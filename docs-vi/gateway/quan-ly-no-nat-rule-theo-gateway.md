# Quản lý NO NAT Rule của Gateway

## Xem danh sách NO NAT Rule

[![Màn hình danh sách NO NAT Rule](/img/migrated/image-1754291539835-89c7d41b.png)](/img/migrated/image-1754291539835-89c7d41b.png)

## Tạo NO NAT Rule

[![Form tạo NO NAT Rule](/img/migrated/image-1754291548065-9e866aa4.png)](/img/migrated/image-1754291548065-9e866aa4.png)

Có hai loại NO NAT:

- **NO Inbound NAT**
- **NO Outbound NAT**

## Chỉnh sửa NO NAT Rule

Ở cột **Actions**, bạn có thể nhấn nút **Edit** để chỉnh sửa thông tin của NO NAT Rule.

## Xóa NO NAT Rule

Ở cột **Actions**, bạn có thể nhấn nút **Delete** để xóa NO NAT Rule.
