# Quản lý Route của Gateway

## Xem danh sách Route

[![Màn hình danh sách Route](/img/migrated/image-1754290897920-2416f962.png)](/img/migrated/image-1754290897920-2416f962.png)

Các thông tin của một Route bao gồm:

- **Route name**
- **CIDR Destination**
- **IP next hop**
- **Status**
- **Created at**

## Tạo Route

Bạn có thể nhấn nút **Create route** để tạo mới một Route.

[![Form tạo Route](/img/migrated/image-1754290924141-cfae2a04.png)](/img/migrated/image-1754290924141-cfae2a04.png)

:::note
Gateway loại **Basic** không hỗ trợ tạo Route. Bạn chỉ có thể tạo Route khi Gateway là loại **Custom**.
:::

## Chỉnh sửa Route

Ở cột **Actions**, bạn có thể nhấn nút **Edit** để chỉnh sửa thông tin của Route.

:::note
Gateway loại **Basic** không hỗ trợ chỉnh sửa Route. Bạn chỉ có thể chỉnh sửa Route khi Gateway là loại **Custom**.
:::

## Xóa Route

Ở cột **Actions**, bạn có thể nhấn nút **Delete** để xóa Route.

:::note
Gateway loại **Basic** không hỗ trợ xóa Route. Bạn chỉ có thể xóa Route khi Gateway là loại **Custom**.
:::
