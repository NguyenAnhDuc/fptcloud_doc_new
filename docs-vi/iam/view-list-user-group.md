# Xem danh sách User Group

1. Trên FPT Portal, chọn **IAM** > **User Groups**.
2. Danh sách User Group được hiển thị. Mỗi User Group bao gồm các thông tin: **Name**, **Description**, **Roles**, **User quantity**, **Created at** và **Actions**.
