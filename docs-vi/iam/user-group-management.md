# Quản lý User Group

Tính năng Quản lý User Group cho phép Tenant owner tạo, xem, chỉnh sửa và xóa các User Group. Mỗi User Group được gán một hoặc nhiều Role, và tất cả IAM user trong cùng User Group sẽ được áp dụng chung các quyền đó.

Xem các hướng dẫn chi tiết trong mục con của trang này.
