# Xoa User Groups

Với các User Groups không còn nhu cầu sử dụng, bạn có thể xóa theo hướng dẫn sau:
**Bước 1** : Ở menu chọn **IAM** > **User Groups** , hệ thống sẽ hiển thị trang **User Groups Management**.
[![file](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)
**Bước 2** : Chọn **Action** ở cuối **Custom Role** cần xóa trên danh sách. Chọn **Delete.**
**Lưu ý: không thể xóa user group đang có user.**
[![file](/img/migrated/image-1734344655938-f07ad3de.png)](/img/migrated/image-1734344655938-f07ad3de.png)
**Bước 3** : Xác nhận thông tin cảnh báo trong popup và chọn **Delete.**
[![file](/img/migrated/image-1734344671384-0a15465e.png)](/img/migrated/image-1734344671384-0a15465e.png)
