# Xóa User Group

Với các User Group không còn nhu cầu sử dụng, thực hiện xóa theo hướng dẫn sau.

1. Chọn **IAM** > **User Groups**. Hệ thống hiển thị trang **User Groups Management**.

[![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. Chọn **Action** ở cuối User Group cần xóa trên danh sách, rồi chọn **Delete**.

:::warning
Không thể xóa User Group đang có user.
:::

[![](/img/migrated/image-1734344655938-f07ad3de.png)](/img/migrated/image-1734344655938-f07ad3de.png)

3. Xác nhận thông tin cảnh báo trong popup và chọn **Delete**.

[![](/img/migrated/image-1734344671384-0a15465e.png)](/img/migrated/image-1734344671384-0a15465e.png)
