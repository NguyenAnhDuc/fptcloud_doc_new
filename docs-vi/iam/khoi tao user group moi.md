# Khởi tạo User Group mới

1. Chọn **IAM** > **User Groups**, sau đó chọn **Create User Group**.

[![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. Trên trang **Create User Group**, nhập các thông tin theo yêu cầu:
   - **Name**: Tên của nhóm người dùng.
   - **Description**: Mô tả nhóm người dùng (không bắt buộc).
   - **Role**: Có thể gán nhiều role cho một User Group.
   - **User**: Có thể thêm user có sẵn trong hệ thống vào User Group.

[![](/img/migrated/image-1734344557099-e110b8a0.png)](/img/migrated/image-1734344557099-e110b8a0.png)

3. Chọn **Create user group**. Hệ thống tiến hành khởi tạo và thông báo kết quả.
