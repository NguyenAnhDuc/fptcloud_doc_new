# Khoi Tao User Group Moi

**Bước 1** : Ở menu chọn **IAM** > **User Groups** , chọn **Create User Group**.
[![file](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)
**Bước 2** : Trên giao diện trang **Create User Group** , nhập các thông tin theo yêu cầu:
  * **Name** : Tên của nhóm người dùng.
  * **Description** : Mô tả nhóm người dùng
  * **Role** : Có thể gán nhiều role cho một nhóm người dùng.
  * **User** : Có thể thêm user có sẵn trong hệ thống để thêm vào user group

[![file](/img/migrated/image-1734344557099-e110b8a0.png)](/img/migrated/image-1734344557099-e110b8a0.png)
**Bước 3** : Chọn **Create user group** , hệ thống sẽ tiến hành khởi tạo và thông báo kết quả.
