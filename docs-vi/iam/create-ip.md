# Khởi tạo một record IP access mới

1. Chọn **IAM**, sau đó chọn tab **IP access**.

[![](/img/migrated/image-1711511716480-fe4d7e22.png)](/img/migrated/image-1711511716480-fe4d7e22.png)

2. Chọn **Create**.

[![](/img/migrated/Screenshot_20-1024x366-374cb3a8.png)](/img/migrated/Screenshot_20-1024x366-374cb3a8.png)

3. Trên trang khởi tạo IP access, nhập các thông tin và đọc kỹ hướng dẫn trước khi cài đặt:
   - **IP Access**: Có thể chọn một hoặc nhiều IP, dãy IP hoặc CIDR IP.
   - **Users**: Có thể chọn một hoặc nhiều user.
   - **User groups**: Có thể chọn một hoặc nhiều User Group — tất cả user trong group sẽ được áp dụng.

[![](/img/migrated/Screenshot_17-1024x549-20d82693.png)](/img/migrated/Screenshot_17-1024x549-20d82693.png)

4. Chọn **Create ip access**. Sau khi khởi tạo thành công, record sẽ xuất hiện trong danh sách.

[![](/img/migrated/Screenshot_19-1024x362-978a41cd.png)](/img/migrated/Screenshot_19-1024x362-978a41cd.png)
