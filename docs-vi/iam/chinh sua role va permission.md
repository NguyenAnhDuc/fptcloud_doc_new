# Chinh Sua Role Va Permission

Bạn có thể thay đổi thông tin và permission của các Custom Role bằng cách.
**Bước 1** : Ở menu chọn **IAM** > **Roles** , hệ thống sẽ hiển thị trang **Role Management**.
[![file](/img/migrated/image-1734344271940-4a804c14.png)](/img/migrated/image-1734344271940-4a804c14.png)
**Bước 2** : Chọn **Action** ở cuối **Custom Role** cần update trên danh sách. 
  * Chọn Edit Basic Information hoặc Edit Permissions **Bước 3** : Thay đổi thông tin Role
  * **Bước 3.1** Edit **Basic Information**

[![file](/img/migrated/image-1734344299974-74ad3c1a.png)](/img/migrated/image-1734344299974-74ad3c1a.png)
  * **Bước 3.2** Edit **Permission**

[![file](/img/migrated/image-1734344312048-ce9f5aea.png)](/img/migrated/image-1734344312048-ce9f5aea.png)
**Lưu ý: Khi role thay đổi sẽ ảnh hưởng đến quyền truy cập của tất cả User trong các User Group có gán role đó.**
