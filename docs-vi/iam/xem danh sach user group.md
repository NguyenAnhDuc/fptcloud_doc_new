# Xem danh sách User Group

1. Trên FPT Portal, chọn **IAM** > **User Groups**.
2. Hệ thống hiển thị danh sách User Group với các thông tin: **Name**, **Description**, **Roles**, **User quantity**, **Created at**, **Actions**.
