# Xem Danh Sach User Group

Bạn có thể xem và quản lý danh sách User Group đã tạo trên trang **User Groups Management.**
Để mở **User Groups Management** , bạn thao tác như sau:
Trên FPT Portal, ở phần **IAM** chọn **User Group**. Hệ thống sẽ hiển thị danh sách User Group bao gồm với các thông tin quan trọng như : 
Name, Description, Roles, User quantity, Created at, Actions
