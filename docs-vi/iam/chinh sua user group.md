# Chinh Sua User Group

**Bước 1** : Ở menu chọn **IAM** >**User Groups** , hệ thống sẽ hiển thị trang **User Groups Management**.
**Bước** 2: Chọn **Action** ở cuối **User Group** cần update trên danh sách. Chọn 1 trong các thông tin muốn edit
  * Manage user
  * Manage role
  * Edit information
