# Chỉnh sửa User Group

1. Chọn **IAM** > **User Groups**. Hệ thống hiển thị trang **User Groups Management**.
2. Chọn **Action** ở cuối User Group cần cập nhật trên danh sách.
3. Chọn một trong các tùy chọn:
   - **Manage user**: Thêm hoặc xóa user trong User Group.
   - **Manage role**: Thêm hoặc xóa role gán cho User Group.
   - **Edit information**: Chỉnh sửa tên và mô tả của User Group.
