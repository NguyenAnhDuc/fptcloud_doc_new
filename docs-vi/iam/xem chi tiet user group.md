# Xem chi tiết User Group

1. Chọn **IAM** > **User Groups**. Hệ thống hiển thị trang **User Groups Management**.
2. Chọn User Group cần xem thông tin trên danh sách. Hệ thống hiển thị trang **User Groups Detail**.
