# Xem Danh Sach Role

Bạn có thể xem và quản lý danh sách role đã tạo trên trang **Roles management.**
Để mở **Roles management** , bạn thao tác như sau:
  * Trên FPT Portal, ở phần IAM chọn Roles. Hệ thống sẽ hiển thị danh sách bao gồm cả basic Role và Custom Role với các thông tin : 
    * **Name, Type (của role), Description, Status, Created at**.
    * **No of group** : số lượng group đang gán role
    * **Actions** : Delete, Edit Basic Information, Edit Permissions
