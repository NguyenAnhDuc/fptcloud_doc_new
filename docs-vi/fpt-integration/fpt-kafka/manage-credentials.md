# Quản lý credential

Credential là cặp username/password dùng để xác thực kết nối đến Kafka theo phương thức SASL/SCRAM-SHA-256. Quản lý credential đúng cách giúp kiểm soát quyền truy cập vào dịch vụ một cách an toàn.

Các thao tác có thể thực hiện:

- [Tạo credential](/docs/vi/fpt-integration/fpt-kafka/create-credentials)
- [Chỉnh sửa credential](/docs/vi/fpt-integration/fpt-kafka/edit-credentials)
- [Xóa credential](/docs/vi/fpt-integration/fpt-kafka/delete-credentials)
