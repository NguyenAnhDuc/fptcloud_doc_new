# Xoa Event Gateway

Để xóa Event Gateway, người dùng thực hiện các bước sau: 
**Bước 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Bước 2:** Tại danh sách **Event Gateway** > chọn **Event Gateway** cần xóa > nhấn **Action** > chọn **Delete**
[![Hình ảnh](/img/migrated/image1b-e0892577.png)](/img/migrated/image1b-e0892577.png)
**Bước 3** : Hiển thị hộp thoại **Delete Application** > nhập **delete** > nhấn **confirm** để xóa hoàn thành việc xóa **Event Gateway**
[![Hình ảnh](/img/migrated/image1c-ca16f7c7.png)](/img/migrated/image1c-ca16f7c7.png)
