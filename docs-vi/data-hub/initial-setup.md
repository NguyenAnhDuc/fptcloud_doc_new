# Thiết lập ban đầu

Hoàn thành các bước sau trước khi sử dụng Data Hub.

## Tạo tài khoản FPT Cloud

1. Truy cập [https://ai.fptcloud.com/](https://ai.fptcloud.com/) và chọn **Sign Up**.
2. Điền thông tin cần thiết để đăng ký.
3. Kiểm tra email để nhận hướng dẫn từ **FPT Smart Cloud** (noreply@fptcloud.com) hoàn tất đăng ký.

## Truy cập Data Hub

1. Đăng nhập [FPT AI Factory Console](https://ai.fptcloud.com/).
2. Trong menu bên trái, chọn **Data Hub**.
