# Hướng dẫn nhanh

## Các bước thực hiện

Làm theo các bước sau để bắt đầu sử dụng **AI Marketplace**.

### 1. Truy cập và đăng ký / đăng nhập

- Truy cập [FPT AI Marketplace Portal](https://marketplace.fptcloud.com/en).
- Đăng ký tài khoản mới hoặc đăng nhập bằng thông tin xác thực hiện có của bạn.

Sau khi đăng nhập, bạn sẽ được chuyển đến trang chủ, nơi bạn có thể duyệt tất cả các mô hình AI hiện có.

---

### 2. Khám phá và chọn mô hình

- Duyệt các mô hình hiện có trên [FPT AI Marketplace Portal](https://marketplace.fptcloud.com/en) (Trang chủ).
- Dùng [Playground](https://marketplace.fptcloud.com/en/playground) để kiểm thử mô hình với các prompt của bạn trước khi quyết định.
- Chọn mô hình phù hợp (ví dụ: LLM hoặc VLM) dựa trên nhu cầu kinh doanh của bạn.

---

### 3. Kích hoạt API và tạo API key

- Vào **My API Keys** trong dashboard của bạn.
- Nhấp **Create API Key** và gán cho dự án của bạn.
- Sao chép và lưu trữ key an toàn — key này cần thiết để xác thực mọi yêu cầu API.

:::tip
Xem [hướng dẫn chi tiết tạo API key](./my-account.md) để biết các bước cụ thể.
:::

---

### 4. Tích hợp API

Tích hợp mô hình đã chọn vào ứng dụng của bạn bằng API.

:::note
Xem [tài liệu tham khảo API đầy đủ](https://github.com/fpt-corp/ai-marketplace) để biết chi tiết endpoint, định dạng yêu cầu và các mẫu code.
:::
