# Thiết lập ban đầu

Để sử dụng **File Storage - High Performance** , bạn cần liên hệ với bộ phận hỗ trợ. Sau khi cung cấp các thông tin cần thiết, tài khoản của bạn sẽ được enable dịch vụ, cấp quota và mở tính năng quản trị **File Storage Management** trên FPT Unify Portal.
Các thông tin cần cung cấp bao gồm:  
| **Metric**  | **Unit**  | **Require**  | **Detail**  |  
| --- | --- | --- | --- |  
| Tenant/Region  |   | Yes  | Tenant/Region cần enable dịch vụ File Storage - High Performance  |  
| Storage Quota  | GB  | Yes  | Dung lượng cần sử dụng  |  
| Danh sách Subnet  |   | Yes  | Mỗi Subnet trên HPC khi sử dụng File Storage - High Performance sẽ có endpoint khác nhau, khách hàng cần cung cấp danh sách subnet muốn truy cập File Storage để team hỗ trợ cung cấp.  |  
| Hiệu năng  | IOPS  | No  | IOPS mong muốn   
_(Nếu không cung cấp sẽ được set theo ngưỡng mặc định)_  |  
| Băng thông  | Gbps  | No  | Băng thông mong muốn   
_(Nếu không cung cấp sẽ được set theo ngưỡng mặc định)_  |  
| Tính năng  |   | No  | Các tính năng mong muốn.  |
