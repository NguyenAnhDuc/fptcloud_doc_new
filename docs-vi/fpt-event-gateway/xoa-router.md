# Xoa Router

Để xóa **Router** , người dùng thực hiện các bước sau: 
**Bước 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Bước 2:** Tại danh sách **Event Gateway** > chọn **Event Gateway** > chọn **tab Security**
**Bước** 3: Tại danh sách **Router** > chọn **Edit** ở Group chứa router cần xóa > chọn icon Delete tại Router cần xóa

[![Hình ảnh](/img/migrated/image28-fb9a35ef.png)](/img/migrated/image28-fb9a35ef.png)
**Bước 3** : Sau khi hoàn tất việc xóa, nhấn **Save** để lưu lại các thay đổi.
