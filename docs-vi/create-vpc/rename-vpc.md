# Đổi tên VPC

:::warning
Hành động đổi tên VPC có thể dẫn đến các cấu hình nội bộ bị sai. Nếu có nhu cầu, liên hệ hotline 1900 638 399 hoặc email support@fptcloud.com để được hỗ trợ.
:::

1. Vào trang danh sách VPC. Tại VPC cần đổi tên, chọn **Actions** → **Rename VPC**.

   ![Menu Actions với tùy chọn Rename VPC](/img/migrated/image-1726823026645-ceaabb35.png)

2. Nhập VPC name mới, nhấn **Rename**.

   ![Hộp thoại Rename VPC](/img/migrated/image-1726823062656-852dc78b.png)
