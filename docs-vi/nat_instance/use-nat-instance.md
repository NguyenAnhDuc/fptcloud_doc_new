# Tính năng chỉ áp dụng cho người dùng mua dịch vụ loại General

:::note
Tính năng này chỉ áp dụng cho người dùng đã mua dịch vụ loại **General**.
:::

Khi bạn chọn mode **isolated** trong quá trình khởi tạo Subnet, hệ thống sẽ hiển thị hướng dẫn cách cấu hình NAT Instance để các instance thuộc isolated network có thể truy cập Internet.
