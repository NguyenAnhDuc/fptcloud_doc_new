# Khởi tạo DDoS Protection

[![Giao diện DDoS Protection chưa kích hoạt](/img/migrated/corero-not-active-1024x462-fa6af4dd.png)](/img/migrated/corero-not-active-1024x462-fa6af4dd.png)

:::note
Nếu bạn chưa thể tự kích hoạt DDoS Protection trên FPT Cloud Portal, hãy liên hệ hỗ trợ:
- Hotline: **1900 638 399**
- Email: **support@fptcloud.com**
:::

Sau khi được kích hoạt, bạn có thể sử dụng các tính năng bên dưới.

[![Giao diện DDoS Protection đã kích hoạt](/img/migrated/corero-active-1024x462-24b1a42c.png)](/img/migrated/corero-active-1024x462-24b1a42c.png)

Hệ thống cho phép quản lý danh sách IP được bảo vệ (toàn bộ IP trong một Tenant).

[![Danh sách IP được bảo vệ](/img/migrated/Create-ip-1-1024x416-b730e124.png)](/img/migrated/Create-ip-1-1024x416-b730e124.png)

## Thêm IP cần bảo vệ

Nhấn **Create IP Protection** và điền các thông tin sau:

- **Traffic Protection**: Gói traffic bảo vệ cho IP.
- **VPC**: Chọn VPC để lọc danh sách Floating IP tương ứng.
- **Floating IP**: Chọn Floating IP cần bảo vệ.

:::tip
Bạn có thể chọn VPC trước để lọc danh sách Floating IP, hoặc chọn Floating IP trước — hệ thống sẽ tự động điền VPC tương ứng.
:::
