# Quản lý applications đã được deploy

App Catalogs Management hỗ trợ quản lý các applications đã được deploy tới cluster. Tại Menu này, khách hàng có thể thực hiện Filter App theo Cluster, Xem chi tiết App đã được deploy, Reconfig, rollback, uninstall applications.
