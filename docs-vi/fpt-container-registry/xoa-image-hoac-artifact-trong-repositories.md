# Xoa Image Hoac Artifact Trong Repositories

Trong trường hợp bạn muốn xóa Image hoặc Artifact đã push lên trước đó để tiết kiệm dung lượng hoặc loại bỏ các project rác, hãy thao tác theo các bước sau: 
**_1. Xoá Repository_**
**Bước 1** : Ở phần **Action** của Image hoặc Artifact chọn **Delete.** [![](/img/migrated/Picture19-1-efab785a.png)](/img/migrated/Picture19-1-efab785a.png)
**Bước 2** : Chọn Delete trong popup Confirmation. [![](/img/migrated/Picture20-1-a67dbf3a.png)](/img/migrated/Picture20-1-a67dbf3a.png)
**_2. Xoá Artifact_**
**Bước 1** : Lấy thông tin Detail Repository
[![](/img/migrated/Picture21-1-14112c71.png)](/img/migrated/Picture21-1-14112c71.png)
**Bước 2** : Ở phần Action của Artifact, chọn Delete 
[![](/img/migrated/Picture22-1-812a2686.png)](/img/migrated/Picture22-1-812a2686.png)
**Bước 3** : Chọn Delete trong popup confirmation
[![](/img/migrated/Picture23-1-7da45a00.png)](/img/migrated/Picture23-1-7da45a00.png)
**_3. Xoá Tag_**
**Bước 1** : Lấy thông tin Detail Tag 
[![](/img/migrated/Picture24-1-65f144c5.png)](/img/migrated/Picture24-1-65f144c5.png)
**Bước 2** : Ở phần Action của Tag, chọn Delete 
[![](/img/migrated/Picture25-609bcb94.png)](/img/migrated/Picture25-609bcb94.png)
**Bước 3** : Chọn Delete trong popup confirmation
[![](/img/migrated/Picture26-f6b122fd.png)](/img/migrated/Picture26-f6b122fd.png)
