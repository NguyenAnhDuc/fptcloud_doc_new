# Enabling Fpt Container Registry Service

If you are the owner of the Tenant or the IAM user who is authorized to use the FPT Container Registry, you can activate the service on the FPT Cloud Portal.
**Step 1** : On the sidebar menu, choose **Container Registry** > click **Active** to activate the service.
[![](/img/migrated/image01-1-8cc0d81b.png)](/img/migrated/image01-1-8cc0d81b.png)
**Step 2** : Hover on the **Help Icon** to get _Docker Login, Tag, and Push image_ commands:
[![](/img/migrated/image02-1-35c7756d.png)](/img/migrated/image02-1-35c7756d.png)
_**Note** : Users who are not the owner of the Tenant will not see the Active button. Choose a Tenant that you have the owner role to continue with setting up the service._
