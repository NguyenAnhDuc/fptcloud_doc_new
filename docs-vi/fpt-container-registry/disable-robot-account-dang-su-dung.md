# Disable Robot Account Dang Su Dung

**Bước 1** : Tương tự như thực hiện Edit Robot Account, tích chọn **Robot Account** muốn disable, sau đó chọn **ACTION DISABLE** : 
[![](/img/migrated/Picture69-5d9785b0.png)](/img/migrated/Picture69-5d9785b0.png)
**Bước 2** : Trên Portal xuất hiện hộp thoại để xác nhận hành động. Nhập **DISABLE** để thực hiện disable account: [![](/img/migrated/Picture70-e994d833.png)](/img/migrated/Picture70-e994d833.png)
**Bước 3** : Kết quả sau khi thực hiện disable, **Enabled State** sẽ hiển thị như hình bên dưới. 
[![](/img/migrated/Picture71-071816c7.png)](/img/migrated/Picture71-071816c7.png)
