# Integration Management - Quản lý tích hợp

Chỉ **Org Admin** hoặc **Team Manager** có quyền chỉnh sửa
**Bước 1** : Đến tab DevSec trong menu Security Platform, chọn **Integration** để xem được danh sách **integration**. Tiến hành chọn Org hoặc Team của bạn:
[![anhfsec](/img/migrated/in1-1-bbf498e7.png)](/img/migrated/in1-1-bbf498e7.png)
_Note: Các loại tích hợp sẽ bị disabled và bạn không thể chọn tích hợp khi chưa chọn org/team_
**Bước 2** : Chọn loại Source Control hoặc Container Registry cần tích hợp:
