# Report

**Bước 1** : Truy cập thanh menu > Queries & Reports
[![file](/img/migrated/image-1765765637950-de5f4b98.png)](/img/migrated/image-1765765637950-de5f4b98.png)
**Bước 2** : Chọn Share Groups > Report
[![file](/img/migrated/image-1765765653374-e18ef375.png)](/img/migrated/image-1765765653374-e18ef375.png)
Mặc định FPT Cloud đã có sẵn các templates để báo cáo theo dõi các hành vi phát hiện mã độc bao gồm:
  * Báo cáo các hành vi của Mã độc (24h/7days/30days)
  * Báo cáo các hành vi exploit (24h/7days/30days)
  * Báo cáo các hành vi chạy crack-tools (24h/7days/30days)

User có thể tự custom các template bằng cách click > New Query
