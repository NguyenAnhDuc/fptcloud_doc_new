# Xóa address pair

## Cách 1: Xóa tại màn hình quản lý Allow address pair

1. Chọn biểu tượng xóa (delete icon).

   [![Biểu tượng xóa address pair](/img/migrated/image-1708395674995-65847886.png)](/img/migrated/image-1708395674995-65847886.png)

2. Xác nhận xóa.

   [![Xác nhận xóa address pair](/img/migrated/image-1708395828130-9894ec41.png)](/img/migrated/image-1708395828130-9894ec41.png)

## Cách 2: Xóa trong Instance Management

1. Vào **Network Interface** trong Instance Management.

   [![Màn hình Network Interface trong Instance Management](/img/migrated/image-1708395867460-1d20cf7a.png)](/img/migrated/image-1708395867460-1d20cf7a.png)

2. Bấm vào biểu tượng X.

   [![Biểu tượng X để xóa address pair](/img/migrated/image-1708395893416-0efccabb.png)](/img/migrated/image-1708395893416-0efccabb.png)

3. Xác nhận xóa.

   [![Xác nhận xóa address pair](/img/migrated/image-1708395920272-52eddcc4.png)](/img/migrated/image-1708395920272-52eddcc4.png)
