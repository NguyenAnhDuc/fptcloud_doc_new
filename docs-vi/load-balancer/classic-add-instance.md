# Thêm máy ảo vào Classic Load Balancer

Để mở rộng Load Balancer, bạn có thể thêm máy ảo vào Load Balancer hiện có. Hãy tạo máy ảo mới và cài đặt dịch vụ cần thiết trong Instance Management trước, sau đó thêm vào Load Balancer.

**Bước 1:** Truy cập trang chi tiết Load Balancer bằng cách nhấn vào tên Load Balancer trong bảng **Load Balancer Management**.

[![](/img/migrated/image-1712807219267-78e51205.png)](/img/migrated/image-1712807219267-78e51205.png)

**Bước 2:** Trong tab **Instances**, danh sách máy ảo đang gắn vào Load Balancer được hiển thị. Nhấn **Select instances**.

[![](/img/migrated/image-1712807226342-138669ae.png)](/img/migrated/image-1712807226342-138669ae.png)

**Bước 3:** Trong popup **Change instances**, chọn các máy ảo cần thêm trong trường **Instances**.

[![](/img/migrated/image-1712807234311-3e65aba3.png)](/img/migrated/image-1712807234311-3e65aba3.png)

**Bước 4:** Nhấn **Save**.

[![](/img/migrated/image-1712807240125-47822fc9.png)](/img/migrated/image-1712807240125-47822fc9.png)
