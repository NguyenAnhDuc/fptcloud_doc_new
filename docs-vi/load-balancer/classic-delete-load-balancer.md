# Xóa Classic Load Balancer

Khi Load Balancer không còn cần thiết, hãy xóa để giải phóng tài nguyên và tránh phát sinh chi phí không cần thiết.

**Bước 1:** Trong bảng **Load Balancer Management**, nhấn **Delete** trong menu **Action** của Load Balancer cần xóa.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-15-1024-225988fc.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-15-1024-225988fc.png)

**Bước 2:** Popup xác nhận xuất hiện. Nhấn **Delete** để xác nhận.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-16-6b6db13c.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-16-6b6db13c.png)
