# Import an SSL certificate

**Step 1:** In the Load Balancer list, select the **Certificate** tab.

**Step 2:** Click **Import SSL Certificate** and enter:

- **Name**: A name for the certificate. Use a name that clearly identifies it, for example the domain name covered by the certificate.
- **Certificate**: The digital certificate in standard PEM format.
- **Private Key**: The private key for the certificate in standard PEM format.

**Step 3:** Click **Save** to import, or **Cancel** to discard.
