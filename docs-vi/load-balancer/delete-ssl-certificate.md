# Xóa chứng chỉ SSL

**Bước 1:** Trong danh sách chứng chỉ SSL, nhấn biểu tượng **thùng rác** của chứng chỉ cần xóa.

**Bước 2:** Hộp thoại xác nhận xuất hiện. Nhấn **Delete SSL Certificate** để xác nhận, hoặc **Cancel** để hủy.
