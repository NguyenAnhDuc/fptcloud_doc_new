# Xóa report

1. Vào **Home > Apps > Reporting**.
2. Tại màn danh sách report, click icon **Trash** của report muốn xóa.

   [![Danh sách report với icon xóa](/img/migrated/Picture15-c1a6f8b0.png)](/img/migrated/Picture15-c1a6f8b0.png)

3. Xuất hiện hộp thoại xác nhận. Chọn **Delete** để xóa report, hoặc **Cancel** để hủy bỏ thao tác.

   [![Hộp thoại xác nhận xóa](/img/migrated/Picture16-19c4a8c7.png)](/img/migrated/Picture16-19c4a8c7.png)
