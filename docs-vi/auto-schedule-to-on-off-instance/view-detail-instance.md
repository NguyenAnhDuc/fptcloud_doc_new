# Xem chi tiết một instance schedule

**Bước 1:** Truy cập trang **Instance Schedule Detail** bằng cách chọn tên instance schedule trên bảng quản trị **Instance Schedule Management**.

[![Danh sách instance schedule](/img/migrated/image-1695609011987-33ef2b1d.png)](/img/migrated/image-1695609011987-33ef2b1d.png)

**Bước 2:** Xem thông tin chi tiết của instance schedule trong mục **Overview**, bao gồm: Name, Description, Start time, Stop time, Start Date, End Date, Frequency và danh sách máy ảo gắn kèm.

[![Tổng quan instance schedule](/img/migrated/image-1695609025652-03872f17.png)](/img/migrated/image-1695609025652-03872f17.png)

**Bước 3:** Xem thông tin chi tiết trong mục **Attached instances**, bao gồm: danh sách máy ảo đính kèm cùng các thông tin như instance name, instance status, latest start time và status, latest stop time và status.

[![Danh sách máy ảo đính kèm](/img/migrated/image-1695609035976-7b5310a1.png)](/img/migrated/image-1695609035976-7b5310a1.png)
