# Xóa VPN Site-to-Site

:::danger
Hành động xóa VPN Site-to-Site không thể hoàn tác. Kết nối sẽ bị ngắt vĩnh viễn.
:::

1. Chọn kết nối VPN, nhấn **Actions** → **Delete**.

   ![Menu Delete VPN](/img/migrated/Config-delete-1024x354-9bb9ed67.png)

2. Xác nhận xóa. VPN chuyển sang trạng thái **Deleting** và bị xóa khỏi danh sách.

   ![Hộp thoại xác nhận xóa VPN](/img/migrated/Config-delete-confirm-1024x445-4c33a072.png)
