# Vô hiệu hóa VPN Site-to-Site

Khi cần tạm dừng kết nối VPN đang hoạt động (Active), bạn có thể vô hiệu hóa (Disable) kết nối đó.

1. Chọn kết nối VPN cần Disable, nhấn **Actions**.

   ![Chọn VPN cần Disable](/img/migrated/Config-15-1024x262-574882a1.png)

   ![Menu Actions với tùy chọn Disable](/img/migrated/Disable-VPN-1024x354-0338f037.png)

2. Xác nhận việc Disable.

   ![Hộp thoại xác nhận Disable VPN](/img/migrated/Disable-VPN-confirm-1024x445-a8ee08e4.png)

3. VPN chuyển sang trạng thái **Deactivating** → **Inactive**.

   ![VPN ở trạng thái Inactive](/img/migrated/Config-inactive-1024x238-c1dec1de.png)
