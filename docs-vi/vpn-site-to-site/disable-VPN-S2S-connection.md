# Disable Vpn S2S Connection

Vô hiệu hóa VPN đang hoạt động (nếu VPN đang Active).
**Bước 1** : Chọn kết nối VPN cần Disable.
[![Config 15](/img/migrated/Config-15-1024x262-574882a1.png)](/img/migrated/Config-15-1024x262-574882a1.png)
[![Disable VPN](/img/migrated/Disable-VPN-1024x354-0338f037.png)](/img/migrated/Disable-VPN-1024x354-0338f037.png)
**Bước 2:** Xác nhận việc Disable. 
[![Disable VPN confirm](/img/migrated/Disable-VPN-confirm-1024x445-a8ee08e4.png)](/img/migrated/Disable-VPN-confirm-1024x445-a8ee08e4.png)
**Bước 3:** VPN sẽ chuyển sang trạng thái **Deactivating** > **Inactive**. 
[![Config inactive](/img/migrated/Config-inactive-1024x238-c1dec1de.png)](/img/migrated/Config-inactive-1024x238-c1dec1de.png)
