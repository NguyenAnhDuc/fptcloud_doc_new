# Xem thông tin VPN

Theo dõi trạng thái kết nối và thông tin cấu hình giúp đảm bảo VPN hoạt động ổn định.

1. Trên trang quản lý VPN, nhấn vào tên VPN muốn kiểm tra.

   ![Danh sách VPN](/img/migrated/image-1742913956469-c8950bf1.png)

2. Tại tab **Status**, xem các thông tin sau:
   - Thông tin kết nối
   - Thông tin lỗi
   - Lưu lượng (Traffic)
   - Hướng dẫn kết nối và VPN test guide

   Nhấn **Refresh** để đồng bộ dữ liệu mới nhất.

   ![Chi tiết VPN - tab Status](/img/migrated/image-1742914122300-0e3c8fb8.png)

3. Tại tab **Connection Info**, xem thông tin cấu hình. Nhấn **Download Config** để lưu cấu hình ra file .txt.

   ![Chi tiết VPN - tab Connection Info](/img/migrated/image-1742914184152-67ad6fa7.png)
