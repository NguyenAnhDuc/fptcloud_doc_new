# Tải cấu hình VPN

Tải file cấu hình VPN giúp bạn nhanh chóng cài đặt kết nối phía local.

Chọn kết nối VPN, nhấn **Actions** → **Download Config**. Hệ thống tải file cấu hình VPN về máy.
