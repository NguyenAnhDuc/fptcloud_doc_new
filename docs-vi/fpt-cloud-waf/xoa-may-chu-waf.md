# Xóa máy chủ WAF

:::danger
Thao tác này sẽ xóa toàn bộ dữ liệu của máy chủ WAF và **không thể phục hồi**. Hãy kiểm tra kỹ trước khi thực hiện.
:::

1. Trên màn hình **Cloud Web Application Firewall Management**, chọn **Delete WAF** tại máy chủ cần xóa.

   [![Chọn Delete WAF](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)

2. Trong hộp thoại xác nhận, chọn **Delete** để hoàn tất.

   [![Xác nhận xóa máy chủ WAF](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)
