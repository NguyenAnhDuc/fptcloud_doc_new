# Xoa Proxy Host

Lưu ý: Xóa Proxy Host sẽ khiến máy chủ web của bạn không còn được bảo vệ, hãy cẩn thận khi thực hiện thao tác này.
Để xóa Proxy Host khỏi hệ thống, bạn thao tác như sau:
**Bước 1:** Ở **WAF Dashboard** , chọn **Hosts** > **Proxy**.
[![Userguide FPT WAF 2022 17](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)
**Bước 2:** Ở **Proxy** cần xóa chọn icon **Delete**.
[![Userguide FPT WAF 2022 24](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)
**Bước 3:** Hộp thoại xác nhận hiện lên. Chọn **Delete** để xóa **Proxy.**
[![Userguide FPT WAF 2022 25](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)
