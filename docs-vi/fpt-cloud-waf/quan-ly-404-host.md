# Quản lý 404 host

**404 host** là tính năng điều hướng trang lỗi 404 về trang mong muốn.

1. Chọn **Hosts** > **404 Not Found** từ menu để mở màn hình quản lý.

   [![Màn hình quản lý 404 host](/img/migrated/Userguide-FPT-WAF-2022-32-1024x502-a4e51c1a.png)](/img/migrated/Userguide-FPT-WAF-2022-32-1024x502-a4e51c1a.png)

2. Chọn **Create 404 Not Found** để mở màn hình thêm mới và nhập thông tin:

   - **Domain Names**: Tên domain/website được điều hướng khi gặp lỗi 404.
   - **Custom Configuration**
   - **SSL Certificate**: Chọn SSL certificate tương ứng cho domain/website.

   [![Nhập thông tin 404 host](/img/migrated/Userguide-FPT-WAF-2022-33-1024x502-98cf258a.png)](/img/migrated/Userguide-FPT-WAF-2022-33-1024x502-98cf258a.png)

3. Chọn **Create 404 Not Found** để lưu và hoàn tất.
