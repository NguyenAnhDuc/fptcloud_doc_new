# Đăng nhập WAF Dashboard

1. Sau khi khởi tạo máy chủ WAF thành công, truy cập hộp thư đến của email đã dùng để tạo WAF. Lấy thông tin đăng nhập gồm: **Link dashboard**, **Username**, **Password**.

   [![Email chứa thông tin đăng nhập WAF Dashboard](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)

2. Truy cập **Link dashboard** và thực hiện đăng nhập.

   [![Màn hình đăng nhập WAF Dashboard](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)
