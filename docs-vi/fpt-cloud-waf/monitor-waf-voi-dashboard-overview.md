# Theo dõi WAF qua Dashboard Overview

**Home dashboard** là trang tổng quan cho phép quản trị viên theo dõi các thông số và số liệu của máy chủ WAF theo thời gian thực.

Trong **WAF Dashboard**, chọn **Overview**. Màn hình hiển thị các chỉ số được cập nhật liên tục:

| Metrics | Mô tả |
|---|---|
| **Total Hosts** | Số lượng proxy host đã tạo |
| **Total Redirection** | Số lượng redirection host đã tạo |
| **Total Stream** | Số lượng stream host đã tạo |
| **Total Not Found Hosts** | Số lượng 404 Not Found host đã tạo |
| **Active Connections** | Số connection đang kết nối đến các proxy host |
| **Security Events** | Tổng số security event WAF đã chặn và ghi nhận |
| **Top By Attack Type** | Các loại tấn công nhiều nhất WAF đã chặn |
| **Top By RuleID** | Các rule bị vi phạm nhiều nhất |
| **Top By Host** | Các proxy host ghi nhận nhiều security event nhất |
| **Top By Source Attack** | Các IP nguồn tấn công nhiều lần nhất |
