# Bật/tắt proxy host

1. Trong **WAF Dashboard**, chọn **Hosts** > **Proxy**.

   [![Danh sách proxy host](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. Tại proxy host cần thay đổi trạng thái, chọn icon **More** > **Toggle**.

   [![Chọn Toggle proxy host](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)

3. Trong hộp thoại xác nhận, chọn **Run Action** để thực hiện.

   [![Xác nhận bật/tắt proxy host](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)
