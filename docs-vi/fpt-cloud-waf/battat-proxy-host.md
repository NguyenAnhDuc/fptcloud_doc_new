# Battat Proxy Host

**Bước 1:** Ở **WAF Dashboard** , chọn **Hosts** > **Proxy**.
[![Userguide FPT WAF 2022 17](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)
**Bước 2:** Ở **Proxy** cần thay đổi trạng thái Bật/Tắt chọn icon **More** > Chọn chức năng **Toggle**.
[![Userguide FPT WAF 2022 22](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)
**Bước 3:** Hộp thoại xác nhận hiện lên. Chọn **Run Action** để thực hiện thao tác**.**
[![Userguide FPT WAF 2022 23](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)
