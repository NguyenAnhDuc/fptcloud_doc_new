# Sử dụng tính năng SSO

Sau khi cấu hình SSO, bạn có thể đăng nhập vào hệ thống theo 2 cách:

## Cách 1: Vào bằng đường dẫn

1. Truy cập đường dẫn: `http://sub-domain.fptcloud.com`
2. Xác thực thông tin đăng nhập

   [![Màn hình xác thực SSO](/img/migrated/image-1734056582232-ddd3fa5b.png)](/img/migrated/image-1734056582232-ddd3fa5b.png)

3. Hệ thống xác thực đăng nhập và chuyển hướng bạn vào trang quản lý instance

## Cách 2: Vào bằng trang select-method

1. Truy cập đường dẫn: `https://console.fptcloud.com/login/select-method`
2. Chọn **Login with your organization**

   [![Chọn phương thức đăng nhập](/img/migrated/image-1734056612511-805a1ea0.png)](/img/migrated/image-1734056612511-805a1ea0.png)

3. Nhập sub-domain bạn đã đăng ký

   [![Nhập sub-domain](/img/migrated/image-1734056636838-3134f1b3.png)](/img/migrated/image-1734056636838-3134f1b3.png)

4. Xác thực thông tin đăng nhập

   [![Xác thực đăng nhập](/img/migrated/image-1734056657577-d806a4b5.png)](/img/migrated/image-1734056657577-d806a4b5.png)

5. Hệ thống xác thực đăng nhập và chuyển hướng bạn vào trang quản lý instance
