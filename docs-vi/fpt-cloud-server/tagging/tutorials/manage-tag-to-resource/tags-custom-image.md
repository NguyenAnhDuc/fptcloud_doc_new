# Quản lý tag cho Custom Image

Gắn tag cho Custom Image giúp bạn phân loại các image theo dự án hoặc môi trường — thuận tiện cho việc tìm kiếm và tái sử dụng.

## Gắn tag khi upload image

1. Chọn **Custom Image** từ menu, nhấn **Upload image**.

   [![Màn hình Custom Image với nút Upload image](/img/migrated/image-1721032879356-5bd46d9a.png)](/img/migrated/image-1721032879356-5bd46d9a.png)

2. Tại hộp thoại **Upload image**, chọn tag trong trường **Add tag (optional)**.

   [![Hộp thoại Upload image với trường Add tag](/img/migrated/image-1721032892734-ae8a432b.png)](/img/migrated/image-1721032892734-ae8a432b.png)

## Quản lý tag cho Custom Image đã tạo

1. Chọn **Custom Image** từ menu, nhấn **Actions** → **Manage tag**.

   [![Menu Actions với tùy chọn Manage tag cho Custom Image](/img/migrated/image-1721032908795-71940b90.png)](/img/migrated/image-1721032908795-71940b90.png)

2. Hộp thoại hiển thị, chọn tag để gắn vào image.

   [![Hộp thoại Manage tags cho Custom Image](/img/migrated/image-1721032920130-4755a194.png)](/img/migrated/image-1721032920130-4755a194.png)

3. Để gỡ tag, nhấn dấu **x** bên cạnh tên tag. Nhấn dấu **x** ở cuối listbox để gỡ toàn bộ tag. Nhấn **Save** để lưu thay đổi.

   [![Hộp thoại Manage tags với tùy chọn gỡ tag](/img/migrated/image-1721032931446-41441a71.png)](/img/migrated/image-1721032931446-41441a71.png)

:::warning
Khi xóa image, hệ thống tự động gỡ các tag liên kết và cập nhật danh sách tag trên màn hình Tagging.
:::
