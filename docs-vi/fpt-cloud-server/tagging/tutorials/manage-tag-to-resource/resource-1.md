# Quản lý tag cho tài nguyên

Tag giúp bạn phân loại và lọc tài nguyên nhanh chóng. Bạn có thể gắn tag cho nhiều loại tài nguyên trên FPT Cloud — giúp theo dõi chi phí và quản lý hạ tầng hiệu quả hơn.

## Tài nguyên hỗ trợ tag

- Instance Management
- Storage Disk
- Snapshot
- Custom Image
- Floating IP
- Subnet
- Security Group
- Classic Load Balancer

## Thời điểm gắn tag

Bạn có thể gắn hoặc gỡ tag tại hai thời điểm:

- Trong lúc khởi tạo tài nguyên.
- Sau khi tài nguyên đã được tạo.

:::note
Một tag có thể gắn cho nhiều tài nguyên và một tài nguyên có thể gắn nhiều tag.
:::
