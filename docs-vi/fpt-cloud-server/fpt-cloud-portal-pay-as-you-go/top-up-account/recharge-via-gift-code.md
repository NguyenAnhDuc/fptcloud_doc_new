# Nạp tiền thông qua gift code

Bạn có thể nạp tiền vào tài khoản bằng gift code nhận được từ FPT Smart Cloud.

1. Truy cập trang **Billing**, chọn menu **Topup**.

[![Trang Billing – Topup](/img/migrated/image-1694073773226-c126e916.png)](/img/migrated/image-1694073773226-c126e916.png)

2. Điền mã gift code vào ô nhập liệu.

[![Nhập mã gift code](/img/migrated/image-1694073782136-bded0c89.png)](/img/migrated/image-1694073782136-bded0c89.png)

3. Nhấn **Redeem gift code**.

:::note
Nếu gift code hợp lệ và còn trong thời gian có hiệu lực, hệ thống sẽ tự động cộng tiền vào tài khoản.
:::
