# Điều kiện yêu cầu

Trước khi truy cập FPT Cloud Desktop, bạn cần đảm bảo các điều kiện sau:

## Đăng nhập với tài khoản SSO (Azure AD)

- Sử dụng PC hoặc laptop có thể truy cập link FPT Cloud Desktop qua Internet hoặc mạng LAN
- Được cung cấp đầy đủ thông tin truy cập máy ảo bao gồm URL FCD, tài khoản và ứng dụng Authenticator tương ứng

## Đăng nhập với Account/Password

- Sử dụng PC hoặc laptop có thể truy cập link FPT Cloud Desktop qua Internet hoặc mạng LAN
- Được cung cấp đầy đủ thông tin truy cập máy ảo bao gồm URL FCD và tài khoản (URL FCD FCI: `fci.fcd.fptcloud.com`)
