# Khóa và mở khóa xóa máy ảo

Tính năng khóa xóa giúp bạn bảo vệ máy ảo quan trọng khỏi việc bị xóa nhầm — giảm rủi ro mất dữ liệu do lỗi thao tác.

## Khóa xóa máy ảo (Lock Deletion)

1. Tại **Instance Management**, chọn máy ảo, nhấn **Actions** → **Lock Deletion**.

   ![Menu Actions với tùy chọn Lock Deletion](/img/migrated/image-1712738228214-76b8b81a.png)

2. Hộp thoại cảnh báo hiển thị tên máy ảo. Nhấn **Lock Instance Deletion** để xác nhận.

   ![Hộp thoại xác nhận Lock Instance Deletion](/img/migrated/image-1712738219490-01d185fc.png)

Máy ảo sẽ không thể bị xóa cho đến khi được mở khóa.

## Mở khóa xóa máy ảo (Unlock Deletion)

1. Tại **Instance Management**, chọn máy ảo, nhấn **Actions** → **Unlock Deletion**.

   ![Menu Actions với tùy chọn Unlock Deletion](/img/migrated/image-1712738202636-17abdc91.png)

2. Hộp thoại cảnh báo hiển thị tên máy ảo. Nhấn **Unlock Instance Deletion** để xác nhận.

   ![Hộp thoại xác nhận Unlock Instance Deletion](/img/migrated/image-1712738209302-86a7aea2.png)

Máy ảo đã mở khóa có thể được xóa bình thường.
