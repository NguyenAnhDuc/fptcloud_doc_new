# Gateway

Gateway là thành phần mạng quan trọng cho phép bạn quản lý lưu lượng vào/ra VPC. Hệ thống hỗ trợ hai loại Gateway: **Basic** (cấu hình tự động) và **Custom** (tự cấu hình theo nhu cầu).

:::note
Tính năng này chỉ dành cho một số Tenant có cấu hình đặc biệt. Vui lòng liên hệ đội ngũ hỗ trợ để được kích hoạt.
:::
