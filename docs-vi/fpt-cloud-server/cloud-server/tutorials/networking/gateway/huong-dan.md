# Hướng dẫn truy cập Gateway

1. Đăng nhập vào [console.fptcloud.com](https://console.fptcloud.com).
2. Sau khi đăng nhập thành công, chọn VPC cần xem dữ liệu Gateway.
3. Ở sidebar, chọn và mở rộng menu **Network** rồi chọn **Gateway**.

:::note
Tính năng này chỉ dành cho một số Tenant có cấu hình đặc biệt. Vui lòng liên hệ đội ngũ hỗ trợ để được kích hoạt.
:::
