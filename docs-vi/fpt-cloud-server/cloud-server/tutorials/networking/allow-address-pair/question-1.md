# Làm thế nào để thay đổi cấu hình address pair đã tạo?

Hiện tại hệ thống chưa hỗ trợ tính năng chỉnh sửa các address pair đã khởi tạo. Để có thể thay đổi cấu hình, bạn vui lòng xóa các address pair đã khởi tạo và tạo address pair mới.
