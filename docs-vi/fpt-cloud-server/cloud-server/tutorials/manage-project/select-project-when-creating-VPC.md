# Chọn Project khi tạo VPC

Khi tạo VPC mới, bạn có thể gắn nó vào một Project hiện có. Điều này đảm bảo tất cả tài nguyên trong VPC được liên kết với đúng dự án để theo dõi chi phí và kiểm soát truy cập.

:::tip
Gắn VPC vào Project ngay khi tạo là cách tiếp cận được khuyến nghị — giúp tránh việc phải phân bổ lại tài nguyên sau này.
:::
