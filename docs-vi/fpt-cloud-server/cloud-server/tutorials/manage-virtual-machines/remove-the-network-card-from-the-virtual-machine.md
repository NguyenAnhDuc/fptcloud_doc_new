# Gỡ card mạng (NIC) khỏi máy ảo

Khi không còn cần kết nối đến một Subnet, bạn có thể gỡ card mạng khỏi máy ảo để đơn giản hóa cấu hình mạng.

## Gỡ card mạng

1. Mở tab **Network Interface** của máy ảo.

   [![Tab Network Interface hiển thị danh sách card mạng](/img/migrated/image-1744793241168-180450d8.png)](/img/migrated/image-1744793241168-180450d8.png)

2. Tại bảng **Network**, chọn **Actions** → **Delete** ở card mạng cần gỡ.

   [![Menu Actions với tùy chọn Delete cho card mạng](/img/migrated/image-1744793619214-47123e3d.png)](/img/migrated/image-1744793619214-47123e3d.png)

3. Hộp thoại xác nhận hiển thị. Nhấn **Delete** để xác nhận gỡ card mạng.

   [![Hộp thoại xác nhận gỡ card mạng](/img/migrated/image-1712723201176-b2e4e0e8.png)](/img/migrated/image-1712723201176-b2e4e0e8.png)
