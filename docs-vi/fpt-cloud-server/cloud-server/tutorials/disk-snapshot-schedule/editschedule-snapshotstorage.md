# Chỉnh sửa lịch snapshot ổ đĩa

1. Tại tab **Storage snapshot schedule**, chọn **Action** → **Edit**.

   [![Menu Action Edit](/img/migrated/image-1766045109643-13e019dd.png)](/img/migrated/image-1766045109643-13e019dd.png)

2. Chỉnh sửa thông tin.

   [![Chỉnh sửa lịch](/img/migrated/image-1766045137152-94c1a305.png)](/img/migrated/image-1766045137152-94c1a305.png)

3. Nhấn **Update schedule**.
