# Tổng quan Custom Images

Custom Images cho phép bạn tải lên và quản lý các image hệ điều hành tùy chỉnh — giúp triển khai máy ảo với cấu hình đã chuẩn bị sẵn, tiết kiệm thời gian thiết lập.
