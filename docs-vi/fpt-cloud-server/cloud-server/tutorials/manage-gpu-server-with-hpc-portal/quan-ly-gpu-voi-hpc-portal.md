# Quản lý GPU Server với HPC Portal

[HPC Portal](https://hpc.fptcloud.com/) là trang quản trị GPU Server vật lý cho các dòng RTX 3090, RTX 4090 của FPT Cloud — cung cấp giao diện quản lý chuyên biệt cho workload GPU hiệu năng cao.

:::note
Để đăng nhập và khởi tạo máy tại HPC Portal, liên hệ quản trị viên để phân quyền và cấp tài nguyên cho tài khoản.
:::
