# Đổi cấu hình máy

Khi nhu cầu thay đổi và cấu hình máy chủ hiện tại quá cao hoặc quá thấp, bạn có thể sử dụng tính năng Change Package để điều chỉnh mà không ảnh hưởng đến thiết lập máy.

:::warning
Cần tắt máy trước khi sử dụng tính năng Change Package.
:::

## Đổi cấu hình

1. Chọn image muốn thay đổi, nhấn **Change package**.
2. Chọn gói cấu hình mới phù hợp và xác nhận.

:::note
Sau khi Change Package, hãy kiểm tra lại card GPU trên máy và update phiên bản mới nhất của NVIDIA Studio Driver để đảm bảo máy chạy ổn định.
:::
