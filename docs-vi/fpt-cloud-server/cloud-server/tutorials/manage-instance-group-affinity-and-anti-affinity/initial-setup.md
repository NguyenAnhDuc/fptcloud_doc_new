# Thiết lập ban đầu

Nếu đây là lần đầu tiên bạn sử dụng, hãy hoàn thành các bước sau.

## Điều kiện tiên quyết

- Có tài khoản FPT Cloud.
- Đã đăng nhập vào FPT Cloud Portal.

## Đăng nhập FPT Cloud Portal

1. Truy cập [https://console.fptcloud.com](https://console.fptcloud.com/).
2. Đăng nhập bằng tài khoản FPT Cloud đã được cấp.
3. Chọn đúng **Tenant**, **Region**, **VPC** cần sử dụng dịch vụ.

:::tip
Nếu chưa có tài khoản FPT Cloud, truy cập [https://fptcloud.com](https://fptcloud.com/) và chọn **Sign Up** để đăng ký. Bộ phận hỗ trợ sẽ liên hệ xác nhận thông tin.
:::
