# Xóa Instance Group

:::danger
Thao tác này không thể hoàn tác. Xóa Instance Group sẽ đồng thời gỡ Instance Group khỏi tất cả Instance đã gắn.
:::

1. Trên bảng **Instance Group Management**, chọn **Delete** trong cột **Action**.

   ![Nút Delete trên bảng Instance Group Management](/img/migrated/image-1705025148984-2af9c584.png)

2. Hộp thoại xác nhận hiện lên. Gõ `delete` và nhấn **Delete Instance Group**.

   ![Hộp thoại xác nhận xóa Instance Group](/img/migrated/image-1705025187846-59abb769.png)

Instance Group bị xóa hoàn toàn. Các Instance đã gắn được giải phóng.
