# Tổng quan NAT Instance

Tính năng NAT Instance hỗ trợ các instance trong mạng cô lập (isolated network) có thể truy cập hệ thống bên ngoài Internet, ví dụ như cài đặt phần mềm hoặc kết nối về On-premise.
