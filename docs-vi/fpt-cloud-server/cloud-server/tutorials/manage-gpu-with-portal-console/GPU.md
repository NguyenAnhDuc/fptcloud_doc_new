# Quản lý GPU với Portal Console

FPT GPU Server là dịch vụ máy chủ ảo triển khai trên FPT Cloud với GPU card chuyên dụng — đem đến khả năng xử lý tính toán vượt trội cho AI, Machine Learning, kết xuất đồ họa và xử lý hình ảnh.

Dịch vụ kết hợp tính linh hoạt của FPT Cloud Server với hiệu năng mạnh mẽ của các dòng GPU Card như NVIDIA Tesla A10, A30, A100, RTX 3090 và 4090.

:::tip
Nếu bạn có nhu cầu sử dụng GPU hoặc gặp vấn đề kỹ thuật, hãy liên hệ đội ngũ CSKH để được hỗ trợ.
:::
