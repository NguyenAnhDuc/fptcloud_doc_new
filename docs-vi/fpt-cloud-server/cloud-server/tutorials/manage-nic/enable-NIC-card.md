# Enable card NIC

Đối với card NIC đang ở trạng thái Inactive, bạn có thể enable để kết nối mạng trở lại.

1. Chọn **Enable Card** trên card NIC cần bật.

[![Nút Enable Card](/img/migrated/image-1691382154697-155c022c.png)](/img/migrated/image-1691382154697-155c022c.png)

2. Xác nhận thao tác Enable Card.

[![Xác nhận Enable Card](/img/migrated/image-1691382171803-02c83019.png)](/img/migrated/image-1691382171803-02c83019.png)

3. Chờ hệ thống xử lý.

[![Hệ thống đang xử lý](/img/migrated/image-1691382523333-cd5f8412.png)](/img/migrated/image-1691382523333-cd5f8412.png)

4. Card chuyển sang trạng thái **Active** và Instance có thể kết nối mạng qua card này.

[![Card NIC ở trạng thái Active](/img/migrated/image-1691382532014-ecb97957.png)](/img/migrated/image-1691382532014-ecb97957.png)
