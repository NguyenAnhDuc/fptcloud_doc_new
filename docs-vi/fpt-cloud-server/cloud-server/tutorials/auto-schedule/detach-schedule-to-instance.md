# Gỡ máy ảo khỏi instance schedule

Bạn có thể gỡ các máy ảo ra khỏi schedule khi không còn nhu cầu sử dụng. Sau khi gỡ, máy ảo sẽ giữ nguyên trạng thái hiện tại cho đến khi bạn thao tác thủ công.

**Bước 1:** Trên bảng quản trị **Instance Schedule Management**, trong phần **Action**, chọn **Apply instances**.

**Bước 2:** Chọn dấu "x" bên cạnh tên instance muốn gỡ để loại khỏi schedule. Bạn cũng có thể chọn dấu "x" ở cuối listbox để gỡ toàn bộ máy ảo cùng lúc. Sau đó chọn **Apply instances**.
