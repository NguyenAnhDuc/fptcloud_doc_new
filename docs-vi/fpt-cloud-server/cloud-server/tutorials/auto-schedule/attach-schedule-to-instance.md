# Gắn máy ảo vào instance schedule

Sau khi tạo instance schedule, bạn cần gắn các máy ảo vào schedule đó để sử dụng tính năng tự động bật/tắt.

**Bước 1:** Trên bảng quản trị **Instance Schedule Management**, trong phần **Action**, chọn **Apply instances**.

[![Nút Apply instances trong danh sách](/img/migrated/image-1695609073911-03ac0ac2.png)](/img/migrated/image-1695609073911-03ac0ac2.png)

**Bước 2:** Chọn tên các instance muốn gắn vào schedule, sau đó chọn **Apply instances**.

[![Giao diện chọn instance](/img/migrated/image-1695609084366-c567a73d.png)](/img/migrated/image-1695609084366-c567a73d.png)
