# Sử dụng instance schedule cho máy ảo

Instance schedule cho phép bạn tự động hóa việc bật và tắt các máy ảo theo lịch định sẵn. Để sử dụng tính năng này, bạn cần thực hiện theo trình tự:

1. [Tạo một instance schedule mới](/docs/vi/fpt-cloud-server/cloud-server/tutorials/auto-schedule/create-new-instance) với thông tin thời gian và tần suất phù hợp.
2. [Gắn các máy ảo](/docs/vi/fpt-cloud-server/cloud-server/tutorials/auto-schedule/attach-schedule-to-instance) vào schedule vừa tạo.
3. Theo dõi trạng thái thực thi trong trang [chi tiết instance schedule](/docs/vi/fpt-cloud-server/cloud-server/tutorials/auto-schedule/view-detail-instance).

Khi không còn nhu cầu sử dụng, bạn có thể [gỡ máy ảo](/docs/vi/fpt-cloud-server/cloud-server/tutorials/auto-schedule/detach-schedule-to-instance) ra khỏi schedule hoặc [xóa schedule](/docs/vi/fpt-cloud-server/cloud-server/tutorials/auto-schedule/delete-instance-schedule).
