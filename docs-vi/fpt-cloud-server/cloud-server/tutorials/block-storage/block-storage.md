# Tổng quan Block Storage

FPT Block Storage cung cấp ổ đĩa sử dụng cho máy ảo trên FPT Cloud — quản lý tương tự một ổ cứng vật lý khi gắn vào máy ảo.

FPT Cloud Portal có **Storage Disk Management** giúp bạn quản lý toàn bộ Storage Disk đã tạo trong VPC: xem danh sách, tạo mới, attach/detach, chỉnh sửa và xóa.

## Truy cập Storage Disk Management

Chọn **Compute Engine** → **Storage Disks** từ menu.

[![Màn hình Storage Disk Management](/img/migrated/Userguide-FPT-Cloud-Server-2022-55-1024x-0c126930.png)](/img/migrated/Userguide-FPT-Cloud-Server-2022-55-1024x-0c126930.png)

Danh sách hiển thị: **Name**, **Storage type**, **Size**, **Created at**, **Attached** (máy ảo đang gắn).

[![Danh sách Storage Disk](/img/migrated/Userguide-FPT-Cloud-Server-2022-56-1024x-abee045f.png)](/img/migrated/Userguide-FPT-Cloud-Server-2022-56-1024x-abee045f.png)
