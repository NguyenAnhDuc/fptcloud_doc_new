# Tôi có thể hiệu chỉnh các Address Pair đã khởi tạo không?

Hiện tại hệ thống chưa hỗ trợ hiệu chỉnh Address Pair đã tạo. Để thay đổi cấu hình, bạn cần xóa Address Pair hiện tại và tạo mới.
