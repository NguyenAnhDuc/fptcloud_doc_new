# Xóa Group

1. Truy cập trang **Autoscaling > Autoscale Group**. Trên dòng tương ứng với group cần xóa, nhấn vào biểu tượng **Action menu**.

   [![Menu hành động của Group](/img/migrated/Screenshot-2024-10-02-023008-8d0f1521.png)](/img/migrated/Screenshot-2024-10-02-023008-8d0f1521.png)

2. Chọn **Delete**.

   [![Tùy chọn Delete trong menu](/img/migrated/Screenshot-2024-10-02-023134-da955e2d.png)](/img/migrated/Screenshot-2024-10-02-023134-da955e2d.png)

3. Một hộp thoại xuất hiện. Nếu tên group chính xác, nhập **delete** và chọn **Delete group** để xác nhận. Chọn **Cancel** để hủy thao tác.

   [![Hộp thoại xác nhận xóa Group](/img/migrated/Screenshot-2024-10-02-023251-6a7e8d6c.png)](/img/migrated/Screenshot-2024-10-02-023251-6a7e8d6c.png)

   [![Hộp thoại xóa Group đã xác nhận](/img/migrated/Screenshot-2024-10-02-023337-021ad4b9.png)](/img/migrated/Screenshot-2024-10-02-023337-021ad4b9.png)
