# Xem Queues

**Bước 1** : Chọn queue bạn muốn xem thông tin chi tiết 
[![](/img/migrated/View-1-bb456837.png)](/img/migrated/View-1-bb456837.png)
**Bước 2** : Chọn nút View 
[![](/img/migrated/View-2-cec0d10d.png)](/img/migrated/View-2-cec0d10d.png)
Hiển thị các thông tin mà bạn đã tạo bao gồm: 
  * **Name** : tên queue
  * **Status** : Trạng thái của queue
  * **Các thông tin đã nhập** bao gồm: max length, max length byte, message TTL, overflow, …

Các thông tin này chỉ có quyền xem, không có quyền chỉnh sửa.
