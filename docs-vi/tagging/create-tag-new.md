# Khởi tạo tag mới

Tạo tag giúp bạn phân loại tài nguyên theo mục đích, môi trường hoặc chủ sở hữu — giúp quản lý và tìm kiếm tài nguyên nhanh hơn.

## Điều kiện tiên quyết

- ✅ Đã đăng nhập vào [FPT Cloud Portal](https://console.fptcloud.com/).
- ✅ Có quyền tạo tag trên Tenant hoặc VPC.

## Tạo tag

1. Chọn **Tagging** từ menu bên trái, nhấn **Create tag**.

   [![Màn hình Tagging với nút Create tag](/img/migrated/image-1721012726449-a6958240.png)](/img/migrated/image-1721012726449-a6958240.png)

2. Điền các thông tin sau:
   - **Tag Level**: chọn Tenant, Project hoặc VPC.
   - **Resource Scope**: đối với level Project/VPC, chọn resource scope tương ứng.

   [![Hộp thoại Create tag với các trường Tag Level và Resource Scope](/img/migrated/image-1764325536204-b41d710f.png)](/img/migrated/image-1764325536204-b41d710f.png)

3. Nhấn **Create** để tạo tag.

Tag mới hiển thị trong danh sách tại trang Tagging.

[![Danh sách tag sau khi tạo thành công](/img/migrated/image-1764325568380-5a0fdb03.png)](/img/migrated/image-1764325568380-5a0fdb03.png)
