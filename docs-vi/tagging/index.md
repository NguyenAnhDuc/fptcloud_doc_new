# Tổng quan Tagging

Tag giúp bạn quản lý, xác định, sắp xếp, tìm kiếm và lọc các tài nguyên trên FPT Cloud. Bạn có thể tạo tag để phân loại tài nguyên theo mục đích, chủ sở hữu, môi trường phát triển hoặc các tiêu chí riêng — giúp tối ưu việc vận hành và theo dõi chi phí.

## Thành phần của tag

Mỗi tag gồm 2 phần:

- **Key**: Tên hiển thị khóa tag. Ví dụ: `Env`, `Project`.
- **Value**: Giá trị cụ thể liên quan đến key. Ví dụ: `Production`, `FPT`. Bạn có thể để trống trường này.
