# Resource 1

Người dùng có thể gắn và gỡ tag đối với các tài nguyên sau:
  * Instance Management
  * Storage disk
  * Snapshot
  * Custom Image
  * Floating IP
  * Security Group
  * Subnet
  * Classic Load Balancer

Người dùng có thể thêm hoặc gỡ tag vào các thời điểm sau:
  * Trong lúc người dùng khởi tạo tài nguyên
  * Sau khi người dùng tạo tài nguyên

**Lưu ý: Một tag có thể được gắn cho nhiều resource name và một resource name cũng có thể được gắn với nhiều tag.**
