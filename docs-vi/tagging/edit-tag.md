# Edit Tag

Người dùng thao tác như sau:
**Bước 1** : Ở menu chọn **Tagging** , tại tag cần điều chỉnh, chọn **Actions** > chọn **Edit**
[![file](/img/migrated/image-1764325579411-6236079e.png)](/img/migrated/image-1764325579411-6236079e.png)
**Bước 2** : Người dùng có thể thay đổi Key – Value – Color của tag. Người dùng KHÔNG THỂ thay đổi Tag level, tuy nhiên có thể thay đổi resource scope đổi với level Project/VPC
[![file](/img/migrated/image-1764325607970-6f8635d5.png)](/img/migrated/image-1764325607970-6f8635d5.png)
