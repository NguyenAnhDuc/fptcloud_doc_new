# Thay đổi cấu hình K8s cluster

Bạn có thể thực hiện các thay đổi cấu hình sau trên Kubernetes cluster hiện có trong D-FKE:

- Scale worker node
- Thay đổi flavor (cấu hình) của node
- Thay đổi flavor (cấu hình) của NFS Server
- Tăng kích thước storage của node
- Tăng kích thước storage của NFS Server
- Nâng cấp phiên bản Kubernetes
- Thêm NFS Persistent Storage
