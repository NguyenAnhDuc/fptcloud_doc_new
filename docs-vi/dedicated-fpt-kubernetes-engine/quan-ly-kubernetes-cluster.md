# Quản lý Kubernetes cluster

Dedicated FPT Kubernetes Engine (D-FKE) hỗ trợ quản lý toàn bộ vòng đời Kubernetes cluster với các thao tác sau:

- Tạo mới Kubernetes cluster
- Xem danh sách cluster đã tạo
- Truy cập thông tin chi tiết cluster
- Lấy thông tin truy cập cluster
- Xoá Kubernetes cluster
- Dừng Kubernetes cluster
- Khởi động lại Kubernetes cluster
