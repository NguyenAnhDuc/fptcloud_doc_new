# Tang Storage Size Cua Nodes

**Bước 1** : Ở menu chọn **Kubernetes** , hệ thống sẽ hiển thị trang **Kubernetes Management**.
[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)
**Bước 2** : Chọn **Cluster** muốn thay đổi cấu hình nodes trên danh sách.
[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)
**Bước 3** : Chọn tab **Node Pools** , bạn có thể thay đổi Storage size của Master Nodes, Worker Nodes bằng cách chọn icon Setting ở config tương ứng.
[![](/img/migrated/Picture46-45189119.png)](/img/migrated/Picture46-45189119.png)
**Bước 4** : Nhập dung lượng cần tăng thêm > Chọn **Extend** để thực hiện tăng dung lượng storage.
[![](/img/migrated/Picture47-e9a1ce8b.png)](/img/migrated/Picture47-e9a1ce8b.png)
