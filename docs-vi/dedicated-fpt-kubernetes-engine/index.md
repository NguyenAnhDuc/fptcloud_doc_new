# Dedicated FPT Kubernetes Engine (DFKE)

Dedicated FPT Kubernetes Engine (DFKE) là dịch vụ Kubernetes được quản lý trên FPT Cloud, cung cấp các cluster chuyên dụng với toàn quyền kiểm soát node, mạng và storage. Người dùng có thể tạo, scale, nâng cấp và quản lý Kubernetes cluster thông qua FPT Cloud Portal.

Với D-FKE, khách hàng quản lý toàn bộ Kubernetes cluster bao gồm Master Nodes (chứa các thành phần control plane) và Worker Nodes, trong khi các tính năng như autoscale, service load balancer, upgrade version và persistent volume vẫn được hỗ trợ đầy đủ.
