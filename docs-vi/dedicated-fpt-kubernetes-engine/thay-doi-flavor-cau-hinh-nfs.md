# Thay Doi Flavor Cau Hinh Nfs

**Bước 1** : Ở menu chọn **Kubernetes** , hệ thống sẽ hiển thị trang **Kubernetes Management**.
[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)
**Bước 2** : Chọn **Cluster** muốn thay đổi cấu hình nodes trên danh sách.
[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)
**Bước 3** : Chọn tab **Advanced** , bạn có thể thay đổi Flavor (cấu hình CPU, RAM) của **NFS Server** bằng cách chọn icon Setting ở config tương ứng.
[![](/img/migrated/Picture42-804cb91c.png)](/img/migrated/Picture42-804cb91c.png)
**Bước 4** : Chọn cấu hình mong muốn > Chọn **Change** để thực hiện thay đổi.
[![](/img/migrated/Picture43-f4575115.png)](/img/migrated/Picture43-f4575115.png)
