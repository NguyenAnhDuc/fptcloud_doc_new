# GPU Container

Hướng dẫn tạo container GPU trên nền tảng AI Factory của FPT Cloud.

## Cách tạo container

Để tạo GPU Container mới, bạn truy cập vào mục **GPU Container** trong menu bên trái và nhấp vào nút **Create New Container**.

Xem hướng dẫn chi tiết tại phần [Tạo Container qua giao diện](/docs/vi/gpu-container/create-via-gui).
