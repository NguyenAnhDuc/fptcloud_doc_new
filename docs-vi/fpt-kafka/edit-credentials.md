# Chỉnh sửa credential

Khi cần cập nhật mật khẩu để tăng cường bảo mật hoặc sau khi xảy ra sự cố, bạn có thể thay đổi mật khẩu của credential trực tiếp từ giao diện quản lý.

1. Chọn credential muốn đổi mật khẩu, sau đó nhấn **Edit**.

   [![Màn hình danh sách credential với nút Edit](/img/migrated/Pic1_sua_cre-83c837b2.png)](/img/migrated/Pic1_sua_cre-83c837b2.png)

2. Nhập mật khẩu mới.

   [![Màn hình nhập mật khẩu mới cho credential](/img/migrated/Pic2_Sua_cre-f249da7b.png)](/img/migrated/Pic2_Sua_cre-f249da7b.png)

3. Nhấn **OK** để lưu.
