# Quản lý topic

Topic là đơn vị tổ chức dữ liệu trung tâm trong Kafka, nơi producer gửi message và consumer đọc message. Quản lý topic đúng cách giúp tối ưu hiệu suất và kiểm soát luồng dữ liệu trong hệ thống.

Các thao tác có thể thực hiện:

- [Tạo topic](./create-topics.md)
- [Xem topic](./view-topics.md)
- [Chỉnh sửa topic](./edit-topics.md)
- [Xóa topic](./delete-topics.md)
