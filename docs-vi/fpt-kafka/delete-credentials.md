# Delete Credentials

In order to delete a credential, follow the steps:
**Step 1** : Select the credential to delete > Click **Remove**.
[![](/img/migrated/Pic1_XoaCre-4d7c6047.png)](/img/migrated/Pic1_XoaCre-4d7c6047.png)
**Step 2** : In the confirmation dialog, select **Yes** to confirm deletion.
[![](/img/migrated/Pic2_remove-354c9e9d.png)](/img/migrated/Pic2_remove-354c9e9d.png)
