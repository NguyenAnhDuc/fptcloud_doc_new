# Delete Session

**Lưu ý:** Xóa một session sẽ xóa vĩnh viễn toàn bộ dữ liệu trong bộ nhớ tạm thời và bộ nhớ lâu dài. Hãy đảm bảo bạn đã lưu lại bất kỳ dữ liệu nào bạn muốn truy cập sau này.

1. Mở **Session list**.
2. Tìm session bạn muốn xóa và nhấp vào biểu tượng 3 chấm trong cột **Actions**.
3. Nhấp vào hành động **Delete**.
4. Hộp thoại xác nhận xuất hiện. Xác nhận bằng cách nhập "delete" vào trường văn bản và nhấp **Confirm**.
