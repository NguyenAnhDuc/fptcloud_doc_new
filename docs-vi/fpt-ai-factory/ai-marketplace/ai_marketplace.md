# AI Marketplace

AI Marketplace là nền tảng Model-as-a-Service của FPT Cloud, cho phép bạn truy cập ngay các mô hình AI được huấn luyện sẵn qua API — không cần thiết lập hạ tầng.

## Bạn có thể làm gì

- **Chạy inference** — gọi các mô hình LLM, VLM, TTS, STT và embedding qua REST API chuẩn.
- **Fine-tune mô hình** — tùy chỉnh các foundation model cho trường hợp sử dụng của bạn bằng LoRA.
- **Triển khai mô hình tùy chỉnh** — phục vụ các LoRA adapter đã fine-tune dưới dạng API endpoint riêng.
- **Kiểm thử trên Playground** — thử nghiệm tương tác với bất kỳ mô hình nào trước khi viết code.
