# Start Pipeline

1. Mở **Danh sách Pipeline**
2. Tìm Pipeline bạn muốn khởi động và nhấp vào biểu tượng 3 chấm trong cột **Actions**
3. Nhấp vào hành động **Start**

**Lưu ý:** Đối với người dùng PAYG

Để chạy pipeline thành công, hãy đảm bảo:
  * Số dư tài khoản đủ. Trang tham khảo: <https://ai.fptcloud.com/pricing>
  * Tạo tối đa **20** pipeline
  * Chỉ **2** pipeline có thể được khởi động đồng thời tại bất kỳ thời điểm nào
  * Mỗi pipeline có thể được cấu hình với tối đa 2 node

**Lưu ý:** Đối với người dùng ký hợp đồng
