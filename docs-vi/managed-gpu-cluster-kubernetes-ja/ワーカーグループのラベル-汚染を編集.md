# Chỉnh sửa Label và Taint của Worker Group

Label và taint là tính năng quan trọng để quản lý và phân phối workload hiệu quả trong hệ thống có nhiều worker group, giúp bạn dễ dàng phân nhóm theo mục đích sử dụng, hiệu năng hoặc khu vực địa lý. Managed GPU Cluster hỗ trợ bạn thêm, chỉnh sửa và xóa label/taint trên Unify Portal.

**Bước 1:** Trong menu, chọn **[AI Infrastructure] > [Managed GPU Cluster]** để hiển thị trang **[Quản lý Managed GPU Cluster]**. Chọn cluster bạn muốn chỉnh sửa label/taint.

[![](/img/migrated/24-2bafdf17.png)](/img/migrated/24-2bafdf17.png)

**Bước 2:** Chọn Node Pool > Chỉnh sửa Worker

[![](/img/migrated/25-62db8512.png)](/img/migrated/25-62db8512.png)

**Bước 3:** Nhập label và taint muốn thêm vào worker group rồi nhấn nút **[Lưu]**.

[![](/img/migrated/26-888e5fac.png)](/img/migrated/26-888e5fac.png)

[![](/img/migrated/27-1e045ce9.png)](/img/migrated/27-1e045ce9.png)

[![](/img/migrated/28-8da95fc4.png)](/img/migrated/28-8da95fc4.png)

**Lưu ý:** Việc chỉnh sửa label và taint sẽ mất vài phút và trạng thái cluster sẽ chuyển sang **Đang xử lý**. Bạn không thể chỉnh sửa cluster cho đến khi quá trình hoàn tất.
