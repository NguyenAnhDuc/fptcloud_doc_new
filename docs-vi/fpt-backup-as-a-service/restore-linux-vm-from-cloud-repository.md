# Restore Linux vm From Cloud Repository

**Bước 1:** Để restore 1 VM Linux bạn cần download file ISO Veeam recovery media từ trang chủ của VEEAM. Download file ISO phù hợp với hệ điều hành của VM cần khôi phục. [![file](/img/migrated/image-1695294618113-7017717b.png)](/img/migrated/image-1695294618113-7017717b.png)
**Bước 2:** Khởi động lại VM và boot từ ISO Veeam recovery media. Đồng ý với điều khoản sau đó tiếp tục. [![file](/img/migrated/image-1695294644895-2a9750be.png)](/img/migrated/image-1695294644895-2a9750be.png) **Bước 3:** Cấu hình lại network nếu cần Trường hợp VM khởi tạo để restore add network có DHCP thì không cần Configure network và có thể bỏ qua bước 3 này Còn lại cần phải click vào “Configure network” để đặt IP static cho VM restore. Note: IP của VM này cần được mở kết nối ra internet hoặc đến baas-sgn-gw.fptcloud.com (site HCM) và baas-han-gw.fptcloud.com port 6180 để thực hiện restore [![file](/img/migrated/image-1695294671316-44391ccf.png)](/img/migrated/image-1695294671316-44391ccf.png) Thực hiện chọn edit a connection để đặt IP manual cho VM restore: [![file](/img/migrated/image-1695294681820-3a5125db.png)](/img/migrated/image-1695294681820-3a5125db.png) Chọn Edit để set IP cho VM [![file](/img/migrated/image-1695294692535-3e2748f0.png)](/img/migrated/image-1695294692535-3e2748f0.png) Chọn IPv4 Configuration sau đó chọn mode Manual -> Show để đặt IP [![file](/img/migrated/image-1695294702919-9bc47e74.png)](/img/migrated/image-1695294702919-9bc47e74.png) Sau khi đặt xong IP chọn OK và nhấn ESC để back lại màn hình chính để tiến hành restore [![file](/img/migrated/image-1695294720487-cfae6dc7.png)](/img/migrated/image-1695294720487-cfae6dc7.png) **Bước 4:** Chọn Add Cloud provider [![file](/img/migrated/image-1695294739863-99ac30bf.png)](/img/migrated/image-1695294739863-99ac30bf.png) **Bước 5:** Nhập URL gateway của BaaS: baas-sgn-gw.fptcloud.com (site HCM) hoặc baas-han-gw.fptcloud.com (site Hà Nội) [![file](/img/migrated/image-1695294750751-e258de28.png)](/img/migrated/image-1695294750751-e258de28.png) **Bước 6:** Nhập Username/password của bạn. Sau đó ấn Enter để tiếp tục. [![file](/img/migrated/image-1695294761178-6db27dc0.png)](/img/migrated/image-1695294761178-6db27dc0.png) **Bước 7:** Chọn tên backup job và restore poin mong muốn. [![file](/img/migrated/image-1695294772745-817c2f15.png)](/img/migrated/image-1695294772745-817c2f15.png) **Bước 8:** Map các ổ đĩa tương ứng. [![file](/img/migrated/image-1695294786115-e28a2a8c.png)](/img/migrated/image-1695294786115-e28a2a8c.png) Restore các ổ đĩa tương ứng.
