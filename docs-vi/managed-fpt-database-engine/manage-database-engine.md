# Manage Database Engine

## View the Database Engine list

1. From the menu select **Database Engine**.
2. The **Database Management** page displays a list of all clusters.

Status descriptions:
- **Running:** Database Engine is operating normally.
- **Processing:** Database Engine is being processed.
- **Stopped:** Database Engine is stopped.
- **Failed:** Database Engine creation or operation has failed.

## Stop, start, and restart a Database Engine

1. Select **Actions** for the Cluster ID you want to operate on.
2. Select the desired operation (**Start** / **Stop** / **Restart**).
3. When the confirmation popup appears, re-enter the Cluster ID to confirm.

## Delete a Database Engine

1. Select **Actions** for the Cluster ID to delete.
2. Select **Delete**.
3. In the confirmation popup, re-enter the Cluster ID and click **Delete**.

:::warning
Deletion cannot be undone. Verify that a backup exists before deleting.
:::
