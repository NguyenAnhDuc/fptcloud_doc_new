# Update and delete backup jobs

After creating a Full or Diff backup job, you can update or delete the backup job schedule.

1. In the **Full backup job** or **Diff backup job** tab, select **Update job**.
   - Update the job schedule information and click **Update**.

2. In the **Full backup job** or **Diff backup job** tab, select **Delete job**.
   - The **Delete Backup job Cluster** popup appears. Re-enter the Cluster ID to confirm deletion and click **Delete**.
