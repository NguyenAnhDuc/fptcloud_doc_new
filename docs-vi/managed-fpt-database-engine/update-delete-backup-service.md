# Update and delete backup service

Use these steps when you want to update or delete a backup service.

:::note
Deleting the backup service immediately stops the two existing backup jobs until the service is re-enabled.
:::

**Case 1: Update backup service**

In the **Action Panel** tab, select **Update Backup**.

- The **Backup Capacity** popup appears. Enter the new Capacity and click **Save**.

**Case 2: Delete backup service**

Select **Delete Backup**.

- The **Delete Backup Cluster** popup appears. Re-enter the Cluster ID to confirm deletion, then click **Delete**.
