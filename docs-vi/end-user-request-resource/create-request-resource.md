# Create Request Resource

**Bước 1** : Ở tab bar chọn **Tenant** cần cung cấp tài nguyên, ở thanh menu bar chọn **Request Resource** , chọn **Create reques** t.
[![file](/img/migrated/image-1703744342441-a0f93b7b.png)](/img/migrated/image-1703744342441-a0f93b7b.png)
**Bước 2** : Điền các thông tin vào form yêu cầu cung cấp tài nguyên
  * Các trường thông tin bắt buộc bao gồm: Title, Request type, Service, Quantity
  * Các trường thông tin còn lại không bắt buộc.

[![file](/img/migrated/image-1703744369903-d102105e.png)](/img/migrated/image-1703744369903-d102105e.png)
[![file](/img/migrated/image-1703744387522-1dd6ab81.png)](/img/migrated/image-1703744387522-1dd6ab81.png)
**Bước 3** : Chọn **Submit request** để gửi yêu cầu tới FCI. Hệ thống sẽ gửi email xác nhận đã tiếp nhận yêu cầu tới người tạo và đồng thời thông báo tới đội ngũ Sale/Support của FCI để hỗ trợ thực hiện yêu cầu. 
[![file](/img/migrated/image-1703744428466-add11d9d.png)](/img/migrated/image-1703744428466-add11d9d.png)
