# Xem thông tin API Gateway

Để xem thông tin **API Gateway**, bạn thực hiện các bước sau:

**Bước 1:** Tại thanh menu chọn **Integration** > chọn **API Gateway**.

**Bước 2:** Tại danh sách **API Gateway** > chọn **API Gateway** cần xem thông tin.

- Tab **Overview**: Hiển thị thông tin cơ bản của API Gateway: Basic information, Launch Configuration, Etcd, LB size, Endpoint.

[![Hình ảnh](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)

- Tab **Autoscaling**: Hiển thị thông tin Autoscaling của **API Gateway**.

[![Hình ảnh](/img/migrated/SCR-20250924-nuvs-077cada2.png)](/img/migrated/SCR-20250924-nuvs-077cada2.png)

- Tab **Dashboard**: Hiển thị thông tin Dashboard của API Gateway. Nếu khi tạo API Gateway bạn chọn **Dashboard = Disable**, tab Dashboard sẽ không hiển thị.

[![Hình ảnh](/img/migrated/z7045379122842_6adcfbe63dafe93587e68b587-a1a084d9.jpg)](/img/migrated/z7045379122842_6adcfbe63dafe93587e68b587-a1a084d9.jpg)
