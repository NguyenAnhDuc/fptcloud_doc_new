# Xóa API Gateway

Để xóa **API Gateway**, bạn thực hiện các bước sau:

**Bước 1:** Tại thanh menu chọn **Integration** > chọn **API Gateway**.

**Bước 2:** Tại danh sách **API Gateway** > chọn **API Gateway** cần xóa > nhấn **Action** > chọn **Delete**.

[![Hình ảnh](/img/migrated/z7045391397635_d1a00a9228efc6171ae123d65-6c413f1c.jpg)](/img/migrated/z7045391397635_d1a00a9228efc6171ae123d65-6c413f1c.jpg)

**Bước 3:** Hiển thị hộp thoại **Delete Application** > nhập **delete** > nhấn **Confirm** để hoàn thành việc xóa **API Gateway**.

[![Hình ảnh](/img/migrated/z7045438315669_cb9abecccf0015459dc9825c2-824ce48c.jpg)](/img/migrated/z7045438315669_cb9abecccf0015459dc9825c2-824ce48c.jpg)
