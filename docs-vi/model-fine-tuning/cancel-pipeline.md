# Cancel Pipeline

**Lưu ý:** Hủy pipeline này sẽ dẫn đến mất toàn bộ dữ liệu và các checkpoint chưa được lưu.

1. Mở **Danh sách Pipeline**
2. Tìm Pipeline bạn muốn hủy và nhấp vào biểu tượng 3 chấm trong cột **Actions**
3. Nhấp vào hành động **Cancel**
4. Hộp thoại xác nhận xuất hiện. Xác nhận bằng cách chọn lý do và nhấp nút **Confirm**
