# Delete Pipeline

**Lưu ý:** Xóa pipeline sẽ xóa vĩnh viễn toàn bộ dữ liệu trong bộ nhớ tạm thời và bộ nhớ liên tục. Hãy đảm bảo bạn đã lưu mọi dữ liệu mà bạn muốn truy cập lại.

1. Mở **Danh sách Pipeline**
2. Tìm Pipeline bạn muốn xóa và nhấp vào biểu tượng 3 chấm trong cột **Actions**
3. Nhấp vào hành động **Delete**
4. Hộp thoại xác nhận xuất hiện. Xác nhận bằng cách nhập "delete" vào trường văn bản và nhấp **Confirm**
