# Select Volume

Chọn loại volume phù hợp để đảm bảo hiệu suất tối ưu và phân bổ tài nguyên trong quá trình model fine-tuning.

[![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/Volume.png)](https://fptcloud.com/wp-content/uploads/2025/02/Volume.png)

Chúng tôi cung cấp hai loại volume:

| Volume | Phù hợp nhất cho | Cách hoạt động |
| --- | --- | --- |
| **Managed volume** | Dataset của bạn nhỏ hơn 20GiB. | - Được hệ thống tự động cấp phát. - Lý tưởng cho các thử nghiệm nhanh và tác vụ fine-tuning |
| **Dedicated network volume** | Dataset của bạn lớn hơn 20GiB. | - Yêu cầu cấp phát thủ công. - Bạn phải liên hệ nhóm hỗ trợ để yêu cầu |
