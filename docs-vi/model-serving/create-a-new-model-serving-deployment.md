# Tạo deployment mới

1. Trong **Model Serving**, nhấn **Create Deployment**.
2. Chọn mô hình và phiên bản cần triển khai.
3. Cấu hình tài nguyên (GPU, RAM, replicas).
4. Nhấn **Deploy** để triển khai.
