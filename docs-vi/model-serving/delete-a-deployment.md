# Xóa deployment

1. Trong danh sách deployment, chọn deployment cần xóa.
2. Nhấn **Delete**.
3. Xác nhận xóa trong popup.

:::warning
Xóa deployment sẽ dừng tất cả API endpoint liên quan. Hành động không thể hoàn tác.
:::
