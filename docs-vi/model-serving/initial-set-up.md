# Thiết lập ban đầu

1. Truy cập [FPT AI Factory Console](https://ai.fptcloud.com/).
2. Trong menu bên trái, chọn **Model Serving**.
3. Đảm bảo bạn đã có mô hình sẵn sàng để triển khai.
