# Danh sách deployment

**Bước 1:** Truy cập **AI Platform → Model Serving → Deployment**

[![Alt text](/img/migrated/10-4142972a.png)](/img/migrated/10-4142972a.png)

Thông tin được hiển thị theo dạng bảng, liệt kê tất cả các cụm Deployment.

| **Thông tin** | **Mô tả** |
| --- | --- |
| Model Serving Name | Tên của Model Serving |
| Model Type | Loại mô hình đang được triển khai |
| Instance Replica | Số lượng instance hiện đang được sử dụng trong Deployment |
| Status | Trạng thái của Deployment (xem phần Trạng thái và Hành động) |
| Action | Các hành động có sẵn cho Deployment (xem phần Trạng thái và Hành động) |
