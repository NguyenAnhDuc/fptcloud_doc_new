# Chi tiết Run

Trang **Run details** cung cấp tổng quan toàn diện về một fine-tuning job. Trang này bao gồm metadata, cài đặt cấu hình và các chỉ số.
[![Alt text](/img/migrated/Run-details-30dc2510.png)](/img/migrated/Run-details-30dc2510.png)
Bạn có thể xem:
  * **Input:** Dữ liệu đầu vào của bài kiểm thử, ví dụ: một câu hỏi trong nhiệm vụ Question Answering.
  * **Ground Truth:** Câu trả lời đúng (nhãn) tương ứng với mỗi đầu vào, dùng để so sánh với kết quả dự đoán.
  * **Output:** Câu trả lời mà mô hình tạo ra dựa trên đầu vào.
  * **Metrics:** Được tính dựa trên sự so sánh giữa output và ground truth.
