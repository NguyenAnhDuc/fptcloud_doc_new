# Standard - Chọn Dữ liệu Kiểm thử

[![Alt text](/img/migrated/Test-data-fe6dfa7e.png)](/img/migrated/Test-data-fe6dfa7e.png)
Bạn có hai cách để truyền **Dữ liệu kiểm thử**:
  * Tải lên file
    1. Giá trị mặc định là Upload file
    2. Chọn file từ máy tính của bạn.
    3. (Tùy chọn) Nhấp **Download sample** để xem ví dụ về định dạng mong đợi.

**Lưu ý:** Đảm bảo file khớp với định dạng dữ liệu đã chọn  
| Định dạng dữ liệu hỗ trợ  | Định dạng file hỗ trợ  | Kích thước file hỗ trợ  |  
| --- | --- | --- |  
| Alpaca  | - CSV   
- JSON   
- JSONLINES   
- ZIP   
- PARQUET  | Giới hạn 100MB  |  
| ShareGPT  | - JSON   
- JSONLINES   
- ZIP   
- PARQUET  | Giới hạn 100MB  |  
| ShareGPT_Image  | - ZIP   
- PARQUET  | Giới hạn 100MB  |  
  * Kết nối đến Data Hub
    1. Nhấp **Data Hub**
    2. Chọn kết nối hoặc bộ dữ liệu từ Data Hub. **Lưu ý:** Đảm bảo bộ dữ liệu tương thích với định dạng đã chọn.
    3. (Tùy chọn) Nhấp **Open Data Hub** để xem trước hoặc quản lý bộ dữ liệu.
    4. (Tùy chọn) Nhấp **biểu tượng Reload** để cập nhật danh sách kết nối và bộ dữ liệu.
    5. Làm theo hướng dẫn chi tiết về Data Hub
