# Khi nào nên sử dụng Model Testing - Test Jobs

**Model Testing - Test Jobs** có giá trị nhất khi bạn cần đánh giá hiệu suất tổng thể, độ tin cậy và khả năng mở rộng của mô hình đã fine-tuned trước khi triển khai.
Bạn nên sử dụng **Test Jobs** khi:
  * Bạn muốn xác nhận hiệu suất mô hình **ở quy mô lớn**.
  * Bạn muốn theo dõi các cải tiến **qua các phiên bản mô hình**.
  * Bạn cần **các chỉ số hiệu suất định lượng**.
