# IaC (Infrastructure as Code)

Infrastructure as Code (IaC) cho phép bạn quản lý và cấu hình hạ tầng FPT Cloud thông qua mã nguồn thay vì thao tác thủ công trên giao diện. FPT Cloud hỗ trợ Terraform với OpenStack Provider để tự động hóa việc tạo và quản lý tài nguyên.
