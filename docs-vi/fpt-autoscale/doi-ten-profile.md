# Đổi tên Profile

1. Truy cập trang **Autoscaling > Autoscale Profile**. Nhấn vào tên profile cần đổi tên.

   [![Nhấn vào tên profile để đổi tên](/img/migrated/Screenshot-2024-10-01-164339-e3a959ae.png)](/img/migrated/Screenshot-2024-10-01-164339-e3a959ae.png)

2. Nhập tên mới cho profile và chọn **Update profile** để xác nhận.

   [![Form đổi tên Profile](/img/migrated/screencapture-console-fptcloud-000823-IN-9da981c0.png)](/img/migrated/screencapture-console-fptcloud-000823-IN-9da981c0.png)

   Sau khi cập nhật thành công, profile hiển thị trong danh sách với tên mới:

   [![Danh sách profile sau khi đổi tên](/img/migrated/Screenshot-2024-10-01-164932-fe6bd0c4.png)](/img/migrated/Screenshot-2024-10-01-164932-fe6bd0c4.png)
