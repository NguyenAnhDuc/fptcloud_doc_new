# Đổi tên Profile

##  **Bước 1** : Truy cập trang **Autoscaling > Autoscale Profile**. Click vào tên profile cần đổi tên.
[![click to rename profile](/img/migrated/Screenshot-2024-10-01-164339-e3a959ae.png)](/img/migrated/Screenshot-2024-10-01-164339-e3a959ae.png)
##  **Bước 2** : Nhập tên mới cho profile và chọn **Update profile** để xác nhận.
[![rename profile](/img/migrated/screencapture-console-fptcloud-000823-IN-9da981c0.png)](/img/migrated/screencapture-console-fptcloud-000823-IN-9da981c0.png)
Sau khi cập nhật thành công, profile hiển thị trong danh sách với tên mới:
[![list profile after rename](/img/migrated/Screenshot-2024-10-01-164932-fe6bd0c4.png)](/img/migrated/Screenshot-2024-10-01-164932-fe6bd0c4.png)
