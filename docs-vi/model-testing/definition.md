# Định nghĩa

**Kiểm thử mô hình** cho phép bạn đánh giá cách mô hình đã chọn phản hồi với các đầu vào cụ thể (văn bản, hình ảnh hoặc cả hai). Bạn có thể xác minh độ chính xác, giọng điệu và mức độ liên quan của mô hình trước khi triển khai lên môi trường production hoặc tích hợp vào ứng dụng.
