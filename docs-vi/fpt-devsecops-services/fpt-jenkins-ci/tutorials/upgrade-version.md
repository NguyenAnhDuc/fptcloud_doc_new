# Nâng cấp phiên bản cho cơ sở dữ liệu

FPT Cloud hỗ trợ người dùng upgrade phiên bản Jenkins lên phiên bản mới hơn trong số các phiên bản FPT Cloud support. Hiện tại trên Portal đang support một số phiên bản bao gồm: 2.452,2.453,2.454.
**Bước 1.** Vào menu **Essential Properties** > **Instance Information** [![](/img/migrated/Picture89-ffa69e1a.png)](/img/migrated/Picture89-ffa69e1a.png) **Bước 2.** Thực hiện chọn version muốn upgrade: [![](/img/migrated/Picture90-bebb276a.png)](/img/migrated/Picture90-bebb276a.png) **Bước 3.** Nhấn **update** để thực hiện Upgrade version. [![](/img/migrated/Picture91-cd0fd45c.png)](/img/migrated/Picture91-cd0fd45c.png) **Bước 4.** Kết quả sau khi upgrade version thành công [![](/img/migrated/Picture92-5539c4c1.png)](/img/migrated/Picture92-5539c4c1.png) **Lưu ý:** Hệ thống sẽ mất một khoảng thời gian để upgrade version.
