# Cấu hình log level

Chức năng này cho phép cấu hình log level trong Image Updater controller. Mặc định log level là `debug`.

1. Tại màn hình **Image Updater** > **Log Level**, thực hiện cấu hình log level.

[![](/img/migrated/Picture6-7-4537dcac.png)](/img/migrated/Picture6-7-4537dcac.png)

2. Chọn mode để cập nhật.

[![](/img/migrated/Picture7-7-c04500ca.png)](/img/migrated/Picture7-7-c04500ca.png)
