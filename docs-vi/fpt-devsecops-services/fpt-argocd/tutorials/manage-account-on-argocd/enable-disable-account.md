# Enable/disable tài khoản

Hệ thống cho phép bật hoặc tắt tài khoản đã tạo.

:::note
Hiện tại Portal chỉ hỗ trợ disable tài khoản trên ArgoCD. Chức năng này tương đương với việc xóa tài khoản.
:::
