# Ket Noi Cluster Toi App Catalog Management

Để có thể deploy một ứng dụng qua App Catalogs Management đã được khởi tạo, cần kết nối một cluster K8s nơi mà ứng dụng được deploy. FPT Cloud hỗ trợ người dùng kết nối cluster qua giao diện
  * **Bước 1** : Chuẩn bị một k8s cluster để deploy ứng dụng, yêu cầu sử dụng kubeconfig để thực hiện Connect cluster
  * **Bước 2** : Tại menu **Management Cluster** > Chọn **Connect Cluster**

[![](/img/migrated/Picture9-1-12077ac5.png)](/img/migrated/Picture9-1-12077ac5.png)
  * **Bước 3** : Thực hiện nhập thông tin để Connect Cluster

[![](/img/migrated/Picture10-1-44a26315.png)](/img/migrated/Picture10-1-44a26315.png)
_Lưu ý: Thông tin dùng để connect k8s cluster bắt buộc phải có quyền admin._
  * Bước 4: Nhấn nút **Connnect Cluster** để hoàn thành.
  * Bước 5: Kiểm tra thông tin k8s cluster đã được connect

[![](/img/migrated/Picture11-1-e7ad5362.png)](/img/migrated/Picture11-1-e7ad5362.png)
