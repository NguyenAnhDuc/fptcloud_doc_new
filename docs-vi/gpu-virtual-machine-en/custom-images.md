# Custom Images

Bạn có thể tạo **GPU VM** dựa trên **custom image**, cho phép bạn di chuyển và mở rộng khối lượng công việc mà không cần tốn thời gian tái tạo lại môi trường từ đầu.

[![Alt text](https://fptcloud.com/wp-content/uploads/2025/03/Custom-Images-1.png)](https://fptcloud.com/wp-content/uploads/2025/03/Custom-Images-1.png)

### Tải lên Custom Image
Chỉ hỗ trợ định dạng file **QCOW** để tải lên GPU VM tại đây. Đối với các định dạng file khác, vui lòng **liên hệ với chúng tôi** để được hỗ trợ thêm.

**Bước 1:** Từ menu, chọn **Compute Engine > Custom Image**. Sau đó nhấp **Upload Image**.

**Bước 2:** Nhập thông tin cần thiết và chọn file từ máy của bạn. Nhấp **Upload** để bắt đầu quá trình tải lên.

### Xóa Custom Image
Nếu bạn không còn cần Custom Image, bạn có thể xóa nó bằng cách thực hiện các bước sau:

**Bước 1:** Từ menu, chọn **Custom Image**, sau đó trong menu **Actions** của image bạn muốn xóa, nhấp **Delete**.

**Bước 2:** Một hộp thoại xác nhận sẽ xuất hiện. Để tiến hành xóa, nhấp **Delete Custom Image**.
