# Tạo Subnet

Để tạo GPU Virtual Machine (instance), bạn cần có Subnet đang hoạt động trong VPC. Hệ thống gán Private IP cho máy ảo mới từ Subnet này.

Các bước tạo Subnet mới:

**Bước 1:** Trong menu bên, vào **Network** > **Subnets** và chọn **Create Subnet**.

[![Alt text](/img/migrated/1-057589e4.png)](/img/migrated/1-057589e4.png)

**Bước 2:** Nhập thông tin cần thiết.

  * **Tên:** Nhập tên dễ nhận biết cho Subnet.
  * **Loại:** Hiện hỗ trợ hai loại.
    * **Routed:** Subnet được định tuyến đến internet qua NAT gateway.
    * **Isolated:** Subnet không có định tuyến internet.
  * **Network address (CIDR):** Nhập CIDR hợp lệ.
  * **Gateway IP:** Nhập địa chỉ gateway hợp lệ.
  * **Static IP pool (tùy chọn):** Chỉ định dải IP cần sử dụng. Nếu để trống, hệ thống cấp phát toàn bộ dải CIDR.
  * **Primary DNS:** DNS server cần thiết cho phân giải tên miền.
  * **Secondary DNS (tùy chọn):** DNS server dự phòng trong trường hợp Primary DNS gặp sự cố.
  * **Add tag (tùy chọn):** Chọn tag để gán cho Subnet.

[![Alt text](/img/migrated/2-b8e89ee4.png)](/img/migrated/2-b8e89ee4.png)

**Bước 3:** Nhấp **Create Subnet** để tạo Subnet mới. Hệ thống xử lý yêu cầu và hiển thị kết quả.

Khi thành công, Subnet mới tạo xuất hiện trong bảng Subnet.
