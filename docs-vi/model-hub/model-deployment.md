# Model Deployment

Mục đích: Kích hoạt, bỏ kích hoạt chức năng deployment của model
Bổ sung thêm 1 tab Setting trong chi tiết model, tại đây bạn có thể kích hoạt, bỏ kích hoạt chức năng deployment. Giá trị default khi model vừa được tạo sẽ là Disable [![](/img/migrated/anh-15-cae85c93.png)](/img/migrated/anh-15-cae85c93.png)
