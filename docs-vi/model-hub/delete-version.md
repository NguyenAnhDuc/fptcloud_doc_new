# Delete Version

Xóa phiên bản là xoá thông tin và các tệp của phiên bản trong hệ thống 
Để xóa phiên bản, bạn thao tác như sau: 
Bước 1: Click vào 1 model để xem chi tiết > Chọn tab Version 
Bước 2: Click vào dấu 3 chấm ở cột Action > Chọn Delete [![](/img/migrated/Picture18-162129ae.png)](/img/migrated/Picture18-162129ae.png) Bước 3: Hiển thị hộp thoại xác nhận, để đồng ý xóa, bạn nhập “delete” vào text box. Lúc này button Delete sẽ enable lên cho người dùng xóa. [![](/img/migrated/Picture19-604dc55c.png)](/img/migrated/Picture19-604dc55c.png) Bước 4: Để xóa, click button Confirm, để hủy bỏ thao tác chọn Cancel
