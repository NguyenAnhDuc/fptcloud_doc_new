# Delete File

Mục đích: Xoá các tệp tương ứng với version 
Các bước thực hiện: 

Bước 1: Click vào 1 model để xem chi tiết > Chọn tab File Browser 

Bước 2: Click vào dấu 3 chấm cột Action > Chọn Delete [![](/img/migrated/Picture25-43e776d4.png)](/img/migrated/Picture25-43e776d4.png) Bước 3: Hiển thị hộp thoại xác nhận, để đồng ý xóa, bạn nhập "delete" vào text box. Lúc này button Delete sẽ enable lên cho người dùng xóa. [![](/img/migrated/Picture26-b12a6e4d.png)](/img/migrated/Picture26-b12a6e4d.png) Bước 4: Để xóa, click button Confirm, để hủy bỏ thao tác chọn Cancel
