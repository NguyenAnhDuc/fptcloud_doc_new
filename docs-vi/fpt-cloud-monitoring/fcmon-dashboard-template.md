# 7.2.1. Cài đặt dashboard template

Giúp bạn triển khai nhanh chóng thông qua các mẫu dashboard mà FPT Cloud Monitoring cung cấp, sẵn sàng ngay để sử dụng.
**7.2.1. Cài đặt dashboard template**
**Bước 1** : Đăng nhập vào dashboard (xem mục 7.1. Truy cập vào dashboard)
**Bước 2** : Trên menu trái, click **Apps** > **FMON Template**
[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)
**Bước 3** : Click **Dashboard Template**
**Bước 4** : Chọn một dashboard, click **Integrate**
[![Alt text](/img/migrated/Picture14-2-d9a00088.png)](/img/migrated/Picture14-2-d9a00088.png)
**7.2.2. Gỡ bỏ Dashboard Template**
**Bước 1** : Đăng nhập vào dashboard (xem mục 7.1. Truy cập vào dashboard)
**Bước 2** : Trên menu trái, click **Apps** > **FMON Template**
[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)
**Bước 3** : Click **Dashboard Template**
**Bước 4** : Chọn một dashboard đã cài đặt, click **Remove**
[![Alt text](/img/migrated/Picture15-2-7940295e.png)](/img/migrated/Picture15-2-7940295e.png)
