# Thêm mới một Credential

Với mỗi loại credential, người dùng có thể thực hiện cấu hình như sau:
**Bước 1.** Tại menu FPT Portal > **Jenkins CI** > **Detail Cluster** > **Credential Managements** : [![](/img/migrated/Picture28-f1ca981e.png)](/img/migrated/Picture28-f1ca981e.png)
  * **Tạo mới Credential loại Secret Text**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **Secret Text** : [![](/img/migrated/Picture29-4e919329.png)](/img/migrated/Picture29-4e919329.png) **Bước 3.** Nhập thông tin để tạo credential với kind Secret Text: [![](/img/migrated/Picture30-6ec104f7.png)](/img/migrated/Picture30-6ec104f7.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind Secret Text. Kết quả sau khi tạo credential: [![](/img/migrated/Picture31-15489933.png)](/img/migrated/Picture31-15489933.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture32-690a204c.png)](/img/migrated/Picture32-690a204c.png) [![](/img/migrated/Picture33-c7a729fd.png)](/img/migrated/Picture33-c7a729fd.png)
  * **Tạo credential với loại Username with password**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **Username with password** : [![](/img/migrated/Picture34-0260b9d4.png)](/img/migrated/Picture34-0260b9d4.png) **Bước 3**. Nhập thông tin để tạo credential với kind **Username with password** : [![](/img/migrated/Picture35-52ee1479.png)](/img/migrated/Picture35-52ee1479.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind **Username with password**. Kết quả sau khi tạo credential: [![](/img/migrated/Picture36-ce02107a.png)](/img/migrated/Picture36-ce02107a.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture37-42448ed6.png)](/img/migrated/Picture37-42448ed6.png) [![](/img/migrated/Picture38-e8817bc1.png)](/img/migrated/Picture38-e8817bc1.png)
  * **Tạo credential với loại GitLab API token**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **GitLab API token** : [![](/img/migrated/Picture39-5f3a117b.png)](/img/migrated/Picture39-5f3a117b.png) **Bước 3.** Nhập thông tin để tạo credential với kind **GitLab API token** : [![](/img/migrated/Picture40-bd944784.png)](/img/migrated/Picture40-bd944784.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind **GitLab API token**. Kết quả sau khi tạo credential: [![](/img/migrated/Picture41-232f7619.png)](/img/migrated/Picture41-232f7619.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture42-addc31f0.png)](/img/migrated/Picture42-addc31f0.png) [![](/img/migrated/Picture43-4d3ae4c9.png)](/img/migrated/Picture43-4d3ae4c9.png)
  * **Tạo credential với loại SSH Username with private key**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **SSH Username with private key** : [![](/img/migrated/Picture44-cc87bfd1.png)](/img/migrated/Picture44-cc87bfd1.png) **Bước 3.** Nhập thông tin để tạo credential với kind **SSH Username with private key** : [![](/img/migrated/Picture45-562215f2.png)](/img/migrated/Picture45-562215f2.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind **SSH Username with private key**. Kết quả sau khi tạo credential: [![](/img/migrated/Picture46-aa140b9e.png)](/img/migrated/Picture46-aa140b9e.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture47-03acc4a6.png)](/img/migrated/Picture47-03acc4a6.png) [![](/img/migrated/Picture48-8be15bf6.png)](/img/migrated/Picture48-8be15bf6.png)
