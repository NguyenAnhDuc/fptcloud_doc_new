# Điều kiện yêu cầu

Trước khi truy cập, người dùng cần đảm bảo:
**Đăng nhập với tài khoản SSO (Azure):**
  * Sử dụng PC, laptop có thể truy cập link FPT Cloud Desktop thông qua Internet/ mạng LAN
  * Được cung cấp đầy đủ thông tin truy cập máy ảo bao gồm URL FCD, tài khoản và Authenticator tương ứng

**Đăng nhập với Account/ Password:**
  * Sử dụng PC, laptop có thể truy cập link FPT Cloud Desktop thông qua Internet/ mạng LAN
  * Được cung cấp đầy đủ thông tin truy cập máy ảo bao gồm URL FCD và tài khoản (URL FCD FCI: fci.fcd.fptcloud.com)
