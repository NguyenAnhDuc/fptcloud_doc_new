# FPT ArgoCD

FPT Smart Cloud cung cấp dịch vụ Managed ArgoCD trên FPT Cloud Portal. Dịch vụ cho phép triển khai ứng dụng trên nền tảng Kubernetes bằng Kustomize, manifest, Helm chart và nhiều phương thức khác. Người dùng có thể quản lý ArgoCD instance bao gồm cấu hình, phiên bản, tài khoản thông qua giao diện tích hợp sẵn.
