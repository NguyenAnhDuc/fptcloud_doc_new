# Cau Hinh Build Option Cho Kustomize

Để cấu hình build options tới kustomize build của kustomize mặc định thực hiện cấu hình như sau: 
**Bước 1** : Tại màn hình **Configuration** > chọn **Kustomize** > chọn **Update**
[![](/img/migrated/Picture120-2-2f369b3e.png)](/img/migrated/Picture120-2-2f369b3e.png)
**Bước 2** : Nhập các thông tin build options 
Ví dụ: Nhập build Options “—enable-helm” 
**_Lưu ý: Các options cách nhau bởi dấu cách_**
[![](/img/migrated/Picture121-2-aa0525af.png)](/img/migrated/Picture121-2-aa0525af.png)
**Bước 3** : Click để lưu cấu hình [![](/img/migrated/Picture122-2-06ab4bf2.png)](/img/migrated/Picture122-2-06ab4bf2.png)
