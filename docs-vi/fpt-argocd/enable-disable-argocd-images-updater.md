# Enable/disable ArgoCD Image Updater

Chức năng này cho phép bật hoặc tắt ArgoCD Image Updater qua giao diện Portal. Mặc định dịch vụ được bật.

1. Tại màn hình **Image Updater** > **Service Status**, thực hiện enable/disable dịch vụ.
