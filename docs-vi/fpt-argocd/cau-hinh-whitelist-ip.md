# Cau Hinh Whitelist Ip

FPT Cloud hỗ trợ người dùng cấu hình whitelistIP truy cập vào URL ArgoCD thông qua giao diện Portal như sau:
**Bước 1:** Vào màn hình **Security & Access** > **Whitelist IP** [![](/img/migrated/Picture41-2-66349b65.png)](/img/migrated/Picture41-2-66349b65.png) **Bước 2:** Nhập IP cho phép truy cập vào **Argo CD**
**Lưu ý: Các IP cách nhau bởi dấu “,”** [![](/img/migrated/Picture42-2-a15b14d8.png)](/img/migrated/Picture42-2-a15b14d8.png) **Bước 3:** Kết quả sau khi cấu hình [![](/img/migrated/Picture43-2-ba2e74dc.png)](/img/migrated/Picture43-2-ba2e74dc.png) **Bước 4:** Kiểm tra kết quả sau khi cấu hình
Truy cập ArgoCD thông qua trình duyệt, xuất hiện lỗi 403 [![](/img/migrated/Picture44-2-d712e561.png)](/img/migrated/Picture44-2-d712e561.png)
