# Quản lý admin và anonymous user

FPT Cloud cho phép quản lý tài khoản admin và người dùng ẩn danh với các chức năng sau:
- Enable/disable tài khoản admin
- Cập nhật mật khẩu admin
- Enable/disable truy cập ẩn danh (anonymous user)
