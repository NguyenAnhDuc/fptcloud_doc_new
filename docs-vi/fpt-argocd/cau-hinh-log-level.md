# Cau Hinh Log Level

Chức năng này cho phép người dùng cấu hình logs level trong images updater controller. Mặc định cấu hình log level là debug
**Bước 1:** Tại màn hình **Image Updater** > **Log Level** , thực hiện cấu hình log level: [![](/img/migrated/Picture6-7-4537dcac.png)](/img/migrated/Picture6-7-4537dcac.png) **Bước 2:** Chọn mode để update [![](/img/migrated/Picture7-7-c04500ca.png)](/img/migrated/Picture7-7-c04500ca.png) **Bước 3:** Kết quả sau khi update [![](/img/migrated/Picture8-7-d55ba191.png)](/img/migrated/Picture8-7-d55ba191.png)
