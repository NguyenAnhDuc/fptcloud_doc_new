# Cấu hình channel để thực hiện gửi thông báo

FPT Cloud cho phép thêm mới/ chỉnh sửa thông tin cấu hình các channel như sau:
**_A. Telegram_**
_1. Thêm mới Telegram config_
  * **Bước 1** : Tại màn hình **Notifications** > **Telegram** :

[![](/img/migrated/Picture38-3-ecf99d40.png)](/img/migrated/Picture38-3-ecf99d40.png)
  * **Bước 2** : Thực hiện cấu hình Token để gửi thông báo tới telegram

[![](/img/migrated/Picture39-3-5750e082.png)](/img/migrated/Picture39-3-5750e082.png)
  * **Bước 3** : Kết quả sau khi cấu hình: 

[![](/img/migrated/Picture40-1-d2f2c786.png)](/img/migrated/Picture40-1-d2f2c786.png)
_2. Edit Telegram config_
Ngoài ra, người dùng có thể thực hiện edit thông tin cấu hình telegram như sau:
  * **Bước 1** : Tại màn hình **Notifications** > **Telegram** :

[![](/img/migrated/Picture41-1-7e331436.png)](/img/migrated/Picture41-1-7e331436.png)
  * **Bước 2** : Thực hiện cấu hình lại Token để gửi thông báo tới telegram

[![](/img/migrated/Picture42-1-ffe0b22f.png)](/img/migrated/Picture42-1-ffe0b22f.png)
  * **Bước 3** : Kết quả sau khi cấu hình: 

[![](/img/migrated/Picture43-1-2262a51b.png)](/img/migrated/Picture43-1-2262a51b.png)
_3. Xoá cấu hình telegram_
Để thực hiện xoá cấu hình thực hiện như sau:
  * **Bước 1** : Tại màn hình **Notifications** > **Telegram** :

[![](/img/migrated/Picture44-1-8b6f543d.png)](/img/migrated/Picture44-1-8b6f543d.png)
  * **Bước 2** : Nhập confirm text để thực hiện xoá channel telegram:

[![](/img/migrated/Picture45-1-ced26400.png)](/img/migrated/Picture45-1-ced26400.png)
  * **Bước 3** : Kết quả sau khi xoá channel telegram:

[![](/img/migrated/Picture46-1-1d1b4ec2.png)](/img/migrated/Picture46-1-1d1b4ec2.png)
**_B. Slack_**
_1. Thêm mới cấu hình Slack_
  * **Bước 1** : Tại màn hình **Notifications** > **Slack** :

[![](/img/migrated/Picture47-1-76675787.png)](/img/migrated/Picture47-1-76675787.png)
  * **Bước 2** : Thực hiện cấu hình Slack token để gửi thông báo tới Slack

[![](/img/migrated/Picture48-1-8c4f6f64.png)](/img/migrated/Picture48-1-8c4f6f64.png)
  * **Bước 3** : Kết quả sau khi cấu hình thành công

[![](/img/migrated/Picture49-1-be406f9e.png)](/img/migrated/Picture49-1-be406f9e.png)
_2. Edit Slack config_
  * **Bước 1** : Tại màn hình **Notifications** > **Slack** :

[![](/img/migrated/Picture50-1-2a168aed.png)](/img/migrated/Picture50-1-2a168aed.png)
  * **Bước 2** : Thực hiện cập nhật cấu hình Slack token để gửi thông báo tới Slack

[![](/img/migrated/Picture51-1-4dba0f1b.png)](/img/migrated/Picture51-1-4dba0f1b.png)
  * **Bước 3** : Kết quả sau khi cấu hình thành công 

[![](/img/migrated/Picture52-1-50e0f2cb.png)](/img/migrated/Picture52-1-50e0f2cb.png)
_3. Xoá Slack config_
  * **Bước 1** : Tại màn hình **Notifications** > **Slack** :

[![](/img/migrated/Picture53-1-f87f0214.png)](/img/migrated/Picture53-1-f87f0214.png)
  * **Bước 2** : Nhập confirm text để thực hiện xoá

[![](/img/migrated/Picture54-1-fbdcbe30.png)](/img/migrated/Picture54-1-fbdcbe30.png)
  * **Bước 3** : Kết quả sau khi cấu hình thành công

[![](/img/migrated/Picture55-1-88d2c494.png)](/img/migrated/Picture55-1-88d2c494.png)
**_D. Email_**
_1. Thêm mới cấu hình Email_
  * **Bước 1** : Tại màn hình **Notifications** > **Email** :

[![](/img/migrated/Picture56-1-e553eb8b.png)](/img/migrated/Picture56-1-e553eb8b.png)
  * **Bước 2** : Thực hiện cấu hình Email config để gửi thông báo tới Email

[![](/img/migrated/Picture57-1-07622439.png)](/img/migrated/Picture57-1-07622439.png)
  * **Bước 3** : Kết quả sau khi cấu hình thành công

[![](/img/migrated/Picture58-1-9deec720.png)](/img/migrated/Picture58-1-9deec720.png)
_2. Edit Email config_
  * **Bước 1** : Tại màn hình **Notifications** > **Email** :

[![](/img/migrated/Picture59-1-a7e74dfc.png)](/img/migrated/Picture59-1-a7e74dfc.png)
  * **Bước 2** : Thực hiện cập nhật cấu hình Email. Người dùng có thể thay đổi một số thông tin sau như **SMTP Host/smtp port/username/password/ from user**

[![](/img/migrated/Picture60-1-77cdc196.png)](/img/migrated/Picture60-1-77cdc196.png)
  * **Bước 3** : Kết quả sau khi cấu hình thành công

[![](/img/migrated/Picture61-1-63551cc8.png)](/img/migrated/Picture61-1-63551cc8.png)
_3. Delete Email config_
  * **Bước 1** : Tại màn hình **Notifications** > **Email** :

[![](/img/migrated/Picture62-1-9dce244f.png)](/img/migrated/Picture62-1-9dce244f.png)
  * **Bước 2** : Nhập confirm text để thực hiện xoá

[![](/img/migrated/Picture63-1-35366acf.png)](/img/migrated/Picture63-1-35366acf.png)
  * **Bước 3** : Kết quả sau khi cấu hình thành công

[![](/img/migrated/Picture64-1-d6b61738.png)](/img/migrated/Picture64-1-d6b61738.png)
Hướng dẫn chi tiết tích hợp với applications tham khảo tại mục **Demo > Hướng dẫn cấu hình notifications để nhận thông báo khi ứng dụng thay đổi**
