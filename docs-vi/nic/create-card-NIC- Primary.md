# Khởi tạo card NIC Primary

Trong quá trình khởi tạo Instance, hệ thống tự động tạo card NIC Primary dựa trên thông tin bạn cung cấp.

[![Màn hình khởi tạo Instance](/img/migrated/image-1691381396202-7c332ae8.png)](/img/migrated/image-1691381396202-7c332ae8.png)

1. Trong menu, chọn **Compute** rồi chọn tab **Instance Management**.

[![Danh sách Instance](/img/migrated/image-1691381410150-1b799854.png)](/img/migrated/image-1691381410150-1b799854.png)

2. Chọn Instance cần xem, sau đó chọn tab **Network Interface**.

   Tại đây bạn có thể quản lý thông tin các card NIC. Nếu chỉ có một card NIC, mặc định đó là card Primary. Đối với card Primary, hệ thống chỉ cho phép thực hiện thao tác Disable.

[![Quản lý Network Interface](/img/migrated/image-1691381424127-1da54a77.png)](/img/migrated/image-1691381424127-1da54a77.png)

[![Chi tiết card NIC Primary](/img/migrated/image-1691381438228-5cb76cd6.png)](/img/migrated/image-1691381438228-5cb76cd6.png)
