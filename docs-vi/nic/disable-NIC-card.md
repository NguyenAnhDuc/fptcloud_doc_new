# Disable card NIC

1. Chọn **Disable Card** trên card NIC cần tắt.

[![Nút Disable Card](/img/migrated/image-1691381843537-90d61d72.png)](/img/migrated/image-1691381843537-90d61d72.png)

2. Xác nhận thao tác Disable Card.

[![Xác nhận Disable Card](/img/migrated/image-1691381854527-265d8dd4.png)](/img/migrated/image-1691381854527-265d8dd4.png)

3. Chờ hệ thống xử lý.

[![Hệ thống đang xử lý](/img/migrated/image-1691381866783-e7158e52.png)](/img/migrated/image-1691381866783-e7158e52.png)

4. Card chuyển sang trạng thái **Inactive** và Instance không thể kết nối mạng qua card này.

[![Card NIC ở trạng thái Inactive](/img/migrated/image-1691381878032-5f233510.png)](/img/migrated/image-1691381878032-5f233510.png)
