# Thay đổi cấu hình Cluster Autoscale

**Bước 1:** Ở menu chọn **Kubernetes** để mở trang **Kubernetes Management**. Chọn cluster muốn thay đổi cấu hình Cluster Autoscale.

[![](/img/migrated/34-1-51e37dcc.png)](/img/migrated/34-1-51e37dcc.png)

**Bước 2:** Chọn **Configure Worker Groups**.

[![](/img/migrated/35-1-fdd253ce.png)](/img/migrated/35-1-fdd253ce.png)

**Bước 3:** Điều chỉnh số lượng worker theo nhu cầu.

[![](/img/migrated/36-1-2bd8d261.png)](/img/migrated/36-1-2bd8d261.png)

**Bước 4:** Kiểm tra lại thông tin và chọn **Save**.

[![](/img/migrated/37-1-60705efe.png)](/img/migrated/37-1-60705efe.png)
