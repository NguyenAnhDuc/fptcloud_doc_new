# Thay đổi mật khẩu

Sau khi IAM user đăng ký và đăng nhập thành công vào FPT Portal, nên đổi mật khẩu để đảm bảo bảo mật tài khoản.

1. Nhấn vào vùng hiển thị tên tài khoản ở góc trên bên phải FPT Portal, sau đó chọn **Change Password**.

[![](/img/migrated/Userguide-FPT-IAM-2022-27-1024x285-88368ee2.png)](/img/migrated/Userguide-FPT-IAM-2022-27-1024x285-88368ee2.png)

2. Trên hộp thoại **Change Password**, nhập các thông tin theo yêu cầu:
   - **Current**: Nhập mật khẩu hiện tại.
   - **New**: Nhập mật khẩu mới.
   - **Confirm**: Xác nhận mật khẩu mới.

[![](/img/migrated/Userguide-FPT-IAM-2022-28-c40f7554.png)](/img/migrated/Userguide-FPT-IAM-2022-28-c40f7554.png)

3. Chọn **Change Password**. Hệ thống kiểm tra, xử lý và thông báo kết quả.

[![](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)
