# Gioi Thieu Chung

Tính năng **IP Access Control** là tính năng bảo mật của FPT Cloud Monitoring, cho phép người dùng thiết lập và quản lý danh sách các địa chỉ IP bên ngoài hệ thống FPT Cloud được phép truy cập vào từng hệ thống IP nội bộ FPT cloud monitoring của mình. 
Tính năng này chủ động chặn các truy cập từ bên ngoài FPT Cloud không nằm trong danh sách cho phép, nhằm hỗ trợ người dùng nâng cao bảo mật truy cập, phòng chống rủi ro truy cập trái phép và đáp ứng các tiêu chuẩn bảo mật.
