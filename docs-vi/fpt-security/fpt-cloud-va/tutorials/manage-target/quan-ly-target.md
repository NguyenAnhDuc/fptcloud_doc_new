# Quản lý Target

**Target** là đối tượng được FPT VA rà quét lỗ hổng bảo mật, do người dùng tạo và cài đặt trên **FPT Portal**. Target có thể là một website, một địa chỉ IP public hoặc một dải IP mà người dùng đang quản lý.
