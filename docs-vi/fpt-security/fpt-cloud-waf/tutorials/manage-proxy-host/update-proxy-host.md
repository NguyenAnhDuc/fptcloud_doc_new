# Cập nhật thông tin proxy host

1. Trong **WAF Dashboard**, chọn **Hosts** > **Proxy**.

   [![Danh sách proxy host](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. Tại proxy host cần cập nhật, chọn icon **Edit**.

   [![Chọn Edit proxy host](/img/migrated/Userguide-FPT-WAF-2022-19-1024x578-234f8c82.png)](/img/migrated/Userguide-FPT-WAF-2022-19-1024x578-234f8c82.png)

3. Nhập các thông tin cần cập nhật.

   [![Chỉnh sửa thông tin proxy host](/img/migrated/Userguide-FPT-WAF-2022-20-1024x578-e34b9e78.png)](/img/migrated/Userguide-FPT-WAF-2022-20-1024x578-e34b9e78.png)

4. Chọn **Update Proxy** để lưu và quay về trang **Proxy Management**.

   [![Lưu thông tin proxy host](/img/migrated/Userguide-FPT-WAF-2022-21-1024x578-9594ad2a.png)](/img/migrated/Userguide-FPT-WAF-2022-21-1024x578-9594ad2a.png)
