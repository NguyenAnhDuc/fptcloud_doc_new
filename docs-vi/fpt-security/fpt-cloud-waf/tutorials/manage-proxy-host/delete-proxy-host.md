# Xóa proxy host

:::danger
Xóa proxy host sẽ khiến máy chủ web của bạn không còn được WAF bảo vệ. Hãy cẩn thận khi thực hiện thao tác này.
:::

1. Trong **WAF Dashboard**, chọn **Hosts** > **Proxy**.

   [![Danh sách proxy host](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. Tại proxy host cần xóa, chọn icon **Delete**.

   [![Chọn Delete proxy host](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)

3. Trong hộp thoại xác nhận, chọn **Delete** để hoàn tất.

   [![Xác nhận xóa proxy host](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)
