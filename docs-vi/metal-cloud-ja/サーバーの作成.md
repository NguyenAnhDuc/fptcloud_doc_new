# Tạo server

Bạn có thể triển khai Bare Metal Server qua FPT Customer Portal. FPT xử lý việc cấp phát, cài đặt OS và cấu hình dựa trên các thiết lập bạn chọn. Sau khi triển khai hoàn tất, bạn có thể truy cập server qua SSH bằng SSH key hoặc mật khẩu.

1. Đăng nhập vào tài khoản FPT Cloud và chọn tenant, region, VPC (nếu có nhiều).
2. Vào AI Infrastructure và Metal Cloud trong sidebar, nhấn Create server.
3. Chọn flavor (khả dụng tùy theo vùng).
4. Nhập số lượng server muốn triển khai cùng lúc (số tối đa phụ thuộc vào quota).
5. Nhập tên server.
6. Chọn hệ điều hành. Nếu OS cần thiết không có, bạn có thể dùng custom image thay thế.
7. Chọn subnet khả dụng. Bạn không thể triển khai server mà không có subnet. Hãy tạo subnet trước khi tạo và triển khai server.
8. Chọn gắn floating IP (tùy chọn).
9. Chọn phương thức xác thực: SSH key hoặc console password. a. Chọn SSH key có sẵn hoặc thêm key mới (xem hướng dẫn tạo và thêm SSH key tại đây).
10. Cấu hình RAID (tùy chọn).
11. Chọn hoặc tạo user data script (tùy chọn). User data script được tự động thực thi khi khởi động server lần đầu thông qua quá trình cloud-init.
12. Nhấn Create server.
