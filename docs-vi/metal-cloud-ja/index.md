# Metal Cloud

Metal Cloud là dịch vụ cloud cho phép bạn thuê máy chủ vật lý tích hợp GPU từ FPT. Máy chủ này dành riêng cho bạn và không chia sẻ với tenant khác. Bạn có thể truy cập server với toàn quyền kiểm soát phần cứng mà không cần lớp ảo hóa hay trừu tượng hóa, giúp tối đa hiệu năng cho các workload tính toán chuyên sâu.
