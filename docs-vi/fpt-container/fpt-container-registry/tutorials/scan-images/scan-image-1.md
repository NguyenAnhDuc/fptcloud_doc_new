# Thực hiện scan image

Quét lỗ hổng bảo mật giúp xác định các CVE (Common Vulnerabilities and Exposures) trong image trước khi triển khai lên môi trường production.

1. Trên menu **FPT Portal**, chọn **Container Registry** > **Repository** > **Detail Repository**.

   [![Chi tiết repository để scan](/img/migrated/Picture90-097f05eb.png)](/img/migrated/Picture90-097f05eb.png)

2. Chọn artifact muốn quét để hiển thị nút **SCAN**.

   [![Chọn artifact để hiển thị nút SCAN](/img/migrated/Picture91-3e24672e.png)](/img/migrated/Picture91-3e24672e.png)

3. Chọn **SCAN** để bắt đầu quét image.

   [![Nút SCAN](/img/migrated/Picture92-c39690de.png)](/img/migrated/Picture92-c39690de.png)

   [![Trạng thái đang scan](/img/migrated/Picture93-55409ce3.png)](/img/migrated/Picture93-55409ce3.png)

4. Kiểm tra kết quả sau khi quét hoàn thành.

   [![Kết quả scan image](/img/migrated/Picture94-fd252260.png)](/img/migrated/Picture94-fd252260.png)
