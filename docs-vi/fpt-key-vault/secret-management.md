# Secret管理

SecretはSecret Engine内にKey/Valueペアとして機密情報を保存します。各Secretはバージョン管理をサポートしており、変更履歴を追跡できます。このセクションでSecretの作成、表示、更新、削除を行います。

- [Secret EngineにSecretを作成する](./create-secret.md)
- [Secret情報を表示・更新する](./view-update-secret.md)
- [Secretを削除する](./delete-secret.md)
