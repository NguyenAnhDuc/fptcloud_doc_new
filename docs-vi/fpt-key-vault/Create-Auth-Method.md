# Create Auth Method

**Step 1:** Select the Key Vault to create **Auth Method** on the Key Vault Management screen:
[![](/img/migrated/acc1-815ef22e.png)](/img/migrated/acc1-815ef22e.png)
**Step 2** : In the **Access** tab, click **Create**.
[![](/img/migrated/acc2-92ea8af3.png)](/img/migrated/acc2-92ea8af3.png)
**Step 3** : In the **Create New Auth Method** popup, enter the necessary information
[![](/img/migrated/acc3-81005be5.png)](/img/migrated/acc3-81005be5.png)
  * **Mount Point** : Partition to store authentication method secret information
  * **Type** : Select App Role

**Step 4** : Click **Create** to proceed. The system will check the resource requirement, send a confirmation message and proceed with the process of creating a new **Auth Method** based on the information entered.
