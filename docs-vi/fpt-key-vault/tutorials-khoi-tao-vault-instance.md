# Khởi tạo Vault Instance

Vault Instance là đơn vị cơ bản của FPT Key Vault, nơi lưu trữ toàn bộ Secret Engine, Secret và cấu hình bảo mật của bạn. Khởi tạo Vault Instance trước khi thực hiện bất kỳ thao tác quản lý Secret nào.

1. Trên menu, chọn **Security** > **Key Vault** > **Create**.

   [![Màn hình danh sách Key Vault với nút Create](/img/migrated/Screenshot-2024-08-20-103813-1-bebc2790.png)](/img/migrated/Screenshot-2024-08-20-103813-1-bebc2790.png)

2. Trên màn hình **Create Key Vault**, nhập các thông tin cần thiết.

   [![Form nhập thông tin khởi tạo Key Vault](/img/migrated/Userguide-FPT-KeyVault-2022-2-125824c1.png)](/img/migrated/Userguide-FPT-KeyVault-2022-2-125824c1.png)

3. Chọn **Create** để hoàn tất. Hệ thống kiểm tra tài nguyên, gửi thông báo xác nhận và tiến hành tạo **Vault Instance** mới dựa trên thông tin đã nhập.

   [![Danh sách Key Vault sau khi khởi tạo thành công](/img/migrated/Userguide-FPT-KeyVault-2022-3-1024x290-02d8d8d5.png)](/img/migrated/Userguide-FPT-KeyVault-2022-3-1024x290-02d8d8d5.png)
