# Model Testing – Interactive Sessions

## Quick Start

Hướng dẫn nhanh để bắt đầu sử dụng **Model Testing – Interactive Sessions**:

1. [Đăng ký tài khoản](./sign-up-for-an-account.md) tại <https://ai.fptcloud.com/> hoặc <https://ai.fptcloud.jp/>.
2. Làm theo [Step By Step](./step-by-step.md) để tạo và quản lý session.
3. Sử dụng [Playground](./playground.md) để interact và evaluate model.
4. Xem [Completion History](./completion-history.md) để theo dõi các completion đã thực hiện.
