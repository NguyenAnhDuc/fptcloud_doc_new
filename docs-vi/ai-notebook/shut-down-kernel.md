# Tắt kernel

1. Trong Jupyter, chọn **Running Kernels/Notebooks** từ menu.
2. Chọn kernel cần tắt.
3. Nhấn **Shut Down** để tắt kernel.

:::tip
Tắt kernel khi không sử dụng giúp tiết kiệm tài nguyên và tránh phát sinh chi phí không mong muốn.
:::
