# Hướng dẫn nhanh

1. Truy cập [https://ai.fptcloud.com/](https://ai.fptcloud.com/) và đăng ký bằng **FPT ID**.
2. Trong menu bên trái, chọn **AI Notebook**.
3. Nhấn **Create Notebook** để tạo notebook mới.
4. Chọn cấu hình GPU và nhấn **Launch** để khởi chạy.
