# Câu hỏi thường gặp

**Q: Làm thế nào để tiết kiệm tài nguyên?**

Tắt kernel khi không sử dụng. AI Notebooks tính phí dựa trên thời gian sử dụng GPU.

**Q: Tôi có thể sử dụng GPU nào?**

FPT AI Factory hỗ trợ nhiều loại GPU khác nhau. Kiểm tra cấu hình khi tạo notebook.
