# Analytics

:::note
**Đối tượng sử dụng:** API Provider.
:::

Analytics giúp bạn theo dõi tình trạng sử dụng API — bao gồm thống kê số lượng gọi API theo consumer và theo từng API.

## Thống kê API

Vào **menu Analytics** để xem thống kê về tình trạng sử dụng API. Phần analytics hiển thị 2 biểu đồ:

- Biểu đồ thống kê số lượng call API theo từng consumer.
- Biểu đồ thống kê số lượng call API theo từng API.

## Logging

Vào mục Logging để xem thông tin về việc gọi API theo từng consumer, bao gồm: API được call từ địa chỉ IP nào, được gọi qua đâu, và các thông tin như thời gian, phương thức, status, base path.

**Bước 1:** Ở menu **Application** > chọn **Analytics** > chọn **Logging**.

**Bước 2:** Chọn consumer cần xem log. Hệ thống hiển thị log bao gồm các thông tin về việc consumer đã gọi API.
