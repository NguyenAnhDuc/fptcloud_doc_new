# Clone image

Clone Image cho phép bạn nhân bản các image có sẵn cùng toàn bộ thiết bị lưu trữ — không cần thiết lập lại môi trường từ đầu.

:::warning
Đảm bảo đã hoàn tất toàn bộ cài đặt môi trường và tắt máy trước khi clone.
:::

## Clone image

1. Chọn image và nhấn **Clone**.
2. Nhập số lượng máy muốn tạo, nhấn **Clone**.
