# Xem chi tiết lịch snapshot ổ đĩa

1. Tại tab **Storage snapshot schedule**, nhấn chọn tên lịch.

   [![Chọn lịch](/img/migrated/image-1766045049451-2b6c1676.png)](/img/migrated/image-1766045049451-2b6c1676.png)

2. Xem thông tin: Name, description, start/end date, frequency, danh sách ổ đĩa.

   [![Chi tiết lịch](/img/migrated/image-1766045074035-865020b6.png)](/img/migrated/image-1766045074035-865020b6.png)
