# Gắn máy ảo vào Instance Schedule

## Gắn máy ảo

1. Tại **Instance Schedule Management**, chọn **Action** → **Apply instances**.

   [![Menu Action](/img/migrated/image-1695609073911-03ac0ac2.png)](/img/migrated/image-1695609073911-03ac0ac2.png)

2. Chọn máy ảo cần gắn, nhấn **Apply instances**.

   [![Chọn máy ảo](/img/migrated/image-1695609084366-c567a73d.png)](/img/migrated/image-1695609084366-c567a73d.png)
