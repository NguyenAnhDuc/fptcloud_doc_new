# Xóa Instance Schedule

:::danger
Xóa Schedule không thể khôi phục. Không thể xóa nếu lịch đang chạy dở.
:::

## Xóa Schedule

1. Tại **Instance Schedule Management**, chọn **Action** → **Delete**.

   [![Menu Action Delete](/img/migrated/image-1695609049584-8754accf.png)](/img/migrated/image-1695609049584-8754accf.png)

2. Nhập **delete**, nhấn **Delete schedule**.

   [![Xác nhận xóa Schedule](/img/migrated/image-1695609061103-74aa2b28.png)](/img/migrated/image-1695609061103-74aa2b28.png)

Máy ảo đang gắn sẽ được giải phóng và giữ nguyên trạng thái.
