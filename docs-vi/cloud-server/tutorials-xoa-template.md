# Xóa template

Khi không còn cần template, bạn có thể xóa để giải phóng tài nguyên.

:::danger
Template đã xóa không thể khôi phục.
:::

## Xóa template

1. Tại trang **Custom Images**, chọn **Actions** → **Delete custom image** trên template cần xóa.

   [![Menu Actions với tùy chọn Delete custom image](/img/migrated/image-1719539882859-4f7b095b.png)](/img/migrated/image-1719539882859-4f7b095b.png)

2. Xác nhận xóa trong hộp thoại.
