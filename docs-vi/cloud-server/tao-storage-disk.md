# Tao Storage Disk

Để tạo mới **Storage Disk** người thao tác như sau:
**Bước 1** : Ở menu chọn **Compute Engine** > **Storage Disk** , chọn **Create Storage**.
[![file](/img/migrated/image-1712215492491-8d7d7c0c.png)](/img/migrated/image-1712215492491-8d7d7c0c.png)
**Bước 2** : Nhập các thông tin theo yêu cầu của hệ thống:
  * **Name** : Tên Storage
  * **Storage Policy** : Loại Storage
  * **Size** : Kích thước Storage
  * **Applied instance (optional)** : Máy ảo gắn vào storage (có thể gắn sau)

[![file](/img/migrated/image-1712215502018-5f358995.png)](/img/migrated/image-1712215502018-5f358995.png)
**Bước 3** : Chọn **Create storage disk**. Hệ thống sẽ tiến hành khởi tạo và thông báo kết quả.
Nếu thành công, **Storage Disk** mới sẽ được hiển thị ở trang **Storage Disk Management**.
[![file](/img/migrated/image-1712215510866-1cd01f25.png)](/img/migrated/image-1712215510866-1cd01f25.png)
