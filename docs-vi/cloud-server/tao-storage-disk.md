# Tạo Storage Disk

Tạo Storage Disk mới giúp bạn mở rộng dung lượng lưu trữ cho máy ảo — phù hợp khi cần thêm không gian cho dữ liệu hoặc ứng dụng.

## Tạo Storage Disk

1. Chọn **Compute Engine** → **Storage Disk**, nhấn **Create Storage**.

   [![Màn hình Storage Disk với nút Create Storage](/img/migrated/image-1712215492491-8d7d7c0c.png)](/img/migrated/image-1712215492491-8d7d7c0c.png)

2. Nhập thông tin:
   - **Name**: Tên Storage.
   - **Storage Policy**: Loại Storage.
   - **Size**: Kích thước Storage.
   - **Applied instance** *(tùy chọn)*: Máy ảo gắn vào (có thể gắn sau).

   [![Hộp thoại Create Storage Disk](/img/migrated/image-1712215502018-5f358995.png)](/img/migrated/image-1712215502018-5f358995.png)

3. Nhấn **Create storage disk**.

Storage Disk mới hiển thị tại trang **Storage Disk Management**.

[![Storage Disk sau khi tạo](/img/migrated/image-1712215510866-1cd01f25.png)](/img/migrated/image-1712215510866-1cd01f25.png)
