# Gắn máy ảo vào Instance Group

Thêm các máy ảo vào Instance Group để áp dụng chính sách đặt vị trí Affinity hoặc Anti-Affinity mà bạn đã chọn.

:::note
Một máy ảo chỉ có thể thuộc về một Instance Group tại một thời điểm.
:::
