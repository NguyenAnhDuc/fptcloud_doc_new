# Filter Security Group

Tính năng Filter giúp bạn nhanh chóng tìm kiếm và thu hẹp danh sách Security Group theo nhiều tiêu chí: tên, tag, IP, trạng thái, loại firewall, ngày tạo.

## Các tiêu chí filter

| Field | Ý nghĩa |
|-------|---------|
| **Name** | Nhập tên hoặc một phần tên. Ví dụ: nhập `sec` để tìm "security-group-9609". |
| **Tag** | Lọc theo tag đã gắn. |
| **IP** | Lọc theo địa chỉ IP. |
| **Status** | Lọc theo trạng thái. |
| **Type** | Security Group hoặc Distributed Firewall. |
| **Created at** | Lọc theo ngày tạo. |
