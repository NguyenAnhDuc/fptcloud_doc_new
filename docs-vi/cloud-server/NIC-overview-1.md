# Quản lý NIC (Network Interface Card)

Card NIC (Network Interface Card) là thành phần quan trọng kết nối máy ảo với tài nguyên mạng trên Cloud — cung cấp khả năng truyền dữ liệu hiệu suất cao và đáng tin cậy.

:::warning
Đây là tính năng nâng cao, dành cho người có kiến thức về quản lý hệ thống mạng. Cần kiểm tra thử nghiệm sau khi cấu hình để đảm bảo card hoạt động đúng.
:::
