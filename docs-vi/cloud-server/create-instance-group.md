# Tạo Instance Group mới

Tạo một Instance Group mới để kiểm soát việc đặt vị trí các máy ảo bằng chính sách Affinity hoặc Anti-Affinity.

:::tip
Trước khi tạo Instance Group, hãy xem lại [chính sách Affinity và Anti-Affinity](./overview-affinity.md) để chọn chiến lược đặt vị trí phù hợp với khối lượng công việc của bạn.
:::

- [Gắn máy ảo vào Instance Group](./add-instance-instance-group.md)
