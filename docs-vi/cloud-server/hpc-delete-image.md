# Xóa image

Khi không còn nhu cầu sử dụng, bạn có thể xóa image khỏi danh sách máy trên HPC Portal.

:::danger
Xóa image là thao tác không thể hoàn tác. Đảm bảo đã sao lưu dữ liệu quan trọng trước khi xóa.
:::

## Xóa image

1. Chọn image muốn xóa, nhấn **Action** → **Remove**.
2. Tại hộp thoại xác nhận, nhấn **Remove** để hoàn tất.
