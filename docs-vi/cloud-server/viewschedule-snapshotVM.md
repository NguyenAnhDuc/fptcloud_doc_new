# Xem chi tiết lịch snapshot máy ảo

1. Tại tab **Instance snapshot schedule**, nhấn chọn tên lịch.

   [![Chọn lịch](/img/migrated/image-1766044475260-90c9037f.png)](/img/migrated/image-1766044475260-90c9037f.png)

2. Xem thông tin: Name, description, start/end date, frequency, danh sách máy ảo.

   [![Chi tiết lịch](/img/migrated/image-1766044510198-7f67e63b.png)](/img/migrated/image-1766044510198-7f67e63b.png)
