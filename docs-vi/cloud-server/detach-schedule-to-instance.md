# Gỡ máy ảo khỏi Instance Schedule

Sau khi gỡ, máy ảo giữ nguyên trạng thái hiện tại cho đến khi bạn thao tác thủ công.

## Gỡ máy ảo

1. Tại **Instance Schedule Management**, chọn **Action** → **Apply instances**.
2. Nhấn dấu **x** bên cạnh tên máy ảo để gỡ (hoặc **x** ở cuối listbox để gỡ toàn bộ).
3. Nhấn **Apply instances**.
