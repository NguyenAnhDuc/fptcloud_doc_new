# Question 2

Khách hàng có thể tùy chỉnh toàn bộ các rule trong Security group, không hạn chế. Ngoài các Type được FCI hỗ trợ sẵn, tính năng cho phép khách hàng tạo các custom rule phù hợp nhu cầu riêng biệt.
