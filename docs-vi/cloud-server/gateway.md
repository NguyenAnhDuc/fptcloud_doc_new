# Gateway

Gateway quản lý kết nối mạng giữa VPC và internet hoặc các mạng bên ngoài.

:::note
Tính năng này chỉ dành cho một số Tenant có cấu hình đặc biệt. Liên hệ đội ngũ hỗ trợ để được kích hoạt.
:::

## Truy cập Gateway

1. Đăng nhập [console.fptcloud.com](https://console.fptcloud.com).
2. Chọn VPC cần xem dữ liệu Gateway.
3. Tại sidebar, mở rộng menu **Network** → chọn **Gateway**.
