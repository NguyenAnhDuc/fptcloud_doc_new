# Bật Network Interface Card

Bạn có thể bật lại card NIC đã bị tắt trước đó để khôi phục kết nối mạng qua giao diện đó.

## Bật card NIC

1. Truy cập trang **Quản lý Instance** và chọn Instance của bạn.
2. Chuyển sang tab **NIC**.
3. Tìm card NIC đang bị tắt, sau đó chọn **Hành động** → **Bật**.
4. Xác nhận hành động trong hộp thoại hiện ra.

Trạng thái card NIC sẽ chuyển thành **Đã bật** và kết nối mạng được khôi phục.
