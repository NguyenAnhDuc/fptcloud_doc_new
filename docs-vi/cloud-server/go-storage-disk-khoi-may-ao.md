# Gỡ Storage Disk khỏi máy ảo

Khi không còn cần sử dụng, bạn có thể gỡ Storage Disk khỏi máy ảo. Dữ liệu trong Storage Disk vẫn được giữ nguyên và có thể gắn vào máy ảo khác.

## Gỡ Storage Disk

1. Tại danh sách Storage Disk, chọn **Actions** → **Detach**.

   [![Menu Actions với tùy chọn Detach](/img/migrated/image-1712215548327-dc507d37.png)](/img/migrated/image-1712215548327-dc507d37.png)

2. Xác nhận thông tin trong popup, nhấn **Detach**.

   [![Hộp thoại xác nhận Detach](/img/migrated/image-1712215553461-87cc9cfe.png)](/img/migrated/image-1712215553461-87cc9cfe.png)
