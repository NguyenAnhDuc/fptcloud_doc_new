# Thu hồi token

Khi Token không còn cần sử dụng hoặc nghi ngờ bị lộ, bạn nên revoke ngay để đảm bảo bảo mật.

## Revoke Token

1. Tại trang **Token**, nhấn **Actions** → **Revoke token**.

   [![Menu Actions với tùy chọn Revoke token](/img/migrated/image-1731025026905-158de23e.png)](/img/migrated/image-1731025026905-158de23e.png)

2. Xác nhận revoke trong hộp thoại.

   [![Hộp thoại xác nhận Revoke token](/img/migrated/image-1731025048482-957c5ad2.png)](/img/migrated/image-1731025048482-957c5ad2.png)

3. Hệ thống tự động revoke Token.

   [![Token đã được revoke](/img/migrated/image-1731025071939-e64b80bc.png)](/img/migrated/image-1731025071939-e64b80bc.png)
