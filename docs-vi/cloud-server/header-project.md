# Thay đổi giao diện header

Tùy chỉnh header của FPT Cloud Portal để hiển thị Project đang hoạt động — giúp bạn dễ dàng xác định mình đang làm việc trong ngữ cảnh Project nào.

:::tip
Hiển thị tên Project trên header giúp tránh việc vô tình tạo tài nguyên trong sai Project.
:::
