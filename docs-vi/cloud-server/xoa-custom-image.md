# Xóa Custom Image

Khi không còn nhu cầu sử dụng, bạn có thể xóa Custom Image khỏi hệ thống.

:::danger
Custom Image đã xóa không thể khôi phục. Đảm bảo không có máy ảo nào đang sử dụng image này trước khi xóa.
:::

## Xóa Custom Image

1. Tại **Custom Image**, chọn **Actions** → **Delete**.

   [![Menu Actions với tùy chọn Delete](/img/migrated/image-1734596886940-61bbf341.png)](/img/migrated/image-1734596886940-61bbf341.png)

2. Xác nhận trong popup, nhấn **Delete custom image**.

   [![Hộp thoại xác nhận xóa Custom Image](/img/migrated/image-1734596914709-3da0ab77.png)](/img/migrated/image-1734596914709-3da0ab77.png)
