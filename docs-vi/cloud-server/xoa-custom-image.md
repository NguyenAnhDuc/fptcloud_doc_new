# Xoa Custom Image

Nếu không còn nhu cầu sử dụng, người dùng có thể xóa **Custom Image** theo hướng dẫn sau: 
**Bước 1:** Ở menu chọn **Custom Image** , trong phần **Actions** của Image cần xóa, chọn **Delete**. [![file](/img/migrated/image-1734596886940-61bbf341.png)](/img/migrated/image-1734596886940-61bbf341.png) **Bước 2:** Hệ thống sẽ hiển thị popup xác nhận thông tin xóa, để đồng ý xóa chọn **Delete custom image**. [![file](/img/migrated/image-1734596914709-3da0ab77.png)](/img/migrated/image-1734596914709-3da0ab77.png)
