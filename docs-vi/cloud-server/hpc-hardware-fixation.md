# Cố định máy vật lý

Mặc định, khi bật máy hệ thống phân phối ngẫu nhiên một máy chủ vật lý cho mỗi phiên. Tính năng Hardware Fixation ghi nhớ địa chỉ MAC của máy chủ vật lý vào image — giúp bạn luôn sử dụng đúng máy vật lý mỗi khi bật.

## Bật Hardware Fixation

1. Chọn image, nhấn **Action** → **Hardware fixation**.
2. Tại hộp thoại xác nhận, chọn **Enable**.

## Tắt Hardware Fixation

1. Chọn **Hardware fixation**.
2. Tại hộp thoại, chọn **Disable**.
