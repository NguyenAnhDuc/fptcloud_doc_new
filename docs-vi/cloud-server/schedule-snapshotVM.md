# Schedule Snapshotvm

Tính năng quản lý đặt lịchy tự động (Schedule Management) đáp ứng nhu cầu của các công ty/ tổ chức lớn có lượng tài nguyên lớn cần lưu trữ dữ liệu thường xuyên. Thực hiện hành động tự động thông qua tính năng tạo lịch giúp hạn chế thời gian và sai sót trong quá trình thao tác.
