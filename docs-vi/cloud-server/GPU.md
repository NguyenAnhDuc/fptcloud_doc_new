# Gpu

• FPT GPU Server là dịch vụ máy chủ ảo triển khai trên FPT Cloud với GPU card chuyên dụng, đem đến khả năng xử lý nhiều tác vụ tính toán cũng như đồ họa phức tạp. Dịch vụ kết hợp hoàn hảo giữa tính linh hoạt của FPT Cloud Server với hiệu năng mạnh mẽ của các dòng GPU Card như NVIDIA Tesla A10, A30, A100, RTX 3090 và 4090 đem đến khả năng xử lý tính toán vượt trội, giúp tăng tốc những ứng dụng trí tuệ nhân tạo (AI), Machine Learning, kết xuất đồ họa, xử lý hình ảnh… • Hiện tại FPT đã hỗ trợ gắn thêm các dòng GPU cao cấp lên máy ảo của khách hàng. Nếu bạn có nhu cầu sử dụng GPU, hoặc có vấn đề kỹ thuật trong quá trình sử dụng, hãy liên hệ ngay với đội ngũ CSKH để được hỗ trợ tốt nhất.
