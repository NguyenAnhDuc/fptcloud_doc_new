# Quản lý Subnets

Subnet là mạng con trong VPC — tương đương mạng LAN ở hệ thống vật lý. Bạn có thể gắn một hoặc nhiều Subnet vào máy ảo tùy nhu cầu.

## Tạo Subnet mới

1. Chọn **Networking** → **Subnets**, nhấn **Create subnet**.

   [![Màn hình Subnets](/img/migrated/image-1716262599480-f51ee2ed.png)](/img/migrated/image-1716262599480-f51ee2ed.png)

2. Nhập thông tin: Name, Type (Routed/Isolated), CIDR, Gateway IP, DNS.

3. Nhấn **Create subnet**.
