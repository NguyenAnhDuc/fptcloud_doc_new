# Remove The Network Card From The Virtual Machine

Để gỡ **card mạng** khỏi máy ảo bạn thao tác như sau:
**Bước 1** : Mở tab **Network Interface** của máy ảo. Hệ thống sẽ hiển thị các card mạng đang được gắn với máy ảo này.
[![file](/img/migrated/image-1744793241168-180450d8.png)](/img/migrated/image-1744793241168-180450d8.png)
**Bước 2** : Trên bảng **Network** , trong phần **Actions** của card mạng cần xóa, chọn **Delete**.
[![file](/img/migrated/image-1744793619214-47123e3d.png)](/img/migrated/image-1744793619214-47123e3d.png)
**Bước 3** : Hệ thống sẽ hiển thị popup xác nhận thông tin. Để đồng ý gỡ, chọn **Delete**.
[![file](/img/migrated/image-1712723201176-b2e4e0e8.png)](/img/migrated/image-1712723201176-b2e4e0e8.png)
