# Chỉnh sửa lịch snapshot máy ảo

1. Tại tab **Instance snapshot schedule**, chọn **Action** → **Edit**.

   [![Menu Action Edit](/img/migrated/image-1766044561523-30e56e38.png)](/img/migrated/image-1766044561523-30e56e38.png)

2. Chỉnh sửa thông tin theo nhu cầu.

   [![Chỉnh sửa lịch](/img/migrated/image-1766044589323-730f2f46.png)](/img/migrated/image-1766044589323-730f2f46.png)

3. Nhấn **Update schedule**.
