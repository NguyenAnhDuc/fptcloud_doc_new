# Sử dụng dịch vụ theo phương thức PAYG

Khi hết tài nguyên trả trước, bạn có thể tiếp tục tạo tài nguyên theo hình thức thanh toán PAYG (Pay As You Go).

## Dịch vụ hỗ trợ PAYG

1. **Máy ảo**: Tạo, clone, resize, tạo snapshot/template.
2. **Storage**: Tạo, resize, tạo snapshot từ storage.
3. **IP**: Allocate IP.
4. **Load Balancer v1** (chỉ hỗ trợ General).
5. **Auto Scaling**.

## Cách sử dụng

Khi tạo tài nguyên mới, nếu quota đã hết, hệ thống thông báo loại tài nguyên nào đã hết và tự động chuyển sang hình thức thanh toán PAYG.
