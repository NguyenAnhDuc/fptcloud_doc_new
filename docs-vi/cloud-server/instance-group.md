# Quản lý Instance Group — Affinity và Anti-Affinity

Instance Group giúp bạn kiểm soát vị trí đặt máy ảo trên host vật lý — đảm bảo các máy ảo cùng nhóm được đặt trên cùng host (Affinity) hoặc khác host (Anti-Affinity) để tối ưu hiệu năng và độ sẵn sàng.
