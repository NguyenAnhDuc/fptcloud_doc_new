# Tổng quan NIC

Card NIC (Network Interface Card) trên FPT Cloud kết nối máy ảo với tài nguyên mạng — cung cấp khả năng truyền dữ liệu hiệu suất cao, linh hoạt và mở rộng.

## Lưu ý trước khi cấu hình

- Đây là tính năng nâng cao, dành cho người có kiến thức về hệ thống mạng.
- Xác định mạng kết nối cho card NIC, đảm bảo máy ảo giao tiếp và truy cập tài nguyên hiệu quả.
- Cấu hình đúng IP address, subnet mask, gateway và DNS.
- Kiểm tra thử nghiệm sau khi cấu hình để đảm bảo không gây vấn đề.
