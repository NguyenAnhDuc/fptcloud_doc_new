# Câu hỏi thường gặp về Token

## Tại sao một số Token không thể revoke được?

Một số Token cũ được tạo trước bản nâng cấp của FPT Cloud Portal không hỗ trợ revoke. Hệ thống hiển thị tooltip: "Tokens created earlier cannot be revoked".

## Token chuyển trạng thái "Expired" khi nào?

Khi Token vượt qua thời gian hết hạn tính từ lúc tạo, hệ thống tự động chuyển sang trạng thái **Expired**. Bạn sẽ không thể truy cập bằng Token đó nữa.
