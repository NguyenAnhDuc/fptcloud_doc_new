# Chi tiết Networking

Kiến trúc mạng FCI được thiết kế để đáp ứng đa dạng nhu cầu — từ hệ thống mạng truyền thống đến VPC hoặc mô hình đặc thù No-gateway.

## Mô hình No-gateway

Bạn có thể tự thiết lập hệ thống riêng, quản lý thông qua máy chủ đóng vai trò router (ví dụ: Check-point firewall). IP được chỉ định trực tiếp vào máy chủ và traffic điều hướng thông qua firewall.
