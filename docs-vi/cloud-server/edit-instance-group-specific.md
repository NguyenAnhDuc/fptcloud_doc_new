# Chỉnh sửa Instance Group

Bạn có thể cập nhật tên hoặc chính sách đặt vị trí của một Instance Group đã có bất kỳ lúc nào.

:::note
Việc thay đổi chính sách của Instance Group có thể ảnh hưởng đến vị trí đặt của các máy ảo đang thuộc nhóm đó.
:::
