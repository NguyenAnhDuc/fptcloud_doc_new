# Xóa Instance Group

Xóa Instance Group khi không còn cần thiết nữa.

:::warning
Việc xóa Instance Group sẽ gỡ bỏ chính sách đặt vị trí. Các máy ảo đã thuộc nhóm này sẽ không được tự động di chuyển — chúng vẫn tiếp tục chạy trên các máy chủ vật lý hiện tại.
:::
