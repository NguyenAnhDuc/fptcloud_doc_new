# Delete Instance Group

_Tính năng áp dụng cho người dùng sử dụng dịch vụ General và Specific._
Lưu ý: Thao tác này sẽ đồng thời gỡ instance group khỏi các instances đã gắn với nó. Việc xóa instance group này sẽ không thể khôi phục lại được.
**Bước 1** : Ở menu chọn **Instance Group** , trong phần **Actions** , chọn **Delete**
[![file](/img/migrated/image-1718788790753-7b92a5c1.png)](/img/migrated/image-1718788790753-7b92a5c1.png)
**Bước 2** : Một hộp thoại cảnh báo sẽ hiện lên, hiển thị tên Instance Group và yêu cầu người dùng xác nhận. Người dùng nhập **delete** và chọn **Delete instance group** để xác nhận xóa. Hệ thống sẽ tiến hành gỡ các máy ảo gắn với instance group. Sau đó hệ thống sẽ xóa hoàn toàn Instance group.
[![file](/img/migrated/image-1718788805066-97bc894a.png)](/img/migrated/image-1718788805066-97bc894a.png)
