# Zalo ticket support

This guide explains how to create FPT Cloud support tickets through the Zalo OA channel or directly via the support portal at [https://support.fptcloud.com](https://support.fptcloud.com).

## Create a support ticket via Zalo OA

Use the Zalo app to scan the QR code and **follow** FPT Cloud's official account (OA).

## Submit a support request from the FPT Cloud Portal

**Step 1:** Access the FPT Cloud support page.

**Option 1:** Go directly to [https://support.fptcloud.com](https://support.fptcloud.com/) and log in with your registered FPT Cloud account.

[![Login page](/img/migrated/image-1702289190502-0108bf6c.png)](/img/migrated/image-1702289190502-0108bf6c.png)

Select **Submit a ticket** to create a support request.

[![Submit a ticket](/img/migrated/image-1702289199279-8b54294a.png)](/img/migrated/image-1702289199279-8b54294a.png)

**Option 2:** Via FPT Cloud Zalo OA — type `/support` in the official support group chat or chat directly with the FPT Cloud Official Account. A link to the support page will be sent to you.

[![Zalo support](/img/migrated/image-1702289383339-7cfae5fb.png)](/img/migrated/image-1702289383339-7cfae5fb.png) [![Support link](/img/migrated/image-1702289397554-222cc3fe.png)](/img/migrated/image-1702289397554-222cc3fe.png)

After clicking the link, log in to create your ticket.

[![Support portal](/img/migrated/image-1702289421857-a00d06a2.png)](/img/migrated/image-1702289421857-a00d06a2.png)

**Step 2:** Fill in the required information.

- **Subject:** Title of the support request.
- **Organization name:** Your tenant name on FPT Cloud.
- **I need support for:** Select the service you need help with.
- **Priority level:**
  - **Urgent:** System is down, inaccessible, or internet connection is lost.
  - **High:** System is slow, intermittent, or there is a major impact on your service.
  - **Medium:** Low impact; the system is still functioning normally.
  - **Low:** Service questions or basic guidance and support requests.
- **Description:** Describe your issue in detail.

[![Ticket form](/img/migrated/image-1702289433410-a3de407d.png)](/img/migrated/image-1702289433410-a3de407d.png)

After filling in all the information, click **Submit**. Your request will be recorded in the system and a confirmation email will be sent.

[![Confirmation](/img/migrated/image-1702289454316-840e1c55.png)](/img/migrated/image-1702289454316-840e1c55.png)

**Step 3:** Enable automatic ticket status notifications via Zalo.

Click the link at the bottom of the confirmation email. A QR code will appear — scan it with the Zalo app and select **Allow** to grant FPT Cloud access to your Zalo name and profile picture.

[![Zalo notification setup](/img/migrated/image-1702289472711-917181f5.png)](/img/migrated/image-1702289472711-917181f5.png)

You will now receive automatic ticket status updates via Zalo chat.

[![Zalo notifications](/img/migrated/image-1702289485901-918b287b.png)](/img/migrated/image-1702289485901-918b287b.png)

## Check ticket status

- If you completed the steps above, Zalo will automatically send ticket status updates.
- You can also check ticket status directly on the [FPT Cloud Support Portal](https://support.fptcloud.com).
