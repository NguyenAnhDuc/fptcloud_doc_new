# View logs

Security logs record all malware detection events, exploits, and suspicious activity on protected servers. Reviewing logs regularly helps you detect threats early and respond quickly.

## View system-wide logs

1. In the administration console, select **Threat Event Log** from the menu bar.

   [![Threat Event Log page in the Trellix administration console](/img/migrated/image-1765765683274-68178a40.png)](/img/migrated/image-1765765683274-68178a40.png)

   This page displays all security events from servers with the agent installed, including malware infections, exploits, and other suspicious activity.

   [![Security event list in Threat Event Log](/img/migrated/image-1765765697231-084be1af.png)](/img/migrated/image-1765765697231-084be1af.png)

## View logs for a specific server

1. In **System Tree** > **Group** > **System**, double-click the server you want to inspect.

   [![System Tree interface for selecting a specific server](/img/migrated/image-1765765708664-117612a4.png)](/img/migrated/image-1765765708664-117612a4.png)

2. Select the **Threat Events** tab to view events for that server.

   [![Threat Events tab showing logs for a single server](/img/migrated/image-1765765721659-77184455.png)](/img/migrated/image-1765765721659-77184455.png)

## View logs directly on the server

1. On the server, right-click the Trellix icon in the task bar and select **Trellix ENS**.

   [![Trellix ENS right-click menu in the Windows task bar](/img/migrated/image-1765765732756-8f460d01.png)](/img/migrated/image-1765765732756-8f460d01.png)

2. In the **Event Log** section, the same Threat Events are displayed.

   [![Event Log section in Trellix ENS on the server](/img/migrated/image-1765765789911-5df77202.png)](/img/migrated/image-1765765789911-5df77202.png)

3. **View Log Folder** contains the Endpoint Security log files, including the history of scheduled scanning tasks.

   [![Endpoint Security log file directory](/img/migrated/image-1765765814382-c01507d6.png)](/img/migrated/image-1765765814382-c01507d6.png)
