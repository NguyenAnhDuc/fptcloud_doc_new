# View reports

Trellix provides built-in report templates for monitoring malware detection, exploit, and crack-tool activity across multiple time ranges. Reports give you a quick overview of the overall security situation without manually reviewing individual logs.

1. In the administration console, select **Queries & Reports** from the menu bar.

   [![Queries & Reports page in the Trellix interface](/img/migrated/image-1765765637950-de5f4b98.png)](/img/migrated/image-1765765637950-de5f4b98.png)

2. Select **Shared Groups** > **Report**.

   [![Report list in Shared Groups](/img/migrated/image-1765765653374-e18ef375.png)](/img/migrated/image-1765765653374-e18ef375.png)

   FPT Cloud has pre-configured the following report templates:

   - Malware behavior report (24 hours / 7 days / 30 days)
   - Exploit behavior report (24 hours / 7 days / 30 days)
   - Crack-tool execution behavior report (24 hours / 7 days / 30 days)

:::note
To create a custom report, click **New Query** and configure the parameters as needed.
:::
