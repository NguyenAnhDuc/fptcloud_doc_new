# Access the administration interface

The Trellix administration console lets you monitor all servers with the agent installed and manage security policies by group. Logging in correctly is the first step to operating the anti-virus system effectively.

1. Open a browser and go to `https://am.fptcloud.com:8443`. Enter the username and password provided by FCI.

   [![Trellix login screen](/img/migrated/image-1765765382780-8e758eaf.png)](/img/migrated/image-1765765382780-8e758eaf.png)

2. After logging in successfully, the Trellix administration interface is displayed.

   [![Trellix administration interface after login](/img/migrated/image-1765765391556-388ea73c.png)](/img/migrated/image-1765765391556-388ea73c.png)

   - **System Tree**: displays a list of servers with the agent installed.
     - **New Subgroups**: create sub-groups to separate administration. Each group or sub-group can have its own security policy applied.

   [![System Tree in the Trellix administration console](/img/migrated/image-1765765400751-504e9a39.png)](/img/migrated/image-1765765400751-504e9a39.png)
