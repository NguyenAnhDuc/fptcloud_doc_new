# Tính năng chính

- **Kiểm thử prompt:** Gửi prompt và quan sát kết quả completion của mô hình.
- **Tùy chỉnh system/user message:** Xác định vai trò và kiểm soát ngữ cảnh cuộc trò chuyện.
- **Điều chỉnh tham số:** Tinh chỉnh các cài đặt như temperature, top P, stop sequence và độ dài đầu ra.
- **Đầu vào hình ảnh (VLM):** Tải lên file .jpg hoặc .jpeg cho Vision-Language Models.
- **Lịch sử completion:** Xem, so sánh và quản lý các kết quả trước đây.
- **Quản lý session:** Dễ dàng bắt đầu, tiếp tục hoặc xóa interactive session.
