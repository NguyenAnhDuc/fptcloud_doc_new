# Chọn Model

[![Alt text](/img/migrated/Create-New-Jobs-Select-Model-bd425c7b.png)](/img/migrated/Create-New-Jobs-Select-Model-bd425c7b.png)
Bạn chọn model từ **Model Hub** theo hai cách:
  * **Model Catalog:** Kho lưu trữ các nguồn model từ nhiều nhà cung cấp khác nhau như DeepSeek, Gemma, Llama và Qwen.
  * **Private Model:** Kho lưu trữ cho các model do người dùng sở hữu và các model đã được fine-tuned. Các model này phải đã **tải lên đầy đủ tất cả các file cần thiết**. Phải bao gồm **các tag cụ thể** để xác định tính tương thích và sẵn sàng sử dụng.

**Lưu ý:** Để sử dụng trong Test Jobs, các model phải đáp ứng các tiêu chí sau:
  * **Loại model:**
    * LLM: Chỉ nhận đầu vào văn bản
    * VLM: Nhận cả đầu vào văn bản và hình ảnh
  * **Kích thước model** > 0
  * **Giai đoạn học:** Phải là **Instruction-tuned**
