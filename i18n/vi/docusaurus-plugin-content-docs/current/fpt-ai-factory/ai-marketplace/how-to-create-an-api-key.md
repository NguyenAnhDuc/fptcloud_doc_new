# Tạo API key

Để tạo và quản lý API key, vào **My Account → My API Keys** trong dashboard của AI Marketplace.

:::tip
Để xem hướng dẫn từng bước đầy đủ bao gồm tạo, chỉnh sửa và thu hồi key, xem hướng dẫn [Tài khoản của tôi](/vi/docs/fpt-ai-factory/ai-marketplace/my-account).
:::
