# Stop Session

1. Mở **Session list**.
2. Tìm session bạn muốn hủy và nhấp vào biểu tượng 3 chấm trong cột **Actions**.
3. Nhấp vào hành động **Stop**.
4. Hộp thoại xác nhận xuất hiện. Xác nhận bằng cách nhấp nút **Confirm** sau 3 giây.

**Lưu ý:**

* Một session đã dừng không thể khởi động lại. Nếu bạn cần tiếp tục làm việc, bạn phải tạo một session mới.
* Mọi kết quả được tạo ra trước khi dừng vẫn còn sẵn cho bạn.
