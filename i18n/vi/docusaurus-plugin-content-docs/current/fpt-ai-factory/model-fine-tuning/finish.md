# Finish

[![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/Finish.png)](https://fptcloud.com/wp-content/uploads/2025/02/Finish.png)

Cuối cùng, bạn phải nhập một tên pipeline duy nhất (ví dụ: ft.pipeline_20250905235412) thỏa mãn các điều kiện sau:
  * Bắt đầu bằng chữ cái hoặc số
  * Chỉ sử dụng chữ cái (a-z, A-Z), số (0-9), dấu gạch dưới "_" và dấu gạch ngang "-"
  * Tối đa 100 ký tự

Tùy chọn thêm mô tả pipeline tối đa 200 ký tự và nhận thông báo qua email (noreply@fptcloud.com) khi pipeline **SUCCEEDED** hoặc **FAILED**.
