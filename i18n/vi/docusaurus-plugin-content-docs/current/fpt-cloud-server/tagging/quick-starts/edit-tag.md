# Chỉnh sửa tag

Khi cần thay đổi thông tin tag đã tạo, bạn có thể chỉnh sửa Key, Value, Color hoặc resource scope mà không cần xóa và tạo lại.

## Chỉnh sửa

1. Chọn **Tagging** từ menu, tại tag cần chỉnh sửa, chọn **Actions** → **Edit**.

   [![Menu Actions với tùy chọn Edit](/img/migrated/image-1764325579411-6236079e.png)](/img/migrated/image-1764325579411-6236079e.png)

2. Thay đổi **Key**, **Value** hoặc **Color** của tag. Bạn không thể thay đổi Tag Level, nhưng có thể thay đổi resource scope đối với level Project/VPC.

   [![Hộp thoại Edit tag với các trường Key, Value và Color](/img/migrated/image-1764325607970-6f8635d5.png)](/img/migrated/image-1764325607970-6f8635d5.png)

:::note
Bạn không thể thay đổi **Tag Level** sau khi tạo.
:::
