# Tìm kiếm toàn cục

Tính năng Global Search trên FPT Cloud cho phép bạn tìm kiếm nhanh các tài nguyên trong hệ thống, bao gồm **Instances, IP Addresses, Subnets, Security Groups và Database**.

Tính năng này giúp tiết kiệm thời gian và tăng hiệu quả quản lý tài nguyên trên nền tảng FPT Cloud.
