# Xóa máy ảo

Khi không còn nhu cầu sử dụng, bạn có thể xóa máy ảo khỏi hệ thống để tránh phát sinh thêm chi phí.

:::danger
Xóa máy ảo sẽ xóa toàn bộ dữ liệu và không thể phục hồi. Đảm bảo máy ảo đang ở trạng thái **Unlock Deletion** trước khi xóa.
:::

## Xóa máy ảo

1. Tại bảng quản trị **Instance Management**, chọn **Actions** → **Delete**.

   [![Menu Actions với tùy chọn Delete](/img/migrated/image-1712200768564-6389b6eb.png)](/img/migrated/image-1712200768564-6389b6eb.png)

   [![Xác nhận chọn Delete](/img/migrated/image-1712200780901-998413fe.png)](/img/migrated/image-1712200780901-998413fe.png)

2. Hộp thoại cảnh báo hiển thị tên máy ảo. Nhập **delete** để xác nhận.

   [![Hộp thoại xác nhận xóa máy ảo](/img/migrated/image-1712200787729-9546cde9.png)](/img/migrated/image-1712200787729-9546cde9.png)

   [![Nhập delete để xác nhận](/img/migrated/image-1712200798790-c16164fa.png)](/img/migrated/image-1712200798790-c16164fa.png)

3. Hệ thống tiến hành xóa và thu hồi tài nguyên.

   [![Máy ảo đang được xóa](/img/migrated/image-1712200803567-91c8fdd5.png)](/img/migrated/image-1712200803567-91c8fdd5.png)
