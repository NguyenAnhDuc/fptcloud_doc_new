# Quản lý Instance Template

Template là bản sao dạng Custom Image của máy ảo — bao gồm disk, thiết bị ảo và settings. Template giúp bạn tạo nhanh nhiều máy ảo cùng cấu hình mà không cần thiết lập lại từ đầu.
