# Đặt mật khẩu cấp 2

Mật khẩu cấp 2 là mật khẩu do bạn thiết lập sau khi đăng nhập HPC Portal — dùng để đăng nhập vào máy khi Remote Desktop hoặc SSH.

## Thiết lập mật khẩu cấp 2

1. Chọn biểu tượng user tại header, nhấn vào username.
2. Tại trang cài đặt, chọn tab **Level 2 password**.
3. Nhập mật khẩu mới, nhấn **Submit**.
