# Xem transaction đã thực hiện

Bạn có thể xem lịch sử các giao dịch đã thực hiện trên hệ thống, bao gồm:

- **Nạp tiền**: qua VNPay hoặc gift code.
- **Thanh toán**: phí dịch vụ vào cuối tháng.
