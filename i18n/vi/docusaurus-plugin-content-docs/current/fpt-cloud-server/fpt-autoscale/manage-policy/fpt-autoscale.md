# FPT Autoscale

FPT Autoscale là dịch vụ cho phép tạo và quản lý đồng thời nhiều Virtual Instance. Dịch vụ hỗ trợ tự động điều chỉnh số lượng Virtual Instance theo nhu cầu sử dụng, nâng cao độ ổn định của ứng dụng, giảm thiểu chi phí phát sinh và tiết kiệm thời gian vận hành.
