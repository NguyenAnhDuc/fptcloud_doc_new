# Xóa superset

Để xóa **Apache Superset**, Người dùng thực hiện các bước sau:

**Bước 1:** Tại thanh menu chọn **Data Platform** > chọn **Workspace Management** > chọn **Workspace name**

**Bước 2:** Tại phần **My serivces** chọn **Apache Superset** > nhấn vào nút **Action** góc phải màn hình chọn **delete**

![Hình ảnh](images/xoa-superset/img-001.jpg)

**Bước 3.** Hiển thị hộp thoại **Delete application** > nhập **delete** > nhấn **Confirm** để xóa hoàn thành việc xóa app

![anh-3](images/xoa-superset/img-002.png)
