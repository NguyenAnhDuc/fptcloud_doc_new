# Metal Cloud là gì

Metal Cloud là dịch vụ cloud cho phép bạn thuê máy chủ vật lý tích hợp GPU từ FPT. Máy chủ này dành riêng cho bạn và không chia sẻ với tenant khác. Bạn có thể truy cập server với toàn quyền kiểm soát phần cứng mà không cần lớp ảo hóa hay trừu tượng hóa, giúp tối đa hiệu năng cho các workload tính toán chuyên sâu.

*(Server này còn được gọi là Bare Metal GPU Server.)*

# Cách thức hoạt động

Khi bạn thuê Bare Metal GPU Server, máy vật lý được phân bổ riêng cho bạn và cấu hình mạng vật lý được thiết lập dành riêng cho bạn.

Môi trường server hoàn toàn độc lập, cho phép bạn truy cập 100% năng lực tính toán và mạng của máy.

Bạn cấu hình mạng (subnet/IP), SSH key, user data và cài đặt hệ điều hành để triển khai server thông qua FPT Customer Portal.

# Tại sao chọn Metal Cloud

- Phần cứng chuyên dụng cho hiệu năng tối đa
- Khả năng sẵn sàng cao và khả năng mở rộng của tài nguyên
- Tùy chỉnh linh hoạt ở mức cao
- Hiệu năng và bảo mật được cải thiện
- Chi phí có thể dự đoán trước
