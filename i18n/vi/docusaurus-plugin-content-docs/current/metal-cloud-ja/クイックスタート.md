# Bắt đầu nhanh

Bare Metal GPU Server là server chuyên dụng single-tenant trang bị 8 GPU NVIDIA H100/H200, có thể hoạt động độc lập hoặc trong cluster đa node.

## Đăng ký tài khoản

1. Truy cập [https://id.fptcloud.com](https://id.fptcloud.com) và chọn Sign Up.
2. Nếu bạn không thấy email xác nhận trong hộp thư đến, hãy kiểm tra thư mục spam.
3. Để bắt đầu sử dụng dịch vụ Metal Cloud, hãy đăng nhập tại [https://console.fptcloud.com](https://console.fptcloud.com).

## Tạo subnet

1. Bạn cần có subnet để tạo và triển khai Bare Metal GPU.
2. Nhấn AI Infrastructure và chọn Subnet trong menu bên.
3. Xem hướng dẫn chi tiết tại đây.

**Tạo Bare Metal GPU Server**

1. Nhấn **AI Infrastructure** và chọn **Metal Cloud** trong menu bên.
2. Chọn **Create server** và cấu hình việc triển khai server.

- Có 2 (02) **FPT image dành cho GPU và AI/ML** khả dụng. Kiểm tra các phụ thuộc phần mềm tại đây.
- Bạn có thể chọn floating IP (public IP) khi tạo để truy cập server qua internet, hoặc gắn sau tại Network/Floating IP.

3. Xem hướng dẫn chi tiết tại đây.

## Truy cập server

1. Thiết lập Network ACL tại đây để tạo inbound rule cho phép public IP.
2. Kết nối SSH vào server bằng SSH key hoặc mật khẩu đã thiết lập khi tạo.
3. Hướng dẫn này cũng giúp bạn thiết lập jump host cho hầu hết các cài đặt bảo mật.

## Lưu ý

1. Sau khi truy cập FPT Cloud Console, **hãy kiểm tra tenant và region**. **AI Factory chỉ được hỗ trợ ở region Hà Nội 2 (Việt Nam) và Tokyo (Nhật Bản).**
2. **Bạn có thể thay đổi mặc định Network ACL, nhưng cần thiết lập các rule cần thiết khác để tạo server.**

Sau đây là các outbound rule mặc định của Network ACL, có thể xóa:

| Độ ưu tiên | Loại | Giao thức | Cổng | Nguồn | Traffic Action |
| --- | --- | --- | --- | --- | --- |
| 100 | ALL | ALL | ALL | 0.0.0.0/0 | ALLOW |

Nếu bạn xóa rule mặc định trên, cần thêm các outbound rule sau để tạo và triển khai server:

| Độ ưu tiên | Loại | Giao thức | Cổng | Nguồn | Traffic Action |
| --- | --- | --- | --- | --- | --- |
| 1 | HTTP | TCP | 80 | 0.0.0.0/0 | ALLOW |
| 2 | HTTPS | TCP | 443 | 0.0.0.0/0 | ALLOW |
| 3 | DNS (UDP) | UDP | 53 | 0.0.0.0/0 | ALLOW |

3. FPT thiết lập giới hạn tài nguyên (quota) theo hợp đồng/thỏa thuận của bạn. Khi hết quota, bạn không thể tạo tài nguyên mới. Hãy liên hệ sales hoặc **FPT Smart Cloud Support** để được hỗ trợ.
