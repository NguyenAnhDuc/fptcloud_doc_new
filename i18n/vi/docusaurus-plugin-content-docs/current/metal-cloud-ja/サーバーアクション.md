# Bật và tắt server

1. Vào **AI Infrastructure** và **Metal Cloud** trong sidebar để xem danh sách server.
2. Chọn server và nhấn **"Actions"**.
3. Nhấn **"Power off"** để tắt server đang chạy, hoặc nhấn **"Power on"** để bật server đang dừng.
4. Xác nhận hành động tắt hoặc bật nguồn.

# Xóa server

_Khi bạn xóa server, tất cả dữ liệu trên ổ đĩa sẽ bị xóa vĩnh viễn và không thể phục hồi. Hãy đảm bảo bạn đang xóa đúng server._

1. Vào **AI Infrastructure** và **Metal Cloud** trong sidebar để xem danh sách server.
2. Chọn server và nhấn **Actions**.
3. Nhấn **Delete**.
4. Nhập văn bản **DELETE** để xác nhận xóa.

# Đổi tên

_Tên server phải duy nhất và không được trùng với tên hiện có._

1. Vào **AI Infrastructure** và **Metal Cloud** trong sidebar để xem danh sách server.
2. Chọn server và nhấn **Actions**.
3. Nhấn **Rename**.
4. Nhập tên server mới theo quy tắc: tên tối đa 63 ký tự, chỉ dùng chữ cái, chữ số và dấu gạch ngang.
5. Nhấn **Rename server**.
