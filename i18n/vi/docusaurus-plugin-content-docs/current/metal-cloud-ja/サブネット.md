# Tạo subnet

Subnet là dải địa chỉ IP trong mạng cloud. Các địa chỉ trong dải này được gán cho một hoặc nhiều server Metal Cloud. Chỉ hỗ trợ IPv4.

_Dải CIDR được tự động phân bổ, được giữ trong 60 giây rồi tự động cập nhật sang dải mới trên màn hình. Cần có Network ACL để tạo subnet. Mặc định, chỉ traffic outbound từ Bare Metal GPU Server ra internet được cho phép. Quota subnet bằng với quota Bare Metal GPU Server._

1. Đăng nhập tài khoản FPTCloud và chọn **tenant, region, VPC** (nếu có nhiều).
2. Vào **AI Infrastructure** và **Subnet for Metal Cloud** trong sidebar, nhấn **Create subnet**.
3. Nhập **tên subnet**.
4. Nhập **tên ACL mạng**.
5. Nhập **mô tả ACL mạng** (tùy chọn).
6. Nhấn **Create Subnet**.

# Xóa subnet

_Bạn không thể xóa subnet đang được liên kết với tài nguyên khác._

1. Vào **AI Infrastructure** và **Subnet for Metal Cloud**.
2. Xem danh sách server trong sidebar.
3. Chọn subnet và nhấn **"Actions"**. 4. Nhấn **Delete**. 5. Nhập văn bản **DELETE** để xác nhận xóa.

# Đổi tên subnet

_Tên subnet có thể trùng với tên hiện có._

1. Chọn subnet và nhấn **"Actions"**.
2. Nhấn **Rename**.
3. Nhập tên subnet mới theo quy tắc. Tên tối đa 32 ký tự, chỉ dùng chữ cái, chữ số và dấu gạch ngang.
4. Nhấn **Rename subnet**.

# Cập nhật mô tả subnet

1. Chọn subnet và nhấn **"Actions"**.
2. Nhấn **Update description**.
3. Nhập mô tả mới. Độ dài không được quá 63 ký tự.
4. Nhấn Save.
