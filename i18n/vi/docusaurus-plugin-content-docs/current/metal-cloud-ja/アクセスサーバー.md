# Console KVM

Bạn có thể kết nối Bare Metal GPU Server bằng SSH key hoặc mật khẩu. SSH protocol (còn gọi là Secure Shell) là phương thức đăng nhập từ xa an toàn từ server này sang server khác. Để kết nối qua SSH, hãy đảm bảo tất cả rule cần thiết cho traffic vào đều có trong cài đặt Network ACL.

1. Vào **AI Infrastructure** và **Metal Cloud** trong sidebar để xem danh sách server.
2. Chọn server từ danh sách hoặc xem chi tiết server.
3. Nhấn **Actions** và nhấn **Console**, hoặc nhấn **Open** trong phần console trên trang chi tiết server.
4. Để đăng nhập, dùng **tên user mặc định: clouduser** và mật khẩu bạn đã nhập khi tạo server.

# SSH

_SSH key được chọn khi tạo và triển khai server. Tham khảo SSH key management._

## Điều kiện tiên quyết trước khi kết nối từ Windows 10

1. Mở **Windows Settings**.
2. Vào mục **Apps & features** và nhấn **Optional features**.
3. Tìm **OpenSSH Client** và nhấn để xem mô tả chi tiết.
4. Nhấn **Install**.
5. Chờ quá trình cài đặt hoàn tất. Sau khi cài SSH client, hãy khởi động lại máy tính để áp dụng cài đặt đúng cách. Lệnh SSH sẽ có thể dùng trong cmd.

## Kết nối từ Windows 10, Linux OS, macOS

1. Mở Command Prompt.

| Phương thức xác thực | Lệnh |
| --- | --- |
| Mật khẩu | `ssh username@192.168.1.92` |
| SSH key | `ssh username@192.168.1.92 -i "C:\\Users\\username\\.ssh\\id_rsa` |

Trong đó:
- Thay "username" bằng tên user của bạn hoặc tên user mặc định khi đăng nhập lần đầu (clouduser).
- Thay "192.168.1.92" bằng địa chỉ floating IP của server.
- Thay "C:\Users\username\\.ssh\id_rsa" bằng đường dẫn đến file private key định dạng PEM trên máy tính của bạn.

_Nếu bạn tạo Bare Metal GPU Server chỉ với private interface, hãy tạo floating IP address và dùng nó khi kết nối SSH vào BareMetal._

2. Tiện ích sẽ cảnh báo bạn đang kết nối đến thiết bị không xác định và hỏi có muốn tiếp tục không. Gõ **"yes"** và nhấn **Enter** để tiếp tục.
3. (Chỉ cho kết nối bằng mật khẩu) Nhập mật khẩu bạn đã thiết lập khi tạo **Bare Metal**.

# Sử dụng Bastion Host (Jump Server)

Jump host là server trung gian giữa máy gốc và Bare Metal GPU Server bạn muốn kết nối. Nó hoạt động như một cổng kết nối giữa hai mạng tin cậy. Bạn chỉ có thể truy cập server đích sau khi jump host cho phép truy cập.

1. Tạo cloud instance có subnet và floating IP.
2. Cấu hình security group.
