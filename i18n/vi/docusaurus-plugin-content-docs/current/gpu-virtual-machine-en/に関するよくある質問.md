# Câu hỏi thường gặp

### 1. GPU Virtual Machine là gì?

GPU Virtual Machine (GPU VM) là một hệ thống độc lập được tạo từ cụm GPU của tài nguyên phần cứng, được trang bị GPU chuyên dụng, CPU, bộ nhớ, giao diện mạng và storage. Có sẵn nhiều máy ảo được cấu hình sẵn được thiết kế để phù hợp với yêu cầu workload, cung cấp các tùy chọn linh hoạt từ 1 đến 8 GPU mỗi VM.

### 2. Tôi có thể thay đổi kích thước GPU instance không — CPU, RAM hoặc disk?

GPU VM cung cấp các flavor được cấu hình sẵn cho GPU, CPU, RAM và disk, và không thể tùy chỉnh theo yêu cầu. Hãy chọn cấu hình phù hợp hoặc liên hệ **FPT Cloud Support**.

### 3. Làm thế nào để gán IP công cộng cho GPU instance?

**Floating IP** chỉ có thể được gán sau khi GPU instance được tạo thành công. Lưu ý rằng bạn cũng phải cấu hình IP port (port forwarding) để hoàn tất cấu hình IP. Xem hướng dẫn chi tiết để biết thêm thông tin.

### 4. Những tính năng nào không được hỗ trợ cho GPU Virtual Machine?

Các tính năng sau không được hỗ trợ cho GPU Virtual Machine với NVIDIA Hopper (H100, H200):

  * Thay đổi kích thước instance: thêm hoặc xóa GPU, thay đổi kích thước CPU, RAM hoặc disk
  * Snapshot
  * Tạo template từ GPU instance
  * Hiện chỉ hỗ trợ backup instance sử dụng Veeam.

### 5. Những tùy chọn storage nào khả dụng cho GPU Virtual Machine?

  * Hiện tại, chỉ NVMe physical disk được hỗ trợ cho GPU instance.
  * GPU Virtual Machine với Block Storage sẽ sớm ra mắt.
  * Dịch vụ File Storage (tầng hiệu năng cao) được hỗ trợ.

### 6. GPU Virtual Machine có sẵn ở những khu vực nào?

GPU Virtual Machine với NVIDIA Hopper (H100, H200) có sẵn tại Hà Nội 2, Việt Nam và Tokyo, Nhật Bản.

### 7. Tôi được tính phí như thế nào?

  * **Reserved:** Tài nguyên bị giới hạn dựa trên nhu cầu với giá cố định, thanh toán trước (một phần hoặc toàn bộ) hoặc thanh toán sau. Chu kỳ thanh toán là hàng tháng (3–9 tháng) hoặc hàng năm (1–5 năm).
  * **Pay-As-You-Go (PAYG):** Bắt đầu với tài nguyên không giới hạn và thanh toán sau. Phí thường được tính theo ngày hoặc giờ.

### 8. FPT có tính phí cho GPU Virtual Machine ở trạng thái dừng không?

Có. Instance ở trạng thái "stopped" vẫn giữ chỗ máy chủ cho bạn sử dụng, vì vậy phí vẫn tiếp tục cho đến khi bạn giải phóng máy chủ. Nếu bạn không muốn bị tính phí cho máy chủ nữa, hãy xóa instance trên FPT Cloud Portal. Bạn cũng bị tính phí cho các dịch vụ hoặc tài nguyên khác được gắn kèm với GPU instance.
