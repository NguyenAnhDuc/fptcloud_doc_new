# Tạo Security Group

**Bước 1:** Sau khi tạo GPU VM, vào **Network** > **Security Groups**.

[![Alt text](/img/migrated/14-1-281806c6.png)](/img/migrated/14-1-281806c6.png)

**Bước 2:** Nhấp **Create Security Group** để tạo Security Group mới.

**Bước 3:** Trong màn hình **Create Security Group**, điền các trường sau.

  * **Tên:** Đặt tên cho Security Group (tên ngẫu nhiên được tạo mặc định).
  * **Applied instance:** Chọn **GPU VM** mới tạo để liên kết với Security Group.
  * **Add tag:** Tùy chọn — giúp tổ chức tài nguyên hiệu quả hơn.
  * **Cấu hình security rules:** Tạo **inbound rule** với các cài đặt sau:

```
Type: SSH
Port range: 22
Source: All IPv4
```

[![Alt text](/img/migrated/15-1-4fa4db41.png)](/img/migrated/15-1-4fa4db41.png)

**Bước 4:** Sau khi nhập thông tin, nhấp **Create Security Group** để xác nhận. Security Group mới tạo xuất hiện trong danh sách.

> 📝 **Lưu ý:** Để xóa Security Group, bạn phải xóa tất cả các rules trước và tách tất cả các instance liên kết.
