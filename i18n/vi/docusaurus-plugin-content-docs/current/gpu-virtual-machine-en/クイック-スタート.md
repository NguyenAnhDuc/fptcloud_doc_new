# Quick Start

GPU Virtual Machine là máy chủ đám mây hiệu năng cao kết hợp tính linh hoạt của đám mây với NVIDIA H100 và H200. Làm theo các bước sau để triển khai GPU VM.

**Đăng nhập vào Console.**
  1. Truy cập <https://console.fptcloud.com> và đăng nhập.
  2. Điều hướng đến màn hình Instance (màn hình mặc định) trên trang console.
  3. Làm theo hướng dẫn thiết lập ban đầu [tại đây](/vi/docs/gpu-virtual-machine-en/)/.

**Tạo Subnet.**
Cần có Subnet để tạo và triển khai GPU VM.
  1. Nhấp **Network** và chọn **Subnets** trong menu bên.
  2. Xem hướng dẫn chi tiết [tại đây](/vi/docs/gpu-virtual-machine-en/)/.

**Tạo GPU VM.**
  1. Nhấp **Compute Engine** và chọn **Instance Management** trong menu bên.
  2. Chọn **Create Instance** trong màn hình Instance Management.
  3. Cấu hình triển khai máy ảo.
  4. **Floating IP và Security Group** có thể được cấu hình sau khi tạo VM, không phải trong bước này.
  5. Xem hướng dẫn chi tiết [tại đây](/vi/docs/gpu-virtual-machine-en/)/.

**Cấp phát địa chỉ IP (Floating IP).**
  1. Nhấp **Network** và chọn **Floating IPs** trong menu bên.
  2. Chọn **Allocate IP address** trong màn hình Floating IPs.
  3. Gán địa chỉ IP cho VM mới tạo.
  4. Xem hướng dẫn chi tiết [tại đây](/vi/docs/gpu-virtual-machine-en/)/.

**Tạo Security Group.**
  1. Nhấp **Network** và chọn **Security Groups** trong menu bên.
  2. Chọn **Create Security Group** trong màn hình Security Groups.
  3. Tạo inbound rules cho VM mới tạo.
  4. Xem hướng dẫn chi tiết [tại đây](/vi/docs/gpu-virtual-machine-en/)/.

**Truy cập máy chủ.**
  1. Sau khi GPU VM được tạo, kết nối đến máy chủ qua SSH bằng SSH key của bạn.
  2. Bạn cũng có thể truy cập bằng cách nhấp **"Open at Console"** trên trang chi tiết máy chủ và đăng nhập bằng mật khẩu.
  3. Xem hướng dẫn chi tiết [tại đây](/vi/docs/gpu-virtual-machine-en/)/.
