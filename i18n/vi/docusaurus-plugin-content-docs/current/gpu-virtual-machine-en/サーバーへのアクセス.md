# Truy cập máy chủ

Sau khi GPU VM (với Ubuntu OS) được tạo thành công trên FPT Portal, bạn có thể truy cập mặc định bằng Web Console tích hợp sẵn.

Bạn cũng có thể đăng nhập từ bên ngoài qua SSH client hoặc phần mềm bên thứ ba như PuTTY hoặc Bitvise.

## Kết nối GPU VM qua Web Console

Web Console cho phép bạn điều khiển tất cả GPU VM trên **FPT Cloud** — kể cả những VM không có IP công cộng.

Trong menu, vào **Instance Management**, tìm máy ảo cần truy cập và chọn **Console** trong phần **Action**.

[![Alt text](/img/migrated/4.1-1-d8d47791.png)](/img/migrated/4.1-1-d8d47791.png)

Trình duyệt ngay lập tức mở cửa sổ mới hiển thị màn hình máy chủ, cho phép bạn kiểm soát và vận hành máy chủ được kết nối.

[![Alt text](/img/migrated/17-a3766778.png)](/img/migrated/17-a3766778.png)

## Kết nối GPU VM qua SSH client

Bạn có thể kết nối đến máy ảo qua SSH bằng SSH client tích hợp sẵn của Windows như **cmd.exe** hoặc **PowerShell**.

**Bước 1:** Mở **CMD** hoặc **PowerShell** trên Windows.

**Bước 2:** Để kết nối bằng mật khẩu, sử dụng lệnh sau:

  * `ssh <Username>@<IP_ADDRESS>`

[![Alt text](/img/migrated/5-2-4e4809f6.png)](/img/migrated/5-2-4e4809f6.png)

Để kết nối bằng SSH key pair, trước tiên điều hướng đến thư mục chứa file SSH key bằng lệnh `cd`:

  * `cd <ĐƯỜNG_DẪN_ĐẾN_SSH_KEY>`

Sau đó kết nối đến máy ảo bằng:

  * `ssh -i "<FILE_KEY>" <Username>@<IP_ADDRESS>`

**Bước 3:** Khi kết nối đến máy ảo lần đầu tiên, bạn được nhắc xác minh fingerprint của host. Nhập **"yes"** và nhấn **Enter** để tiếp tục.

[![Alt text](/img/migrated/6-2-20d1ad6c.png)](/img/migrated/6-2-20d1ad6c.png)

Bạn đã kết nối thành công đến máy chủ qua SSH client.

[![Alt text](/img/migrated/7-2-b23b7851.png)](/img/migrated/7-2-b23b7851.png)

> 📝 **Lưu ý:** Nếu bạn gặp lỗi **"WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!"**, sử dụng lệnh sau để xóa fingerprint host đã lưu trước đó và thực hiện lại từ đầu:
>
> `ssh-keygen -R "<IP_ADDRESS>"`
