# Xem chi tiết report

1. Vào **Home > Apps > Reporting**.

   [![Màn hình danh sách report](/img/migrated/Picture11-f73c4b93.png)](/img/migrated/Picture11-f73c4b93.png)

   Danh sách report hiển thị dưới dạng bảng, bao gồm các thông tin:

   | Cột | Mô tả |
   |-----|-------|
   | Name | Tên report đã tạo |
   | Report status | **Active** hoặc **Draft** |
   | Dashboard name | Tên dashboard đã chọn |
   | Recipient | Email của người nhận |
   | Created date | Thời điểm tạo hoặc chỉnh sửa report lần cuối |
   | Last sent | Ngày gửi email report cuối cùng |
   | Schedule | Lịch gửi email report |
   | Schedule status | **Active** hoặc **Inactive**. Không hiển thị với report ở trạng thái **Draft** |

2. Click vào tên report để xem chi tiết.

   [![Màn hình chi tiết report](/img/migrated/Picture12-94a2f87d.png)](/img/migrated/Picture12-94a2f87d.png)

   Thông tin chi tiết report bao gồm:

   - **Tên report**
   - **Status report**:
     - **Active**: Report được tạo thành công.
     - **Draft**: Người dùng chọn **Save as Draft**.
   - **Initialization time**: Thời gian tạo hoặc chỉnh sửa report lần cuối.
   - **Status Schedule**:
     - **Scheduled**: Đã đặt lịch gửi report.
     - **Expired**: Report đã quá hạn gửi.
     - **Draft**: Report đang ở trạng thái Draft.
     - **Paused**: Schedule vẫn trong lịch trình nhưng người dùng đã tắt trạng thái (tạm dừng gửi báo cáo).
