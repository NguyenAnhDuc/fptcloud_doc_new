# Truy cập Grafana

Sau khi workspace được khởi tạo thành công, bạn hãy:

1. Truy cập trang console portal <https://console.fptcloud.com>
2. Chọn Workspace cần sử dụng để xem thông tin chi tiết
3. Tại màn hình Configuration, hệ thống cung cấp Dashboard URL của Grafana
4. Click vào Dashboard URL để truy cập Grafana

Hệ thống mặc định tự động đăng nhập bằng FPT ID. Trong trường hợp đăng nhập không thành công, vui lòng liên hệ đội ngũ hỗ trợ.
