# Secret Engine Management

Secret EngineはVault内でSecretを保存・管理するコンポーネントです。各Secret Engineは固有のマウントポイントを持ち、Key/Value構造で複数のSecretを格納できます。このセクションでSecret Engineの作成と削除を行います。

- [Secret Engineを作成する](./create-secret-engine.md)
- [Secret Engineを削除する](./delete-secret-engine.md)
