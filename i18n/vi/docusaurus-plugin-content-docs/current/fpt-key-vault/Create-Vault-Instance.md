# Create Vault Instance

Vault Instanceは、すべてのSecret Engine、Secret、セキュリティ設定を格納するFPT Key Vaultの基本単位です。Secret管理操作を行う前にVault Instanceを作成してください。

1. FPT Portalメニューで **Security** > **Key Vault** > **Create** を選択します。

   [![Createボタンが表示されたKey Vault一覧画面](/img/migrated/k1-ff2abcff.png)](/img/migrated/k1-ff2abcff.png)

2. **Create Key Vault** ポップアップで必要な情報を入力します。

   [![設定フィールドが表示されたCreate Key Vaultフォーム](/img/migrated/k2-1e9263de.png)](/img/migrated/k2-1e9263de.png)

3. **Create** をクリックして続行します。システムがリソース要件を確認し、確認メッセージを送信後、入力した情報に基づいて新しい **Vault Instance** を作成します。

   [![新しく作成されたインスタンスが表示されたKey Vault一覧](/img/migrated/k3-612b2fb5.png)](/img/migrated/k3-612b2fb5.png)
