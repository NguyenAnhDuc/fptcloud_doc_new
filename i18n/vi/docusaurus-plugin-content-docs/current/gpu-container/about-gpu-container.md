# Giới thiệu GPU Container

## GPU Container là gì?
GPU Container là nền tảng điện toán được quản lý cho phép bạn chạy container trực tiếp trên hạ tầng FPT AI Factory. Chỉ với vài thao tác đơn giản, bạn có thể triển khai và quản lý các khối lượng công việc AI một cách dễ dàng — bằng cách sử dụng image container của chính bạn hoặc tận dụng các image có sẵn do FPT cung cấp.

## Hoạt động như thế nào?
GPU Container chạy ứng dụng trong môi trường cô lập với quyền truy cập trực tiếp vào tài nguyên GPU. Nó sử dụng các công cụ như NVIDIA Container Toolkit để tận dụng khả năng tăng tốc GPU cho các tác vụ AI, ML hoặc xử lý dữ liệu — mà không cần cài đặt phức tạp.

## Tại sao nên dùng GPU Container?
1. **Triển khai AI bằng container:** Chạy khối lượng công việc AI trong container.
2. **Điện toán GPU hiệu năng cao:** Truy cập GPU NVIDIA cho các khối lượng công việc AI/ML.
3. **Triển khai nhanh:** Khởi động container với GPU mạnh mẽ chỉ bằng 1 lần click thông qua các template có sẵn.
4. **Lưu trữ bền vững:** Dữ liệu được lưu trữ bền vững và vẫn khả dụng ngay cả khi container bị dừng.
