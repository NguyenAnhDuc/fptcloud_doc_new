# Xuất cấu hình

Để tái sử dụng sau này, tính năng Xuất cấu hình cho phép bạn lưu cấu hình container và tải xuống dưới dạng file YAML.

1. Từ màn hình danh sách container, nhấp menu Action (dấu 3 chấm) và chọn **Export Configuration**. [![Alt text](/img/migrated/export-2-2-430ec6f3.png)](/img/migrated/export-2-2-430ec6f3.png)
2. Hoặc, mở trang Container Details và nhấp **Export Configuration**. [![Alt text](/img/migrated/export-1-2-adff5d09.png)](/img/migrated/export-1-2-adff5d09.png)

File YAML sẽ được tải xuống tự động với định dạng tên: **container-name.yaml**
