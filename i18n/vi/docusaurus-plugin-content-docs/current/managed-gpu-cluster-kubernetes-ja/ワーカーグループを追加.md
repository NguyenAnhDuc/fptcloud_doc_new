# Thêm Worker Group

Trong Managed GPU Cluster, bạn có thể thêm worker group vào cluster theo nhu cầu. Để thêm worker, thực hiện các bước sau trên portal.

**Bước 1:** Trong menu, chọn **[AI Infrastructure] > [Managed GPU Cluster]** để hiển thị trang **"Quản lý Managed GPU"**. Chọn cluster bạn muốn thêm worker group.

[![](/img/migrated/17-502dd1f3.png)](/img/migrated/17-502dd1f3.png)

**Bước 2:** Chọn **[Node Pool] > [Chỉnh sửa Worker]**.

[![](/img/migrated/18-13f798d7.png)](/img/migrated/18-13f798d7.png)

**Bước 3:** Chọn **"Thêm Worker Group"**.

[![](/img/migrated/19-437d9b85.png)](/img/migrated/19-437d9b85.png)

**Bước 4:** Điền vào các trường bắt buộc rồi nhấn **[Lưu]**.

[![](/img/migrated/20-d9c3a1fa.png)](/img/migrated/20-d9c3a1fa.png)

- **Tên nhóm:** Đặt tên cho worker group để phân biệt
- **Container Runtime:** Chọn container runtime. Hiện tại hệ thống mới chỉ hỗ trợ Containerd container runtime
- **Flavor:** Tài nguyên flavor của worker GPU
- **Số lượng server:** Số MetalCloud server được tạo để chạy worker của cluster
- **Label:** Label của worker group
- **Taint:** Taint cho worker group

**Lưu ý:** Quá trình thêm cluster mới sẽ mất vài phút và trạng thái cluster sẽ chuyển sang **Đang xử lý**. Khi thêm worker group mới, cluster vẫn hoạt động bình thường.
