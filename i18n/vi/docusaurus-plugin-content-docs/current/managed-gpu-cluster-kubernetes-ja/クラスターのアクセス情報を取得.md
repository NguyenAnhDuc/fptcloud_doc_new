# Lấy thông tin truy cập cluster

File Kube-config được dùng để lưu thông tin kết nối tới Kubernetes cluster, giúp các công cụ như kubectl, kubelet, kubeadm xác định cách giao tiếp với Kubernetes API server. Vì file kubeconfig đóng vai trò quan trọng trong việc quản lý quyền truy cập vào Kubernetes, bạn cần bảo vệ file này cẩn thận.

Để tải file Kube-config, thực hiện theo các bước sau:

**Bước 1:** Trong menu, chọn **[AI Infrastructure] > [Managed GPU Cluster]**. Hệ thống sẽ hiển thị trang **[Quản lý GPU Cluster]**. Chọn cluster bạn muốn lấy thông tin truy cập.

[![](/img/migrated/12-7952cac6.png)](/img/migrated/12-7952cac6.png)

**Bước 2:** Vào Basic Properties > Cluster Information > Configuration, tải file kubeconfig để đăng nhập vào cluster bằng kubectl hoặc các công cụ khác.

[![](/img/migrated/13-b5e2d2ac.png)](/img/migrated/13-b5e2d2ac.png)

**Lưu ý:** Managed GPU Cluster sử dụng Kubernetes cluster core gốc, bạn sử dụng cluster với các công cụ kubectl và dashboard giống như với Kubernetes cluster thông thường.
