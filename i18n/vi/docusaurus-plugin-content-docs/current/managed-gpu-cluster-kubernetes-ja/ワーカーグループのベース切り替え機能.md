# Tính năng chuyển đổi base của Worker Group

Tính năng chuyển đổi base cho phép bạn thay đổi worker group được chỉ định là base trong Managed GPU Cluster. Worker group base là nhóm worker mặc định không thể xóa và đóng vai trò nền tảng cho cluster.
