# Thiết lập ban đầu

Nếu bạn sử dụng MANAGED GPU CLUSTER lần đầu tiên, hãy xác nhận và hoàn thành các bước sau.

### 1. Tạo tài khoản FPT Cloud và đăng nhập vào FPT Portal

Để đăng ký tài khoản FPT Cloud, hãy truy cập trang chủ tại đây. Tiếp theo, chọn Sign Up và nhập thông tin theo hướng dẫn của hệ thống. Đội ngũ hỗ trợ sẽ liên hệ với bạn ngay sau đó để xác nhận thông tin và tạo tài khoản.

Để đăng nhập FPT Portal, hãy truy cập [console.fptcloud.com](https://console.fptcloud.com/login/select-method). Sau khi đăng nhập bằng tài khoản và mật khẩu được cấp, hãy chọn đúng tenant, region và VPC.

Nếu bạn có thắc mắc về thông tin trên hoặc gặp lỗi hệ thống sau 3 lần thử, hãy liên hệ ngay với đội ngũ hỗ trợ để được giúp đỡ.

**Lưu ý:** Để sử dụng sản phẩm AI Factory, tài khoản của bạn cần bật xác thực 2 bước (MFA).

### 2. Tạo subnet cho Bare Metal GPU server dùng làm worker node trong Managed GPU Cluster

Để tạo Managed GPU Cluster, trước tiên bạn cần có subnet trên Bare Metal GPU server – server này sẽ đóng vai trò worker node trong K8s cluster. Địa chỉ IPv4 của worker Bare Metal GPU sẽ được cấp từ subnet này.

**Bước 1:** Vào [AI Infrastructure] > chọn [Subnet] > chọn [Tạo Subnet]

![](/img/migrated/1-1-4f1767cc.png)

**Bước 2:** Nhập tên subnet

![](/img/migrated/2-1-e7e458f5.png)

**Bước 3:** Nhập tên Network ACL cho subnet

**Bước 4:** Nhấn [Tạo Subnet] để hoàn tất việc tạo subnet cho Bare Metal GPU

**Lưu ý:** Network ACL được tạo mặc định cho subnet sẽ chặn tất cả traffic vào và cho phép tất cả traffic ra. Để sử dụng load balancer của Managed GPU Cluster, bạn cần mở đúng rule trên subnet load balancer để cho phép kết nối.

### 3. Tạo subnet cho Load Balancer

Managed GPU Cluster chỉ hoạt động với subnet có tùy chọn static pool được bật, vì vậy bạn cần tạo subnet có static pool theo các bước sau.

**Bước 1:** Trong mục [Network], chọn [Subnet].

![Subnet K8s](/img/migrated/Screenshot-2025-01-10-at-10.35.34-7d3142bd.png)

**Bước 2:** Trên trang [Quản lý Subnet], chọn [Tạo Subnet].

![Subnet K8s](/img/migrated/Screenshot-2025-01-10-at-10.39.58-b09ab0a1.png)

**Bước 3:** Nhập các thông tin sau:
- **Tên:** Nhập tên gợi nhớ cho subnet
- **CIDR:** Nhập CIDR hợp lệ
- Chọn tùy chọn **[Cài đặt nâng cao]**
- **Static IP Pool:** Nhập một dải IP hợp lệ lấy từ CIDR

Chọn **[Lưu]** để tạo subnet mới. Hệ thống sẽ xử lý và thông báo kết quả.

![Userguide K8s 3](/img/migrated/Initial-setup-3-9bccff9a.png)

### 4. Kích hoạt dịch vụ Managed GPU Cluster và yêu cầu cấp quota tài nguyên

Nếu bạn sử dụng FPT Cloud lần đầu, một số dịch vụ có thể chưa được mở trên tài khoản. Hãy liên hệ với đội ngũ hỗ trợ để cung cấp thông tin về dịch vụ và cấu hình cần thiết. Chúng tôi sẽ cung cấp các tài nguyên cần thiết như RAM, CPU, lưu trữ, Public IP để bạn có thể bắt đầu sử dụng dịch vụ Managed GPU Cluster.
