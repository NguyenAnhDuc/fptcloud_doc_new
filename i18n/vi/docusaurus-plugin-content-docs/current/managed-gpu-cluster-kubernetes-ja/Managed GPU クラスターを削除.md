# Xóa Managed GPU Cluster

Đối với các Managed GPU Cluster không còn cần thiết, bạn có thể xóa theo các bước sau.

**Bước 1:** Trong menu, chọn **[AI Infrastructure] > [Managed GPU Cluster]** để hiển thị trang **[Quản lý GPU Cluster]**. Chọn cluster bạn muốn truy cập.

[![](/img/migrated/14-b4dc395d.png)](/img/migrated/14-b4dc395d.png)

**Bước 2:** Trong cột Action của cluster muốn xóa, chọn **[Xóa]**.

[![](/img/migrated/15-05e09c5c.png)](/img/migrated/15-05e09c5c.png)

**Bước 3:** Xem xét cảnh báo trong popup rồi nhấn **[Xóa]** để xác nhận.

[![](/img/migrated/16-52efdb8d.png)](/img/migrated/16-52efdb8d.png)
