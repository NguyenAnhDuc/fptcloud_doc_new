# Tính năng Manual Scale

Manual Scale cho phép bạn scale tài nguyên hệ thống theo nhu cầu. Bạn có thể tăng hoặc giảm số lượng MetalCloud server theo ngày trên portal theo các bước sau.

**Bước 1:** Trong menu, chọn **[AI Infrastructure] > [Managed GPU Cluster]** để hiển thị trang **"Quản lý Managed GPU"**. Chọn cluster bạn muốn thêm worker group.

[![](/img/migrated/17-502dd1f3.png)](/img/migrated/17-502dd1f3.png)

**Bước 2:** Nhấn vào cluster bạn muốn scale, chọn **[Node Pool] > [Chỉnh sửa Worker]**.

[![](/img/migrated/21-de403ab8.png)](/img/migrated/21-de403ab8.png)

[![](/img/migrated/22-ba5ffbfa.png)](/img/migrated/22-ba5ffbfa.png)

**Bước 3:** Tăng **Số lượng server** theo nhu cầu rồi nhấn nút **[Lưu]**.

[![](/img/migrated/23-82548293.png)](/img/migrated/23-82548293.png)

**Lưu ý:** Quá trình scale server sẽ mất vài phút và trạng thái cluster sẽ chuyển sang **Đang xử lý** cho đến khi worker mới tham gia thành công vào cluster. Cluster vẫn hoạt động bình thường khi scale server mới.
