# Tải file đối soát chi tiết

Để tải đối soát chi tiết dạng file Excel, thực hiện như sau:

1. Sau khi chọn kỳ hóa đơn, contract và đối soát, nhấn nút **Download**.

   [![Nút Download đối soát](/img/migrated/image-1744088593606-63e60ab6.png)](/img/migrated/image-1744088593606-63e60ab6.png)

2. File đối soát sẽ được gửi về email của bạn (email của user đang đăng nhập vào hệ thống) trong vòng 15–20 phút.

   [![Email xác nhận tải file đối soát](/img/migrated/image-1744088600025-5d3582dc.png)](/img/migrated/image-1744088600025-5d3582dc.png)

:::warning
Hệ thống chỉ nhận hành động Download với tần suất 15 phút/lần. Vui lòng không thực hiện hành động download nhiều lần trong vòng 15 phút.
:::
