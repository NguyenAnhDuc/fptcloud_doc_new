# Lọc Cost Explorer theo resource cụ thể

Customize Filter của Cost Explorer cho phép bạn lọc theo tag. Sử dụng tính năng Tagging để đánh tag các tài nguyên cụ thể mà bạn muốn xem chi phí, sau đó lọc Cost Explorer theo tag này.
