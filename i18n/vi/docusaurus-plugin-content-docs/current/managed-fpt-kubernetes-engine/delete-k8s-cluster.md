# Xóa Kubernetes cluster

**Bước 1:** Ở menu FPT Cloud Portal, chọn **Kubernetes** để mở trang **Kubernetes Management**.

[![](/img/migrated/12-20fee2a3.png)](/img/migrated/12-20fee2a3.png)

**Bước 2:** Chọn **Action** của cluster muốn xóa, sau đó chọn **Delete**.

[![](/img/migrated/13-4f4d9034.png)](/img/migrated/13-4f4d9034.png)

**Bước 3:** Chọn **Delete** trong popup xác nhận để hoàn tất việc xóa.

[![](/img/migrated/14-8e0da3fa.png)](/img/migrated/14-8e0da3fa.png)
