# Quản lý queues

Queue là hàng đợi lưu trữ message cho đến khi consumer nhận. FPT Message Bus for RabbitMQ cho phép bạn tạo, xem, gửi/nhận message và xóa queue trực tiếp từ giao diện quản lý.
