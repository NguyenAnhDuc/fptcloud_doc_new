# VPN Site-to-Site

VPN Site-to-Site giúp bạn thiết lập kết nối bảo mật giữa mạng on-premise và tài nguyên trên FPT Cloud thông qua tunnel IPSec — cho phép nhân viên truy cập ứng dụng và dịch vụ cloud mà không tiết lộ dữ liệu mật.
