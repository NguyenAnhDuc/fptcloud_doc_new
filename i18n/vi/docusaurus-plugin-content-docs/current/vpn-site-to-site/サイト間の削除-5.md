# Xóa VPN Site-to-Site

:::danger
Xóa kết nối VPN Site-to-Site không thể hoàn tác. Kết nối sẽ bị ngắt vĩnh viễn.
:::

1. Chọn kết nối VPN, sau đó nhấn **Actions** → **Delete**.

   ![Menu xóa VPN](/img/migrated/image-1742916525951-13239d02.png)

2. Xác nhận thao tác xóa. Trạng thái VPN chuyển sang **Deleting** và kết nối bị xóa khỏi danh sách.

   ![Hộp thoại xác nhận xóa VPN](/img/migrated/image-1742916584099-b13362ab.png)
