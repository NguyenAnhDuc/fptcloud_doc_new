# Chỉnh sửa thông tin VPN

Nếu cần thay đổi cấu hình VPN sau khi tạo, bạn có thể chỉnh sửa thông tin kết nối.

1. Chọn kết nối VPN, sau đó nhấn **Actions** → **Edit VPN Information**.

   ![Menu Edit VPN Information](/img/migrated/image-1742915365993-6486c2a4.png)

2. Cập nhật thông tin VPN. Các giá trị ban đầu được điền sẵn theo cấu hình hiện tại.

   ![Form chỉnh sửa VPN](/img/migrated/image-1742915433625-2d068d8c.png)

3. Nhấn **Update** để xác nhận. Trạng thái VPN chuyển từ **Updating** sang **Active**.

   ![Cập nhật VPN thành công](/img/migrated/image-1742915544560-614508af.png)
