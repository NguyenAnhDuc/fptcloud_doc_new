# Manage resources

1. Select **Database Engine** > select a **Cluster ID** > on **Database Detail**, select the **Resource** tab.

2. Choose how to change resources.

   - **Using a preset configuration:**
     - Select **Choose Preset**.
     - Select a preset from the list.

   - **Using a custom configuration:**
     - Select **Customized**.
     - Enter the CPU, RAM, and Disk values.

3. Click **Apply** to apply the changes.

:::note
Resources can only be changed within your remaining quota. If the quota is exceeded, the system will display an error and the resource change cannot be applied.
:::
