# Import/tạo SSL Certificate (Classic)

**Bước 1:** Truy cập trang **Manage SSL Certificate for Load balancer** và nhấn **Import SSL Certificate**.

[![](/img/migrated/image-1712807252615-cef0c042.png)](/img/migrated/image-1712807252615-cef0c042.png)

**Bước 2:** Nhập thông tin chi tiết SSL Certificate vào popup **Import SSL Certificate** và nhấn **Save**.

[![](/img/migrated/image-1712808061542-b08fa903.png)](/img/migrated/image-1712808061542-b08fa903.png)
