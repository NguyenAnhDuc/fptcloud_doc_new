# Hướng dẫn download file config VPN

Để tải cấu hình VPN, người dùng chọn **Download config** ở mục **Action** của VPN cần tải. [![file](/img/migrated/image-1744269925166-3989e131.png)](/img/migrated/image-1744269925166-3989e131.png) Tại file đã tải về dưới dạng .txt, người dùng có thể xem được các thông số cấu hình của VPN.
