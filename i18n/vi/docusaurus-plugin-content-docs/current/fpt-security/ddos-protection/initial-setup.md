# Thiết lập ban đầu

Trước khi sử dụng DDoS Protection, hãy hoàn thành các bước sau.

## Tạo tài khoản FPT Cloud và đăng nhập

1. Truy cập [https://fptcloud.com/](https://fptcloud.com/) và chọn **Sign Up**.
2. Nhập thông tin theo hướng dẫn để tạo tài khoản. Bộ phận hỗ trợ sẽ liên hệ xác nhận.
3. Đăng nhập tại [https://console.fptcloud.com](https://console.fptcloud.com) bằng tài khoản và mật khẩu FPT Cloud.
4. Chọn đúng **Tenant**, **Region** và **VPC** cần sử dụng dịch vụ.

:::tip
Nếu gặp bất kỳ lỗi nào trong quá trình thao tác, hãy liên hệ đội ngũ hỗ trợ qua hotline **1900 638 399** hoặc email **support@fptcloud.com**.
:::
