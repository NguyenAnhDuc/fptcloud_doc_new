# Manage proxy host

A proxy host is the core component of WAF that forwards end-user requests through the WAF server to the web server while applying security rule sets.

Proxy host management operations include:

- [Create a proxy host](/vi/docs/fpt-security/fpt-cloud-waf/tutorials/manage-proxy-host/create-proxy-host)
- [View proxy host details](/vi/docs/fpt-security/fpt-cloud-waf/tutorials/manage-proxy-host/view-proxy-host-detail)
- [Update a proxy host](/vi/docs/fpt-security/fpt-cloud-waf/tutorials/manage-proxy-host/update-proxy-host)
- [Enable or disable a proxy host](/vi/docs/fpt-security/fpt-cloud-waf/tutorials/manage-proxy-host/enable-disable-proxy-host)
- [Delete a proxy host](/vi/docs/fpt-security/fpt-cloud-waf/tutorials/manage-proxy-host/delete-proxy-host)
