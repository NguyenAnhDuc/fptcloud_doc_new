# Thiết lập ban đầu

Nếu đây là lần đầu tiên bạn sử dụng **FPT Cloud WAF**, hãy hoàn thành các bước sau trước khi bắt đầu.

## Tạo tài khoản và đăng nhập FPT Cloud Portal

Để sử dụng dịch vụ **FPT Cloud WAF**, bạn cần đăng nhập vào **FPT Portal**.

1. Truy cập FPT Portal và đăng nhập bằng tài khoản/mật khẩu FPT Cloud đã được cấp.
2. Chọn đúng Tenant, Region và VPC cần sử dụng dịch vụ.

:::note
Nếu chưa có tài khoản **FPT Cloud**, truy cập trang chủ và chọn **Sign Up**, sau đó nhập thông tin theo hướng dẫn. Bộ phận hỗ trợ sẽ liên hệ để xác nhận và tạo tài khoản cho bạn.
:::

Nếu gặp bất kỳ lỗi nào trong quá trình thao tác, hãy liên hệ đội ngũ Support để được hỗ trợ.

Sau khi hoàn thành các bước trên, bạn đã sẵn sàng sử dụng WAF trên hệ thống FPT Cloud.
