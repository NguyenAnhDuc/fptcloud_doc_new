# Chỉnh sửa role và permission

Bạn có thể thay đổi thông tin và permission của các Custom Role theo hướng dẫn sau.

1. Chọn **IAM** > **Roles**. Hệ thống hiển thị trang **Role Management**.

[![](/img/migrated/image-1734344271940-4a804c14.png)](/img/migrated/image-1734344271940-4a804c14.png)

2. Chọn **Action** ở cuối Custom Role cần cập nhật trên danh sách.
3. Chọn **Edit Basic Information** hoặc **Edit Permissions**.

   - **Edit Basic Information**: Chỉnh sửa tên và mô tả role.

[![](/img/migrated/image-1734344299974-74ad3c1a.png)](/img/migrated/image-1734344299974-74ad3c1a.png)

   - **Edit Permissions**: Chỉnh sửa danh sách quyền của role.

[![](/img/migrated/image-1734344312048-ce9f5aea.png)](/img/migrated/image-1734344312048-ce9f5aea.png)

:::warning
Khi role thay đổi sẽ ảnh hưởng đến quyền truy cập của tất cả user trong các User Group có gán role đó.
:::
