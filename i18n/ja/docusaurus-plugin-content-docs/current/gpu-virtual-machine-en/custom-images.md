# Custom Images

You can create **GPU VMs** based on **custom images** , which lets you migrate and scale your workloads without spending time recreating your environment from scratch.
[![Alt text](https://fptcloud.com/wp-content/uploads/2025/03/Custom-Images-1.png)](https://fptcloud.com/wp-content/uploads/2025/03/Custom-Images-1.png)

### Upload a Custom Image
Only the **QCOW** file format is supported for uploading GPU VMs here. For other file formats, please **contact us** for further assistance.
**Step 1:** From the menu, select **Compute Engine > Custom Image**. Then click **Upload Image**. 
**Step 2:** Enter the required information and choose the file from your machine. Click **Upload** to begin the upload process.

### Delete a Custom Image
If you no longer need a Custom Image, you can delete it by following these steps:
**Step 1:** From the menu, select **Custom Image** , then under the **Actions** menu for the image you wish to delete, click **Delete**. 
**Step 2:** A confirmation pop-up will appear. To proceed with the deletion, click **Delete Custom Image**.
