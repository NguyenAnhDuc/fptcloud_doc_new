# Kubernetes (K8S Cluster) で File Storage – High Performance を使用する

Kubernetes クラスターで File Storage – High Performance を利用するには、サポートチームにお問い合わせいただくか、最新の設定ガイドをご参照ください。

利用を開始する前に、[初期セットアップ](/ja/docs/fpt-storage/file-storage-high-performance/initial-setup)を完了し、必要なクォータとサブネット情報をサポートチームにご提供ください。
