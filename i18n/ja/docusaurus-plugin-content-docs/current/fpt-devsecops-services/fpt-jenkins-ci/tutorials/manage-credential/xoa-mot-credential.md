# クレデンシャルの削除

不要になったクレデンシャルを削除するには：

**ステップ 1:** FPT Portal メニュー > **Jenkins CI** > **Detail Cluster** > **Credential Managements** に移動します。

[![](/img/migrated/Picture62-44b50e96.png)](/img/migrated/Picture62-44b50e96.png)

**ステップ 2:** **Credential Managements** 画面で、削除するクレデンシャルを選択 > **Actions** > **Delete**。

[![](/img/migrated/Picture63-8d44ed2a.png)](/img/migrated/Picture63-8d44ed2a.png)

**ステップ 3:** **delete** と入力し、**Confirm** をクリックして削除を確認します。

[![](/img/migrated/Picture64-3c5f5142.png)](/img/migrated/Picture64-3c5f5142.png)

**ステップ 4:** Portal でのクレデンシャル削除後の結果。
