# クレデンシャルの新規追加

各クレデンシャルの種類ごとに、以下の手順で追加できます。

**ステップ 1:** FPT Portal メニュー > **Jenkins CI** > **Detail Cluster** > **Credential Managements** に移動します。

[![](/img/migrated/Picture28-f1ca981e.png)](/img/migrated/Picture28-f1ca981e.png)

### Secret Text

**ステップ 2:** **Credential Managements** 画面 > **Add Credential** で、**Secret Text** 種別のクレデンシャルを追加します。

[![](/img/migrated/Picture29-4e919329.png)](/img/migrated/Picture29-4e919329.png)

**ステップ 3:** Secret Text クレデンシャルの必要情報を入力します。

[![](/img/migrated/Picture30-6ec104f7.png)](/img/migrated/Picture30-6ec104f7.png)

**ステップ 4:** **Add** をクリックしてクレデンシャルを作成します。Portal での結果：

[![](/img/migrated/Picture31-15489933.png)](/img/migrated/Picture31-15489933.png)

Jenkins での結果：

[![](/img/migrated/Picture32-690a204c.png)](/img/migrated/Picture32-690a204c.png)

### Username with password

**ステップ 2:** **Credential Managements** 画面 > **Add Credential** で、**Username with password** 種別のクレデンシャルを追加します。

[![](/img/migrated/Picture34-0260b9d4.png)](/img/migrated/Picture34-0260b9d4.png)

**ステップ 3:** Username with password クレデンシャルの必要情報を入力します。

[![](/img/migrated/Picture35-52ee1479.png)](/img/migrated/Picture35-52ee1479.png)

**ステップ 4:** **Add** をクリックしてクレデンシャルを作成します。Portal での結果：
