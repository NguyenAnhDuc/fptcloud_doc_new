# Telegram Bot の設定

FPT Cloud では、Portal インターフェースを通じてユーザーへの通知送信のために Jenkins に Telegram Bot を設定できます。

### Telegram Bot 設定の編集

**ステップ 1:** FPT Portal メニュー > **Jenkins CI** > **Detail Cluster** > **Configurations** > **Telegram Bot** > **Edit** を選択します。
