# K8s cluster構成の変更

D-FKEでは、既存のKubernetes clusterに対して以下の構成変更を実行できます。

- worker nodeのスケール
- nodeのフレーバー（構成）の変更
- NFS Serverのフレーバー（構成）の変更
- nodeのストレージサイズの拡張
- NFS Serverのストレージサイズの拡張
- Kubernetesバージョンのアップグレード
- NFS Persistent Storageの追加
