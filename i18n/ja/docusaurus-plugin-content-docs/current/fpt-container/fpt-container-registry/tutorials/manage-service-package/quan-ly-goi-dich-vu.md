# Quan Ly Goi Dich Vu

FPT Container Registry cung cấp nhiều gói dịch vụ với mức storage và pull-limit khác nhau. Phần này hướng dẫn cách xem thông tin quota hiện tại và nâng cấp gói khi cần.
