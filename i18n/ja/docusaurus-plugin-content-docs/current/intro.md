# FPT Cloud ドキュメント

FPT Cloud の公式ドキュメントへようこそ。本ドキュメントでは、FPT Cloud が提供するすべてのサービスの概要、設定手順、ベストプラクティスをご確認いただけます。

## FPT Cloud とは

FPT Cloud は、FPT Corporation が提供するエンタープライズ向けクラウドプラットフォームです。仮想マシン、ストレージ、ネットワーク、AI/ML サービスなど、幅広いクラウドサービスをベトナムおよびアジア太平洋地域のデータセンターから提供します。高い可用性とセキュリティを備えたインフラを通じて、お客様のデジタルトランスフォーメーションを支援します。

## 主なサービスカテゴリ

- **コンピューティング**: 仮想マシン（vServer）、コンテナサービス、Auto Scaling
- **ストレージ**: オブジェクトストレージ、ブロックストレージ、ファイルストレージ
- **ネットワーク**: 仮想ネットワーク（vNetwork）、ロードバランサー、CDN
- **セキュリティ**: WAF（WAPPLES）、DDoS 保護、SSL/TLS 管理
- **AI Factory**: モデルトレーニング、Model Serving、Data Hub、Model Testing
- **DevOps**: Jenkins CI、Kubernetes サービス

## ドキュメントの使い方

左側のサイドバーからサービスカテゴリを選択してください。各サービスのページには、概要・設定手順・よくある質問が掲載されています。お探しの情報が見つからない場合は、ページ上部の検索バーをご利用ください。
