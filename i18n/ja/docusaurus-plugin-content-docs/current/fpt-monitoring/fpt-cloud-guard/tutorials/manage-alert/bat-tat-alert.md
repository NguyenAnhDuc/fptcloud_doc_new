# アラートを有効化・無効化する

有効化/無効化機能を使うと、アラートを削除せずに一時的に停止できます。特定のアラートからの通知を一定期間停止したい場合に便利です。

**Alert Management** テーブルで有効化または無効化したいアラートを選択します。**スイッチ**ボタン [![Userguide FPT Alert 2022 5](/img/migrated/Userguide-FPT-Alert-2022-5-b7cf3f24.png)](/img/migrated/Userguide-FPT-Alert-2022-5-b7cf3f24.png) をクリックして状態を切り替えます。

[![file](/img/migrated/image-1725532697828-cdc78b79.png)](/img/migrated/image-1725532697828-cdc78b79.png)

アラートの状態は Alert Management テーブルの **Status** フィールドに表示されます:

- **有効:** システムが継続的に評価し、設定したしきい値に達した際に通知を送信します。
- **無効:** システムが評価を停止し、自動通知は送信されません。
