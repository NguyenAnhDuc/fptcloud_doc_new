# テスト送信メール

**Test send email** 機能を使用すると、レポートの内容を確認し、レポートが受信者に正常に配信されるかどうかを検証できます。

この機能を使用するには：

1. **Home > Apps > Reporting > Create a new report** に移動します。
2. レポートの詳細をすべて入力します：**Select Dashboard**、**Format Report**、**Schedule**、**Confirm**。
3. **Confirm** 画面で **Send email test** をクリックします。

   [![確認画面のテスト送信メールボタン](/img/migrated/Picture10-0a9f7fe5.png)](/img/migrated/Picture10-0a9f7fe5.png)

4. ダイアログボックスが表示されます。テスト用のメールアドレスを入力し、**Send** をクリックします。
5. 受信者のメールボックスでレポートが届いているか確認します。
