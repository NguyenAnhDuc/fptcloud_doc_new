# Grafana での metric data query の自作方法ガイド

このセクションは、独自に query を作成したいユーザー向けです。

## 手順

**ステップ 1:** Grafana のメニューで **Explore** を選択します。

[![Alt text](/img/migrated/Screenshot_22-f2b7b75d.png)](/img/migrated/Screenshot_22-f2b7b75d.png)

**ステップ 2:** data source を選択します。例: loki

[![Alt text](/img/migrated/Screenshot_31-0fcd00d8.png)](/img/migrated/Screenshot_31-0fcd00d8.png)

**ステップ 3:** コードモードを選択 >> query を入力します。

**ステップ 4:** 画面右上の **Run query** ボタンをクリックします。

画面に以下のような結果が表示されます。

[![Alt text](/img/migrated/Screenshot_35-4151422d.png)](/img/migrated/Screenshot_35-4151422d.png)
