# Grafana へのアクセス

ワークスペースの初期化が完了したら、以下の手順を実行します。

1. コンソールポータル <https://console.fptcloud.com> にアクセスします。
2. 使用するワークスペースを選択して詳細情報を確認します。
3. Configuration 画面で、システムが Grafana の Dashboard URL を提供しています。
4. Dashboard URL をクリックして Grafana にアクセスします。

システムはデフォルトで FPT ID を使用して自動的にログインします。ログインに失敗した場合は、サポートチームにご連絡ください。
