# データの可視化

FPT Cloud Monitoring は Grafana ダッシュボードを通じてデータを可視化します。ダッシュボードへのアクセス、テンプレートのインストール、メトリクス・ログ・トレースの視覚的な探索が可能です。

詳細については以下のセクションを参照してください:

- [ダッシュボードにアクセスする](/ja/docs/fpt-monitoring/fpt-cloud-monitoring/detailed-guide/7-data-visualization/truy-cap-vao-dashboard)
- [ダッシュボードテンプレートをインストールする](/ja/docs/fpt-monitoring/fpt-cloud-monitoring/detailed-guide/7-data-visualization/fcmon-dashboard-template)
