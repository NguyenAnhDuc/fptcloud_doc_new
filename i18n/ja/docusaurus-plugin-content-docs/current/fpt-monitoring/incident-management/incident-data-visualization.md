# インシデントデータの可視化

**Incident Insights** を使用してインシデントデータを可視化し、インシデント対応パフォーマンスの指標やトレンドを探索できます。

## Incident Insightsの表示

1. **FMON Grafana** にログインします。
2. 左メニューで **Incident** を見つけ、**Insights** をクリックします。

可視化グラフで主要指標が表示されます：

[![Incident Insightsの主要指標グラフ](/img/migrated/Picture34-56839c68.png)](/img/migrated/Picture34-56839c68.png)

- **Total Incident**: インシデントの総数
- **Incident by severity**: 深刻度別インシデント統計
- **Incident by status**: ステータス別インシデント統計
- **Incident by label**: ラベル別インシデント統計
