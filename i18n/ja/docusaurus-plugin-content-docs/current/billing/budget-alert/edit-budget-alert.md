# 予算アラートの編集

予算、アラートしきい値、またはリソースグループを変更する必要がある場合、作成済みの予算アラートを編集できます。更新後、システムは新しい構成に基づいて実績コストを再計算し、新しいしきい値に基づいてアラートを送信します。

1. **Billing** → **Budget Alert** を選択します。作成済み予算アラート一覧が表示されます。編集したい予算アラートを選択し、**Actions** → **Update** を選択します。

   ![Update予算アラートを選択](/img/migrated/image-1722238825640-0606b59c.png)

2. 予算アラート作成時と同様に情報を編集します。

3. **Update** をクリックして変更を保存します。

   ![予算アラート更新の確認](/img/migrated/image-1722238882870-6e80e188.png)

保存後、システムは新しい構成に基づいてActual costを計算し、新しいしきい値に基づいてアラートを送信します。
