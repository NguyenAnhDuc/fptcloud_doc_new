# ユーザープロファイルの権限割り当て

JupyterHub にログイン後、**Admin** ロールを持つユーザーは **Service** > **Profile** メニューを選択し、**Profile Assignments** タブをクリックします。

システムには作成済みのプロファイルに対応するカードが表示されます。権限を設定するプロファイルをクリックして詳細を展開します。

 * **Add User**: システム内のユーザーリストから選択します。

 * **Add Group**: システム内のグループリストから選択します。

選択後、**Save Changes** をクリックしてプロファイルへのユーザー/グループの権限を適用します。
