# JupyterHub

**JupyterHub** は、データサイエンティスト、データエンジニア、ソフトウェア開発者がデータ分析、データ処理、機械学習モデル開発のために計算リソースにアクセスできるマルチユーザー対応の Jupyter Notebook 環境を提供するために設計されたオープンソースプラットフォームです。Cloud Data Platform に統合することで、JupyterHub はコアコンポーネントとなり、クラウドサービス全体にわたるリソースの管理、スケーリング、最適化を可能にし、大規模なデータの保存および処理ワークフローを支援します。
