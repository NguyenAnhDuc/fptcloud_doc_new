# Tag Management

**1, タグの作成**

定義：タグとは、業務フローまたはビジネスラインの識別名です。

例：FRT では Long Chau、Vaccine などのシステムラインがあり、それぞれのラインがタグと呼ばれます（タグ Long Chau、タグ Vaccine など）。

タグを作成するには、以下の手順に従ってください。

**ステップ 1**: **Relation management** サービスに正常にアクセスします。

**ステップ 2**: サービスメニューで **Tag management** >> **Button Create tag** をクリックします。

[![Alt text](/img/migrated/Picture6-f6221c1b.png)](/img/migrated/Picture6-f6221c1b.png)

**ステップ 3**: タグ作成フォームが表示されます。以下の情報を入力してください。
  * Name（必須）：タグ名
  * Description（任意）：タグの説明

[![Alt text](/img/migrated/Picture7-1fead4bd.png)](/img/migrated/Picture7-1fead4bd.png)

**注意**：タグ名は最大100文字まで入力できます。

**ステップ 4**: 全ての情報を入力したら、**Create** をクリックしてタグを作成します。

**2, タグの更新**

タグを編集するには、以下の手順に従ってください。

**ステップ 1**: メニューで **Tag management** を選択 >> 一覧画面の **Action** 列にある **Update** ボタンをクリックします。

[![Alt text](/img/migrated/Picture8-ea440913.png)](/img/migrated/Picture8-ea440913.png)

**ステップ 2**: タグ情報を編集します。編集可能なフィールドは **Description** のみです。Name フィールドは編集できません。

[![Alt text](/img/migrated/Picture9-2b92aca1.png)](/img/migrated/Picture9-2b92aca1.png)

**ステップ 3**: **Update** ボタンをクリックして完了します。

操作をキャンセルするには、**Cancel** をクリックして編集画面を終了します。

**3, タグの削除**

タグを削除するには、以下の手順に従ってください。

**ステップ 1**: メニューで **Tag management** を選択 >> 一覧画面の **Action** 列にある **Delete** ボタンをクリックします。

[![Alt text](/img/migrated/Picture11-68f55546.png)](/img/migrated/Picture11-68f55546.png)

**ステップ 2**: 確認ポップアップが表示されます。削除を確定するには、ポップアップ内の **Delete** をクリックします。

削除をキャンセルするには、**Cancel** をクリックします。

[![Alt text](/img/migrated/Picture10-c19c9196.png)](/img/migrated/Picture10-c19c9196.png)
