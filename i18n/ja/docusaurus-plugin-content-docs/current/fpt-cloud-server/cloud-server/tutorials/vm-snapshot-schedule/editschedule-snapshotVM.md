# インスタンスSnapshotスケジュールの編集

1. **Instance snapshot schedule** タブで **Action** → **Edit** をクリックします。

   [![Action Editメニュー](/img/migrated/image-1766044561523-30e56e38.png)](/img/migrated/image-1766044561523-30e56e38.png)

2. 必要に応じて情報を更新します。

   [![スケジュールの編集](/img/migrated/image-1766044589323-730f2f46.png)](/img/migrated/image-1766044589323-730f2f46.png)

3. **Update schedule** をクリックします。
