# Storage Disk の編集

ストレージ構成を変更する必要がある場合、Storage Disk の名前、ストレージタイプを変更したり、容量を増加したりできます。

:::note
アタッチされていない Storage Disk のみ編集できます。
:::

## Storage Disk の編集

1. Storage Disk 一覧で **Actions** → **Edit** をクリックします。

   [![Edit オプションがある Actions メニュー](/img/migrated/image-1712215559562-97286a15.png)](/img/migrated/image-1712215559562-97286a15.png)

2. 内容を更新します。
   - **Name**: Storage Disk の名前。
   - **Storage Policy**: ストレージタイプ。
   - **Size**: ストレージサイズ。

   **Edit storage disk** をクリックして保存します。

   [![Edit Storage Disk ダイアログ](/img/migrated/image-1712215566139-704f934e.png)](/img/migrated/image-1712215566139-704f934e.png)

:::warning
新しいサイズは現在のサイズより大きくする必要があります — Storage Disk の容量を縮小することはできません。
:::
