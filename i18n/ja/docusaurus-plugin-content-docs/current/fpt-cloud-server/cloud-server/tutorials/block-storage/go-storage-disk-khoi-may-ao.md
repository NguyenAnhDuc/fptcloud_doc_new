# Storage Disk を Instance からデタッチする

Storage Disk が Instance で不要になった場合、デタッチできます。データはそのまま保持され、後で別の Instance にアタッチすることができます。

## Storage Disk のデタッチ

1. Storage Disk 一覧で **Actions** → **Detach** をクリックします。

   [![Detach オプションがある Actions メニュー](/img/migrated/image-1712215548327-dc507d37.png)](/img/migrated/image-1712215548327-dc507d37.png)

2. ポップアップで内容を確認し、**Detach** をクリックします。

   [![Detach 確認ダイアログ](/img/migrated/image-1712215553461-87cc9cfe.png)](/img/migrated/image-1712215553461-87cc9cfe.png)
