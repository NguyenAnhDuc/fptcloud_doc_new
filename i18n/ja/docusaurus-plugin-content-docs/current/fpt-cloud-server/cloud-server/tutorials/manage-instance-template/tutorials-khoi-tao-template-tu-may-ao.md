# 仮想マシンからTemplateを作成する

設定済みの仮想マシンからTemplateを作成することで、同じ環境の仮想マシンを素早く複数デプロイできます。

## Templateを作成する

1. **Instance Management** にて、**Actions** → **Create Template** を選択します。

   [![Create TemplateオプションのあるActionsメニュー](/img/migrated/image-1715855344608-be2905c7.png)](/img/migrated/image-1715855344608-be2905c7.png)

2. Templateの情報を入力し、**Create template** をクリックします。

3. システムが **Custom Images** ページにリダイレクトし、作成の進捗を確認できます。

   [![Template作成の進捗](/img/migrated/image-1715855896164-3bb4e6c2.png)](/img/migrated/image-1715855896164-3bb4e6c2.png)

   [![Template作成完了](/img/migrated/image-1715855902172-92a593fa.png)](/img/migrated/image-1715855902172-92a593fa.png)
