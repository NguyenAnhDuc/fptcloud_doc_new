# 初期設定

FPT Cloudを初めてご利用の場合は、開始前に以下の手順を完了してください。

## アカウントの作成とFPT Cloudへのサインイン

FPT Cloudサービスを利用するには、FPT Portalにサインインする必要があります。

1. [https://console.fptcloud.com](https://console.fptcloud.com/) にアクセスし、FPT Cloudの認証情報でサインインします。
2. 使用する**テナント**、**リージョン**、**VPC**を正しく選択します。

アカウントをお持ちでない場合は、**Sign Up**をクリックして画面の指示に従って情報を入力してください。サポートチームが連絡を取り、アカウントの作成を確認します。

:::note
操作中にエラーが発生した場合は、すぐにサポートチームにお問い合わせください。
:::
