# プライマリNICの作成

Instanceを作成すると、システムが入力した情報に基づいてプライマリNICを自動的に生成します。

[![Instance作成画面](/img/migrated/image-1691381396202-7c332ae8.png)](/img/migrated/image-1691381396202-7c332ae8.png)

1. メニューから **Compute** を選択し、**Instance Management** タブを開いて対象のInstanceを選択します。

[![Instanceリスト](/img/migrated/image-1691381410150-1b799854.png)](/img/migrated/image-1691381410150-1b799854.png)

2. **Network Interface** タブを選択します。

   このタブではInstanceのすべてのNICカードを管理できます。カードが1枚だけの場合、それがデフォルトのプライマリNICです。プライマリNICでは **Disable** 操作のみ使用できます。

[![Network Interfaceタブ](/img/migrated/image-1691381424127-1da54a77.png)](/img/migrated/image-1691381424127-1da54a77.png)

[![プライマリNICの詳細](/img/migrated/image-1691381438228-5cb76cd6.png)](/img/migrated/image-1691381438228-5cb76cd6.png)
