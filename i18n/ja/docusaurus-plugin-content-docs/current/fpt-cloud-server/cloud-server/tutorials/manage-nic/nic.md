# ネットワークインターフェースカード (NIC)

## 概要

FPT Cloud上のネットワークインターフェースカード (NIC) は、InstanceをCloudのネットワークリソースに接続するための重要なコンポーネントです。NICは高いパフォーマンスと信頼性を持つ接続・データ転送機能を提供し、クラウドベースのアプリケーションやサービス向けの柔軟なネットワーク環境を実現します。

## NICを設定する前の注意事項

- これはネットワークインフラの構築・管理に関する知識を持つユーザー向けの高度な機能です。
- Instanceがネットワークリソースに効率的にアクセスできるよう、NICのネットワーク接続を決定してください。
- NICにIPアドレス、サブネットマスク、ゲートウェイ、DNSを正しく設定してください。
- 設定後、NICが正常に動作し、InstanceおよびネットワークでNICが問題を起こさないことをテストして確認してください。
