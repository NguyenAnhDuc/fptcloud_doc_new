# 概要

ユーザートークンは短期間有効な認証情報で、FPT Cloud AI Studio のAPIにパスワードを送信することなくアクセスできます。リクエストのたびにパスワードを送信する代わりに、トークンを提示することで身元を証明し、特定サービスへのアクセス権を取得します。

ユーザートークン機能は以下の操作をサポートします。

| 機能 | 説明 |
| --- | --- |
| トークン発行 | 名前と有効期限を指定して新しいアクセストークンを生成する |
| トークン検証 | トークンが正当で有効期限内かつ適切な権限を持つかを確認してからAPIアクセスを許可する |
| トークン取り消し | 不要になったトークンや漏洩のおそれがあるトークンを手動で無効化する |
| 有効期限管理 | 有効期限を過ぎたトークンを自動的に「Expired」状態に変更する |

:::note
FPT Cloud Portalはトークン作成後にその内容を保存しません。生成直後に必ずコピーして安全な場所に保管してください。
:::
