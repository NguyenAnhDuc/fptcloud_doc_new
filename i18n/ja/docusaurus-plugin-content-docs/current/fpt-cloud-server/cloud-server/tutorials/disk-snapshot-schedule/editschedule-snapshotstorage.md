# ストレージディスクSnapshotスケジュールの編集

1. **Storage snapshot schedule** タブで **Action** → **Edit** をクリックします。

   [![Action Editメニュー](/img/migrated/image-1766045109643-13e019dd.png)](/img/migrated/image-1766045109643-13e019dd.png)

2. 必要に応じて情報を更新します。

   [![スケジュールの編集](/img/migrated/image-1766045137152-94c1a305.png)](/img/migrated/image-1766045137152-94c1a305.png)

3. **Update schedule** をクリックします。
