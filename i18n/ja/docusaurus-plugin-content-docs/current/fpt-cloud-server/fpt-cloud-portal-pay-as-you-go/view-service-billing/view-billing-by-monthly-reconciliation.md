# 月次請求の確認

当月に請求されたサービスの一覧を確認できます。

1. **Billing – Bills**ページに移動します。

2. デフォルトでは、システムは当月の請求書を表示します。過去の月を選択して確認することもできます：
   - 当月の総使用金額。
   - 仮想マシン、Public IP、ストレージディスクを含む使用済みサービスの一覧。
