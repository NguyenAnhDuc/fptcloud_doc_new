# 作成済みの Managed Gpu クラスターの一覧を表示

[Managed GPU Cluster] ページでは、作成された GPU クラスタのリストを表示および管理できます
管理。[管理] ページを開くには:
**FPTポータルで、[AIインフラストラクチャ] >[マネージドGPUクラスター]**を選択します。作成されたクラスターのリストの**名前、バージョン、ワーカーグループ、ステータス、作成時、アクション** などの重要な情報とともに表示されます。
[![](/img/migrated/8-1-75374b83.png)](/img/migrated/8-1-75374b83.png)
**クラスターの詳細にアクセス** **ステップ1: [Managed GPU Cluster]** メニューに、**[ マネージドGPU管理]** ページが表示されます。詳細を表示するクラスターを選択します。
[![](/img/migrated/9-1-1670bae1.png)](/img/migrated/9-1-1670bae1.png)
**ステップ2: [エッセンシャルプロパティ]** タブに、クラスターの情報が表示されます。
[![](/img/migrated/10-1-6e0a4cbd.png)](/img/migrated/10-1-6e0a4cbd.png)
  * **クラスター情報：** クラスター名、バージョン、構成、ネットワーク、ステータス
  * **ロードバランサーサービス:** インポートされた内部LBサブネット情報
  * **API:** クラスターにつながる API URL

**ステップ3: [ノードプール]** タブには、クラスター内のすべてのワーカーグループと各ワーカーグループの構成情報が表示されます。
[![](/img/migrated/11-1-3eeaf351.png)](/img/migrated/11-1-3eeaf351.png)
  * **名前:** ワーカーグループ名
  * **Is Based:** ワーカーベースの場合は () が表示し、ワーカーベースではないの場合は (✘) が表示します。
  * **フレーバータイプ:** 選択したフレーバーリソースを表示します。
  * **サーバー数:** ワーカーのメタルクラウドのサーバー数
