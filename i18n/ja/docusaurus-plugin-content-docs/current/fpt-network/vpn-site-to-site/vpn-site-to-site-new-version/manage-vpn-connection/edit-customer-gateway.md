# Customer Gatewayの編集

:::note
Customer Gatewayは、VPN Connectionに割り当てられていない場合のみ編集できます。
:::

1. **Customer Gateway** 一覧画面で、変更するゲートウェイのActionカラムにある **Edit Customer Gateway** をクリックします。

[![Customer Gatewayの編集](/img/migrated/image-1744269804714-c8f82034.png)](/img/migrated/image-1744269804714-c8f82034.png)

2. 以下のフィールドを編集します：

[![Customer Gatewayフォーム](/img/migrated/image-1744269819671-b090793a.png)](/img/migrated/image-1744269819671-b090793a.png)

| **フィールド** | **説明** |
| --- | --- |
| **Customer gateway name** | Customer Gatewayの名前。 |
| **Description** | Customer Gatewayの説明。 |
| **Remote IP public** | オンプレミスVPN機器のパブリックIPアドレス。 |
| **Remote network** | オンプレミスネットワークのサブネット。 |
| **Healthcheck IP address** | 2サイト間のVPN接続ヘルスを監視するためのIPアドレス。 |

3. **Update Customer Gateway** をクリックして変更を保存するか、**Cancel** をクリックして破棄します。

[![更新結果](/img/migrated/image-1744269832390-ef24471f.png)](/img/migrated/image-1744269832390-ef24471f.png)
