# Disable VPN Site-to-Site

Disable an active VPN connection (applicable when the VPN status is **Active**).

1. Select the VPN connection to disable.

[![Select VPN](/img/migrated/Config-15-1024x262-574882a1.png)](/img/migrated/Config-15-1024x262-574882a1.png)

[![Disable VPN](/img/migrated/Disable-VPN-1024x354-0338f037.png)](/img/migrated/Disable-VPN-1024x354-0338f037.png)

2. Confirm the disable action.

[![Confirm disable](/img/migrated/Disable-VPN-confirm-1024x445-a8ee08e4.png)](/img/migrated/Disable-VPN-confirm-1024x445-a8ee08e4.png)

3. The VPN status changes from **Deactivating** to **Inactive**.

[![Inactive status](/img/migrated/Config-inactive-1024x238-c1dec1de.png)](/img/migrated/Config-inactive-1024x238-c1dec1de.png)
