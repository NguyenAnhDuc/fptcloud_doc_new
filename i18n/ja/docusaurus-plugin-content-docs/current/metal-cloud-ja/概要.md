# **Metal Cloud とは**

Metal Cloud とは、ユーザーが FPT から GPU コンポーネントを内蔵した物理サーバーをレンタルするクラウドサービスの一種であります。このサーバー* はユーザー専用であり、他のテナントと共有されません。 仮想化レイヤーや抽象化レイヤーを使用せずにハードウェアを完全に制御しながらサーバーにアクセスできるため、計算集約型のワークロードのパフォーマンスを最大限に高めることができます。

※このサーバーは Bare Metal GPUサーバーとも呼ばれます。

# **運用方法**
Bare Metal GPU サーバーをレンタルする場合、物理マシンをお客様専用に割り当て、物理ネットワーク構成を設定します。
サーバー環境は完全に孤立するため、マシンのコンピューティング能力とネットワーク能力に100%アクセスできます。
FPT カスタマーポータルでネットワーク (サブネット/IP)、SSH キー、ユーザーデータを構成し、オペレーティングシステムをインストールしてサーバーを展開できます。

# **Metal Cloud を選ぶ理由**
  * 最高のパフォーマンスを実現する専用ハードウェア
  * ソースの高い可用性と拡張性
  * 高度なカスタマイズ
  * パフォーマンスとセキュリティの向上
  * 予測可能なコスト
