# サーバーの電源オンとオフ

1. サイドバーの**AI Infrastructure** と **Metal Cloud** に移動してサーバーリストを表示します
  2. サーバーを選択して**「Actions」** をクリックします
  3. 実行中のサーバーのを**「Power off」** をクリックするか、**停止中の** サーバーを**「Power off」** をクリックします
  4. 電源オフまたは電源オンのアクションを確認します

# サーバーの削除
_サーバーを削除すると、ディスクのすべてのデータが完全に消去され、元に戻すことはできません。正しいサーバーを削除することを確認してください。_
  1. サイドバーの **AI Infrastructure** と **Metal Cloud** に移動してサーバーリストを表示します。
  2. サーバーを選択して**Actions** をクリックします
  3. **Delete** をクリックします。
  4. **DELETE** テキストを入力して削除を確認します

# 名前を更新
_サーバー名は一意であり、既存の名前と重複することはできません。_
  1. サイドバーの **AI Infrastructure** と**Metal Cloud** に移動してサーバーリストを表示します。
  2. サーバーを選択して **Actions** をクリックします
  3. **Rename** をクリックします
  4. ルールに準拠する新しいサーバー名を入力します：名前は最大 63 文字までで、文字、数字、ダッシュのみ使用できます。
  5. **Rename server** をクリックします
