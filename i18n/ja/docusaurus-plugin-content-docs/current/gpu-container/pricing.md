# Pricing

USD払いは世界中で受け付けられ、ベトナムドンはベトナムで、日本円は日本で適用されます。利用可能なプランの詳細については、[料金ページ](https://ai.fptcloud.com/pricing/container)をご参照ください。

## コスト構成

GPU Container利用料金は以下の要素で構成されています：

- **GPU Instanceコスト**：コンテナが実行状態にある間、毎秒課金されます。
- **永続ストレージコスト**：コンテナが存在する間（停止中も含む）毎秒課金されます。現在は$0.0066/GB/時間です。
- **一時ストレージ**：現在無料です。

詳細については [Payment Model](/ja/docs/gpu-container/payment-model/) および [Billing Overview](/ja/docs/gpu-container/billing-overview/) をご参照ください。
