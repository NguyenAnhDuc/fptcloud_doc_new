# Action

## コンテナを停止するとデータは失われますか？
一時ディスク上のデータは失われ、復元できません。永続ディスク上のデータのみ保持されます。重要なデータは必ず永続ディスクに保存してください。

## コンテナ操作時のストレージ動作まとめ

| ストレージタイプ | コンテナ停止時 | コンテナ削除時 |
| --- | --- | --- |
| 一時ディスク | データは完全に削除される | データは完全に削除される |
| 永続ディスク | データは保持される | データは完全に削除される |

ストレージの詳細については [Storage](/ja/docs/gpu-container/storage/) をご参照ください。
