# Payment Model

## PAYG
従量課金制（PAYG）は、事前コミットメントなしで実際に使用したリソースに対してのみ支払う柔軟な支払いモデルです。料金はコンピューティング時間やストレージなど、実際の使用量に基づいて計算されます。

例えば、1つのH100 GPU（15 CPU、250 GB RAM、1 GPU）を搭載したコンテナを1時間$2.31でレンタルする場合、コンテナを15分間実行すると$2.31 × 0.25 = $0.5775かかります。この金額は15分ごとに計算が完了次第、残高から直接差し引かれます。コンソールのBillingページで「Credit History」をクリックすると、詳細な請求履歴を確認できます。

## 請求サイクル

コンテナが実行中の間、料金は**15分ごと**に計算されて残高から差し引かれます。また、永続ストレージの料金はコンテナが存在する間（実行中・停止中を問わず）毎秒課金されます。

詳細については [Billing Overview](/ja/docs/gpu-container/billing-overview/) をご参照ください。
