# Add Credit

残高が不足して新しいGPU Containerを作成できない場合は、以下の手順に従ってアカウントにチャージして再試行してください。
  1. AI Factory Portalの左側メニューで **Billing** をクリックします
  2. Billingに初めてアクセスする場合は**通貨を選択**する必要があります。 [![Alt text](/img/migrated/image2025-6-22_11-39-26-9352e9a8.png)](/img/migrated/image2025-6-22_11-39-26-9352e9a8.png)
  3. **Add credit** ボタンをクリックします [![Alt text](/img/migrated/image2025-6-22_11-49-23-92aca56f.png)](/img/migrated/image2025-6-22_11-49-23-92aca56f.png)
  4. **Add creditポップアップ**では、提案された金額を選択するか、残高に追加したい**クレジット金額を入力**できます。 [![Alt text](/img/migrated/image2025-6-22_11-47-16-e08e227b.png)](/img/migrated/image2025-6-22_11-47-16-e08e227b.png)
  5. 次に**支払い方法**を選択します：
     * **VNPay**：ベトナム地域とベトナムドンのみ対応しています。**VNPay**のチェックアウトウェブサイトで取引を完了します。
     * **Stripe**：ポップアップウィンドウにカード情報を入力して支払いを進めます。**すべての取引はStripeによって安全に処理され、FPTはカード情報を保存しません。**
[![Alt text](/img/migrated/image2025-6-22_11-55-30-5be3b48e.png)](/img/migrated/image2025-6-22_11-55-30-5be3b48e.png)
  6. 支払い完了後、成功通知を表示するポップアップが表示され、残高が更新されます。 [![Alt text](/img/migrated/Picture1-9ed4b9be.png)](/img/migrated/Picture1-9ed4b9be.png)
