# Managed GPU Clusterを削除する

不要になったManaged GPU Clusterは、以下の手順で削除できます。

**手順1:** メニューで **AI Infrastructure** > **Managed GPU Cluster** を選択します。システムは **GPU Cluster Management** ページを表示します。削除したいクラスターを選択します。

[![](/img/migrated/14-b4dc395d.png)](/img/migrated/14-b4dc395d.png)

**手順2:** 一覧で削除したいクラスターの行末にある **Action** を選択し、**Delete** を選択します。

[![](/img/migrated/15-05e09c5c.png)](/img/migrated/15-05e09c5c.png)

**手順3:** ポップアップの警告情報を確認し、**Delete** をクリックします。

[![](/img/migrated/16-52efdb8d.png)](/img/migrated/16-52efdb8d.png)
