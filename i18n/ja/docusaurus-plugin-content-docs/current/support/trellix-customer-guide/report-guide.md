# レポートの表示

Trellix は、複数の時間範囲にわたってマルウェア検出、エクスプロイト、クラックツールのアクティビティを監視するための組み込みレポートテンプレートを提供します。レポートを使用すると、個別のログを手動で確認することなく、全体的なセキュリティ状況を素早く把握できます。

1. 管理コンソールのメニューバーから **Queries & Reports** を選択します。

   [![Trellix インターフェースの Queries & Reports ページ](/img/migrated/image-1765765637950-de5f4b98.png)](/img/migrated/image-1765765637950-de5f4b98.png)

2. **Shared Groups** > **Report** を選択します。

   [![Shared Groups のレポート一覧](/img/migrated/image-1765765653374-e18ef375.png)](/img/migrated/image-1765765653374-e18ef375.png)

   FPT Cloud には以下の事前設定済みレポートテンプレートが含まれています:

   - マルウェア動作レポート（24 時間 / 7 日間 / 30 日間）
   - エクスプロイト動作レポート（24 時間 / 7 日間 / 30 日間）
   - クラックツール実行動作レポート（24 時間 / 7 日間 / 30 日間）

:::note
カスタムレポートを作成するには、**New Query** をクリックして必要なパラメーターを設定します。
:::
