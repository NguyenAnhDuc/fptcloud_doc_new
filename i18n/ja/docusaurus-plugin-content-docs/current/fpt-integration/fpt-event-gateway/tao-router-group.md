# Router Groupの作成

新しい **Router Group** を作成するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **Event Gateway** を選択します。

**ステップ2:** **Event Gateway** リストで **Event Gateway** を選択 > **Router group** タブを選択します。

**ステップ3:** **Router group** リストで **Create** をクリック > **Create router group** 画面が表示されます。

**ステップ4:** 以下の情報を入力します。

- **Name**（必須）：Router Group名。
  - a～z、A～Z、ハイフン（-）、アンダースコア（_）、数字を使用できます。先頭は英字。最大30文字。
- **Description**（任意）：説明。

**ステップ5:** **Add router** をクリックしてグループにRouterを追加します。

**ステップ6:** **Create** をクリックして新しい **Router Group** を作成します。
