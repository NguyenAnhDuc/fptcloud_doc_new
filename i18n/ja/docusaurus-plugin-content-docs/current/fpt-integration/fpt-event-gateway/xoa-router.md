# Routerの削除

**Router** を削除するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **Event Gateway** を選択します。

**ステップ2:** **Event Gateway** リストで **Event Gateway** を選択 > **Security** タブを選択します。

**ステップ3:** **Router** リストで、削除したいRouterを含むGroupの **Edit** をクリック > 削除したいRouterの削除アイコンをクリックします。

[![削除アイコンが表示された編集モードのRouterリスト](/img/migrated/image27-bc4dff9c.png)](/img/migrated/image27-bc4dff9c.png)

[![削除対象として選択されたRouter](/img/migrated/image28-fb9a35ef.png)](/img/migrated/image28-fb9a35ef.png)

**ステップ4:** 削除が完了したら、**Save** をクリックして変更を保存します。

[![Router削除後の保存確認](/img/migrated/image29-579cd85c.png)](/img/migrated/image29-579cd85c.png)
