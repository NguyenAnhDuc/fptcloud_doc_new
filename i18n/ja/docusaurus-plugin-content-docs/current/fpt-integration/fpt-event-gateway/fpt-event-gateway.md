# FPT Event Gateway

**Event Gateway** は、システム間でリアルタイムにイベントを受信、処理、ルーティングする中間コンポーネントです。**集中型イベントデータゲートウェイ**として機能し、イベントソースとKafka、webhook、分析パイプラインなどの処理システムとの間で柔軟な接続を実現します。

**イベントのフィルタリング、変換、ルーティング**などの機能をサポートするEvent Gatewayは、イベント駆動型アーキテクチャに適しており、**リアルタイム分析**、**モニタリング**、**IoT**、および信頼性の高い制御されたイベントデータ処理が必要な分散システムでよく使用されます。
