# 初期セットアップ

FPT App Catalog を初めて使用する場合は、まず以下の作業を確認・完了してください。

1. FPT Cloud アカウントを作成し、FPT Portal にログインします。

   - FPT App Catalogs サービスを使用するには、FPT Portal にログインする必要があります。
   - [https://console.fptcloud.com](https://console.fptcloud.com) にアクセスし、割り当てられた FPT Cloud のアカウント / パスワードでログインしてください。使用するサービスの **Tenant、Region、VPC** を正しく選択してください。
   - FPT Cloud アカウントをお持ちでない場合は、[https://fptcloud.com/](https://fptcloud.com/) のホームページにアクセスしてください。
   - **Sign Up** 機能を選択し、システムの指示に従って情報を入力してください。その後、サポートチームが連絡してアカウント作成の確認を行います。

操作中に問題が発生した場合は、サポートチームにお問い合わせください。
