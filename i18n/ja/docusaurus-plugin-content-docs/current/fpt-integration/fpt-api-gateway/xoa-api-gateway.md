# API Gatewayの削除

**API Gateway** を削除するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **API Gateway** を選択します。

**ステップ2:** **API Gateway** 一覧から、削除したいAPI Gatewayを選択し、**Action** をクリックして **Delete** を選択します。

[![画像](/img/migrated/z7045391397635_d1a00a9228efc6171ae123d65-6c413f1c.jpg)](/img/migrated/z7045391397635_d1a00a9228efc6171ae123d65-6c413f1c.jpg)

**ステップ3:** **Delete Application** ダイアログで **delete** と入力し、**Confirm** をクリックして削除を完了します。

[![画像](/img/migrated/z7045438315669_cb9abecccf0015459dc9825c2-824ce48c.jpg)](/img/migrated/z7045438315669_cb9abecccf0015459dc9825c2-824ce48c.jpg)
