# FPT API Gateway

**API Gateway** は、クライアントとバックエンドサービスの間でリクエストを管理・制御・ルーティングするミドルウェアコンポーネントです。すべてのAPIへの集中アクセスポイントとして機能し、安全で一貫性のあるスケーラブルな通信を実現します。

**API Gateway** は、認証、認可、レート制限、ロードバランシング、監視、APIライフサイクル管理などの機能をサポートしています。**APISIX Dashboard** を使用すると、route、service、upstream、plugin、TLS証明書を視覚的に設定でき、管理・運用を簡素化できます。

API Gatewayはマイクロサービスアーキテクチャや分散システムに適しており、APIの管理におけるセキュリティ、柔軟性、可観測性の向上に広く活用されています。
