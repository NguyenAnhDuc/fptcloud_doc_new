# 初期設定

FPT Message Bus を使用する前に、FPT Cloud アカウントが必要で、正しい Tenant、Region、VPC にログインする必要があります。この手順により、サービスが適切な環境で動作することが保証されます。

## アカウントの作成とログイン

1. [https://fptcloud.com/](https://fptcloud.com/) にアクセスして **Sign Up** をクリックし、手順に従って情報を入力します。サポートチームがアカウントの確認と有効化のために連絡します。
2. [https://console.fptcloud.com](https://console.fptcloud.com) にアクセスし、FPT Cloud の認証情報でサインインします。
3. サービスに使用する正しい **Tenant**、**Region**、**VPC** を選択します。

:::note
設定中にエラーが発生した場合は、サポートチームに連絡してすぐにサポートを受けてください。
:::
