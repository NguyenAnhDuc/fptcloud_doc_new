# queue の表示

任意の queue の設定と状態を確認して、セットアップを監視・検証できます。すべての詳細は読み取り専用で表示されます。

1. 詳細を確認したい queue を選択します。

   [![queue を選択した queue 一覧画面](/img/migrated/View-1-bb456837.png)](/img/migrated/View-1-bb456837.png)

2. **View** ボタンをクリックします。

   [![Name、Status、設定フィールドを表示する queue 詳細画面](/img/migrated/View-2-cec0d10d.png)](/img/migrated/View-2-cec0d10d.png)

画面には設定済みの情報が表示されます。

- **Name**: queue 名。
- **Status**: queue の状態。
- 設定フィールド: max length、max length byte、message TTL、overflow など。

:::note
すべての詳細は読み取り専用で、ここでは編集できません。
:::
