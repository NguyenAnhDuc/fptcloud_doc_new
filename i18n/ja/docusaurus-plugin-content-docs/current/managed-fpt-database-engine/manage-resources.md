# リソースの管理

1. **Database Engine** を選択し、**ID Cluster** を選択します。**Database Detail** の **Resource** タブを選択します。

2. リソースを変更する方法を選択します。

   - **プリセット構成を使用する場合:**
     - **Choose Preset** を選択します。
     - リストからプリセットを選択します。

   - **カスタム構成を使用する場合:**
     - **Customized** を選択します。
     - CPU、RAM、Disk の情報を入力します。

3. **Apply** をクリックして変更を適用します。

:::note
リソースは残りのクォータ範囲内でのみ変更できます。クォータを超過した場合、システムがエラーを表示してリソースを変更できません。
:::
