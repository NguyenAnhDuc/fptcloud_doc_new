# サイト間VPNの削除

:::danger
VPN Site-to-Site接続の削除は元に戻せません。接続は永久に切断されます。
:::

1. VPN接続を選択し、**Actions** → **Delete** をクリックします。

   ![Delete VPNメニュー](/img/migrated/image-1742916525951-13239d02.png)

2. 削除を確認します。VPNステータスが **Deleting** に変わり、一覧から削除されます。

   ![VPN削除の確認ダイアログ](/img/migrated/image-1742916584099-b13362ab.png)
