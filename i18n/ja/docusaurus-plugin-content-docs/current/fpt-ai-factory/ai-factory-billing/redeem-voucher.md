# バウチャーの引き換え

AI Factory サービスを開始するには、**チャージクレジット（Top-up Credits）** または **バウチャークレジット（Voucher Credits）** を使用できます。
プロモーションオファーなどで **バウチャーコード** を受け取った場合は、それを適用してクレジットを獲得できます。

## 注意事項

  * バウチャーは **有効期間内に引き換える必要があります**。有効期限が切れたコードは使用できません。
  * **バウチャークレジットも有効期間後に失効し**、期限切れ後は使用できません。
  * **チャージクレジットとバウチャークレジットの両方をお持ちの場合**、AI Factory サービス使用時にシステムはバウチャークレジットを先に控除します。

## バウチャーの追加

バウチャーコードを受け取った後、以下の手順に従ってください。
  1. 左側メニューで **Account** セクションの **Billing** をクリックします。
  2. **Add Voucher** エリアでバウチャーコードを入力し、**Apply Code** をクリックします。（_注意：バウチャーコードは大文字の英字と数字を含む10文字で構成されます。_） [![Alt text](/img/migrated/image2025-6-30_13-37-5-94c9066a.png)](/img/migrated/image2025-6-30_13-37-5-94c9066a.png)
  3. 正常に適用されると、確認メッセージが表示されます：「**バウチャーが正常に適用されました（Your voucher has been applied successfully.）**」 [![Alt text](/img/migrated/image2025-6-30_13-56-37-f36f6894.png)](/img/migrated/image2025-6-30_13-56-37-f36f6894.png)
  4. 適用されたバウチャーは参照用として **バウチャー一覧（Voucher List）タブ** に表示されます。
