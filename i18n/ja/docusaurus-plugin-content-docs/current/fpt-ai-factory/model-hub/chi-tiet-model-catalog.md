# Model Catalog の詳細

Model Catalog 内の model の詳細を表示します。

Model Catalog 内の model には以下の情報が含まれます。
- **Model title**: model の名前。
- **Description**: model の説明。
- **Publisher**: model を作成した tenant。
- **Created date**: model の作成日。
- **Modified**: model 情報の最終更新日。
- **Tags**: model に付与された tag。
- **Model card**: model の概要情報。
[![](/img/migrated/anh-16-4af6707f.png)](/img/migrated/anh-16-4af6707f.png)

- **Version**:

  > **注意:** Model Catalog の Version タブは、**Model visibility = Public** かつ **Public** タイプの version を持つ model のデータのみ表示されます。**Model visibility = Public metadata** の model はこのタブを表示できません。

  - **Version**: version コード。
  - **Create date**: version の作成日。
  - **Size**: ストレージサイズ。
  - **Type**: version のステータス。
  - **Actions**: Model Catalog では **Download** のみ利用可能。
[![](/img/migrated/anh-17-66b2395d.png)](/img/migrated/anh-17-66b2395d.png)

- **File Browser**: 各 version にアップロードされたファイルを表示します。
  - **File**: ファイル名。
  - **Modified date**: 最終更新日。
  - **Size**: ファイルサイズ。
[![](/img/migrated/anh-18-0a98f64e.png)](/img/migrated/anh-18-0a98f64e.png)
