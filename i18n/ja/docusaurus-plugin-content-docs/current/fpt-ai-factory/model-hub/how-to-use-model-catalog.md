# Model Catalog の使用

各 model カードには以下が含まれます：
  * **Model Name**（例：`Qwen/Qwen3-14B`、`meta-llama/Llama-3-70B`）
  * **Provider**（例：Qwen、Meta LLaMA、Google Gemma、DeepSeek、IBM Granite、Mistral）
  * **バージョンまたはサイズ**（例：`7B`、`14B`、`70B`）

各 model カードは **三点メニュー（⋮）** 経由で複数のオプションを提供します：
  * **Playground**
Playground インターフェースで model を直接テストします。deploy 前に操作を確認できます。
  * **Create API Endpoint**
model の API endpoint を生成して、アプリケーションやサービスに統合できます。
  * **Fine-tune**
独自のデータセットを使用して model をカスタマイズし、特定のユースケースのパフォーマンスを向上させます。

**model の閲覧と検索**
  * 上部の **検索バー** を使用して、名前で model をすばやく検索します。
  * **フィルタータブ**（例：DeepSeek、GPT、Qwen など）を使用して、特定のファミリーの model を表示します。
  * 右下の **「Models per page」** 設定を調整して、1 ページに 8、16、32、64、またはそれ以上の model を表示します。
  * ページネーションコントロールを使用してページ間を移動します。
