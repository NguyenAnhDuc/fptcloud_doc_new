# バージョンの作成

**目的**：model のトレーニング情報をシステムに保存するためのバージョンを作成します。
新しいバージョンを作成するには、以下の手順に従います：
**ステップ 1**：model をクリックして詳細を表示 > **Version** タブを選択します。
**ステップ 2**：**New version** ボタンをクリックして作成フォームを開きます。
[![file](/img/migrated/image-1759303671098-2d129273.png)](/img/migrated/image-1759303671098-2d129273.png)
**ステップ 3**：バージョン作成フォームが表示されます。以下の情報を入力します：
**Version**：バージョン名
  * **注意**：バージョン名には a-z、A-Z、数字、「-」、「_」、「.」を使用でき、最大 20 文字まで入力できます。バージョン名は各 model に対して一意でなければなりません。

**Version visibility**：バージョンの公開状態。Private または Public を選択できます。
**Tags**：このバージョンの分類や説明に役立つキーと値のペアを追加します。
  * **Key**：タグ名
  * **Value**：タグの値

**ステップ 4**：**Create** をクリックしてバージョンを作成します。
