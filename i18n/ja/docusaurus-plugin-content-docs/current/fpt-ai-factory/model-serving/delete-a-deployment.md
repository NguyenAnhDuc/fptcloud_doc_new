# デプロイメントの削除

1. デプロイメント一覧から削除するデプロイメントを選択します。
2. **Delete**をクリックします。
3. ポップアップで削除を確認します。

:::warning
デプロイメントを削除すると、関連するすべてのAPIエンドポイントが停止します。この操作は元に戻せません。
:::
