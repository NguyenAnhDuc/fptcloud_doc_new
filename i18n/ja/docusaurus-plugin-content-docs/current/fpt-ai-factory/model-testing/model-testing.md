# モデルテスト

モデルテストは、FPT AI Factory上でAIモデルの応答品質を評価・検証するための機能です。

## 概要

モデルテスト機能を使用すると、特定の入力に対するモデルの応答を体系的にテストし、品質を確認できます。

## 主な機能

- **定義の確認** — モデルテストの基本概念と用語を理解
- **インタラクティブセッション** — リアルタイムでモデルの応答を確認
- **補完履歴** — 過去のテスト結果を確認・比較
- **主要機能** — テスト設定のカスタマイズと自動評価

サイドバーから各セクションを選択して、モデルテストの詳細をご確認ください。
