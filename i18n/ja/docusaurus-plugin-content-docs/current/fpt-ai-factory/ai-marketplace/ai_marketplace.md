# AI Marketplace

AI Marketplace は FPT Cloud の Model-as-a-Service プラットフォームです。インフラ構築不要で、事前学習済み AI モデルに API 経由で即座にアクセスできます。

## できること

- **推論の実行** — LLM、VLM、TTS、STT、Embedding モデルを標準 REST API で呼び出せます。
- **モデルのファインチューニング** — LoRA を使用して、独自のデータでファウンデーションモデルをカスタマイズできます。
- **カスタムモデルのデプロイ** — ファインチューニング済みの LoRA アダプターを専用 API エンドポイントとして公開できます。
- **Playground でのテスト** — コードを書く前に、任意のモデルをインタラクティブに試すことができます。
