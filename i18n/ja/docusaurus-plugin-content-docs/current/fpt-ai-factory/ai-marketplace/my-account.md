# マイアカウント

## 1.1 API キーを確認するには？

API キーを確認するには、以下の手順に従ってください。

:::note
有料プランに加入済みのアカウントが必要です。お持ちでない場合は、[お問い合わせ](./お問い合わせ.md)ください。
:::

**ステップ 1：** ヘッダーのアカウントアイコンをクリックします。

[![アカウントアイコンのスクリーンショット](/img/migrated/0-fb82a997.png)](/img/migrated/0-fb82a997.png)

**ステップ 2：** **「マイアカウント」** を選択します。

API キーの一覧が **「My API Keys」** ページに表示されます。

[![My API Keys ページのスクリーンショット](/img/migrated/my-api-key-02299856.png)](/img/migrated/my-api-key-02299856.png)

## 1.2 新しい API キーを作成するには？

API キーを作成するには、以下の手順に従ってください。

**ステップ 1：** **「My API Keys」** ページに移動し、**「Create New API Key」** ボタンをクリックします。

**ステップ 2：** **名前** を入力します。

**ステップ 3：** アクセスするモデルを選択します。

**ステップ 4：** **「Create」** ボタンをクリックして新しい API キーを生成します。

[![API キー作成フォームのスクリーンショット](/img/migrated/create-my-api-key-ac675818.png)](/img/migrated/create-my-api-key-ac675818.png)

新しい API キーが作成されたら、キーをコピーまたは保存してください。

:::warning
API キーは作成時に一度だけ表示・コピーできます。キーを安全な場所に保管し、他者と共有したり、ブラウザやクライアント側のコードで公開したりしないでください。API キーを持つ人は誰でもあなたの代わりにリクエストを送信できます。アカウントのセキュリティ保護のため、AI Marketplace は公開された API キーを自動的に無効にする場合があります。
:::

**「Done」** ボタンをクリックして完了します。

[![完了ボタンのスクリーンショット](/img/migrated/done-7329bd74.png)](/img/migrated/done-7329bd74.png)

## 1.3 既存の API キーを編集するには？

API キーの **名前** と **権限** を編集できます。以下の手順に従ってください。

**ステップ 1：** **「My API Keys」** ページに移動し、編集する API キーの横にある **「Action」** をクリックします。

**ステップ 2：** ドロップダウンメニューから **「Edit」** を選択します。

**ステップ 3：** 情報を編集します。

**ステップ 4：** **「Save」** ボタンをクリックして変更を確定します。

## 1.4 API キーを削除するには？

:::warning
削除後、この API キーは即座に無効になります。このキーを使用した API リクエストは拒否され、依存しているシステムが動作しなくなる可能性があります。
:::

**ステップ 1：** **「My API Keys」** ページに移動し、削除する API キーの横にある **「Action」** をクリックします。

**ステップ 2：** ドロップダウンメニューから **「Revoke」** を選択します。

**ステップ 3：** **「Revoke key」** ボタンをクリックして削除を確認します。

[![API キー削除確認のスクリーンショット](/img/migrated/revoke-55f19ebc.png)](/img/migrated/revoke-55f19ebc.png)

## 2. 使用状況を確認するには？

各モデルと API キーの入力トークン数、出力トークン数、合計使用量などを確認するには、**マイアカウント → My usage** にアクセスしてください。

[![使用状況ページのスクリーンショット](/img/migrated/my-usage-9f9ef039.png)](/img/migrated/my-usage-9f9ef039.png)
