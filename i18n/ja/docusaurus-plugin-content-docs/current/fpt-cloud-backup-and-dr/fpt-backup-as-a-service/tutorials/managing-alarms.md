# Managing Alarms

**ステップ 1:** Backup As a Service の管理ポータルサイトにログインします。<https://baas.fptcloud.com> [![file](/img/migrated/image-1695292948402-3616b178.png)](/img/migrated/image-1695292948402-3616b178.png)

**ステップ 2:** ウィンドウが表示されたら **Configuration** を選択します。 [![file](/img/migrated/image-1695292954154-840bf510.png)](/img/migrated/image-1695292954154-840bf510.png)

**ステップ 3:** **Alarms Management** を選択します。次に、使用したい Alarm を選択して Enable をクリックします。 [![file](/img/migrated/image-1695292963049-43668c28.png)](/img/migrated/image-1695292963049-43668c28.png)

**ステップ 4:** **Edit** をクリックして、ご自身のニーズに合わせてパラメータをカスタマイズします。 [![file](/img/migrated/image-1695292969858-8e06f50e.png)](/img/migrated/image-1695292969858-8e06f50e.png)

詳細については、[https://helpcenter.veeam.com/docs/vac/provider_admin/configure_alarms.html?ver=60](https://helpcenter.veeam.com/docs/vac/provider_admin/configure_alarms.html?ver=60%20) をご参照ください。
