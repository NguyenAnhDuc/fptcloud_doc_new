# Detection Log Analysis

設定されたポリシーに基づき、URL ディレクトリへのアクセスログや Web 脆弱性の悪用ログを含むすべてのログを検索・フィルタリングします。[![file](/img/migrated/image-1714120758618-b85a7127.png)](/img/migrated/image-1714120758618-b85a7127.png)

**Advanced Condition** から各 Web 宛先ごとにフィルタリングすることも可能です。[![file](/img/migrated/image-1714112428666-12b53682.png)](/img/migrated/image-1714112428666-12b53682.png)
