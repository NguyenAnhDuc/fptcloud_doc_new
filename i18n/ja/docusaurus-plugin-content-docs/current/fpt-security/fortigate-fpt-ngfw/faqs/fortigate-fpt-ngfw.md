# Fortigate – FPT NGFW

FortiGate NGFW は Fortinet Security Fabric の一部であり、脅威からの自動保護機能を備えたファイアウォールソリューションです。ネットワーク攻撃、マルウェア、その他の脆弱性など、内部および外部の脅威から組織を保護するよう設計されています。NGFW は SSL インスペクション、アプリケーションコントロール、高度な可視性、その他のセキュリティ機能を提供しています。

## 主な機能

- 包括的な可視性と保護：Ransomware・Command & Control のブロック、SSL/TLS 1.3 暗号化、脅威に対する自動保護を統合しています。
- FortiGuard サポート：IPS、Web・動画フィルタリング、DNS セキュリティを統合して同時実行し、コスト削減とリスク管理を実現します。
- Proxy 統合：Zero Trust Network Access（ZTNA）によるシームレスなユーザー体験とデバイスセキュリティを提供します。
- 高度なセキュリティ：不正アクセスの防止、アクセス分離、明確なパケットフィルタリングを実現します。
- 自動化されたネットワーク管理：トラフィックの可視化、詳細なポリシー制御、セキュリティの拡張性を提供します。

## ドキュメント構成

- **管理インターフェースへのアクセス** — Fortigate NGFW の管理インターフェースにログインします。
- **Network Interfaces の設定** — LAN および WAN インターフェースを設定します。
- **Static Routes の設定** — スタティックルートを作成・管理します。
- **Firewall Policy の作成** — トラフィックを許可または拒否するルールを定義します。
- **Security Profile の作成** — AntiVirus、Web Filter、Application Control、IPS、WAF を設定します。
- **System User の管理** — 管理者プロファイルとユーザーアカウントを作成します。
- **Feature Visibility** — システム内の機能を有効・無効にします。
- **Logs Traffic の表示** — Firewall Policy からのトラフィックログを監視します。
- **よくある質問** — Fortigate – FPT NGFW に関するよくある質問です。
