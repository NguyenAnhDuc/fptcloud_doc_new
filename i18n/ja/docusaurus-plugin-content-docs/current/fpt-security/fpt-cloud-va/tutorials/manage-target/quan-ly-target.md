# ターゲット管理

**ターゲット**とは、FPT VAがセキュリティ脆弱性をスキャンするスキャン対象です。ターゲットは **FPT Portal** 上で作成・設定します。ターゲットには、Webサイト、パブリックIPアドレス、または管理するIPレンジを指定できます。

ターゲット管理で実行できる操作：

- [ターゲットを作成する](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/tao-target)
- [ターゲットのスキャンステータスを確認する](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/kiem-tra-trang-thai-target)
- [ターゲットのスキャン設定を編集する](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/update-target)
- [今すぐスキャンする](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/scan-now)
- [ターゲットを削除する](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/xoa-target)
- [Webサイトのヘッダーを取得する](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/lay-header)
- [Webサイトのクッキーを取得する](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/lay-cookie)
- [SMB用にWindowsでRemote Registryを有効にする](/ja/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/mo-remote-registry-window)
