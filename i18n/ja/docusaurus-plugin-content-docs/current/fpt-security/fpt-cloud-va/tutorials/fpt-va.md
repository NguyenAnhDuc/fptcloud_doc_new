# FPT Cloud VA

FPT Cloud Vulnerability Assessment（FPT Cloud VA）は、FPT Portal上でユーザーが管理するWebアプリケーションおよびネットワークシステムのセキュリティ脆弱性を自動的にスキャンして特定するサービスです。

## 主な機能

- Webサイト（Webアプリケーション）およびネットワーク（IPアドレスまたはIPレンジ）のターゲットのスキャンをサポートします。
- 複数のスキャンモードを提供します：Host Discovery、Standard Scan、Deep Scan、Brute Force Scan、Log4Shell Scan。
- スキャンスケジュールを設定できます：1回のみ、毎日、毎週、または毎月。
- スキャン完了後にメール通知を送信します。
- 複数のフォーマットで詳細な脆弱性レポートをエクスポートできます。

## ドキュメント構成

- **初期設定** — FPT Portalでサービスを登録し、クォータを確認します。
- **ターゲット管理** — スキャンターゲットの作成、編集、ステータス確認、スキャン実行、削除を行います。
- **スキャン結果の表示** — ダッシュボードへのアクセス、詳細結果の確認、スキャン履歴の閲覧を行います。
- **レポートのエクスポート** — 脆弱性評価レポートをエクスポートします。
