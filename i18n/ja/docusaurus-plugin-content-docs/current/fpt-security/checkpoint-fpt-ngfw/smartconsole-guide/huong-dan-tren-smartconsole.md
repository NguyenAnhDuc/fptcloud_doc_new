# SmartConsoleでの操作ガイド

このセクションでは、SmartConsoleアプリケーションを通じてCheckPoint – FPT NGFWを管理する方法について説明します。SmartConsoleへのアクセス、ポリシーのインストール、オブジェクトの作成、ファイアウォールルール、NATルール、ログの調査、BladeとAccountの管理が含まれます。
