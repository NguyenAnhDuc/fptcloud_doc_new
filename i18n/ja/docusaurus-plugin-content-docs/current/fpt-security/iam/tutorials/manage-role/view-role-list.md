# Role一覧の表示

**Roles management** ページで作成済みRoleの一覧を確認・管理できます。

1. FPT Portalで **IAM** > **Roles** を選択します。
2. Basic RoleとCustom Roleの両方を含む一覧が表示されます。各Roleには **Name**、**Type**、**Description**、**Status**、**Created at**、**No of group**、**Actions** が表示されます。

[![](/img/migrated/image-1734344067631-5186635a.png)](/img/migrated/image-1734344067631-5186635a.png)

:::note
**No of group** 列はそのRoleを割り当てているUser Groupの数を表示します。**Actions** 列では Delete、Edit Basic Information、Edit Permissions が実行できます（Custom Roleのみ）。
:::
