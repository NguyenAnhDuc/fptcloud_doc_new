# IAM userの招待

1. **IAM** > **Users** を選択し、**Invite users** をクリックします。

[![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. **Invite User** ダイアログで必要な情報を入力します：
   - **Users**: 招待するメールアドレスを正確に入力します。システムはこのアドレスに確認メールを送信します。
   - **User group**: userに割り当てるUser Groupを選択します（必須）。

[![](/img/migrated/image-1734344703145-b92230b6.png)](/img/migrated/image-1734344703145-b92230b6.png)

3. **Invite** をクリックします。システムがアカウントを確認し、新しいuserをVPCに招待して結果を表示します。

[![](/img/migrated/image-1734344718091-7b7d1882.png)](/img/migrated/image-1734344718091-7b7d1882.png)

招待が成功すると、新しいuserがUser Managementページに **Name**、**Email**、**User Groups** の情報とともに表示されます。
