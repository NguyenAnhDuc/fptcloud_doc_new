# 2FAの設定

**2FA（Two Factor Authentication）** は、パスワードに加えてもう一つのセキュリティ層を追加することで、アカウントをより安全に認証する方法です。両方の認証ステップを満たした場合のみ、アカウントへのログインが成功します。

二段階認証を有効にするには、以下の手順を実行します。

1. FPT Portalの右上にあるアカウント名表示エリアをクリックし、**Account Security** を選択します。

[![](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)

2. **Two Factor Authentication** セクションで、**Set up Authenticator Application** を選択します。

[![](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)

3. **Mobile Authenticator Setup** ページの手順に従ってモバイルデバイスに2FAをインストールし、**Submit** を選択します。

[![](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)

2FAの設定が完了すると、ログインのたびに2FAアプリが生成するOTPの入力が求められます。
