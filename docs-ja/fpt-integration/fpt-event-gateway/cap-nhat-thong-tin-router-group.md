# Router Group情報の更新

**Router Group** の情報を更新するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **Event Gateway** を選択します。

**ステップ2:** **Event Gateway** リストで **Event Gateway** を選択 > **Security** タブを選択します。

**ステップ3:** **Router group** リストで、更新したい **Router Group** を選択します。

**ステップ4:** 更新したいセクションのEditアイコンをクリックします。

- **ポップアップ：Update Basic Information**
  - **Name**（必須）：Router Group名。
    - a～z、A～Z、ハイフン（-）、アンダースコア（_）、数字を使用できます。先頭は英字。最大30文字。
  - **Description**（任意）：説明。

[![基本情報更新ポップアップ](/img/migrated/image23-aadfbc71.png)](/img/migrated/image23-aadfbc71.png)

- **ポップアップ：Update routers**
  - グループ内のRouterを追加または削除できるポップアップが表示されます。

[![Router更新ポップアップ](/img/migrated/image24-bd82d929.png)](/img/migrated/image24-bd82d929.png)

**ステップ5:** 各ポップアップで更新後、**Save** をクリックして変更を保存します。
