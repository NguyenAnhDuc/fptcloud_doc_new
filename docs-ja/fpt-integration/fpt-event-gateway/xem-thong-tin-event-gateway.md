# Event Gateway情報の確認

**Event Gateway** の情報を確認するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **Event Gateway** を選択します。

**ステップ2:** **Event Gateway** リストから、情報を確認したい **Event Gateway** を選択します。

- **Essential Information** タブ

  Event Gatewayの基本情報を表示します：**Basic information**、**Launch Configuration**、**Kafka cluster**、**Endpoint**。

[![Event Gatewayの詳細を表示するEssential Informationタブ](/img/migrated/image10-60575f21.png)](/img/migrated/image10-60575f21.png)

- **Security** タブ

  **Event Gateway** のSecurityの一覧を表示します。

[![JWT auth一覧を表示するSecurityタブ](/img/migrated/image18-37e84895.png)](/img/migrated/image18-37e84895.png)

- **Router group** タブ

  **Event Gateway** の **Router group** の一覧を表示します。

[![Router group一覧を表示するRouter groupタブ](/img/migrated/image19-cfa604c3.png)](/img/migrated/image19-cfa604c3.png)

- **Router** タブ

  **Event Gateway** のRouter groupごとの **Router** の一覧を表示します。

[![グループ別に整理されたRouterを表示するRouterタブ](/img/migrated/image1a-f6ba0423.png)](/img/migrated/image1a-f6ba0423.png)
