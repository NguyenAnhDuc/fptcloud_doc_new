# 初期設定

**FPT Event Gateway** サービスを使用するには、FPT Portalにログインする必要があります。

<https://console.fptcloud.com> にアクセスし、発行されたFPT Cloudのアカウント情報でログインしてください。使用するサービスに合わせて、正しいTenant、Region、VPCを選択してください。

FPT Cloudのアカウントをお持ちでない場合は、<https://fptcloud.com/> のホームページにアクセスしてください。

**Sign Up** 機能を選択し、システムの案内に従って情報を入力してください。サポートチームがその後すぐに連絡し、アカウント作成のために情報を確認します。

操作中にエラーが発生した場合は、すぐにサポートチームにご連絡ください。
