# consumer groupの管理

consumer groupは1つまたは複数のtopicからメッセージを読み取る複数のconsumerをまとめたグループで、consumerがグループに参加または離脱する際に並列処理と自動負荷分散を実現します。

実行可能な操作：

- [consumer groupの作成](/docs/ja/fpt-integration/fpt-kafka/create-consumer-groups)
- [consumer groupの表示](/docs/ja/fpt-integration/fpt-kafka/view-consumer-groups)
- [consumer groupの削除](/docs/ja/fpt-integration/fpt-kafka/delete-consumer-groups)
