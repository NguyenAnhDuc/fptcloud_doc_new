# topicの管理

topicはKafkaの中心的なデータ整理単位で、producerがメッセージを公開し、consumerがそれを読み取る場所です。topicを適切に管理することで、システム全体のパフォーマンスとデータフローを最適化できます。

実行可能な操作：

- [topicの作成](/docs/ja/fpt-integration/fpt-kafka/create-topics)
- [topicの表示](/docs/ja/fpt-integration/fpt-kafka/view-topics)
- [topicの編集](/docs/ja/fpt-integration/fpt-kafka/edit-topics)
- [topicの削除](/docs/ja/fpt-integration/fpt-kafka/delete-topics)
