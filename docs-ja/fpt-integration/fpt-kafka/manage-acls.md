# ACLの管理

ACL（Access Control List）は、Kafkaにおいてどのcredentialがどのtopicまたはconsumer groupにアクセスできるかを制御します。ACLを正しく設定することで、許可されたアプリケーションのみがデータを読み取りまたは書き込みできることを保証します。

実行可能な操作：

- [ACLの作成](/docs/ja/fpt-integration/fpt-kafka/create-acls)
- [ACLの削除](/docs/ja/fpt-integration/fpt-kafka/delete-acls)
