# 初期設定

**FPT API Gateway** の利用を開始するには、FPT Portalにログインする必要があります。

[https://console.fptcloud.com](https://console.fptcloud.com) にアクセスし、FPT Cloudのアカウント認証情報でサインインしてください。サービスに使用するTenant、Region、VPCを正しく選択してください。

FPT Cloudのアカウントをお持ちでない場合は、[https://fptcloud.com](https://fptcloud.com) にアクセスし、**Sign Up** をクリックして画面の指示に従って登録してください。サポートチームが連絡し、情報を確認してアカウントを作成します。

:::note
操作中にエラーが発生した場合は、すぐにサポートチームにお問い合わせください。
:::
