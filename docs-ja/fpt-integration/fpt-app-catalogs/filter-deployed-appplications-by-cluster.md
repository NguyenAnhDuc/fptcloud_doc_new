# クラスター別にデプロイ済みアプリをフィルタリングする

1. **Deployed App** 画面でクラスターを選択し、デプロイ済みアプリの一覧を確認します。

   [![クラスター選択画面](/img/migrated/Picture19-1-fa364f8b.png)](/img/migrated/Picture19-1-fa364f8b.png)

2. デプロイ済みアプリの一覧が表示されます。デフォルトでは、App Catalogs Management に最後に追加されたクラスターでフィルタリングされます。

   アプリがデプロイされているクラスター:

   [![デプロイ済みアプリあり](/img/migrated/Picture20-1-a5138541.png)](/img/migrated/Picture20-1-a5138541.png)

   アプリがデプロイされていないクラスター:

   [![デプロイ済みアプリなし](/img/migrated/Picture21-1-9bb1d91f.png)](/img/migrated/Picture21-1-9bb1d91f.png)

   フィルターをクリアする場合:

   [![フィルタークリア](/img/migrated/Picture22-1-45255b77.png)](/img/migrated/Picture22-1-45255b77.png)
