# Install Veeam Agent for Windows

**ステップ 1:** backup が必要なサーバー上で、ご自身のアカウントでポータルサイトにログインします。Backup as a Service: <https://baas.fptcloud.com> [![file](/img/migrated/image-1695292106378-41580e19.png)](/img/migrated/image-1695292106378-41580e19.png) ログイン成功後の管理画面は以下のとおりです。 [![file](/img/migrated/image-1695292115345-71164f38.png)](/img/migrated/image-1695292115345-71164f38.png)

**ステップ 2:** **Managed Computers** メニューから **Download Agent** を選択し、**Window** を選択します。 [![file](/img/migrated/image-1695292124121-91d52bda.png)](/img/migrated/image-1695292124121-91d52bda.png) 必要に応じて Token の有効期限を変更します。デフォルトは 12 ヶ月です。 [![file](/img/migrated/image-1695292132688-e41cc91e.png)](/img/migrated/image-1695292132688-e41cc91e.png)

**ステップ 3:** ダウンロードが完了したら、backup が必要なサーバーに Management Agent をインストールします。 [![file](/img/migrated/image-1695292142746-c8534ab5.png)](/img/migrated/image-1695292142746-c8534ab5.png) Next をクリックして続けます。 [![file](/img/migrated/image-1695292150003-ee0a607d.png)](/img/migrated/image-1695292150003-ee0a607d.png) 利用規約に同意してから、

Cloud Gateway: baas-sgn-gw.fptcloud.com または baas-han-gw.fptcloud.com Port: 6180 [![file](/img/migrated/image-1695292180601-c17bd7f4.png)](/img/migrated/image-1695292180601-c17bd7f4.png)

**ステップ 5:** ポータルに VM が表示され、Veeam Agent のインストールが正常に完了したことを確認します。 [![file](/img/migrated/image-1695292191409-db0a966c.png)](/img/migrated/image-1695292191409-db0a966c.png)

**ステップ 6:** ポータルに backup job が表示され、Veeam Agent For Windows がサーバーにインストール済みであることを確認します。 [![file](/img/migrated/image-1695292197657-f521f8f1.png)](/img/migrated/image-1695292197657-f521f8f1.png) [![file](/img/migrated/image-1695292206361-fd9d30b7.png)](/img/migrated/image-1695292206361-fd9d30b7.png)
