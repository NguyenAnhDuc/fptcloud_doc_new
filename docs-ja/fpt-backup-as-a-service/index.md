FPT Backup as a Service は、FPT Cloud がクラウド技術をベースに提供するバックアップソリューションサービスです。バックアップのプロセスを自動化し、情報を監視することで、天災・人的ミス・その他の重大な障害によるデータ損失リスクを最小限に抑えます。FPT Cloud Backup as a Service は、オンラインでのデータの backup と restore を実行するためのさまざまなオプションを備えており、仮想化システム・クラウド・オンプレミスシステムなどの backup 対象に必要なデバイスとソフトウェアを提供します。本サービスは、企業がデータを効果的に保護しながら、運用の複雑さ・コスト・時間を削減するためのソリューションです。

FPT Cloud Backup as a Service は、バックアップスケジュール・頻度・保存ポリシーを含むバックアップ計画およびソリューションの策定をサポートし、ユーザーのビジネスニーズに対応します。FPT Cloud は、多様な環境やシステムにおけるシステムデータの restore テストから障害復旧テストまで、さまざまなサポートオプションを提供します。バックアップ容量の柔軟な拡張ニーズに常に対応できるよう準備が整っており、高帯域幅により backup と restore を迅速かつ効率的に実行できます。

## Backup as a Service 利用時の注意事項
  * オンプレミスサーバー・仮想化システム・その他のクラウドシステムを backup する場合、管理者はバックアップ作業の提供と管理を担当します。
  * すでに backup システムを持つオンプレミスシステムの場合、FPT Cloud のデータセンターに保存される外部バックアップコピーを作成でき、バックアップインフラとバックアップ作業の提供・管理を受けることができます。
  * オンプレミスからクラウドへのシステム移行を支援し、移行を迅速に行いながら、移行時のリスクを軽減するための包括的なデータ保護を実現します。
  * VM 数・backup storage・利用中のシステム数について、各アカウントには FPT Cloud から一定のリソース使用上限が付与されます。FPT Cloud が最適なアドバイスとサポートを提供できるよう、保護対象のサーバー数とサーバーシステムをあらかじめ確認しておくことをお勧めします。

バックアップシナリオは、実際のニーズに応じてユーザーが主体的に定義する必要があります。FPT Cloud は、日次・週次・月次などの backup をサポートしており、ユーザーが利用ニーズに合わせて柔軟に選択またはカスタマイズできます。
