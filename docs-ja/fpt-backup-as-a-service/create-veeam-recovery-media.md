# Create Veeam Recovery Media

**ステップ 1:** Create Recovery Media アプリケーションを起動します。 [![file](/img/migrated/image-1695292809658-75c1dd96.png)](/img/migrated/image-1695292809658-75c1dd96.png)

**ステップ 2:** ウィンドウが表示されたら、Veeam Recovery Media ファイルに統合するネットワーク接続とドライバーを選択し、Next をクリックして続けます。 [![file](/img/migrated/image-1695292817914-6d9e55d8.png)](/img/migrated/image-1695292817914-6d9e55d8.png)

**ステップ 3:** ISO ファイルの名前と保存場所を指定します。その後、Next をクリックして続けます。
  * 注意: サーバーが削除されたり起動できなくなった場合にファイルを取り出せなくなるため、Veeam Recovery Media ファイルは backup 対象のサーバーに保存しないことをお勧めします。 [![file](/img/migrated/image-1695292826165-585e6c60.png)](/img/migrated/image-1695292826165-585e6c60.png)

**ステップ 4:** 情報を確認してから Create をクリックします。 [![file](/img/migrated/image-1695292833515-81e26e7e.png)](/img/migrated/image-1695292833515-81e26e7e.png)

**ステップ 5:** 作成が完了したら、Finish をクリックして終了します。 [![file](/img/migrated/image-1695292841371-d35664ce.png)](/img/migrated/image-1695292841371-d35664ce.png)

**Veeam Recovery Media for Linux:** ステップ 1: Linux の場合は、以下のリンクから直接ダウンロードできます。 <https://download2.veeam.com/VAL/v5/veeam-recovery-media-5.0.0.4318_x86_64.iso>
