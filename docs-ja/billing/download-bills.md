# 詳細照合ファイルをダウンロードする

詳細照合データをExcelファイルとしてダウンロードするには、以下の手順で操作してください。

1. 請求期間、contract、照合エントリを選択した後、**Download**ボタンをクリックします。

   [![Downloadボタン](/img/migrated/image-1744088593606-63e60ab6.png)](/img/migrated/image-1744088593606-63e60ab6.png)

2. 照合ファイルは、ログイン中のアカウントのメールアドレスに15〜20分以内に送信されます。

   [![ダウンロード確認メール](/img/migrated/image-1744088600025-5d3582dc.png)](/img/migrated/image-1744088600025-5d3582dc.png)

:::warning
システムはDownloadリクエストを15分に1回の頻度で受け付けます。15分以内に複数回Downloadをクリックしないようにしてください。
:::
