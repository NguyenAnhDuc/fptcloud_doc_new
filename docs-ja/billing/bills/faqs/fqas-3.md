# 同じ請求期間に複数の照合エントリがあるのはなぜですか？

同一Tenant内のProject/VPCやリソースを複数のbilling groupに分けて支払いを管理している場合、システムはbilling groupの数に応じた照合エントリを表示します。

:::tip
billing groupの設定について詳しくは、FCI Sale Supportまたは FCIチームにお問い合わせください。
:::
