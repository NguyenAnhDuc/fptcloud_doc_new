# リソース別の請求明細を確認する

**Charges by resource**タブでは、リソースタイプを選択して各リソースの請求数量、単価、合計金額を確認できます。このデータはダウンロードした照合ファイルの**Details**シートに対応しています。

[![Charges by resourceの画面](/img/migrated/image-1744088548570-8b7ba389.png)](/img/migrated/image-1744088548570-8b7ba389.png)
[![リソース一覧](/img/migrated/image-1744088556698-8e3a4ec2.png)](/img/migrated/image-1744088556698-8e3a4ec2.png)

## リソースの検索

**Search by name**バーを使用して、名前に特定の文字列を含むリソースを検索できます。検索条件に一致するリソースのAmountの合計が合計金額として表示されます。

[![名前で検索バー](/img/migrated/image-1744088563151-bd609838.png)](/img/migrated/image-1744088563151-bd609838.png)
[![検索結果](/img/migrated/image-1744088568905-881fb46e.png)](/img/migrated/image-1744088568905-881fb46e.png)

## リソースの絞り込み

フィルターパネルを使用して、以下の条件でリソースを絞り込めます。

- **Start date – End date** – Start timeとEnd timeが指定期間内のリソース
- **Service** – 特定のサービスに属するリソース。サービス分類がないリソースにはこのフィルターは表示されません
- **Configuration** – リソースの設定またはサイズ。設定/サイズがないリソースにはこのフィルターは表示されません
