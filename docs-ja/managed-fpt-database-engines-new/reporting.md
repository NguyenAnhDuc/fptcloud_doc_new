# レポーティング

**レポーティング**機能により、ユーザーは選択した期間のデータベースクラスターのアクティビティに関する統計データを抽出できます。レポートは手動または自動スケジュールで生成できます。

レポートはユーザーの以下を支援します。

- アクティビティ履歴の追跡。
- 実行状況の評価。
- 運用監査または内部記録のサポート。

現在、FPT Database Engineはデータベースクラスターのバックアップアクティビティレポートという1種類のレポートのみをサポートしています。

**必要な権限：**

| 権限名 | 操作種別 | 説明 |
| --- | --- | --- |
| manageDatabase:ReportConfig | 編集 | データベースアクティビティに関するレポートの受信スケジュールを設定する権限。 |
