# 初期セットアップ

**FPT Container Registry** を使用する前に、以下の準備手順を完了してアカウントと環境を整えてください。

## FPT Cloud アカウントの作成と Portal へのログイン

1. [fptcloud.com](https://fptcloud.com) にアクセスし、**Sign Up** をクリックします。
2. 必要な情報を入力します。カスタマーサポートがアカウントの確認と有効化のために連絡します。
3. 発行された認証情報を使用して [FPT Cloud Portal](https://console.fptcloud.com) にログインします。
4. 正しい **Tenant**、**Region**、**VPC** を選択します。

:::note
ログイン情報が不明な場合や 3 回試行後にエラーが発生する場合は、サポートチームにお問い合わせください。
:::

これらの手順が完了したら、**FPT Container Registry** のアクティベートと使用を開始できます。
