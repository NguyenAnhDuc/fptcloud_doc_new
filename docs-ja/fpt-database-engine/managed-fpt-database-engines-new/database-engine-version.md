# データベースエンジンバージョン

FPT Database Engineは、FPTがサポートするさまざまなデータベース管理システムに対して完全マネージドなデータベースサービスを提供しており、お客様がシステムを効率的に展開・運用・拡張しやすくしています。**データベースエンジンバージョン**セクションでは、これらのデータベースエンジンのバージョンサポートポリシー（バージョンライフサイクル、可用性、更新、およびサポート終了日）を説明します。

バージョンポリシーを理解することで、お客様は適切なデータベースエンジンバージョンを積極的に選択・管理し、システムが常に安全で安定しており、最新の改善との互換性を保ちながら、アップグレードやバージョン移行時のリスクと中断を最小化できます。
