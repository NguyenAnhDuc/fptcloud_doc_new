# FPT Console Portalへのアクセス

FPT Console PortalでFPT Database Engineサービスを使用するには、登録済みのアカウント認証情報でログインする必要があります。

### ステップ1：ログインページへのアクセス

<https://fptcloud.com/> のFPT Cloudホームページにアクセスし、**ログイン**ボタンをクリックしてログインページに移動します。

[![](/img/migrated/Login-scaled-48903630.png)](/img/migrated/Login-scaled-48903630.png)

### ステップ2：ログイン

2つのログイン方法があります。

- **FPT IDでログイン**: アカウントが個人のFPT IDである場合、このオプションを選択してログインします。FPTアカウントのログインページにリダイレクトされます。必要な情報を入力して続行してください。
- **組織アカウントでログイン**: 組織に企業アカウントがある場合、このオプションを選択して組織アカウントの認証情報でログインします。組織のログインページにリダイレクトされます。必要な情報を入力して続行してください。

ログインに成功すると、FPT Consoleのデフォルトページにリダイレクトされ、DBaaSリソースやその他のクラウドサービスを管理できます。

### ログインのトラブルシューティング

ログインに問題がある場合は、以下の手順をお試しください。

- **パスワードを忘れた場合**: パスワードを忘れた場合は、ログインページの「パスワードを忘れた場合」リンクをクリックし、手順に従ってパスワードをリセットしてください。
- **アカウントがロックされている場合**: アカウントがロックされている場合は、FPT Cloud Supportに連絡してアカウントのロック解除を依頼してください。
- **ログイン情報が正しくない場合**: ユーザー名とパスワードを再確認して、入力ミスがないことを確認してください。

**注意：** ログイン問題が続く場合は、FPT Cloud Supportにお問い合わせください。
