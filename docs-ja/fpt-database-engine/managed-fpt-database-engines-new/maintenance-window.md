# メンテナンスウィンドウ

**メンテナンスウィンドウ**により、お客様はデータベースクラスターの定期メンテナンス時間枠を積極的に設定できます。この時間帯にFPTはアップグレード、パッチ適用、システム最適化のアクティビティを実施します。

この機能はビジネス運営への影響を最小化し、サービスの安定性を向上させるのに役立ちます。

#### 動作ルール

- メンテナンスウィンドウは週次サイクルです。
- 各データベースクラスターに個別に適用されます。
- 各クラスターには有効なメンテナンスウィンドウが1つのみあります。

#### 必要な権限

| 権限名 | 操作種別 | 説明 |
| --- | --- | --- |
| manageDatabase:Configuration | 編集 | データベース設定（パラメーター、スケーリング設定など）を変更する権限。 |

#### 主な機能

- 定期メンテナンスの日時を設定します。
- 運用中の編集をサポートします。
- 予定されているメンテナンスイベントを表示します。
- メンテナンス前に通知を送信します。

#### 手順

新しいクラスターを作成する際、ユーザーは**Additional Service**設定ステップでメンテナンスウィンドウを選択できます（詳細は[データベースの作成](/docs/ja/fpt-database-engine/managed-fpt-database-engines-new/)/を参照）。システムは最初の操作から設定を適用します。

使用中、ユーザーは以下の手順でメンテナンス時間枠を更新できます。

**ステップ1：機能へのアクセス**

FPT Cloud Portalにログインします。メインメニューから**Database Platform** → **All Database**を選択します。Database listページで、メンテナンスウィンドウを設定するクラスターのCluster IDを選択します。**Maintenance**タブを選択してメンテナンスウィンドウ設定管理画面を開きます。

[![](/img/migrated/view-maintenance-3d911f64.png)](/img/migrated/view-maintenance-3d911f64.png)

**ステップ2：メンテナンスウィンドウの設定**

**Maintenance**タブで、**Maintenance Window**の横にある**Configuration**アイコン（歯車アイコン）をクリックして**Update Maintenance Window**ダイアログを開きます。

[![](/img/migrated/update-maintenance-wwindow-d95e9f42.png)](/img/migrated/update-maintenance-wwindow-d95e9f42.png)

情報を入力します。

- **Day of Week**：週のメンテナンス実施日。**Monday**から**Sunday**までのオプションがあります。
- **Start Time**：メンテナンスを開始する時刻。

**Save**をクリックして設定を保存します。システムは次回以降のメンテナンス操作に設定を適用します。

**注意**：データベースクラスターのメンテナンスがまだスケジュールされていない場合のみ、スケジュール設定を更新できます。

**ステップ3：今後のメンテナンススケジュールの表示**

スケジュールされたメンテナンスイベントは**Pending Maintenance Event**セクションに表示されます。詳細情報には以下が含まれます。

- **Maintenance Event**：メンテナンスイベントの名前。
- **Type**：メンテナンスの種類。
- **Duration (Minute)**：メンテナンスの予定所要時間（分単位）。
- **Apply At**：メンテナンスが開始されるスケジュール時刻。

ユーザーはメンテナンススケジュールを定期的に監視して、運用を積極的に計画し、サービスへの影響を最小化する必要があります。

システムはメンテナンスアクティビティが行われる前にメール通知を送信します。現在、7日前と1日前の2つの通知ポイントがサポートされています。メンテナンス通知を受け取るメールリストを設定するには、**Notification**タブにアクセスし、**Maintenance**サービスの設定を調整します（詳細は[Notification](/docs/ja/fpt-database-engine/managed-fpt-database-engines-new/)/を参照）。
