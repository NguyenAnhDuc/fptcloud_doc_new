# データベース一覧の表示

**データベース一覧の表示**機能は、FPT Database Engine環境で稼働しているすべてのデータベースクラスターの概要を提供し、ユーザーが各データベースのステータス、設定を素早く監視し、管理操作を実行できるようにします。

データベースクラスターの一覧にアクセスするには、FPT Cloud Portalにログインします。メインメニューから**Database Platform > All Databases**を選択するか、ニーズに応じてデータベースグループに対応するメニューを選択します。

- **リレーショナルデータベース**：PostgreSQL、MySQL、MariaDB、Microsoft SQL Server。
- **NoSQL**：MongoDB、Cassandra、Redis。
- **検索エンジン**：OpenSearch。
- **データストリーミング**：Kafka。
- **時系列データベース**：TimescaleDB。
- **OLAP**：ClickHouse。

**Database List**画面には環境内の既存データベースの一覧が表示され、監視と管理のための稼働状態と主要情報が含まれます。

- **データベースが存在しない場合**：一覧は空で、データベース作成オプションが表示され、新しいデータベースクラスターをプロビジョニングできます。

[![](/img/migrated/dblist_nodb-scaled-af2686bc.png)](/img/migrated/dblist_nodb-scaled-af2686bc.png)

- **データベースが存在する場合**：VPC内にデプロイされたすべてのデータベースクラスターが概要と必要な管理操作（起動/停止/再起動/削除）とともに表示されます。

[![](/img/migrated/db-list-warning-a77a8c3c.png)](/img/migrated/db-list-warning-a77a8c3c.png)

一覧に表示される情報：

| 列名 | 説明 |
| --- | --- |
| ID | データベースのIDを表示します。**再起動警告**：データベースのコンピュートリソース（CPU/RAM）が調整されたが新しい設定を適用するための再起動がまだ行われていない場合、システムは警告アイコンを表示します。新しい設定を完全に適用してパフォーマンスと安定性を確保するため、ユーザーはトラフィックが少ない時間帯に再起動を実行してください。再起動が完了し新しい設定が適用されると警告は消えます。 |
| Name | データベース名を表示します。 |
| Type | データベースエンジンタイプを表示します。 |
| Tag | データベースに付けられたタグの一覧を表示します。 |
| Version | データベースのバージョンを表示します。ベータバージョンは「Beta」とラベル表示され、GA（一般提供）バージョンはラベルなしで通常表示され、非推奨バージョンは「Dep.」、サポート終了バージョンは「EoS」とラベル表示されます。 |
| Node | データベースクラスター内のノード数を表示します。 |
| Status | データベースの現在のステータス（Running、Stopped、Warningなど）を表示します。 |
| Backup | バックアップサービスの有効化ステータスを表示します。 |
| Vertical scaling | 垂直スケーリングサービスの有効化ステータスを表示します。 |
| Monitoring | 監視サービスの有効化ステータスを表示します。 |
| Created at | データベースの作成時刻を表示します。 |
| Action | データベースで利用可能な操作（起動、停止、再起動、削除、リサイズなど）を表示します。 |
