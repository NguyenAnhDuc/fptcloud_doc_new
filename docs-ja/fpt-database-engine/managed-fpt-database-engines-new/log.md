# イベントログ

FPT Database Engineの**Event Logs**機能により、ユーザーは2つの主要なログタイプを通じてデータベースシステムのすべての運用アクティビティを監視・追跡できます。

- **Process Log（Process Monitoring）** – システムタスクの詳細な処理フローを表示します。
- **Event Logs** – 内部のステップごとの処理詳細を表示せずに、データベースのライフサイクルと設定に関連する重要なイベントを記録します。

この機能は以下をサポートします。

- 操作が正常に完了したかどうかの確認。
- 変更履歴の追跡（監査証跡）。
- 管理操作のレビュー。

**Event Logs**ページにアクセスするには、FPT Cloud Consoleにログインします。メインメニューから「**Database Platform**」>「**All Databases**」を選択します。管理するデータベースクラスターのIDをクリック > 「**Event Logs**」タブを選択します。

- プロセスログを表示するには「**Process Monitoring**」タブを選択します。アクションを選択して「**View**」をクリックして結果を確認します。

[![](/img/migrated/view-process-monitor-scaled-85e128b6.png)](/img/migrated/view-process-monitor-scaled-85e128b6.png)

- イベントログを表示するには「**Event Logs**」タブを選択します。

[![](/img/migrated/view-event-log-scaled-61adb5ff.png)](/img/migrated/view-event-log-scaled-61adb5ff.png)
