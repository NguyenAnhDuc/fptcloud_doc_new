# Jenkins バージョンのアップグレード

FPT Cloud では、サポートされている新しいバージョンへ Jenkins インスタンスをアップグレードできます。現在 Portal でサポートされているバージョン：2.452、2.453、2.454。

**ステップ 1:** **Essential Properties** > **Instance Information** に移動します。

[![](/img/migrated/Picture89-ffa69e1a.png)](/img/migrated/Picture89-ffa69e1a.png)

**ステップ 2:** アップグレードするバージョンを選択します。

[![](/img/migrated/Picture90-bebb276a.png)](/img/migrated/Picture90-bebb276a.png)

**ステップ 3:** **Update** をクリックしてアップグレードを実行します。

[![](/img/migrated/Picture91-cd0fd45c.png)](/img/migrated/Picture91-cd0fd45c.png)

**ステップ 4:** アップグレード成功後の結果。

[![](/img/migrated/Picture92-5539c4c1.png)](/img/migrated/Picture92-5539c4c1.png)

:::note
バージョンアップグレードの完了には少し時間がかかります。
:::
