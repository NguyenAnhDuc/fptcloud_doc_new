# Jenkins インスタンスの削除

不要になった Jenkins インスタンスを削除するには：

**ステップ 1:** FPT Cloud メニュー > **Jenkins CI** > 削除するインスタンスを選択 > **Actions** > **Delete**。

[![](/img/migrated/Picture96-8e14d6a9.png)](/img/migrated/Picture96-8e14d6a9.png)

**ステップ 2:** **delete** と入力し、**Confirm** をクリックして削除を確認します。

[![](/img/migrated/Picture97-9e157809.png)](/img/migrated/Picture97-9e157809.png)

**ステップ 3:** インスタンス削除後の結果。
