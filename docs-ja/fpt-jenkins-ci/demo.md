# デモ

FPT Jenkins CI のデモ環境を使用すると、実際のプロジェクトに影響を与えることなく、Jenkins の機能を試すことができます。本ページでは、Jenkins インスタンスへの接続方法と基本的な操作手順を説明します。

## 概要

FPT Cloud 上にプロビジョニングされた Jenkins インスタンスに接続し、CI/CD パイプラインの動作を確認できます。デモ環境では、ビルド・テスト・デプロイの一連のフローを体験できます。

## Jenkins インスタンスへの接続手順

1. **ポータルにログイン**: FPT Cloud ポータル（[https://console.fptcloud.com](https://console.fptcloud.com)）にアクセスします。
2. **Jenkins CI メニューを選択**: 左サイドバーから「FPT Jenkins CI」を選択します。
3. **インスタンス一覧を確認**: プロビジョニング済みの Jenkins インスタンスが一覧表示されます。
4. **接続情報を取得**: 対象インスタンスの「詳細」をクリックし、URLおよびログイン認証情報を確認します。
5. **Jenkins にアクセス**: 表示された URL をブラウザで開き、提供された認証情報でログインします。

## 基本操作の確認

接続後、以下の操作をデモとして試すことができます。

- **新規ジョブの作成**: 「新しいジョブ」から Freestyle または Pipeline ジョブを作成
- **ビルドの実行**: ジョブページの「ビルドを実行」ボタンをクリック
- **ログの確認**: ビルド番号をクリックして「コンソール出力」を表示

詳細な設定については、Portal メニューの詳細ドキュメントをご参照ください。
