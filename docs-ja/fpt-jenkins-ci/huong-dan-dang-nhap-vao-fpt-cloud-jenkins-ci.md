# FPT Cloud Jenkins CI へのログイン

Portal で作成されたユーザー名とパスワードを使用して Jenkins インスタンスにログインします。

**ステップ 1:** **FPT Portal** メニュー > **DevOps** > **Jenkins CI** > **インスタンス名** をクリックしてクラスターの詳細を表示します。

[![](/img/migrated/Picture99-9e038c5d.png)](/img/migrated/Picture99-9e038c5d.png)

最初の画面には、URL やログイン認証情報など、Jenkins CI インスタンスへのアクセスに必要な詳細情報が表示されます。

[![](/img/migrated/Picture100-5653186e.png)](/img/migrated/Picture100-5653186e.png)

**ステップ 2:** Jenkins インスタンスにログインします。

**Essential Properties** 画面で URL をコピーしてアプリケーションを開きます：

- **URL:** Essential Properties 画面から取得します。
- **ユーザー名:** Essential Properties 画面からコピーします。
- **パスワード:** Essential Properties 画面からコピーします。

[![](/img/migrated/Picture101-956cda0d.png)](/img/migrated/Picture101-956cda0d.png)

ログイン成功後の画面：

[![](/img/migrated/Picture102-8ac75f23.png)](/img/migrated/Picture102-8ac75f23.png)

:::note
Jenkins システム上で管理者のユーザー名またはパスワードを直接変更した場合、Portal に表示されている認証情報は無効になり、システムへの認証に使用できなくなります。
:::
