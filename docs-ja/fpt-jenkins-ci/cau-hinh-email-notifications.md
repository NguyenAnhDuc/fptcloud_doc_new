# メール通知の設定

FPT Cloud では、Portal インターフェースを通じて Jenkins インスタンスのメール通知を設定できます。

### メール設定の編集

**ステップ 1:** FPT Portal メニュー > **Jenkins CI** > **Detail Cluster** > **Configurations** > メールの **Edit** をクリックします。

[![](/img/migrated/Picture68-f44a375a.png)](/img/migrated/Picture68-f44a375a.png)

**ステップ 2:** メール設定の更新情報を入力し、**Update** をクリックします。

[![](/img/migrated/Picture69-333db764.png)](/img/migrated/Picture69-333db764.png)

**ステップ 3:** メール設定更新後の結果。
