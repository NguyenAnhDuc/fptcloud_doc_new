# FPT Jenkins CI の新規作成

**ステップ 1:** FPT Portal メニュー > **DevOps** > **Jenkins CI** > **Create Jenkins CI** をクリックします。

[![](/img/migrated/Picture1-38ff2a6f.png)](/img/migrated/Picture1-38ff2a6f.png)

**ステップ 2:** 新規作成フォームに以下の情報を入力します。

[![](/img/migrated/Picture2-1-9c7a0368.png)](/img/migrated/Picture2-1-9c7a0368.png)

- **Instance name:** インスタンス名を入力します。
- **Kubernetes version:** Kubernetes のバージョンを選択します。
- **Service package:** リソースパッケージを選択します。

**ステップ 3:** **Create** をクリックしてインスタンスを作成します。システムが作成を開始し、Portal に結果が表示されます。

[![](/img/migrated/Picture3-1-47e2570d.png)](/img/migrated/Picture3-1-47e2570d.png)

**ステップ 4:** インスタンスのステータスが **Succeeded** に変わったことを確認します。

[![](/img/migrated/Picture4-48e1c05d.png)](/img/migrated/Picture4-48e1c05d.png)
