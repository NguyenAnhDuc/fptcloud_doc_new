# 初期設定

FPT Jenkins CI を使用する前に、以下の初期設定を行ってください。

**ステップ 1:** FPT Portal にログインし、**DevOps** > **Jenkins CI** に移動します。

**ステップ 2:** サービスパッケージの情報を確認します。

**ステップ 3:** FPT Jenkins CI インスタンスを新規作成します。

**ステップ 4:** インスタンスの詳細情報を確認し、Jenkins にログインします。
