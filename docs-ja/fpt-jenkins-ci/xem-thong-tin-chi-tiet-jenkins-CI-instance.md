# Jenkins CI インスタンスの詳細確認

既存の Jenkins クラスターを操作するには、まず詳細情報を確認します。

**ステップ 1:** **FPT Portal** メニュー > **DevOps** > **Jenkins CI** > **インスタンス名** をクリックしてクラスターの詳細を表示します。

[![](/img/migrated/Picture5-1-e44d4a18.png)](/img/migrated/Picture5-1-e44d4a18.png)

最初の画面には、URL やログイン認証情報など、Jenkins CI インスタンスにアクセスするための詳細情報が表示されます。

[![](/img/migrated/Picture6-69dec98b.png)](/img/migrated/Picture6-69dec98b.png)

**ステップ 2:** **Jenkins インスタンス** にログインします。

**Essential Properties** 画面で URL をコピーしてアプリケーションを開きます。

[![](/img/migrated/Picture7-1-c8ec9386.png)](/img/migrated/Picture7-1-c8ec9386.png)

Jenkins へのログイン成功後の画面：
