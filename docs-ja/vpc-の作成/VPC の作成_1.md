# Vpc の作成_1

新しいVPCを作成するには、テナント所有者権限でシステムにログインする必要があります。
**ステップ1** ：VPC選択メニューにて[VPCの作成]を選択します 。
[![file](/img/migrated/image-1744258756717-b8306892.png)](/img/migrated/image-1744258756717-b8306892.png)
ステップ2：以下の情報を入力します。
  * VPC 名：VPC名称を 10 文字以上50 文字以下 (大文字、数字、ダッシュ、アンダースコアを含む) で入力します。小文字はシステムによって自動的に大文字に変換されます。
  * [VPC 所有者]：VPC 所有者権限が付与されるアカウントを選択します。なおシステムはプロセスを迅速化するために、デフォルトでVPCの作成を行っているアカウントをVPC 所有者権限の付与先に選択します。
  * ネットワーク設定（任意）：関連のサブネット情報を設定します。任意の設定で、VPC作成後にも設定できます。
    * 名称：サブネットの名前を入力します。
    * タイプ：現在はルーティング済みと分離の2つのタイプをサポートしています。
    * ネットワークアドレス（CIDR）：有効なCIDRを入力します。
    * ゲートウェイ IP：有効なゲートウェイ IP アドレスを入力します。
    * 静的IPプール(（任意）：使用するIP範囲を入力します。指定しない場合はCIDRからIP全体を取得します。

[![file](/img/migrated/image-1744258458944-38445575.png)](/img/migrated/image-1744258458944-38445575.png)
**ステップ3** ：[VPC作成]にクリックしてVPCを作成します。その後システムが初期化され、結果が通知されます。VPCが正常に作成された場合、VPCページに表示されます。
[![file](/img/migrated/image-1744258774497-f77ee8ce.png)](/img/migrated/image-1744258774497-f77ee8ce.png)
