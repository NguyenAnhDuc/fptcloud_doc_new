# VPCの名前変更

:::warning
VPCの名前変更は内部構成に影響を与える可能性があります。必要な場合は、ホットライン 0800 300 9739 またはメール support@fptcloud.jp にてサポートを受けてください。
:::

1. VPC一覧ページで、名前変更したいVPCの **Actions** → **Rename VPC** を選択します。

   ![Rename VPCオプションのあるActionsメニュー](/img/migrated/image-1726823026645-ceaabb35.png)

2. 新しいVPC名を入力し、**Rename** をクリックします。

   ![Rename VPCダイアログ](/img/migrated/image-1726823062656-852dc78b.png)
