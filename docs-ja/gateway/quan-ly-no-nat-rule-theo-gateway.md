# GatewayのNO NAT Rule管理

## NO NAT Ruleリストの確認

[![NO NAT Ruleリスト画面](/img/migrated/image-1754291539835-89c7d41b.png)](/img/migrated/image-1754291539835-89c7d41b.png)

## NO NAT Ruleの作成

[![NO NAT Rule作成フォーム](/img/migrated/image-1754291548065-9e866aa4.png)](/img/migrated/image-1754291548065-9e866aa4.png)

NO NATには2種類あります：

- **NO Inbound NAT**
- **NO Outbound NAT**

## NO NAT Ruleの編集

**Actions**列の**Edit**ボタンをクリックして、NO NAT Rule情報を編集します。

## NO NAT Ruleの削除

**Actions**列の**Delete**ボタンをクリックして、NO NAT Ruleを削除します。
