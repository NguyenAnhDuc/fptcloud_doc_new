# Huong Dan

ステップ1: Đăng nhập console.fptcloud.com
ステップ2: ログイン成功後, chọn VPC cần xem dữ liệu Gateway
ステップ3: Ở sidebar, chọn và mở rộng menu Network → Chọn Menu Gateway
注意: Tính năng này chỉ dành riêng cho một số Tenant có cấu hình đặc biêt, vui lòng liên hệ để được hỗ trợ.
