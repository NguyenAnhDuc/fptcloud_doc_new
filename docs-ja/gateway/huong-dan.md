# Huong Dan

ステップ 1: Đăng nhập console.fptcloud.com
ステップ 2: Sau khi đăng nhập thành công, chọn VPC cần xem dữ liệu Gateway
ステップ 3: Ở sidebar, chọn và mở rộng menu Network → Chọn Menu Gateway
注意: Tính năng này chỉ dành riêng cho một số Tenant có cấu hình đặc biêt, vui lòng liên hệ để được hỗ trợ.
