# ArgoCDバージョンのアップグレード

FPT Cloudはサポートされているバージョンリストから新しいArgoCDバージョンへのアップグレードをサポートしています。現在PortalはバージョンをサポートしていますL v2.8.1、v2.8.2、v2.8.3。

:::warning
アップグレードにより、使用中のArgoCDサービスのダウンタイムが発生する可能性があります。実行前に適切なタイミングを計画してください。
:::

1. **Essential Properties** > **Instance Information**に移動します。

[![](/img/migrated/Picture79-fddde53e.png)](/img/migrated/Picture79-fddde53e.png)

2. アップグレードするバージョンを選択します。

[![](/img/migrated/Picture80-1a5785f9.png)](/img/migrated/Picture80-1a5785f9.png)

3. **Update**をクリックしてアップグレードを実行します。
