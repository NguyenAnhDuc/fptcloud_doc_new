# Secretの管理

この機能により、ArgoCDの`argocd-image-updater-secret`ファイルにsecretデータを追加できます。

## A. 新しいsecretを追加します

1. **Image Updater** > **Secret**画面で、新しいsecretを追加するためにクリックします。

[![](/img/migrated/Picture9-7-f0b6dc09.png)](/img/migrated/Picture9-7-f0b6dc09.png)

2. secret情報を入力します。

[![](/img/migrated/Picture10-7-8beb9d04.png)](/img/migrated/Picture10-7-8beb9d04.png)

3. secret作成後の結果：

[![](/img/migrated/Picture11-7-0e6c02c4.png)](/img/migrated/Picture11-7-0e6c02c4.png)

## B. Secretを更新します

1. **Image Updater** > **Secret**画面で、更新するsecretを選択します。

[![](/img/migrated/Picture12-6-08eae6e2.png)](/img/migrated/Picture12-6-08eae6e2.png)

2. secret情報を更新します。

[![](/img/migrated/Picture13-6-9b05fe0c.png)](/img/migrated/Picture13-6-9b05fe0c.png)

3. 更新後の結果：

[![](/img/migrated/Picture14-6-6e5c43d8.png)](/img/migrated/Picture14-6-6e5c43d8.png)

## C. Secretを削除します

:::danger
この操作は元に戻せません。
:::

1. **Image Updater** > **Secret**画面で、削除するsecretを選択します。

[![](/img/migrated/Picture15-6-a77bbc61.png)](/img/migrated/Picture15-6-a77bbc61.png)

2. 削除するsecretを選択して確認します。

[![](/img/migrated/Picture16-6-2cdcf902.png)](/img/migrated/Picture16-6-2cdcf902.png)

3. 削除後の結果：

[![](/img/migrated/Picture17-6-b2c862d9.png)](/img/migrated/Picture17-6-b2c862d9.png)
