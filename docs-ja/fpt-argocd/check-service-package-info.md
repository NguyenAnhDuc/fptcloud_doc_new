# サービスパッケージ情報の確認

FPT CloudはArgoCD サービスパッケージを**Quota Detail**で提供しています。

1. FPT Portalメニューで**Argo CD** > **View Details**を選択して、利用可能なFPT ArgoCDサービスパッケージを確認します。

[![](/img/migrated/Picture2-6-053e4ca3.png)](/img/migrated/Picture2-6-053e4ca3.png)

2. Portalに利用可能なサービスパッケージが表示されます。

[![](/img/migrated/Picture3-7-3e2895dc.png)](/img/migrated/Picture3-7-3e2895dc.png)

デフォルトでは、**Basic**パッケージでサービスをアクティベートできます。より高いパッケージにアップグレードするには、Salesにお問い合わせください。
