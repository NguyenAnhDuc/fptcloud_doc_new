# KustomizeのBuild optionsの設定

1. **Configuration** > **Kustomize** > **Update**画面に移動します。

[![](/img/migrated/Picture120-2-2f369b3e.png)](/img/migrated/Picture120-2-2f369b3e.png)

2. 希望するbuild optionsを入力します。

例：`--enable-helm`を入力します

:::note
オプションはスペースで区切ります。
:::

[![](/img/migrated/Picture121-2-aa0525af.png)](/img/migrated/Picture121-2-aa0525af.png)

3. 保存ボタンをクリックして設定を適用します。

[![](/img/migrated/Picture122-2-06ab4bf2.png)](/img/migrated/Picture122-2-06ab4bf2.png)
