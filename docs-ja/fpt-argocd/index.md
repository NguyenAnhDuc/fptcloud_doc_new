# FPT ArgoCD

FPT Smart CloudはFPT Cloud Portal上でManaged ArgoCDサービスを提供しています。このサービスにより、Kustomize、manifest、Helm chart、その他の方法を使用してKubernetes上にアプリケーションをデプロイできます。統合インターフェースを通じて、設定、バージョン、アカウントを含むArgoCD instanceを管理できます。
