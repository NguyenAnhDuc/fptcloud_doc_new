# OIDC configの削除

:::danger
削除後、OIDC ScopeはDisable状態に戻ります。ArgoCDはadminアカウントが有効またはローカルアカウントが存在する場合にのみ、username/passwordでのログインをサポートします。
:::

1. **Security & Access** > **OIDC Scope** 画面で、削除するOIDC Scopeの **Delete** をクリックします。

[![](/img/migrated/Picture30-3-7c37461f.png)](/img/migrated/Picture30-3-7c37461f.png)

2. アクションを確認するために `delete` と入力します。

[![](/img/migrated/Picture31-3-c8f7c979.png)](/img/migrated/Picture31-3-c8f7c979.png)
