# ArgoCD Image Updaterのenable/disable

この機能により、Portalインターフェースを通じてArgoCD Image Updaterを有効または無効にできます。デフォルトではサービスは有効になっています。

1. **Image Updater** > **Service Status**画面でサービスをenable/disableします。
