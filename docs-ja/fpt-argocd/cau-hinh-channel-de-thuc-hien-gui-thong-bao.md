# 通知送信チャンネルの設定

FPT Cloudでは以下の通知チャンネルの追加・編集が可能です：

**_A. Telegram_**

_1. Telegram設定の新規追加_

- **ステップ1**：**Notifications** > **Telegram** 画面に移動します：

[![](/img/migrated/Picture38-3-ecf99d40.png)](/img/migrated/Picture38-3-ecf99d40.png)

- **ステップ2**：Telegramへの通知送信用TokenをI設定します。

[![](/img/migrated/Picture39-3-5750e082.png)](/img/migrated/Picture39-3-5750e082.png)

- **ステップ3**：設定後の結果：

[![](/img/migrated/Picture40-1-d2f2c786.png)](/img/migrated/Picture40-1-d2f2c786.png)

_2. Telegram設定の編集_

また、以下の手順でTelegram設定を編集できます：

- **ステップ1**：**Notifications** > **Telegram** 画面に移動します：

[![](/img/migrated/Picture41-1-7e331436.png)](/img/migrated/Picture41-1-7e331436.png)

- **ステップ2**：Telegramへの通知送信用Tokenを再設定します。

[![](/img/migrated/Picture42-1-ffe0b22f.png)](/img/migrated/Picture42-1-ffe0b22f.png)

- **ステップ3**：設定後の結果：

[![](/img/migrated/Picture43-1-2262a51b.png)](/img/migrated/Picture43-1-2262a51b.png)

_3. Telegram設定の削除_

設定を削除するには：

- **ステップ1**：**Notifications** > **Telegram** 画面に移動します：

[![](/img/migrated/Picture44-1-8b6f543d.png)](/img/migrated/Picture44-1-8b6f543d.png)

- **ステップ2**：確認テキストを入力してTelegramチャンネルを削除します：

[![](/img/migrated/Picture45-1-ced26400.png)](/img/migrated/Picture45-1-ced26400.png)

- **ステップ3**：Telegramチャンネル削除後の結果：

[![](/img/migrated/Picture46-1-1d1b4ec2.png)](/img/migrated/Picture46-1-1d1b4ec2.png)
