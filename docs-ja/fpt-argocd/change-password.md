# アカウントパスワードの変更

:::note
パスワードを忘れた場合のみパスワードを変更できます。現在のパスワードはPortalからコピーできません。
:::

1. **Security & Access** > **Local Account** > **Change Password**画面に移動します。

[![](/img/migrated/Picture88-2d97201c.png)](/img/migrated/Picture88-2d97201c.png)

2. 新しいパスワードを入力します。

[![](/img/migrated/Picture89-37eec235.png)](/img/migrated/Picture89-37eec235.png)

3. **Change Password**をクリックして完了します。
