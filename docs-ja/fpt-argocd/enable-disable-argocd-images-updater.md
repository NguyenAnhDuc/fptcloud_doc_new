# ArgoCD Image Updaterの有効化/無効化

この機能により、Portalインターフェースを通じてArgoCD Image Updaterを有効化または無効化できます。サービスはデフォルトで有効です。

1. **Image Updater** > **Service Status** 画面でサービスを有効化または無効化します。
