# Getting Started

To create a new ArgoCD instance, you can do the following steps: 
**Step 1:** Select **ArgoCD** on **FPT Cloud Portal** menu
[![](/img/migrated/Picture1-1-5dc58dbd.png)](/img/migrated/Picture1-1-5dc58dbd.png)
**Step 2:** Select **Create Instance** > Fill out the information to create a **ArgoCD** instance.
[![](/img/migrated/Picture2-1-1cd926ba.png)](/img/migrated/Picture2-1-1cd926ba.png)
**Step 3:** Select **Create Instance** and the system will initialize an ArgoCD Instance. It will notify you about the processing status
[![](/img/migrated/Picture3-1-f4accb41.png)](/img/migrated/Picture3-1-f4accb41.png)
After successfully creating an **ArgoCD instance** , the cluster status will change to **Succeed** , then users can access the ArgoCD instance via the URL and account/password in details views of the instance.
[![](/img/migrated/Picture4-1-b767bf84.png)](/img/migrated/Picture4-1-b767bf84.png)
