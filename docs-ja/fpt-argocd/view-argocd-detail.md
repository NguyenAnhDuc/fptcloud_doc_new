# ArgoCD インスタンスの詳細確認とログイン

## 詳細情報を確認する

1. **Instance Name** をクリックして、インスタンスの詳細情報を確認します。

[![](/img/migrated/Picture58-fd48905e.png)](/img/migrated/Picture58-fd48905e.png)

画面にはArgoCD クラスターへのアクセスに必要な詳細情報（URL、ユーザー名、ログインパスワード）が表示されます。

[![](/img/migrated/Picture59-1438cb52.png)](/img/migrated/Picture59-1438cb52.png)

## ArgoCD クラスターにログインする

2. **Essential Properties** 画面でURLをコピーしてアプリケーションにアクセスします:
   - **URL**: **Essential Properties** 画面から取得
   - **Username**: **Essential Properties** 画面から取得
   - **Password**: **Essential Properties** 画面から取得

[![](/img/migrated/Picture60-3afa2ba7.png)](/img/migrated/Picture60-3afa2ba7.png)

ログイン成功後の画面:
