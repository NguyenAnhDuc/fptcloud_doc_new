# Dedicated FPT Kubernetes Engine (DFKE)

Dedicated FPT Kubernetes Engine (DFKE) は FPT Cloud 上で提供されるマネージド Kubernetes サービスです。node、ネットワーク、ストレージを完全に制御できる専用 cluster を提供します。ユーザーは FPT Cloud Portal から Kubernetes cluster の作成、スケール、アップグレード、管理を行うことができます。

D-FKE では、お客様が Master Nodes（control plane コンポーネントを含む）と Worker Nodes を含む Kubernetes cluster 全体を管理します。autoscale、service load balancer、バージョンアップグレード、persistent volume などの機能も完全にサポートされています。
