# Kubernetes クラスターの管理

Dedicated – FPT Kubernetes Engine (D-FKE) では、Kubernetes クラスターのライフサイクル全体を管理できます。

このセクションでは以下の操作を説明します。

- 新しい Kubernetes クラスターの作成
- 作成済みクラスターの一覧表示
- クラスターの詳細情報へのアクセス
- クラスターアクセス情報の取得
- クラスターの削除
- クラスターの停止
- クラスターの起動
