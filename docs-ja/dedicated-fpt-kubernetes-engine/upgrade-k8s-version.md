# Kubernetes バージョンのアップグレード

1. メニューから **Kubernetes** を選択します。**Kubernetes Management** ページが表示されます。
[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)
2. 一覧からバージョンを変更したい **Cluster** を選択します。
[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)
3. **Essential Properties** タブを選択し、**Cluster Information** > **Version** の **Setting** アイコンをクリックします。
[![](/img/migrated/Picture50-471159a1.png)](/img/migrated/Picture50-471159a1.png)
4. アップグレードするバージョンを選択し、**Upgrade** をクリックします。
[![](/img/migrated/Picture51-82e15c60.png)](/img/migrated/Picture51-82e15c60.png)

:::note
バージョンのアップグレードのみ可能です。ダウングレードはサポートされていません。
:::
