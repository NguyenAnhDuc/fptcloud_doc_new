# K8s バージョンのアップグレード

**手順 1**: FPT Cloud Portal のメニューで **Kubernetes** を選択します。**Kubernetes Management** ページが表示されます。

[![](/img/migrated/50-8aa6c97f.png)](/img/migrated/50-8aa6c97f.png)

**手順 2**: 一覧から K8s バージョンをアップグレードしたい Cluster を選択します。

[![](/img/migrated/51-9e351d4a.png)](/img/migrated/51-9e351d4a.png)

**手順 3**: **Essential Properties** タブ > **Cluster Information** > **Version** で、**Setting** アイコンをクリックします。

[![](/img/migrated/52-f135b93a.png)](/img/migrated/52-f135b93a.png)

**手順 4**: アップグレード先の **Version** を選択し、**Upgrade** をクリックします。

[![](/img/migrated/53-0f2d1cc1.png)](/img/migrated/53-0f2d1cc1.png)

:::note
バージョンのアップグレードのみ可能です。ダウングレードはできません。
:::
