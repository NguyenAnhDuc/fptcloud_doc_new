# Quan Ly Kubernetes Cluster

Managed FPT Kubernetes Engine hỗ trợ các thao tác quản lý cluster sau:

- Tạo mới Kubernetes cluster
- Xem danh sách Kubernetes cluster đã tạo
- Truy cập thông tin chi tiết của cluster
- Lấy thông tin truy cập cluster
- Rotate kubeconfig
- Xoá Kubernetes cluster
