# K8S バージョンのアップグレード

**ステップ 1:** メニューで **Kubernetes** を選択すると、**Kubernetes Management** ページが表示されます。K8s バージョンをアップグレードしたい **Cluster** を選択します。

[![](/img/migrated/Picture41-d322b494.png)](/img/migrated/Picture41-d322b494.png)

**ステップ 2:** **Cluster Information** > **Version** セクションで **Setting** アイコンを選択します。

[![](/img/migrated/Picture42-6db86334.png)](/img/migrated/Picture42-6db86334.png)

**ステップ 3:** アップグレードしたい **Version** を選択し、**Upgrade** を選択します。

[![](/img/migrated/Picture7-3-d45ba609.png)](/img/migrated/Picture7-3-d45ba609.png)

_注意: バージョンのアップグレードのみ可能で、ダウングレードはできません。_
_処理中の問題を避けるため、バージョンは順番にアップグレードすることを推奨します。_
