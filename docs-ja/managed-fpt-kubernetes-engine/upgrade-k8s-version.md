# Kubernetes バージョンのアップグレード

**ステップ 1:** メニューから **Kubernetes** を選択して **Kubernetes Management** ページを開きます。Kubernetes バージョンをアップグレードしたい cluster を選択します。

[![](/img/migrated/24-1-4f6944fa.png)](/img/migrated/24-1-4f6944fa.png)

**ステップ 2:** **General information** > **Version** セクションで、設定アイコンをクリックします。

[![](/img/migrated/25-1-dcc21c83.png)](/img/migrated/25-1-dcc21c83.png)

**ステップ 3:** 希望するバージョンを選択し、**Upgrade** をクリックします。

[![](/img/migrated/26-1-84cc9b69.png)](/img/migrated/26-1-84cc9b69.png)

:::note
バージョンのアップグレードのみ可能で、ダウングレードはできません。問題を避けるため、バージョンを順番にアップグレードしてください。
:::
