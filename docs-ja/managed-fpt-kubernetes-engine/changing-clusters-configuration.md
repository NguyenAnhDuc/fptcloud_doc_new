# K8s Cluster の設定変更

K8s Cluster の設定を変更するには、以下の操作が利用できます。

- Cluster Endpoint Access の変更
- ワーカーグループの追加
- Internal subnet Load Balancer (CIDR) の設定変更
- ワーカーグループの label/taint の編集
- Node Auto-repair の有効化/無効化
- ベースワーカーグループの切り替え
- K8S バージョンのアップグレード
- Cluster Autoscale
