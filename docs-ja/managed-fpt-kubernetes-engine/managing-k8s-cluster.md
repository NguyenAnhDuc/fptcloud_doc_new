# Kubernetes Cluster の管理

Managed FPT Kubernetes Engine では、以下の操作で Kubernetes Cluster を管理できます。

- 新しい Kubernetes Cluster の作成
- 作成済み Kubernetes Cluster の一覧表示
- クラスターの詳細情報へのアクセス
- クラスターのアクセス情報の取得
- kubeconfig の Rotate
- Kubernetes Cluster の削除
