# kubeconfig のローテーション機能

Kubernetes クラスターの使用中に、クラスター内の admin ユーザーのトークンを変更したい場合は kubeconfig のローテーションを実行できます。

**ステップ 1:** メニューで **Kubernetes** を選択すると、**Kubernetes Management** ページが表示されます。

[![](/img/migrated/Picture15-31f6358c.png)](/img/migrated/Picture15-31f6358c.png)

**ステップ 2:** **Essential Properties** タブの **Configuration** セクションで **Renew kubeconfig** を選択します。

[![](/img/migrated/Picture16-abcd6aa3.png)](/img/migrated/Picture16-abcd6aa3.png)

**ステップ 3:** ポップアップの警告情報を確認し、**Renew** ボタンをクリックします。

[![](/img/migrated/Picture17-4bc52df3.png)](/img/migrated/Picture17-4bc52df3.png)
