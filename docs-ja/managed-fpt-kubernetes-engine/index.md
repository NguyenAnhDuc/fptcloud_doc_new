# Managed FPT Kubernetes Engine

Managed FPT Kubernetes Engine (MFKE)はFPT Cloudが提供するフルマネージドKubernetesサービスです。

MFKEを利用することで、インフラの管理を最小限に抑えながら、Kubernetesクラスターを簡単に作成・運用できます。FPT CloudはネイティブKubernetesをベースとしており、AWS・Azure・GCPなど他のクラウドプラットフォームとの互換性も確保されています。

## 主な機能

- Kubernetes クラスターの作成・管理
- ワーカーグループの追加・設定変更
- クラスター autoscale
- バージョン自動アップグレード
- Persistent Storage (CSI)
- Hibernate & Wake-up
- セキュリティ強化機能（Benchmark、Runtime Security、Workload Scan、Audit Log）
- Backup & Restore
- GPU サポート
