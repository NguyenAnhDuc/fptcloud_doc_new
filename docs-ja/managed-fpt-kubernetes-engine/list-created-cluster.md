# 作成済み Kubernetes Cluster の一覧表示

**Kubernetes Management** ページで作成済みの Kubernetes Cluster の一覧を表示・管理できます。**Kubernetes Management** を開くには、以下の手順で操作します。

**FPT Portal** のメニューで **Kubernetes** を選択します。**Managed** タブを選択します。システムが作成済み Cluster の一覧を表示します。**Name**、**Version**、**Worker Group**、**Type**、**Status**、**Created at**、**Actions** などの重要な情報が表示されます。
