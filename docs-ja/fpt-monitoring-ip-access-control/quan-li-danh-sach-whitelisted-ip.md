# ホワイトリスト IP 一覧を管理する

IP Access Control 機能は、ワークスペースの FPT Cloud Monitoring システムコンポーネントごとにホワイトリスト IP を管理します:

- Grafana dashboard URL
- Metric datasource URL
- Log datasource URL
- Trace datasource URL

各コンポーネントは独立した IP ホワイトリストテーブルを持ちます。
