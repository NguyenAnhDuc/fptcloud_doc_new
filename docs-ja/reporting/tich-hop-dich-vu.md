# サービス連携

この管理ツールを使用するには、まずFPT Cloud Monitoringのサポート管理チームに連絡して、ワークスペースにReportingサービスを**連携**してもらう必要があります。

連携が完了したら、ダッシュボードから直接サービスをご利用いただけます。

ご利用をご希望の際は、お気軽にお問い合わせください。
