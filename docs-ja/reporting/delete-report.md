# Xóa report

Để xóa report, người dùng thực hiện như sau:
**ステップ 1** : Click **Home > Apps > Reporting**
**ステップ 2** : Tại màn danh sách report, click icon **Trash**
[![](/img/migrated/Picture15-c1a6f8b0.png)](/img/migrated/Picture15-c1a6f8b0.png)
**ステップ 3** : Xuất hiện hộp thoại xác nhận, chọn **Delete** để xóa report, chọn **Cancel** để hủy bỏ thao tác 
[![](/img/migrated/Picture16-19c4a8c7.png)](/img/migrated/Picture16-19c4a8c7.png)
