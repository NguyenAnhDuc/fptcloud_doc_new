# レポートの削除

1. **Home > Apps > Reporting** に移動します。
2. レポート一覧で、削除したいレポートの**ゴミ箱**アイコンをクリックします。

   [![ゴミ箱アイコン付きレポート一覧](/img/migrated/Picture15-c1a6f8b0.png)](/img/migrated/Picture15-c1a6f8b0.png)

3. 確認ダイアログが表示されます。**Delete** をクリックして確認するか、**Cancel** をクリックして中止します。

   [![削除確認ダイアログ](/img/migrated/Picture16-19c4a8c7.png)](/img/migrated/Picture16-19c4a8c7.png)
