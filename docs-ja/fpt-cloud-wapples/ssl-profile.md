# SSL プロファイルの保存と表示

SSL プロファイルは、FPT Cloud WAPPLES（Web Application Firewall）における SSL/TLS 通信の設定をテンプレートとして保存・再利用するための機能です。証明書や暗号化設定を一元管理することで、複数のサービスに対して一貫したセキュリティポリシーを適用できます。

## 概要

SSL プロファイルを使用することで、SSL 証明書のアップロード、暗号スイートの設定、プロトコルバージョンの指定などをひとつのプロファイルとして保存できます。保存したプロファイルは複数のウェブサービスに適用可能で、設定の変更も一箇所で管理できます。

## SSL プロファイルの操作

### 新規プロファイルの追加

1. WAPPLES 管理画面の「SSL プロファイル」ページに移動します。
2. 画面右上の **+（プラス）アイコン** をクリックします。
3. プロファイル名、SSL 証明書、秘密鍵を入力します。
4. 必要に応じて暗号スイートやプロトコルバージョンを設定します。
5. 「保存」ボタンをクリックして設定を保存します。

### プロファイルの確認と編集

保存済みのSSLプロファイルは一覧形式で表示されます。各プロファイルの詳細を確認するには、対象の行をクリックしてください。設定の変更や証明書の更新もこの画面から行えます。

## 注意事項

- SSL 証明書の有効期限を定期的に確認し、期限切れ前に更新してください。
- 秘密鍵は安全に管理し、不要な共有を避けてください。
