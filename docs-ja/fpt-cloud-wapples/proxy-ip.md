# Proxy IP

Chức năng thực hiện thiết lập địa chỉ proxy IP nếu có sử dụng nhằm thiết lập các dịch vụ của ip private trong cloud [![file](/img/migrated/image-1714112480774-859e0576.png)](/img/migrated/image-1714112480774-859e0576.png)
  * Các chức năng có thể sử dụng bao gồm thêm, sửa, xoá Proxy IP

  1. Thêm IP Proxy
     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn vào biểu tượng + để thêm Proxy IP
     * Nhập vào các giá trị cần thiết để thiết lập IP như hình và chọn Apply [![file](/img/migrated/image-1714112494432-ca5d7a26.png)](/img/migrated/image-1714112494432-ca5d7a26.png)
     * Chọn **Save** để lưu thay đổi [![file](/img/migrated/image-1714112501548-e3275d5b.png)](/img/migrated/image-1714112501548-e3275d5b.png)
  2. Sửa IP Proxy
     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn vào Modify
     * Thực hiện sửa đổi các thông tin cần thiết: [![file](/img/migrated/image-1714112513079-760a45f0.png)](/img/migrated/image-1714112513079-760a45f0.png)
     * Nhấn chọn Apply và chọn Save để lưu thiết lập [![file](/img/migrated/image-1714115110737-1154d081.png)](/img/migrated/image-1714115110737-1154d081.png)
  3. Xoá IP Proxy
     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn vào ip cần xoá [![file](/img/migrated/image-1714115123188-4dfc9591.png)](/img/migrated/image-1714115123188-4dfc9591.png)
     * Chọn vào biểu tượng thùng rác để thực hiện xoá ip đã chọn và chọn **Save** nhằm thực hiện lưu lại thiết lập. [![file](/img/migrated/image-1714115133281-726201bd.png)](/img/migrated/image-1714115133281-726201bd.png)
