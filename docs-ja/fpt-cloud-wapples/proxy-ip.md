# Proxy IP

クラウド内のプライベートIPサービスを設定するために、使用している proxy IPアドレスを設定する機能です。 [![file](/img/migrated/image-1714112480774-859e0576.png)](/img/migrated/image-1714112480774-859e0576.png)

Proxy IP の追加、編集、削除が可能です。

1. **Proxy IP の追加**
   - Security Settings / Networks Settings / Proxy IP ページで、`+` アイコンをクリックして Proxy IP を追加します。
   - 図のように必要な値を入力してIPを設定し、Apply をクリックします。 [![file](/img/migrated/image-1714112494432-ca5d7a26.png)](/img/migrated/image-1714112494432-ca5d7a26.png)
   - **Save** をクリックして変更を保存します。 [![file](/img/migrated/image-1714112501548-e3275d5b.png)](/img/migrated/image-1714112501548-e3275d5b.png)

2. **Proxy IP の編集**
   - Security Settings / Networks Settings / Proxy IP ページで、Modify をクリックします。
   - 必要な情報を編集します。 [![file](/img/migrated/image-1714112513079-760a45f0.png)](/img/migrated/image-1714112513079-760a45f0.png)
   - Apply をクリックし、Save をクリックして設定を保存します。 [![file](/img/migrated/image-1714115110737-1154d081.png)](/img/migrated/image-1714115110737-1154d081.png)

3. **Proxy IP の削除**
   - Security Settings / Networks Settings / Proxy IP ページで、削除したいIPを選択します。 [![file](/img/migrated/image-1714115123188-4dfc9591.png)](/img/migrated/image-1714115123188-4dfc9591.png)
   - ゴミ箱アイコンをクリックして選択したIPを削除し、**Save** をクリックして設定を保存します。 [![file](/img/migrated/image-1714115133281-726201bd.png)](/img/migrated/image-1714115133281-726201bd.png)
