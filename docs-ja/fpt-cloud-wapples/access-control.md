# Access Control

スパミングやクリッキングなどの不正なアクセス行為を防ぐためのアクセス制御機能です。

  * Access Control を追加します [![file](/img/migrated/image-1714116678455-4855ca28.png)](/img/migrated/image-1714116678455-4855ca28.png)
  * 名前を設定します
  * ソース IP を設定します
  * 宛先 IP を設定します
  * 宛先情報（Host: ドメイン、Port、Method: GET・POST など、URL: サブドメイン）を設定します [![file](/img/migrated/image-1714116684905-93057588.png)](/img/migrated/image-1714116684905-93057588.png)
  * 行為が発生してからアクセスをブロックするまでの時間、ブロックを実行するための最大行為数、最大行為数が許容される時間範囲を設定します
  * [Source IP / Destination IP / URL / Host / Method] で選択したフィルターは「,」で区切ります
