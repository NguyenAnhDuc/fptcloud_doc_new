# ネットワークフィルタースケジューリング

ネットワークフィルター内のセキュリティグループの実行スケジュールを設定します。

入力が必要なフィールド：
  * **Schedule name**: スケジュール名
  * **Schedule description**: スケジュールの説明
  * **Day of Week**: 実行する曜日を選択します。個別の曜日、または平日・週末の一括選択が可能です
  * **Start Time と End Time**: セキュリティグループをその日に有効化する開始時刻と終了時刻 [![file](/img/migrated/image-1714115260868-de7cc26d.png)](/img/migrated/image-1714115260868-de7cc26d.png) [![file](/img/migrated/image-1714115269843-844c70ba.png)](/img/migrated/image-1714115269843-844c70ba.png)
