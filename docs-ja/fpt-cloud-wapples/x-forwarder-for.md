# Bao gồm các chức năng

Bao gồm các chức năng
  * Block X-Forwarded-For IP: thực hiện block các ip không có X-Forwarded-For header
  * Detect by X-Forwarded-For IP: thực hiện deteck và ghi lại log các ip không có X-Forwarded-For header
