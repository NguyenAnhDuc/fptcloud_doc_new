# X-Forwarded-For ヘッダーの設定

以下の機能が含まれています。

- **Block X-Forwarded-For IP**: X-Forwarded-For ヘッダーを持たないIPをブロックします。
- **Detect by X-Forwarded-For IP**: X-Forwarded-For ヘッダーを持たないIPを検出し、ログに記録します。
