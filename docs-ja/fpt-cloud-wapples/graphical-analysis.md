# Graphical Analysis

各ウェブサイトのネットワークトラフィック、ネットワーク状態、システム状態などのパラメーターを検索・フィルタリングします。

確認したいウェブサイトとグラフの種類を各フィールドから選択できます。デフォルトではすべてのウェブサイトに対して棒グラフ（Bar）が表示されます。
