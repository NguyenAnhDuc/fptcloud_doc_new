# 初期設定

DDoS保護を使用する前に、以下の手順を完了してください。

## FPT Cloudアカウントの作成とログイン

1. [https://fptcloud.com/](https://fptcloud.com/) にアクセスし、**Sign Up** をクリックします。
2. 必要な情報を入力してアカウントを登録します。サポートチームがアカウント確認のために連絡します。
3. [https://console.fptcloud.com](https://console.fptcloud.com) にFPT Cloudの認証情報でログインします。
4. 使用するサービスに合わせて正しい **Tenant**、**Region**、**VPC** を選択します。

:::tip
操作中に問題が発生した場合は、ホットライン **1900 638 399** またはメール **support@fptcloud.com** までお問い合わせください。
:::
