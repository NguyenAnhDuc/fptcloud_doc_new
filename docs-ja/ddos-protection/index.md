# DDoS保護

FPT CloudのDDoS保護は、分散型サービス拒否攻撃（DDoS）を自動的に検出・遮断・対応し、サービスの可用性と安定性を確保します。

## 主な機能

- **リアルタイム監視**: ネットワーク上のトラフィックと異常な動作を自動的に追跡します。
- **自動防御の起動**: 攻撃が検出された際に、トラフィックフィルタリングとDDoS対策メカニズムを自動的に起動します。
- **透明性のあるレポート**: お客様向けにリアルタイムおよび定期的なレポートを提供します。
