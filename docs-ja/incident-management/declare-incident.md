# インシデントの直接宣言

1. 左メニューで **Apps** > **Incident** をクリックします。
2. **Declare Incident** をクリックします。
3. インシデント情報を入力します：
   - **What's going on?**: 発生中のインシデントの説明または名前
   - **Severity**: インシデントの深刻度
   - **Status**: インシデントの現在のステータス
   - **Labels**: インシデントに割り当てるラベルのリスト
4. **Declare incident** をクリックして完了します。
