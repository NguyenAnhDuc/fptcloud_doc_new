# Incident status

インシデントのステータスは、そのインシデントが解決済みかどうかを示します。対応活動が進行中かどうかを即座に反映する必要があります。

FMON Incidentに推奨設定済みのステータス：

| Status | 説明 |
|--------|------|
| Active | インシデントが発生中 |
| Resolved | インシデントが解決済み |

## Incident statusの編集

1. 左メニューで **Apps** > **Incident** をクリックします。
2. **Settings** をクリックします。
3. **Status** タブで、編集するstatusを選択し、**鉛筆アイコン** をクリックします。

   [![編集するstatusを選択](/img/migrated/Screenshot_13-469bf8a6.png)](/img/migrated/Screenshot_13-469bf8a6.png)

4. **Name** と **Description** を更新します。
5. **Edit Status** をクリックして保存します。

   [![Status編集フォーム](/img/migrated/Screenshot_14-91e539ad.png)](/img/migrated/Screenshot_14-91e539ad.png)
