# Incident label

Incident に label を付けることで、インシデントをフィルタリングし、時間の経過に伴うトレンドを分析できます。label を使用してインシデントをグループ化し、関連するチーム、影響を受けているサービス、顧客への影響度などの情報をタグ付けします。

## label を作成する

1. 左メニューで **Apps** > **Incident** をクリックします。
2. **Settings** をクリックします。
3. **Labels** タブで **Create Label** をクリックします。

   [![label 一覧](/img/migrated/Screenshot_5-81e86153.png)](/img/migrated/Screenshot_5-81e86153.png)

4. label の **Name** と **Description** を入力します。
5. (任意) label の色を選択します。
6. **Add Label** をクリックして作成します。

   [![label 作成フォーム](/img/migrated/Screenshot_6-441947ed.png)](/img/migrated/Screenshot_6-441947ed.png)

追加の label を作成する場合は上記の手順を繰り返します。

## label を編集する

1. 左メニューで **Apps** > **Incident** をクリックします。
2. **Settings** をクリックします。
3. **Labels** タブで編集したい label を選択し、**鉛筆** アイコンをクリックします。

   [![編集する label を選択](/img/migrated/Screenshot_7-351b6d16.png)](/img/migrated/Screenshot_7-351b6d16.png)

4. **Name** と **Description** を更新します。
5. (任意) 色を変更します。
6. **Edit Label** をクリックして保存します。

   [![label 編集フォーム](/img/migrated/Screenshot_8-2f86e3dd.png)](/img/migrated/Screenshot_8-2f86e3dd.png)

## label を削除する

1. 左メニューで **Apps** > **Incident** をクリックします。
2. **Settings** をクリックします。
3. **Labels** タブで削除したい label を選択し、**Recycle Bin** アイコンをクリックします。

   [![削除する label を選択](/img/migrated/Screenshot_9-9db19efd.png)](/img/migrated/Screenshot_9-9db19efd.png)

4. 確認ダイアログで **Delete** をクリックします。
