# 特定リソースでコストエクスプローラーをフィルタリング

コストエクスプローラーのCustomize Filterではタグによるフィルタリングが可能です。タグ付け機能を使用してコストを確認したい特定のリソースにタグを付け、そのタグでコストエクスプローラーをフィルタリングしてください。
