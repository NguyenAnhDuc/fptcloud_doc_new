# フィルタをカスタマイズして期間／リソース グループ別のコストを確認_2

ステップ1： タブバーメニューにて[請求（Billing）]→[コストエクスプローラー（Cost Explorer）] →[フィルター（Filter）] を選択します。 
[![file](/img/migrated/image-1744253659165-fb43cfb4.png)](/img/migrated/image-1744253659165-fb43cfb4.png)
ステップ2：[フィルタをカスタマイズ]のダイアログボックスにてフィルタリング条件を1つ以上指定します。
フィルタリング条件には以下が含まれます。
  * 期間（Start date/End date）
  * 期間タイプ（Period tye）：コスト変動グラフの表示形式（月ごと・日ごと）の切替
  * サービス（Service）：ORGで使用される各種サービス
  * VPC：ORG内の全VPC
  * タグ（Tag）：タグはポータルで管理され、リソースに割り当てられます。
