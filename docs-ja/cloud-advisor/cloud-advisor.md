# Cloud Advisorの紹介

FPT Cloud Advisorは、以下のことを支援するツールです：

- クラウドリソースの安全性、パフォーマンス、コスト最適化の評価
- 設定ミスのリスクの検出
- システム改善のための推奨事項の提供
