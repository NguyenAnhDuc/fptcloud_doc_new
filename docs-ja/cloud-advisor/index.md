# Tính năng chính

Cloud Advisor là dịch vụ của FPT Cloud cung cấp các khuyến nghị tối ưu hóa cho hạ tầng cloud của bạn, bao gồm các lĩnh vực:

- **Chi phí (Cost)**: Phân tích và đề xuất cách giảm chi phí sử dụng cloud
- **Hiệu suất (Performance)**: Tối ưu hóa hiệu năng của các tài nguyên
- **Bảo mật (Security)**: Đánh giá và cải thiện tình trạng bảo mật
- **Độ tin cậy (Reliability)**: Đảm bảo tính sẵn sàng và khả năng phục hồi của hệ thống

## Tính năng chính

- Phân tích tự động các tài nguyên cloud đang sử dụng
- Đưa ra khuyến nghị cụ thể để tối ưu hóa
- Dashboard trực quan để theo dõi trạng thái
- Báo cáo chi tiết theo từng danh mục

## Bắt đầu sử dụng

Truy cập Cloud Advisor từ [FPT Cloud Console](https://console.fptcloud.com) để bắt đầu nhận các khuyến nghị tối ưu hóa cho hạ tầng của bạn.
