# Groupの管理

Autoscale Groupは、技術仕様が均一で同一の目的を持つInstance Nodeの集合であり、各Instance NodeはVirtual Instanceに対応します。Autoscale Groupは、事前に作成したAutoscale Profileに基づいて、Instance Nodeの管理と自動スケール（Instance Node数の増減）を提供します。
