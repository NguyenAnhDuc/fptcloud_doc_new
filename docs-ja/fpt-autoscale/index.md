# FPT Autoscale

FPT Autoscaleは、複数のVirtual Instanceを同時に作成・管理できるサービスです。需要に応じてVirtual Instanceの数を自動的に調整し、アプリケーションの安定性を向上させ、不要なコストを最小化し、運用時間を節約します。
