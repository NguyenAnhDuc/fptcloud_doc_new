# FAQ

### 1. API キーを作成してモデルで使用するには？

**マイアカウント → My API Keys** で API キーを作成できます。
このキーは **Inference API** 経由でモデルを呼び出す際に必要です。

---

### 2. モデルの使用料金はどのように計算されますか？

料金は **入力トークンと出力トークンの数** に基づいて計算されます。
詳細は **「Product Information」→「Pricing」** または **マイアカウント** 内の **「Billing Management」** で確認できます。

---

### 3. モデル使用のレート制限はどうなっていますか？

各モデルには固有の **レート制限**（例：1 秒あたりのリクエスト数またはトークン数）があります。
この情報は **「Product Information」→「Rate Limit」** で確認できます。

---

### 4. Marketplace はモデルエンドポイントの自動スケーリングをサポートしていますか？

はい。エンドポイントはトラフィック負荷に基づく **自動スケーリング** で設定できます。
これにより、トラフィックの急増時にも安定性を維持しながらコストを最適化できます。
