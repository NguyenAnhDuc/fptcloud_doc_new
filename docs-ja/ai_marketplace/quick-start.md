# クイックスタート

## ステップバイステップ

**AI Marketplace** を使い始めるには、以下の手順に従ってください。

### 1. アクセスとサインアップ / ログイン

- [FPT AI Marketplace ポータル](https://marketplace.fptcloud.com/en) にアクセスします。
- 新規アカウントにサインアップするか、既存の資格情報でログインします。

ログイン後、ホームページに移動し、利用可能なすべての AI モデルを参照できます。

---

### 2. モデルの探索と選択

- [FPT AI Marketplace ポータル](https://marketplace.fptcloud.com/en)（ホームページ）で利用可能なモデルを参照します。
- [Playground](https://marketplace.fptcloud.com/en/playground) を使用して、決定する前に独自のプロンプトでモデルをテストします。
- ビジネスニーズに基づいて、希望するモデル（LLM または VLM など）を選択します。

---

### 3. API の有効化と API キーの作成

- ダッシュボードの **「My API Keys」** に移動します。
- **「Create API Key」** をクリックしてプロジェクトに割り当てます。
- キーをコピーして安全な場所に保管してください — すべての API リクエストの認証に必要です。

:::tip
詳細な手順については、[API キー作成ガイド](./マイアカウント.md)を参照してください。
:::

---

### 4. API 連携

API を使用して選択したモデルをアプリケーションに組み込みます。

:::note
エンドポイントの詳細、リクエスト形式、コードサンプルについては、[API リファレンス全文](https://github.com/fpt-corp/ai-marketplace)を参照してください。
:::
