# 請求

## 請求を管理するには？

- マイアカウントのドロップダウンから **「Add to credit balance」** または **「Buy credit to start using API」** をクリックして、請求情報と設定にアクセスします。

[![マイアカウントの請求メニューのスクリーンショット](/img/migrated/1-1-29c4dda0.png)](/img/migrated/1-1-29c4dda0.png)

- **AI Marketplace — Model as a Service** の料金を確認できます。

[![料金表のスクリーンショット](/img/migrated/2-b02ccb07.png)](/img/migrated/2-b02ccb07.png)

- クレジットを購入するには、**「Purchase」** ボタンをクリックします。推奨金額（**1,500,000 VND** または **2,500,000 VND**）を選択するか、希望の金額を入力してください。最低取引金額は **26,000 VND** です。入力後、**「Purchase」** をクリックします。

[![クレジット購入画面のスクリーンショット](/img/migrated/3-973418ce.png)](/img/migrated/3-973418ce.png)

- **VNPay** 決済ページにリダイレクトされ、選択した金額を希望の支払い方法（**VNPayQR** または **国際決済カード** など）で完了できます。

[![VNPay 決済画面のスクリーンショット](/img/migrated/image-1742892528929-dc30d234.png)](/img/migrated/image-1742892528929-dc30d234.png)

- チャージ後は、**「Credit History」** でチャージ取引履歴を確認できます。

[![クレジット履歴ページのスクリーンショット](/img/migrated/image-1742892550518-3a6ee216.png)](/img/migrated/image-1742892550518-3a6ee216.png)
