# 製品情報

## AI モデルと API 連携の使い方

このタブでは、各 AI モデルの入力パラメーター、サンプル入出力、および API 連携ドキュメントを確認できます。

[![入力パラメーターとサンプルコードを表示する製品情報タブ](/img/migrated/Thiet-ke-chua-co-ten-2-e1754983791648-e6aef34e.png)](/img/migrated/Thiet-ke-chua-co-ten-2-e1754983791648-e6aef34e.png)

## モデルの詳細情報を確認するには？

ホームページまたは製品 URL から製品詳細画面にアクセスしてください。このページでは、AI モデルの詳細情報の確認、モデルの試用、推論 API のサンプルコードの参照ができます。

[![ベンチマーク結果と推論サンプルコードが表示されたモデル詳細ページ](/img/migrated/detail-8eb65be6.png)](/img/migrated/detail-8eb65be6.png)

## モデルのベンチマークを確認するには？

このタブでは、モデルのベンチマーク結果を確認し、他のモデルと比較できます。

[![モデルのパフォーマンス指標を表示するベンチマーク比較タブ](/img/migrated/detail-8eb65be6.png)](/img/migrated/detail-8eb65be6.png)

## レート制限を確認するには？

このページでは API のレート制限について説明しています。公平で安定したシステムパフォーマンスを確保するため、各モデルサイズごとに許可される最大トークン数とリクエスト数（1 分あたり）が記載されています。

[![各モデルのトークン数とリクエスト数（分あたり）を表示するレート制限ページ](/img/migrated/rate-limit-849886a5.png)](/img/migrated/rate-limit-849886a5.png)
