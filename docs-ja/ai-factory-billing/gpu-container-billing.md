# GPU Container 課金

| **課金ロジック** | **最小課金単位** | **課金タイミング** |
| --- | --- | --- |
| - GPU Instance：<br/>- GPU 数<br/>- 使用時間（分）<br/>- ステータス：実行中（Running）<br/><br/>- 永続ストレージ：<br/>- GB 容量<br/>- 使用時間（分）<br/>- ステータス：実行中（Running）/ 停止中（Stopped） | 1分 | 15分ごと |

**課金計算式：**
  * **使用時間** : 時間単位（切り上げなし）
  * **ストレージ容量** : GB 単位
  * **GPU コスト** : 使用時間（時間）× GPU 時間単価
  * **ストレージコスト** : 使用時間（時間）× GB × ストレージ時間単価
  * **合計コスト** : `GPU コスト + ストレージコスト`

**計算例：**

> H100 GPU 1基（CPU 15コア、RAM 250 GB、GPU 1基）のコンテナと、$2.31/時間の 1000 GB 一時ストレージ、$0.00013/GB/時間の 1000 GB 永続ストレージを 30分間使用した場合
  * 使用時間：30分 = 0.5時間
  * GPU コスト：`0.5 × $2.31 = $1.16`
  * ストレージコスト：`1000 × $0.00013 × 0.5 = $0.0015`
  * 合計：`$1.16 + $0.0015 = $1.1615`
