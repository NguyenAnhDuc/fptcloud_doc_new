# クレジットの追加

PAYG 課金プランで AI Factory サービスを使用開始するには、先にクレジットを購入する必要があります。
以下の手順に従ってください。
  1. 左側メニューで **Account** セクションの **Billing** をクリックします。
  2. **Add credit** ボタンをクリックします。 [![Alt text](/img/migrated/image2025-6-22_11-49-23-c0ffac4e.png)](/img/migrated/image2025-6-22_11-49-23-c0ffac4e.png)
  3. **Add Credit ポップアップ** で、提案金額を選択するか、残高に追加したい **クレジット金額を入力** します。*_最低金額が設定されており、それ以下の金額は追加できません。_ [![Alt text](/img/migrated/image2025-7-6_17-32-55-29e3537a.png)](/img/migrated/image2025-7-6_17-32-55-29e3537a.png)
  4. 次に **支払い方法** を選択します。
     * **VNPay** : ベトナムリージョンおよびベトナムドンのみ対応。VNPay のチェックアウトウェブサイトで取引を完了します。
     * **PayPal** : ベトナムリージョンおよび米ドルのみ対応。PayPal のチェックアウトポップアップで取引を完了します。
     * **Stripe** : ポップアップウィンドウでカード情報を入力して支払いを進めます。**すべての取引は Stripe によって安全に処理され、FPT はカード情報を保存しません。**

支払い取引の確認のために OTP が必要な場合があります。これはカード発行会社によって異なります。 [![Alt text](/img/migrated/image2025-7-6_17-34-54-77174d1a.png)](/img/migrated/image2025-7-6_17-34-54-77174d1a.png) [![Alt text](/img/migrated/image2025-8-10_11-23-14-86accd6e.png)](/img/migrated/image2025-8-10_11-23-14-86accd6e.png)
  5. 支払い完了後、成功通知を表示するポップアップが表示され、**合計残高（Total Balance）** および **チャージ残高（Top-up Balance）** が更新されます。 [![Alt text](/img/migrated/Picture1-1-9f3038ad.png)](/img/migrated/Picture1-1-9f3038ad.png)
  6. **クレジット履歴（Credit history）** テーブルでクレジット追加の取引を確認することもできます。 [![Alt text](/img/migrated/image2025-6-28_12-17-42-eecb0352.png)](/img/migrated/image2025-6-28_12-17-42-eecb0352.png)
