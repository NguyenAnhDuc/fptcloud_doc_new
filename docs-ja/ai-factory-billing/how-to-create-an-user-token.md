# ユーザートークンの作成方法

ユーザートークンは、ログイン情報を公開することなくAI StudioサービスへのAPIコールを認証します。トークンは作成時に一度だけ表示されるため、ダイアログを閉じる前に必ずコピーしてください。

## ステップ1：トークンを作成する

1. [ai.fptcloud.com](https://ai.fptcloud.com/) のトークンサービスにアクセスし、**Create New Token** をクリックします。

   [![「Create New Token」ボタンが強調表示されたトークン一覧ページ](/img/migrated/image-1760341748499-2ab11f23.png)](/img/migrated/image-1760341748499-2ab11f23.png)

2. ポップアップでトークンの詳細を入力します。
   - **Name（名前）**: 最大50文字。使用できるのは英数字、アンダースコア、ハイフン、スペース、ドットのみです。
   - **Expiration（有効期限）**: ドロップダウンから有効期限を選択します（30日、60日、または90日）。

   [![名前と有効期限フィールドを表示する「Create New Token」ポップアップ](/img/migrated/image-1760341777314-c998e416.png)](/img/migrated/image-1760341777314-c998e416.png)

3. **Create Token** をクリックします。

4. 確認画面に表示されたトークンをすぐにコピーします。

:::warning
トークンは一度しか表示されません。このダイアログを閉じると再度確認することはできません。**Done** をクリックする前に必ず安全な場所に保存してください。
:::

:::note
トークンは**モデルファインチューニング**および**モデルハブ**サービスでのみ有効です。
:::

## ステップ2：テナントIDを取得する

トークンを作成すると、トークンページのトークン一覧に表示されます。一覧から**テナントID**を確認してください。ファインチューニングやモデルハブのAPIコールを認証するには、`access_token` と `tenant_id` の両方が必要です。
