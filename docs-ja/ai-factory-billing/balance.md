# 残高

**残高は特定の tenant とリージョンに紐付いています。**
  * 現在、複数のリージョンまたは tenant にまたがる単一の残高はサポートしていません。
  * **ベトナム** の GPU H100 と **日本** の GPU H200 の両方をインフラとして使用する場合は、各リージョンに **別々の tenant** を作成する必要があります。*

**合計残高（Total Balance）** は、サービスに使用できる現在の利用可能なクレジットです。 [![Alt text](/img/migrated/image2025-6-28_21-29-54-7c6b5942.png)](/img/migrated/image2025-6-28_21-29-54-7c6b5942.png) 残高は2つのコンポーネントで構成されます。
  * **チャージ残高（Top-up Balance）** : **VNPAY**、**Stripe** などの支払い方法を通じて購入したクレジット。
  * **バウチャー残高（Voucher Balance）** : **バウチャーコード** から換金されたクレジット。通常、**試用（Trial）、概念実証（POC）、プロモーション** などの特別プログラムとして **FPT** から提供されます。

*_ai.fptcloud.com（ベトナムリージョン）または ai.fptcloud.jp（日本リージョン）のポータルへのサインインが完了すると、新しい tenant が自動的に作成されます。これら2つのリージョンの tenant は、残高とサービスリソースの面で完全に分離されていることにご注意ください。_
