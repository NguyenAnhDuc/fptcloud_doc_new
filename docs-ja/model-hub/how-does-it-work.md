# 仕組み

### ステップ 1. Model Catalog からベース model を選択する
  * model を検索します（例：NLP 向けの `"Llama-3.2-1B"`、ビジョン向けの `"Qwen2-VL-72B"`）。
  * これらの model はすでに汎用的な表現を学習しています。

### ステップ 2. ファインチューニングを開始する
  * 選択した model を使用してファインチューニングを開始します。

### ステップ 3. ファインチューニング済み model を Hub にプッシュする
  * トレーニング完了後、ファインチューニングした model を hub にアップロードします。
  * 新しいバージョン ID が付与され、コラボレーターと共有したり（またはプライベートに保持したり）できます。

### ステップ 4. どこからでも利用する
  * ファインチューニング完了後、**SDK で model をダウンロード** できます。
  * これにより、再現性と容易な deploy が保証されます。
