# model の編集

作成済みの model の情報を編集します。

## 手順

1. **Model list** 画面で、編集したい model のタイトルをクリックします。**Model detail** 画面が表示されます。

2. **Actions** > **Edit** をクリックします。
[![](/img/migrated/Picture12-38e4faed.png)](/img/migrated/Picture12-38e4faed.png)

3. Model detail 画面が編集モードに切り替わり、各フィールドを編集できます。
[![](/img/migrated/Picture13-bbfea244.png)](/img/migrated/Picture13-bbfea244.png)

4. 編集したいフィールドを変更します。**Save** をクリックして変更を保存するか、**Cancel** をクリックして操作を取り消します。
