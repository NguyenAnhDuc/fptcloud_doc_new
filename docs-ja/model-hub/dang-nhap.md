# サインイン

アカウント作成が完了したら、Portal にサインインして **Model Hub** サービスを利用します。

## 手順

1. <https://console.fptcloud.com> にアクセスします。

2. 作成したアカウントとパスワードでサインインします。

   > **注意:** Tenant、Region、VPC を正しく選択してください。

3. ポータルの画面が表示されます。
[![](/img/migrated/Picture1-67e96351.png)](/img/migrated/Picture1-67e96351.png)

4. メニューで **AI Platform** > **Model Hub** を選択します。

**Model Hub** サービス画面が表示され、サービスの利用を開始できます。
