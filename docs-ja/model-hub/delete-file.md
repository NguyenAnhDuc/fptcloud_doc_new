# ファイルの削除

**目的：** バージョンに関連するファイルを削除します。
ファイルを削除するには、以下の手順に従います：
**ステップ 1：** model をクリックして詳細を表示 > **File Browser** タブを選択します。
**ステップ 2：** **Action** 列の三点ボタンをクリック > **Delete** を選択します。
[![Alt text](/img/migrated/23.-Delete-5c3168c6.png)](/img/migrated/23.-Delete-5c3168c6.png)
**ステップ 3**：確認ダイアログが表示されます。削除を確認するには、テキストボックスに「delete」と入力します。テキストを入力すると **Delete** ボタンが有効になります。
[![Alt text](/img/migrated/24.-delete-4496c8c8.png)](/img/migrated/24.-delete-4496c8c8.png)
**ステップ 4**：削除するには **Confirm** ボタンをクリックします。アクションをキャンセルするには **Cancel** をクリックします。
