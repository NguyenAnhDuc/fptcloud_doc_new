# model の編集

**Edit Model** を使用すると、既存のプライベート model の詳細（メタデータ、設定、アクセス設定を含む）を更新できます。
### **ステップ 1：model を見つける**
  * 上部の **検索バー** を使用して、名前またはキーワードで model を検索します。
  * **フィルター**（例：バージョン、サイズ、タグ）を適用してリストを絞り込みます。

### **ステップ 2：model を選択する**
  * model 一覧では、各エントリに以下が表示されます：
    * **Model Name**
    * **Version**
    * **Size**
    * **Tags**
  * 変更したい model の横にある **Edit** ボタンをクリックします。 [![file](/img/migrated/image-1757326429664-b76cb241.png)](/img/migrated/image-1757326429664-b76cb241.png)
### **ステップ 3：model を編集する**

**Edit** をクリックすると、model 設定インターフェースに移動し、以下のことができます：
  * **基本情報の更新**：タイトル、説明、ベース model、サイズ、アバター、ホワイトリスト IP を変更します。
  * **タグの変更**：Frameworks、Architecture、Subject、Task、Others などのカテゴリーのタグを追加、削除、または更新します。
  * **変更のプレビュー**：保存前にプレビューパネルを使用して更新内容を確認します。 [![file](/img/migrated/image-1757326379648-f3c0bb0d.png)](/img/migrated/image-1757326379648-f3c0bb0d.png)

### **ステップ 4：変更の確定**
  * **Submit** をクリックして編集内容を保存します。
  * **Back** を使用して保存せずに戻ります。
