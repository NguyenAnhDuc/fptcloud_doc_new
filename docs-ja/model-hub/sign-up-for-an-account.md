# アカウントの登録

### **1. サインイン / サインアップページにアクセスする**
  * ウェブブラウザで FPT AI Factory にアクセスします（<https://ai.fptcloud.com/>）
  * ホームページの右上隅にある **Sign in/Sign up** をクリックします。
  * 画面中央にログインダイアログが表示されます。

[![file](/img/migrated/image-1756379868688-c8bb05fd.png)](/img/migrated/image-1756379868688-c8bb05fd.png)
### **2. サインインオプション**
以下のいずれかの方法でサインインできます：
  * **FPT ID で続ける**：既存の FPT アカウントの認証情報を使用してログインします。

[![file](/img/migrated/image-1756380038671-d20b4728.png)](/img/migrated/image-1756380038671-d20b4728.png)
  * **Google で続ける**：このオプションを選択し、Google 認証プロセスに従って Google アカウントでサインインします。

[![file](/img/migrated/image-1756380184145-996657dc.png)](/img/migrated/image-1756380184145-996657dc.png)
  * **Microsoft で続ける**：Microsoft アカウントを使用して安全にログインします。

[![file](/img/migrated/image-1756380229709-9aa75e66.png)](/img/migrated/image-1756380229709-9aa75e66.png)
### **3. 新しいアカウントへのサインアップ**
  * アカウントをお持ちでない場合は、利用可能なログインオプション（FPT ID、Google、または Microsoft）のいずれかを選択します。
  * 指示に従って登録プロセスを完了します。
  * 登録後は、同じ方法で次回もサインインできます。

### **4. ログイン成功**
サインイン後、**AI Factory ダッシュボード** にリダイレクトされ、以下のことができます：
  * GPU コンテナの起動
  * AI/ML model の管理
  * inference API の探索
  * 料金と請求へのアクセス

[![file](/img/migrated/image-1756380788218-07094d0e.png)](/img/migrated/image-1756380788218-07094d0e.png)
### **5. トラブルシューティング**
  * 外部ログインメソッドを使用する場合は、Google または Microsoft アカウントがアクティブであることを確認してください。
  * FPT ID のログイン問題については、**FPT アカウント回復ページ** でパスワードをリセットしてください（<https://id.fptcloud.com/>）。
  * 技術的な問題が発生した場合は、左メニューの **Help & Support** をクリックするか、以下にお問い合わせください：
📧 **support@fptcloud.com** | ☎️ **1900 638 399**
