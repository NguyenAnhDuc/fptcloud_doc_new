# model の削除

**Delete a model** を使用すると、作成した model の一覧から削除できます。

### **ステップ 1：model を見つける**
  * 上部の **検索バー** を使用して、名前またはキーワードで model を検索します。
  * **フィルター**（例：バージョン、サイズ、タグ）を適用してリストを絞り込みます。

### **ステップ 2：model を選択する**
  * model 一覧では、各エントリに以下が表示されます：
    * **Model Name**
    * **Version**
    * **Size**
    * **Tags**
  * 削除したい model の横にある **Delete** ボタンをクリックします。 [![file](/img/migrated/image-1757326429664-b76cb241.png)](/img/migrated/image-1757326429664-b76cb241.png)

### **ステップ 3：model を削除する**
**Delete** をクリックすると、確認ダイアログが表示されます。削除を確認するには、テキストボックスに「**confirm**」と入力します。テキストを入力すると **Confirm** ボタンが有効になります。 [![file](/img/migrated/image-1757329571101-178be0fe.png)](/img/migrated/image-1757329571101-178be0fe.png)

### **ステップ 4**：変更の確定
  * **Confirm** ボタンをクリックして model を削除します。
  * **Cancel** をクリックしてアクションをキャンセルします。
