# バージョンの編集

**目的**：既存のバージョンを編集できます。
バージョンを更新するには、以下の手順に従います：
**ステップ 1**：model をクリックして詳細を表示 > **Version** タブを選択します。
**ステップ 2**：バージョン一覧で、編集したいバージョンの横にある三点ボタンをクリックし、**Edit** オプションを選択します。
[![file](/img/migrated/image-1759305159846-cc479dd1.png)](/img/migrated/image-1759305159846-cc479dd1.png)
**ステップ 3**：システムが **Edit version** 画面に移動し、変更を加えることができます。
  * **注意**：編集できるのは **Version visibility** と **Tags** のみです。

**ステップ 4**：変更が完了したら、**Save** ボタンをクリックして更新されたバージョンを保存します。
