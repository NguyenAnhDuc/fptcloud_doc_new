# model の削除

作成済みの model リストから model を削除します。

## 手順

1. **Model list** 画面で、削除したい model のタイトルをクリックします。**Model detail** 画面が表示されます。

2. **Actions** ボタンをクリックし、**Delete** を選択します。
[![](/img/migrated/Picture14-d464a2e0.png)](/img/migrated/Picture14-d464a2e0.png)

3. 確認ダイアログが表示されます。テキストボックスに `delete` と入力して確認します。**Delete** ボタンが有効になります。
[![](/img/migrated/Picture15-64ad0878.png)](/img/migrated/Picture15-64ad0878.png)

4. **Confirm** をクリックして削除するか、**Cancel** をクリックして操作を取り消します。
