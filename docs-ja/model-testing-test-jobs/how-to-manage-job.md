# テストジョブの管理方法

テストジョブの管理は、FPT AI Factory の Model Testing 機能において、AIモデルの評価・検証プロセスを効率的に運用するための重要な操作です。本ページでは、テストジョブの作成・実行・確認・削除の手順を説明します。

## 概要

テストジョブとは、指定したデータセットとモデルに対して評価タスクを実行する単位です。ジョブを管理することで、複数のモデルバージョンの性能を体系的に比較し、本番環境へのデプロイ判断に役立てることができます。

## テストジョブの基本操作

### ジョブの作成

1. Model Testing の管理画面から「テストジョブ」タブを選択します。
2. 「新規ジョブ作成」ボタンをクリックします。
3. テスト対象のモデルとデータセットを選択します。
4. 評価指標（精度、F1スコア、レイテンシ等）を設定します。
5. 「実行」ボタンをクリックしてジョブを開始します。

### ジョブの状態確認

ジョブ一覧では、各ジョブのステータス（待機中・実行中・完了・失敗）がリアルタイムで更新されます。完了したジョブをクリックすると、評価結果の詳細レポートを確認できます。

### ジョブの再実行と削除

- **再実行**: ジョブ詳細ページの「再実行」ボタンから同じ設定でジョブを再度実行できます。
- **削除**: 不要なジョブは一覧の「削除」アイコンから削除できます。実行中のジョブを削除するとジョブが中断されます。

## 結果の活用

テスト結果はCSV形式でエクスポートでき、モデルの改善サイクルに活用してください。
