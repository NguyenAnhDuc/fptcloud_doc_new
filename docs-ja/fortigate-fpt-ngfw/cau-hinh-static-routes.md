# Static Routes の設定

**ステップ 1：** **Network** > **Static Routes** を選択します。CLI の初期設定では、ゲートウェイ 10.0.0.1 を持つ WAN ポート向けのスタティックルートがあらかじめ作成されています。  
[![Userguide Fortigate FPT NGFW 2022 5](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-5-1024-f0a1691a.png)](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-5-1024-f0a1691a.png)  
**ステップ 2：** Web インターフェースからスタティックルートを作成できます。**Create New** > ルーティング対象の Interface を選択 > **Gateway** または **Destination** を選択します（特定の宛先がない場合は Destination をデフォルトのままにできます）。
