# アラートのステータスを確認する

アラートが作成されると、システムは継続的に FPT Portal に情報を同期します。アラートの稼働状況は以下の手順で確認できます。

**Alert Management** テーブルで監視したいアラートを見つけます。アラートの稼働状況は **State** フィールドに表示されます。
