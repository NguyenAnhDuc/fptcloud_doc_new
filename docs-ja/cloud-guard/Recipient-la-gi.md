# 受信者とは？

**受信者**（Recipient）は、FPT Alert システムからアラート通知を受け取る人またはチャネルです。Tenant Owner は受信者リストを作成・管理できます。

受信者リストを表示するには、**Alert** ページを開いて **Recipients** タブを選択します。
