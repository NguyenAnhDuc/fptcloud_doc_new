# 編集 Recipient

**ステップ1:** Ở menu chọn **Cloud Guard** > **Alert** , chọn tab **Recipients.**
[![file](/img/migrated/image-1725532997863-419e42b8.png)](/img/migrated/image-1725532997863-419e42b8.png)
**ステップ2:** **Recipients** cần chỉnh sửa. システムが表示します màn hình **Update**.
**ステップ3:** Nhập thông tin mới vào các trường tương ứng và chọn **Update**.
