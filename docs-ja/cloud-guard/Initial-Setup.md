# 初期設定

**FPT Cloud Guard – Alert** を初めて使用する場合は、最初のアラートを作成する前に以下の手順を完了してください。

## FPT Cloud アカウントの作成と FPT Portal へのログイン

1. FPT Cloud アカウントを登録するには、[https://fptcloud.com/](https://fptcloud.com/) にアクセスします。
2. **Sign Up** を選択し、画面の指示に従って必要な情報を入力してください。サポートチームが折り返し連絡し、アカウントの詳細を確認します。
3. FPT Portal にログインするには、[https://console.fptcloud.com/](https://console.fptcloud.com/) にアクセスします。
4. 発行された認証情報でログインし、監視するリソースが含まれる正しい Tenant、Region、VPC を選択します。
5. これで最初の **Alert** を作成する準備が整いました。

:::tip
上記の情報が不明な場合、または 3 回試みてもシステムがエラーを返す場合は、サポートチームに直ちにお問い合わせください。
:::
