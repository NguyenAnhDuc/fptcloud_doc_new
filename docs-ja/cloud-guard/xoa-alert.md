# アラートを削除する

:::warning
この操作はアラートのすべての履歴データを完全に削除し、元に戻すことはできません。
:::

**手順 1:** **Alert Management** テーブルで削除したいアラートを見つけ、**Action** 列の **Delete** をクリックします。

[![file](/img/migrated/image-1725532930908-040fd200.png)](/img/migrated/image-1725532930908-040fd200.png)

**手順 2:** アラート名が表示された確認ダイアログが表示されます。**Delete** をクリックして削除を確定します。

[![Userguide FPT Alert 2022 11](/img/migrated/Userguide-FPT-Alert-2022-11-71bc76a4.png)](/img/migrated/Userguide-FPT-Alert-2022-11-71bc76a4.png)
