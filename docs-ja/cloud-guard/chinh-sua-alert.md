# 編集 Alert

**ステップ1:** Ở menu chọn **Cloud Guard** , chọn tab **Resource alert**
[![file](/img/migrated/image-1725532789529-6dfc0d32.png)](/img/migrated/image-1725532789529-6dfc0d32.png)
**ステップ2:** Chọn Alert cần chỉnh sửa. システムが表示します màn hình Update.
[![file](/img/migrated/image-1725532811064-3d095f59.png)](/img/migrated/image-1725532811064-3d095f59.png)
**ステップ3:** Nhập thông tin mới vào các trường tương ứng và chọn **Update**.
注意: FPT Cloud Guard chỉ hỗ trợ người dùng chỉnh sửa một số trường thông tin cần thiết
