# ストレージディスクの購入

アカウントにチャージした後、ストレージディスクを購入できます。

:::note
- ディスクを作成するには、少なくとも50,000 VNDと1時間分のストレージ使用料金が必要です。
- 作成時に1時間分のストレージ使用料金が即時引き落とされます。
:::

1. **Compute Engine – Storage Disks**ページに移動します。

[![Storage Disksページ](/img/migrated/image-1694073928674-97d41009.png)](/img/migrated/image-1694073928674-97d41009.png)

2. **New Storage**をクリックします。

[![新しいストレージの作成](/img/migrated/image-1694073937844-c9e0ea1f.png)](/img/migrated/image-1694073937844-c9e0ea1f.png)

3. **Storage policy**と必要な容量を選択します。システムが参考として1時間あたりの使用コストを表示します。

4. **Create storage disk**をクリックします。
   - 情報が有効な場合、システムはストレージを作成し、Storage Disksページにリダイレクトします。
   - リソース制限に達している場合、システムはエラーメッセージを表示します。

[![ストレージディスク作成結果](/img/migrated/image-1694073957418-7a7b3dac.png)](/img/migrated/image-1694073957418-7a7b3dac.png)

[![作成後に引き落とされたコスト](/img/migrated/image-1694073966294-6d3c6da1.png)](/img/migrated/image-1694073966294-6d3c6da1.png)
