# Nạp tiền thông qua gift code.

Khách hàng có thể nạp tiền thông qua gift code nhận được từ FPT Smart Cloud. 
  * B1. Vào trang Billing, chọn menu Topup. [![file](/img/migrated/image-1694073773226-c126e916.png)](/img/migrated/image-1694073773226-c126e916.png)
  * B2. Nhập mã gift code. [![file](/img/migrated/image-1694073782136-bded0c89.png)](/img/migrated/image-1694073782136-bded0c89.png)
  * B3. Bấm Redeem gift code 
    * Nếu gift code đúng và đang trong thời gian có hiệu lực, hệ thống sẽ cộng tiền vào tài khoản.
