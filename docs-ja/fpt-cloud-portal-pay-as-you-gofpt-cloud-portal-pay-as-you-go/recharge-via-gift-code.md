# ギフトコードによるチャージ

FPT Smart Cloudから受け取ったギフトコードを使用してアカウント残高にチャージできます。

1. **Billing**ページに移動し、**Topup**メニューを選択します。

[![Billing – Topupページ](/img/migrated/image-1694073773226-c126e916.png)](/img/migrated/image-1694073773226-c126e916.png)

2. 入力フィールドにギフトコードを入力します。

[![ギフトコードの入力](/img/migrated/image-1694073782136-bded0c89.png)](/img/migrated/image-1694073782136-bded0c89.png)

3. **Redeem gift code**をクリックします。

:::note
ギフトコードが有効で有効期限内であれば、システムは自動的に金額をアカウントに加算します。
:::
