# Organizationの作成

OTPで電話番号を認証すると、システムはOrganization作成ページにリダイレクトします。

1. 組織または個人の情報を入力します：
   - **Organization name**：組織名または顧客名。
   - **Organization code**：システム上での組織の表示名。
   - **Organization address**：組織の住所（任意）。

[![Organization情報の入力](/img/migrated/image-1694073747297-30c6d924.png)](/img/migrated/image-1694073747297-30c6d924.png)

2. Organizationの作成ボタンをクリックします。システムがサービス利用開始のためのデフォルトリソースをプロビジョニングします。リソースのプロビジョニングには約1〜3分かかります。

[![システムがリソースを初期化中](/img/migrated/image-1694073758532-40c000e0.png)](/img/migrated/image-1694073758532-40c000e0.png)
