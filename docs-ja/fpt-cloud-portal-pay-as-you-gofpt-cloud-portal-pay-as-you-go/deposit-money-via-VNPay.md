# VNPayによるチャージ

VNPay決済ゲートウェイを通じてアカウント残高にチャージできます。

1. **Billing**ページに移動し、**Topup**メニューを選択します。

[![Billing – Topupページ](/img/migrated/image-1694073789071-dcc595d7.png)](/img/migrated/image-1694073789071-dcc595d7.png)

2. プリセットの金額を選択するか、カスタム金額を入力します。

[![チャージ金額の選択](/img/migrated/image-1694073800926-6af15e22.png)](/img/migrated/image-1694073800926-6af15e22.png)

3. **Pay via VNPay**をクリックし、VNPayの決済プロセスを完了します。

:::note
決済が成功した場合、システムは確認メッセージを表示し、金額をアカウントに加算します。
:::

[![チャージ完了確認画面](/img/migrated/image-1694073814699-e6b48ffa.png)](/img/migrated/image-1694073814699-e6b48ffa.png)
