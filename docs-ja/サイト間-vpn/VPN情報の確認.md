# Vpn情報の確認

**ステップ1** ：VPN管理ページにて確認したいVPNの名称にクリックします。 ![file](/img/migrated/image-1742913956469-c8950bf1.png)
**ステップ2** ： **ステータスタブ** に表示され るVPN情報には以下のものが含まれます。
  * 接続情報 
  * エラー情報 
  * トラフィック 
  * 接続手順、VPNテスト 

現在の時刻で同期するには、**更新 ボタンをクリックしてください。なおシステムはデフォルトで定期的に更新されます。 
![file](/img/migrated/image-1742914122300-0e3c8fb8.png)
**ステップ3** : 接続情報 タブにおける接続構成情報 
*.txt形式で構成情報を保存したい場合は 構成情報をダウンロード にクリックしてください。
![file](/img/migrated/image-1742914184152-67ad6fa7.png)
