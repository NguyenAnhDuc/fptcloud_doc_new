# サイト間Vpnの情報編集  3

作成済みVPNの情報を変更するための手順です。 
**ステップ1** ：VPN情報の編集 を選択します。
[![file](/img/migrated/image-1742915365993-6486c2a4.png)](/img/migrated/image-1742915365993-6486c2a4.png)
**ステップ2** ：VPN情報を変更します。＊注：デフォルトで各フィールドに読み込まれる情報はVPN作成時のものです。
[![file](/img/migrated/image-1742915433625-2d068d8c.png)](/img/migrated/image-1742915433625-2d068d8c.png)
**ステップ3** ：変更後の情報を確認します。VPNステータスは 更新中 → アクティブ に切り替わります。
[![file](/img/migrated/image-1742915544560-614508af.png)](/img/migrated/image-1742915544560-614508af.png)
