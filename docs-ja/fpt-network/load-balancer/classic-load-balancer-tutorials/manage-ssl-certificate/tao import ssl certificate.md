# SSL証明書のインポート

1. **Manage SSL Certificate for Load Balancer**ページに移動し、**Import SSL Certificate**を選択します。

   [![Import SSL Certificateページ](/img/migrated/image-1712807252615-cef0c042.png)](/img/migrated/image-1712807252615-cef0c042.png)

2. **Import SSL Certificate**ダイアログにSSL証明書の情報を入力し、**Save**を選択します。

   [![Import SSL Certificateダイアログ](/img/migrated/image-1712808061542-b08fa903.png)](/img/migrated/image-1712808061542-b08fa903.png)
