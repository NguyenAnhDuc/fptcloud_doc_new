# Activity logとタスクステータスの確認

VPC上の全てのアクションはActivity logに記録されます。必要に応じて確認できます。

![VPC Activity logがVPC上のアクションを記録](/img/migrated/image-1744258669711-406a501d.png)

また、VPCのアクションは **Bell - Recent task** にも記録されます。進行中のタスクのステータスを確認できます。

![Bell Recent taskがVPCで実行中のタスクを記録](/img/migrated/image-1714811345638-2e8d0492.png)
