# View Execution History

**定義:** パイプライン実行とは、コードのビルド、テスト、デプロイなど、事前定義された一連の自動化ステップを実行するプロセスを指します。各ステップは順番に実行され、通常はイベント(コードコミットなど)によってトリガーされ、1つのステップのアウトプットを次のステップに渡すことができます。通常、実行ステータス、ログ、および結果は、各ステージで成功または失敗を監視するために追跡されます。
  * 特定のパイプラインを選択する
  * 実行中および完了したすべてのパイプライン実行が表示されます
  * 実行 ID で検索したり、フィルターを適用してステータスなどの検索を絞り込んだりできます
  * 右上隅の **アクション** で「**開始** 」をクリックして新しく実行するか、「**編集** 」、「**テンプレートとして保存** 」、または 「**削除** 」 を選択できます。
