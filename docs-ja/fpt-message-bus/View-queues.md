# queue の表示

**ステップ1:** 詳細情報を表示したい queue を選択します。

**ステップ2:** **View** をクリックします。

入力した情報が表示されます:
- **Name**: queue 名。
- **Status**: queue のステータス。
- max length、max length byte、message TTL、overflow などの設定情報も表示されます。

[![](/img/migrated/queue_view_1-98f8393e.png)](/img/migrated/queue_view_1-98f8393e.png)

これらの詳細情報は読み取り専用で、編集することはできません。
