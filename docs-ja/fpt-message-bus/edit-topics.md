# topic の編集

topic を編集すると、新しい subscription を追加したり既存の subscription を削除したりできます。topic を作り直さずに、メッセージを受信する consumer の数を調整できます。

1. **Application** メニューから **Topics** > 対象の topic を選択 > **Edit** に移動します。

   [![topic を選択して Edit ボタンをクリックした topic 一覧画面](/img/migrated/topic_edit_1-84c0c55b.png)](/img/migrated/topic_edit_1-84c0c55b.png)

2. **Subscriptions** セクションで subscription を追加または削除します。

   [![追加または削除できる subscription 一覧を表示する Edit topic 画面](/img/migrated/topic_edit_2-3a07d393.png)](/img/migrated/topic_edit_2-3a07d393.png)

3. **OK** をクリックして変更を保存します。
