# Adminアカウントのenable/disable

デフォルトでは、ArgoCDインスタンス作成時にadminアカウントは有効になっています。

## A. Adminアカウントをdisableする

1. **Essential Properties** 画面で **Disable Admin Account** をクリックします。

[![](/img/migrated/Picture60-2-5541f2bd.png)](/img/migrated/Picture60-2-5541f2bd.png)

2. **Confirm** をクリックして確認します。

[![](/img/migrated/Picture61-2-8c88f74e.png)](/img/migrated/Picture61-2-8c88f74e.png)

3. Disable後の結果：

[![](/img/migrated/Picture62-2-bbc635b4.png)](/img/migrated/Picture62-2-bbc635b4.png)

Disable後、ユーザーはLocal AccountまたはOIDCでArgoCDにアクセスできます。設定がない場合のログイン画面：

[![](/img/migrated/Picture63-2-e3980c02.png)](/img/migrated/Picture63-2-e3980c02.png)

Local AccountまたはOIDC Scopeがある場合：

[![](/img/migrated/Picture64-2-c91c0f62.png)](/img/migrated/Picture64-2-c91c0f62.png)

## B. Adminアカウントをenableする

1. **Essential Properties** 画面で **Enable Admin Account** をクリックします。

[![](/img/migrated/Picture65-2-3de06794.png)](/img/migrated/Picture65-2-3de06794.png)

2. **Confirm** をクリックして確認します。

[![](/img/migrated/Picture66-2-c7f796fd.png)](/img/migrated/Picture66-2-c7f796fd.png)

3. Enable後の結果：

[![](/img/migrated/Picture67-2-07f5266a.png)](/img/migrated/Picture67-2-07f5266a.png)

Enable後、adminアカウントでログインできます：

[![](/img/migrated/Picture68-2-3bc07d4a.png)](/img/migrated/Picture68-2-3bc07d4a.png)
