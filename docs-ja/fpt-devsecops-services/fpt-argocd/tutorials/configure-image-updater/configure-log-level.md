# ログレベルの設定

この機能により、Image Updater controllerのログレベルを設定できます。デフォルトのログレベルは`debug`です。

1. **Image Updater** > **Log Level**画面でログレベルを設定します。

[![](/img/migrated/Picture6-7-4537dcac.png)](/img/migrated/Picture6-7-4537dcac.png)

2. 更新するモードを選択します。

[![](/img/migrated/Picture7-7-c04500ca.png)](/img/migrated/Picture7-7-c04500ca.png)
