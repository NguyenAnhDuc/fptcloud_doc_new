# Vault Instanceを作成する

Vault Instanceは、すべてのSecret Engine、Secret、セキュリティ設定を格納するFPT Key Vaultの基本単位です。Secret管理操作を行う前にVault Instanceを作成してください。

1. メニューで **Security** > **Key Vault** > **Create** を選択します。

   [![Createボタンが表示されたKey Vault一覧画面](/img/migrated/Screenshot-2024-08-20-103813-1-bebc2790.png)](/img/migrated/Screenshot-2024-08-20-103813-1-bebc2790.png)

2. **Create Key Vault** 画面で必要な情報を入力します。

   [![Key Vault初期化情報入力フォーム](/img/migrated/Userguide-FPT-KeyVault-2022-2-125824c1.png)](/img/migrated/Userguide-FPT-KeyVault-2022-2-125824c1.png)

3. **Create** をクリックして完了します。システムがリソースを確認し、確認メッセージを送信後、入力した情報に基づいて新しい **Vault Instance** を作成します。

   [![作成成功後のKey Vault一覧](/img/migrated/Userguide-FPT-KeyVault-2022-3-1024x290-02d8d8d5.png)](/img/migrated/Userguide-FPT-KeyVault-2022-3-1024x290-02d8d8d5.png)
