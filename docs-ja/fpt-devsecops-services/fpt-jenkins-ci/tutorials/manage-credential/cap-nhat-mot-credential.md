# クレデンシャルの更新

クレデンシャルの情報を更新するには：

**ステップ 1:** FPT Portal メニュー > **Jenkins CI** > **Detail Cluster** > **Credential Managements** に移動します。

[![](/img/migrated/Picture49-64efc5d7.png)](/img/migrated/Picture49-64efc5d7.png)

### Secret Text

**ステップ 2:** **Credential Managements** 画面で、編集する Secret Text クレデンシャルを選択 > **Actions** > **Edit**。

[![](/img/migrated/Picture50-0e401103.png)](/img/migrated/Picture50-0e401103.png)

**ステップ 3:** クレデンシャル情報を編集します。

[![](/img/migrated/Picture51-dbcf07bc.png)](/img/migrated/Picture51-dbcf07bc.png)

**ステップ 4:** 更新後の結果。

[![](/img/migrated/Picture52-7c7e78e0.png)](/img/migrated/Picture52-7c7e78e0.png)

### Username with password

**ステップ 2:** **Credential Managements** 画面で、編集する Username with password クレデンシャルを選択 > **Actions** > **Edit**。

[![](/img/migrated/Picture53-c95f4268.png)](/img/migrated/Picture53-c95f4268.png)

**ステップ 3:** クレデンシャル情報を編集します。

[![](/img/migrated/Picture54-0edd4bec.png)](/img/migrated/Picture54-0edd4bec.png)

**ステップ 4:** 更新後の結果。

[![](/img/migrated/Picture55-cda81603.png)](/img/migrated/Picture55-cda81603.png)

### GitLab API token

**ステップ 2:** **Credential Managements** 画面で、編集する GitLab API token クレデンシャルを選択 > **Actions** > **Edit**。

[![](/img/migrated/Picture56-c87fb773.png)](/img/migrated/Picture56-c87fb773.png)

**ステップ 3:** クレデンシャル情報を編集します。

[![](/img/migrated/Picture57-23d42fb3.png)](/img/migrated/Picture57-23d42fb3.png)

**ステップ 4:** 更新後の結果。

[![](/img/migrated/Picture58-0f86bc09.png)](/img/migrated/Picture58-0f86bc09.png)

### SSH Username with private key

**ステップ 2:** **Credential Managements** 画面で、編集する SSH Username with private key クレデンシャルを選択 > **Actions** > **Edit**。

[![](/img/migrated/Picture59-5564d869.png)](/img/migrated/Picture59-5564d869.png)

**ステップ 3:** クレデンシャル情報を編集します。
