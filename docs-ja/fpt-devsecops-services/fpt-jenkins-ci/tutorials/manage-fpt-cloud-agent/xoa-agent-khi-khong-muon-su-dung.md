# エージェントの削除

不要になったエージェントを削除するには：

**ステップ 1:** FPT Portal メニュー > **Jenkins CI** > **Detail Cluster** > **Agent Managements** > 削除するエージェントの **Actions** をクリックします。

[![](/img/migrated/Picture23-5442fdee.png)](/img/migrated/Picture23-5442fdee.png)

**ステップ 2:** **Delete** を選択してエージェントを削除します。

[![](/img/migrated/Picture24-3dd2460d.png)](/img/migrated/Picture24-3dd2460d.png)

**ステップ 3:** 確認ダイアログで **delete** と入力し、**Confirm** をクリックします。

[![](/img/migrated/Picture25-675a92fd.png)](/img/migrated/Picture25-675a92fd.png)

**ステップ 4:** エージェント削除後の結果。エージェントは Portal に表示されなくなります。

[![](/img/migrated/Picture26-c4890775.png)](/img/migrated/Picture26-c4890775.png)

エージェントは Jenkins からも削除されます。

[![](/img/migrated/Picture27-b7f58552.png)](/img/migrated/Picture27-b7f58552.png)
