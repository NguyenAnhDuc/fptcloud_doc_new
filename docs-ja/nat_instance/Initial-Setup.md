# 初期設定

FPT Cloudを初めてご利用の場合は、開始前に以下の手順を完了してください。

## FPT Cloudアカウントの作成とFPT Portalへのログイン

FPT Cloudサービスを利用するには、FPT Portalにログインする必要があります。

1. [https://console.fptcloud.com](https://console.fptcloud.com/) にアクセスし、FPT Cloudアカウントの認証情報でログインします。
2. 使用するサービスに合わせて、正しいTenant、Region、VPCを選択します。

アカウントをお持ちでない場合は、**Sign Up**をクリックして必要事項を入力してください。サポートチームが連絡してお客様の情報を確認し、アカウントを作成します。

:::note
操作中にエラーが発生した場合は、すぐにサポートチームにお問い合わせください。
:::
