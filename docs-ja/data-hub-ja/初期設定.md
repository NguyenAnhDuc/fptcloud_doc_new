# 初期設定

### 1. 契約するユーザー場合
  * **FPT クラウドアカウントの作成** ：<https://id.fptcloud.com/> にアクセスし、[サインアップ]をクリックして、アカウント登録するために必要な情報を入力します。次に、FPT Smart Cloud (noreply@fptcloud.com)からの指示メールを確認し、登録の完了とアカウントを承認します。

[![file](/img/migrated/image-1742891973865-70c602e1.png)](/img/migrated/image-1742891973865-70c602e1.png)
  * **テナントのプロビジョニングを待つ**
    * FPTは、お客様のアカウントにテナントを提供します。(ユーザーがSGリージョンのAIStudioクォータを持つテナントに追加され、モデルハブにアクセスできることを確認してください) 。
ユーザーが新しいテナントを排他的に使用する必要がある場合は、サポートチーム(ホットライン:1900 638 399 またはメール：support@fptcloud.com)に連絡してテナントクォータを付与させてもらってください。
    * テナントの準備が整うと、メールで通知が届きます。
  * **サインイン**
    * 通知が届いたら、ログインページに移動します <https://ai.fptcloud.com/>
    * 資格情報を入力してサインインします。
    * 正常にサインインしたら、FPTから電子メールで送信されたプロビジョニングされたテナントに切り替えて、モデルの微調整を使用します。

### 2. 従量課金制のユーザーの場合
  * FPT クラウドアカウントの作成：<https://ai.fptcloud.com/> にアクセスし、[サインアップ]をクリックして、必要な情報を入力し、アカウントを登録します。次に、FPT Smart Cloud (noreply@fptcloud.com)からの指示メールを確認し、登録の完了とアカウントを認証します。

[![file](/img/migrated/image-1742891973865-70c602e1.png)](/img/migrated/image-1742891973865-70c602e1.png)
  * FPT AI Factory ポータルにログインします: <https://ai.fptcloud.com/> に移動し、FPTクラウドアカウントとパスワードでログイン

[![file](/img/migrated/image-1742891982448-756e7f02.png)](/img/migrated/image-1742891982448-756e7f02.png)
  * ログインできたら、AI Factory ポータルに移動します。

[![file](/img/migrated/image-1740036087795-6bc5c01a.png)](/img/migrated/image-1740036087795-6bc5c01a.png)
