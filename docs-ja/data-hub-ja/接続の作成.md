# 接続の作成

* ステップ1: でオブジェクトストレージ管理にアクセスし <https://console.fptcloud.com/> 、「バケット」を選択し、「バケットを作成」をクリックします。 

  * **ステップ 2** ：新しいオブジェクトをバケットにアップロードします。

  * **ステップ3** ：Data Hubサービスにアクセスし、接続リストに移動して、「接続を作成」をクリックします [![file](/img/migrated/image-1742974995525.33.34-760613b1.png)](/img/migrated/image-1742974995525.33.34-760613b1.png)
  * 接続を作成するには、必要な情報を入力する必要があります。 [![file](/img/migrated/image-1742975565417.51.05-8588afb6.png)](/img/migrated/image-1742975565417.51.05-8588afb6.png)
    * **名前：** 接続名を入力します 
      * **有効な文字:** 文字 (a から z、大文字と小文字を区別しない)、数字 (0から 9)、ハイフン (-) のみ。
      * **長さ:** 3〜64文字
      * **データソース：** データへの接続を有効にするためのデータソースを選択します
    * **エンドポイントURL：** 接続するバケットのエンドポイントアドレス例: <https://test-bucket-name.s3-han02.fptcloud.com>
    * **アクセスキー と 秘密キー:** アクセス許可の認証情報
  * **[保存]** をクリックして 、接続を作成します。
