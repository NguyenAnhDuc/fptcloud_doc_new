# バックアップ＆リカバリー履歴を表示する

**FPT Portal**では、**Backup Job**の作成/削除、リストアポイントの復元/クローンなど、**Backup & Recovery**サービスに対するすべての操作履歴を表示できます。

**Backup & Recovery**履歴を表示するには、以下の手順に従ってください：

メニューで**Backup & Recovery** > **Backup & Recovery Management**を選択します。**History**タブを開きます。

**主な操作タイプ：**

| **タイプ** | **説明** |
| --- | --- |
| **Create backup schedule** | 新しいBackup Jobを作成 |
| **Delete backup schedule** | Backup Jobを削除 |
| **Disable backup schedule** | Backup Jobを無効化 |
| **Enable backup schedule** | Backup Jobを有効化 |
| **Full backup** | フルバックアップを手動で開始 |
| **Restore** | リストアポイントからVMを復元 |
| **Clone** | リストアポイントから新しいVMをクローン |
