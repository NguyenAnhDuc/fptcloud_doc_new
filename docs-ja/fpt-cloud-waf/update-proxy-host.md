# Proxy hostの更新

1. **WAF Dashboard**で**Hosts** > **Proxy**を選択します。

   [![Proxy hostリスト](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. 更新するproxy hostの**Edit**アイコンをクリックします。

   [![Proxy hostのEditを選択](/img/migrated/Userguide-FPT-WAF-2022-19-1024x578-234f8c82.png)](/img/migrated/Userguide-FPT-WAF-2022-19-1024x578-234f8c82.png)

3. 更新する情報を入力します。

   [![Proxy host詳細の編集](/img/migrated/Userguide-FPT-WAF-2022-20-1024x578-e34b9e78.png)](/img/migrated/Userguide-FPT-WAF-2022-20-1024x578-e34b9e78.png)

4. **Update Proxy**をクリックして保存し、**Proxy Management**ページに戻ります。

   [![Proxy hostの保存](/img/migrated/Userguide-FPT-WAF-2022-21-1024x578-9594ad2a.png)](/img/migrated/Userguide-FPT-WAF-2022-21-1024x578-9594ad2a.png)
