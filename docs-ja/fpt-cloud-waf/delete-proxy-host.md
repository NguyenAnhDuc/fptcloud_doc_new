# Proxy hostの削除

:::danger
Proxy hostを削除すると、Webサーバーへの WAF保護が解除されます。この操作は慎重に行ってください。
:::

1. **WAF Dashboard**で**Hosts** > **Proxy**を選択します。

   [![Proxy hostリスト](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. 削除するproxy hostの**Delete**アイコンをクリックします。

   [![Proxy hostのDeleteを選択](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)

3. 確認ダイアログで**Delete**をクリックして削除を完了します。

   [![Proxy host削除の確認](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)
