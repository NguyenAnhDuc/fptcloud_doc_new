# WAFサーバーの編集

1. 変更するWAFサーバーを選択します。

   [![WAFサーバーの選択](/img/migrated/Userguide-FPT-WAF-2022-11-1024x538-67ed51c4.png)](/img/migrated/Userguide-FPT-WAF-2022-11-1024x538-67ed51c4.png)

2. WAFサーバーの詳細を編集します。更新できる項目: **Name**、**Description**、**Size**。

   [![WAFサーバー編集フォーム](/img/migrated/Userguide-FPT-WAF-2022-12-1024x538-e643bd5f.png)](/img/migrated/Userguide-FPT-WAF-2022-12-1024x538-e643bd5f.png)

   :::warning
   - サイズを変更する場合、現在のサイズより**大きい**サイズのみ選択できます。
   - サイズの変更により、完了するまで数分間のダウンタイムが発生する場合があります。トラフィックの少ない時間帯に実施してください。
   :::

3. **Update WAF**をクリックして変更を適用します。
