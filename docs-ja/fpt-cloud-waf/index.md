# FPT Cloud WAF

FPT Cloud Web Application Firewall（FPT Cloud WAF）は、DOS、Local and Remote File Inclusion、SQL Injection、Cross Site Scripting、NodeJS・Java・PHPの脆弱性を利用した攻撃などからWebアプリケーションを保護するファイアウォールサービスです。

## 主な機能

- Small、Medium、Largeの3つの構成サイズで迅速にセットアップできます。
- OWASP Top 10の脅威からWebアプリケーションを保護します。
- ボットネットトラフィックを自動的に監視、分析、ブロックします。
- SSL certificateの登録と自動更新をサポートします。
- ゼロデイ攻撃を防ぐため、仮想セキュリティパッチを継続的に更新します。

## ドキュメント構成

- **初期設定** — アカウントを作成してFPT Portalに初めてログインします。
- **クイックスタート** — WAFサーバーの追加、dashboardへのログイン、接続の設定を行います。
- **詳細ガイド** — WAF Dashboardのすべての機能を管理します。
