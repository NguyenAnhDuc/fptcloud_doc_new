# Proxy hostの管理

Proxy hostはWAFの中心コンポーネントであり、エンドユーザーのリクエストをWAFサーバーを経由してWebサーバーに転送しながら、セキュリティルールセットを適用します。

Proxy hostの管理操作には以下が含まれます。

- [Proxy host詳細の表示](view-proxy-host-detail.md)
- [Proxy hostの更新](update-proxy-host.md)
- [Proxy hostの有効化/無効化](enable-disable-proxy-host.md)
- [Proxy hostの削除](delete-proxy-host.md)
