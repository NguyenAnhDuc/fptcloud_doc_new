# 初期設定

**FPT Cloud WAF**を初めてご利用の場合は、開始前に以下の手順を完了してください。

## アカウントの作成とFPT Cloud Portalへのログイン

**FPT Cloud WAF**サービスを使用するには、**FPT Portal**にログインする必要があります。

1. FPT Portalにアクセスし、FPT Cloudのユーザー名とパスワードでログインします。
2. 使用するサービスに対応したTenant、Region、VPCを選択します。

:::note
FPT Cloudアカウントをお持ちでない場合は、ホームページにアクセスして**Sign Up**をクリックし、画面の指示に従って情報を入力してください。サポートチームが詳細を確認し、アカウントを作成するためにご連絡します。
:::

操作中にエラーが発生した場合は、サポートチームにお問い合わせください。

以上の手順を完了すると、FPT Cloud上でWAFを使用する準備が整います。
