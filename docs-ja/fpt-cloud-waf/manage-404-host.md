# 404 hostの管理

**404 host** は、404エラーページを任意のページへリダイレクトする機能です。

1. メニューから **Hosts** > **404 Not Found** を選択して管理画面を開きます。

   [![404 host管理画面](/img/migrated/Userguide-FPT-WAF-2022-32-1024x502-a4e51c1a.png)](/img/migrated/Userguide-FPT-WAF-2022-32-1024x502-a4e51c1a.png)

2. **Create 404 Not Found** をクリックして新規作成画面を開き、情報を入力します。

   - **Domain Names**: 404エラー発生時にリダイレクトされるドメイン名。
   - **Custom Configuration**
   - **SSL Certificate**: ドメインに対応するSSL certificateを選択します。

   [![404 host情報の入力](/img/migrated/Userguide-FPT-WAF-2022-33-1024x502-98cf258a.png)](/img/migrated/Userguide-FPT-WAF-2022-33-1024x502-98cf258a.png)

3. **Create 404 Not Found** をクリックして保存します。
