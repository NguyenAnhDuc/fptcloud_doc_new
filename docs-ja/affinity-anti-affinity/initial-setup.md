# 初期設定

初めてサービスを利用する場合は、以下の手順を完了してください。

## 前提条件

- FPT Cloudアカウントを持っていること。
- FPT Cloud Portalにログイン済みであること。

## FPT Cloud Portalへのログイン

1. [https://console.fptcloud.com](https://console.fptcloud.com/) にアクセスします。
2. FPT Cloudの資格情報でログインします。
3. 使用するサービスに対応する **Tenant**、**Region**、**VPC** を選択します。

:::tip
FPT Cloudアカウントをお持ちでない場合は、[https://fptcloud.com](https://fptcloud.com/) にアクセスして **Sign Up** を選択し、登録してください。サポートチームが確認のためご連絡いたします。
:::
