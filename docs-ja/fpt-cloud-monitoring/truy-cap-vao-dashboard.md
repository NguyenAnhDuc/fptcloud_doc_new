# 7.1. にアクセスします dashboard

**ステップ1:** にログインします FPT portal cloud **<https://console.fptcloud.com>**
**ステップ2:** Trên menu của FPT Cloud Portal, click **Monitoring**. 
**ステップ3:** Click chi tiết vào một Workspace, sau đó di chuyển tới **Configuration Data** tab.
**ステップ4:** Click vào dashboard url, dashboard sẽ tự động login bằng Single Sign On (SSO)
