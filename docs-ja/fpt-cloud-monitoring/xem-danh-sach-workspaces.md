# ワークスペース一覧を表示する

**手順 1:** FPT Cloud Portal（[https://console.fptcloud.com](https://console.fptcloud.com)）にログインします。

**手順 2:** FPT Cloud Console Portal のメニューで **Monitoring** をクリックします。ワークスペース一覧が表示されます。

[![](/img/migrated/Screenshot-2025-04-16-145353-2e29b6c0.png)](/img/migrated/Screenshot-2025-04-16-145353-2e29b6c0.png)

各ワークスペースエントリには以下の情報が表示されます:

| フィールド | 説明 |
|---|---|
| **Workspace name** | ワークスペース名。作成後は変更できません。 |
| **Package Type** | ワークスペースのサービスパッケージ。FMON では TRIAL、BASE、PRO、ADVANCE の 4 種類を提供しています。 |
| **Status** | ワークスペースの稼働状態。 |
| **Created At** | ワークスペースの作成日時。 |
| **Metric Usages** | 現在のメトリクスデータ使用量（単位: metric records）。 |
| **Log Usages** | 現在のログデータ使用量（単位: GB）。 |
| **Trace Usages** | 現在のトレースデータ使用量（単位: GB）。 |
