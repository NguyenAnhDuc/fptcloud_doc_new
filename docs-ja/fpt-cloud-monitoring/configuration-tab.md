# Configuration タブ

**手順 1:** FPT Cloud Portal（**[https://console.fptcloud.com](https://console.fptcloud.com)**）にログインします。

**手順 2:** FPT Cloud Portal のメニューで **Monitoring** をクリックし、ワークスペース一覧を表示します。

**手順 3:** ワークスペース名をクリックして詳細を表示します。

**手順 4:** **Configuration** タブを選択します。

[![](/img/migrated/Screenshot-2025-04-16-145854-56757256.png)](/img/migrated/Screenshot-2025-04-16-145854-56757256.png)

Configuration タブには以下の情報が表示されます:

**ダッシュボード情報**
- **Dashboard ID:** ダッシュボードページの ID。
- **Dashboard URL:** ダッシュボードページの URL。

**Contact Point**
- **Basic Authentication – Username:** 認証用のユーザー名。
- **Basic Authentication – Password:** 認証用のパスワード（`*` でマスク表示）。
- **Webhook URL:** Webhook URL。

**データソース**
- **Endpoint – remote_write / query URL:** メトリクス、ログ、トレースのエンドポイント URL。
- **Datasource ID:** メトリクス、ログ、トレースの ID。
- **Datasource Secret / Password:** データソースのパスワード（`*` でマスク表示）。
