# 10. Quản trị User

Cho phép user phân quyền được vào grafana dashboard với 1 trong các quyền: Admin, Editor, Viewer.
**ステップ1:** にログインします FPT portal cloud <https://console.fptcloud.com>
**ステップ2:** Trên menu của FPT Cloud Portal, click **Monitoring** hiển thị danh sách workspace
**ステップ3:** Click tên workspace để xem chi tiết, hiển thị màn hình thông tin chi tiết của workspace đó.
**ステップ4:** Click chọn tab **“User Management”**.
**ステップ5:** Click nút **Add user**
[![Alt text](/img/migrated/Picture19-2-f56180f8.png)](/img/migrated/Picture19-2-f56180f8.png)
Có thể add thêm user vào Dashboard Grafana với các role như: Admin, Editor, Viewer. Với mỗi role sẽ có các quyền sử dụng khác nhau.
