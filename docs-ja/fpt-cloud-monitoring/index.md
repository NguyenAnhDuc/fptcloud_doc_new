# FPT Cloud Monitoring

FPT Cloud Monitoring (FMON) は、FPT Cloud 上のインフラ、アプリケーション、パブリックエンドポイントのメトリクス、ログ、トレースを監視する包括的な可観測性プラットフォームです。Grafana をベースに構築されており、ダッシュボード、アラート、データ探索を 1 つのワークスペースで提供します。
