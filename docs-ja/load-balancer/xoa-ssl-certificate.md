# SSL証明書の削除

**SSL証明書**を削除するには、以下の手順に従ってください。

1. **SSL Certificate**画面で、削除したい証明書のゴミ箱アイコンを選択します。
2. 確認ダイアログが表示されたら、**Delete SSL Certificate**を選択して削除を実行するか、**Cancel**を選択してキャンセルします。
