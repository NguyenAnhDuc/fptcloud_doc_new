# SSL 証明書の詳細確認（クラシック）

**Manage SSL Certificate for Load balancer** セクションで、VPC 内に作成されたすべての SSL 証明書を確認できます。

[![](/img/migrated/image-1712808075335-de5bd0cf.png)](/img/migrated/image-1712808075335-de5bd0cf.png)

特定の SSL 証明書を選択すると、その証明書の **Name** と **Public Key** が表示されます。

[![](/img/migrated/image-1712808088237-03369751.png)](/img/migrated/image-1712808088237-03369751.png)
