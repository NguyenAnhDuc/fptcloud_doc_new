# サーバープールメンバーの追加・削除

**メンバー**はロードバランサーを通じて転送されたリクエストを処理する**サーバープール**内のサーバーです。

サーバープールメンバーはロードバランサー作成時に自動的に作成されます。

## メンバーの追加

**手順 1:** サーバープール一覧でメンバーを追加するサーバープールをクリックします。

[![](/img/migrated/image-1710989399033-82cb406f.png)](/img/migrated/image-1710989399033-82cb406f.png)

**手順 2:** **Add server** で**インスタンス選択**パネルに利用可能なインスタンスが表示されます。追加するインスタンスを選択します。

[![](/img/migrated/image-1710989438271-ca679a80.png)](/img/migrated/image-1710989438271-ca679a80.png)

各メンバーを設定します：

- **Port**: メンバーがリクエストを受け取るポート。
- **Weight**: リクエスト分散のための優先度の重み。重みが高いメンバーほど多くのリクエストを受け取ります。

## メンバーの削除

サーバープールメンバーを削除するには、メンバーの **Action** メニューで **Remove** をクリックします。

[![](/img/migrated/image-1710989470976-16ee6e91.png)](/img/migrated/image-1710989470976-16ee6e91.png)
