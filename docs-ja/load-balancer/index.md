# ロードバランサー

FPT Cloud ロードバランサーは、受信トラフィックをバックエンドサーバー（Webサーバー、Webアプリ、コンテナなど）のグループに分散し、速度とパフォーマンスを最大化します。

設定が完了すると、FPT ロードバランサーはトラフィックを自動的にルーティングし、プール内のサーバー間でリクエストを均等に分散します。各サーバーのヘルス状態を継続的に監視し、利用不能になったサーバーからトラフィックを自動的に切り替えます。

## ロードバランサーを作成する前に確認すること

- **サーバー台数:** ワークロードを計算して適切なバックエンドサーバー数を選択します。
- **ロードバランサーパッケージ:** FPT Cloud は Basic から Premium までのパッケージを提供しており、それぞれ最大容量の仕様が異なります。
- **地理的な場所 (Region):** トラフィックの発生源に最も近い Region を選択して速度を最適化します。
- **負荷分散アルゴリズム:** FPT は Round Robin と Least Connections を提供します。それぞれのトレードオフを理解して最適なものを選択します。
