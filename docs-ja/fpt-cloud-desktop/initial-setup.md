# 前提条件

FPT Cloud Desktopにアクセスする前に、以下の条件を満たしていることを確認してください。

## SSOログイン（Azure AD）

- インターネットまたはLAN経由でFPT Cloud DesktopリンクにアクセスできるPCまたはノートパソコン
- 管理者から提供されたバーチャルデスクトップへのアクセス情報一式（FCD URL、アカウント情報、対応するAuthenticatorアプリ）

## Account/Passwordログイン

- インターネットまたはLAN経由でFPT Cloud DesktopリンクにアクセスできるPCまたはノートパソコン
- 管理者から提供されたバーチャルデスクトップへのアクセス情報一式（FCD URLおよびアカウント情報。FCD FCI URL：`fci.fcd.fptcloud.com`）
