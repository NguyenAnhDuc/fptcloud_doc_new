# 詳細概要

## 従来のPCに対する優位性

- **データセキュリティ：** 暗号化ストレージ、集中管理、アクセス制御
- **継続的なセキュリティパッチ適用：** 一元的かつ一貫した更新により、時間とリソースを節約
- **障害時のデータ整合性確保：** 自動バックアップとディザスタリカバリソリューションの組み合わせ
- **迅速かつ柔軟なリソース拡張/縮小：** 数時間で数千台のバーチャルデスクトップをプロビジョニング

## 対応接続デバイス

- デスクトップ、PC、ノートパソコン
- モバイル（スマートフォン）またはタブレット
- ThinClientまたはFPT MiniPC

## アクセス方法

- **Native FCDClient（推奨）：** アプリケーションを通じてバーチャルデスクトップに直接アクセスし、最もスムーズで安定した体験を提供します。
- **Webブラウザ：** インストール不要でブラウザから素早くアクセスできます。

## 用語と定義

| 用語（英語） | 用語（ベトナム語） | 略語 | 定義 |
| --- | --- | --- | --- |
| **Virtual Desktop** | Máy ảo / Máy tính ảo | VD | 物理コンピューターと同等のソフトウェア機能とパフォーマンスを提供する仮想マシン。バーチャルデスクトップはFPT Cloudインフラを共有します。各ユーザーは異なる構成とOSで複数のバーチャルデスクトップを同時に割り当てられます。 |
| **Desktop Pool** | Nhóm máy ảo | Pool | 同じCloudの処理・ストレージインフラを共有するバーチャルデスクトップのグループ。部門、機能、利用時間帯などでグループ化するために使用します。 |
| **Operating System** | Hệ điều hành | OS | 例：Microsoft Windows、macOS |
| **End-User / User** | Người dùng cuối | N/A | FCDバーチャルデスクトップのエンドユーザー。FCD User Portalへのログイン、割り当てられたバーチャルデスクトップへのアクセス、通常の物理コンピューターとしての使用が可能です。 |
| **Supported Device** | Thiết bị hỗ trợ | N/A | 割り当てられたバーチャルデスクトップにアクセスして使用できるデバイス |
| **Local Device** | Thiết bị kết nối | N/A | バーチャルデスクトップへの接続に使用するデバイス |
| **Local Area Network** | Mạng nội bộ | LAN | 組織内のコンピューターを接続するネットワークシステム |
| **Authenticator App** | Phần mềm xác thực | N/A | アカウントログインに使用する30秒間有効な6桁のランダムな認証コードを生成するアプリ |
| **Active Directory** | Quản lý thư mục và xác thực | AD | 集中型ユーザー管理システム |
