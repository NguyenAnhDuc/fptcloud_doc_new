# FPT App Catalogs

FPT Smart Cloud は、ポータルインターフェース上で FPT App Catalog サービスを正式に提供しています。このサービスを使用すると、App Catalog から Kubernetes プラットフォーム経由でアプリケーションをデプロイできます。また、App Catalog の管理、チャートの検索とデプロイ、Helm Chart アプリケーションの管理、クラスター管理などをポータルインターフェースから簡単に行えます。
