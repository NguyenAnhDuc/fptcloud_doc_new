# リポジトリから App Catalog を検索する

App Catalog を検索するには、以下の手順を実行します。

1. **App Catalogs** 画面でデプロイしたいチャートを検索します。
2. 検索したい情報を入力します。

例として、bitnami リポジトリを選択し、クラスターにデプロイする bitnami の `cert-manager` チャートを検索します。
