# デプロイ済みアプリケーションの詳細を表示する

FPT Cloud では、デプロイ済みアプリの詳細情報を以下の手順で確認できます。

1. デプロイ済みアプリを選択します。

   [![デプロイ済みアプリ選択](/img/migrated/Picture23-1-418a191b.png)](/img/migrated/Picture23-1-418a191b.png)

2. デプロイ済みアプリの詳細情報が表示されます。

   - アプリのステータス
   - Revisions list: デプロイ / アップグレードの回数情報
   - Resource
   - Manifest
   - Values

   [![アプリ詳細画面](/img/migrated/Picture24-1-e30daa6d.png)](/img/migrated/Picture24-1-e30daa6d.png)
