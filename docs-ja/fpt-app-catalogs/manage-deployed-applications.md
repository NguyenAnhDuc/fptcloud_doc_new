# デプロイ済みアプリケーションの管理

App Catalogs Management では、クラスターにデプロイ済みのアプリケーションを管理できます。このメニューから、クラスター別のアプリフィルタリング、デプロイ済みアプリの詳細表示、再設定 (Reconfig)、ロールバック、アンインストールなどの操作が行えます。
