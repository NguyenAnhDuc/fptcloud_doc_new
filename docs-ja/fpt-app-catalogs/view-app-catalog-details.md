# App Catalog の詳細を表示する

作成済みの App Catalog クラスターで作業するには、まずクラスターの詳細情報を表示します。

**Catalog Id** をクリックして詳細情報を表示します。

[![Catalog Id の選択](/img/migrated/Picture5-1-6d637f64.png)](/img/migrated/Picture5-1-6d637f64.png)

App Catalogs Management では、各リポジトリに対応した App Catalog の詳細情報が表示され、アプリケーションをデプロイするための Kubernetes クラスターを管理できます。

- 最初の画面では、各リポジトリに対応した App Catalog の詳細情報が表示されます。

  [![リポジトリ一覧](/img/migrated/Picture6-1-144a7e3f.png)](/img/migrated/Picture6-1-144a7e3f.png)

- 次のメニュー **Deployed Apps** では、デプロイ後のアプリケーションを管理します。

  [![Deployed Apps メニュー](/img/migrated/Picture7-1-97eba568.png)](/img/migrated/Picture7-1-97eba568.png)

- 最後のメニュー **Management Clusters** では、アプリのデプロイに使用する Kubernetes クラスターを管理します。

  [![Management Clusters メニュー](/img/migrated/Picture8-1-e08365c5.png)](/img/migrated/Picture8-1-e08365c5.png)
