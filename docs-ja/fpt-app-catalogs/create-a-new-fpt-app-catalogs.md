# 新しい App Catalog を作成する

新しい **App Catalog** を作成するには、以下の手順を実行します。

1. **FPT Portal** のメニューから **App Catalogs** を選択します。

   [![App Catalogs メニュー](/img/migrated/Picture1-1-58bf0bcc.png)](/img/migrated/Picture1-1-58bf0bcc.png)

2. **Create App Catalog** をクリックし、App Catalog 作成に必要な情報を入力します。

   [![App Catalog 作成フォーム](/img/migrated/Picture2-1-e371fa76.png)](/img/migrated/Picture2-1-e371fa76.png)

3. **Create** をクリックします。システムが App Catalog クラスターを作成し、処理状況を通知します。

   [![作成処理中](/img/migrated/Picture3-1-f536a3bd.png)](/img/migrated/Picture3-1-f536a3bd.png)

App Catalog の作成が成功すると、クラスターのステータスが **Succeeded** に変わります。その後、ポータルインターフェースから App Catalog クラスターにアクセスしてアプリを管理できます。

[![作成成功](/img/migrated/Picture4-1-d851e811.png)](/img/migrated/Picture4-1-d851e811.png)
