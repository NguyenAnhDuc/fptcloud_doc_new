# App Catalog Management を削除する

不要になった App Catalogs Management を削除できます。

1. App Catalogs 管理画面で削除したい Catalog Id を選択し、**Delete** をクリックします。

   [![Delete ボタン](/img/migrated/Picture38-1-bfecf9e8.png)](/img/migrated/Picture38-1-bfecf9e8.png)

2. 確認画面で **delete** と入力して削除を確認します。

   [![削除確認](/img/migrated/Picture39-1-97218177.png)](/img/migrated/Picture39-1-97218177.png)
