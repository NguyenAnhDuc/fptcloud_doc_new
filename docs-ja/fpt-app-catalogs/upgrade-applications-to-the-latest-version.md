# アプリケーションを最新バージョンにアップグレードする

システムでは、アプリのバージョンをリポジトリ内の最新バージョンにアップグレードできます。ただし、アップグレード前に影響範囲を評価する必要があります。

1. アプリの詳細画面で **Upgrade to** を選択します（最新バージョン以外がインストールされている場合にボタンが表示されます）。

   [![Upgrade to ボタン](/img/migrated/Picture33-1-a6c06913.png)](/img/migrated/Picture33-1-a6c06913.png)

2. 希望する設定をカスタマイズします。

   [![アップグレード設定](/img/migrated/Picture34-1-e7b42635.png)](/img/migrated/Picture34-1-e7b42635.png)

3. **Confirm** をクリックしてチャートのアップグレードを実行します。

   :::note
   アップグレードの成否はチャートをインストールするクラスターによって異なります。チャートのバージョンによってサポートされる Kubernetes クラスターのバージョンが異なります。
   :::
