# アプリケーションをアンインストールする

クラスターにインストールされたチャートを削除できます。

1. クラスター詳細画面で **Uninstall** ボタンをクリックします。

   [![Uninstall ボタン](/img/migrated/Picture35-1-825b929d.png)](/img/migrated/Picture35-1-825b929d.png)

2. チャートの削除を確認するため **uninstall** と入力します。

   [![アンインストール確認](/img/migrated/Picture36-1-9e7c9fb3.png)](/img/migrated/Picture36-1-9e7c9fb3.png)

3. **Uninstall** をクリックしてチャートを削除し、結果を確認します。

   [![アンインストール結果](/img/migrated/Picture37-1-566a9f4c.png)](/img/migrated/Picture37-1-566a9f4c.png)
