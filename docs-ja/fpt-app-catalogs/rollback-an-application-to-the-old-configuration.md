# アプリケーションを旧設定にロールバックする

システムでは、アプリケーションを以前の設定にロールバックできます。

1. アプリの詳細画面で **Rollback to #** を選択します。

   [![Rollback ボタン](/img/migrated/Picture30-1-da0a3712.png)](/img/migrated/Picture30-1-da0a3712.png)

2. 確認のため **Rollback** と入力します。

   [![ロールバック確認](/img/migrated/Picture31-1-45650e38.png)](/img/migrated/Picture31-1-45650e38.png)

3. ロールバック後の情報を確認します。

   [![ロールバック結果](/img/migrated/Picture32-1-a330cd39.png)](/img/migrated/Picture32-1-a330cd39.png)
