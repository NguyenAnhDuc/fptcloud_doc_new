# Router Groupの削除

**Router Group** を削除するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **Event Gateway** を選択します。

**ステップ2:** **Event Gateway** リストで **Event Gateway** を選択 > **Router group** タブを選択します。

**ステップ3:** **Router group** リストで削除したいグループを選択 > **Action** をクリック > **Delete** を選択します。

[![Actionメニューが開いたRouter Groupリスト](/img/migrated/image25-3bec02b1.png)](/img/migrated/image25-3bec02b1.png)

**ステップ4:** **Delete Application** ダイアログが表示されます > **Delete** をクリックして **Router Group** の削除を完了します。

[![Router Group削除確認ダイアログ](/img/migrated/image26-fb5457d4.png)](/img/migrated/image26-fb5457d4.png)
