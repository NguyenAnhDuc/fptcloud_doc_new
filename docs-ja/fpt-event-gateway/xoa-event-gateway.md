# Event Gatewayの削除

**Event Gateway** を削除するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **Event Gateway** を選択します。

**ステップ2:** **Event Gateway** リストで、削除したい **Event Gateway** を選択 > **Action** をクリック > **Delete** を選択します。

[![Actionメニューが開いたEvent Gatewayリスト](/img/migrated/image1b-e0892577.png)](/img/migrated/image1b-e0892577.png)

**ステップ3:** **Delete Application** ダイアログが表示されます > **delete** と入力 > **Confirm** をクリックして **Event Gateway** の削除を完了します。

[![削除確認ダイアログ](/img/migrated/image1c-ca16f7c7.png)](/img/migrated/image1c-ca16f7c7.png)
