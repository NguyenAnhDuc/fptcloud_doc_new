# Quick Start Jp

**アカウントの登録**
**ステップ 1: アカウントを作成**
  1. <https://ai.fptcloud.com/> にアクセスし、FPT ID を使用してサインアップするか、Google アカウントで続行してください。
  2. FPT ID を使用する場合は、**FPT Smart Cloud** からの確認メールでアカウントを確認してください。

**ステップ2：アカウントにログイン**
  1. [https://ai.fptcloud.com/に戻り、ログインしてください](https://ai.fptcloud.com/%E3%81%AB%E6%88%BB%E3%82%8A%E3%80%81%E3%83%AD%E3%82%B0%E3%82%A4%E3%83%B3%E3%81%97%E3%81%A6%E3%81%8F%E3%81%A0%E3%81%95%E3%81%84)。

**ステップ3: AIノートブックを起動**
  1. 左メニューから「**AIノートブック** 」を選択してください。ノートブックの起動には約1**～2分かかります** 。
  2. 詳細なガイドはこちらをご覧ください。

**ステップ4: アカウントにクレジットを追加**
1. **「アカウント」** セクションに移動し、**「請求」** をクリック
  2. 「**クレジットを追加** 」ボタンをクリックし、金額と支払い情報を入力して完了します。

または、FPT発行のバウチャーをお持ちの場合は、**「バウチャーを追加** 」セクションで有効なコードを入力し、クレジットを交換してください。
  3. 詳細なガイドはこちらをご覧ください。

**ステップ5: AIノートブックの作成**
  1. ランチャーから、CPU/GPU接続のノートブックを作成できます
  2. 詳細なガイドはこちらをご覧ください

**ステップ6：ノートブックを新規/既存カーネルに接続**
  1. 新しいカーネルを起動するには、**カーネル設定メニューの「その他のカーネルを起動」** から目的のカーネルを選択してください
  2. 既存カーネルに接続するには、**カーネル設定メニューの** 「**実行中のカーネル」** から目的のカーネルを選択してください
  3. 詳細なガイドはこちらをご覧ください。

**ステップ7: カーネルのシャットダウン**
  1. **実行中のカーネル/ノートブック** （ナビゲーションバー上から2番目のアイコン）に移動します。
  2. 詳細なガイドはこちらをご覧ください。
