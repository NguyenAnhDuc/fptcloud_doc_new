# Create Jp

[![Alt text](/img/migrated/Screenshot-2025-09-24-at-15.45.42-0c7fedab.png)](/img/migrated/Screenshot-2025-09-24-at-15.45.42-0c7fedab.png)
## CPUに接続するノートブックを作成する:
**ステップ1:**
  * **「+」ボタンをクリック（1）**
  * **CPUノートブック** を選択**（2）**

**ステップ2** : ノートブックの準備が整ったら:ノートブックの左下に「CPU (free)| idle」と表示されます選択したフレーバーが右上の**カーネル設定メニュー内「実行中カーネル」セクションに** 表示されます
**ステップ3** : コーディングを開始する 
## GPUに接続するノートブックを作成する:
**ステップ1** : 開始前に、アカウントに少なくとも**1時間のGPU利用料を** カバーできる十分なクレジットがあることを確認してください。詳細なガイドはこちらをご覧ください。 
**ステップ2** : **希望のGPU** を搭載したノートブックを選択する **(3)**
**ステップ3** : 確認ポップアップで**「確認」** を選択
[![Alt text](/img/migrated/worddavcbde47ee263356d9fcbb156662265af0-2f05a191.png)](/img/migrated/worddavcbde47ee263356d9fcbb156662265af0-2f05a191.png) **ステップ4** : ノートブックの準備が整うまで約**1～2分** お待ちください。
**ステップ5** : ノートブックの準備が完了したら:
  * ノートブックの左下に「GPU フレーバー名| idle」と表示されます
  * 選択したフレーバーが右上の**カーネル設定メニュー内「実行中カーネル」セクションに** 選択済みとして表示されます

**ステップ6:** コーディングを開始する
