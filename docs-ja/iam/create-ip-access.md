# 新しいIP Accessレコードの作成

1. **IAM** を選択し、**IP access** タブをクリックします。

[![](/img/migrated/image-1711511716480-fe4d7e22.png)](/img/migrated/image-1711511716480-fe4d7e22.png)

2. **Create** を選択します。

[![](/img/migrated/Screenshot_20-1024x366-374cb3a8.png)](/img/migrated/Screenshot_20-1024x366-374cb3a8.png)

3. IP Access作成ページで情報を入力し、設定前に案内をよく読みます：
   - **IP Access**: 1つまたは複数のIP、IP範囲、またはCIDR IPを選択できます。
   - **Users**: 1人または複数のユーザーを選択できます。
   - **User groups**: 1つまたは複数のUser Groupを選択できます — グループ内の全ユーザーに適用されます。

[![](/img/migrated/Screenshot_17-1024x549-20d82693.png)](/img/migrated/Screenshot_17-1024x549-20d82693.png)

4. **Create ip access** を選択します。作成が成功すると、レコードが一覧に表示されます。

[![](/img/migrated/Screenshot_19-1024x362-978a41cd.png)](/img/migrated/Screenshot_19-1024x362-978a41cd.png)
