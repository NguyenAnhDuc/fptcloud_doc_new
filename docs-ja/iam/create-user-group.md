# 新しいUser Groupの作成

1. **IAM** > **User Groups** を選択し、**Create User Group** をクリックします。

[![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. **Create User Group** ページで必要な情報を入力します：
   - **Name**: ユーザーグループ名。
   - **Description**: ユーザーグループの説明（任意）。
   - **Role**: User Groupに複数のRoleを割り当てることができます。
   - **User**: システムに存在するユーザーをUser Groupに追加できます。

[![](/img/migrated/image-1734344557099-e110b8a0.png)](/img/migrated/image-1734344557099-e110b8a0.png)

3. **Create user group** を選択します。システムが作成を実行し、結果を通知します。
