新しい IP アクセスレコードの作成

**手順 1**: メニューで **IAM** を選択し、**IP access** タブを選択します。ここでアクセスを許可するユーザーの一覧を管理できます。

![file](images/create-ip-/img-001.png)

**手順 2**: **Create** を選択します。

![](images/create-ip-/img-002.png)

**手順 3**: IP アクセス作成ページで、IP アクセスアドレス情報とユーザーリストを入力します。設定前にガイドをよくお読みください。その後「Create ip access」をクリックします。

  * IP Access: 1 つまたは複数の IP、IP レンジ、または CIDR IP を選択できます。

  * Users: 1 つまたは複数のユーザーを選択できます。

  * User groups: 1 つまたは複数のユーザーグループを選択できます。グループ内のすべてのユーザーに適用されます。

![](images/create-ip-/img-003.png)

作成が成功すると、レコードが一覧画面に表示されます。

![](images/create-ip-/img-004.png)
