# Role管理

Role管理機能により、Tenant ownerはFPT IAMシステム内のRoleを確認、作成、編集、削除できます。FPT IAMは2種類のRoleをサポートします：

- **Basic Role**: Super AdminとReadonly — 編集・削除不可。
- **Custom Role**: カスタム権限セットを持つユーザー作成のRole。

詳細な手順はこのページのサブセクションを参照してください。
