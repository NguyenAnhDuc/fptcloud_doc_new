# User Group一覧の表示

1. FPT Portalで **IAM** > **User Groups** を選択します。
2. User Groupの一覧が表示されます。各User Groupには **Name**、**Description**、**Roles**、**User quantity**、**Created at**、**Actions** が表示されます。
