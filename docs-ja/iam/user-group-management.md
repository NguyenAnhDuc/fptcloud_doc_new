# User Group管理

User Group管理機能により、Tenant ownerはUser Groupを作成、確認、編集、削除できます。各User Groupには1つまたは複数のRoleが割り当てられ、同じUser Group内の全IAM userにそれらの権限が適用されます。

詳細な手順はこのページのサブセクションを参照してください。
