# User Groupの編集

1. **IAM** > **User Groups** を選択します。**User Groups Management** ページが表示されます。
2. 一覧から更新するUser Groupの末尾にある **Action** を選択します。
3. 以下のオプションから選択します：
   - **Manage user**: User Group内のユーザーを追加または削除します。
   - **Manage role**: User Groupに割り当てられたRoleを追加または削除します。
   - **Edit information**: User Groupの名前と説明を編集します。
