# User Groupの編集

1. メニューから **IAM** > **User Groups** を選択します。
2. 更新したいUser Groupの末尾にある **Action** を選択し、操作を選びます：

   - **Manage user**: グループ内のユーザーを追加または削除します。
   - **Manage role**: グループに割り当てられたroleを追加または削除します。
   - **Edit information**: グループ名と説明を更新します。
