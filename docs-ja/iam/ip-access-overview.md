# IP Access概要

IP Access機能はシステムアクセスを制御し、許可されたIPアドレスリストに含まれるIPからのみユーザーのログインを許可します。許可されたIPリストに含まれないユーザーアカウントはシステムにアクセスできません。

- アクセスを許可するIPに対応するユーザーリストを正確に特定する必要があります。
- 設定中は、現在アクセスしているパブリックIPアドレスと自分のアカウントを追加してください（最初のレコードを優先）。これにより設定プロセスが中断されません。

:::warning
IP Access機能はTenant Ownerには適用されません — 全Tenant OwnerはIPによるアクセス制限を受けません。
:::
