# 初期設定

FPT IAMの利用を開始するには、以下の作業を確認して完了してください。

## FPT Portalでのアカウント登録

1. https://fptcloud.com/ のトップページにアクセスします。
2. **Sign Up** を選択し、システムの案内に従って情報を入力します。
3. サポートチームが連絡を取り、アカウント作成のための情報を確認します。

## FPT Portalへのログイン

1. https://console.fptcloud.com/ にアクセスします。
2. 提供されたアカウントとパスワードでログインします。
3. 権限設定を行う正しいTenantを選択します。

:::note
上記の情報が不明な場合、または3回試行後にシステムエラーが表示される場合は、すぐにサポートチームにお問い合わせください。
:::

上記の操作を完了すると、FPT IAMを使用する準備が整います。
