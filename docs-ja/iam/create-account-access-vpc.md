# アカウントの作成と招待されたVPCへのアクセス

Tenant ownerが招待を送信すると、システムは登録されたメールアドレスに確認メールを送信します。招待されたユーザーはメールを開いて招待を承認し、アカウントを有効化する必要があります。

- **メールアドレスがFPT IDアカウントにない場合**: このメールアドレスを使用してFPT IDアカウントを作成する必要があります。
- **メールアドレスがすでにFPT IDアカウントにある場合**: そのFPT IDアカウントでFPT Cloudサービスを有効化するだけです。
