# アカウントの作成と招待されたVPCへのアクセス

Tenant ownerが招待を送信すると、システムは申告されたメールアドレスに確認メールを送信します。招待されたユーザーはメールを開いて招待を確認し、アカウントを有効化する必要があります。

- **メールがFPT IDアカウントに登録されていない場合**: ユーザーはこのメールを使ってFPT IDアカウントを作成する必要があります。
- **メールがすでにFPT IDアカウントに登録されている場合**: ユーザーはそのFPT IDアカウントでFPT Cloudサービスを有効化するだけです。
