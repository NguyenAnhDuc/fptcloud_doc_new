# Fpt_Images

**FPT Image** là custom image được xây dựng bởi **FPT** mà bạn có thể sử dụng để **bắt đầu nhanh chóng với bất kỳ GPU VM nào hiện có**.
Image này đi kèm với một số thành phần cần thiết cho **khối lượng công việc AI** và chọn **Ubuntu** làm **Hệ điều hành (OS).**
Các phiên bản của các dependency được cài đặt* được **tối ưu hóa cho khả năng tương thích** và có thể không phải là phiên bản mới nhất hiện có.

| **Hệ điều hành**  | **Phiên bản**  |
| --- | --- |
| Ubuntu  | 22.04  |
| Ubuntu  | 24.04  |

[![Alt text](https://fptcloud.com/wp-content/uploads/2025/03/OS-Images-1-1024x494.png)](https://fptcloud.com/wp-content/uploads/2025/03/OS-Images-1-1024x494.png)

* Dependency không bao gồm trong OS.

| **Dependencies**  | **Phiên bản**  |
| --- | --- |
| **NVIDIA Driver**  | 560.35.03  |
| **NVIDIA CUDA Toolkit**  | 12.6.85  |
| **NVIDIA Fabric Manager (NVSwitch Driver)**  | 560.35  |
| **NVIDIA Datacenter GPU Manager**  | 3.3.9  |
