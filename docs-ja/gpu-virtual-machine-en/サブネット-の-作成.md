# サブネット の 作成

GPU仮想マシン(インスタンス)を作成するには、VPC内にアクティブなサブネットが必要です。システムは、このサブネットから新規仮想マシンにプライベートIPを割り当てます。
新しいサブネットを作成する手順:
**ステップ1：** サイドメニューで、**[Network]** > **[Subnets]** に移動し、**[Create subnet]** を選択します。
[![Alt text](/img/migrated/1-057589e4.png)](/img/migrated/1-057589e4.png)
**ステップ2：** 必要な情報を入力します。
  * **名前：** サブネットの認識可能な名前を入力します。
  * **タイプ：** 現在、2つのタイプがサポートされています。
    * **ルート：** サブネットはNATゲートウェイを介してインターネットにルーティングされます。
    * **孤立:** サブネットにはインターネットルーティングがありません。
  * **ネットワークアドレス(CIDR):** 有効なCIDRを入力します。
  * **ゲートウェイIP：** 有効なゲートウェイアドレスを入力します。
  * **静的IPプール(オプション):** 使用するIP範囲を指定します。空白のままにすると、システムはCIDR範囲全体を割り当てます。
  * **プライマリDNS:** ドメイン名解決に必要なDNSサーバーです。
  * **セカンダリDNS（オプション）：** プライマリDNSサーバーに障害が発生した場合のバックアップDNSサーバーです。
  * **タグを追加(オプション)：** サブネットに割り当てるタグを選択します。

[![Alt text](/img/migrated/2-b8e89ee4.png)](/img/migrated/2-b8e89ee4.png)
**ステップ3：** 新しいサブネットを作成するには、 **「Create subnet」** をクリックします。システムはリクエストを処理し、結果を表示します。
成功すると、新しく作成された**サブネットがサブネット** テーブルに表示されます。
