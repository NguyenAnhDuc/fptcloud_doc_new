# 概要

**GPU 仮想マシン(GPU VM)** を使用すると、**高性能 GPU サーバーを簡単に導入および管理** できます。当社のプラットフォームは、**クラウド コンピューティングの柔軟性と専用GPU** のパ**ワー、スピード、効率性** を組み合わせて、AI、機械学習、データ集約型のワークロードに最適なパフォーマンスを保証します。
このドキュメントでは、**FPT クラウド ポータル** を使用して**GPU VM を** 管理する方法について説明します。
**GPU VMとは?**
**GPU VMはパススルー GPU を使用して専用 GPU を取得し、** アプリケーションはゲストOS とハイパーバイザーのレイヤーを介してそれにアクセスします。RAM、ストレージ、ネットワークなど、アプリケーションが使用するその他の重要なVMリソースも仮想化されます。
運用方法
GPU VMは、集中的なワークロードのために専用のGPUを備えた強力なクラウドベースのコンピューターのように運用します。できること：
  * FPTクラウドポータルを介して、最新のGPU世代を搭載したGPU VMを数分で簡単に展開できます。
  * 高度なGPUアクセラレーションにより、AI、機械学習、データ処理タスクを高速に実行できます。
  * 必要に応じてリソースを拡張し、ダウンタイムなしで CPU、RAM、ストレージ、または追加のGPUを追加できます。
  * 社内にITスタッフを置く必要がなく、シンプルなオンラインポータルからすべてを管理できます。

GPU仮想マシンを選択する理由
  * FPTスマートクラウドポータルからのワンクリック展開。
  * ネットワーク、ストレージ、セキュリティのシンプルな構成。
  * より高速な AI トレーニングと深層学習:最先端のGPUは、大規模なAIおよび機械学習モデルの処理に最適な、優れた速度と効率を実現します。
  * AI推論向けに最適化:これまで以上に速くデータを処理し、リアルタイムで意思決定を行います。
