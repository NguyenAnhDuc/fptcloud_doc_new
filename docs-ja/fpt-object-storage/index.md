# FPT Object Storage の概要

**FPT Object Storage** は FPT Cloud が提供するクラウドストレージサービスで、画像・動画・ドキュメント・その他の大容量ファイルなど非構造化データの保存・管理のために設計されています。高い信頼性、高可用性、柔軟なスケーラビリティを備え、大規模なデータ保存に対応します。

Amazon の S3 プロトコルおよび Azure Blob と完全互換に設計・開発されており、企業の既存アプリケーションやサービスと容易に統合できます。また、データの管理と保存において高い柔軟性と利便性を提供します。

[![overview](/img/migrated/2-FPT-Object-Storage-Service_anh-sp-1-f482e951.png)](/img/migrated/2-FPT-Object-Storage-Service_anh-sp-1-f482e951.png)

## 主な特長

- **柔軟なスケーラビリティ:** 技術的な制限なく、必要に応じて大容量データの保存要件に対応・拡張できます。
- **データセキュリティの保証:** データは保存時および転送中に暗号化されます。versioning、SSE-C などの高度な機能をサポートしています。
- **統合性:** Amazon S3 および Azure Blob プロトコルと互換性があり、S3 クライアント、SDK、一般的な API を通じた接続が可能です。
- **厳格なアクセス管理:** FPT IAM、bucket policy、bucket ACL、object ACL、subuser を通じた権限管理をサポートしています。
- **Unify Portal での簡単管理:** FPT Unify Portal の直感的なインターフェースにより、データの管理・監視・操作が容易です。
- **コスト最適化:** 実際に使用したリソースの料金のみお支払いいただきます（Pay-As-You-Go）。

## メリット

- **高い信頼性:** 分散ストレージシステムにより、ハードウェア障害が発生した場合でも、データは常に保護され、いつでもアクセスできます。
- **高いパフォーマンス:** 大量のデータを高速に処理するよう設計されており、現代のアプリケーションのアクセス時間要件を満たします。
- **使いやすさ:** 直感的な管理インターフェースにより、ユーザーがデータを簡単に操作・管理できます。

FPT Object Storage は、強力で安全かつ既存システムとの統合が容易なデータストレージを必要とする企業にとって理想的なソリューションです。
