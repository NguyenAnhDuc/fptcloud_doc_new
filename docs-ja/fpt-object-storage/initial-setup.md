# 初期設定

**FPT Object Storage** の利用を開始するには、以下の手順を完了してください。

## ステップ 1: FPT Cloud アカウントの作成とログイン

1. [https://fptcloud.com/](https://fptcloud.com/) のトップページにアクセスします。
2. **Sign Up** を選択し、システムの案内に従って情報を入力します。サポートチームが連絡を取り、情報を確認したうえでアカウントを作成します。
3. FPT Portal にログインするには、FPT Cloud にアクセスし、発行されたアカウントとパスワードを入力します。
4. ログイン後、**Tenant**、**Region**、**VPC** などのアカウント情報が表示されます。

## ステップ 2: FPT Object Storage サービスの登録

FPT Cloud アカウントをお持ちで **Object Storage** のご利用を希望される場合は、サポートチームまでお問い合わせください。必要な情報を提供いただいた後、アカウントに **Object Storage** 管理機能が有効化されます。
