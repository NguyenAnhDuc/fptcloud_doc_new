# API Gatewayの詳細確認

**API Gateway** の詳細情報を確認するには、以下の手順に従ってください。

**ステップ1:** メニューバーで **Integration** > **API Gateway** を選択します。

**ステップ2:** **API Gateway** 一覧から、確認したいAPI Gatewayを選択します。

- **Overview** タブ: Basic information、Launch Configuration、Etcd、LB size、Endpointなどの基本情報が表示されます。

[![画像](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)

- **Autoscaling** タブ: **API Gateway** のオートスケーリング設定が表示されます。

[![画像](/img/migrated/SCR-20250924-nuvs-077cada2.png)](/img/migrated/SCR-20250924-nuvs-077cada2.png)

- **Dashboard** タブ: API GatewayのDashboard情報が表示されます。作成時に **Dashboard = Disable** を選択した場合、このタブは表示されません。

[![画像](/img/migrated/z7045379122842_6adcfbe63dafe93587e68b587-a1a084d9.jpg)](/img/migrated/z7045379122842_6adcfbe63dafe93587e68b587-a1a084d9.jpg)
