# FPT API Gateway

  * [ ![](/img/migrated/initial-setup-714537aa.png) Initial Setup ](javascript: "Initial Setup")
  * [ ![](/img/migrated/icon-networking-2-09c1ef04.png) Hướng dẫn chi tiết (Tutorial)  ](javascript: "Hướng dẫn chi tiết \(Tutorial\) ")

  2. FPT API Gateway

**API Gateway** là thành phần trung gian giúp quản lý, kiểm soát và định tuyến các yêu cầu (requests) giữa client và các dịch vụ backend. Nó hoạt động như một cổng truy cập tập trung cho toàn bộ API, đảm bảo việc giao tiếp an toàn, thống nhất và dễ dàng mở rộng. 
**API Gateway** hỗ trợ các tính năng như xác thực, phân quyền, giới hạn tần suất (rate limiting), cân bằng tải, giám sát và quản lý vòng đời API. Với **APISIX Dashboard** , người dùng có thể cấu hình trực quan các route, service, upstream, plugin và chứng chỉ TLS, giúp đơn giản hóa việc quản trị và vận hành. 
API Gateway phù hợp với các kiến trúc microservices và hệ thống phân tán, thường được sử dụng để tăng cường bảo mật, tính linh hoạt và khả năng quan sát trong việc quản lý API.
