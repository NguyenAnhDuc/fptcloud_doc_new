# ネットワーキング詳細

FCIのネットワークアーキテクチャは、従来型ネットワークシステムからVPC、または特殊なNo-gatewayモデルまで、多様なニーズに対応するよう設計されています。

## No-gatewayモデル

ルーターとして機能するサーバー（例：Check-pointファイアウォール）を通じて独自のシステムをセットアップし、管理することができます。IPはサーバーに直接割り当てられ、トラフィックはファイアウォールを通じてルーティングされます。
