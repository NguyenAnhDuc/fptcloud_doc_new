# Gateway

GatewayはVPCとインターネットまたは外部ネットワーク間のネットワーク接続を管理します。

:::note
この機能は、特別な設定を持つ一部のテナントのみが利用できます。有効化するにはサポートチームにお問い合わせください。
:::

## Gatewayへのアクセス

1. [console.fptcloud.com](https://console.fptcloud.com) にログインします。
2. Gatewayデータを確認したいVPCを選択します。
3. サイドバーで **Network** メニューを展開し、**Gateway** を選択します。
