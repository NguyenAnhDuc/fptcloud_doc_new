# マシンのリセット

使用中に画面が真っ黒になる、リモートサーバーに接続できない、マシンがフリーズするなどの問題が発生することがあります。2〜3分待って再接続しても解決しない場合は、マシンをリセットする必要があります。

## リセット手順

1. HPC Portalの **Hosts** 画面でイメージを選択し、**Reset machine** をクリックします。
2. 確認ダイアログで **Reset** をクリックします。

   [![Reset machine確認ダイアログ](/img/migrated/reset-1a4b229d.jpg)](/img/migrated/reset-1a4b229d.jpg)

3. 数分待ってから、RDPファイルを再度ダウンロードして再接続します。

:::warning
- マシンのリセットは、マシンが応答しない場合に物理サーバーのリスタートボタンを押すのと同等の操作です。
- データが失われる可能性があるため、慎重に使用してください。
- リセット後もマシンに接続できない場合は、管理者にサポートを依頼してください。
:::
