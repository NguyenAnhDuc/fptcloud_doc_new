# Instance Group管理 — AffinityとAnti-Affinity

Instance Groupを使用すると、物理ホスト上のインスタンスの配置を制御できます。同じグループ内のインスタンスを同じホストに配置（Affinity）するか、異なるホストに配置（Anti-Affinity）するかを指定し、パフォーマンスと可用性を最適化します。
