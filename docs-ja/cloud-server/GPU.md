# Portal ConsoleでのGPU管理

FPT GPU Serverは、FPT Cloud上に展開された仮想マシンサービスで、専用GPUカードを搭載しています。AI、Machine Learning、グラフィックスレンダリング、画像処理において卓越した演算能力を提供します。

本サービスは、FPT Cloud Serverの柔軟性と、NVIDIA Tesla A10、A30、A100、RTX 3090、RTX 4090などのGPUカードの強力なパフォーマンスを組み合わせています。

:::tip
GPU サービスの利用や技術的な問題が発生した場合は、カスタマーサポートチームまでお問い合わせください。
:::
