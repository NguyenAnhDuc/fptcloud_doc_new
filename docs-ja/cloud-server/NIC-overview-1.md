# NIC（ネットワークインターフェースカード）の管理

NIC（Network Interface Card）は仮想マシンをクラウド上のネットワークリソースに接続する重要なコンポーネントであり、高性能で信頼性の高いデータ転送を実現します。

:::warning
これはネットワークシステム管理の知識を持つユーザー向けの高度な機能です。設定変更後は必ずテストを行い、カードが正常に動作していることを確認してください。
:::
