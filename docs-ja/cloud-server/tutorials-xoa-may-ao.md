# 仮想マシンの削除

使用する必要がなくなった場合、追加コストの発生を防ぐために仮想マシンをシステムから削除できます。

:::danger
仮想マシンを削除するとすべてのデータが削除され、復元できません。
:::

## 仮想マシンの削除

1. **Instance Management** で **Actions** → **Delete** を選択します。

   [![DeleteオプションのあるActionsメニュー](/img/migrated/image-1712200768564-6389b6eb.png)](/img/migrated/image-1712200768564-6389b6eb.png)

   [![Deleteの選択確認](/img/migrated/image-1712200780901-998413fe.png)](/img/migrated/image-1712200780901-998413fe.png)

2. 警告ダイアログに仮想マシン名が表示されます。確認のために **delete** と入力します。

   [![削除確認ダイアログ](/img/migrated/image-1712200787729-9546cde9.png)](/img/migrated/image-1712200787729-9546cde9.png)

   [![deleteと入力して確認](/img/migrated/image-1712200798790-c16164fa.png)](/img/migrated/image-1712200798790-c16164fa.png)

3. システムが削除を実行してリソースを回収します。

   [![削除中の仮想マシン](/img/migrated/image-1712200803567-91c8fdd5.png)](/img/migrated/image-1712200803567-91c8fdd5.png)
