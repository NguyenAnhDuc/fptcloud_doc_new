# Token の失効

Token が不要になった場合や漏洩が疑われる場合は、直ちに失効させてセキュリティを確保してください。

## Token の失効

1. **Token** ページで **Actions** → **Revoke token** をクリックします。

   [![Revoke token オプションがある Actions メニュー](/img/migrated/image-1731025026905-158de23e.png)](/img/migrated/image-1731025026905-158de23e.png)

2. ダイアログで失効を確認します。

   [![Revoke token 確認ダイアログ](/img/migrated/image-1731025048482-957c5ad2.png)](/img/migrated/image-1731025048482-957c5ad2.png)

3. システムが Token を自動的に失効させます。

   [![Token が失効された状態](/img/migrated/image-1731025071939-e64b80bc.png)](/img/migrated/image-1731025071939-e64b80bc.png)
