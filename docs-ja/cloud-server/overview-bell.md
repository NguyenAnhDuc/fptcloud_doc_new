# 通知ベル

通知ベル（ベルアイコン）は新しいイベントが発生したときに通知します。実行したアクションのステータスに素早くアクセスできます。

## 使い方

1. FPT Cloud Portalのヘッダーにあるベルアイコンをクリックします。
2. アクションの詳細とステータスを含む通知の一覧を確認します。
