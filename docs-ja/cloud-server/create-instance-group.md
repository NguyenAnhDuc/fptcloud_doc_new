# 新しいInstance Groupの作成

AffinityまたはAnti-Affinityポリシーを使用してインスタンスの配置を制御するために、新しいInstance Groupを作成します。

:::tip
Instance Groupを作成する前に、[AffinityとAnti-Affinityポリシー](./overview-affinity.md)を確認して、ワークロードに適した配置戦略を選択してください。
:::
