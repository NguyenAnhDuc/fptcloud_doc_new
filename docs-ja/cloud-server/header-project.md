# ヘッダーインターフェースの変更

FPT Cloud Portalのヘッダーをカスタマイズして、アクティブなProjectを表示します。これにより、現在どのProjectコンテキストで作業しているかを簡単に識別できます。

:::tip
ヘッダーにProject名を表示することで、誤ったProjectにリソースを作成してしまうことを防ぐことができます。
:::
