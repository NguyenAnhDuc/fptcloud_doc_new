# Priority Rule管理

:::note
この機能は、特別な設定を持つ一部のテナントのみが利用できます。有効化するにはサポートチームにお問い合わせください。
:::

Priority Ruleを使用すると、ファイアウォールルールの実行順序を設定できます。トラフィックを許可または拒否する際に、どのルールを最初に評価するかを決定します。

## メリット

- 重要度に応じてルールを並べ替えられます。
- 汎用ルールよりも前に詳細なルールが処理されるようにできます。

## 優先度の設定

1. Security Groupで変更したいルールを選択します。
2. **Priority** の値を設定します（数値が小さいほど優先度が高い）。
3. **Save** をクリックします。
