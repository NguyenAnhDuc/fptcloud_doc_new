# Storage Disk の削除

Storage Disk が不要になった場合、削除することで追加のコスト発生を防ぐことができます。

:::danger
いずれの Instance にもアタッチされていない Storage Disk のみ削除できます。削除したデータは復元できません。
:::

## Storage Disk の削除

1. Storage Disk 一覧で **Actions** → **Delete** をクリックします。

   [![Delete オプションがある Actions メニュー](/img/migrated/image-1712215573891-02608fa9.png)](/img/migrated/image-1712215573891-02608fa9.png)

2. ポップアップで内容を確認し、**Delete storage disk** をクリックします。

   [![Storage Disk 削除確認ダイアログ](/img/migrated/image-1712215582910-1868700d.png)](/img/migrated/image-1712215582910-1868700d.png)
