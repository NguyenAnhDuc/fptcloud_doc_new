# ScheduleへのInstanceのアタッチ

## インスタンスのアタッチ

1. **Instance Schedule Management** で **Action** → **Apply instances** をクリックします。

   [![Actionメニュー](/img/migrated/image-1695609073911-03ac0ac2.png)](/img/migrated/image-1695609073911-03ac0ac2.png)

2. アタッチするインスタンスを選択し、**Apply instances** をクリックします。

   [![インスタンスの選択](/img/migrated/image-1695609084366-c567a73d.png)](/img/migrated/image-1695609084366-c567a73d.png)
