# Security Groupのフィルター

フィルター機能を使用すると、名前、タグ、IP、ステータス、ファイアウォールタイプ、作成日など複数の条件でSecurity Groupリストをすばやく検索・絞り込むことができます。

## フィルター条件

| フィールド | 説明 |
|-----------|------|
| **Name** | 完全な名前または部分的な名前を入力します。例：`sec` と入力すると "security-group-9609" が見つかります。 |
| **Tag** | 付与されたタグでフィルタリングします。 |
| **IP** | IPアドレスでフィルタリングします。 |
| **Status** | ステータスでフィルタリングします。 |
| **Type** | Security GroupまたはDistributed Firewall。 |
| **Created at** | 作成日でフィルタリングします。 |
