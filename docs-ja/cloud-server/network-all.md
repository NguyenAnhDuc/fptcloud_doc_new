# ネットワーキング概要

FPT Cloud（FCI）のネットワークシステムは、企業向けの包括的なソリューションを提供しています。従来型ネットワークから仮想プライベートクラウド（VPC）、またはNo-gatewayモデルまで、ネットワークインフラの構築・管理が可能です。

:::note
No-gatewayモデルでは、ルーターとして機能するサーバー（例：Check-pointファイアウォール）を通じて独自のシステムをセットアップし、管理することができます。IPはサーバーに直接割り当てられ、トラフィックはファイアウォールを通じてルーティングされます。
:::
