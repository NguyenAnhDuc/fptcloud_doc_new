# ネットワーキング概要

Hệ thống mạng FPT Cloud (FCI) cung cấp giải pháp toàn diện cho doanh nghiệp — cho phép xây dựng và quản lý hạ tầng mạng từ mô hình truyền thống đến mạng riêng ảo (VPC) hoặc mô hình No-gateway.

:::note
Với mô hình No-gateway, bạn có thể tự thiết lập hệ thống riêng, quản lý thông qua máy chủ đóng vai trò router (ví dụ: Check-point firewall). IP được chỉ định trực tiếp vào máy chủ và traffic điều hướng qua firewall.
:::
