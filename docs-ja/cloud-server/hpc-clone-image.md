# Imageのクローン

Clone Imageを使用すると、既存のイメージとすべてのストレージデバイスを複製できます。環境をゼロからセットアップし直す必要はありません。

:::warning
クローンを実行する前に、すべての環境設定が完了していることを確認し、マシンをシャットダウンしてください。
:::

## Imageのクローン手順

1. イメージを選択し、**Clone** をクリックします。
2. 作成するマシンの台数を入力し、**Clone** をクリックします。
