# Token に関するよくある質問

## 一部の Token が失効できないのはなぜですか？

FPT Cloud Portal のアップグレード前に作成された一部の古い Token は失効をサポートしていません。システムには「Tokens created earlier cannot be revoked」というツールチップが表示されます。

## Token が「Expired」ステータスに変わるのはいつですか？

Token が作成時から設定された有効期限を過ぎると、システムは自動的にそのステータスを **Expired** に変更します。その Token を使用して認証することはできなくなります。
