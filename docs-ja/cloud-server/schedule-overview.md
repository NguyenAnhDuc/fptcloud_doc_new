Instance スケジュール概要


1つのスケジュールで仮想マシンの起動/停止アクションをすべて完了するには、選択した開始時刻または停止時刻から最大15分かかる場合があります。そのため、特定の時刻までにすべての仮想マシンを起動/停止させる必要がある場合は、必要な時刻より15分早めにスケジュールを設定し、各スケジュール間を少なくとも15分空けてください。スケジュールを作成した後は、時刻、頻度、繰り返し設定は編集できません。変更できるのは、スケジュールにアタッチまたはデタッチする仮想マシンのみです。スケジュールを変更したい場合は、現在のスケジュールを削除して新しいスケジュールを作成してください。システムでは、選択した開始時刻または停止時刻の少なくとも1時間前にしかスケジュールを作成できません。1時間の期間内では、起動または停止のいずれか1つのアクションのみ実行できます。
