# HPC Portalクイックスタート

HPC Portalでは、RTX 3090およびRTX 4090向けの物理GPU Serverを管理できます。以下の基本手順に従って開始してください。

## 前提条件

- ✅ HPC Portalアカウントを持っている（アクセス権の付与は管理者に連絡してください）。
- ✅ アカウントにGPUリソースが割り当てられている。

## はじめに

1. [HPC Portal](https://hpc.fptcloud.com/) にアクセスしてログインします。
2. **Hosts** 画面でイメージを選択し、**Start** をクリックしてマシンを起動します。
3. マシンが起動したら、**RDP** ファイルをダウンロードしてリモートデスクトップで接続します。
