# ストレージディスクSnapshotスケジュールの削除

:::danger
スケジュールの削除は元に戻せません。
:::

1. **Storage snapshot schedule** タブで **Action** → **Delete** をクリックします。

   [![Action Deleteメニュー](/img/migrated/image-1766045198605-b5baee8c.png)](/img/migrated/image-1766045198605-b5baee8c.png)

2. **delete** と入力し、**Delete schedule** をクリックします。

   [![削除の確認](/img/migrated/image-1766045228117-02dc5687.png)](/img/migrated/image-1766045228117-02dc5687.png)
