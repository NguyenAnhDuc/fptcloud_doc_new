# Security Group内のすべてのルールをカスタマイズできますか？

はい。Security Group内のすべてのルールを制限なくカスタマイズできます。FCIがネイティブにサポートするタイプに加えて、特定のニーズに合わせたカスタムルールを作成することもできます。
