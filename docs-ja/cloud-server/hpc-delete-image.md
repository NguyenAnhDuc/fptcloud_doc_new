# Imageの削除

不要になったイメージは、HPC Portalのマシンリストから削除できます。

:::danger
イメージの削除は取り消しできません。削除前に重要なデータをバックアップしていることを確認してください。
:::

## Imageの削除手順

1. 削除するイメージを選択し、**Action** → **Remove** をクリックします。
2. 確認ダイアログで **Remove** をクリックして削除を完了します。
