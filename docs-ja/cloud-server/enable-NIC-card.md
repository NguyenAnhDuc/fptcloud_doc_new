# Network Interface Cardを有効化する

無効化されたNICカードを再度有効化して、そのインターフェース経由のネットワーク接続を復元できます。

## NICカードを有効化する

1. **Instance Management** ページに移動し、対象のInstanceを選択します。
2. **NIC** タブに移動します。
3. 無効化されたNICカードを見つけ、**Actions** → **Enable** を選択します。
4. ダイアログボックスで操作を確認します。

NICカードのステータスが **Enabled** に変わり、ネットワーク接続が復元されます。
