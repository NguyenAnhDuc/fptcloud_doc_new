# ScheduleからのInstanceのデタッチ

デタッチ後、インスタンスは手動で操作するまで現在の状態を維持します。

## インスタンスのデタッチ

1. **Instance Schedule Management** で **Action** → **Apply instances** をクリックします。
2. デタッチするインスタンス名の横にある **x** をクリックします（またはリストボックス末尾の **x** をクリックしてすべてデタッチします）。
3. **Apply instances** をクリックします。

   [![ScheduleからInstanceをデタッチ](/img/migrated/image-1695609097406-f1f13ac3.png)](/img/migrated/image-1695609097406-f1f13ac3.png)
