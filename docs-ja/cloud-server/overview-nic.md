# NIC概要

FPT Cloud上のNIC（Network Interface Card）は仮想マシンをネットワークリソースに接続し、高性能で柔軟かつスケーラブルなデータ転送を実現します。

## 前提条件

- これはネットワークシステムの知識を持つユーザー向けの高度な機能です。
- NICカードを接続するネットワークを特定し、仮想マシンが効率的に通信・リソースアクセスできるようにしてください。
- 正しいIP address、サブネットマスク、ゲートウェイ、DNSを設定してください。
- 設定後は問題が発生しないことを確認するためにテストを行ってください。
