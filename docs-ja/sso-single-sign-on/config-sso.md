# SSOの設定

現時点では、FPT Cloud上でSSOをセルフサービスで設定することはできません。SSOサービスを利用するには、チケットを作成するかサポートチームに連絡し、以下の情報をご提供ください。

- システムにすばやくアクセスするためのsub-domain。SSO設定後は、以下のURLでFPT Cloudにアクセスします: `http://sub-domain.fptcloud.com`
- Tenant ownerとなるアカウントのメールアドレス
- SSOサービスを提供するIdentity Provider

技術サポートチームから、追加の内部設定情報についてご連絡いたします。
