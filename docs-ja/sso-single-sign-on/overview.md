# 概要

## 1. 概要

SSO（Single Sign-On）は、ユーザーが1つのIDを使って複数のアプリケーションやサービスに安全にアクセスできる認証メカニズムです。SSOを利用すると、毎回ログインすることなく多くのアプリケーションに同時にアクセスできます。SSOはフェデレーテッドアイデンティティ（federated identity）という概念に基づいており、信頼された独立したシステム間でID情報を共有することができます。

## 2. メリット

Single Sign-Onの主なメリットは以下の通りです。

- 記憶するusernameとpasswordの数を減らせます
- usernameとpasswordを入力する回数を減らせます
- ユーザーの認証情報が漏洩するリスクを低減できます
