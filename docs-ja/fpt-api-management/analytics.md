# Analytics

## 1. Analytics

:::note
**対象ユーザー:** API Provider
:::

プロバイダーは **Analytics メニュー**で API の利用状況の統計を確認できます。

現在 Analytics では以下の 2 つのグラフが表示されます。

- Consumer ごとの API 呼び出し数の統計グラフ
- API ごとの API 呼び出し数の統計グラフ

## 2. Logging

API Provider は Logging セクションで、各 Consumer の API 呼び出し情報を確認できます。呼び出し元 IP アドレス、呼び出し経路、時刻・メソッド・ステータス・base path などの詳細情報が含まれます。

1. メニューの **Application** > **Analytics** > **Logging** を選択します。
2. ログを確認したい Consumer を 1 つ選択します。システムは Consumer の API 呼び出し情報を含むログを表示します。
