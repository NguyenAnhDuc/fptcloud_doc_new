# Managed Gpu クラスターを削除

不要になったマネージド GPU クラスターについては、次の手順で削除できます。
**ステップ1:** メニューで、**[AIインフラストラクチャ] >[マネージドGPUクラスター]**を選択して、**[GPUクラスター管理]** ページが表示されます。アクセスするクラスターを選択します。
[![](/img/migrated/14-b4dc395d.png)](/img/migrated/14-b4dc395d.png)
**ステップ2:** 削除したいクラスターのアクション列で、**[削除]** を選択します。
[![](/img/migrated/15-05e09c5c.png)](/img/migrated/15-05e09c5c.png)
**ステップ3:** ポップアップの警告事項を確認し、**[削除]** をクリックします。
[![](/img/migrated/16-52efdb8d.png)](/img/migrated/16-52efdb8d.png)
