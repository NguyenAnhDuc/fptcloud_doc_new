# Secret情報を表示・更新する

PortalインターフェースからいつでもSecret値を表示・編集できます。変更を保存するたびに新しいバージョンが作成され、更新履歴が保持されます。

1. Secret Engine Management画面で表示するSecretを含むSecret Engineを選択する。

   [![Secret Engine Management画面](/img/migrated/v1-88a45c58.png)](/img/migrated/v1-88a45c58.png)

2. 表示または更新する**Secret**を選択する。

   [![利用可能なSecretが表示されたSecret一覧](/img/migrated/v2-90d1e587.png)](/img/migrated/v2-90d1e587.png)

3. **Edit Secret**画面で現在の情報を確認するか、新しい値を入力して更新する。**Edit**をクリックして変更を保存する。

   [![Key/Valueフィールドと編集ボタンが表示されたEdit Secret画面](/img/migrated/v4-534b24ad.png)](/img/migrated/v4-534b24ad.png)
