# 初期設定

FPT Key Vaultを使用する前に、FPT Cloudアカウントを持ち、FPT Portalにログインする必要があります。このステップにより、サービスを初期化するための正しいTenant、Region、VPCへのアクセスが確保されます。

## FPT Cloudアカウントを作成してFPT Portalにログインする

FPT Cloudアカウントをお持ちでない場合は、[https://fptcloud.com/](https://fptcloud.com/) にアクセスし、**Sign Up** をクリックして手順に従ってください。サポートチームがお客様の情報を確認し、アカウントを作成するために連絡します。

アカウントを取得したら、以下の手順を実行してください:

1. [FPT Cloud Portal](https://console.fptcloud.com) にアクセスします。
2. メールアドレスとパスワードを入力し、**Sign In** をクリックします。
3. サービスに使用する正しい **Tenant**、**Region**、**VPC** を選択します。

:::note
この手順中にエラーが発生した場合は、サポートチームに即座にお問い合わせください。
:::

以上の手順が完了すると、FPT CloudシステムでKey Vaultを作成する準備が整います。
