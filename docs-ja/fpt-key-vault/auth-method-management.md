# Auth Method管理

Auth MethodはアプリケーションまたはユーザーがVaultで認証してSecretのアクセストークンを取得する方法を定義します。FPT Key VaultはAppRole Auth Methodをサポートしており、マシン間認証に適しています。このセクションでAuth MethodとRoleの作成を行います。
