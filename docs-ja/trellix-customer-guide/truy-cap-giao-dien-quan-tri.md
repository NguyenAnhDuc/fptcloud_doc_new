# 管理インターフェースへのアクセス

Trellix 管理コンソールを使用すると、agent がインストールされたすべてのサーバーを監視し、グループごとにセキュリティポリシーを管理できます。正しくログインすることが、アンチウイルスシステムを効果的に運用するための最初のステップです。

1. ブラウザを開き、`https://am.fptcloud.com:8443` にアクセスします。FCI から提供されたユーザー名とパスワードを入力します。

   [![Trellix ログイン画面](/img/migrated/image-1765765382780-8e758eaf.png)](/img/migrated/image-1765765382780-8e758eaf.png)

2. ログインに成功すると、Trellix 管理インターフェースが表示されます。

   [![ログイン後の Trellix 管理インターフェース](/img/migrated/image-1765765391556-388ea73c.png)](/img/migrated/image-1765765391556-388ea73c.png)

   - **System Tree**: agent がインストールされたサーバーの一覧を表示します。
     - **New Subgroups**: 管理を分離するためのサブグループを作成します。各グループまたはサブグループに個別のセキュリティポリシーを適用できます。

   [![Trellix 管理コンソールの System Tree 表示](/img/migrated/image-1765765400751-504e9a39.png)](/img/migrated/image-1765765400751-504e9a39.png)
