# クイックスタート

Bare Metal GPU サーバーは、スタンドアロンまたはマルチノードクラスターで動作できる 8つの NVIDIA H100/H200 GPU を搭載した専用のシングルテナントサーバーであります。
## アカウントのサインアップ
  1. [https://id.fptcloud.com/にアクセスし、サインアップを選択します](https://id.fptcloud.com/%E3%81%AB%E3%82%A2%E3%82%AF%E3%82%BB%E3%82%B9%E3%81%97%E3%80%81%E3%82%B5%E3%82%A4%E3%83%B3%E3%82%A2%E3%83%83%E3%83%97%E3%82%92%E9%81%B8%E6%8A%9E%E3%81%97%E3%81%BE%E3%81%99)。
  2. 受信トレイに確認メールが見つからない場合は、迷惑メールフォルダを確認してください。
  3. Metal Cloud サービスの利用を開始するには、[https://console.fptcloud.com/にサインインしてください](https://console.fptcloud.com/%E3%81%AB%E3%82%B5%E3%82%A4%E3%83%B3%E3%82%A4%E3%83%B3%E3%81%97%E3%81%A6%E3%81%8F%E3%81%A0%E3%81%95%E3%81%84)。

## サブネットの作成。
  1. Bare Metal GPU を作成してデプロイするにはサブネットが必要であります。
  2. AI Infrastructure をクリックし、サイドメニューで Subnet を選択します。
  3. 詳細なガイドはこちらを参照してください。

**Bare Metal GPU サーバーの作成**
  1. **AI Infrastructure** をクリックし、サイドメニューで **Metal Cloud** を選択します。
  2. **Create server** を選択し、サーバーのデプロイメントを構成します。

  * 2 つの(02) **GPU と AI/ML 用の FPT 画像** 利用可能です。ソフトウェアの依存関係性については、こちらで確認してください。
  * 作成時にフローティング IP (パブリック IP)を選択してインターネット経由でサーバーにアクセスするか、後でネットワーク/フローティング IP でアタッチすることができます。

  3. 詳細なガイドはこちらを参照してください。

## サーバーへのアクセス
  1. こちらで Network ACL を設定して、パブリック IP を許可するインバウンドルールを作成します。
  2. 作成時に設定した SSH キーまたはパスワードでサーバーに SSH 接続します。
  3. このガイドで、ほとんどのセキュリティ設定のジャンプホストも設定します。

## 通知
  1. FPT クラウドコンソールにアクセスした後、**テナントとリージョンを確認して** ください。**AI ファクトリーは、ハノイ 2 (ベトナム) および東京 (日本) のリージョンでのみサポートされています。**
  2. **ネットワーク ACL のデフォルトを変更することはできますが、サーバーを作成するために必要なその他のルールを設定する必要があります。**

以下はネットワーク ACL のデフォルトのアウトバウンドルールであり、削除することができます。  
| 優先度  | 種類  | プロトコル  | ポート  | ソース  | トラフィックアクション  |  
| --- | --- | --- | --- | --- | --- |  
| 100  | ALL  | ALL  | ALL  | 0.0.0.0/0  | ALLOW  |  
上記のデフォルトルールを削除する場合は、サーバーを作成して展開するために次のアウトバウンドルールを追加する必要があります。  
| 優先度  | 種類  | プロトコル  | ポート  | ソース  | トラフィックアクション  |  
| --- | --- | --- | --- | --- | --- |  
| 1  | HTTP  | TCP  | 80  | 0.0.0.0/0  | ALLOW  |  
| 2  | HTTPS  | TCP  | 443  | 0.0.0.0/0  | ALLOW  |  
| 3  | DNS (UDP)  | UDP  | 53  | 0.0.0.0/0  | ALLOW  |  
  3. FPT は、お客様の契約/合意に従ってリソース量の制限(割り当て)を設定します。それが不足すると、新しいリソースを作成することはできません。営業担当者にお問い合わせいただくか、 **FPT Smart Cloud Support へご連絡ください。**
