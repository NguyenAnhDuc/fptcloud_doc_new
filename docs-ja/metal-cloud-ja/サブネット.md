# サブネットの作成。

サブネットは、クラウドネットワーク内の IP アドレスの範囲であります。この範囲のアドレスは、Metal Cloud の 1 つまたは複数のサーバーに割り当てられます。IPv4 のみがサポートされています。
_CIDR 範囲は自動的に割り当てられます。60 秒間保持され、その後画面上で新しいものに更新されます。 サブネットを作成するにはネットワーク ACL が必要です。デフォルトでは、BareMetal GPU サーバーからインターネットへのアウトバウンドトラフィックのみが許可されます。 サブネットのクォータは、Bare Metal GPU サーバーのクォータと同じです。_
  1. FPTCloud アカウントにサインインし、**テナント、リージョン、VPC** を選択します。（複数あるの場合）
  2. サイドバーの **AI Infrastructure** と **Subnet for Metal Cloud** に移動し、**Createsubnet** をクリックします。
  3. **サブネット名** を入力します
  4. ネットワーク **ACL 名** を入力します
  5. ネットワーク **ACL の説明** を入力します (オプション)
  6. **Create Subnet** をクリックします

# サブネットの削除
_他のリソースに関連付けられているサブネットを削除することはできません。_
  1. **AI Infrastructure** と **Subnet for Metal Cloud** に移動します
  2. サイドバーでサーバーリストを表示します
  3. サブネットを選択して**「Actions」** をクリックします 4.**Delete** をクリックします。 5.**DELETE** テキストを入力して削除を確認します

# サブネット名を更新
_サブネット名は既存の名と重複する可能性があります。_
  1. サブネットを選択して**「Actions」** をクリックします
  2. **Rename** をクリックします
  3. ルールに準拠する新しいサブネット名を入力します。名前は最大 32 文字までで、文字、数字、ダッシュのみ使用できます。
  4. **Rename subnet** をクリックします

# サブネットの説明を更新
  1. サブネットを選択して**「Actions」** をクリックします
  2. **Update description** をクリックします
  3. 新しい説明を入力します。長さは 63 文字以下でなければなりません
  4. 保存をクリックします。
