# コンソール KVM

SSH キーまたはパスワードを使用して、Bare Metal GPU サーバーに接続できます。SSH プロトコル (セキュアシェルとも呼ばれる)は、あるサーバーから別のサーバーに安全にリモートログインするメソードであります。SSH 経由で接続するには、着信トラフィックに必要なすべてのルールがネットワーク ACL 設定に含まれていることを確認します。
  1. サイドバーの **AI Infrastructure** と **Metal Cloud** に移動してサーバーリストを表示します。
  2. リストからサーバーを選択するか、サーバーの詳細を表示します
  3. **Actions** をクリックし、**Console** をクリックするか、サーバー詳細ページのコンソールで**Open** をクリックします。
  4. ログインするには、**デフォルトのユーザー名: clouduser** と、サーバーの作成時に入力したパスワードを使用します。

# SSH
_SSH キーは、サーバーを作成してデプロイするときに選択されます。SSH keymanagement をご参照くだいさい。_
## Windows 10 から接続する前の前提条件
  1. **Windows 設定** を開きます。
  2. **Apps & features** セクションに移動し、 **Optional features** をクリックします。
  3. **OpenSSH Client** を見つけてクリックすると、詳細な説明が展開されます。
  4. **Install** をクリックします。
  5. インストールが完了するまでお待ちください。SSH クライアントをインストールした後、設定を正しく適用するためにコンピューターを再起動します。SSH ユーティリティが cmd で使用できるようになります。

## Windows 10、Linux OS、macOS から接続
  1. コマンドプロンプトを開きます。

  
| 認証方法  | コマンド  |  
| --- | --- |  
| パスワード  | `ssh username@192.168.1.92`  |  
| SSH キー  | `ssh username@192.168.1.92 -i "C:\\Users\\username\\.ssh\\id_rsa`  |  
そこで：
  * 「username」を自分のユーザー名または初回ログイン時のデフォルトのユーザー名（clouduser）に換えます
  * 「192.168.1.92」をサーバーのフローティング IP アドレスに置き換えます。
  * 「C:\Users\username\\.ssh\id_rsa」を、コンピュータ上の PEM 形式のプライベートキーファイルへのパスに置き換えます。

_プライベートインターフェイスのみで Bare Metal GPU サーバーを作成した場合は、フローティング IP アドレスを作成し、SSH 経由で BareMetal に接続するときにそれを使用します。_
  2. ユーティリティは、不明なデバイスに接続しようとしていることを警告し、続行するかどうかを尋ねます。続行するには、**「yes」** と入力して **Enter** を押します。
  3. (パスワードのみで接続する手順) **Bare Metal** の作成時に設定したパスワードを入力します。

# 要塞ホスト（ジャンプサーバー）を使用する
ジャンプホストは、元のマシンと接続しようとしている Bare Metal GPU サーバーとの中間サーバーです。これは、2 つの信頼できるネットワーク間のゲートのように機能します。ジャンプホストがアクセスを許可した後でのみ、宛先サーバーにアクセスできます。
  1. サブネットとフローティング IP を持つクラウドインスタンスを作成します
  2. セキュリティグループを構成します
