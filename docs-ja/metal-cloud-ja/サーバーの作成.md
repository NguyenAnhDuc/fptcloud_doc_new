# サーバーの作成

FPT カスタマー ポータル経由でベアメタルサーバーをデプロイできます。FPT は、選択した設定に基づいてプロビジョニング、OS のインストール、構成を処理します。デプロイが完了すると、SSH キーまたはパスワードを使用して SSH 経由でサーバーにアクセスできるようになります。
  1. FPT Cloud アカウントにサインインし、テナント、リージョン、VPC を選択します。（複数お持ちの場合）
  2. サイドバーの AI Infrastructure と Metal Cloud に移動し、Create server をクリックします。
  3. フレーバーを選択します（可用性が地域によって異なります）。
  4. 一度にデプロイするサーバーの数を入力します。（最大数は割り当て量によって異なります。）
  5. サーバー名を入力します
  6. オペレーティングシステムを選択します必要な OS が提供されていない場合は、代わりにカスタム イメージを利用できます。
  7. 利用可能なサブネットを選択しますサブネットなしでサーバーを展開することはできません。サーバーを作成して展開する前に作成してください。
  8. フローティング IP のアタッチを選択します (オプション)
  9. 認証には、SSH キーまたはコンソールパスワードを選択できます。a. 利用可能な SSH キーを選択するか、新しいキーを追加します (SSH キーの作成と追加の詳細については、こちらをご覧ください)。
  10. RAID を構成します (オプション)
  11. ユーザーデータスクリプトを選択または作成します(オプション)。ユーザーデータスクリプトは、cloud-init プロセスを通じてサーバーの最初の起動時に自動的に実行されます。
  12. Create server をクリックします
