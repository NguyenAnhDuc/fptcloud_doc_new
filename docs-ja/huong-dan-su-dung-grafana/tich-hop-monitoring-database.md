# Monitoring Database の統合

FPT Database Engine の Monitoring は、FPT Monitoring システムと統合されており、metrics および logs の収集を行うとともに、必要に応じて dashboard とアラートを提供します。このガイドでは、FPT Cloud ポータルで Database クラスターの健全性とパフォーマンスを監視するための Monitoring へのアクセス、設定、および使用方法について説明します。
