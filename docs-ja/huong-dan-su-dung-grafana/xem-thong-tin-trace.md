# Trace 情報の確認

**ステップ 1:** メニューで **Explore** を選択します。

[![Alt text](/img/migrated/Screenshot_22-f2b7b75d.png)](/img/migrated/Screenshot_22-f2b7b75d.png)

**ステップ 2:** data source として **tempo** を選択します。

システムが **TraceQL** タブを開き、query を入力できます。

[![Alt text](/img/migrated/Screenshot_29-a8f6ab7d.png)](/img/migrated/Screenshot_29-a8f6ab7d.png)

**ステップ 3:** _Run query_ をクリックして結果を表示します。

_イメージ図_

[![Alt text](/img/migrated/Screenshot_30-e54fb90f.png)](/img/migrated/Screenshot_30-e54fb90f.png)

詳細については、Grafana の公式ドキュメント [こちら](https://grafana.com/docs/tempo/latest) をご参照ください。
