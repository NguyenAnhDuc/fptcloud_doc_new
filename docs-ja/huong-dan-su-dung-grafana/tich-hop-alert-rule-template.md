# Alert Rule Template の統合

**ステップ 1:** Grafana インターフェースでサイドバーメニューを開きます。

[![Alt text](/img/migrated/Picture3-a5fead32.png)](/img/migrated/Picture3-a5fead32.png)

**ステップ 2:** **Apps** を開き、**Fmon – Templates** を選択します。

[![Alt text](/img/migrated/Screenshot_7-4e0e3943.png)](/img/migrated/Screenshot_7-4e0e3943.png)

**ステップ 3:** Fmon – Templates インターフェースで、**Alert rule template** タブを選択し、Alert Rule を追加したいサービスの **Topic** を選択します。

[![Alt text](/img/migrated/Screenshot_8-ca3c0b94.png)](/img/migrated/Screenshot_8-ca3c0b94.png)

**ステップ 4:** 希望する各 Alert rule の **Integrate** をクリックします。

[![Alt text](/img/migrated/Screenshot_9-cc567b1a.png)](/img/migrated/Screenshot_9-cc567b1a.png)

**ステップ 5:** Alert Rule の統合状態が **Integrated** に変わると、希望する Alert Rule がお客様の Alert Rules に追加されています。

[![Alt text](/img/migrated/Screenshot_10-3b3797a4.png)](/img/migrated/Screenshot_10-3b3797a4.png)

[![Alt text](/img/migrated/Screenshot_11-17e20818.png)](/img/migrated/Screenshot_11-17e20818.png)
