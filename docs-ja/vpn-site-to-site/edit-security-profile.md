# Edit Security Profile

Change the connection Security Profile for an existing VPN.

1. Select **Edit Security Profile**.

[![Edit Security Profile](/img/migrated/Config-edit-security-profile-1024x354-60d3d6e6.png)](/img/migrated/Config-edit-security-profile-1024x354-60d3d6e6.png)

2. Configure the parameters and save.
