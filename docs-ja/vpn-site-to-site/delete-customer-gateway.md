# Customer Gatewayの削除

:::note
Customer Gatewayは、VPN Connectionが割り当てられていない場合（Usage VPN Connection = なし）のみ削除できます。
:::

1. **Customer Gateway** 一覧画面で、削除するゲートウェイのActionカラムにある **Delete** をクリックします。

[![Customer Gatewayの削除](/img/migrated/image-1744270299322-d556bbcc.png)](/img/migrated/image-1744270299322-d556bbcc.png)

2. ダイアログボックスで `delete` と入力し、**Delete Customer Gateway** をクリックして確認するか、**Cancel** をクリックして破棄します。

[![削除の確認](/img/migrated/image-1744183626743-165bf74f.png)](/img/migrated/image-1744183626743-165bf74f.png)
