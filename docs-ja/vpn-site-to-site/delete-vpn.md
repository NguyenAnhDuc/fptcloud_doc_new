# Delete VPN Site-to-Site

Remove an active VPN connection.

1. Select **Delete**.

[![Delete VPN](/img/migrated/Config-delete-1024x354-9bb9ed67.png)](/img/migrated/Config-delete-1024x354-9bb9ed67.png)

2. Confirm the deletion. The VPN status changes to **Deleting** and is then removed.

[![Confirm delete](/img/migrated/Config-delete-confirm-1024x445-4c33a072.png)](/img/migrated/Config-delete-confirm-1024x445-4c33a072.png)
