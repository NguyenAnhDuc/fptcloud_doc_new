# Edit Customergw

**_Lưu ý: Chỉ chỉnh sửa được Customer gateway khi CGW không được gán vào VPN connection_**
Để chỉnh sửa **Customer gateway** , người dùng thao tác theo hướng dẫn sau:
**Bước 1:** Tại màn hiển thị danh sách **Customer gateway,** nhấn **Edit Customer gateway** tại mục **Action** của **Customer gateway** cần chỉnh sửa. [![file](/img/migrated/image-1744269804714-c8f82034.png)](/img/migrated/image-1744269804714-c8f82034.png)
**Bước 2:** Tại đây, người dùng chỉnh sửa các thông tin bao gồm: [![file](/img/migrated/image-1744269819671-b090793a.png)](/img/migrated/image-1744269819671-b090793a.png)  
| **Trường**  | **Mô tả**  |  
| --- | --- |  
| **Customer gateway name**  | Tên của Customer gateway  |  
| **Description**  | Mô tả Customer gateway  |  
|  **Remote IP public**   
  
 | Địa chỉ IP public của VPN Connection phía On-premise  |  
| **Remote network**  | Subnet phía On-premise  |  
| **Healthcheck IP address**  | Địa chỉ IP được sử dụng để kiểm tra tình trạng hoạt động của kết nối VPN giữa hai địa điểm  |  
  
**Bước 3:** Nhấn **Update Customer gateway** để cập nhật hoặc **Cancel** để huỷ bỏ [![file](/img/migrated/image-1744269832390-ef24471f.png)](/img/migrated/image-1744269832390-ef24471f.png)
