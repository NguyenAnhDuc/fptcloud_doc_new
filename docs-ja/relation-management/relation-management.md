# 1, リレーションの作成

**1, リレーションの作成** リレーション画面では、タグ内のコンポーネント間の関係を視覚化できます。

コンポーネントをボードエリアにドラッグ＆ドロップして関係を設定できます。

各コンポーネントは1つのノードに対応します。

例：

[![Alt text](/img/migrated/Picture20-e7d6591d.png)](/img/migrated/Picture20-e7d6591d.png)

**2, ノードの更新**

- **ノードへの値の追加**

ノードに値を追加するには、以下の手順で行います。

該当ノードの **Pencil** アイコンをクリック >> **Update node** ダイアログが表示されます。チェックボックスにチェックを入れてノードに値を追加します。

**複数の値を同時に追加できます。**

[![Alt text](/img/migrated/Picture21-ecc9182e.png)](/img/migrated/Picture21-ecc9182e.png)
[![Alt text](/img/migrated/Picture22-1c0be09b.png)](/img/migrated/Picture22-1c0be09b.png)

- **ノードからの値の削除**

ノードから1つまたは複数のアイテムを削除したい場合は、Pencil アイコンをクリックして Update node ダイアログを開き、チェックを外してください。

[![Alt text](/img/migrated/Picture23-80246e33.png)](/img/migrated/Picture23-80246e33.png)

- **説明の更新**

ノードの説明を追加または編集するには、以下の手順で行います。

ノードを右クリックして **Update Description** を選択します。説明を入力するテキストボックスが表示されます。

[![Alt text](/img/migrated/Picture24-2854752a.png)](/img/migrated/Picture24-2854752a.png)
[![Alt text](/img/migrated/Picture25-0ddf7c2a.png)](/img/migrated/Picture25-0ddf7c2a.png)

- **複数選択の無効化**

Disable multiple selection を選択すると、アイテムがドロップダウンリストとしてノードに追加され、**1つのノードに1アイテムのみ**追加できます。

[![Alt text](/img/migrated/Picture26-f9ef2589.png)](/img/migrated/Picture26-f9ef2589.png)
[![Alt text](/img/migrated/Picture27-0f3c25b0.png)](/img/migrated/Picture27-0f3c25b0.png)

1つのノードに複数のアイテムを追加できるようにするには、右クリックして **Enable multiple selection** を選択してください。

[![Alt text](/img/migrated/Picture28-abb1fec1.png)](/img/migrated/Picture28-abb1fec1.png)

- **ノードの削除**

ノードを削除するには、ノードを右クリックして **Delete node** を選択してください。

[![Alt text](/img/migrated/Picture29-a8665830.png)](/img/migrated/Picture29-a8665830.png)
