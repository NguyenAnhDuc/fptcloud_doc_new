# Bảng điều khiển

Để tạo một dashboard, người dùng thực hiện như sau:
**ステップ1:** Vào Home >> Dashboard >> New dashboard
**ステップ2:** Người dùng sẽ add variable trong dashboard bằng cách click icon Settings >> Chọn Variables >> Add variables
**ステップ3:** Khai báo tag có trong dashboard. 
[![Alt text](/img/migrated/Picture30-c72550b8.png)](/img/migrated/Picture30-c72550b8.png)
Sau khi tạo xong sẽ có một variable được khai báo và hiển thị ở màn hình.
[![Alt text](/img/migrated/Picture31-b13fb689.png)](/img/migrated/Picture31-b13fb689.png)
**ステップ4:** Tạo các biến tương ứng với các component bên trong tag.
[![Alt text](/img/migrated/Picture33-152d5300.png)](/img/migrated/Picture33-152d5300.png)
Click Apply để lưu biến.
Các biến đã khai báo sẽ được hiển thị ở màn hình.
[![Alt text](/img/migrated/Picture35-86ebccbf.png)](/img/migrated/Picture35-86ebccbf.png)
**ステップ5:** Save dashboard. Quay lại trang dashboard bằng cách chọn Home >> Dashboard >> Chọn dashboard đã tạo
Trên dashboard lúc này sẽ hiển thị các variables mà người dùng đã khai báo.
[![Alt text](/img/migrated/Picture36-abda8417.png)](/img/migrated/Picture36-abda8417.png)
Bạn có thể add panel theo mong muốn để theo dõi.
