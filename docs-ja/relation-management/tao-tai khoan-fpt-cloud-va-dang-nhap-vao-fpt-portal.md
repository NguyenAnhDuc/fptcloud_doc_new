# FPT Cloudアカウントの作成とFPT Portalへのログイン

**Relation management** サービスをご利用いただくには、**FPT Portal** にログインする必要があります。

**<https://console.fptcloud.com>** にアクセスし、付与されたFPT Cloudのアカウント/パスワードでログインしてください。ご利用になるサービスに合わせて、正しい **Tenant、Region、VPC** を選択してください。FPT Cloudアカウントをお持ちでない場合は、**[https://fptcloud.com](https://fptcloud.com/)** のホームページにアクセスしてください。

**Sign Up** 機能を選択し、システムの案内に従って必要情報を入力してください。その後、サポート担当者よりご連絡の上、情報を確認してアカウントを作成いたします。

操作中に何らかのエラーが発生した場合は、すぐにサポートチームにご連絡ください。
