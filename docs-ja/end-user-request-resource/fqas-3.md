# サービスのリクエスト時にVPCを選択または新規作成できますか？

新しいVPC（New）または管理権限を持つ任意のVPCを選択できます。VPC Owner権限を持つアカウントの場合、New VPCまたは現在管理しているVPCのみ選択でき、管理外の他のVPCは選択できません。
