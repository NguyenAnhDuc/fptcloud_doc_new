# Can I select a VPC or create a new VPC when requesting a service?

You can select a new VPC (New) or any VPC that you have permission to manage. For accounts with VPC Owner permissions, you can only select a New VPC or the VPC you are currently managing, and cannot select other VPCs outside your management scope.
