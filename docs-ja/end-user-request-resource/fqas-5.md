# Submitをクリックしてチケットを送信できないのはなぜですか？

以下の情報をご確認ください：

  * TitleとRequest typeのフィールドを完全に入力する必要があります。
  * 少なくとも1つのサービスを選択し、そのサービスのQuantity（数量）を選択する必要があります。
  * 各サービスのQuantityは整数で、1以上の値である必要があります。
  * 有効なStart dateは、リクエストチケットを作成した現在の日付より後の日付である必要があります。
  * 有効なEnd dateは、Start date（設定されている場合）より後、またはStart dateが設定されていない場合は現在のチケット作成日より後の日付である必要があります。
