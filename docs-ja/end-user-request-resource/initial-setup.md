# 初期設定

FPT Cloud Serverを初めてご利用の場合は、まず以下の作業を確認・完了してください：

  * FPT CloudのSales/Supportチームに連絡し、最低1件のCommercial契約を作成・稼働させてください。
  * Tenant OwnerまたはVPC Ownerの役割でFPT Portalにアクセスするためのアカウントを取得してください。

上記の手順を完了した後、FPT Cloud Serverシステム上でリソースリクエストを作成する準備が整います。
