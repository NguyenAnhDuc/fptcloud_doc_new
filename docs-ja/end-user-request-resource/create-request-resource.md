# リソースリクエストの作成

**ステップ1**：タブバーでリソースをプロビジョニングする**Tenant**を選択します。メニューバーで**Request Resource**を選択し、**Create request**をクリックします。

[![file](/img/migrated/image-1703744342441-a0f93b7b.png)](/img/migrated/image-1703744342441-a0f93b7b.png)

**ステップ2**：リソースリクエストフォームに情報を入力します。

  * 必須項目：Title、Request type、Service、Quantity。
  * その他の項目は任意です。

[![file](/img/migrated/image-1703744369903-d102105e.png)](/img/migrated/image-1703744369903-d102105e.png)
[![file](/img/migrated/image-1703744387522-1dd6ab81.png)](/img/migrated/image-1703744387522-1dd6ab81.png)

**ステップ3**：**Submit request**を選択してFCIにリクエストを送信します。システムは作成者にリクエスト受領確認メールを送信し、同時にFCIのSales/Supportチームにリクエスト処理の通知を行います。

[![file](/img/migrated/image-1703744428466-add11d9d.png)](/img/migrated/image-1703744428466-add11d9d.png)
