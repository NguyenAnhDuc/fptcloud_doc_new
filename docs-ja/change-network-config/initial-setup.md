# 初期設定

初めてご利用の場合は、以下の手順を確認して完了してください。

## FPT Cloudアカウントの作成とFPT Portalへのサインイン

1. [https://console.fptcloud.com](https://console.fptcloud.com/) にアクセスし、FPT Cloudの認証情報でサインインします。

2. 使用するサービスに応じて、正しい **Tenant**、**Region**、**VPC** を選択します。

FPT Cloudアカウントをお持ちでない場合は、[https://fptcloud.com/](https://fptcloud.com/) にアクセスし、**Sign Up** を選択して必要な情報を入力してください。サポートチームからアカウント作成の確認連絡があります。

:::note
操作中にエラーが発生した場合は、サポートチームにお問い合わせください。
:::
