# 新規デプロイメントの作成

1. **Model Serving**で**Create Deployment**をクリックします。
2. デプロイするモデルとバージョンを選択します。
3. リソースを設定します（GPU、RAM、レプリカ数）。
4. **Deploy**をクリックしてデプロイします。
