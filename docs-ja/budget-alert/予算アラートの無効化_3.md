# 予算アラートの一時停止

一時的にアラートの受信を停止し、コストの追跡は継続したい場合、予算アラートを一時停止（無効化）できます。

1. **Billing** → **Budget Alert** を選択します。作成済み予算アラート一覧が表示されます。一時停止したい予算アラートの **Actions** を選択します。

   ![予算アラートのActionsを選択](/img/migrated/image-1722238345270-ab339c50.png)

2. **Change Status** を選択します。

   ![Change Statusボタン](/img/migrated/image-1722238355811-5adac673.png)

3. 確認ダイアログが表示されます。**Inactive** をクリックします。

   ![予算アラート無効化の確認ダイアログ](/img/migrated/image-1722238369070-8458a3f4.png)

予算アラートのステータスが **Inactive** に変わります。アラートの送信は停止されますが、リソースグループのコスト計算は継続され、Budget historyで確認できます。

![Inactiveステータスの予算アラート](/img/migrated/image-1722238382972-e8af626d.png)
