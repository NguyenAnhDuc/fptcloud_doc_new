# 予算アラートの削除

:::danger
予算アラートの削除は元に戻せません。削除後、システムはその予算アラートに対するコスト計算とアラート送信を停止します。
:::

1. **Billing** → **Budget Alert** を選択します。作成済み予算アラート一覧が表示されます。削除したい予算アラートの **Actions** を選択します。

   ![予算アラートのActionsを選択](/img/migrated/image-1722238402560-f7ffe076.png)

2. **Delete** を選択します。

   ![Deleteボタン](/img/migrated/image-1722238623649-f0020ae9.png)

3. 確認ダイアログが表示されます。**Delete** をクリックします。

   ![予算アラート削除の確認ダイアログ](/img/migrated/image-1722396453644-8e9f9bd6.png)

予算アラートが一覧から削除されます。
