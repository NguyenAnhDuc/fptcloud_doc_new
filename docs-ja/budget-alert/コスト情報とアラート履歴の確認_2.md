# コスト情報とアラート履歴の確認

予算に対する実績コストの追跡とアラート履歴の確認により、クラウドコスト管理の効果を評価できます。

1. **Billing** → **Budget Alert** を選択します。作成済み予算アラート一覧が表示されます。確認したい予算アラート名を選択します。

   ![予算アラート一覧](/img/migrated/image-1722249621354-9d427c95.png)

2. **View budget details** 画面で **Additional budget parameters** を選択し、予算とアラートしきい値の情報を確認します。

   ![View budget details画面](/img/migrated/image-1722249628381-884401a2.png)

3. **Budget history** セクションで、スクロールしてActual costとBudget amountの比較チャートをズームイン／ズームアウトできます。
   - Budget Periodが **Monthly** の場合、過去6か月間の相関関係が表示されます。
   - Budget Periodが **Daily** の場合、過去30日間の相関関係が表示されます。

   ![Budget historyチャート](/img/migrated/image-1722249653673-f81c46ae.png)

4. **Alert history** セクションで、予算アラートのアラート送信履歴を確認できます。
   - Budget Periodが **Monthly** の場合、過去6か月間の履歴が表示されます。
   - Budget Periodが **Daily** の場合、過去30日間の履歴が表示されます。

   ![アラート履歴](/img/migrated/image-1722249921424-ecf944ca.png)
