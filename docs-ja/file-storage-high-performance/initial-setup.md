# 初期セットアップ

**File Storage – High Performance** を利用するには、サポートチームにお問い合わせください。必要な情報を提供いただいた後、アカウントのサービスが有効化され、クォータが割り当てられ、FPT Unify Portal 上で **File Storage Management** 管理機能が有効になります。

提供が必要な情報は以下のとおりです。

| **メトリック** | **単位** | **必須** | **詳細** |
|---|---|---|---|
| テナント / リージョン | | 必須 | File Storage – High Performance を有効化するテナント / リージョン |
| ストレージクォータ | GB | 必須 | 必要なストレージ容量 |
| サブネットリスト | | 必須 | HPC 上の各サブネットには異なるエンドポイントが割り当てられます。File Storage にアクセスしたいサブネットのリストを提供してください。 |
| パフォーマンス | IOPS | 任意 | 希望する IOPS _(指定しない場合はデフォルト値が設定されます)_ |
| 帯域幅 | Gbps | 任意 | 希望する帯域幅 _(指定しない場合はデフォルト値が設定されます)_ |
| 機能 | | 任意 | 希望する追加機能 |
