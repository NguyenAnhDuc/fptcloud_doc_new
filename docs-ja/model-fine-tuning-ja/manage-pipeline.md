# Manage Pipeline

**パイプラインの検索**
  * **検索** バーを使用して、パイプラインを名前で検索します。
  * **フィルター** ドロップダウンを使用して、基本モデル、ステータス、トレーナー、トリガー、または最終更新日時にパイプラインを絞り込みます。

**パイプラインの編集**
  * 特定のパイプラインを選択する
  * **「アクション」** 列で、**「編集」** をクリックして **** 必要な情報を変更します。

**_注:_** ベースモデルは編集できません。 [![file](/img/migrated/image-1747211211191-d3dea1e5.png)](/img/migrated/image-1747211211191-d3dea1e5.png)
**パイプラインの削除**
**_注:_** パイプラインを削除すると、すべての実行が完全に削除されます。
  * 特定のパイプラインを選択する
  * **[アクション**] で、**[削除]** を選択します。
  * 警告プロンプトで削除を確認します。
  * パイプラインを複数選択して一度に削除することができます。

[![file](/img/migrated/image-1747211250617-61363ae4.png)](/img/migrated/image-1747211250617-61363ae4.png)
**テンプレートとして保存**
  * 特定のパイプラインを選択する
  * **[アクション]** で、**[テンプレートとして保存]** を選択して保存します
