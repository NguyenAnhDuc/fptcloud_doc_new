# Allow Address Pairの削除

## 方法1: Allow Address Pair管理画面から削除する

**ステップ1** : 削除アイコンをクリックします。

[![削除アイコンの選択](/img/migrated/image-1708395674995-65847886.png)](/img/migrated/image-1708395674995-65847886.png)

**ステップ2** : 削除を確認します。

[![削除確認ダイアログ](/img/migrated/image-1708395828130-9894ec41.png)](/img/migrated/image-1708395828130-9894ec41.png)

## 方法2: Instance管理画面から削除する

**ステップ1** : Instance ManagementのNetwork Interfaceを開きます。

[![Instance ManagementのNetwork Interface画面](/img/migrated/image-1708395867460-1d20cf7a.png)](/img/migrated/image-1708395867460-1d20cf7a.png)

**ステップ2** : Xアイコンをクリックします。

[![Xアイコンの選択](/img/migrated/image-1708395893416-0efccabb.png)](/img/migrated/image-1708395893416-0efccabb.png)

**ステップ3** : 削除を確認します。

[![削除確認ダイアログ](/img/migrated/image-1708395920272-52eddcc4.png)](/img/migrated/image-1708395920272-52eddcc4.png)

:::warning
削除したAllow Address Pairは復元できません。削除前に設定内容を必ず確認してください。
:::
