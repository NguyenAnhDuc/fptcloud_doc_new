# 初期セットアップ

初めてご利用される場合は、まず以下の事項を確認・完了してください。

## FPT Cloudアカウントの作成とFPT Portalへのログイン

FPT Cloudサービスをご利用いただくには、FPT Portalへのログインが必要です。

[**https://console.fptcloud.com**](https://console.fptcloud.com/) にアクセスし、発行済みのFPT Cloudアカウントとパスワードでログインしてください。ご利用のサービスに対応したTenant、Region、VPCを正しく選択してください。

アカウントをお持ちでない場合は、**Sign Up** を選択し、案内に従って情報を入力してください。入力後、サポートチームよりご連絡を差し上げ、アカウント作成に必要な情報を確認させていただきます。

:::note
操作中に何らかのエラーが発生した場合は、速やかにサポートチームまでお問い合わせください。
:::
