# consumer groupの表示

consumer groupの詳細を確認することで、グループの稼働状況・メンバー数・現在subscribeしているtopicをチェックできます。

1. 確認したいconsumer groupを選択し、**View**をクリックします。

   [![Viewボタンとステータス情報が表示されたconsumer groupリスト](/img/migrated/Pic1_viewconsumer-da0c5124.png)](/img/migrated/Pic1_viewconsumer-da0c5124.png)

詳細ビューにはグループ名・ステータス・メンバー数などの関連情報が表示されます。
