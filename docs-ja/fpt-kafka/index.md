# FPT Kafka

FPT Kafkaは、リアルタイムでデータを送受信できる分散メッセージングサービスです。メッセージキュー、データストリーミングパイプライン、データレプリケーションの構築に適しています。topic、consumer group、credential、ACLを管理するための統合管理インターフェースを提供します。
