# credentialの編集

セキュリティ上の理由やインシデント発生後にパスワードを更新する必要がある場合、管理インターフェースから直接credentialのパスワードを変更できます。

1. パスワードを変更したいcredentialを選択し、**Edit**をクリックします。

   [![Editボタンが強調表示されたcredentialリスト](/img/migrated/Pic1_sua_cre-83c837b2.png)](/img/migrated/Pic1_sua_cre-83c837b2.png)

2. 新しいパスワードを入力します。

   [![credentialのパスワード更新入力フィールド](/img/migrated/Pic2_Sua_cre-f249da7b.png)](/img/migrated/Pic2_Sua_cre-f249da7b.png)

3. **OK**をクリックして保存します。
