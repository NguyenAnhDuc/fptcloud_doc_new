# ACLの削除

ACLを削除すると、対応するtopicまたはconsumer groupに対するcredentialのアクセス権が取り消されます。変更は即座に反映されます。

:::danger
ACLを削除すると、この権限に依存する接続が即座にブロックされます。システムへの影響を確認してから実行してください。
:::

1. 削除するACLを選択し、**Remove**をクリックします。
2. 確認ダイアログで**Yes**をクリックして削除を実行します。
