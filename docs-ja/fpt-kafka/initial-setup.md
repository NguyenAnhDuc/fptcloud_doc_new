# 初期設定

FPT Cloud上のKafkaサービスを使用する前に、アカウントの登録とポータルへのサインインを完了して、管理インターフェースへのアクセス権を取得する必要があります。

## FPT Cloudアカウントの登録

1. [FPT Cloud Portal](https://console.fptcloud.com)にアクセスします。
2. ページ下部の**Sign Up**をクリックし、手順に従ってアカウントを作成します。
3. サポートチームが連絡を取り、情報を確認してアカウントを有効化します。

## FPT Cloud Portalへのサインイン

1. [FPT Cloud Portal](https://console.fptcloud.com)にアクセスします。
2. メールアドレスとパスワードを入力し、**Sign In**をクリックします。
3. 正しいTenant、Region、VPCを選択します。

:::note
サインイン中にエラーが発生した場合は、サポートチームに連絡して迅速なサポートを受けてください。
:::
