# 予算アラートの編集_4

**ステップ1** ：メニューにて[請求] > [予算アラート] を選択します。作成済み予算アラート一覧が表示され > 編集したいアラートの アクション一覧から [更新] を選択します。
[![file](/img/migrated/image-1722238825640-0606b59c.png)](/img/migrated/image-1722238825640-0606b59c.png)
**ステップ 2** ：予算アラート情報を編集します。
**ステップ3** ：「更新」ボタンを選択して編集内容を保存します。保存に成功した後、システムは新しい構成に従って実績コストを計算し、新しいアラートしきい値に従ってアラートを通知します。
[![file](/img/migrated/image-1722238882870-6e80e188.png)](/img/migrated/image-1722238882870-6e80e188.png)
