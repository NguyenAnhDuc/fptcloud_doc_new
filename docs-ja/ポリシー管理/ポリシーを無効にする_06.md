# ポリシーを無効にする_06

ポリシーの無効化機能を使用するには、ユーザーはテナント所有者の権限でシステムにログインする必要があります。
**ステップ1** :メニューで、[ポリシー管理]を選択します。ユーザーは 、アクティブ状態のポリシーを選択し、[アクション] を選択し、[ポリシ無効化] を選択します 
[![file](/img/migrated/image-1741840895326-a00f7753.png)](/img/migrated/image-1741840895326-a00f7753.png)
**ステップ2** :ダイアログボックスが表示されたら、ユーザーは[ポリシー無効化]をクリックしてアクションを確認します。システムはポリシーを無効にし、 ポリシー管理画面のステータスを更新します
[![file](/img/migrated/image-1741840965966-6dcf34fa.png)](/img/migrated/image-1741840965966-6dcf34fa.png)
**ステップ3** :システムはポリシーを無効にし、 ポリシー管理画面でステータスをアクティブから非アクティブに更新します
[![file](/img/migrated/image-1741840999581-d2d6b0ae.png)](/img/migrated/image-1741840999581-d2d6b0ae.png)
