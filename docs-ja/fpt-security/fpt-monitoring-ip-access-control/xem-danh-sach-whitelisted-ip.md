# ホワイトリスト IP 一覧を表示する

各エンドポイント（Dashboard、Metric、Log、Trace）は独自の IP テーブルを持ちます。

- 各テーブルの左側のアイコンをクリックして、そのエンドポイントの IP リストを展開または折りたたみます。
- エンドポイントの隣のアイコンをクリックして、Grafana ダッシュボードを開くか、エンドポイント URL をクリップボードにコピーします。
- IP/CIDR リストはページネーション、IP/CIDR による検索、ホワイトリストステータスによるフィルタリングに対応しています。
- IP/CIDR の隣のアイコンをクリックして、その IP/CIDR をクリップボードにコピーします。
