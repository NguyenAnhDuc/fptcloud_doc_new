# ネットワーク設定（セキュリティグループに相当）

ネットワークの設定（セキュリティグループに相当）を行うために使用します。

表示フィールドの説明は以下のとおりです。

- **Filter No.**: ポリシーの番号を表示します。
- **Source IP**: アクセス元のIPアドレスです。
- **Source Port**: アクセス元のポート番号です。
- **Source Netmask**: アクセス元のサブネットです。
- **Destination IP**: アクセス先サービスの宛先IPアドレスです。
- **Destination Port**: アクセス先サービスの宛先ポート番号です。
- **Destination Netmask**: アクセス先の宛先サブネットです。
- **Protocol**: アクセスに使用するプロトコルです。
- **Connection**: 接続が検出された際に実行するアクション（Allow、Deny、Detection）です。
