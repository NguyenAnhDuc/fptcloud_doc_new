# Destination Server

クラウド内のホスト IP から Web サービスを設定する機能です。[![file](/img/migrated/image-1714121626519-4364137e.png)](/img/migrated/image-1714121626519-4364137e.png)

  * Destination Server の追加・編集・削除 [![file](/img/migrated/image-1714719775318-5890dbc9.png)](/img/migrated/image-1714719775318-5890dbc9.png)
  * **Mode**: Proxy
  * **Proxy IP**: 設定済みの Proxy IP
  * **Proxy Port**: 元のポートから変換して使用する Proxy ポート
  * **Web Server IP (Domain)**: Web サーバーの IP またはドメイン
  * **Web Server Port**: Web サーバーのサービスポート
  * **Health Check**: Web サーバーの稼働状態を確認します

[![file](/img/migrated/image-1714719898104-ec1e303d.png)](/img/migrated/image-1714719898104-ec1e303d.png)
  * **Gateway**: 設定済みのゲートウェイ IP
  * **VLAN ID**: 入力された VLAN ID を表示します。SSL を使用しない場合は空白になります
  * **SSL Use**: 使用しない場合は「X」、使用する場合は「O」を表示します
  * **SSL Termination**: 使用しない場合は「X」、使用する場合は「O」を表示します
  * **Certificate Expiration**: SSL を使用している場合、選択した証明書の有効期限を表示します
