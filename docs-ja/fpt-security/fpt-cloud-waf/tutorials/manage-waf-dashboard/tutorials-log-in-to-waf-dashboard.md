# WAF dashboardへのログイン

1. WAFサーバーが正常に作成されたら、WAF作成に使用したメールアドレスの受信箱を確認します。ログイン情報（**Dashboard link**、**Username**、**Password**）を取得します。

   [![WAF Dashboard ログイン情報メール](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)

2. メールのリンクまたは**Cloud Web Application Firewall Management**画面から**WAF Dashboard**を開きます。

   [![ポータルからWAF Dashboardにアクセス](/img/migrated/Userguide-FPT-WAF-2022-15-1024x538-bf0f51e8.png)](/img/migrated/Userguide-FPT-WAF-2022-15-1024x538-bf0f51e8.png)

3. メールで受け取ったユーザー名とパスワードを使用してログインします。

   [![WAF Dashboard ログイン画面](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)

:::warning
アカウントのセキュリティを確保するため、初回ログイン後すぐにWAF Dashboardのパスワードを変更してください。
:::
