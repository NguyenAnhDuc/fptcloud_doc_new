# WAFサーバーの削除

:::danger
この操作はWAFサーバーに関連するすべてのデータを完全に削除し、**元に戻すことはできません**。実行前に必ず確認してください。
:::

1. **Cloud Web Application Firewall Management**画面で、削除するサーバーの**Delete WAF**を選択します。

   [![Delete WAFの選択](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)

2. 確認ダイアログで**Delete**を選択して削除を完了します。

   [![WAFサーバー削除の確認](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)
