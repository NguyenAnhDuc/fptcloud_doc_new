# Proxy hostの有効化/無効化

1. **WAF Dashboard**で**Hosts** > **Proxy**を選択します。

   [![Proxy hostリスト](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. 切り替えるproxy hostの**More**アイコンをクリックし、**Toggle**を選択します。

   [![Proxy hostのToggleを選択](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)

3. 確認ダイアログで**Run Action**をクリックして確定します。

   [![Proxy hostの有効化/無効化の確認](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)
