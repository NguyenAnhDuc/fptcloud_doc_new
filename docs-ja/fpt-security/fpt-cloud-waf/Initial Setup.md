# 初期設定

If this is your first time using **FPT Cloud WAF**, complete the following steps before you begin.

## Create an account and log in to FPT Cloud Portal

To use the **FPT Cloud WAF** service, you need to log in to **FPT Portal**.

1. Access FPT Portal and log in with your FPT Cloud account credentials.
2. Select the correct Tenant, Region, and VPC for the service.

:::note
If you do not yet have an **FPT Cloud** account, visit the homepage and click **Sign Up**, then follow the on-screen instructions. The support team will contact you to confirm and create your account.
:::

If you encounter any errors during the process, contact the Support team for assistance.

After completing the steps above, you are ready to use WAF on the FPT Cloud platform.
