# Install Policy

:::note
**SmartConsole** で行った変更は、**Install Policy** を実行して公開・適用するまで有効になりません。
:::

**ステップ1：** **Install Policy** (**Ctrl + Shift + Enter**) を選択し、**Publish & Install** > **Install** をクリックします。
