# Roleの削除

不要になったCustom Roleは以下の手順で削除できます。

1. **IAM** > **Roles** を選択します。**Role Management** ページが表示されます。
2. 一覧から削除するCustom Roleの末尾にある **Action** を選択し、**Delete** を選択します。

[![](/img/migrated/image-1734344452227-353ab722.png)](/img/migrated/image-1734344452227-353ab722.png)

3. ポップアップで警告情報を確認し、**Delete** を選択します。

[![](/img/migrated/image-1734344485077-982d3c57.png)](/img/migrated/image-1734344485077-982d3c57.png)

:::warning
Roleを削除できるのは、そのRoleがどのUser Groupにも割り当てられていない場合のみです。
:::
