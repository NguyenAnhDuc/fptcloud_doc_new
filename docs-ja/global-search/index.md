# グローバル検索

FPT CloudのGlobal Search機能では、**Instances、IPアドレス、Subnets、Security Groups、Databases**など、システム内のすべてのリソースを素早く検索できます。

この機能により、FPT Cloudプラットフォーム上でのリソース管理の時間を節約し、効率を向上させます。
