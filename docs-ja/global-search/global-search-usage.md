# グローバル検索の使い方

1. [FPT Cloud Portal](https://portal.fptcloud.com)にログインします。
2. 上部ナビゲーションバーの**検索**アイコンをクリックします。
3. 検索キーワード（リソース名、IPアドレスなど）を入力します。
4. 結果がリソースタイプごとにグループ化されて表示されます。
5. 結果をクリックしてリソースの詳細に移動します。
