# Groupの削除

## ステップ1: Autoscaling > Autoscale Groupページに移動します。削除したいGroupの行で**Action menu**アイコンをクリックします。

[![open group action menu](/img/migrated/Screenshot-2024-10-02-023008-8d0f1521.png)](/img/migrated/Screenshot-2024-10-02-023008-8d0f1521.png)

## ステップ2: **Delete**を選択します。

[![delete group](/img/migrated/Screenshot-2024-10-02-023134-da955e2d.png)](/img/migrated/Screenshot-2024-10-02-023134-da955e2d.png)

## ステップ3: 確認ダイアログが表示されます。Group名が正しければ、**delete**と入力して**Delete group**をクリックして確定します。操作を中止するには**Cancel**をクリックします。

[![delete group confirm dialog](/img/migrated/Screenshot-2024-10-02-023251-6a7e8d6c.png)](/img/migrated/Screenshot-2024-10-02-023251-6a7e8d6c.png)

[![delete group confirmed dialog](/img/migrated/Screenshot-2024-10-02-023337-021ad4b9.png)](/img/migrated/Screenshot-2024-10-02-023337-021ad4b9.png)
