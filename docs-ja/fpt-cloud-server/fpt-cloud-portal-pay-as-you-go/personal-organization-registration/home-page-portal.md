# Portalのホームページ

FPT Cloud Portalのホームページには以下の情報が表示されます：

- **Available Services**：現在PAYG課金をサポートしているサービスの一覧。
- **Upcoming Services**：近日中にPAYG課金をサポートする予定のサービス。
- **Balance**：現在のアカウント残高。
