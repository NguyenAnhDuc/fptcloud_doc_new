# Public IPの購入

アカウントにチャージした後、追加のPublic IPを購入できます。

:::note
- IPを割り当てるには、少なくとも50,000 VNDと1時間分のIP使用料金が必要です。
- 割り当て時に1時間分のIP使用料金が即時引き落とされます。
:::

1. **Networking – Floating IPs**ページに移動します。

[![Floating IPsページ](/img/migrated/image-1694073889613-41f69bf5.png)](/img/migrated/image-1694073889613-41f69bf5.png)

2. **Allocate IP address**をクリックします。

3. **Allocate new IP from pool**を選択します：
   - 割り当てるIPの数を選択します。
   - システムが参考としてIPの1時間あたりの使用コストを表示します。

[![IP割り当て設定](/img/migrated/image-1694073898945-f373c2b3.png)](/img/migrated/image-1694073898945-f373c2b3.png)

4. **Allocate floating IP**をクリックします。情報が有効な場合、システムはアカウントにIPを割り当てます。

[![IP割り当て結果](/img/migrated/image-1694073904922-0dd35f26.png)](/img/migrated/image-1694073904922-0dd35f26.png)

[![割り当て後に引き落とされたコスト](/img/migrated/image-1694073916507-22fc26cb.png)](/img/migrated/image-1694073916507-22fc26cb.png)
