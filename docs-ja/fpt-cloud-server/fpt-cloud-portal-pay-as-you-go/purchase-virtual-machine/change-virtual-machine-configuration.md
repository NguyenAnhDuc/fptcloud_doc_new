# 仮想マシン構成の変更

現在稼働中の仮想マシンの構成を変更できます。

:::note
構成変更時に、新しい構成に基づく1時間分の使用料金が即時引き落とされます。
:::

1. **Instance Management**ページに移動します。

2. 変更したい仮想マシンの**Action**をクリックし、**Resize VM**を選択します。

[![Resize VMの選択](/img/migrated/image-1694073866010-73330cc8.png)](/img/migrated/image-1694073866010-73330cc8.png)

3. 新しい構成を選択します。

[![新しい構成の選択](/img/migrated/image-1694073874856-91ac98f3.png)](/img/migrated/image-1694073874856-91ac98f3.png)

4. **Resize instance**をクリックします。

[![構成変更の確認](/img/migrated/image-1694073882943-1c379eb9.png)](/img/migrated/image-1694073882943-1c379eb9.png)
