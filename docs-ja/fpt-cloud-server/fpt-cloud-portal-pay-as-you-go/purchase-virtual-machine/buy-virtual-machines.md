# 仮想マシンの購入

インスタンスページで仮想マシンを作成し、PAYGモデルで支払いができます。

:::note
仮想マシンを作成するには、少なくとも50,000 VNDと1時間分のサービス料金が必要です。作成時に1時間分の使用料金が即時引き落とされます。
:::

1. **Instance Management**ページに移動します。

[![Instance Managementページ](/img/migrated/image-1694073823080-04d5dd8a.png)](/img/migrated/image-1694073823080-04d5dd8a.png)

2. **New Instance**をクリックします。

3. 仮想マシンの構成を選択します：
   - 構成パッケージを選択すると、システムが構成の詳細と対応する価格を表示します。
   - システムが月間使用コストの見積もりを参考として表示します。

[![仮想マシンの構成選択](/img/migrated/image-1694073838317-c01bfab4.png)](/img/migrated/image-1694073838317-c01bfab4.png)

4. **Create instance**をクリックします。
   - 情報が有効な場合、システムは仮想マシンを作成し、Instance Managementページにリダイレクトします。
   - リソース制限に達している場合、システムはエラーメッセージを表示し、追加購入を防ぎます。

[![仮想マシン作成結果](/img/migrated/image-1694073846242-2bb9076d.png)](/img/migrated/image-1694073846242-2bb9076d.png)

[![作成後に引き落とされたコスト](/img/migrated/image-1694073853907-420d469b.png)](/img/migrated/image-1694073853907-420d469b.png)
