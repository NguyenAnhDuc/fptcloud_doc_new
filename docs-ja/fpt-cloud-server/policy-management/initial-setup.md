# 初期設定

初めて使用する場合は、まず次の点を確認して完了してください。

**FPT Cloudアカウントを作成し、FPTポータルにログインする**

FPT Cloudのサービスを使用開始するには、FPTポータルにログインする必要があります。[https://console.fptcloud.com](https://console.fptcloud.com/) にアクセスし、発行されたFPT Cloudアカウント・パスワードでログインしてください。適切なテナント、リージョン、VPCを選択します。

FPT Cloudアカウントをお持ちでない場合は、ホームページ [https://fptcloud.com](https://fptcloud.com/) にアクセスしてください。**サインアップ**機能を選択し、システムの指示に従って情報を入力します。その後すぐにサポートチームから連絡があり、アカウントを作成するための情報を確認します。

操作中にエラーが発生した場合は、すぐにサポートチームに連絡してサポートを受けてください。
