# Custom Imageのタグ管理

Custom Imageにtagを割り当てると、イメージをプロジェクトや環境別に分類でき、検索と再利用が容易になります。

## イメージアップロード時のタグ割り当て

1. メニューから **Custom Image** を選択し、**Upload image** をクリックします。

   [![Upload imageボタンのあるCustom Image画面](/img/migrated/image-1721032879356-5bd46d9a.png)](/img/migrated/image-1721032879356-5bd46d9a.png)

2. **Upload image** ダイアログボックスで、**Add tag (optional)** フィールドからtagを選択します。

   [![Add tagフィールドのあるUpload imageダイアログ](/img/migrated/image-1721032892734-ae8a432b.png)](/img/migrated/image-1721032892734-ae8a432b.png)

## 既存Custom Imageのタグ管理

1. メニューから **Custom Image** を選択し、**Actions** → **Manage tag** をクリックします。

   [![Custom ImageのManage tagオプションのあるActionsメニュー](/img/migrated/image-1721032908795-71940b90.png)](/img/migrated/image-1721032908795-71940b90.png)

2. ダイアログボックスが表示されます。イメージに割り当てるtagを選択します。

   [![Custom ImageのManage tagsダイアログ](/img/migrated/image-1721032920130-4755a194.png)](/img/migrated/image-1721032920130-4755a194.png)

3. tagを削除するには、tag名の横にある **x** をクリックします。リストボックス末尾の **x** をクリックすると、すべてのtagを一度に削除できます。**Save** をクリックして確定します。

   [![タグ削除オプションのあるManage tagsダイアログ](/img/migrated/image-1721032931446-41441a71.png)](/img/migrated/image-1721032931446-41441a71.png)

:::warning
イメージを削除すると、システムは関連するすべてのtagを自動的に削除し、Tagging画面のtagリストを更新します。
:::
