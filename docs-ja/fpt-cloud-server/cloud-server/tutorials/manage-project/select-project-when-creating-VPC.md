# VPC作成時のProject選択

新しいVPCを作成する際に、既存のProjectに割り当てることができます。これにより、VPC内のすべてのリソースが正しいProjectに関連付けられ、コスト追跡とアクセス制御が適切に行われます。

:::tip
作成時にVPCをProjectに割り当てることが推奨されます。後でリソースを再割り当てする手間を省けます。
:::
