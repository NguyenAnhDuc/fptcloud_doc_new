# インスタンス削除のロック

以下の手順で仮想マシンをロックして削除を防ぐことができます。

1. **Instance Management** で仮想マシンを選択し、**Actions** → **Lock Deletion** をクリックします。

2. 確認ダイアログに仮想マシン名が表示されます。**Lock Instance Deletion** をクリックして確認します。

[![Lock Instance Deletion確認ダイアログ](/img/migrated/image-1702889848019-0f69d8a3.png)](/img/migrated/image-1702889848019-0f69d8a3.png)

ロックが解除されるまで、仮想マシンは削除できなくなります。
