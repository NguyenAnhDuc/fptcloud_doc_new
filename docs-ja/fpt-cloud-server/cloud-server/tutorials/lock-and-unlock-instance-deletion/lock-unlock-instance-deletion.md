# インスタンス削除のロック & ロック解除

削除ロック機能を使用すると、重要な仮想マシンを誤削除から保護できます。操作ミスによるデータ損失のリスクを低減できます。

任意の仮想マシンをロックすることで、意図しない削除や終了を防ぐことができます。本番環境のマシンをテスト用マシンと誤って削除してしまうリスクを回避できます。
