# シャットダウンスケジュール

Schedule Shutdownを使用すると、マシンの将来の自動シャットダウン時刻を設定できます。作業完了後にマシンを監視する時間がない場合などに便利です。

## シャットダウンスケジュールの設定

1. イメージを選択し、**Schedule** をクリックします。
2. ダイアログで希望の時刻を選択し、**Save settings** をクリックします。

   [![Schedule shutdownの設定ダイアログ](/img/migrated/schedule-2-c46f16f3.jpg)](/img/migrated/schedule-2-c46f16f3.jpg)

設定後、**Hosts** リスト画面で自動シャットダウンまでの残り時間を確認できます。
