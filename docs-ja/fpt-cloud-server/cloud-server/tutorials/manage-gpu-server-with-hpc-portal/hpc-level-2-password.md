# 第2レベルパスワードの設定

第2レベルパスワードは、HPC Portalにログインした後に設定するパスワードです。リモートデスクトップまたはSSHでマシンに接続する際のログインに使用します。

## 第2レベルパスワードの設定手順

1. ヘッダーのユーザーアイコンをクリックし、ユーザー名をクリックします。
2. 設定ページで **Level 2 password** タブを選択します。
3. 新しいパスワードを入力し、**Submit** をクリックします。
