# HPC PortalでのGPU Server管理

[HPC Portal](https://hpc.fptcloud.com/) は、FPT Cloud の RTX 3090、RTX 4090 向け物理GPU Serverの管理インターフェースです。高性能GPUワークロード向けに特化した管理画面を提供します。

:::note
HPC Portalへのログインおよびマシンの作成には、管理者に連絡してアカウントへの権限付与とリソース割り当てを依頼してください。
:::
