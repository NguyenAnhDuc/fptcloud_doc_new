# サービスへのアクセス

手順は以下のとおりです。

**ステップ 1**: **<https://console.fptcloud.com>** から **FPT Cloud Portal** にログインします。

**ステップ 2**: **FPT Cloud Console Portal** のメニューで **Monitoring** をクリックします。ワークスペースの一覧が以下のように表示されます。

[![Alt text](/img/migrated/Picture1-931a5e8c.png)](/img/migrated/Picture1-931a5e8c.png)

**ステップ 3**: ワークスペース名をクリックして詳細を確認します。そのワークスペースの詳細画面が表示されます。

**ステップ 4**: "**Configuration**" タブをクリックします。

[![Alt text](/img/migrated/Picture2-d44be0fa.png)](/img/migrated/Picture2-d44be0fa.png)

ここに **Dashboard URL** が表示されます。そのリンクをクリックしてダッシュボードページに移動します。

ダッシュボードページは以下のように表示されます。

[![Alt text](/img/migrated/Picture5-d0c28feb.png)](/img/migrated/Picture5-d0c28feb.png)

**ステップ 5**: メニューで **App > Fmon - Relation** をクリックします。これでサービスをご利用いただけます。
