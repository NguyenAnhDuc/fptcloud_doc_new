# Component Management

**1, コンポーネントの作成**

定義：コンポーネントとは、業務フローまたはビジネス内の要素を表します。

例：Long Chau タグ内には、LoadBalancer、Kubernetes、DB などのコンポーネントがあります。

コンポーネントを作成するには、以下の手順に従ってください。

**ステップ 1**: **Relation management** サービスに正常にアクセスします。

**ステップ 2**: サービスメニューで **Component management** >> **Create component** ボタンをクリックします。

[![Alt text](/img/migrated/Picture45-51564469.png)](/img/migrated/Picture45-51564469.png)

**ステップ 3**: 作成フォームが表示されます。以下の情報を入力してください。
  * Name（必須）：コンポーネント名
  * Icon（任意）
  * Query type：デフォルトは Custom
  * Value：コンポーネントに値を追加できます。

[![Alt text](/img/migrated/Picture46-14525c51.png)](/img/migrated/Picture46-14525c51.png)

値の入力方法は2種類あります。

- **Builder**

テキストボックスに値を入力 >> **Add** ボタンをクリックします。

複数の値を一度に追加したい場合は、テキストボックスにカンマ（,）区切りで値を入力 >> **Add** ボタンをクリックします。

例：

[![Alt text](/img/migrated/Picture47-99f02769.png)](/img/migrated/Picture47-99f02769.png)

- **Code**

Code を選択した場合、カンマ（,）区切りで値を入力して追加します。

例：[![Alt text](/img/migrated/Picture48-4adc6eb8.png)](/img/migrated/Picture48-4adc6eb8.png)

**ステップ 4**: 全ての情報を入力したら、**Create** をクリックしてコンポーネントを作成します。

作成が成功すると、一覧画面にコンポーネントが表示されます。

**2, コンポーネントの更新**

ステップ 1：メニューで Component management を選択 >> 一覧画面の Action 列にある **Update** ボタンをクリックします。

[![Alt text](/img/migrated/Picture17-7b5c3a89.png)](/img/migrated/Picture17-7b5c3a89.png)

**ステップ 2**: コンポーネント情報を編集します。編集可能なフィールドは以下のとおりです。
  * Name
  * Icon
  * Query type
  * Value：コンポーネントの値を追加または削除できます。

[![Alt text](/img/migrated/Picture18-28271c7d.png)](/img/migrated/Picture18-28271c7d.png)

**ステップ 3**: **Update** ボタンをクリックして完了します。

操作をキャンセルするには、**Cancel** をクリックして編集画面を終了します。

**3, コンポーネントの削除**

コンポーネントを削除するには、以下の手順に従ってください。

**ステップ 1**: メニューで **Component management** を選択 >> 一覧画面の **Action** 列にある **Delete** ボタンをクリックします。

[![Alt text](/img/migrated/Picture19-c948da9f.png)](/img/migrated/Picture19-c948da9f.png)

**ステップ 2**: 確認ポップアップが表示されます。削除を確定するには、ポップアップ内の **Delete** をクリックします。

削除をキャンセルするには、**Cancel** をクリックします。

[![Alt text](/img/migrated/Picture49-eb07667d.png)](/img/migrated/Picture49-eb07667d.png)
