# 初期設定

初めてFPT Cloudをご利用になる場合は、以下の手順を完了してから開始してください。

**FPT Cloudアカウントの作成とFPT Portalへのサインイン**

1. [https://console.fptcloud.com](https://console.fptcloud.com/) にアクセスし、FPT Cloudの認証情報でサインインします。
2. 使用するサービスに対応したTenant、Region、VPCを選択します。
3. アカウントをお持ちでない場合は、**Sign Up** をクリックして必要な情報を入力してください。サポートチームがお客様に連絡し、情報を確認した上でアカウントを作成します。

:::note
操作中にエラーが発生した場合は、すぐにサポートチームまでお問い合わせください。
:::
