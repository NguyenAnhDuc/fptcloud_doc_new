# Gatewayリストの管理

[![Gatewayリスト画面](/img/migrated/image-1754290296265-667afaca.png)](/img/migrated/image-1754290296265-667afaca.png)

この画面では、VPCのGatewayリストを管理できます。各Gatewayに表示される情報は以下の通りです：

- **Gateway name**：Gatewayの名前。名前をクリックすると詳細ページが表示されます。
- **Gateway type**：2種類あります。
  - **Basic**：デフォルト設定で自動的にセットアップされるGatewayです。
  - **Custom**：用途に合わせて設定をカスタマイズできるGatewayです。
- **Number of Route**：GatewayのRoute数です。
- **Connect to Network**：Gatewayが接続されているネットワークです。

**Gateway Type**タブにはGatewayのタイプ情報が表示されます：
