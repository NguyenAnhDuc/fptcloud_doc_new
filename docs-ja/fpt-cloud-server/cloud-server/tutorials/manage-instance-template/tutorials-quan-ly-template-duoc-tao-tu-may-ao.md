# Custom ImagesページでTemplateを管理する

**Custom Images** ページの **General Image** タブに、仮想マシンから作成されたTemplateの一覧が表示されます。

## Template情報

- **Name**: Templateの名前。
- **Description**: Templateの説明。
- **Status**: 現在の状態 — Pending / Success / Failed。

[![Custom ImagesのTemplate一覧](/img/migrated/image-1715855590500-9c6d17bc.png)](/img/migrated/image-1715855590500-9c6d17bc.png)

- **進捗**: プログレスバーで初期化の進捗が確認できます。

[![Templateの初期化進捗](/img/migrated/image-1715855691493-7d373533.png)](/img/migrated/image-1715855691493-7d373533.png)
