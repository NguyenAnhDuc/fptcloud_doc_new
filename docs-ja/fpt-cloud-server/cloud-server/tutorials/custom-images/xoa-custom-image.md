# Custom Image の削除

Custom Image が不要になった場合、システムから削除できます。

:::danger
削除した Custom Image は復元できません。削除する前に、このイメージを使用している Instance がないことを確認してください。
:::

## Custom Image の削除

1. **Custom Image** で **Actions** → **Delete** をクリックします。

   [![Delete オプションがある Actions メニュー](/img/migrated/image-1734596886940-61bbf341.png)](/img/migrated/image-1734596886940-61bbf341.png)

2. ポップアップで確認し、**Delete custom image** をクリックします。

   [![Custom Image 削除確認ダイアログ](/img/migrated/image-1734596914709-3da0ab77.png)](/img/migrated/image-1734596914709-3da0ab77.png)
