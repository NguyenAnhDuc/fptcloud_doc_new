# インスタンスSnapshotスケジュール

Schedule Managementは、大量のリソースを持つ組織の定期的なデータ保存ニーズに対応します。スケジュールに従ってSnapshotを自動作成することで、作業時間の削減とヒューマンエラーの最小化を実現します。
