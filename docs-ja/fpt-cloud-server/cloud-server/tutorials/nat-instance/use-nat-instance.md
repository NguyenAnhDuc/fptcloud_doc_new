# Generalサービスプランのユーザーのみ利用可能

:::note
この機能は**General**サービスプランを購入したユーザーのみ利用できます。
:::

Subnet作成時に**isolated**モードを選択すると、isolated network内のインスタンスがインターネットにアクセスできるようにNAT Instanceを設定するための手順が表示されます。
