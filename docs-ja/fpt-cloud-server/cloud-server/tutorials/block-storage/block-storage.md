# Block Storage 概要

FPT Block Storage は FPT Cloud 上の仮想マシン向けにディスクを提供します — 仮想マシンに接続した物理ハードドライブと同様に管理できます。

FPT Cloud Portal には **Storage Disk Management** が搭載されており、VPC 内に作成したすべての Storage Disk を管理できます。一覧表示、新規作成、アタッチ／デタッチ、編集、削除が可能です。

## Storage Disk Management へのアクセス

メニューから **Compute Engine** → **Storage Disks** を選択します。

[![Storage Disk Management 画面](/img/migrated/Userguide-FPT-Cloud-Server-2022-55-1024x-0c126930.png)](/img/migrated/Userguide-FPT-Cloud-Server-2022-55-1024x-0c126930.png)

一覧には **Name**、**Storage type**、**Size**、**Created at**、**Attached**（現在アタッチされている Instance）が表示されます。

[![Storage Disk 一覧](/img/migrated/Userguide-FPT-Cloud-Server-2022-56-1024x-abee045f.png)](/img/migrated/Userguide-FPT-Cloud-Server-2022-56-1024x-abee045f.png)
