# Storage Disk の作成

新しい Storage Disk を作成することで、Instance のストレージ容量を拡張できます — データやアプリケーションのための追加スペースが必要なときに便利です。

## Storage Disk の作成

1. **Compute Engine** → **Storage Disk** を選択し、**Create Storage** をクリックします。

   [![Create Storage ボタンがある Storage Disk 画面](/img/migrated/image-1712215492491-8d7d7c0c.png)](/img/migrated/image-1712215492491-8d7d7c0c.png)

2. 必要事項を入力します。
   - **Name**: Storage Disk の名前。
   - **Storage Policy**: ストレージのタイプ。
   - **Size**: ストレージのサイズ。
   - **Applied instance** *(省略可)*: アタッチする Instance（後からアタッチすることも可能です）。

   [![Create Storage Disk ダイアログ](/img/migrated/image-1712215502018-5f358995.png)](/img/migrated/image-1712215502018-5f358995.png)

3. **Create storage disk** をクリックします。

新しい Storage Disk が **Storage Disk Management** ページに表示されます。

[![作成後の Storage Disk](/img/migrated/image-1712215510866-1cd01f25.png)](/img/migrated/image-1712215510866-1cd01f25.png)
