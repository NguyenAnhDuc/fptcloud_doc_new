# 仮想マシンの起動/停止

FPT Cloud Portalの管理画面から、物理サーバーと同様の操作で仮想マシンを直接停止または起動できます。

:::warning
仮想マシンが **停止** 状態の場合でも、リソースが固定割り当てされているため、引き続きコストが発生します。
:::

## 仮想マシンの起動/停止

1. **Instance Management** 管理画面で **Action** → **Power** を選択します。
2. 実行する操作を選択します。
   - **Power Off**: 仮想マシンを停止します（稼働中の場合）。
   - **Power On**: 仮想マシンを起動します（停止中の場合）。
