# 仮想マシンの削除

使用する必要がなくなった仮想マシンは、システムから削除することでコストの発生を防ぐことができます。

:::danger
仮想マシンを削除すると、すべてのデータが削除されて復元できなくなります。削除前に仮想マシンが **Unlock Deletion** 状態であることを確認してください。
:::

## 仮想マシンの削除

1. **Instance Management** 管理画面で **Actions** → **Delete** を選択します。

   [![DeleteオプションがあるActionsメニュー](/img/migrated/image-1712200768564-6389b6eb.png)](/img/migrated/image-1712200768564-6389b6eb.png)

   [![Deleteの選択を確認](/img/migrated/image-1712200780901-998413fe.png)](/img/migrated/image-1712200780901-998413fe.png)

2. 警告ダイアログに仮想マシン名が表示されます。確認のために **delete** と入力します。

   [![仮想マシン削除確認ダイアログ](/img/migrated/image-1712200787729-9546cde9.png)](/img/migrated/image-1712200787729-9546cde9.png)

   [![deleteと入力して確認](/img/migrated/image-1712200798790-c16164fa.png)](/img/migrated/image-1712200798790-c16164fa.png)

3. システムが削除とリソースの回収を実行します。

   [![削除中の仮想マシン](/img/migrated/image-1712200803567-91c8fdd5.png)](/img/migrated/image-1712200803567-91c8fdd5.png)
