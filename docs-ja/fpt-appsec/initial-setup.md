# 初期設定

FPT ASPM_v1.0を初めてご利用の場合は、まず以下の作業を確認・完了してください。
  * **FPT Cloud** アカウントを作成し、FPT Portalにログインする

FSP DevSec Serviceの利用を開始するには、FPT Portalへのログインが必要です。<https://console.fptcloud.com> にアクセスし、発行済みのFPT Cloudアカウント／パスワードでログインしてください。使用するTenant、Region、VPCを正しく選択してください。FPT Cloudアカウントをお持ちでない場合は、<https://fptcloud.com/> のトップページにアクセスしてください。
その後、**Sign Up** 機能を選択し、システムの案内に従って必要情報を入力してください。入力後、サポート担当者がすぐに連絡し、アカウント作成に必要な情報を確認します。
操作中に何らかのエラーが発生した場合は、弊社のサポートチームにお問い合わせください。
