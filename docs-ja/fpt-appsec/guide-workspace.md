# Workspace Management - AssetおよびIssue管理

**手順1** : メニューの **Workspace** をクリックします。
**手順2** : 使用するTeamを選択します。
[![anhfsec](/img/migrated/1-52d0b798.png)](/img/migrated/1-52d0b798.png)
画面にはチームのasset一覧と、**Status**、**Last request time**、**Issues Severity** などの情報が表示されます。Scan/Rescan、Disable/Enable、Deleteなどの操作も実行できます。また、**Name**、**Severity**、**Recently Import** によるコンテンツのフィルタリングも可能です。
[![anhfsec](/img/migrated/w2-bde88680.png)](/img/migrated/w2-bde88680.png)
