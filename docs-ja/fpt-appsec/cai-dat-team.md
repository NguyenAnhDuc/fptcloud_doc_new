# チーム設定

**Org Admin** および **Team Manager** のみ編集権限を持ちます。
**手順1** : チームのセクションで **Setting** を選択します。
[![anhfsec](/img/migrated/st1-3d42cb2c.png)](/img/migrated/st1-3d42cb2c.png)
ここで **Code**、**Name**、**Description** などの情報とEdit操作を確認できます。
[![anhfsec](/img/migrated/st2-7be4dc7e.png)](/img/migrated/st2-7be4dc7e.png)
**手順2** : **Edit** を選択して情報を編集します。
[![anhfsec](/img/migrated/st3-368d910e.png)](/img/migrated/st3-368d910e.png)
**手順3** : **Team Name** および **Team Description** を入力します。
[![anhfsec](/img/migrated/st4-2d7deb7e.png)](/img/migrated/st4-2d7deb7e.png)
**手順4** : **Save** を選択して編集内容を保存します。
[![anhfsec](/img/migrated/st5-f523ff20.png)](/img/migrated/st5-f523ff20.png)
