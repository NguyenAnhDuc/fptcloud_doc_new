# チーム管理

**Organization** の **Team** をクリックします。
[![anhfsec](/img/migrated/ot1-4890d1ae.png)](/img/migrated/ot1-4890d1ae.png)
ここでは **Name**、**Description**、**Number of Members**、**Status**、**Created at** などの情報を確認でき、これらを基に検索や **Create Team**、**Create Sub-team**、**Edit**、**Disable/Enable**、**Delete** などの操作を実行できます。

## 新しいチームの作成
**Org Admin** および **Team Manager** のみ編集権限を持ちます。
**手順1** : **Teams** の **Create team** をクリックします。
[![anhfsec](/img/migrated/CT1-9725d6d4.png)](/img/migrated/CT1-9725d6d4.png)
**手順2** : **Team Name**、**Description**、**Parent Org Unit** をすべて入力します。
[![anhfsec](/img/migrated/CT2-8455b3c6.png)](/img/migrated/CT2-8455b3c6.png)
**手順3** : **Create** をクリックします。

## サブチームの作成
**Org Admin** および **Team Manager** のみ編集権限を持ちます。
**手順1** : 対象チームのActionから **Create Sub-team** をクリックします。
[![anhfsec](/img/migrated/CST1-411e33b4.png)](/img/migrated/CST1-411e33b4.png)
**手順2** : **Org Name**、**Org Description**、**Parent Org Unit** をすべて入力し、**Create** をクリックします。
[![anhfsec](/img/migrated/CST2-6ab10ae3.png)](/img/migrated/CST2-6ab10ae3.png)

## チームの編集
**Org Admin** および **Team Manager** のみ編集権限を持ちます。
**手順1** : 対象チームのActionから **Edit** をクリックします。
[![anhfsec](/img/migrated/EDO-d1cc8e5e.png)](/img/migrated/EDO-d1cc8e5e.png)
**手順2** : **Org Name**、**Org Description**、**Parent Org Unit** をすべて入力します。
[![anhfsec](/img/migrated/EDO2-285e9fb4.png)](/img/migrated/EDO2-285e9fb4.png)
**手順3** : **Save** をクリックします。

## チームの削除
**Org Admin** および **Team Manager** のみ編集権限を持ちます。
**手順1** : 対象チームのActionから **Delete** をクリックします。
[![anhfsec](/img/migrated/DETO-9a2f3ce9.png)](/img/migrated/DETO-9a2f3ce9.png)
**手順2** : **delete** と入力し、**Confirm** をクリックします。
[![anhfsec](/img/migrated/DETO2-031a46c8.png)](/img/migrated/DETO2-031a46c8.png)

## チームの無効化
**Org Admin** のみ編集権限を持ちます。
**手順1** : 対象チームのActionから **Disable** をクリックします。
[![anhfsec](/img/migrated/DTO-7f086bfe.png)](/img/migrated/DTO-7f086bfe.png)
**手順2** : **disable** と入力し、**Confirm** をクリックします。
[![anhfsec](/img/migrated/DTO2-fe8efa07.png)](/img/migrated/DTO2-fe8efa07.png)

## チームの有効化
**Org Admin** のみ編集権限を持ちます。
**手順1** : 対象チームのActionから **Enable** をクリックします。
[![anhfsec](/img/migrated/En1-1-7958e94c.png)](/img/migrated/En1-1-7958e94c.png)
**手順2** : **Confirm** をクリックします。
[![anhfsec](/img/migrated/En2-1-570ad3a9.png)](/img/migrated/En2-1-570ad3a9.png)
