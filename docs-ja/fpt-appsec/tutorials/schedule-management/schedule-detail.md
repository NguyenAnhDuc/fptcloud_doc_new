# スケジュール一覧と詳細の表示ガイド

**手順1:** 左サイドメニューでSecurity Platform → DevSec → Scheduleを選択します。**Schedule Management** 画面には、デフォルトで作成日の新しい順にスケジュール一覧が表示されます。
[![Alt text](/img/migrated/1-1-461969c3.png)](/img/migrated/1-1-461969c3.png)
**手順2** : 表示される情報を確認します。
| フィールド | 説明 |
| --- | --- |
| Schedule ID | システムが自動採番する識別子。 |
| Schedule Name | スケジュールの名前。クリックすると詳細が表示されます。 |
| Frequency | 実行頻度（On-demand / Weekly / Monthly）。 |
| Interval | ジョブの実行タイミング。「日付 + 時刻」形式で表示。 |
| Last Launch | 直近の実行日時とステータス（Running / Success Finished / Failed Error / None）。 |
| Next Run | 次回の実行時刻、または空白。 |
| Created By | スケジュールの作成者。 |
| Action | Edit（編集）、Delete（削除）。 |
**手順3:** スケジュールの詳細を表示します。
任意のスケジュール名をクリックしてSchedule detail画面に移動します。
[![Alt text](/img/migrated/Screenshot-2025-11-14-141945-fef523cc.png)](/img/migrated/Screenshot-2025-11-14-141945-fef523cc.png)
Create Scheduleと同様のフィールドを含むschedule detail画面が表示されます。
