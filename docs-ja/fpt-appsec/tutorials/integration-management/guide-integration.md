# Integration Management - 連携管理

**Org Admin** または **Team Manager** のみ編集権限を持ちます。
**手順1** : Security Platformメニューの DevSecタブに移動し、**Integration** を選択して連携（integration）の一覧を表示します。使用するOrgまたはTeamを選択します。
[![anhfsec](/img/migrated/in1-1-bbf498e7.png)](/img/migrated/in1-1-bbf498e7.png)
_注意: org/teamを選択していない場合、各連携タイプは無効化されており選択できません。_
**手順2** : 連携するSource ControlまたはContainer Registryの種類を選択します。
[![anhfsec](/img/migrated/in2-6701890e.png)](/img/migrated/in2-6701890e.png)
