# 組織情報の編集

**Org Admin** のみ編集権限を持ちます。
**手順1** : サブタブをクリックし、**Organization** を選択します。
[![anhfsec](/img/migrated/od1-c5e2421f.png)](/img/migrated/od1-c5e2421f.png)
**手順2** : **Organization** 内の **Settings** を選択すると、**Organization** の情報を編集できます。
[![anhfsec](/img/migrated/od2-0ef111f4.png)](/img/migrated/od2-0ef111f4.png)
ここで **Code**、**Org Name**、**Description** などの情報を確認できます。[![anhfsec](/img/migrated/od3-new-5f772255.png)](/img/migrated/od3-new-5f772255.png)
**手順3** : **Information Tab** の **Edit** を選択します。
[![anhfsec](/img/migrated/od4-69f817e8.png)](/img/migrated/od4-69f817e8.png)
**手順4** : **Org Name** および **Org Description** を入力します。
[![anhfsec](/img/migrated/od5-0623354b.png)](/img/migrated/od5-0623354b.png)
**手順5** : **Save** をクリックします。
[![anhfsec](/img/migrated/od6-0ead19d4.png)](/img/migrated/od6-0ead19d4.png)
