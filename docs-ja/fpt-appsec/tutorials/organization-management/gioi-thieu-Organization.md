# Organization Management - 組織管理

組織管理機能の利用を開始するには、以下の操作を行ってください。
**手順1** : **FPT Portal** のメニューから **Security Platform** を選択し、次に **DevSec** を選択します。
[![anhfsec](/img/migrated/o1-9874c5f2.png)](/img/migrated/o1-9874c5f2.png)
**手順2** : サブタブに **Workspace**、**Integration**、**Organization**、**Team** の各セクションが表示されます（下図参照）。
各セクションの説明：
  * **Workspace** : プロジェクトの開発ライフサイクル全体を追跡し、各プロジェクトのセキュリティ状態を管理する集中型のワークスペース管理システムを提供します。
  * **Integration** : ソースコード管理（SCM）システムとのシームレスな連携を可能にし、ソフトウェア開発プロセスの継続性とセキュリティを確保します。
  * **Organization** : メンバー管理（Member Management）、チーム管理（Team Management）、組織内ロール管理（Role Management）、組織の共通設定（Settings）など、組織管理機能の概要を提供します。
  * **Team** : チームのメンバー管理（Member Management）およびチームの共通設定（Settings）など、チーム管理機能の概要を提供します。
