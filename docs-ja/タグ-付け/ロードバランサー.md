# Load Balancerのタグ管理

## 1. Classic Load Balancer作成時のタグ付け

**ステップ1**: メニューで **Classic load balancer** を選択し、**Load Balancerの作成** をクリックします。
[![file](/img/migrated/image-1721034946476-8389286e.png)](/img/migrated/image-1721034946476-8389286e.png)
**ステップ2**: **Load Balancerの作成** ダイアログボックスで、**タグの追加（任意）** フィールドにてtagを選択して、Load Balancerにtagを割り当てます。
[![file](/img/migrated/image-1721034960349-bc880785.png)](/img/migrated/image-1721034960349-bc880785.png)

## 2. Load Balancerのタグ管理

**ステップ1**: **Classic Load Balancer** 画面でLoad Balancerを選択し、**アクション** をクリックして **タグの管理** を選択します。
[![file](/img/migrated/image-1721034970802-6fc88cac.png)](/img/migrated/image-1721034970802-6fc88cac.png)
**ステップ2**: ダイアログボックスが表示され、Load Balancerに割り当てるtagを選択できます。
[![file](/img/migrated/image-1721034981643-db1aeca2.png)](/img/migrated/image-1721034981643-db1aeca2.png)
**ステップ3**: Load Balancerからtagを削除するには:
  * タグ名の横にある「x」をクリックして個別に削除します。
  * リストボックスの末尾の「x」をクリックすると、すべてのtagを一度に削除できます。
  * **保存** をクリックして変更を確定します。

> ⚠️ **警告:** Load Balancerを削除すると、システムは関連するすべてのtagを自動的に削除し、タグ付け画面のtagリストを更新します。

[![file](/img/migrated/image-1721034991057-8426f650.png)](/img/migrated/image-1721034991057-8426f650.png)
