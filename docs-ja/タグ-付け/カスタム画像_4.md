# カスタム画像_4

## **1. 画像をアップロードする際にタグを付ける**
**ステップ1** :メニューで[カスタム画像]を選択し>[画像のアップロード]を選択します
[![file](/img/migrated/image-1741233971912-f5345022.png)](/img/migrated/image-1741233971912-f5345022.png)
**ステップ2:**[画像のアップロード]ダイアログボックスで、ユーザーは[タグの追加（オプション)]フィールドでタグを選択して、画像にタグを付けることができます 
注: ユーザーが付けられるタグは最大 50 個までです 
[![file](/img/migrated/image-1741234007389-334e1a8a.png)](/img/migrated/image-1741234007389-334e1a8a.png)
## **2. カスタム画像のタグ管理**
**ステップ1** :メニューで[カスタム画像]を選択し>、[アクション]を選択し>[タグの管理]を選択します 
[![file](/img/migrated/image-1741234051442-5347452b.png)](/img/migrated/image-1741234051442-5347452b.png)
**ステップ2** :ダイアログボックスが表示され、ユーザーは画像に付けるタグを選択します 
注:1枚の 画像に付けられるタグは最大 50 個までです 
[![file](/img/migrated/image-1741234075468-47033485.png)](/img/migrated/image-1741234075468-47033485.png)
ステップ 3: ユーザーが画像からタグを削除する場合 
タグ名の横にある「x」印を選択して、タグを外す操作を実行します（ユーザーはリストボックスの最後にある「x」をクリックすることで、すべてのタグを画像から一度に外すこともできます）。その後、保存を選択します。 
**ユーザーが画像を削除すると、関連付けられたタグが自動的に削除され、タグ付け画面のタグリストが再度更新されることに注意してください**
[![file](/img/migrated/image-1741234075468-47033485.png)](/img/migrated/image-1741234075468-47033485.png)
