# タグ別請求

タグ別請求を使用すると、割り当てたtagに基づいてリソースグループごとのコストを追跡できます。プロジェクト、部門、環境別にコストを配分する際に特に便利です。

tagごとのコスト分析は [**Cost Explorer**](../cost-explorer/index.md) で確認できます。
