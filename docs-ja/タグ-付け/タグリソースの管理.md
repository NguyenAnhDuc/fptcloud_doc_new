# タグリソースの管理

ユーザーは、次のリソースにタグを付けたり解除したりすることができます。 
  * インスタンス管理 
  * ストレージディスク 
  * スナップショット 
  * カスタムイメージ 
  * フローティングIP 
  * セキュリティグループ 
  * サブネット 
  * クラシックロードバランサー 

ユーザーは、次のタイミングでタグを追加または削除できます。 
  * ユーザーがリソースを作成している間 
  * ユーザーがリソースを作成した後 

注: 1個のタグは複数のリソース名に付けでき、1つのリソース名は複数のタグに付けることもできます。
