# ストレージディスク_2

## **1. ストレージディスクのタグ管理**
**ステップ1** : ストレージディスク画面で、ユーザーはディスクを選択し、[アクション]を選択してから、[タグの管理]を選択します
[![file](/img/migrated/image-1741232003164-9a2bf5ea.png)](/img/migrated/image-1741232003164-9a2bf5ea.png)
**ステップ2** :モーダルを表示し、タグを付け外し、[保存]をクリックしてアクションを保存します 
[![file](/img/migrated/image-1741232208621-39a84873.png)](/img/migrated/image-1741232208621-39a84873.png)
**ステップ3** :ユーザーがストレージディスクからタグを削除する場合。 
タグ名の横にある「x」印を選択して、タグを外す操作を実行します（ユーザーはリストボックスの最後にある「x」をクリックすることで、すべてのタグをストレージディスクから一度に外すこともできます）。その後、保存を選択します。 
ユーザーがストレージディスクを削除すると、関連付けられたタグが自動的に削除され、タグ付け画面のタグリストが更新されることに注意してください。 
[![file](/img/migrated/image-1741232252765-cc76d6f6.png)](/img/migrated/image-1741232252765-cc76d6f6.png)
