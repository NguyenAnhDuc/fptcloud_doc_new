# インスタンス管理 _1

## **1. ユーザーが仮想マシンを作成するとすぐに、タグを仮想マシンに付けます**
**ステップ 1** : [インスタンス管理] メニューで、[インスタンスの作成] を選択して仮想マシンを作成します 
[![file](/img/migrated/image-1741232376997-16719c16.png)](/img/migrated/image-1741232376997-16719c16.png)
**ステップ 2** : [インスタンスの作成] ダイアログ ボックスで、ユーザーは [タグ (オプション)] フィールドでタグを選択して、タグを仮想マシンに付けられますす 
注: ユーザーは、1 つの仮想マシンに最大 50 個のタグしか付けられません 
[![file](/img/migrated/image-1741232489382-5d0601ff.png)](/img/migrated/image-1741232489382-5d0601ff.png)
**ステップ3** :ユーザーは[作成]をクリックして仮想マシンを作成すると、仮想マシンに付けられたタグがインスタンス管理画面の[タグ ]フィールドに表示されます 
[![file](/img/migrated/image-1741232935245-ba16e74d.png)](/img/migrated/image-1741232935245-ba16e74d.png)
## **2. 仮想マシンのタグの管理**
**ステップ 1** : [インスタンス管理] 画面で、仮想マシンを選択し、[アクション] を選択して、[タグの管理] を選択します
[![file](/img/migrated/image-1741232826455-0af4d5aa.png)](/img/migrated/image-1741232826455-0af4d5aa.png)
**ステップ2** : ダイアログボックスが表示され、ユーザーは仮想マシンに付けるタグを選択します。 
注: 1つの仮想マシンに付けられるタグは最大 50 個までです 
[![file](/img/migrated/image-1741232880854-4c47d756.png)](/img/migrated/image-1741232880854-4c47d756.png)
