# スナップショット_3

## **1. インスタンスからスナップショットを作成する際にタグを付ける**
**ステップ 1** : メニューで [インスタンス管理] を選択し、[アクション] を選択し、[スナップショットの作成 ] を選択します 
[![file](/img/migrated/image-1741233306395-34941a5d.png)](/img/migrated/image-1741233306395-34941a5d.png)
**ステップ2** :[インスタンススナップショットの作成]ダイアログボックスで、 ユーザーは[タグの追加（オプション)]フィールドでタグを選択してスナップショットにタグを付けることができます。 
注: ユーザーが付けられるタグは最大 50 個までです
[![file](/img/migrated/image-1741233246636-cc4f52f2.png)](/img/migrated/image-1741233246636-cc4f52f2.png)
## **2. ストレージディスクからスナップショットを作成する際のタグ付け**
**ステップ1** :メニューでストレージディスクを選択し>「アクション」を選択し>「ボリュームスナップショット作成」を選択します 
[![file](/img/migrated/image-1741233440129-ce943b78.png)](/img/migrated/image-1741233440129-ce943b78.png)
**ステップ2** :[ボリュームスナップショット作成]ダイアログボックスで、 ユーザーは[タグの追加（オプション)]フィールドでタグを選択して、仮想マシンにタグを付けることができます。 
注: ユーザーが付けられるタグは最大 50 個までです 
[![file](/img/migrated/image-1741233491213-22274c4d.png)](/img/migrated/image-1741233491213-22274c4d.png)
## **3. スナップショットのタグ管理**
**ステップ1** :メニューで「スナップショット」を選択します>「インスタンススナップショット」フィールドで、「タグの管理」を選択します 
[![file](/img/migrated/image-1741233544097-fcd1048c.png)](/img/migrated/image-1741233544097-fcd1048c.png)
**ステップ2** :ダイアログボックスが表示され、ユーザーはスナップショットに付けるタグを選択します 
注: 1つのインスタンススナップショット に付けられるタグは最大 50 個までです 
[![file](/img/migrated/image-1741233570389-8ff5bc12.png)](/img/migrated/image-1741233570389-8ff5bc12.png)
**ステップ 3** : ユーザーがインスタンススナップショット からタグを削除する場合 
タグ名の横にある「x」印を選択して、タグを外す操作を実行します（ユーザーはリストボックスの最後にある「x」をクリックすることで、すべてのタグをインスタントスナップショットから一度に外すこともできます）。その後、保存を選択します。 
[![file](/img/migrated/image-1741233570389-8ff5bc12.png)](/img/migrated/image-1741233570389-8ff5bc12.png)
