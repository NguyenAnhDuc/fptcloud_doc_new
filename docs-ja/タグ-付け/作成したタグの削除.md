# 作成したタグの削除

注: これにより、タグに関連付けられたリソースが同時に削除されます。このタグを削除すると、復元 できません
ユーザーは次の操作を行います。
**ステップ1** :メニューで[タグ付け]を選択し、[アクション]>[削除]を選択します
[![file](/img/migrated/image-1740552377439-b880b0b2.png)](/img/migrated/image-1740552377439-b880b0b2.png)
**ステップ2** : タグ名とタグにアタッチされているリソースの数を表示する警告ダイアログ が表示され、削除アクションを確認します。ユーザーは [削除] ボタンをクリックして確認します。 タグにアタッチされたリソースが除去され、タグが削除されます。 
[![file](/img/migrated/image-1740552393637-bac57f0d.png)](/img/migrated/image-1740552393637-bac57f0d.png)
