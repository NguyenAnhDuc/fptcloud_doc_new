# タグの編集

ユーザーは次の操作を行います。
**ステップ1** : メニューで「タグ付け」を選択して、調整するタグで、「アクション」を選択し >「編集」を選択します 
[![file](/img/migrated/image-1740552096380-e009ae94.png)](/img/migrated/image-1740552096380-e009ae94.png)
**ステップ2** :ユーザーは情報を修正し、変更を適用するために
**[変更保存]** ボタンをクリックします。テナントでタグ付け権限を持つユーザーの場合:テナント（全てのVPC ）で現在適用されているタグは、ユーザーが[VPCのカスタマイズを有効にする]チェックボックスをオンにすることで、特定のVPCを選択するように変更できます。
**注：**
  * このシステムでは、ユーザーが特定の VPC の適用からテナント (全ての VPC) への適用に切り替えることをまだサポートしていません
  * リソーススコープ内の VPC を削除する場合、またはユーザーが特定の VPC への適用を切り替えた場合、システムは削除された VPC のリソースからタグを自動的に削除します。
