# サブネット_6

## **1. サブネットを作成タグを付けます**
**ステップ 1** : メニューで [サブネット] を選択し、[サブネットの作成] を選択します
[![file](/img/migrated/image-1741769962155-af65f72d.png)](/img/migrated/image-1741769962155-af65f72d.png)
**ステップ2** : [サブネットの作成]のページで、ユーザーは[タグの追加（オプション)]フィールドでタグを選択して、サブネットにタグを付けることができます。
注: ユーザーは、1つのサブネットに最大 50 個のタグを付けることができます
[![file](/img/migrated/image-1741770063377-0fc380fb.png)](/img/migrated/image-1741770063377-0fc380fb.png)
**ステップ3** : ユーザーは他の情報を入力し、[サブネットの作成]をクリックしてサブネットを作成します。サブネットに付けられたタグはサブネット画面の[タグ ]フィールドに表示されます。
[![file](/img/migrated/image-1741770167082-aee1673e.png)](/img/migrated/image-1741770167082-aee1673e.png)
## **2. サブネットのタグ管理**
**ステップ 1** : メニューで「サブネット」 を選択し、[アクション] を選択し> [タグの管理] を選択します 
[![file](/img/migrated/image-1741770305123-08435efa.png)](/img/migrated/image-1741770305123-08435efa.png)
**ステップ2** : ダイアログボックスが表示され、ユーザーはサブネットに付けるタグを選択します。
注: 1つのサブネットに付けられるタグは最大 50 個までです
[![file](/img/migrated/image-1741770336310-9c85ad83.png)](/img/migrated/image-1741770336310-9c85ad83.png)
**ステップ 3** : ユーザーがサブネットからタグを削除する場合
タグ名の横にある「x」印を選択して、タグを外す操作を実行します（ユーザーはリストボックスの最後にある「x」をクリックすることで、すべてのタグをサブネットから一度に外すこともできます）。その後、保存を選択します。
**ユーザーがサブネットを削除すると、関連付けられたタグが自動的に削除され、[タグ付け]画面のタグリストが更新されることに注意してください。**
[![file](/img/migrated/image-1741770396357-69b1e002.png)](/img/migrated/image-1741770396357-69b1e002.png)
