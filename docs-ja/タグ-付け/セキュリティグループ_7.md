# セキュリティグループ_7

## **1. セキュリティグループを作成する際にタグを付けます**
**ステップ 1** : メニューで [セキュリティ グループ] を選択し> [セキュリティ グループの作成] を選択します
[![file](/img/migrated/image-1741771341856-b10796c1.png)](/img/migrated/image-1741771341856-b10796c1.png)
**ステップ2** : [セキュリティグループの作成]のページで、ユーザーは[タグの追加（オプション)]フィールドでタグを選択して、サブネットにタグを付けることができます。
注: ユーザーが1つのセキュリティ グループに付けられるタグは最大 50 個までです
[![file](/img/migrated/image-1741771388075-031914ca.png)](/img/migrated/image-1741771388075-031914ca.png)
**ステップ3** : ユーザーは他の情報を入力し、[セキュリティグループの作成]をクリックしてセキュリティグループを作成します。セキュリティグループに付けられたタグがセキュリティグループ画面の[タグ ]フィールドに表示されます
## **2. セキュリティグループのタグ管理**
ステップ 1: メニューで [セキュリティ グループ] を選択し、[アクション] を選択し> [タグの管理] を選択します 
[![file](/img/migrated/image-1741771427642-b18dc4dc.png)](/img/migrated/image-1741771427642-b18dc4dc.png)
**ステップ2** :ダイアログボックスが表示され、ユーザーはセキュリティグループに付けるタグを選択します。
**注:1つの セキュリティ グループに付けられるタグは最大 50 個までです**
[![file](/img/migrated/image-1741771456062-bdeacac6.png)](/img/migrated/image-1741771456062-bdeacac6.png)
**ステップ 3:** ユーザーがセキュリティ グループからタグを削除する場合
タグ名の横にある「x」印を選択して、タグを外す操作を実行します（ユーザーはリストボックスの最後にある「x」をクリックすることで、すべてのタグをセキュリティグループから一度に外すこともできます）。その後、保存を選択します。
ユーザーがセキュリティグループを削除すると、関連付けられたタグが自動的に削除され、[タグ付け]画面のタグリストが更新されることに注意してください。
[![file](/img/migrated/image-1741771456062-bdeacac6.png)](/img/migrated/image-1741771456062-bdeacac6.png)
