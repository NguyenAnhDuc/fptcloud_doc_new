# タグ作成

ユーザーは以下のように操作してください。
**ステップ1** :「タグ付け」メニューで、「タグを作成」を選択します 
[![file](/img/migrated/image-1740551689064-5838a863.png)](/img/migrated/image-1740551689064-5838a863.png)
**ステップ2** : 以下の情報を入力します。
  * キー: ユーザーのキータグを表示する名前を入力します。例: env project,...
  * 値: タグの値を入力します。例: 「 プロダクション」または 「 FPT 」 。 

※ ユーザーはこのフィールドを空白のままにすることができます。
  * 色: タグの色を選択します。現在、システムは赤、黄、緑の3色のみをサポートしており、デフォルトのタグは白です。
  * テナントでタグ付け権限を持つユーザーの場合:生成されたタグはVPC全体に適用されます。特定のVPCにタグを作成するために、ユーザーは「VPCのカスタマイズを有効にする」を選択し、テナントのVPCリストからVPCを選択します。
  * VPC でタグ付け権限を持つユーザーの場合: ユーザーは、 権限を与えられたVPC のリストからVPC を選択します。 作成されたタグは、選択した VPC に適用されます。

[![file](/img/migrated/image-1740551705309-e51d222f.png)](/img/migrated/image-1740551705309-e51d222f.png)
ステップ3: [作成]をクリックしてタグを作成します。 タグが作成されると、タグ設定ページに表示されます
[![file](/img/migrated/image-1740551792619-5d66ac78.png)](/img/migrated/image-1740551792619-5d66ac78.png)
