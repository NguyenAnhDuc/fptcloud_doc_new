# コストエクスプローラー

コストエクスプローラーは、FPT Cloudのサービスコストをサービス別、リソースタイプ別、VPC別、タグ別など多角的に分析・追跡できるツールです。クラウドコストの管理と予算の最適化が必要な場合に特に有用です。

:::note
コストエクスプローラーに表示されるコストは、Customize Filterで設定したフィルタに基づいてカスタマイズされたものであり、請求期間による月末の実際の請求額と異なる場合があります。
:::
