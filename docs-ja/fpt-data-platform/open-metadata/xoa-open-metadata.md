# Open Metadata service の削除

**Open Metadata** サービスを削除するには、以下の手順に従ってください。

**ステップ 1:** メニューバーで **Data Platform** > **Workspace Management** > **Workspace name** を選択します。

**ステップ 2:** **My Service** セクションで **Open Metadata** サービスを選択します。**Detail Open Metadata Service** 画面で **Delete** アイコンをクリックします。

![Image](images/xoa-open-metadata/img-001.png)

**ステップ 3:** **Delete Open Metadata** ダイアログボックスが表示されます。入力欄に「**delete**」と入力し、**Confirm** をクリックして削除を完了します。

![Image](images/xoa-open-metadata/img-002.png)
