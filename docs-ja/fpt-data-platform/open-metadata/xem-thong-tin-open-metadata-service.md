# Open Metadata service の詳細確認

**Open Metadata** の情報を確認するには、以下の手順に従ってください。

**ステップ 1:** メニューバーで **Data Platform** > **Workspace Management** > **Workspace name** を選択します。

**ステップ 2:** **Workspace** の詳細セクションで **Open Metadata** を選択します。

  * **Tab Essential Information**

    * ユーザーが設定した **Open Metadata** サービスの詳細情報が表示されます。画面に表示されている **URL/Username/Password** を使用して **Open Metadata** にアクセスします。

![Image](images/xem-thong-tin-open-metadata-service/img-001.png)

  * **Advanced**

    * **Database** および **Search Engine Database** の情報が表示されます。

![Image](images/xem-thong-tin-open-metadata-service/img-002.png)
