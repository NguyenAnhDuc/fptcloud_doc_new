# Processing service

**Processing Service** は、FPT Data Platform 上に展開されるサービスであり、ユーザーが設定したコンピューティングリソース（Compute）を通じて、バッチまたはリアルタイムのデータ処理機能を提供します。CPU および GPU 環境の両方をサポートし、分散環境で高性能なデータ処理タスクを柔軟かつ効率的に実行することができます。
