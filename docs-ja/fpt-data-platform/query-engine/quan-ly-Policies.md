# Policies の管理

**Query Engine** の **Policies** 情報が表示されます。Authentication type が **Integration Ranger** の場合、**Query Engine** は **Policies** タブを表示せず、すべてのアクセス制御設定は **Ranger-Admin** を通じて行われます。

![Image](images/quan-ly-Policies/img-001.png)

ユーザーは編集アイコンをクリックして **Policies** 情報を編集できます。

アクセス制御ポリシーの詳細については、[**こちら**](https://trino.io/docs/current/security/file-system-access-control.html)を参照してください。
