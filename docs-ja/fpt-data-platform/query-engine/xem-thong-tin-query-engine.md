# Query Engine の詳細表示

**Query Engine** の情報を表示するには、以下の手順に従ってください。

**ステップ 1:** メニューバーで **Data Platform** > **Workspace Management** > **Workspace name** を選択します。

**ステップ 2:** **My services** セクションで **Query Engine** を選択します。

画面には 5 つのタブが表示されます: **Essential Information**、**Authentication**、**Connectors**、**Users**、**Policies**

**_1\. Essential Information_**

Query Engine の詳細情報が表示されます。

![Image](images/xem-thong-tin-query-engine/img-001.png)

**_2\. Authentication_**

**Query Engine** の認証情報が表示されます。

![Image](images/xem-thong-tin-query-engine/img-002.png)

**_3\. Connectors_**

**Query Engine** の **Connector** 情報が表示されます。

![Image](images/xem-thong-tin-query-engine/img-003.png)

**_4\. Users_**

**Query Engine** の **User** リストが表示されます。

![Image](images/xem-thong-tin-query-engine/img-004.png)

**_5\. Policies_**

**Query Engine** の **Policies** 情報が表示されます。Authentication type が **Integration Ranger** の場合、**Query Engine** は **Policies** タブを表示せず、すべてのアクセス制御設定は **Ranger-Admin** を通じて行われます。

![Image](images/xem-thong-tin-query-engine/img-005.png)
