# 初期設定

**FPT Cloud Data Platform** サービスを利用開始するには、FPT Portal にログインする必要があります。

<https://console.fptcloud.com> にアクセスし、付与された FPT Cloud アカウントの認証情報でサインインしてください。使用するサービスに対応した Tenant、Region、VPC を正しく選択してください。

FPT Cloud アカウントをお持ちでない場合は、<https://fptcloud.com/> のホームページをご覧ください。

**Sign Up** 機能を選択し、システムの指示に従って必要な情報を入力してください。その後、サポートチームよりご連絡し、情報を確認した上でアカウントを作成いたします。

操作中に何らかのエラーが発生した場合は、直ちにサポートチームまでご連絡ください。
