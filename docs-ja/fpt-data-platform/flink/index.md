# Apache Flink

Apache Flink は、**リアルタイムのストリーム処理（stream processing）** に特化したオープンソースの**分散データ処理フレームワーク**です。**バッチ処理（batch processing）** もサポートしていますが、**低レイテンシでの継続的なデータ処理**能力が最も際立っています。

Flink は柔軟なスケーラビリティを持ち、ステートフル処理（stateful processing）をサポートし、**一貫性**を保証します。そのため、**ビッグデータ分析（Big Data Analytics）、Machine Learning、IoT、金融、システム監視**などのシステムにおいてトップの選択肢となっています。
