# Instance scheduleからの仮想マシンの解除

スケジュールが不要になった仮想マシンはいつでも解除できます。解除後、仮想マシンは手動で操作するまで現在の状態を維持します。

**ステップ1:** **Instance Schedule Management** ページの **Action** 列で **Apply instances** をクリックします。

**ステップ2:** 解除したいinstanceの名前の横にある "x" をクリックします。すべての仮想マシンを一括で解除するには、リストボックスの末尾にある "x" をクリックします。その後、**Apply instances** をクリックします。
