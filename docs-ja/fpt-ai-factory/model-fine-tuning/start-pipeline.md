# Start Pipeline

パイプラインは手動で開始することも、自動的に実行するようにスケジュールすることもできます。
  * **手動実行:**

**パイプライン管理** ダッシュボードの **[アクション**] 列で、目的のパイプラインの **[開始]** を選択します **** 。これにより、パイプラインの実行がすぐにトリガーされます。
  * **スケジュール実行:**

パイプラインを特定の間隔で実行するようにスケジュールするには、パイプラインの作成時にスケジュールを設定するか、既存のパイプラインを編集します。
パイプラインを選択し、**[編集]** をクリックして **[トレーニング設定]** に移動し、[**トリガー] > [スケジュール]** を選択し、スケジュールを指定します。
**注意：**
パイプラインを正常に実行するには、次のことを確定してください。
  * 残高に十分なお金があります。参考ページ: <https://ai.fptcloud.com/pricing>
  * 合計で最大 **20** 個のパイプラインを作成できます。
  * 一度に開始できるパイプラインは **2** つだけです
  * 各パイプラインは、最大 **2 つのノード** で構成できます。
