# View Execution Details

**実行の詳細**
  * 特定の実行を選択する
  * 「**実行の詳細** 」 タブを選択すると、実行の詳細、パイプライン設定、モデルアウトプット、メトリックが表示されます
  * **「モデルリンク」** フィールドのリンクを使用して、モデルハブで正常に微調整されたモデルにアクセスします。
  * テストレポートのダウンロード
