# 支払い請求書

:::note
この機能は現在、**ベトナムリージョン** のみに適用されます。
:::

## 請求情報

クレジットを追加する前に、請求および税務目的のために **請求情報（Billing Information）** を提供する必要があります。

## クレジットの追加

1. 左側メニューで **Account** → **Billing** に移動するか、**FPT AI Factory ホームページ** で **Add Credit** をクリックします。
2. 請求情報が未入力の場合、入力を求めるポップアップが表示されます。

![クレジット追加時の請求情報ポップアップ](/img/migrated/image2025-9-10_10-48-26-ae626517.png)

## 請求情報の更新

後から請求情報を更新する必要がある場合：

1. 左側メニューで **Account** → **Billing** に移動します。
2. **Settings** タブを開きます。
3. **編集（Edit）** アイコンをクリックして情報を更新します。

![請求設定の編集画面](/img/migrated/image2025-9-8_23-26-45-eef4a774.png)

:::tip
請求情報は一度だけ入力すれば十分です。後から請求情報を更新した場合、変更は将来の請求書と支払いに適用されます。請求情報に入力したメールアドレスにベトナムの税務請求書が送付されます。
:::

:::note
国としてベトナムを選択した場合、通貨は自動的に **VND（ベトナムドン）** に設定されます。
:::

## 支払い履歴

クレジットを追加するたびに支払い取引が作成されます。
**Account** → **Billing** → **Payment History** に移動して、過去のすべての支払いを確認できます。

![支払い履歴ページ](/img/migrated/image2025-9-10_10-54-3-b3bee087.png)

| フィールド | 説明 |
|---|---|
| **Date** | 取引の日時。 |
| **Source** | 取引に使用された決済ゲートウェイ。 |
| **Amount** | 支払い金額。 |
| **Payment Status** | 支払いのステータス（例：Paid、Failed）。 |
| **Recipient Email** | 支払い時に請求情報に入力されたメールアドレス。 |
| **Invoice Status** | 請求書の生成および配信の進捗状況（例：Processing、Ready）。 |
| **Invoice** | 公式の商業請求書（または購入領収書）ファイルをダウンロードします。 |

## 請求書

FPT AI Factory では2種類の請求書をご利用いただけます。

1. **ベトナム税務請求書（Vietnam Tax Invoice）** — ベトナムの税務規制に準拠するために発行されます。請求情報に指定されたメールアドレスに送付されます。
2. **商業請求書（または購入領収書）（Commercial Invoice）** — 会計または業務目的の支払い証明として発行されます。**支払い履歴（Payment History）** ページからダウンロードできます。
