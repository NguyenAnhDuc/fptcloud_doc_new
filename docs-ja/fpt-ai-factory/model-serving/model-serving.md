# Model Serving

Model Serving は、FPT AI Factory 上でトレーニング済みAIモデルをスケーラブルな REST API エンドポイントとしてデプロイ・運用するためのサービスです。モデルの公開から負荷分散、モニタリングまでを一貫して管理できます。

## 概要

データサイエンティストやMLエンジニアが構築したモデルを、本番環境で安定的に運用するための基盤を提供します。インフラの管理をFPT Cloudに任せることで、モデルの改善と価値提供に集中できます。オートスケーリングにより、トラフィックの増減に自動的に対応します。

## 主な機能

- **モデルデプロイ**: トレーニング済みモデルをワンクリックでAPIエンドポイントとして公開
- **オートスケーリング**: リクエスト数に応じたリソースの自動増減
- **バージョン管理**: 複数のモデルバージョンを並行運用し、段階的なロールアウトが可能
- **モニタリング**: レイテンシ、スループット、エラー率のリアルタイム監視
- **A/Bテスト**: 異なるモデルバージョンへのトラフィック分割テスト
- **認証・認可**: APIキーやトークンによるアクセス制御

## 対応フレームワーク

TensorFlow、PyTorch、scikit-learn、ONNX など主要なMLフレームワークに対応しています。

## 始め方

FPT Cloud ポータルから「AI Factory」→「Model Serving」に移動し、デプロイするモデルファイルをアップロードしてエンドポイントを作成してください。
