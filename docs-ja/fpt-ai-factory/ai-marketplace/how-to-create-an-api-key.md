# API キーの作成

API キーの作成・管理を行うには、AI Marketplace ダッシュボードの **マイアカウント → My API Keys** に移動してください。

:::tip
キーの作成・編集・削除を含む詳細な手順については、[マイアカウント](./マイアカウント.md)ガイドを参照してください。
:::
