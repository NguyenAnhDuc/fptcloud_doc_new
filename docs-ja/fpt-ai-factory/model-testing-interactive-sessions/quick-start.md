# Quick Start

## Quick Start

Hướng dẫn nhanh để bắt đầu sử dụng **Model Testing – Interactive Sessions**:

1. [Đăng ký tài khoản](/docs/ja/fpt-ai-factory/model-testing-interactive-sessions/sign-up-for-an-account) tại <https://ai.fptcloud.com/> hoặc <https://ai.fptcloud.jp/>.
2. Làm theo [Step By Step](/docs/ja/fpt-ai-factory/model-testing-interactive-sessions/step-by-step) để tạo và quản lý session.
3. Sử dụng [Playground](/docs/ja/fpt-ai-factory/model-testing-interactive-sessions/playground) để interact và evaluate model.
4. Xem [Completion History](/docs/ja/fpt-ai-factory/model-testing-interactive-sessions/completion-history) để theo dõi các completion đã thực hiện.
