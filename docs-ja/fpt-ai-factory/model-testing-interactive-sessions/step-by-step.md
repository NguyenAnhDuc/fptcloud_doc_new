# Step By Step

After signing in to the FPT AI Factory Portal at at <https://ai.fptcloud.com/> or <https://ai.fptcloud.jp/>, follow the instruction below:

### Step 1: Add credit to account
  1. Select **Billing** in the Sidemenu
  2. Click **Add Credit** button and enter an amount and payment information to complete. Or, if you have a voucher from FPT, apply your valid code in the **Voucher code** to redeem credits.

### Step 2: Create session
  1. Select **Model Testing** in the Sidemenu, and navigate to **Interactive Sessions**.
  2. Click the button **Create New Session** and configure pipeline deployment.

### Step 3: Manage session
  1. The session list created successfully, displays the **Interactive Sessions** page.
  2. In the **Actions** column, you can **run** , **cancel** and **delete sessions** to manage effectively.

### Step 4: Evaluation model
  1. Open **Session list**
  2. In the **Name** column, you click a session you want to chat with the model
  3. Open **Playground** , where you chat with the model to evaluate

### Step 5: Integrate model
  1. Open **Session list**
  2. In the **Name** column, you click a session you want to chat with the model
  3. Open **Playground**
  4. Click icon **Get API Endpoint** and copy link
