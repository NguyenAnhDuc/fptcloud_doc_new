# Data Hub

Data Hub は、FPT AI Factory 上のAI/MLワークフロー向けに、データの収集・変換・管理・パイプライン処理を一括して提供するデータ管理サービスです。大規模なデータセットを効率的に扱い、モデルのトレーニングや推論に必要なデータを安定して供給します。

## 概要

AI/MLプロジェクトにおけるデータ管理の複雑さを解消するため、Data Hub はデータの取り込みから前処理、バージョン管理、パイプライン実行までを統合的にサポートします。データエンジニアとデータサイエンティストが協力してデータ基盤を構築できる環境を提供します。

## 主な機能

- **データ取り込み**: 各種ソースからのデータ収集と一元管理
- **データ変換**: ETLパイプラインによるデータの前処理・クレンジング
- **バージョン管理**: データセットのバージョン管理と再現性の確保
- **パイプライン実行**: スケジュール実行や依存関係に基づいたワークフロー管理
- **アクセス制御**: チームやプロジェクト単位でのデータアクセス権限管理

## 始め方

FPT Cloud ポータルから「AI Factory」→「Data Hub」に移動し、新しいデータソースを登録してください。データパイプラインの作成と実行は、直感的なUIから操作できます。
