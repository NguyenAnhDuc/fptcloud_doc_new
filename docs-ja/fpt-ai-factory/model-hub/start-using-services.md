# サービスの利用開始

残高に十分なクレジットがあれば、すぐに AI Factory のサービスを使い始めることができます。クレジットは使用量に応じて自動的に差し引かれます。
Model Hub の請求方法の詳細については、[請求計算ガイド](/docs/ja/fpt-ai-factory/ai-factory-billing/)/を参照してください。
