# model 詳細の表示

model には以下のコンポーネントが含まれます： [![file](/img/migrated/image-1757330874098-15240c3e.png)](/img/migrated/image-1757330874098-15240c3e.png)
  * **Model title：** model の名前
  * **Base model：** 使用しているベース model
  * **Model size (Parameters)：** model の合計サイズ
  * **Latest version**：詳細を表示した時点に最も近い日時に作成された最新バージョン
  * **Created date：** model が作成された日付
  * **Created by：** model を作成したユーザーアカウント
  * **Modified**：model 情報が最後に更新された日付
  * **Tags**：model に関連付けられたタグ
  * **Actions**：model で使用可能なアクション
    * Fine-Tuning
    * Deploy
    * Download Model
    * Upload Model

### **model カード**
[![file](/img/migrated/image-1757330874098-15240c3e.png)](/img/migrated/image-1757330874098-15240c3e.png) model カード：model の概要情報

### **バージョン**
[![file](/img/migrated/image-1757331088375-4ce0f3b9.png)](/img/migrated/image-1757331088375-4ce0f3b9.png)
  * **Index**：バージョンの番号順。
  * **Version**：バージョンの一意識別子（詳細についてはクリック可能）。
  * **Status**：バージョンの現在の状態（例：_Created_、_Training_、_Deployed_）、色付きドットで示されます。
  * **Tags**：カテゴリー分類のためのオプションラベル。
  * **Size**：バージョンのファイルサイズ。
  * **Created Date**：バージョンが作成されたときのタイムスタンプ。
  * **Actions**：バージョンを操作するオプション（例：編集、削除）。

### **ファイルブラウザー：**
[![file](/img/migrated/image-1757331260313-705d0d8c.png)](/img/migrated/image-1757331260313-705d0d8c.png)
各バージョンに対応するアップロードされたファイルを表示します
  * **File**：ファイルまたはフォルダーの名前（例：`artifacts`）。
  * **Size**：ファイルサイズ。
  * **Download**：ファイルをダウンロードするオプション。
  * **Modified Date**：最後に変更されたときのタイムスタンプ。

### **共有**
[![file](/img/migrated/image-1757331669543-18821cb1.png)](/img/migrated/image-1757331669543-18821cb1.png)
model を他のユーザーと共有できます
  * **Email**：model が共有されている人物のメールアドレス
  * **Tenant**：オーナーのメールに関連するテナント
  * **Role**：model が共有されている人物のロール
  * **Date invited**：ユーザーが model に追加された日付
  * **Action**：model からユーザーを更新または削除するオプション
