# Completion History

セッションを選択し、**Completion History**タブを開いて、そのセッションの過去のすべてのcompletionを表示します。

[![file](/img/migrated/image-1746697738348.39.04-a9a17fa3.png)](/img/migrated/image-1746697738348.39.04-a9a17fa3.png)

- リストから好みのcompletionを選択するか、text completionの全履歴をダウンロードできます。
- 「Continue」をクリックして、選択したcompletionを使用してモデルをテストします。
