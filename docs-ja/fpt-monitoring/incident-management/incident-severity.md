# Incident severity

**Incident severity** は、インシデントの影響度レベルを定義・測定する機能です。明確で一貫した定義により、組織内の全員が深刻度を素早く把握し、適切に対応できます。

FMON Incidentに推奨設定済みの深刻度レベル：

| Severity | 説明 |
|----------|------|
| Critical | 影響が非常に大きく、即時対応が必要。例：顧客向けサービスの全面停止、セキュリティ/プライバシー違反、顧客データの損失。 |
| Major | 影響が大きい。例：一部の顧客に対する顧客向けサービスの停止、システムのコア機能への影響。 |
| Minor | 影響が軽微。例：代替手段がある小さな顧客への不便。 |
| Pending | 深刻度が未確定。できるだけ早く決定する必要がある。 |

## Incident severityの編集

1. 左メニューで **Apps** > **Incident** をクリックします。
2. **Settings** をクリックします。
3. **Severity** タブで、編集するseverityを選択し、**鉛筆アイコン** をクリックします。

   [![編集するseverityを選択](/img/migrated/Screenshot_10-5ea36620.png)](/img/migrated/Screenshot_10-5ea36620.png)

4. **Name** と **Description** を更新します。
5. （任意）色を変更します。
6. **Edit Severity** をクリックして保存します。

   [![Severity編集フォーム](/img/migrated/Screenshot_11-c1f3d936.png)](/img/migrated/Screenshot_11-c1f3d936.png)

## Incident severityの有効化/無効化

1. 左メニューで **Apps** > **Incident** をクリックします。
2. **Settings** をクリックします。
3. **Severity** タブで、変更するseverityを選択し、**トグルスイッチ** をクリックします。

   [![Severityの有効化/無効化](/img/migrated/Screenshot_12-9146f0c1.png)](/img/migrated/Screenshot_12-9146f0c1.png)
