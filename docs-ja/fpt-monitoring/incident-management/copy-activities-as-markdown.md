# Activity logのMarkdownエクスポート

この機能により、インシデントの全 **activity log** を **Markdown** 形式でエクスポートできます。

1. インシデントの詳細を開きます。
2. **Copy activities as markdown** をクリックします。Activity logがMarkdown形式でエクスポートされます。
