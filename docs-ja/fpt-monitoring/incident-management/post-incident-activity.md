# 対応活動の記録

インシデント対応プロセスに短いメモを追加できます。対応活動を記録することで、将来の類似インシデントの防止や影響軽減に役立ちます。

## Activityの作成

### テキストノート

1. 左メニューで **Apps** > **Incident** > **Home** をクリックします。
2. インシデントのタイトルをクリックして詳細を表示します。
3. テキストボックスに内容を入力します。
4. **Add note** をクリックします。

[![テキストノートの追加](/img/migrated/Screenshot_18-6d439a64.png)](/img/migrated/Screenshot_18-6d439a64.png)

### Query

1. 左メニューで **Apps** > **Incident** > **Home** をクリックします。
2. インシデントのタイトルをクリックして詳細を表示します。
3. **Query** タブを選択します。
4. **target** を選択し、**run query** をクリックします。
5. **Add query** をクリックしてactivityを追加します。

[![Queryアクティビティの追加](/img/migrated/Screenshot_12-1-81ff9fc0.png)](/img/migrated/Screenshot_12-1-81ff9fc0.png)

### Panel

システム上の既存パネルをインシデントに直接追加できます。

1. インシデント一覧でインシデントをクリックして詳細を表示します。
2. **Panel** タブをクリックします。
3. **Dashboard**、**Panel**、**Target** を選択します。
4. **Add query** をクリックします。

:::note
ダッシュボードがない場合は、先にDashboardページでダッシュボードを作成してください。
:::

[![Panelアクティビティの追加](/img/migrated/Screenshot_13-1-209145fc.png)](/img/migrated/Screenshot_13-1-209145fc.png)

### Alert

既存のアラートルールをリンクとしてインシデントに追加できます。

1. インシデント一覧でインシデントをクリックして詳細を表示します。
2. **Alert** タブをクリックします。
3. 作成済みのアラート一覧が表示されます。ない場合は **Alert** セクションで作成してください。
4. **Attach to incident** をクリックしてアラートをインシデントに関連付けます。

関連付けられたアラートは **Activity** セクションに表示されます。詳細表示や非表示の切り替えができます。

[![インシデントへのアラートの関連付け](/img/migrated/Screenshot_14-1-759066e9.png)](/img/migrated/Screenshot_14-1-759066e9.png)

## Activityの編集

1. インシデントのタイトルをクリックして詳細を表示します。
2. 編集するactivityにマウスを合わせ、右側の **三点アイコン** をクリックして **Edit Note** を選択します。

[![Edit Noteの選択](/img/migrated/Screenshot_19-f5138b1d.png)](/img/migrated/Screenshot_19-f5138b1d.png)

3. 内容を更新します。
4. **Save** をクリックして保存します。

## Activityの削除

1. インシデントのタイトルをクリックして詳細を表示します。
2. 削除するactivityにマウスを合わせ、右側の **三点アイコン** をクリックして **Delete Activity** を選択します。

[![Delete Activityの選択](/img/migrated/Screenshot_19-f5138b1d.png)](/img/migrated/Screenshot_19-f5138b1d.png)

## Activityへのコメント

1. インシデントのタイトルをクリックして詳細を表示します。
2. activityにマウスを合わせ、右端の **コメントアイコン** をクリックします。

[![コメントアイコン](/img/migrated/Screenshot_20-19b7c54b.png)](/img/migrated/Screenshot_20-19b7c54b.png)

3. テキストボックスに内容を入力します。
4. **Save** をクリックします。
