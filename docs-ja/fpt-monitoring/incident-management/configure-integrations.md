# Integration の設定

Incident Management は既存のツールとの連携により、インシデント対応中の可視性とコラボレーションを向上させます。

サポートされている integration は、incoming webhook、Grafana webhook、outgoing webhook、Jira、Slack、Telegram、Microsoft Teams です。

## Integration のインストールとアンインストール

1. **FMON Grafana** にログインします。
2. 左メニューで **Incident** を探し、**Integration** をクリックします。
3. integration 一覧で、**Integrate** をクリックしてインストール、または **Uninstall** をクリックして削除します。
