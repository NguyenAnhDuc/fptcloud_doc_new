# 初期設定

**FPT Monitoring** の **Incident Management** サービスを使用するには、以下の手順を実行します。

1. **FPT Monitoring Support** チームに連絡し、ダッシュボードで Incident Management サービスを有効化します。
2. **[https://console.fptcloud.com](https://console.fptcloud.com)** にアクセスし、発行された FPT Cloud アカウントでログインします。
3. 使用するサービスの **Tenant**、**Region**、**VPC** を正しく選択します。
4. メニューバーで **Monitoring** を選択し、Dashboard ページに移動してサービスを使用します。

:::note
FPT Cloud アカウントをお持ちでない場合は、**[https://fptcloud.com](https://fptcloud.com)** にアクセスし、**Sign Up** をクリックして必要な情報を入力します。サポートチームが確認のうえアカウントを作成し、ご連絡します。
:::

操作中にエラーが発生した場合は、すぐにサポートチームにお問い合わせください。
