# ワークスペースの作成

**Dashboard Grafana** にアクセスして使用するには、以下が必要です。

1. Monitoring サービスを有効化する
2. ワークスペースを正常に初期化する

**ご利用を希望される場合**や初期化の過程で問題が発生した場合は、サポートチームにご連絡いただければご案内いたします。
