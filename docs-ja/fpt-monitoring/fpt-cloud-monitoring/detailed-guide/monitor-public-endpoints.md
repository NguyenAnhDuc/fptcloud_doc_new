# パブリックエンドポイントを監視する

:::note
パブリックエンドポイント監視機能は Grafana ダッシュボードに移行されました。FPT Cloud Portal でのこの機能の開発は終了しています。

最良の体験のため、ダッシュボード上の **Endpoint Monitoring** サービスを直接ご利用ください。サポートが必要な場合はお問い合わせください。
:::
