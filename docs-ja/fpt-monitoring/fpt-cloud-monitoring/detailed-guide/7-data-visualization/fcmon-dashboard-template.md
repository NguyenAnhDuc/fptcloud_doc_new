# ダッシュボードテンプレート

ダッシュボードテンプレートを使用すると、FPT Cloud Monitoring が提供する既製のダッシュボードを使ってすぐに監視を開始できます。

## ダッシュボードテンプレートをインストールする

**手順 1:** [ダッシュボードにアクセスします](truy-cap-vao-dashboard.md)。

**手順 2:** 左メニューで **Apps** > **FMON Template** をクリックします。

[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)

**手順 3:** **Dashboard Template** をクリックします。

**手順 4:** ダッシュボードを選択して **Integrate** をクリックします。

[![Alt text](/img/migrated/Picture14-2-d9a00088.png)](/img/migrated/Picture14-2-d9a00088.png)

## ダッシュボードテンプレートを削除する

**手順 1:** [ダッシュボードにアクセスします](truy-cap-vao-dashboard.md)。

**手順 2:** 左メニューで **Apps** > **FMON Template** をクリックします。

[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)

**手順 3:** **Dashboard Template** をクリックします。

**手順 4:** インストール済みのダッシュボードを選択して **Remove** をクリックします。

[![Alt text](/img/migrated/Picture15-2-7940295e.png)](/img/migrated/Picture15-2-7940295e.png)
