# レポートの編集

1. **Home > Apps > Reporting** に移動します。
2. レポート名をクリックして詳細を開きます。
3. **Detail report** 画面で、編集したいセクション（**Dashboard**、**Schedule**、または **Format report**）の**鉛筆**アイコンをクリックします。

   システムが対応するセクションに移動し、必要な情報を更新できます。
