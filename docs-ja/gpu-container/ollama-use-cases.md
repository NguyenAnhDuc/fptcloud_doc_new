# Ollama Use Cases

このガイドでは、サポートされているNVIDIA GPU H100を使用してOllamaとOpenWebUIでdeepseek-r1:1.5bモデルの推論を実行する方法を説明します。
  1. **ステップ1：** Ollama-OpenWebUIテンプレートを使用してGPU Containerを作成します ![Alt text](/img/migrated/ollama-webui-create-1-d45ba747.png) ![Alt text](/img/migrated/ollama-webui-details-1-41dc42f5.png)
  2. **ステップ2：** OpenWebUIのrootユーザーを作成します ![Alt text](/img/migrated/ollama3-d0ee1755.png)
  3. **ステップ3：** このチュートリアルでは、deepseek-r1:1.5bモデルを例として使用します。ニーズに最も合った別のモデルを自由に選択してください。 ![Alt text](/img/migrated/Screenshot-2025-05-29-at-09.34.58-a0e4eb3d.png)
  4. **ステップ4：** プルが成功したら、モデルをテストするためにプロンプトを送信します ![Alt text](/img/migrated/Screenshot-2025-05-29-at-09.37.50-18de2e0c.png)
