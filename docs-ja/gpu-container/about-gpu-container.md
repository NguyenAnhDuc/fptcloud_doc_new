# About Gpu Container

## GPU Containerとは？
GPU ContainerはFPTのAI Factoryインフラ上でコンテナを直接実行できるマネージドコンピューティングプラットフォームです。数回のクリックで、独自のコンテナイメージを使用するか、FPTが提供する組み込みイメージを活用して、AIワークロードを簡単にデプロイおよび管理できます。
## どのように動作しますか？
GPU ContainerはGPUリソースへの直接アクセスを持つ隔離された環境でアプリケーションを実行します。NVIDIA Container Toolkitなどのツールを使用して、複雑な設定なしにAI、ML、データ処理などのタスクにGPUアクセラレーションを活用します。
## なぜGPU Containerを使うのですか？
  1. **コンテナ化されたAIデプロイ:** コンテナでAIワークロードを実行します。
  2. **高性能GPU演算:** AI/MLワークロード向けにNVIDIA GPUにアクセスできます。
  3. **迅速なデプロイ:** 組み込みテンプレートを使用して、ワンクリックで強力なGPUを搭載したコンテナを起動できます。
  4. **永続ストレージ:** このストレージは永続的で、コンテナが停止しても利用可能です。
