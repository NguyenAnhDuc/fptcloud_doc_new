# Export Config

後で再利用するために、設定のエクスポート機能でコンテナの設定を保存してYAMLファイルとしてダウンロードできます。
  1. コンテナ一覧画面から、Action（3点）メニューをクリックして「設定のエクスポート」を選択します。 [![Alt text](/img/migrated/export-2-2-430ec6f3.png)](/img/migrated/export-2-2-430ec6f3.png)
  2. または、コンテナ詳細ページを開いて「設定のエクスポート」をクリックします。 [![Alt text](/img/migrated/export-1-2-adff5d09.png)](/img/migrated/export-1-2-adff5d09.png)

YAMLファイルが **container-name.yaml** という形式の名前で自動的にダウンロードされます。
