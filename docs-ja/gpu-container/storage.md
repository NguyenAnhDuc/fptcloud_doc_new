# Storage

## 永続ディスク
GPU ContainerはコンテナのライフタイムにわたってHigh-Performance Storage（HPS）を提供します。ハードディスクと同様に機能し、コンテナが停止しても保持する必要があるデータを保存できます。
**主な特徴：**
  * コンテナが完全に削除されるまで利用可能です。
  * コンテナの再起動や再設定をまたいで保持する必要があるデータ、モデル、ファイルを保存することでデータの損失を防ぎます。

## 一時ディスク
一時ディスク（NVMe）はコンテナに一時的なストレージを提供するストレージの一種です。一時ディスクに保存されたデータはコンテナが停止または削除されると失われるため、コンテナをシャットダウンする前に重要なデータを必ずバックアップしてください。
## ストレージタイプの比較
| 機能 | 一時ディスク | 永続ディスク |
| --- | --- | --- |
| データの永続性 | 停止時に失われる | コンテナ削除まで保持 |
| ライフサイクル | コンテナのアクティブセッションに直接紐付け | コンテナのリース期間に紐付け |
| パフォーマンス | 最速（ローカルアタッチ） | 信頼性が高く、一般的に一時ディスクより低速 |
| 容量 | 選択したGPU instanceに従って固定 | 作成時に選択可能 |
| コスト | 無料 | $ 0.0066 GB/時間 |
| 最適な用途 | 一時的なセッションデータ、キャッシュ | 永続的なアプリケーションデータ、モデル、データセット |
