# ファイアウォールルールの作成

**ステップ1：** **Menu** から **Security Policies** (**Ctrl + 2**) > **Access Control** > **Policy** を選択します。

[![Userguide Checkpoint FPT NGFW 2022 51](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-51-10-d67afe3a.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-51-10-d67afe3a.png)

**ステップ2：** **Add rule above/below** をクリックするか、既存のルールを右クリックして **New rule Above/Below** を選択し、新しいルールを上または下に追加します。

[![Userguide Checkpoint FPT NGFW 2022 52](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-52-10-dc654ab1.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-52-10-dc654ab1.png)

**ステップ3：** 新しいルールを設定します：

- **Name：** [![Userguide Checkpoint FPT NGFW 2022 53](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-53-10-d3bd4303.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-53-10-d3bd4303.png)
- **Source**（送信元IP）：[![Userguide Checkpoint FPT NGFW 2022 54](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-54-10-7de8a148.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-54-10-7de8a148.png)
- **Destination**（宛先IP）：[![Userguide Checkpoint FPT NGFW 2022 55](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-55-10-59fca814.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-55-10-59fca814.png)
- **Service**（サービス/ポート）：[![Userguide Checkpoint FPT NGFW 2022 56](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-56-10-420bc60e.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-56-10-420bc60e.png)

**ステップ4：** Install Policyを実行します。
