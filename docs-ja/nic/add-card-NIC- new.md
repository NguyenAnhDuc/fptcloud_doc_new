# 新しいNICカードの追加

FPT Cloudは、必要に応じて新しいNICカードを簡単かつ柔軟に追加する機能を提供します。

1. **Add NIC** をクリックします。

[![Add NICボタン](/img/migrated/image-1691382699972-7f2687d8.png)](/img/migrated/image-1691382699972-7f2687d8.png)

2. 必要な **Subnet** を選択します。

[![Subnetの選択](/img/migrated/image-1691382717830-0934edbe.png)](/img/migrated/image-1691382717830-0934edbe.png)

[![Subnetの確認](/img/migrated/image-1691382726787-0afe397a.png)](/img/migrated/image-1691382726787-0afe397a.png)

[![Subnet選択済み](/img/migrated/image-1691382743112-1538d5f6.png)](/img/migrated/image-1691382743112-1538d5f6.png)

3. システムが処理を完了するまで待ちます。システムは選択したSubnetからIPアドレスをランダムに割り当てます。

[![システム処理中](/img/migrated/image-1691382798352-22ba262c.png)](/img/migrated/image-1691382798352-22ba262c.png)

4. カードが **Active** 状態になります。このカードを通じてInstanceはネットワークに接続できるようになります。

[![ActiveになったNICカード](/img/migrated/image-1691382818519-8982a88f.png)](/img/migrated/image-1691382818519-8982a88f.png)
