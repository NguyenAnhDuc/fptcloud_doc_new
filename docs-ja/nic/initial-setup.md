# 初期設定

FPT CloudのNIC（ネットワークインターフェースカード）機能を使用する前に、以下の手順を完了してください。

## FPT Cloudアカウントの作成とログイン

1. [https://console.fptcloud.com](https://console.fptcloud.com) にFPT Cloudの認証情報でログインします。
2. 使用するサービスに合わせて正しい **Tenant**、**Region**、**VPC** を選択します。

アカウントをお持ちでない場合は、[https://fptcloud.com/](https://fptcloud.com/) にアクセスして **Sign Up** をクリックし、画面の指示に従ってください。サポートチームがアカウント確認のために連絡します。

:::tip
操作中に問題が発生した場合は、サポートチームにお問い合わせください。
:::
