# NICカードの有効化

**Inactive** 状態のNICカードを有効化してネットワーク接続を復元できます。

1. 有効化したいNICカードの **Enable Card** をクリックします。

[![Enable Cardボタン](/img/migrated/image-1691382154697-155c022c.png)](/img/migrated/image-1691382154697-155c022c.png)

2. 有効化操作を確認します。

[![Enable Cardの確認](/img/migrated/image-1691382171803-02c83019.png)](/img/migrated/image-1691382171803-02c83019.png)

3. システムが処理を完了するまで待ちます。

[![システム処理中](/img/migrated/image-1691382523333-cd5f8412.png)](/img/migrated/image-1691382523333-cd5f8412.png)

4. カードが **Active** 状態になります。このカードを通じてInstanceはネットワークに接続できるようになります。

[![ActiveになったNICカード](/img/migrated/image-1691382532014-ecb97957.png)](/img/migrated/image-1691382532014-ecb97957.png)
