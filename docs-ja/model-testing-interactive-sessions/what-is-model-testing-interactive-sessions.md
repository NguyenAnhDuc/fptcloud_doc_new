# Model Testing Interactive Sessionsとは

**Model Testing - Interactive Sessions** は、ファインチューニング後の次の重要なステップです。カスタマイズされたAIモデルが実際のシナリオでどのように機能するかを検証・評価するために設計されています。FPT AI Factory Portalに組み込まれており、シンプルで直感的なインターフェースを通じて、ファインチューニング済みモデルと直接やり取りすることができます。

[![Alt text](https://fptcloud.com/wp-content/uploads/2025/05/Playground-Interactive-Session-1.png)](https://fptcloud.com/wp-content/uploads/2025/05/Playground-Interactive-Session-1.png)

精度やlossのような静的な指標だけに頼るのではなく、Interactive Sessionsは次のことができる動的な環境を提供します。

- **自然言語でモデルとチャット**する。
- **ドメイン固有のクエリをテスト**して文脈理解を評価する。
- **応答の品質を評価**する（関連性、トーン、完全性に基づいて）。
- **プロンプトを調整**し、リアルタイムでモデルの動作を観察することで**素早く反復**する。

このハンズオンのアプローチにより、モデルが数字の上だけでなく、顧客サービス、医療、金融、法律などのビジネスコンテキストに合わせた意味のある正確な出力を提供できることを確認できます。

Model Fine-tuningとのシームレスな統合により、**Model Testing - Interactive Sessions** は、強力かつ実用的なAIソリューションを自信を持って展開できるよう組織を支援します。トレーニングから本番環境へのギャップを埋め、モデルをスケールアップする前にチームが十分な情報に基づいた意思決定を行えるようにします。
