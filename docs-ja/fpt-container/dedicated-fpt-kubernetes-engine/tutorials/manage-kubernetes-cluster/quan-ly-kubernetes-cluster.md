# Kubernetes clusterの管理

Dedicated FPT Kubernetes Engine（D-FKE）は、以下の操作でKubernetes clusterのライフサイクル全体を管理できます。

- 新しいKubernetes clusterの作成
- 作成済みclusterの一覧表示
- cluster詳細情報へのアクセス
- clusterアクセス情報の取得
- Kubernetes clusterの削除
- Kubernetes clusterの停止
- Kubernetes clusterの起動
