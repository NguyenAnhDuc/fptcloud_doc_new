# Worker Nodes のスケール

**手順 1**: メニューで **Kubernetes** を選択します。**Kubernetes Management** ページが表示されます。

[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)

**手順 2**: 一覧から Worker Nodes をスケールしたい Cluster を選択します。

[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)

**手順 3**: **Node Pools** タブ > **Auto Scale** で、アイコン ![](/img/migrated/Picture32-9e7541e2.png) を選択します。

[![](/img/migrated/Picture33-042905c4.png)](/img/migrated/Picture33-042905c4.png)

**手順 4**: **Auto Scale Nodes Of Cluster** 画面が表示されます。必要な情報を入力します。
- **Min node / Max node**: 追加・削除する Worker Nodes の数を入力します。

[![](/img/migrated/Picture34-53f1906b.png)](/img/migrated/Picture34-53f1906b.png)

クラスターのスケール処理が実行されます。処理が完了するまで、クラスターの編集操作はできません。
