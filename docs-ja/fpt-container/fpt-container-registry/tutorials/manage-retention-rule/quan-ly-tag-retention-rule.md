# Tag retention rule を管理する

Tag retention rule を作成した後、要件の変化に応じてルールを編集・一時停止・削除できます。FPT Cloud は Portal 画面からすべての管理操作をサポートしています。

## Tag retention rule を更新する

1. **FPT Portal** メニューで **Container Registry** > **Policy** > 更新したい tag retention rule を選択 > **Actions** > **Edit** を選択します。

   [![Actions の Edit を選択](/img/migrated/Picture34-1-011e1b7c.png)](/img/migrated/Picture34-1-011e1b7c.png)

2. Retention rule の設定情報を更新します。

   [![Retention rule 編集フォーム](/img/migrated/Picture35-1-86268cbd.png)](/img/migrated/Picture35-1-86268cbd.png)

3. 更新後の結果を確認します。

   [![Retention rule 更新後の結果](/img/migrated/Picture36-1-de8e7ba1.png)](/img/migrated/Picture36-1-de8e7ba1.png)

## Tag retention rule を無効化する

1. **FPT Portal** メニューで **Container Registry** > **Policy** > 無効化したい tag retention rule を選択 > **Actions** > **Disable** を選択します。

   [![Actions の Disable を選択](/img/migrated/Picture37-24e9ec98.png)](/img/migrated/Picture37-24e9ec98.png)

2. 無効化後の状態を確認します。

   [![Retention rule の disabled 状態](/img/migrated/Picture38-1-1c93acbf.png)](/img/migrated/Picture38-1-1c93acbf.png)

## Tag retention rule を有効化する

1. **FPT Portal** メニューで **Container Registry** > **Policy** > 有効化したい tag retention rule を選択 > **Actions** > **Enable** を選択します。

   [![Actions の Enable を選択](/img/migrated/Picture39-1-7345b6d1.png)](/img/migrated/Picture39-1-7345b6d1.png)

2. 有効化後の状態を確認します。

   [![Retention rule の enabled 状態](/img/migrated/Picture40-1-d3963172.png)](/img/migrated/Picture40-1-d3963172.png)

## Tag retention rule を削除する

:::danger
Tag retention rule の削除は元に戻せません。削除されたルールは以降の実行には適用されません。
:::

1. **FPT Portal** メニューで **Container Registry** > **Policy** > 削除したい tag retention rule を選択 > **Actions** > **Delete** を選択します。

   [![Actions の Delete を選択](/img/migrated/Picture41-1-1a763d9c.png)](/img/migrated/Picture41-1-1a763d9c.png)

2. 削除後の結果を確認します。

   [![Retention rule 削除後の結果](/img/migrated/Picture42-1-004f9073.png)](/img/migrated/Picture42-1-004f9073.png)
