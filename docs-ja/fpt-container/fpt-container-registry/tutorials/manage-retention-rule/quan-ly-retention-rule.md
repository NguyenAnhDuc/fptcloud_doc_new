# Retention rule を管理する

Repository には時間の経過とともに多くの image が tag ごとに蓄積され、古くなった使われなくなった tag がストレージを消費します。

FPT Cloud の **Retention Rule** 機能を使用すると、定義した基準に従って image/tag を自動的に保持または削除するルールを設定でき、ストレージ使用量を最適化できます。
