# バージョン自動アップグレード機能（Automatic Upgrade Version）

Managed Kubernetes Cluster のバージョンを自動的にアップグレードする機能です。

この機能により、クラスターのバージョンが標準サポート終了に達したタイミングで自動的に次のバージョンへアップグレードされます。

詳細な使用方法については、以下のドキュメントを参照してください。

- [FPTCloud における Kubernetes バージョン管理プロセスの概要](./version-management-overview.md)
- [auto upgrade version 機能の詳細ガイド](./auto-upgrade-guide.md)
