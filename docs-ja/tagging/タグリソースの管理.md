# リソースのタグ管理

tagを使用すると、リソースをすばやく分類およびフィルタリングできます。FPT Cloud上の複数のリソースタイプにtagを割り当てでき、コスト管理とインフラストラクチャ管理の効率化に役立ちます。

## tag対応リソース

- Instance Management
- Storage Disk
- Snapshot
- Custom Image
- Floating IP
- Subnet
- Security Group
- Classic Load Balancer

## tagの割り当てタイミング

tagの追加または削除は、次の2つのタイミングで行えます:

- リソースの作成中。
- リソースの作成後。

:::note
1つのtagを複数のリソースに割り当てでき、1つのリソースに複数のtagを割り当てることもできます。
:::
