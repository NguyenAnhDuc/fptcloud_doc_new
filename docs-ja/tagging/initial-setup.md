# 初期設定

タグ付け機能を使用する前に、FPT Cloudアカウントを取得し、FPT Cloud Portalにログインする必要があります。

## 前提条件

- ✅ FPT Cloudアカウントを所有していること。
- ✅ 必要なTenant、Region、VPCへのアクセス権が付与されていること。

## FPT Cloud Portalにログイン

1. [https://console.fptcloud.com](https://console.fptcloud.com/) にアクセスします。
2. 発行されたFPT Cloudアカウントとパスワードでログインします。
3. 使用するサービスの **Tenant**、**Region**、**VPC** を選択します。

:::tip
FPT Cloudアカウントをお持ちでない場合は、[https://fptcloud.com](https://fptcloud.com/) にアクセスし、**Sign Up** を選択して必要な情報を入力してください。サポートチームからアカウント作成の確認のご連絡をいたします。
:::

:::note
操作中にエラーが発生した場合は、サポートチームにお問い合わせください。
:::
