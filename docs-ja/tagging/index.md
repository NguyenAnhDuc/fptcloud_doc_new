# タグ付けの概要

tagを使用すると、FPT Cloud上のリソースを管理、識別、整理、検索、フィルタリングできます。目的、所有者、開発環境などの基準でリソースを分類するtagを作成でき、運用とコスト管理の効率化に役立ちます。

## tagの構成要素

各tagは2つの部分で構成されます:

- **Key**: tagキーの表示名。例: `Env`、`Project`。
- **Value**: キーに関連する具体的な値。例: `Production`、`FPT`。このフィールドは空欄にできます。
