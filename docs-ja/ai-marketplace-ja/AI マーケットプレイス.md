# Ai マーケットプレイス

## 1.1 紹介
Model-as-a-Service は、API エンドポイントとして LLM モデルを提供するプラットフォームであり、企業はモデルを開発、トレーニング、または管理することなく、事前にトレーニングされた LLM モデルをアプリケーションに統合することができます。
## 1.2 主な利点
このシステムには、次の機能があります。
  * API を介して既存のアプリケーションと**簡単に統合できます。**
  * **コスト効率** の従量課金制モデルでは、使用した分だけ支払うことができます。さらに、技術チームは、ハードウェアまたはソフトウェアのリソースのプロビジョニング、デプロイ、管理から解放されます。
  * 需要に応じた**自動スケーリング。**

このサービスはサーバーレスプラットフォームで動作し、高いパフォーマンスと99.9%の可用性を保証します。
## 1.3 主な機能
-**事前学習済みモデル：** 大規模言語モデル (LLM) と視覚言語モデル (VLM) が含まれます。
  * **API による簡単な統合：** 企業はゼロから構築することなく、アプリケーションにすばやく接続できます。
  * **需要に応じた自動スケーリング：** システムは、変化するワークロードに合わせて柔軟にスケーリングできます。
  * **継続的な更新：** パフォーマンスの向上と新しい要件へ適応します。
  * **モデルのカスタマイズ：** 特定のビジネスニーズを満たすために、FPT AI スタジオで直接微調整をサポートします。

Model-as-a-Service は、インフラストラクチャや社内の AI 専門知識に多額の投資をせずに、AI の力を活用したいと考えている企業にとって最適な選択肢です。
