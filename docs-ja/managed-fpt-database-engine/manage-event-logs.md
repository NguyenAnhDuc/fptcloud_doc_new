# イベントログの管理

1. **Database Engine** を選択し、**ID Cluster** を選択します。**Database Detail** の **Event Logs** タブを選択します。

2. イベントログの一覧が表示されます。データベースで発生したイベントの詳細情報（タイムスタンプ、イベントタイプ、説明など）を確認できます。

3. フィルターを使用して特定の期間や種類のイベントに絞り込むことができます。
