# Configuration の管理

1. **Database Engine** を選択し、**ID Cluster** を選択します。**Database Detail** の **Configuration** タブを選択します。

2. 変更したいパラメーターの種類（**Dynamic params** または **Static params**）を選択します。

3. 希望のパラメーターを変更します。

   **ケース 1: Dynamic params の変更**

   **Dynamic params** を選択し、変更したいパラメーターの Action 列にある編集アイコン（ペン型）をクリックします。

   - **Value** 列に変更したい情報を入力し、チェックマークをクリックして保存します。変更しない場合は下部の円形アイコンをクリックして戻ります（情報はそのまま保持されます）。

   **ケース 2: Static params の変更**

   **Static params** を選択し、変更したいパラメーターの Action 列にある編集アイコン（ペン型）をクリックします。

   - **Value** 列に変更したい情報を入力し、チェックマークをクリックして保存します。変更しない場合は下部の円形アイコンをクリックして戻ります（情報はそのまま保持されます）。
