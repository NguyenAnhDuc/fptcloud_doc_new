# バックアップジョブの更新と削除

Full バックアップジョブまたは Diff バックアップジョブを作成した後、バックアップジョブのスケジュールを更新または削除できます。

1. **Full backup job** または **Diff backup job** タブで **Update job** を選択します。
   - ジョブスケジュールの変更情報を更新し、**Update** をクリックします。

2. **Full backup job** または **Diff backup job** タブで **Delete job** を選択します。
   - **Delete Backup job Cluster** ポップアップが表示されます。ジョブの削除を確認するために Cluster ID を再入力し、**Delete** をクリックします。
