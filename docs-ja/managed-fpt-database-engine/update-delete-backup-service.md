# バックアップサービスの更新と削除

バックアップサービスを更新または削除する手順です。

:::note
バックアップサービスを削除すると、2 つのバックアップジョブは即座に停止され、再度有効にするまで実行されません。
:::

**ケース 1: バックアップサービスの更新**

**Action Panel** タブで **Update Backup** を選択します。

- **Backup Capacity** ポップアップが表示されます。変更したい Capacity を入力し、**Save** をクリックします。

**ケース 2: バックアップサービスの削除**

**Delete Backup** を選択します。

- **Delete Backup Cluster** ポップアップが表示されます。削除を確認するために Cluster ID を再入力し、**Delete** をクリックします。
