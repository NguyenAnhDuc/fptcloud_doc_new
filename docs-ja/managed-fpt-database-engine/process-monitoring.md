# プロセスモニタリングの管理

1. **Database Engine** を選択し、**ID Cluster** を選択します。**Database Detail** の **Monitoring** タブを選択します。

2. ドロップダウンリストから確認したいアクションのプロセスを選択します。

3. **View** を選択します。選択したプロセスのモニタリング情報が表示されます。
