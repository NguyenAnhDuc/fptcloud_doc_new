# Search and filter incidents

You can search or filter incidents by name, declaration time, severity, status, and label.

## Search by name

1. On the incident list screen, enter a keyword in the **Search by title** bar.

   [![Incident search bar](/img/migrated/Screenshot_15-1-9ecf7363.png)](/img/migrated/Screenshot_15-1-9ecf7363.png)

2. Press **Enter**.

The system returns a list of incidents whose titles contain the search keyword.

## Filter by time range

1. On the incident list screen, click the time filter field (default is **Last 30 days**).

   [![Time range filter](/img/migrated/Screenshot_16-1-7d52cb46.png)](/img/migrated/Screenshot_16-1-7d52cb46.png)

2. Select the desired time range, then click **Apply time range**.

   [![Select time range](/img/migrated/Screenshot_17-1-bdac4b1c.png)](/img/migrated/Screenshot_17-1-bdac4b1c.png)

The system returns a list of incidents created within that time range.

## Filter by severity, status, or label

Click **Edit Filters** and select the desired criteria.

[![Filter by severity, status, label](/img/migrated/Screenshot_18-1-0d12e8d9.png)](/img/migrated/Screenshot_18-1-0d12e8d9.png)

To clear all filters, click **Clear all filters**. The system will display all incidents.

[![Clear all filters button](/img/migrated/Screenshot_19-1-96822bc5.png)](/img/migrated/Screenshot_19-1-96822bc5.png)
