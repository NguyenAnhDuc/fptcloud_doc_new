# Enable or disable an alert

The enable/disable feature lets you temporarily pause an alert without deleting it. This is useful when you want to stop receiving notifications from a specific alert for a period of time.

In the **Alert Management** table, select the alert you want to enable or disable. Click the **switch** button [![Userguide FPT Alert 2022 5](/img/migrated/Userguide-FPT-Alert-2022-5-b7cf3f24.png)](/img/migrated/Userguide-FPT-Alert-2022-5-b7cf3f24.png) to toggle the state.

[![file](/img/migrated/image-1725532697828-cdc78b79.png)](/img/migrated/image-1725532697828-cdc78b79.png)

The alert status is shown in the **Status** field of the Alert Management table:

- **Enabled:** The system continuously evaluates and sends notifications when usage reaches the configured threshold.
- **Disabled:** The system stops evaluating and will not send automatic notifications.
