# Dashboard templates

Dashboard templates let you deploy quickly using pre-built dashboards provided by FPT Cloud Monitoring.

## Install a dashboard template

**Step 1:** [Access the dashboard](truy-cap-vao-dashboard.md).

**Step 2:** In the left menu, click **Apps** > **FMON Template**.

[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)

**Step 3:** Click **Dashboard Template**.

**Step 4:** Select a dashboard and click **Integrate**.

[![Alt text](/img/migrated/Picture14-2-d9a00088.png)](/img/migrated/Picture14-2-d9a00088.png)

## Remove a dashboard template

**Step 1:** [Access the dashboard](truy-cap-vao-dashboard.md).

**Step 2:** In the left menu, click **Apps** > **FMON Template**.

[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)

**Step 3:** Click **Dashboard Template**.

**Step 4:** Select an installed dashboard and click **Remove**.

[![Alt text](/img/migrated/Picture15-2-7940295e.png)](/img/migrated/Picture15-2-7940295e.png)
