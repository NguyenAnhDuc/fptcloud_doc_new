User Guide Tab

Provides a user guide for using, configuring, and integrating services.

**Step 1**: Log in to the FPT Cloud portal at **<https://console.fptcloud.com>**

**Step 2**: In the FPT Cloud Portal menu, click **Monitoring** to display the list of workspaces.

**Step 3**: Click the workspace name to view details. The detailed information screen for that workspace is displayed.

**Step 4**: Click the **"User guide"** tab.
