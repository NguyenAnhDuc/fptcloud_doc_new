# Integrate alert rule template

**Step 1:** In the Grafana interface, open the sidebar menu.

[![](/img/migrated/Picture3-a5fead32.png)](/img/migrated/Picture3-a5fead32.png)

**Step 2:** Open **Apps** and select **Fmon – Templates**.

[![](/img/migrated/Screenshot_7-4e0e3943.png)](/img/migrated/Screenshot_7-4e0e3943.png)

**Step 3:** In the Fmon – Templates interface, select the **Alert rule template** tab and choose the **Topic** for the service you want to add an alert rule for.

[![](/img/migrated/Screenshot_8-ca3c0b94.png)](/img/migrated/Screenshot_8-ca3c0b94.png)

**Step 4:** Click **Integrate** for each desired alert rule.

[![](/img/migrated/Screenshot_9-cc567b1a.png)](/img/migrated/Screenshot_9-cc567b1a.png)

**Step 5:** When the alert rule status changes to **Integrated**, the alert rule has been added to the Alert Rules section.

[![](/img/migrated/Screenshot_10-3b3797a4.png)](/img/migrated/Screenshot_10-3b3797a4.png)

[![](/img/migrated/Screenshot_11-17e20818.png)](/img/migrated/Screenshot_11-17e20818.png)
