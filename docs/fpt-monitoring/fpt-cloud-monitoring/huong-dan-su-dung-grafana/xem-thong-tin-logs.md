# View log data

**Step 1:** On the menu, select **Explore**.

[![](/img/migrated/Screenshot_22-f2b7b75d.png)](/img/migrated/Screenshot_22-f2b7b75d.png)

**Step 2:** Select the datasource: **loki**.

- Select the desired label filters.

[![](/img/migrated/Screenshot_27-3b3a74e9.png)](/img/migrated/Screenshot_27-3b3a74e9.png)

- You can add more labels by clicking the icon [![](/img/migrated/Screenshot_24-7ed16aa5.png)](/img/migrated/Screenshot_24-7ed16aa5.png).
- To view logs over a specific time range, select a time filter.

[![](/img/migrated/Screenshot_25-639736f9.png)](/img/migrated/Screenshot_25-639736f9.png)

**Step 3:** Click **Run query** and the system returns the corresponding logs.

[![](/img/migrated/Screenshot_28-4d1b494f.png)](/img/migrated/Screenshot_28-4d1b494f.png)
