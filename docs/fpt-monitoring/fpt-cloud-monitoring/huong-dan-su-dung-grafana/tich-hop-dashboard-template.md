# Integrate dashboard template

You need to integrate the appropriate dashboard templates for each service to monitor your metrics.

**Step 1:** Open the menu on the left side of the screen.

[![](/img/migrated/Picture3-a5fead32.png)](/img/migrated/Picture3-a5fead32.png)

**Step 2:** Select **App** > **Fmon – Templates** > **Dashboard Template** tab.

[![](/img/migrated/Screenshot_2-b3e58395.png)](/img/migrated/Screenshot_2-b3e58395.png)

The screen displays all available dashboard templates.

[![](/img/migrated/Screenshot_3-0152f813.png)](/img/migrated/Screenshot_3-0152f813.png)

**Step 3:** Select the relevant **Topic** (for example: Kubernetes).

The system displays the dashboards available for the selected topic.

[![](/img/migrated/Screenshot_4-9a1e9a31.png)](/img/migrated/Screenshot_4-9a1e9a31.png)

**Step 4:** Click **Integrate** to integrate the dashboard.

[![](/img/migrated/Screenshot_18-f0ee84d1.png)](/img/migrated/Screenshot_18-f0ee84d1.png)

After successful integration, the dashboard status changes to **Integrated**.

[![](/img/migrated/Screenshot_19-0864e4df.png)](/img/migrated/Screenshot_19-0864e4df.png)

If an error occurs during integration, please contact the support team.

:::note
The number of dashboards you can integrate depends on your current service package. For example, the V3_P1 package allows a maximum of 2 dashboards. If the limit is exceeded, the system displays a quota warning.
:::

**Step 5:** After successful integration, refresh Grafana > open the menu > select **Dashboards**.

The system displays the integrated dashboards. You can open them to monitor the corresponding metrics.

[![](/img/migrated/Screenshot_6-624450fb.png)](/img/migrated/Screenshot_6-624450fb.png)
