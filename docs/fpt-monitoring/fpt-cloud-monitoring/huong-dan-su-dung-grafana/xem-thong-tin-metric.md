# View metric data

You can monitor cluster performance, resource usage, and status metrics directly on the Grafana Dashboard.

**Step 1:** On the menu, select **Explore**.

[![](/img/migrated/Screenshot_22-f2b7b75d.png)](/img/migrated/Screenshot_22-f2b7b75d.png)

**Step 2:** Select the datasource: **mimir**.

- Select the metric type.
- Select the desired label filters.

[![](/img/migrated/Screenshot_23-b0dab0df.png)](/img/migrated/Screenshot_23-b0dab0df.png)

- You can add more labels by clicking the icon [![](/img/migrated/Screenshot_24-7ed16aa5.png)](/img/migrated/Screenshot_24-7ed16aa5.png).
- To view metrics over a specific time range, select a time filter.

[![](/img/migrated/Screenshot_25-639736f9.png)](/img/migrated/Screenshot_25-639736f9.png)

**Step 3:** Click **Run query** and the system returns the corresponding results.

[![](/img/migrated/Screenshot_26-0072eac5.png)](/img/migrated/Screenshot_26-0072eac5.png)
