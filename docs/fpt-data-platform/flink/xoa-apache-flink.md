# Delete Apache Flink

To delete Flink, follow these steps:

**Step 1:** In the menu bar, select **Data Platform** > **Workspace Management** > select the **Workspace name**

**Step 2:** In the **My Service** section, select **Flink** > click **Action** > select **Delete**

![Alt text](images/xoa-apache-flink/img-001.png)

**Step 3.** The **Delete Application** dialog box appears. Enter **delete** > click **confirm** to complete the deletion of the Application from the workspace.

![Alt text](images/xoa-apache-flink/img-002.png)
