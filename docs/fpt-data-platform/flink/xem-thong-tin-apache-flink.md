# View Apache Flink information

To view **Flink** information, follow these steps:

**Step 1:** In the menu bar, select **Data Platform** > **Workspace Management** > select the **Workspace name**

**Step 2:** In the **My Service** section, select **Flink**

**Essential Information** tab

Displays the basic information of Flink: **Entry Information, Instance Information, Job manager, Task manager**

Access **Apache Flink** using the displayed **URL**, **Username**, and **Password**.

![Alt text](images/xem-thong-tin-apache-flink/img-001.png)

**Advanced**

Displays the **Checkpoint/Savepoint storage**, **Job Storage**, and **Task slot** information.
