# Manage Policies

Displays the **Policies** information of the **Query Engine**. If the Authentication type is **Integration Ranger**, the **Query Engine** does not display the **Policies** tab — all access control configuration is performed through **Ranger-Admin**.

![Image](images/quan-ly-Policies/img-001.png)

Users can edit **Policies** information by clicking the edit icon.

For more information about access control policies, see [**here**](https://trino.io/docs/current/security/file-system-access-control.html).
