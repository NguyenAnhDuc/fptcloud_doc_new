# Delete Query Engine

To delete a Query Engine, follow the steps below:

**Step 1:** In the menu bar, select **Data Platform** > **Workspace Management** > **Workspace name**

**Step 2:** In the applications section, select **Query Engine** > click the **Action** button in the upper-right corner of the screen and select **Delete**

![Image](images/xoa-query-engine/img-001.png)

**Step 3.** The **Delete application** dialog appears > enter **delete** > click **Confirm** to complete the deletion of the application.

![Image](images/xoa-query-engine/img-002.png)
