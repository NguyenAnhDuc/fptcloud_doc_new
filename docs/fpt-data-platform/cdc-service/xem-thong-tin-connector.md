# View details connector

To view connector information, follow these steps:

  * **Step 1:** From the menu bar, select **Data Platform** > **Workspace Management** > **Workspace name**

  * **Step 2:** Under **My services**, select **CDC service**

  * **Step 3:** On the **CDC service** detail screen > Select the **Connectors** tab > Select the connector to view
