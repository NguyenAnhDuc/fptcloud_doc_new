# View JupyterHub Details

**Step 1:** In the menu bar, select **Data Platform** > **Workspace Management** > **Workspace name**

**Step 2:** In the **My service** section, select JupyterHub

 * **Tab Essential information**

Displays detailed information and configuration details of **JupyterHub**

   * **Guide to accessing JupyterHub and creating an admin account**

     1. Open a browser and navigate to the provided JupyterHub URL.

     2. On the displayed page, click the **Create User** button.

     3. Enter the login name (Username = admin) and password (Password) as required.

     4. Confirm the information and complete the account creation.

     5. Use the newly created account to log in to JupyterHub and start using it.
