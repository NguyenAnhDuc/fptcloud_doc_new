# Truy Cap Tinh Nang

* Trên portal FPT Cloud Console, ando workspace FPT Cloud Monitoring cần cấu hình, chọn tab **“IP Access Control”**. 
  * Nếu feature chưa bật, you will thấy nút “Enable IP Access Control". Sau that nhấn nút “Enable IP Access Control” to bật feature.

[![Enable IP Access Control](/img/migrated/Screenshot_2-37902d64.png)](/img/migrated/Screenshot_2-37902d64.png)
  * Nếu feature has been bật, you will thấy 4 bảng whitelisted IP corresponding with 4 thành phần hệ thống FPT Cloud Monitoring, and nút “Disable IP Access Control". Nhấn nút “Disable IP Access Control” to tắt feature.

[![Màn hình chính](/img/migrated/Screenshot_3-2a6306f9.png)](/img/migrated/Screenshot_3-2a6306f9.png)
**Note:**
  * **Khi feature IP Access Control tắt, mọi IP đều can truy cập ando the thành phần hệ thống of workspace FPT Cloud Monitoring.**
  * **Ngay when bật feature, mặc định hệ thống will chặn hết mọi truy cập from bên ngoài FPT Cloud (nghĩa là the agent monitoring trên FPT Cloud vẫn hoạt động bình thường)**. **Các IP from bên ngoài FPT Cloud cần is whitelist to can truy cập hệ thống FPT Cloud Monitoring.**
