# Gioi Thieu Chung

Tính năng **IP Access Control** là feature security of FPT Cloud Monitoring, for phép user setup and quản lý list the địa chỉ IP bên ngoài system FPT Cloud is phép truy cập ando fromng system IP nội bộ FPT cloud monitoring of mình. 
Tính năng this chủ động chặn the truy cập from bên ngoài FPT Cloud không nằm in list for phép, nhằm hỗ trợ user nâng cao security truy cập, phòng chống rủi ro truy cập trái phép and đáp ứng the tiêu chuẩn security.
