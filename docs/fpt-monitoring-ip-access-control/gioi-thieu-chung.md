# Gioi Thieu Chung

Tính năng **IP Access Control** là feature security of FPT Cloud Monitoring, for phép user thiết lập and quản lý danh sách the địa chỉ IP bên ngoài hệ thống FPT Cloud is phép truy cập ando fromng hệ thống IP nội bộ FPT cloud monitoring of mình. 
Tính năng this chủ động chặn the truy cập from bên ngoài FPT Cloud không nằm in danh sách for phép, nhằm hỗ trợ user nâng cao security truy cập, phòng chống rủi ro truy cập trái phép and đáp ứng the tiêu chuẩn security.
