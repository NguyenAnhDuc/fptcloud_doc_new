# View the whitelisted IP list

Each endpoint (Dashboard, Metric, Log, Trace) has its own IP table.

- Click the icon on the left side of each table to expand or collapse the IP list for that endpoint.
- Click the icons next to the endpoint to open the Grafana dashboard or copy the endpoint URL to the clipboard.
- The IP/CIDR list supports pagination, search by IP/CIDR, and filtering by whitelist status.
- Click the icon next to an IP/CIDR to copy it to the clipboard.
