# Create firewall rule

**Step 1:** In the **Menu**, select **Security Policies** (**Ctrl + 2**) > select **Access Control** > select **Policy**.

[![Userguide Checkpoint FPT NGFW 2022 51](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-51-10-d67afe3a.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-51-10-d67afe3a.png)

**Step 2:** Click **Add rule above/below** or right-click an existing rule and select **New rule Above/Below** to add a new rule above or below the current one.

[![Userguide Checkpoint FPT NGFW 2022 52](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-52-10-dc654ab1.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-52-10-dc654ab1.png)

- **Source** (source IP): [![Userguide Checkpoint FPT NGFW 2022 54](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-54-10-7de8a148.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-54-10-7de8a148.png)
