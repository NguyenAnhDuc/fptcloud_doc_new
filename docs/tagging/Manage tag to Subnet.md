# Manage tags for Subnet

Assigning tags to Subnets helps you categorize subnets by project or environment — making network infrastructure management easier.

## Assign tags when creating a Subnet

1. Select **Subnet** from the menu, then click **Create subnet**.

   [![Subnet screen with Create subnet button](/img/migrated/image-1721034781650-31f7f246.png)](/img/migrated/image-1721034781650-31f7f246.png)

2. On the **Create subnet** page, select tags in the **Add tag (optional)** field.

   [![Create subnet page with Add tag field](/img/migrated/image-1721034795338-032c129b.png)](/img/migrated/image-1721034795338-032c129b.png)

3. Fill in the remaining information and click **Create subnet**.

The assigned tags are displayed in the **Tags** field on the **Subnet** screen.

[![Subnet list with assigned tags](/img/migrated/image-1721034807246-4eb48b05.png)](/img/migrated/image-1721034807246-4eb48b05.png)

## Manage tags for an existing Subnet

1. Select **Subnet** from the menu, choose a subnet, then click **Actions** → **Manage tags**.

   [![Actions menu with Manage tags option for Subnet](/img/migrated/image-1721034830614-0a44d416.png)](/img/migrated/image-1721034830614-0a44d416.png)

2. A dialog appears. Select tags to assign to the Subnet.

   [![Manage tags dialog for Subnet](/img/migrated/image-1721034842841-94a4bdb4.png)](/img/migrated/image-1721034842841-94a4bdb4.png)

3. To remove a tag, click the **x** next to the tag name. Click the **x** at the end of the list box to remove all tags at once. Click **Save** to confirm.

   [![Manage tags dialog with remove option](/img/migrated/image-1721034855299-a138f9c7.png)](/img/migrated/image-1721034855299-a138f9c7.png)

:::warning
When a Subnet is deleted, the system automatically removes all associated tags and updates the tag list on the Tagging screen.
:::
