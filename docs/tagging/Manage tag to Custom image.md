# Manage Tag To Custom Image

## **1. Assigning Tags While Uploading an Image**
**Step 1** : In the menu, select **Custom Image** , then click **Upload image**.
[![file](/img/migrated/image-1721032879356-5bd46d9a.png)](/img/migrated/image-1721032879356-5bd46d9a.png)
**Step 2** : In the **Upload image** dialog box, users can select tags in the **Add tag (optional)** field to assign tags to the image.
[![file](/img/migrated/image-1721032892734-ae8a432b.png)](/img/migrated/image-1721032892734-ae8a432b.png)
## **2. Managing Tags for a Custom Image**
**Step 1** : In the menu, select **Custom Image** , then click **Actions** and choose **Manage tags**.
[![file](/img/migrated/image-1721032908795-71940b90.png)](/img/migrated/image-1721032908795-71940b90.png)
**Step 2** : A dialog box will appear, allowing users to select tags to assign to the image.
[![file](/img/migrated/image-1721032920130-4755a194.png)](/img/migrated/image-1721032920130-4755a194.png)
**Step 3** : Removing Tags from an Image
To remove a tag, click the “x” next to the tag name. Users can also click the “x” at the end of the list box to remove all tags at once. Then, click Save to confirm the changes.
**Note: When an image is deleted, the system will automatically remove all associated tags and update the tag list on the Tagging screen.**
[![file](/img/migrated/image-1721032931446-41441a71.png)](/img/migrated/image-1721032931446-41441a71.png)
