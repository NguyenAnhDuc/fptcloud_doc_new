# Manage tags for Custom Image

Assigning tags to Custom Images helps you categorize images by project or environment — making search and reuse easier.

## Assign tags when uploading an image

1. Select **Custom Image** from the menu, then click **Upload image**.

   [![Custom Image screen with Upload image button](/img/migrated/image-1721032879356-5bd46d9a.png)](/img/migrated/image-1721032879356-5bd46d9a.png)

2. In the **Upload image** dialog, select tags in the **Add tag (optional)** field.

   [![Upload image dialog with Add tag field](/img/migrated/image-1721032892734-ae8a432b.png)](/img/migrated/image-1721032892734-ae8a432b.png)

## Manage tags for an existing Custom Image

1. Select **Custom Image** from the menu, then click **Actions** → **Manage tag**.

   [![Actions menu with Manage tag option for Custom Image](/img/migrated/image-1721032908795-71940b90.png)](/img/migrated/image-1721032908795-71940b90.png)

2. A dialog appears. Select tags to assign to the image.

   [![Manage tags dialog for Custom Image](/img/migrated/image-1721032920130-4755a194.png)](/img/migrated/image-1721032920130-4755a194.png)

3. To remove a tag, click the **x** next to the tag name. Click the **x** at the end of the list box to remove all tags at once. Click **Save** to confirm.

   [![Manage tags dialog with remove option](/img/migrated/image-1721032931446-41441a71.png)](/img/migrated/image-1721032931446-41441a71.png)

:::warning
When an image is deleted, the system automatically removes all associated tags and updates the tag list on the Tagging screen.
:::
