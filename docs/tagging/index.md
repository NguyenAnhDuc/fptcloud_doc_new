# Tagging overview

Tags help you manage, identify, sort, search, and filter resources on FPT Cloud. You can create tags to categorize resources by purpose, owner, development environment, or other criteria — making operations and cost tracking more efficient.

## Tag components

Each tag consists of two parts:

- **Key**: The display name of the tag key. Example: `Env`, `Project`.
- **Value**: The specific value associated with that key. Example: `Production`, `FPT`. You can leave this field blank.
