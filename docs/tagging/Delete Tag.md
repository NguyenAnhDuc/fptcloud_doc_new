# Delete a tag

When a tag is no longer needed, you can delete it to keep your tag list organized. Deleting a tag will remove it from all associated resources.

:::danger
Deleting a tag removes all resource associations. This action is irreversible.
:::

## Delete a tag

1. Select **Tagging** from the menu, then at the tag you want to delete, select **Actions** → **Delete**.

   [![Actions menu with Delete option](/img/migrated/image-1721013188045-86d98c1e.png)](/img/migrated/image-1721013188045-86d98c1e.png)

2. A warning dialog displays the tag name and the number of associated resources. Click **Delete** to confirm.

   [![Delete confirmation dialog](/img/migrated/image-1721013198178-ba064154.png)](/img/migrated/image-1721013198178-ba064154.png)

The system removes all resource associations and deletes the tag from the list.
