# Delete Tag

**Note: This will simultaneously remove the resource from the tags associated with it. Deleting this tag is irreversible**
The user does the following: 
**Step 1** : In the menu select **Tagging** , then select **Actions** > Select **Delete**
[![file](/img/migrated/image-1721013188045-86d98c1e.png)](/img/migrated/image-1721013188045-86d98c1e.png)
**Step 2** : A warning dialog, displaying the **tag name and the number of resources attached to the tag** , will appear to confirm the removal action. The user confirms by tapping the **Delete button**. The system will remove the resources attached to the tag and delete the tag.
[![file](/img/migrated/image-1721013198178-ba064154.png)](/img/migrated/image-1721013198178-ba064154.png)
