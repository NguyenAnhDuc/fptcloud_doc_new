# Manage tags for Security Group

Assigning tags to Security Groups helps you categorize security policies by project or environment — making access management easier.

## Assign tags when creating a Security Group

1. Select **Security group** from the menu, then click **Create security group**.

   [![Security group screen with Create security group button](/img/migrated/image-1721034868364-9582cb2e.png)](/img/migrated/image-1721034868364-9582cb2e.png)

2. On the **Create security group** page, select tags in the **Add tag (optional)** field.

   [![Create security group page with Add tag field](/img/migrated/image-1721034882491-a8d79b00.png)](/img/migrated/image-1721034882491-a8d79b00.png)

3. Fill in the remaining information and click **Create security group**.

The assigned tags are displayed in the **Tags** field on the **Security group** screen.

[![Security Group list with assigned tags](/img/migrated/image-1721034894848-efbc40e0.png)](/img/migrated/image-1721034894848-efbc40e0.png)

## Manage tags for an existing Security Group

1. Select **Security group** from the menu, choose a security group, then click **Actions** → **Manage tags**.

   [![Actions menu with Manage tags option for Security Group](/img/migrated/image-1721034908522-a8960341.png)](/img/migrated/image-1721034908522-a8960341.png)

2. A dialog appears. Select tags to assign to the Security Group.

   [![Manage tags dialog for Security Group](/img/migrated/image-1721034922176-ded21a16.png)](/img/migrated/image-1721034922176-ded21a16.png)

3. To remove a tag, click the **x** next to the tag name. Click the **x** at the end of the list box to remove all tags at once. Click **Save** to confirm.

   [![Manage tags dialog with remove option](/img/migrated/image-1721034933906-b25bc9ee.png)](/img/migrated/image-1721034933906-b25bc9ee.png)

:::warning
When a Security Group is deleted, the system automatically removes all associated tags and updates the tag list on the Tagging screen.
:::
