# Initializing Tags 

The user does the following: 
**Step 1** : In the Tagging menu, select **Create tag**
[![file](/img/migrated/image-1721012726449-a6958240.png)](/img/migrated/image-1721012726449-a6958240.png)
**Step 2** : Enter the following information: 
  * **Key** : Enter a name that displays the user's key tag. Examples: Env, Project, ... 
  * **Value** : Enter the value of the corresponding tag. The specific value associated with that key. For example: Production or FPT. Users can leave this field blank. 
  * **Color** : Select the color for the tag. Currently, the system only supports three colors including red, yellow and green, the default tag is white. 
  * **For users with tag permissions on the Tenant** : The generated tag will be applied across the entire VPC. To create tags for certain VPCs, the user selects "Enable customize VPC" and selects the VPC from the list of Tenant VPCs. 
  * **For users with tagging permissions on VPC** : The user selects the VPC in the list of delegated VPCs. The initialized tag will be applied on the selected VPCs. 

[![file](/img/migrated/image-1721012710366-7b7920f7.png)](/img/migrated/image-1721012710366-7b7920f7.png)
**Step 3** : Click Create to create a tag. Once the tag is initialized, it will be displayed on the **Tagging page**
[![file](/img/migrated/image-1721012757339-fb91dbbd.png)](/img/migrated/image-1721012757339-fb91dbbd.png)
