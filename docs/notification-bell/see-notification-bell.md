# View notifications

Clicking the notification bell opens a dropdown list of recent actions. Unread notifications are marked with a blue dot; read notifications have no indicator.

[![Notification dropdown list](/img/migrated/image-1710299730130-8b36f355.png)](/img/migrated/image-1710299730130-8b36f355.png)

Each notification contains activity details, the notification timestamp, and a direct link to the related content. A link at the bottom of the dropdown provides quick access to all your notifications.

[![Notification detail view](/img/migrated/image-1710299747410-1e381598.png)](/img/migrated/image-1710299747410-1e381598.png)
[![View all notifications link](/img/migrated/image-1710299754850-658fb832.png)](/img/migrated/image-1710299754850-658fb832.png)

Clicking anywhere inside a notification takes you to the detailed information page for that action.

[![Detailed information page](/img/migrated/image-1710299771152-56464fc7.png)](/img/migrated/image-1710299771152-56464fc7.png)

:::note
- Clicking **Mark all read** changes all unread notifications to read status.
- If you perform multiple actions at the same time, notifications appear in the order those actions were executed.
:::
