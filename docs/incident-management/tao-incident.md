# Create Incident

Bạn có thể tạo incident theo hai cách:

1. Trực tiếp từ tính năng **Declare incident** trong giao diện hệ thống.
2. Qua **Incoming webhook** bằng cách gửi POST request từ hệ thống bên ngoài.
