# Component Management

**1, Create component**
Định nghĩa: Component là các thành phần bên trong luồng nghiệp vụ hoặc một business.
Example: Bên trong tag Long Châu có các component như: LoadBalancer, Kubernetes, DB, ...
Để tạo component, bạn thực hiện theo hướng dẫn sau:
**Step 1** : Truy cập thành công vào dịch vụ **Relation management**
**Step 2** : Trên menu của dịch vụ, click **Component management** >> Button **Create component**
[![Alt text](/img/migrated/Picture45-51564469.png)](/img/migrated/Picture45-51564469.png)
**Step 3** : Hiển thị form tạo, người dùng điền các thông tin bao gồm:
  * Name (required): tên component
  * Icon (optional)
  * Query type: mặc định là Custom
  * Value: người dùng có thể add value trong component đó.

[![Alt text](/img/migrated/Picture46-14525c51.png)](/img/migrated/Picture46-14525c51.png)
Có 2 cách nhập value:
• Builder
Người dùng nhập value vào text box >> Click button Add.
Trong trường hợp bạn muốn add nhiều value cùng lúc, bạn nhập value vào text box được cách nhau bởi dấu phẩy (,) >> Click button Add
Example: 
[![Alt text](/img/migrated/Picture47-99f02769.png)](/img/migrated/Picture47-99f02769.png)
• Code
Nếu người dùng chọn Code, để add value bạn nhập value cách nhau bởi dấu phẩy (,)
Example: [![Alt text](/img/migrated/Picture48-4adc6eb8.png)](/img/migrated/Picture48-4adc6eb8.png)
**Step 4** : Sau khi điền hết các thông tin, click **Create** để tạo
Sau khi tạo thành công, component được hiển thị trong màn danh sách.
**2, Update component**
Step 1: Menu chọn Component management >> Tại màn danh sách, ở cột Action, click button Update
[![Alt text](/img/migrated/Picture17-7b5c3a89.png)](/img/migrated/Picture17-7b5c3a89.png)
**Step 2** : Thực hiện chỉnh sửa thông tin component. Các thông tin cho phép chỉnh sửa bao gồm:
  * Name
  * Icon
  * Query type
  * Value: người dùng có thể thêm hoặc xóa value trong component

[![Alt text](/img/migrated/Picture18-28271c7d.png)](/img/migrated/Picture18-28271c7d.png)
**Step 3** : Click button **Update** để hoàn tất.
Để hủy bỏ thao tác, bạn chọn **Cancel** để thoát khỏi màn chỉnh sửa.
**3, Delete component**
Để xóa component, người dùng thực hiện như sau:
**Step 1** : Menu chọn **Component management** >> Tại màn danh sách, ở cột **Action** , click button **Delete**
[![Alt text](/img/migrated/Picture19-c948da9f.png)](/img/migrated/Picture19-c948da9f.png)
**Step 2** : Hệ thống sẽ show pop up confirmation. Để xác nhận xóa tag, click **Delete** trong popup.
Để hủy việc xóa tag, chọn **Cancel.**
[![Alt text](/img/migrated/Picture49-eb07667d.png)](/img/migrated/Picture49-eb07667d.png)
