# Component Management

**1, Create component**

Definition: A component represents elements within a business flow or a business.

Example: Inside the Long Chau tag there are components such as: LoadBalancer, Kubernetes, DB, etc.

To create a component, follow these steps:

**Step 1**: Successfully access the **Relation management** service.

**Step 2**: In the service menu, click **Component management** >> Button **Create component**

[![Alt text](/img/migrated/Picture45-51564469.png)](/img/migrated/Picture45-51564469.png)

**Step 3**: The creation form is displayed. Fill in the following information:
  * Name (required): component name
  * Icon (optional)
  * Query type: default is Custom
  * Value: you can add values to the component.

[![Alt text](/img/migrated/Picture46-14525c51.png)](/img/migrated/Picture46-14525c51.png)

There are 2 ways to enter a value:

- **Builder**

Enter the value in the text box >> Click the **Add** button.

If you want to add multiple values at once, enter values in the text box separated by commas (,) >> Click the **Add** button.

Example:

[![Alt text](/img/migrated/Picture47-99f02769.png)](/img/migrated/Picture47-99f02769.png)

- **Code**

If you select Code, add values by entering them separated by commas (,).

Example: [![Alt text](/img/migrated/Picture48-4adc6eb8.png)](/img/migrated/Picture48-4adc6eb8.png)

**Step 4**: After filling in all information, click **Create** to create the component.

After successful creation, the component appears in the list screen.

**2, Update component**

Step 1: In the menu, select Component management >> In the list screen, under the Action column, click the **Update** button.

[![Alt text](/img/migrated/Picture17-7b5c3a89.png)](/img/migrated/Picture17-7b5c3a89.png)

**Step 2**: Edit the component information. The following fields can be modified:
  * Name
  * Icon
  * Query type
  * Value: you can add or remove values from the component.

[![Alt text](/img/migrated/Picture18-28271c7d.png)](/img/migrated/Picture18-28271c7d.png)

**Step 3**: Click the **Update** button to complete.

To cancel the operation, click **Cancel** to exit the edit screen.

**3, Delete component**

To delete a component, follow these steps:

**Step 1**: In the menu, select **Component management** >> In the list screen, under the **Action** column, click the **Delete** button.

[![Alt text](/img/migrated/Picture19-c948da9f.png)](/img/migrated/Picture19-c948da9f.png)

**Step 2**: The system will display a confirmation popup. To confirm deletion, click **Delete** in the popup.

To cancel the deletion, click **Cancel**.

[![Alt text](/img/migrated/Picture49-eb07667d.png)](/img/migrated/Picture49-eb07667d.png)
