# Tag Management

**1, Create tag**
Định nghĩa: Tag là a tên định danh for a luồng nghiệp vụ or a business.
Example: Trong FRT, có the line hệ thống như Long Châu, Vaccine,... Thì mỗi line that is gọi là a tag: tag Long Châu, tag Vaccine, ...
Để thực hiện tạo tag, user thao tác as follows:
**Step 1** : Truy cập successfully ando service **Relation management**
**Step 2** : Trên menu of service, click **Tag management** >> **Button Create tag**
[![Alt text](/img/migrated/Picture6-f6221c1b.png)](/img/migrated/Picture6-f6221c1b.png)
**Step 3** : Hiển thị form tạo tag, user điền the thông tin includes:
  * Name (required): tên tag
  * Description (optional): mô tả tag

[![Alt text](/img/migrated/Picture7-1fead4bd.png)](/img/migrated/Picture7-1fead4bd.png)
**Save ý** : Tên tag chứa tối đa 100 kí tự
**Step 4** : Sau when điền hết the thông tin, click **Create** to tạo
**2, Update tag**
Để chỉnh sửa tag, user thực hiện as follows:
**Step 1** : Menu chọn **Tag management** >> Tại màn danh sách, ở cột **Action** , click button **Update**
[![Alt text](/img/migrated/Picture8-ea440913.png)](/img/migrated/Picture8-ea440913.png)
**Step 2** : Thực hiện chỉnh sửa thông tin tag. Các thông tin for phép chỉnh sửa includes: **Description**. Trường Name không for phép 
chỉnh sửa.
[![Alt text](/img/migrated/Picture9-2b92aca1.png)](/img/migrated/Picture9-2b92aca1.png)
**Step 3** : Click button **Update** to hoàn tất.
Để hủy bỏ thao tác, you chọn **Cancel** to thoát khỏi màn chỉnh sửa.
**3, Delete tag**
Để xóa tag, user thực hiện as follows:
**Step 1** : Menu chọn **Tag management** >> Tại màn danh sách, ở cột **Action** , click button **Delete**
[![Alt text](/img/migrated/Picture11-68f55546.png)](/img/migrated/Picture11-68f55546.png)
**Step 2** : System will show pop up confirmation. Để xác nhận xóa tag, click **Delete** in pop up.
Để hủy việc xóa tag, chọn **Cancel**.
[![Alt text](/img/migrated/Picture10-c19c9196.png)](/img/migrated/Picture10-c19c9196.png)
