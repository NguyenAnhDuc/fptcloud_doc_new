# Dashboard

To create a dashboard, follow these steps:

**Step 1**: Go to Home >> Dashboard >> New dashboard

**Step 2**: Add a variable to the dashboard by clicking the Settings icon >> Select Variables >> Add variables

**Step 3**: Declare the tag in the dashboard.

[![Alt text](/img/migrated/Picture30-c72550b8.png)](/img/migrated/Picture30-c72550b8.png)

After creation, a declared variable will be displayed on the screen.

[![Alt text](/img/migrated/Picture31-b13fb689.png)](/img/migrated/Picture31-b13fb689.png)

**Step 4**: Create variables corresponding to the components inside the tag.

[![Alt text](/img/migrated/Picture33-152d5300.png)](/img/migrated/Picture33-152d5300.png)

Click **Apply** to save the variable.

The declared variables will be displayed on the screen.

[![Alt text](/img/migrated/Picture35-86ebccbf.png)](/img/migrated/Picture35-86ebccbf.png)

**Step 5**: Save the dashboard. Return to the dashboard page by selecting Home >> Dashboard >> Select the dashboard you created.

The dashboard will now display the variables you declared.

[![Alt text](/img/migrated/Picture36-abda8417.png)](/img/migrated/Picture36-abda8417.png)

You can add panels as desired to monitor your data.
