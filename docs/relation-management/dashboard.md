# Dashboard

Để tạo a dashboard, user thực hiện as follows:
**Step 1** : Vào Home >> Dashboard >> New dashboard
**Step 2** : Người dùng will add variable in dashboard bằng theh click icon Settings >> Chọn Variables >> Add variables
**Step 3** : Khai báo tag có in dashboard. 
[![Alt text](/img/migrated/Picture30-c72550b8.png)](/img/migrated/Picture30-c72550b8.png)
Sau when tạo xong will có a variable is khai báo and displayed ở màn hình.
[![Alt text](/img/migrated/Picture31-b13fb689.png)](/img/migrated/Picture31-b13fb689.png)
**Step 4** : Create the biến corresponding with the component bên in tag.
[![Alt text](/img/migrated/Picture33-152d5300.png)](/img/migrated/Picture33-152d5300.png)
Click Apply to lưu biến.
Các biến has been khai báo will is displayed ở màn hình.
[![Alt text](/img/migrated/Picture35-86ebccbf.png)](/img/migrated/Picture35-86ebccbf.png)
**Step 5** : Save dashboard. Quay lại trang dashboard bằng theh chọn Home >> Dashboard >> Chọn dashboard has been tạo
Trên dashboard lúc this will displayed the variables mà user has been khai báo.
[![Alt text](/img/migrated/Picture36-abda8417.png)](/img/migrated/Picture36-abda8417.png)
Bạn can add panel theo mong muốn to theo dõi.
