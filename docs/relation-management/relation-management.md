# 1, Create relation

**1, Create relation** The relation screen allows users to visualize the relationships between components within a tag.

You can drag and drop components into the board area to establish relationships.

Each component corresponds to one node.

Example:

[![Alt text](/img/migrated/Picture20-e7d6591d.png)](/img/migrated/Picture20-e7d6591d.png)

**2, Update node**

- **Add value to a node**

To add a value to a node, proceed as follows:

On that node, click the **Pencil** icon >> The **Update node** dialog is displayed. Tick the checkbox to add a value to the node.

**Multiple values can be added at once.**

[![Alt text](/img/migrated/Picture21-ecc9182e.png)](/img/migrated/Picture21-ecc9182e.png)
[![Alt text](/img/migrated/Picture22-1c0be09b.png)](/img/migrated/Picture22-1c0be09b.png)

- **Remove value from a node**

If you want to remove one or more items from a node, simply click the Pencil icon to open the Update node dialog and uncheck the selected items.

[![Alt text](/img/migrated/Picture23-80246e33.png)](/img/migrated/Picture23-80246e33.png)

- **Update description**

To add or edit a description for a node, proceed as follows:

Right-click on the node and select **Update Description**. A textbox is displayed for you to enter the description.

[![Alt text](/img/migrated/Picture24-2854752a.png)](/img/migrated/Picture24-2854752a.png)
[![Alt text](/img/migrated/Picture25-0ddf7c2a.png)](/img/migrated/Picture25-0ddf7c2a.png)

- **Disable multiple selection**

If the user selects Disable multiple selection, items are added to the node as a dropdown list, allowing only **1 item per node**.

[![Alt text](/img/migrated/Picture26-f9ef2589.png)](/img/migrated/Picture26-f9ef2589.png)
[![Alt text](/img/migrated/Picture27-0f3c25b0.png)](/img/migrated/Picture27-0f3c25b0.png)

To allow multiple items in a node, right-click and select **Enable multiple selection**.

[![Alt text](/img/migrated/Picture28-abb1fec1.png)](/img/migrated/Picture28-abb1fec1.png)

- **Delete node**

To delete a node, right-click on the node and select **Delete node**.

[![Alt text](/img/migrated/Picture29-a8665830.png)](/img/migrated/Picture29-a8665830.png)
