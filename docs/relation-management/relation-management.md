# 1, Create relation

**1, Create relation** Màn relation for phép user thể hiện mối quan hệ giữa the component in a tag.
Bạn can kéo thả the component ando khu vực bảng to setup mối quan hệ.
Mỗi component corresponding with a node.
Example:
[![Alt text](/img/migrated/Picture20-e7d6591d.png)](/img/migrated/Picture20-e7d6591d.png)
**2, Update node**
• Add value in node
Để add value in node, you thực hiện as follows:
Tại node that, click icon **Pencil** >> Hiển thị hộp thoại **Update node** , user tick ando checkbox to add value for node that. 
**Có thể add nhiều value cùng lúc.**
[![Alt text](/img/migrated/Picture21-ecc9182e.png)](/img/migrated/Picture21-ecc9182e.png)
[![Alt text](/img/migrated/Picture22-1c0be09b.png)](/img/migrated/Picture22-1c0be09b.png)
• **Remove value ra khỏi node**
Trong trường hợp you muốn xóa bỏ a or nhiều items ra khỏi node, you chỉ need to click ando icon Pencil to mở hộp thoại Update node and bỏ chọn items has been tick.
[![Alt text](/img/migrated/Picture23-80246e33.png)](/img/migrated/Picture23-80246e33.png)
• **Update description** Để thêm or chỉnh sửa mô tả for node, you thao tác as follows:
Click chuột must ando node that, chọn **Update Description**. Hiển thị textbox for user nhập description.
[![Alt text](/img/migrated/Picture24-2854752a.png)](/img/migrated/Picture24-2854752a.png)
[![Alt text](/img/migrated/Picture25-0ddf7c2a.png)](/img/migrated/Picture25-0ddf7c2a.png)
• **Disable multiple selection**
Nếu user chọn Disable multiple selection, thì add item ando node dưới dạng droplist, chỉ add is **1 item in 1 node.**
[![Alt text](/img/migrated/Picture26-f9ef2589.png)](/img/migrated/Picture26-f9ef2589.png)
[![Alt text](/img/migrated/Picture27-0f3c25b0.png)](/img/migrated/Picture27-0f3c25b0.png)
Để can add is nhiều item in 1 node, click chuột must and chọn **Enable multiple selection.**
[![Alt text](/img/migrated/Picture28-abb1fec1.png)](/img/migrated/Picture28-abb1fec1.png)
• **Delete node**
Để xóa node, click chuột must ando node that, chọn **Delete node**.
[![Alt text](/img/migrated/Picture29-a8665830.png)](/img/migrated/Picture29-a8665830.png)
