# Tao Tai Khoan Fpt Cloud Va Dang Nhap Vao Fpt Portal

To start using service **Relation management** you need to đăng nhập ando **FPT Portal.**
Hãy truy cập ando **<https://console.fptcloud.com>** and đăng nhập bằng account/password FPT Cloud has been is cấp. Chọn đúng **Tenant, Region, VPC** need to sử dụng service. Nếu chưa có account FPT Cloud, you hãy truy cập home page tại **[https://fptcloud.com](https://fptcloud.com/)**.
Sau that chọn chức năng **Sign Up** and nhập the information theo guide of system. Bạn will is bộ phận hỗ trợ liên hệ ngay sau that and xác nhận the information to create account.
Trong quá trình thao tác if gặp bất kỳ lỗi nào hãy liên hệ ngay for đội ngũ Support of chúng tôi to is hỗ trợ.
