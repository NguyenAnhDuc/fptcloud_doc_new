# Update agent configuration

To update an agent's configuration:

**Step 1:** In the FPT Portal menu > **Jenkins CI** > **Detail Cluster** > **Agent Managements** > click **Actions** for the agent to update.

[![](/img/migrated/Picture19-519a8553.png)](/img/migrated/Picture19-519a8553.png)

**Step 2:** Select **Edit** to modify the agent configuration.

[![](/img/migrated/Picture20-4d54db71.png)](/img/migrated/Picture20-4d54db71.png)

**Step 3:** Edit the configuration: select the resources to update.

[![](/img/migrated/Picture21-e8aaf98b.png)](/img/migrated/Picture21-e8aaf98b.png)

**Step 4:** Click **Update** to apply the changes.

[![](/img/migrated/Picture22-ec7e1242.png)](/img/migrated/Picture22-ec7e1242.png)

:::note
You can only upgrade to a higher resource tier, not downgrade.
:::
