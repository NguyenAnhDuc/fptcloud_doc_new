# Update thông tin cấu hình Agent

Để cập nhật thông tin cấu hình Agent user thực hiện as follows:
**Step 1.** Tại menu FPT Portal > **Jenkins CI** > **Detail Cluster** > **Agent Managements** > Click **Actions** đối with agent muốn cập nhật: [![](/img/migrated/Picture19-519a8553.png)](/img/migrated/Picture19-519a8553.png) **Step 2.** Select **Edit** to thực hiện chỉnh sửa cấu hình for Agent: [![](/img/migrated/Picture20-4d54db71.png)](/img/migrated/Picture20-4d54db71.png) **Step 3.** Thực hiện chỉnh sửa thông tin cấu hình: Lựa chọn tài nguyên muốn update [![](/img/migrated/Picture21-e8aaf98b.png)](/img/migrated/Picture21-e8aaf98b.png) **Step 4.** Click **Update** to thực hiện cập nhật thông tin cấu hình. [![](/img/migrated/Picture22-ec7e1242.png)](/img/migrated/Picture22-ec7e1242.png) **Save ý** : Chỉ is Update lên tài nguyên cao hơn
