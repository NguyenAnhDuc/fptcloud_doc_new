# Create mới a FPT Jenkins CI

Để tạo mới **Jenkins CI instance** , you can thực hiện the thao tác sau:
**Step 1.** Trên menu of **FPT Portal** > **DevOps** > **Jenkins CI** > Click **Create a Jenkins**
[![](/img/migrated/Picture1-38ff2a6f.png)](/img/migrated/Picture1-38ff2a6f.png)
**Step 2.** Nhấn chọn **Create a Jenkins** > Enter thông tin to khởi tạo a Jenkins CI instance:
[![](/img/migrated/Picture2-1-9c7a0368.png)](/img/migrated/Picture2-1-9c7a0368.png)
**Step 3.** Select **Create** , hệ thống will khởi tạo a cụm Jenkins Instance and notification tình trạng xử lý.
[![](/img/migrated/Picture3-1-47e2570d.png)](/img/migrated/Picture3-1-47e2570d.png)
Sau when tạo successfully a Jenkins instance, status of cụm will chuyển sang **Successed** , when that user can truy cập ando Jenkins instance thông qua url and account/password has been is khởi tạo. [![](/img/migrated/Picture4-48e1c05d.png)](/img/migrated/Picture4-48e1c05d.png)
