# Create a new FPT Jenkins CI

To create a new Jenkins CI instance, follow these steps:

**Step 1:** In the **FPT Portal** menu > **DevOps** > **Jenkins CI** > click **Create a Jenkins**.

[![](/img/migrated/Picture1-38ff2a6f.png)](/img/migrated/Picture1-38ff2a6f.png)

**Step 2:** Click **Create a Jenkins** > enter the information to create a Jenkins CI instance.

[![](/img/migrated/Picture2-1-9c7a0368.png)](/img/migrated/Picture2-1-9c7a0368.png)

**Step 3:** Click **Create**. The system will create a Jenkins Instance and display the processing status.

[![](/img/migrated/Picture3-1-47e2570d.png)](/img/migrated/Picture3-1-47e2570d.png)

Once the Jenkins instance is successfully created, the cluster status changes to **Succeeded**. You can then access the Jenkins instance via the URL and the account/password created during setup.

[![](/img/migrated/Picture4-48e1c05d.png)](/img/migrated/Picture4-48e1c05d.png)
