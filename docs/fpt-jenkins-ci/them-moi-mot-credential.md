# Add mới a Credential

Với mỗi loại credential, user can thực hiện cấu hình as follows:
**Step 1.** Tại menu FPT Portal > **Jenkins CI** > **Detail Cluster** > **Credential Managements** : [![](/img/migrated/Picture28-f1ca981e.png)](/img/migrated/Picture28-f1ca981e.png)
  * **Create mới Credential loại Secret Text**

**Step 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới a credential loại **Secret Text** : [![](/img/migrated/Picture29-4e919329.png)](/img/migrated/Picture29-4e919329.png) **Step 3.** Enter thông tin to tạo credential with kind Secret Text: [![](/img/migrated/Picture30-6ec104f7.png)](/img/migrated/Picture30-6ec104f7.png) **Step 4.** Click **Add** to tạo a credential with kind Secret Text. Kết quả after tạo credential: [![](/img/migrated/Picture31-15489933.png)](/img/migrated/Picture31-15489933.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture32-690a204c.png)](/img/migrated/Picture32-690a204c.png) [![](/img/migrated/Picture33-c7a729fd.png)](/img/migrated/Picture33-c7a729fd.png)
  * **Create credential with loại Username with password**

**Step 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới a credential loại **Username with password** : [![](/img/migrated/Picture34-0260b9d4.png)](/img/migrated/Picture34-0260b9d4.png) **Step 3**. Enter thông tin to tạo credential with kind **Username with password** : [![](/img/migrated/Picture35-52ee1479.png)](/img/migrated/Picture35-52ee1479.png) **Step 4.** Click **Add** to tạo a credential with kind **Username with password**. Kết quả after tạo credential: [![](/img/migrated/Picture36-ce02107a.png)](/img/migrated/Picture36-ce02107a.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture37-42448ed6.png)](/img/migrated/Picture37-42448ed6.png) [![](/img/migrated/Picture38-e8817bc1.png)](/img/migrated/Picture38-e8817bc1.png)
  * **Create credential with loại GitLab API token**

**Step 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới a credential loại **GitLab API token** : [![](/img/migrated/Picture39-5f3a117b.png)](/img/migrated/Picture39-5f3a117b.png) **Step 3.** Enter thông tin to tạo credential with kind **GitLab API token** : [![](/img/migrated/Picture40-bd944784.png)](/img/migrated/Picture40-bd944784.png) **Step 4.** Click **Add** to tạo a credential with kind **GitLab API token**. Kết quả after tạo credential: [![](/img/migrated/Picture41-232f7619.png)](/img/migrated/Picture41-232f7619.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture42-addc31f0.png)](/img/migrated/Picture42-addc31f0.png) [![](/img/migrated/Picture43-4d3ae4c9.png)](/img/migrated/Picture43-4d3ae4c9.png)
  * **Create credential with loại SSH Username with private key**

**Step 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới a credential loại **SSH Username with private key** : [![](/img/migrated/Picture44-cc87bfd1.png)](/img/migrated/Picture44-cc87bfd1.png) **Step 3.** Enter thông tin to tạo credential with kind **SSH Username with private key** : [![](/img/migrated/Picture45-562215f2.png)](/img/migrated/Picture45-562215f2.png) **Step 4.** Click **Add** to tạo a credential with kind **SSH Username with private key**. Kết quả after tạo credential: [![](/img/migrated/Picture46-aa140b9e.png)](/img/migrated/Picture46-aa140b9e.png) Kết quả tạo credential dưới Jenkins instance: [![](/img/migrated/Picture47-03acc4a6.png)](/img/migrated/Picture47-03acc4a6.png) [![](/img/migrated/Picture48-8be15bf6.png)](/img/migrated/Picture48-8be15bf6.png)
