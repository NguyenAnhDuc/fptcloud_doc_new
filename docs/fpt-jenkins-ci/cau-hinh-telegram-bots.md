# Configure Telegram Bots

FPT Cloud hỗ trợ user cấu hình Telegram Bot in hệ thống Jenkins phục vụ việc gửi notification tới user thông qua giao diện Portal. Để thực hiện cấu hình, user thực hiện as follows:
  * **Chỉnh sửa cấu hình Telegram Bots in cấu hình mặc định of Jenkins**

**Step 1.** Tại menu FPT Portal > **Jenkins CI** > Detail Cluster > **Configurations** > **Telegram Bot** > Select **Edit:** [![Alt text](/img/migrated/Picture84-ace31d4a.png)](/img/migrated/Picture84-ace31d4a.png) **Step 2.** Thực hiện cấu hình Telegram Bot: [![Alt text](/img/migrated/Picture85-5c549fcf.png)](/img/migrated/Picture85-5c549fcf.png) [![Alt text](/img/migrated/Picture86-5f8ffdb1.png)](/img/migrated/Picture86-5f8ffdb1.png) **Step 3.** Kết quả after cấu hình [![Alt text](/img/migrated/Picture87-7a77dfa4.png)](/img/migrated/Picture87-7a77dfa4.png) Check kết quả dưới Jenkins: [![Alt text](/img/migrated/Picture88-254bc3bc.png)](/img/migrated/Picture88-254bc3bc.png)
