# Upgrade gói service

Khi số lượng jobs trên Jenkins chạy vượt quá tài nguyên is dùng to chạy ứng dụng, user có nhu cầu tăng tài nguyên for cụm Jenkins Controller is chạy can thực hiện chọn the gói tài nguyên has been is cung cấp bởi FPT Cloud as follows:
**Step 1.** Vào menu **Essential Properties** > **Instance Information** [![](/img/migrated/Picture93-650c22b2.png)](/img/migrated/Picture93-650c22b2.png) **Step 2.** Chọn gói tài nguyên muốn upgrade: [![](/img/migrated/Picture94-74bdab8d.png)](/img/migrated/Picture94-74bdab8d.png) **Note:** System chỉ for phép update lên gói service cao hơn, không hỗ trợ downgrade resource *_Step 3._ Kết quả after update resource: [![](/img/migrated/Picture95-a73deeab.png)](/img/migrated/Picture95-a73deeab.png)
