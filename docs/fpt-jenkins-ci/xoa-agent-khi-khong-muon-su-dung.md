# Delete Agent when không muốn sử dụng

Để xoá a Agent không sử dụng, user thực hiện as follows:
**Step 1.** Tại menu FPT Portal > **Jenkins CI** > Detail Cluster > **Agent Managements** > Click **Actions** đối with agent muốn cập nhật: [![](/img/migrated/Picture23-5442fdee.png)](/img/migrated/Picture23-5442fdee.png) **Step 2.** Select **Delete** to thực hiện xoá Agent: [![](/img/migrated/Picture24-3dd2460d.png)](/img/migrated/Picture24-3dd2460d.png) **Step 3.** Trong màn confirm delete, nhập **delete** and nhấn **confirm** to xoá agent: [![](/img/migrated/Picture25-675a92fd.png)](/img/migrated/Picture25-675a92fd.png) **Step 4.** Kết quả after xoá agent Trên Portal không displayed thông tin agent: [![](/img/migrated/Picture26-c4890775.png)](/img/migrated/Picture26-c4890775.png) Information agent dưới Jenkins cũng bị xoá: [![](/img/migrated/Picture27-b7f58552.png)](/img/migrated/Picture27-b7f58552.png)
