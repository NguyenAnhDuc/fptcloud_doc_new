# Log in to FPT Cloud Jenkins CI

To log in to the Jenkins instance, use the username and password created on the Portal:

**Step 1:** In the **FPT Portal** menu > **DevOps** > **Jenkins CI** > click the **Instance Name** to view cluster details.

[![](/img/migrated/Picture99-9e038c5d.png)](/img/migrated/Picture99-9e038c5d.png)

On the first screen, you will see the connection details for the Jenkins CI instance, including the URL and login credentials.

[![](/img/migrated/Picture100-5653186e.png)](/img/migrated/Picture100-5653186e.png)

**Step 2:** Log in to the Jenkins instance.

On the **Essential Properties** screen, copy the URL and open the application:

- **URL:** From the **Essential Properties** screen.
- **Username:** Copied from the **Essential Properties** screen.
- **Password:** Copied from the **Essential Properties** screen.

[![](/img/migrated/Picture101-956cda0d.png)](/img/migrated/Picture101-956cda0d.png)

Interface after successfully logging in:

[![](/img/migrated/Picture102-8ac75f23.png)](/img/migrated/Picture102-8ac75f23.png)

:::note
If you change the admin username or password directly in Jenkins, the credentials shown on the Portal will no longer be valid for authentication.
:::
