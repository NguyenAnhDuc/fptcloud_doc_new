# Update a credential

Để cập nhật thông tin a credential, user can thực hiện as follows:
**Step 1.** Tại menu FPT Portal > **Jenkins CI** > Detail Cluster > **Credential Managements:**
[![](/img/migrated/Picture49-64efc5d7.png)](/img/migrated/Picture49-64efc5d7.png)
  * **Update Credential loại Secret Text**

**Step 2.** Tại màn **Credential Managements** > Chọn credential loại Secret Text cần Edit > Actions **Edit** : [![](/img/migrated/Picture50-0e401103.png)](/img/migrated/Picture50-0e401103.png) **Step 3.** Thực hiện chỉnh sửa the thông tin: [![](/img/migrated/Picture51-dbcf07bc.png)](/img/migrated/Picture51-dbcf07bc.png) **Step 4.** Kết quả after cập nhật thông tin [![](/img/migrated/Picture52-7c7e78e0.png)](/img/migrated/Picture52-7c7e78e0.png)
  * **Update Credential loại Username with password**

**Step 2.** Tại màn **Credential Managements** > Chọn credential loại Username with password cần Edit > Actions **Edit** : [![](/img/migrated/Picture53-c95f4268.png)](/img/migrated/Picture53-c95f4268.png) **Step 3.** Thực hiện chỉnh sửa the thông tin: [![](/img/migrated/Picture54-0edd4bec.png)](/img/migrated/Picture54-0edd4bec.png) **Step 4.** Kết quả after cập nhật thông tin [![](/img/migrated/Picture55-cda81603.png)](/img/migrated/Picture55-cda81603.png)
  * **Update Credential loại GitLab API token**

**Step 2.** Tại màn **Credential Managements** > Chọn credential loại GitLab API token cần Edit > Actions **Edit** : [![](/img/migrated/Picture56-c87fb773.png)](/img/migrated/Picture56-c87fb773.png) **Step 3.** Thực hiện chỉnh sửa the thông tin: [![](/img/migrated/Picture57-23d42fb3.png)](/img/migrated/Picture57-23d42fb3.png) **Step 4.** Kết quả after cập nhật thông tin [![](/img/migrated/Picture58-0f86bc09.png)](/img/migrated/Picture58-0f86bc09.png)
  * **Update Credential loại SSH Username with private key**

**Step 2.** Tại màn **Credential Managements** > Chọn credential loại SSH Username with private key cần Edit > Actions **Edit** : [![](/img/migrated/Picture59-5564d869.png)](/img/migrated/Picture59-5564d869.png) **Step 3.** Thực hiện chỉnh sửa the thông tin: [![](/img/migrated/Picture60-b5e5b888.png)](/img/migrated/Picture60-b5e5b888.png) **Step 4.** Kết quả after cập nhật thông tin [![](/img/migrated/Picture61-37e0a81e.png)](/img/migrated/Picture61-37e0a81e.png)
