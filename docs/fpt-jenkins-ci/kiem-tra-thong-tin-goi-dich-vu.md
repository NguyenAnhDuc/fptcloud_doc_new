# Check service package information

FPT Cloud provides **Jenkins CI** service packages accessible under **Quota Detail**. To check available packages:

**Step 1:** In the FPT Portal menu, select **Jenkins CI** > click **View Details** to check the package for activating the **FPT Jenkins CI** service.

[![](/img/migrated/Picture2-2-8836c9d9.png)](/img/migrated/Picture2-2-8836c9d9.png)

**Step 2:** The portal displays the available service packages.

[![](/img/migrated/Picture3-2-61deedb9.png)](/img/migrated/Picture3-2-61deedb9.png)

By default, you can activate the service with the Basic package. If you need a higher tier, contact Sales to upgrade your package.
