# Nâng cấp version for database

FPT Cloud hỗ trợ user upgrade version Jenkins lên version mới hơn in số the version FPT Cloud support. Hiện tại trên Portal is support a số version includes: 2.452,2.453,2.454.
**Step 1.** Vào menu **Essential Properties** > **Instance Information** [![](/img/migrated/Picture89-ffa69e1a.png)](/img/migrated/Picture89-ffa69e1a.png) **Step 2.** Thực hiện chọn version muốn upgrade: [![](/img/migrated/Picture90-bebb276a.png)](/img/migrated/Picture90-bebb276a.png) **Step 3.** Click **update** to thực hiện Upgrade version. [![](/img/migrated/Picture91-cd0fd45c.png)](/img/migrated/Picture91-cd0fd45c.png) **Step 4.** Kết quả after upgrade version successfully [![](/img/migrated/Picture92-5539c4c1.png)](/img/migrated/Picture92-5539c4c1.png) **Note:** System will mất a khoảng thời gian to upgrade version.
