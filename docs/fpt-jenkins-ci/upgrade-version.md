# Upgrade Jenkins version

FPT Cloud allows you to upgrade your Jenkins instance to a newer supported version. The Portal currently supports the following versions: 2.452, 2.453, 2.454.

**Step 1:** Go to **Essential Properties** > **Instance Information**.

[![](/img/migrated/Picture89-ffa69e1a.png)](/img/migrated/Picture89-ffa69e1a.png)

**Step 2:** Select the version to upgrade to.

[![](/img/migrated/Picture90-bebb276a.png)](/img/migrated/Picture90-bebb276a.png)

**Step 3:** Click **Update** to perform the upgrade.

[![](/img/migrated/Picture91-cd0fd45c.png)](/img/migrated/Picture91-cd0fd45c.png)

**Step 4:** Result after a successful upgrade.

[![](/img/migrated/Picture92-5539c4c1.png)](/img/migrated/Picture92-5539c4c1.png)

:::note
The system requires some time to complete the version upgrade.
:::
