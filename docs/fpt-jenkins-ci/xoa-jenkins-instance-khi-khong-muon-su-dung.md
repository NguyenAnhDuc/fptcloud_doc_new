# Delete a Jenkins instance

To delete a Jenkins instance you no longer need:

**Step 1:** In the FPT Cloud menu > **Jenkins CI** > select the instance to delete > **Actions** > **Delete**.

[![](/img/migrated/Picture96-8e14d6a9.png)](/img/migrated/Picture96-8e14d6a9.png)

**Step 2:** Type **delete** and click **Confirm** to confirm the deletion.

[![](/img/migrated/Picture97-9e157809.png)](/img/migrated/Picture97-9e157809.png)

**Step 3:** Result after deleting the instance.

[![](/img/migrated/Picture98-aea6488f.png)](/img/migrated/Picture98-aea6488f.png)
