# Delete Jenkins instance when không muốn sử dụng

Để xoá a Jenkins instance không muốn sử dụng, can thực hiện as follows:
**Step 1.** Tại menu FPT Cloud > **Jenkins CI** > Chọn instance muốn xoá > Actions **Delete** : [![](/img/migrated/Picture96-8e14d6a9.png)](/img/migrated/Picture96-8e14d6a9.png) **Step 2.** Enter **delete** > click **Confirm** to xác nhận xoá instance: [![](/img/migrated/Picture97-9e157809.png)](/img/migrated/Picture97-9e157809.png) **Step 3.** Kết quả after xoá instance: [![](/img/migrated/Picture98-aea6488f.png)](/img/migrated/Picture98-aea6488f.png)
