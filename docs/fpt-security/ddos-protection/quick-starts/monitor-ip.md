# Monitor protected IPs

You can monitor your system and protection status through the dashboard on the FPT Cloud Portal.

1. Go to the protected IP list and select the IP you want to inspect.

[![Protected IP list](/img/migrated/image-1750405781163-2bd000bd.png)](/img/migrated/image-1750405781163-2bd000bd.png)

2. Review the protection status:

   - **No attacks**: The system shows that the IP is protected and no attacks have been recorded this month.

[![IP protected, no attacks](/img/migrated/image-1750405795153-92b66506.png)](/img/migrated/image-1750405795153-92b66506.png)

   - **Attacks detected**: The system displays detailed information about each attack event.

[![DDoS attack information](/img/migrated/image-1750405809600-ee4429cf.png)](/img/migrated/image-1750405809600-ee4429cf.png)

## Monthly protection summary

- **Total traffic**: Total inbound traffic to the IP this month.
- **Block traffic**: Total attack traffic detected and blocked.
- **Block packet**: Total attack packets detected and blocked.

## Attack sources

The system lists the main attack sources.

[![Attack sources](/img/migrated/image-1750405829915-04f543c3.png)](/img/migrated/image-1750405829915-04f543c3.png)

## Attacked protocols

The system displays protocols by ID. For reference, see [IANA Protocol Numbers](https://www.iana.org/assignments/protocol-numbers/protocol-numbers.xhtml).

- **6**: TCP
- **17**: UDP

[![Attacked protocols](/img/migrated/image-1750405849332-5868a949.png)](/img/migrated/image-1750405849332-5868a949.png)
