# Create object

**Objects** define the hosts, network ranges, ports, and other entities used when setting up rules.

**Step 1:** In the **Object Pane** (**F11**) > click **New** > select **Network**.

[![Userguide Checkpoint FPT NGFW 2022 44](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-44-10-e9586759.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-44-10-e9586759.png)

**Step 2:** Enter the name, address, mask, and comment for the network range > click **OK**.

[![Userguide Checkpoint FPT NGFW 2022 45](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-45-721733e9.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-45-721733e9.png)

**Step 3:** Install Policy.

**Step 1:** In the **Object Pane** (**F11**) > click **New** > select **Host**.

[![Userguide Checkpoint FPT NGFW 2022 46](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-46-10-6a86ce67.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-46-10-6a86ce67.png)

**Step 2:** Enter the name, IPv4 address, and comment for the host > click **OK**.

[![Userguide Checkpoint FPT NGFW 2022 47](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-47-6603e0e1.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-47-6603e0e1.png)

**Step 3:** Install Policy.

**Step 1:** In the **Object Pane** (**F11**) > click **New** > **More** > **Service** > select the service type (**UDP/TCP/RPC/...**).

[![Userguide Checkpoint FPT NGFW 2022 48](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-48-10-caf6afa6.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-48-10-caf6afa6.png)

**Step 2:** Enter the name, port, and comment for the service > click **OK**.

[![Userguide Checkpoint FPT NGFW 2022 49](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-49-adcedff8.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-49-adcedff8.png)

**Step 3:** Install Policy.
