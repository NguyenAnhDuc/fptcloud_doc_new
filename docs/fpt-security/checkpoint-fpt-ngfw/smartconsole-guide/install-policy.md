# Install Policy

:::note
Any changes made in **SmartConsole** must be published and installed via **Install Policy** before they take effect.
:::

**Step 1:** Select **Install Policy** (**Ctrl + Shift + Enter**) > choose **Publish & Install** > click **Install**.
