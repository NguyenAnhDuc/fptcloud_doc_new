# Static Route

**Step 1:** In the menu, go to **Network Management** > select **IPv4 Static Routes** > **Add**.

[![Userguide Checkpoint FPT NGFW 2022 21](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-21-10-3fdc31da.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-21-10-3fdc31da.png)

**Step 2:** Enter the **Destination** and **Subnet mask** for the network range to route to.

[![Userguide Checkpoint FPT NGFW 2022 22](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-22-72d2141f.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-22-72d2141f.png)

**Step 3:** Click **Add Gateway** to enter the next-hop address.

[![Userguide Checkpoint FPT NGFW 2022 23](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-23-2b3e7d55.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-23-2b3e7d55.png)

You can route to an **IP Address** or out through an **Interface**.

[![Userguide Checkpoint FPT NGFW 2022 24](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-24-9ff6d0df.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-24-9ff6d0df.png)

**Step 1:** In the menu, go to **Network Management** > select **IPv4 Static Routes** > select the route to edit > **Edit**.

[![Userguide Checkpoint FPT NGFW 2022 26](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-26-10-51cbfb6b.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-26-10-51cbfb6b.png)

**Step 2:** Enter the new Gateway parameters.

[![Userguide Checkpoint FPT NGFW 2022 27](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-27-e489c334.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-27-e489c334.png)

:::note
The destination subnet cannot be edited. If needed, delete the route and create a new one.
:::

**Step 1:** In the menu, go to **Network Management** > select **IPv4 Static Routes**.

[![Userguide Checkpoint FPT NGFW 2022 28](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-28-10-50601952.png)](/img/migrated/Userguide-Checkpoint-FPT-NGFW-2022-28-10-50601952.png)

**Step 2:** Select the route to delete > click **Delete** > click **OK** to confirm.
