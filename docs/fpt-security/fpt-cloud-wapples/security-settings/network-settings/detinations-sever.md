# Destination Server

This feature configures web services from host IPs located in the cloud.

[![file](/img/migrated/image-1714121626519-4364137e.png)](/img/migrated/image-1714121626519-4364137e.png)

**Add / Edit / Delete a Destination Server** [![file](/img/migrated/image-1714719775318-5890dbc9.png)](/img/migrated/image-1714719775318-5890dbc9.png)

- **Mode**: Proxy
- **Proxy IP**: The configured proxy IP address
- **Proxy Port**: The proxy port converted from the original port
- **Web Server IP (Domain)**: The IP address or domain of the web server service
- **Web Server Port**: The service port of the web server
- **Health Check**: Checks the status of the web server

[![file](/img/migrated/image-1714719898104-ec1e303d.png)](/img/migrated/image-1714719898104-ec1e303d.png)

- **Gateway**: The configured gateway IP
- **VLAN ID**: Displays the entered VLAN ID. Empty if SSL is not used
- **SSL Use**: Displays X if not used, O if used
- **SSL Termination**: Displays X if not used, O if used
- **Certificate Expiration**: Displays the expiration date of the selected certificate when SSL is in use
