# Proxy IP

This feature configures proxy IP addresses to set up private IP services within the cloud.

[![file](/img/migrated/image-1714112480774-859e0576.png)](/img/migrated/image-1714112480774-859e0576.png)

Available operations include adding, editing, and deleting proxy IPs.

**1. Add a Proxy IP**

- On the Security Settings / Network Settings / Proxy IP page, click the **+** icon to add a Proxy IP
- Enter the required values to configure the IP as shown, then click **Apply** [![file](/img/migrated/image-1714112494432-ca5d7a26.png)](/img/migrated/image-1714112494432-ca5d7a26.png)
- Click **Save** to save the changes [![file](/img/migrated/image-1714112501548-e3275d5b.png)](/img/migrated/image-1714112501548-e3275d5b.png)

**2. Edit a Proxy IP**

- On the Security Settings / Network Settings / Proxy IP page, click **Modify**
- Make the necessary changes: [![file](/img/migrated/image-1714112513079-760a45f0.png)](/img/migrated/image-1714112513079-760a45f0.png)
- Click **Apply** then **Save** to save the settings [![file](/img/migrated/image-1714115110737-1154d081.png)](/img/migrated/image-1714115110737-1154d081.png)

**3. Delete a Proxy IP**

- On the Security Settings / Network Settings / Proxy IP page, select the IP to delete [![file](/img/migrated/image-1714115123188-4dfc9591.png)](/img/migrated/image-1714115123188-4dfc9591.png)
- Click the trash icon to delete the selected IP, then click **Save** to apply the changes [![file](/img/migrated/image-1714115133281-726201bd.png)](/img/migrated/image-1714115133281-726201bd.png)
