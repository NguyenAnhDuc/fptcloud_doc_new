# Network Filters

Used to configure network rules, similar to security groups.

**Display fields:**

- **Filter No.**: The policy sequence number
- **Source IP**: The accessing IP address
- **Source Port**: The accessing port
- **Source Netmask**: The subnet being accessed
- **Destination IP**: The destination IP of the service being accessed
- **Destination Port**: The destination service port
- **Destination Netmask**: The destination subnet being accessed
- **Protocol**: The access protocol
- **Connection**: The action taken when a connection is detected (Allow, Deny, Detection)
