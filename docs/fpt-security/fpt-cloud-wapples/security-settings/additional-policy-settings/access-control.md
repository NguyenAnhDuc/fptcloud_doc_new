# Access Control

The Access Control feature prevents invalid behaviors such as spamming and clicking.

- **Add Access Control** [![file](/img/migrated/image-1714116678455-4855ca28.png)](/img/migrated/image-1714116678455-4855ca28.png)
- Set the rule name
- Set the source IP
- Set the destination IP
- Set the destination details (Host: domain, Port, Method: GET/POST, URL: subdomain) [![file](/img/migrated/image-1714116684905-93057588.png)](/img/migrated/image-1714116684905-93057588.png)
- Set the blocking duration from the time of the behavior, the maximum number of behaviors required to trigger a block, and the time interval for the maximum allowed behaviors
- Separate selected filters in [Source IP / Destination IP / URL / Host / Method] with a comma `,`.
