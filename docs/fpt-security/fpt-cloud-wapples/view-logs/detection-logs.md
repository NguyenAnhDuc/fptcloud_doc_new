# Detection Logs

Displays logs recorded for websites at IP addresses configured behind WAPPLES.

You can create preset filter list configurations for individual cases by filling in the required fields and then clicking **Register**.

You can use advanced filtering to identify specific log types — such as logs for a particular IP — by clicking **Advanced Condition**.
