# FPT Cloud WAPPLES

FPT Cloud WAPPLES is a web application firewall (WAF) solution that protects web applications from common threats and vulnerabilities. It provides real-time detection and blocking of web attacks, along with comprehensive logging and reporting.

## Main sections

- **Dashboard** - Overview of traffic and device status
- **Data Analysis** - Graphical analysis and detection log analysis
- **View Logs** - Detection logs, audit logs, website logs
- **Security Settings** - Network configuration, policy settings, detection exceptions
- **Reports** - Report generation, scheduling, and history
- **Configuration** - Accounts, backup, integration, system settings
