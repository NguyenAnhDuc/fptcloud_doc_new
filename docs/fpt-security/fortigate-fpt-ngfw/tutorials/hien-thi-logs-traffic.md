# View traffic logs

When firewall policies are created, traffic connection logs appear in **Forward Traffic** after enabling **Generate Logs When Session Starts**.

[![Enable log generation in Firewall Policy](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-35-102-0e039dba.png)](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-35-102-0e039dba.png)

- Navigate to **Log & Report > Forward Traffic**. Double-click a log entry to view its details.

[![Forward Traffic log list](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-36-102-53136a75.png)](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-36-102-53136a75.png)

- Security profile logs are stored in the corresponding sections under **Log & Report > Antivirus / Web Filter / WAF / IPS**, etc.
