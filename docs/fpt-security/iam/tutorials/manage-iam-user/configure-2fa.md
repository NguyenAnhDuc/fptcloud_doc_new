# Configure 2FA

**2FA (Two Factor Authentication)** is a more secure account verification method that adds an extra layer of security beyond a password. An account can only log in successfully if both verification steps are completed.

To enable two-step verification, follow these steps.

1. Click on the account name area in the upper right corner of the FPT Portal, then select **Account Security**.

[![](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)

2. Under **Two Factor Authentication**, click **Set up Authenticator Application**.

[![](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)

3. Install 2FA on your mobile device by following the instructions on the **Mobile Authenticator Setup** page, then click **Submit**.

[![](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)

After successfully configuring 2FA, you will be prompted to enter the OTP generated by your 2FA application each time you log in.
