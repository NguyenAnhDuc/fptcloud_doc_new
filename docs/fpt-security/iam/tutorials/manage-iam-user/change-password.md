# Change password

After an IAM user registers and successfully logs in to the FPT Portal, it is recommended to change the password to keep the account secure.

1. Click on the account name area in the upper right corner of the FPT Portal, then select **Change Password**.

[![](/img/migrated/Userguide-FPT-IAM-2022-27-1024x285-88368ee2.png)](/img/migrated/Userguide-FPT-IAM-2022-27-1024x285-88368ee2.png)

2. In the **Change Password** dialog, enter the required information:
   - **Current**: Enter the current password.
   - **New**: Enter the new password.
   - **Confirm**: Confirm the new password.

[![](/img/migrated/Userguide-FPT-IAM-2022-28-c40f7554.png)](/img/migrated/Userguide-FPT-IAM-2022-28-c40f7554.png)

3. Click **Change Password**. The system verifies, processes, and displays the result.

[![](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)
