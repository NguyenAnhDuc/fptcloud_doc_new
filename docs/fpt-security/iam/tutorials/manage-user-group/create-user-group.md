# Create a User Group

1. From the menu, select **IAM** > **User Groups**, then click **Create User Group**.

   [![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. On the **Create User Group** page, enter the required information:

   - **Name**: The name of the User Group.
   - **Description**: A description of the User Group.
   - **Role**: Assign one or more roles to the User Group.
   - **User**: Add existing users to the User Group.

   [![](/img/migrated/image-1734344557099-e110b8a0.png)](/img/migrated/image-1734344557099-e110b8a0.png)

3. Click **Create User Group**. The system processes the request and displays the result.
