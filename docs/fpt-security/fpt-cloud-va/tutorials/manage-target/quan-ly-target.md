# Manage targets

A **target** is a scan object that FPT VA scans for security vulnerabilities. Targets are created and configured on **FPT Portal**. A target can be a website, a public IP address, or an IP range that you manage.

Target management operations include:

- [Create a target](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/tao-target)
- [Check target scan status](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/kiem-tra-trang-thai-target)
- [Edit target scan configuration](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/update-target)
- [Scan now](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/scan-now)
- [Delete a target](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/xoa-target)
- [Get website headers](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/lay-header)
- [Get website cookies](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/lay-cookie)
- [Enable Remote Registry on Windows for SMB](/docs/fpt-security/fpt-cloud-va/tutorials/manage-target/mo-remote-registry-window)
