# Scan now

The **Scan Now** feature lets you scan a target immediately without waiting for the scheduled time.

1. From the menu, select **Security > Cloud VA**. In the **Action** column of the target you want to scan, click **Scan Now**.

2. The system will confirm, check resources, and begin scanning immediately. To cancel during the scan, click **Stop**.

:::warning
Stopping an in-progress scan will produce incomplete scan results and still counts as a paid scan. Consider carefully before stopping a scan.
:::
