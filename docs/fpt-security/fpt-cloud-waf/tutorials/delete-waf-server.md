# Delete WAF server

:::danger
This action permanently deletes all data associated with the WAF server and **cannot be undone**. Verify carefully before proceeding.
:::

1. On the **Cloud Web Application Firewall Management** screen, click **Delete WAF** for the server you want to remove.

   [![Select Delete WAF](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)

2. In the confirmation dialog, click **Delete** to complete the deletion.

   [![Confirm WAF server deletion](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)
