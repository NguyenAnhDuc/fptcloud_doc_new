# Update WAF server information

1. Select the WAF server you want to edit.

   [![Select WAF server](/img/migrated/Userguide-FPT-WAF-2022-11-1024x538-67ed51c4.png)](/img/migrated/Userguide-FPT-WAF-2022-11-1024x538-67ed51c4.png)

2. Edit the WAF server information. You can update: **Name**, **Description**, **Size**.

   [![WAF server edit form](/img/migrated/Userguide-FPT-WAF-2022-12-1024x538-e643bd5f.png)](/img/migrated/Userguide-FPT-WAF-2022-12-1024x538-e643bd5f.png)

   :::warning
   - When changing the size, you can only select a size **larger** than the current one.
   - Changing the size may cause a few minutes of downtime until the change is complete. Perform this action during low-traffic periods.
   :::

3. Click **Update WAF** to save the changes.
