# Delete a proxy host

:::danger
Deleting a proxy host removes WAF protection from your web server. Proceed with caution.
:::

1. In the **WAF Dashboard**, select **Hosts** > **Proxy**.

   [![Proxy host list](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. On the proxy host you want to delete, click the **Delete** icon.

   [![Select Delete on proxy host](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)](/img/migrated/Userguide-FPT-WAF-2022-24-1024x578-0b7b74ca.png)

3. In the confirmation dialog, click **Delete** to complete the deletion.

   [![Confirm proxy host deletion](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)](/img/migrated/Userguide-FPT-WAF-2022-25-1a7b6311.png)
