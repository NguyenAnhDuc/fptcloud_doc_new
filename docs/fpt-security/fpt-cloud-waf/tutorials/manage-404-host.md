# Manage 404 host

A **404 host** redirects 404 error pages to a desired page.

1. Select **Hosts** > **404 Not Found** from the menu to open the management screen.

   [![404 host management screen](/img/migrated/Userguide-FPT-WAF-2022-32-1024x502-a4e51c1a.png)](/img/migrated/Userguide-FPT-WAF-2022-32-1024x502-a4e51c1a.png)

2. Click **Create 404 Not Found** to open the creation screen and enter the details:

   - **Domain Names**: Domain or website to redirect when a 404 error occurs.
   - **Custom Configuration**
   - **SSL Certificate**: Select the corresponding SSL certificate for the domain or website.

   [![Enter 404 host details](/img/migrated/Userguide-FPT-WAF-2022-33-1024x502-98cf258a.png)](/img/migrated/Userguide-FPT-WAF-2022-33-1024x502-98cf258a.png)

3. Click **Create 404 Not Found** to save and complete.
