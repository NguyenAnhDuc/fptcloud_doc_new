# Lock instance deletion

You can lock a virtual machine to prevent it from being deleted using the following steps.

1. In **Instance Management**, select the virtual machine, then click **Actions** → **Lock Deletion**.

2. A confirmation dialog displays the virtual machine name. Click **Lock Instance Deletion** to confirm.

[![Lock Instance Deletion confirmation dialog](/img/migrated/image-1702889848019-0f69d8a3.png)](/img/migrated/image-1702889848019-0f69d8a3.png)

The virtual machine cannot be deleted until it is unlocked.
