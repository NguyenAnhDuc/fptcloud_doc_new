# Lock Instance Deletion

Bạn can khóa virtual machine lại to ngăn chúng không bị xóa with the thao tác đơn giản sau. 
Step 1: Trên bảng quản trị Instance Schedule Management, in phần Action, chọn Lock Deletion
Step 2: Một hộp thoại cảnh báo sẽ hiện lên, displayed tên virtual machine và yêu cầu người dùng xác nhận. Bạn chọn Lock Instance Deletion để tiến hành khóa, sau đó hệ thống will khóa virtual machine lại to ngăn ko for bị xóa when chưa mở khóa.
