# Lock Instance Deletion

Bạn có thể khóa máy ảo lại để ngăn chúng không bị xóa với các thao tác đơn giản sau. 
Step 1: Trên bảng quản trị Instance Schedule Management, trong phần Action, chọn Lock Deletion
Step 2: Một hộp thoại cảnh báo sẽ hiện lên, hiển thị tên máy ảo và yêu cầu người dùng xác nhận. Bạn chọn Lock Instance Deletion để tiến hành khóa, sau đó hệ thống sẽ khóa máy ảo lại để ngăn ko cho bị xóa khi chưa mở khóa.
