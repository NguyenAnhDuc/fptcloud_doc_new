# Unlock Instance Deletion

Để can xóa virtual machine, you cần tiến hành mở khóa trước with the thao tác đơn giản sau. 
Step 1: Trên bảng quản trị Instance Schedule Management, in phần Action, chọn Unlock Deletion
Step 2: Một hộp thoại cảnh báo sẽ hiện lên, displayed tên virtual machine và yêu cầu người dùng xác nhận. Bạn chọn Unlock Instance Deletion để tiến hành mở khóa, sau đó hệ thống will mở khóa virtual machine to you can xóa máy như bình thường.
