# Create a server

You can deploy a Bare Metal Server through the FPT Customer Portal. FPT handles provisioning, OS installation, and configuration based on the settings you select. Once deployment is complete, you can access the server via SSH using an SSH key or password.

1. Sign in to your FPT Cloud account and select the tenant, region, and VPC (if you have multiple).
2. Go to AI Infrastructure and Metal Cloud in the sidebar, then click Create server.
3. Select a flavor (availability varies by region).
4. Enter the number of servers to deploy at once (the maximum depends on your quota).
5. Enter the server name.
6. Select the operating system. If the required OS is not available, you can use a custom image instead.
7. Select an available subnet. You cannot deploy a server without a subnet. Create a subnet before creating and deploying the server.
8. Select to attach a floating IP (optional).
9. Choose an authentication method: SSH key or console password. a. Select an available SSH key or add a new one (see the guide for creating and adding SSH keys here).
10. Configure RAID (optional).
11. Select or create a user data script (optional). The user data script runs automatically during the server's first boot through the cloud-init process.
12. Click Create server.
