# Create a subnet

A subnet is a range of IP addresses within a cloud network. Addresses in this range are assigned to one or more Metal Cloud servers. Only IPv4 is supported.

_The CIDR range is automatically assigned, held for 60 seconds, and then refreshed with a new one on screen. A Network ACL is required to create a subnet. By default, only outbound traffic from the Bare Metal GPU Server to the internet is allowed. The subnet quota is the same as the Bare Metal GPU Server quota._

1. Sign in to your FPTCloud account and select the **tenant, region, and VPC** (if you have multiple).
2. Go to **AI Infrastructure** and **Subnet for Metal Cloud** in the sidebar, then click **Create subnet**.
3. Enter the **subnet name**.
4. Enter the **network ACL name**.
5. Enter the **network ACL description** (optional).
6. Click **Create Subnet**.

# Delete a subnet

_You cannot delete a subnet that is associated with other resources._

1. Go to **AI Infrastructure** and **Subnet for Metal Cloud**.
2. View the server list in the sidebar.
3. Select the subnet and click **"Actions"**. 4. Click **Delete**. 5. Enter the text **DELETE** to confirm the deletion.

# Rename a subnet

_Subnet names may duplicate existing names._

1. Select the subnet and click **"Actions"**.
2. Click **Rename**.
3. Enter a new subnet name following the rules. The name must be at most 32 characters long and may only contain letters, numbers, and hyphens.
4. Click **Rename subnet**.

# Update a subnet description

1. Select the subnet and click **"Actions"**.
2. Click **Update description**.
3. Enter the new description. The length must not exceed 63 characters.
4. Click Save.
