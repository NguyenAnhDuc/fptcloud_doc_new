# Power a server on and off

1. Go to **AI Infrastructure** and **Metal Cloud** in the sidebar to view the server list.
2. Select the server and click **"Actions"**.
3. Click **"Power off"** to power off a running server, or click **"Power on"** to power on a stopped server.
4. Confirm the power-off or power-on action.

# Delete a server

_When you delete a server, all data on the disks is permanently erased and cannot be recovered. Make sure you are deleting the correct server._

1. Go to **AI Infrastructure** and **Metal Cloud** in the sidebar to view the server list.
2. Select the server and click **Actions**.
3. Click **Delete**.
4. Enter the text **DELETE** to confirm the deletion.

# Rename a server

_The server name must be unique and cannot duplicate an existing name._

1. Go to **AI Infrastructure** and **Metal Cloud** in the sidebar to view the server list.
2. Select the server and click **Actions**.
3. Click **Rename**.
4. Enter a new server name following the rules: the name must be at most 63 characters long and may only contain letters, numbers, and hyphens.
5. Click **Rename server**.
