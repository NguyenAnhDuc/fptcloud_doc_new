# Upgrade Kubernetes version

**Step 1** : In the menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page. Choose a Cluster that you want to upgrade the K8s version.
[![](/img/migrated/24-1-4f6944fa.png)](/img/migrated/24-1-4f6944fa.png)
**Step 2** : In the **GENERAL INFORMATION** > **Version** section, click on the **Setting icon**.
[![](/img/migrated/25-1-dcc21c83.png)](/img/migrated/25-1-dcc21c83.png)
**Step 3** : Choose the desired Version and click **Upgrade**.
[![](/img/migrated/26-1-84cc9b69.png)](/img/migrated/26-1-84cc9b69.png)
**Note** : Only version upgrades are possible; downgrading is not supported.
To avoid issues during the processing, it is recommended to upgrade versions sequentially in order.
