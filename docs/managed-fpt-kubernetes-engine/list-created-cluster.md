# List created clusters

You can view and manage the list of created Kubernetes Clusters on the Kubernetes Management page. To open Kubernetes Management, follow these steps:
In the FPT Portal menu, select **Kubernetes**. Choose the **Managed** tab. The system will display a list of created Clusters with important information such as **Name** , **Version** , **Worker Group** , **Type** , **Status** , **Created at**.
