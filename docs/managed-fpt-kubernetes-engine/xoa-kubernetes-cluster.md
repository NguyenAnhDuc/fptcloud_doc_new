# Delete a Kubernetes cluster

For Kubernetes clusters that are no longer needed, you can delete them by following the steps below.

**Step 1:** From the menu, select **Kubernetes**. The system displays the **Kubernetes Management** page.
[![](/img/migrated/Picture18-905db25e.png)](/img/migrated/Picture18-905db25e.png)

**Step 2:** Select **Action** at the end of the cluster row you want to delete. Select **Delete**.
[![](/img/migrated/Picture1-4-8d1555b6.png)](/img/migrated/Picture1-4-8d1555b6.png)

**Step 3:** Confirm the warning information in the popup and select **Delete**.
[![](/img/migrated/Picture20-fe6427cb.png)](/img/migrated/Picture20-fe6427cb.png)
