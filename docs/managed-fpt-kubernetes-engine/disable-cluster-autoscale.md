# Disable cluster autoscale

**Step 1** : In the menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page. Choose a Cluster that you want to disable Cluster Autoscale.
[![](/img/migrated/30-1-afd6524b.png)](/img/migrated/30-1-afd6524b.png)
**Step 2** : Click **Configure Worker Groups**.
[![](/img/migrated/31-1-f850853b.png)](/img/migrated/31-1-f850853b.png)
**Step 3** : Adjust the minimum and maximum number of workers to be equal.
[![](/img/migrated/32-1-dd7a83a3.png)](/img/migrated/32-1-dd7a83a3.png)
_Note: When the minimum and maximum worker count in a worker pool are equal, the Cluster Autoscale feature is automatically turned off._
**Step 4** : Verify the information and click **Save**.
[![](/img/migrated/33-1-ea1e7929.png)](/img/migrated/33-1-ea1e7929.png)
