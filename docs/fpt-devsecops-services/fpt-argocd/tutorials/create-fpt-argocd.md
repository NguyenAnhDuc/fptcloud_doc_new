# Create a new FPT ArgoCD instance

1. On the FPT Portal menu, select **Argo CD**.

[![](/img/migrated/Picture1-1-5dc58dbd.png)](/img/migrated/Picture1-1-5dc58dbd.png)

2. Click **Create Instance** and enter the required information to create the ArgoCD instance.

[![](/img/migrated/Picture119-2-400611b8.png)](/img/migrated/Picture119-2-400611b8.png)

3. Click **Create Instance**. The system creates the ArgoCD instance and displays the processing status.

[![](/img/migrated/Picture3-1-f4accb41.png)](/img/migrated/Picture3-1-f4accb41.png)

After successful creation, the instance status changes to **Succeeded**. You can access the ArgoCD cluster using the URL and account/password that were created.

[![](/img/migrated/Picture4-1-b767bf84.png)](/img/migrated/Picture4-1-b767bf84.png)
