# Configure OIDC for SSO

Ngoài tài khoản admin và local account, FPT Cloud hỗ trợ tích hợp ArgoCD với hệ thống Keycloak để quản lý tài khoản đăng nhập vào ArgoCD qua SSO (Single Sign-On).
