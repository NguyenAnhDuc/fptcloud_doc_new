# View Jenkins CI instance details

To work with an existing Jenkins cluster, start by viewing its details.

**Step 1:** In the **FPT Portal** menu > **DevOps** > **Jenkins CI** > click the **Instance Name** to view cluster details.

[![](/img/migrated/Picture5-1-e44d4a18.png)](/img/migrated/Picture5-1-e44d4a18.png)

On the first screen, you will see detailed information to access the Jenkins CI instance, including the URL and login credentials.

[![](/img/migrated/Picture6-69dec98b.png)](/img/migrated/Picture6-69dec98b.png)

**Step 2:** Log in to the **Jenkins Instance**.

On the **Essential Properties** screen, copy the URL and open the application.

[![](/img/migrated/Picture7-1-c8ec9386.png)](/img/migrated/Picture7-1-c8ec9386.png)
