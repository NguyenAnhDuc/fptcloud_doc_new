# Add a new credential

The following steps describe how to add each supported credential type.

**Step 1:** In the FPT Portal menu > **Jenkins CI** > **Detail Cluster** > **Credential Managements**.

[![](/img/migrated/Picture28-f1ca981e.png)](/img/migrated/Picture28-f1ca981e.png)

### Secret Text

**Step 2:** On the **Credential Managements** screen > **Add Credential**, add a new credential of type **Secret Text**.

[![](/img/migrated/Picture29-4e919329.png)](/img/migrated/Picture29-4e919329.png)

**Step 3:** Enter the required information for a Secret Text credential.

[![](/img/migrated/Picture30-6ec104f7.png)](/img/migrated/Picture30-6ec104f7.png)

**Step 4:** Click **Add** to create the credential. Result on the Portal:

[![](/img/migrated/Picture31-15489933.png)](/img/migrated/Picture31-15489933.png)

Result in Jenkins:

[![](/img/migrated/Picture32-690a204c.png)](/img/migrated/Picture32-690a204c.png)

[![](/img/migrated/Picture33-c7a729fd.png)](/img/migrated/Picture33-c7a729fd.png)

### Username with password

**Step 2:** On the **Credential Managements** screen > **Add Credential**, add a new credential of type **Username with password**.

[![](/img/migrated/Picture34-0260b9d4.png)](/img/migrated/Picture34-0260b9d4.png)

**Step 3:** Enter the required information for a Username with password credential.

[![](/img/migrated/Picture35-52ee1479.png)](/img/migrated/Picture35-52ee1479.png)

**Step 4:** Click **Add** to create the credential. Result on the Portal:

[![](/img/migrated/Picture36-ce02107a.png)](/img/migrated/Picture36-ce02107a.png)
