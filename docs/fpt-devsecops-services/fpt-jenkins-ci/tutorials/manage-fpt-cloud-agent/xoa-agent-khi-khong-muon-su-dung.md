# Delete an agent

To delete an agent you no longer need:

**Step 1:** In the FPT Portal menu > **Jenkins CI** > **Detail Cluster** > **Agent Managements** > click **Actions** for the agent to delete.

[![](/img/migrated/Picture23-5442fdee.png)](/img/migrated/Picture23-5442fdee.png)

**Step 2:** Select **Delete** to remove the agent.

[![](/img/migrated/Picture24-3dd2460d.png)](/img/migrated/Picture24-3dd2460d.png)

**Step 3:** In the confirmation dialog, type **delete** and click **Confirm**.

[![](/img/migrated/Picture25-675a92fd.png)](/img/migrated/Picture25-675a92fd.png)

**Step 4:** Result after deleting the agent. The agent no longer appears on the Portal.

[![](/img/migrated/Picture26-c4890775.png)](/img/migrated/Picture26-c4890775.png)

The agent is also removed from Jenkins.

[![](/img/migrated/Picture27-b7f58552.png)](/img/migrated/Picture27-b7f58552.png)
