# Delete a Secret Engine

Delete a Secret Engine when it is no longer needed to free up resources. This operation deletes all Secrets stored inside the Secret Engine.

:::danger
Deleting a Secret Engine cannot be undone. All Secrets inside will be permanently deleted. Make sure you have backed up any necessary information before proceeding.
:::

1. On **Key Vault Management**, select the **Key Vault** containing the **Secret Engine** to delete.

   [![Key Vault Management list screen](/img/migrated/Userguide-FPT-KeyVault-2022-4-1024x538-3dcfc356.png)](/img/migrated/Userguide-FPT-KeyVault-2022-4-1024x538-3dcfc356.png)

2. Open the **Secret** tab and click **Delete** in the **Action** menu of the **Secret Engine** to delete.

   [![Secret Engine list with Delete option in Action menu](/img/migrated/Userguide-FPT-KeyVault-2022-8-1024x385-60cfced4.png)](/img/migrated/Userguide-FPT-KeyVault-2022-8-1024x385-60cfced4.png)

3. A confirmation dialog appears. Click **Delete** to confirm deleting the **Secret Engine**.

   [![Secret Engine deletion confirmation dialog](/img/migrated/Userguide-FPT-KeyVault-2022-9-b25853dd.png)](/img/migrated/Userguide-FPT-KeyVault-2022-9-b25853dd.png)
