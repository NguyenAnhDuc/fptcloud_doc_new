View and update Secret value

**Step 1:** On the **Secret Engine Detail** screen, select the **Secret Engine** containing the **Secret** whose value you want to view.

![Userguide FPT KeyVault 2022 10](/img/migrated/docs-fpt-key-vault/tutorials-xem-va-cap-nhat-gia-tri-secret/img-001.png)

**Step 2:** Select the **Secret** you want to view and update.

![Userguide FPT KeyVault 2022 14](/img/migrated/docs-fpt-key-vault/tutorials-xem-va-cap-nhat-gia-tri-secret/img-002.png)

**Step 3:** The **Edit Secret** screen appears. View the existing information or enter new information to update. Select **Edit** to save.

![Userguide FPT KeyVault 2022 15](/img/migrated/docs-fpt-key-vault/tutorials-xem-va-cap-nhat-gia-tri-secret/img-003.png)
