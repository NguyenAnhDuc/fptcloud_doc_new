# Activate the service

After confirming the appropriate service plan, follow these steps to activate FPT Container Registry.

1. On the **Container Registry** screen, click **Active** to activate the service.

   [![Container Registry activation screen](/img/migrated/Picture4-1-57ba717c.png)](/img/migrated/Picture4-1-57ba717c.png)

2. On the activation screen, select the service plan you want to activate.

   [![Select Container Registry service plan](/img/migrated/Picture5-1-07c30cc9.png)](/img/migrated/Picture5-1-07c30cc9.png)

3. Verify the service activation result.

   [![Container Registry activation result](/img/migrated/Picture6-1-5e1fe773.png)](/img/migrated/Picture6-1-5e1fe773.png)

4. Hover over the **ⓘ** icon to get the login credentials, tag, and push image commands.

   [![Container Registry login information](/img/migrated/Picture7-1-41e2a703.png)](/img/migrated/Picture7-1-41e2a703.png)

:::note
Users who are not the Tenant Owner will not see the **Active** button. Select a tenant where you have the owner role to activate the service.
:::
