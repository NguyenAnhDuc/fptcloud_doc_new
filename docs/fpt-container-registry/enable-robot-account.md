# Enable a robot account

To restore access for a disabled robot account, follow the steps below. The **Enable** option only appears for accounts that are currently in the disabled state.

1. In the **Robot Account** list, select the robot account you want to enable, then select **ACTION ENABLE**.

   [![Select ACTION ENABLE](/img/migrated/Picture72-22b86e1b.png)](/img/migrated/Picture72-22b86e1b.png)

2. In the confirmation dialog, enter **ENABLE** to confirm.

   [![Enter ENABLE to confirm](/img/migrated/Picture73-973e765b.png)](/img/migrated/Picture73-973e765b.png)

3. Verify the **Enabled State** after enabling.

   [![Enabled state of robot account](/img/migrated/Picture74-e9f34e4e.png)](/img/migrated/Picture74-e9f34e4e.png)
