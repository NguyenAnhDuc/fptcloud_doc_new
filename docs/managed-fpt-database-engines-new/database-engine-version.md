# Database engine version

FPT Database Engine provides a fully managed database service for various database management systems supported by FPT, making it easy for customers to deploy, operate, and scale their systems efficiently. The **Database engine version** section describes the version support policy for these database engines, including the version lifecycle, availability, updates, and end-of-support dates.

By understanding the version policy, customers can proactively select and manage the appropriate database engine version, ensuring the system remains secure, stable, and compatible with the latest improvements, while minimizing risk and disruption during upgrades or version migrations.
