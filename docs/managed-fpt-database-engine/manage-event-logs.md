# Manage event logs

1. Select **Database Engine** > select a **Cluster ID** > on **Database Detail**, select the **Event Logs** tab.

2. The event log list is displayed. Review detailed information about events that occurred in the database (timestamp, event type, description, etc.).

3. Use filters to narrow down events by time period or type.
