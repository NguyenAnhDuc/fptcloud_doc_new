# Manage configuration

1. Select **Database Engine** > select a **Cluster ID** > on **Database Detail**, select the **Configuration** tab.

2. Select the type of parameter to change: **Dynamic params** or **Static params**.

3. Change the desired parameters.

   **Case 1: Change dynamic params**

   Select **Dynamic params**. For the parameter to change, click the edit icon (pen) in the Action column.

   - In the **Value** column, enter the new value and click the checkmark to save. To cancel, click the circle icon at the bottom to exit without saving.

   **Case 2: Change static params**

   Select **Static params**. For the parameter to change, click the edit icon (pen) in the Action column.

   - In the **Value** column, enter the new value and click the checkmark to save. To cancel, click the circle icon at the bottom to exit without saving.
