# Create DB proxy

1. Select **Database Engine** > select a **Cluster ID** > on **Database Detail**, select the **DB Proxy** tab.
[![](/img/migrated/Db-pro-1-7e7b7ca8.png)](/img/migrated/Db-pro-1-7e7b7ca8.png)
2. In the **DB Proxy Management** panel, select **Create DB Proxy**.
[![](/img/migrated/Db-pro-2-d46a7cbe.png)](/img/migrated/Db-pro-2-d46a7cbe.png)
3. Select **Create DB Proxy** to begin the provisioning process. The system verifies resources, displays a confirmation message, and starts creating the new **DB proxy** based on the information entered.
[![](/img/migrated/Db-pro-3-ac3ae541.png)](/img/migrated/Db-pro-3-ac3ae541.png)

After creation, the system automatically creates the LB cluster within 5 to 7 minutes. The status shows **Processing** during this time and changes to **Running** when the DB proxy is successfully created.

[![](/img/migrated/Db-pro-34-cf0e838d.png)](/img/migrated/Db-pro-34-cf0e838d.png)
