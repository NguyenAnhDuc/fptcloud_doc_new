# Process monitoring

1. Select **Database Engine** > select a **Cluster ID** > on **Database Detail**, select the **Monitoring** tab.

2. Select the process for the action you want to view from the drop-down list.

3. Select **View**. The monitoring information for the selected process is displayed.
