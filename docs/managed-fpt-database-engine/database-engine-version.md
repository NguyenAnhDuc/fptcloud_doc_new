# Database Engine versions

FPT Database Engine provides fully managed database services for multiple database engines supported by FPT, enabling customers to easily deploy, operate, and scale their database systems efficiently. The **Database engine version** section describes the version support policy for these database engines, including version availability, lifecycle management, updates, and end-of-support timelines.
By understanding the version policy, customers can proactively select and manage appropriate database engine versions, ensuring their systems remain secure, stable, and compatible with the latest improvements, while minimizing risks and service disruption during version upgrades or transitions.
