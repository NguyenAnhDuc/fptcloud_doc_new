# Delete Consumer Groups

To delete a consumer group, follow these steps:
**Step 1** : Select the consumer group to delete > Click **Remove**. 
[![](/img/migrated/Pic1_xoaconsumer-d47fd17f.png)](/img/migrated/Pic1_xoaconsumer-d47fd17f.png)
**Step 2** : A confirmation dialog appears, click Yes to Confirm deletion.
**Note** : You cannot delete a consumer group when it is currently active.
