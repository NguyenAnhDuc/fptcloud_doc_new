# Add Share

**Purpose:** Update the permissions of a shared user.
To update a user's permissions, follow these steps:
**Step 1** : Click on a model to view its details > Select the **Share** tab.
**Step 2** : Click the three dots in the **Action column** > Select **Update**.
[![Alt text](/img/migrated/27.-Update-5a4c65b7.png)](/img/migrated/27.-Update-5a4c65b7.png)
**Step 3** : The update form will appear, and you can edit the following information:
[![Alt text](/img/migrated/28.-edit-share-49e2b21f.png)](/img/migrated/28.-edit-share-49e2b21f.png)
[![Alt text](/img/migrated/29.-edit-share-5564d0c2.png)](/img/migrated/29.-edit-share-5564d0c2.png)
  * **Role** : Modify the permissions for the user by changing their role.

_(Images illustrating the update process may be displayed here, showing the statuses of "invited" and "joined" users)_
