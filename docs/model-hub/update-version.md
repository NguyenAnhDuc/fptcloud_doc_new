# Update Version

Edit an existing version of a model.

1. Click on a model to view its details, then select the **Version** tab.

2. In the version list, click the 3-dot icon next to the version you want to edit, then select **Edit**.
[![](/img/migrated/anh-1-709a2381.png)](/img/migrated/anh-1-709a2381.png)

3. The Edit version screen appears. You can modify the version details.

   > **Note:** You can only edit **Version visibility**.

4. After completing your edits, click **Save**.
