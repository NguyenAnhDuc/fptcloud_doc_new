# Model Testing – Interactive Sessions

## Quick Start

Quick start guide for **Model Testing – Interactive Sessions**:

1. [Sign up for an account](./sign-up-for-an-account.md) at <https://ai.fptcloud.com/> or <https://ai.fptcloud.jp/>.
2. Follow the [Step By Step](./step-by-step.md) guide to create and manage sessions.
3. Use the [Playground](./playground.md) to interact with and evaluate the model.
4. View [Completion History](./completion-history.md) to track completed interactions.
