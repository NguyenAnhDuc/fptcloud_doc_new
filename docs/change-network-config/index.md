Change Network Config
## Introduction 
Có nhiều lý do whenến you can need to thay đổi and đặt lại địa chỉ IP tĩnh for thiết bị of mình, chẳng hạn như cấu hình 1 server cố định or need to đặt quy luật for a nhóm máy tính in nội bộ. Một kịch bản nữa là when you thay đổi cấu hình card NIC trên system Cloud is cung cấp bởi FPT Smart Cloud, địa chỉ IP bên in máy will không tự cập nhật theo and you need to cấu hình thủ công (ngoài ra có a số phương pháp khác xử lý in tình huống this nhưng không is đề cập in nội dung bài viết).
**Những điều need to lưu ý before you thay** **đổi information file config network** : 
Đặt địa chỉ IP tĩnh can giúp tránh xung đột network nhưng can whenến a số thiết bị ngừng hoạt động bình thường. Đặt địa chỉ IP tĩnh là a chức năng network nâng cao and need to có kiến thức cơ bản về TCP/IP.
