# Delete Queues

**Step 1** : From the **Application** menu > **Queues** > Choose a desired queue > **Remove**
[![](/img/migrated/queue_delete_1-5046bde7.png)](/img/migrated/queue_delete_1-5046bde7.png)
**Step 2** : A Confirmation dialog will appear. Press **Yes** to delete the queue or **Cancel** to cancel the operation.
[![](/img/migrated/queue_delete_2-3787a6dd.png)](/img/migrated/queue_delete_2-3787a6dd.png)
