# Edit Topics

**Step 1** : From the **Application** menu > **Topics** > Choose a desired topic > **Edit**.
[![](/img/migrated/topic_edit_1-84c0c55b.png)](/img/migrated/topic_edit_1-84c0c55b.png)
**Step 2** : Here, you can either register additional subscriptions or delete existing subscriptions within that topic.
[![](/img/migrated/topic_edit_2-3a07d393.png)](/img/migrated/topic_edit_2-3a07d393.png)
**Step 3** : Click OK to complete.
