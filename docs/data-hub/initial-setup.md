# Initial setup

Complete the following steps before using Data Hub.

## Create an FPT Cloud account

1. Visit [https://ai.fptcloud.com/](https://ai.fptcloud.com/) and click **Sign Up**.
2. Fill in the required information to register.
3. Check your email for instructions from **FPT Smart Cloud** (noreply@fptcloud.com) to complete the registration.

## Access Data Hub

1. Log in to [FPT AI Factory Console](https://ai.fptcloud.com/).
2. In the left menu, select **Data Hub**.
