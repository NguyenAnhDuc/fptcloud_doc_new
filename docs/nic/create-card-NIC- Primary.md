# Create a primary NIC

When an Instance is created, the system automatically generates a primary NIC based on the information you provide.

[![Instance creation screen](/img/migrated/image-1691381396202-7c332ae8.png)](/img/migrated/image-1691381396202-7c332ae8.png)

1. In the menu, select **Compute**, then open the **Instance Management** tab. Select the Instance you want to manage.

[![Instance list](/img/migrated/image-1691381410150-1b799854.png)](/img/migrated/image-1691381410150-1b799854.png)

2. Select the **Network Interface** tab.

   This tab shows all NIC cards for the Instance. If only one card is present, it is the primary NIC by default. For the primary NIC, only the **Disable** action is available.

[![Network Interface tab](/img/migrated/image-1691381424127-1da54a77.png)](/img/migrated/image-1691381424127-1da54a77.png)
