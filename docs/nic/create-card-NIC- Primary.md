# Initialize card Network Interface Card - Primary

Trong quá trình khởi tạo Instance, hệ thống will tự khởi tạo card NIC (Primary) mặc định theo thông tin do you cung cấp.
[![file](/img/migrated/image-1691381396202-7c332ae8.png)](/img/migrated/image-1691381396202-7c332ae8.png)
**Step 1:** Ở menu chọn **Compute** , chọn tab Instance **Management**.
Tại đây will quản lý the Instance, chọn Intance.
[![file](/img/migrated/image-1691381410150-1b799854.png)](/img/migrated/image-1691381410150-1b799854.png)
**Step 2:** Select **Network Interface**
Tại đây will quản lý thông tin the card NIC. Nếu chỉ có 1 card NIC, mặc định that là card Primary. Đối with card Primary, hệ thống chỉ for phép Disable card. [![file](/img/migrated/image-1691381424127-1da54a77.png)](/img/migrated/image-1691381424127-1da54a77.png)
