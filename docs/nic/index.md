Network Interface Card (NIC)
## Overview 
### Introduction khái quát về sản phẩm:
  * Card NIC (Network Interface Card) trên nền tảng Cloud là a thành phần quan trọng to kết nối giữa the virtual machine and tài nguyên trên Cloud. Card NIC cung cấp khả năng kết nối, truyền dữ liệu with hiệu suất cao and đáng tin cậy. Với sự linh hoạt and khả năng mở rộng, Card NIC thatng role quan trọng in việc xây dựng and quản lý hạ tầng network, đồng thời tạo ra môi trường network linh hoạt, mở rộng for the ứng dụng and service dựa trên Cloud.

### Những điều cần lưu ý before you cấu hình card NIC:
  * Đây là feature nâng cao, chỉ dành for người có kiến thức nhất định về xây dựng and quản lý hệ thống network máy tính.
  * Set up network kết nối: Xác định network kết nối for card NIC, đảm bảo virtual machine can giao tiếp and truy cập tài nguyên network a theh hiệu quả.
  * Xác định cấu hình IP: Configure địa chỉ IP, subnet mask, gateway and DNS for card NIC sao for virtual machine can kết nối ando network.
  * Thực hiện kiểm tra thử nghiệm: Sau when cấu hình card NIC, thực hiện kiểm tra thử nghiệm to đảm bảo card hoạt động đúng and không gây ra vấn đề in virtual machine and network.
