View User Group List


You can view and manage the list of User Groups created on the **User Groups Management** page.

To open **User Groups Management**, follow these steps:

On the FPT Portal, in the **IAM** section, select **User Group**. The system will display the User Group list with important information such as:

Name, Description, Roles, User quantity, Created at, Actions
