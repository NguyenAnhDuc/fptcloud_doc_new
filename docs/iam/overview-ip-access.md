# IP Access overview

* The IP access feature controls system access (allows) users with configured IP addresses, other user accounts will not be able to access the system.
  * It is necessary to accurately determine the list of users corresponding to the accessed IPs. 
  * If you are in the installation process, you need to add the public IP address you are accessing and your account (priority is the first record) to ensure the installation process is not interrupted. After initialization, all users outside the installed conditions will not be able to access the system.
