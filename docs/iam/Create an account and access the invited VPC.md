# Create An Account And Access The Invited Vpc

After the Tenant Owner sends an invitation to join the FPT Cloud Portal, the system will send a confirmation email to the email address provided. The invited user needs to open the email and confirm the invitation to activate their account.
  * If the email is not associated with any FPT ID account: The user will need to create an FPT ID account using this email.
  * If the email is already associated with an FPT ID account: The user only needs to activate the FPT Cloud service on that FPT ID account.
