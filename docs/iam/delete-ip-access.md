# Delete an IP access record

1. Click on the record and select the delete action.

   [![](/img/migrated/image-1711511766855-5fa1b9c4.png)](/img/migrated/image-1711511766855-5fa1b9c4.png)

2. Confirm the deletion by typing **delete** in the box, then click **Delete IP access**.

   [![](/img/migrated/image-1711511780469-0177ce56.png)](/img/migrated/image-1711511780469-0177ce56.png)
