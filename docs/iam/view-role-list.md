# View Role List

You can view and manage the list of roles created on the Roles Management page.
To open **Roles Management** , follow these steps:
  * On the FPT Portal, go to the IAM section and select Roles. The system will display a list that includes both basic roles and custom roles with the following details:
    * Name, Type (of the role), Description, Status, Created at.
    * No. of Groups: The number of groups assigned to the role.
    * Actions: Delete, Edit Basic Information, Edit Permissions.
