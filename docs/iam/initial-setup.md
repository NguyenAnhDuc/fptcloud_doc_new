# Initial setup

To start using FPT IAM, complete the following prerequisite tasks.

**Register an account on FPT Portal**

To register an FPT Cloud account, visit the homepage at <https://fptcloud.com/>.

1. Select **Sign Up** and enter the required information as instructed.
2. The support team will contact you to confirm the details and create your account.
3. After logging in with the provided account and password, select the appropriate Tenant to manage permissions.

If you are unsure about the above steps or encounter a system error after three attempts, contact the support team for assistance.

Once complete, you are ready to use FPT IAM.
