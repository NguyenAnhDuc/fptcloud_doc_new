# Invite new IAM users

1. From the menu, select **IAM** > **User**, then choose **Invite users**.

   [![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. In the **Invite User** dialog, enter the required information:

   - **Users**: Enter the email address to invite. This email is used to send the verification email and to access the Cloud Portal.
   - **User group**: Select the User Group to assign (required).

   [![](/img/migrated/image-1734344703145-b92230b6.png)](/img/migrated/image-1734344703145-b92230b6.png)

3. Click **Invite**. The system checks the account and sends the invitation.

   [![](/img/migrated/image-1734344718091-7b7d1882.png)](/img/migrated/image-1734344718091-7b7d1882.png)

Once the invitation succeeds, the new user appears on the User Management page with their **Name**, **Email**, and **User Groups**.
