# Invite new IAM users

1. Select **IAM** > **Users**, then click **Invite users**.

[![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. In the **Invite User** dialog, enter the required information:
   - **Users**: Enter the exact email address to invite. The system will send a verification email to this address.
   - **User group**: Select the User Group to assign to the user (required).

[![](/img/migrated/image-1734344703145-b92230b6.png)](/img/migrated/image-1734344703145-b92230b6.png)

3. Click **Invite**. The system verifies the account and invites the new user into the VPC, then displays the result.

[![](/img/migrated/image-1734344718091-7b7d1882.png)](/img/migrated/image-1734344718091-7b7d1882.png)

After a successful invitation, the new user will appear on the User Management page with their **Name**, **Email**, and **User Groups**.
