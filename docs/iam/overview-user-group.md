# User Group overview

A User Group represents a group of users with the same role. All IAM users in the same User Group share the Roles & Permissions assigned to that group.

- This saves time managing permissions: instead of granting or updating permissions for each IAM user individually, you only need to update the User Group once.
- A User Group can contain multiple IAM users, and an IAM user can belong to multiple User Groups.
