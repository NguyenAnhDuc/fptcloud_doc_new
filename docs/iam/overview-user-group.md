# User Group overview

A **User Group** represents a group of users with the same role. All IAM users within the same User Group are assigned the same roles and permissions.

- Manage permissions for many users at once — update the User Group instead of updating each user individually.
- A User Group can contain multiple IAM users, and an IAM user can belong to multiple User Groups.
