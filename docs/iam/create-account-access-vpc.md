# Create an account and access the invited VPC

After the Tenant Owner sends an invitation, the system sends a confirmation email to the provided address. Open the email and confirm the invitation to activate your account.

- **If the email is not associated with any FPT ID account**: Create a new FPT ID account using this email.
- **If the email is already associated with an FPT ID account**: Activate the FPT Cloud service on that FPT ID account.
