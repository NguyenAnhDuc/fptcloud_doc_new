# Delete User Group

For **User Groups** that are no longer needed, you can delete them by following these steps:
**Step 1** : From the menu, select **IAM** > **User Groups** , and the system will display the **User Groups Management** page.
[![file](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)
**Step 2** : Select **Action** at the end of the User Group you want to delete from the list. Choose **Delete**.
**Note** : You cannot delete a user group that currently has users.
[![file](/img/migrated/image-1734344655938-f07ad3de.png)](/img/migrated/image-1734344655938-f07ad3de.png)
**Step 3** : Confirm the warning information in the popup and click Delete.
[![file](/img/migrated/image-1734344671384-0a15465e.png)](/img/migrated/image-1734344671384-0a15465e.png)
