# Delete a User Group

You can delete User Groups that are no longer needed.

1. From the menu, select **IAM** > **User Groups**.

   [![](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)

2. Select **Action** at the end of the User Group you want to delete, then choose **Delete**.

   [![](/img/migrated/image-1734344655938-f07ad3de.png)](/img/migrated/image-1734344655938-f07ad3de.png)

3. Confirm the warning in the popup and click **Delete**.

   [![](/img/migrated/image-1734344671384-0a15465e.png)](/img/migrated/image-1734344671384-0a15465e.png)

:::note
You cannot delete a User Group that currently has users. Remove all users from the group first.
:::
