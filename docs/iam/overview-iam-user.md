# IAM User overview

An IAM User is a user invited to a Tenant. They may already have an FPT Cloud/SSO account or be a new user.

- If the invited user already has an account, they can immediately access the VPC with the granted permissions after confirming the invitation. If they do not have an account, they are guided to create one.
- If an IAM User needs access to a separate new Tenant, contact the support team to be assigned a Tenant quota.
- To change a user's role or revoke their access to a Tenant, modify the User Group they belong to. For example: if User A is in a User Group with the Reviewer role for Tenant B, remove User A from the group to revoke access, or assign a different role to the group to change their permissions.
