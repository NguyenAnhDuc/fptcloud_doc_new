# 2Fa Configuration

**2FA (Two Factor Authentication)** is a more secure method of account authentication by adding another layer of security beyond the password. A user's account can only successfully log in to the system if it meets both of these authentication steps..
To enable 2-step authentication, do the following:
**Step 1** : Click on the area displaying the account name in the upper right corner of FPT Portal > Select **Account Security**.
[![](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)
**Step 2** : In section**Two Factor Authentication** select**Set up Authenticator Application.**
[![](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)
**Step 3:** Install 2FA on mobile device according to instructions on page**Mobile Authenticator Setup** and select **Submit**.
[![](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)
After successfully configuring 2FA, each time you log in you will be asked to enter an additional OTP generated by the 2FA app..
