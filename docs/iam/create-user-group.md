# Create User Group

**Step 1** : From the menu, select **IAM** > _User Groups_ , then click **Create User Group**.
[![file](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)](/img/migrated/Screenshot-2024-12-21-225822-39855176.png)
**Step 2** : On the **Create User Group** page, enter the required information:
  * **Name** : The name of the user group.
  * **Description** : A description of the user group.
  * **Role** : You can assign multiple roles to a user group.
  * **User** : You can add existing users from the system to the user group.

[![file](/img/migrated/image-1734344557099-e110b8a0.png)](/img/migrated/image-1734344557099-e110b8a0.png)
**Step 3** : Click **Create User Group**. The system will process the request and display the result.
