# Configure 2FA

**Two-Factor Authentication (2FA)** adds a second layer of security beyond your password. Your account can only log in if both authentication steps succeed.

1. Click the account name in the upper-right corner of FPT Portal, then select **Account Security**.

   [![](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)](/img/migrated/Userguide-FPT-IAM-2022-29-7da35c1e.png)

2. In the **Two Factor Authentication** section, select **Set up Authenticator Application**.

   [![](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)](/img/migrated/Userguide-FPT-IAM-2022-31-1024x538-eebf2dbe.png)

3. Install the 2FA app on your mobile device following the instructions on the **Mobile Authenticator Setup** page, then click **Submit**.

   [![](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)](/img/migrated/Userguide-FPT-IAM-2022-32-1024x538-2960f5be.png)

After setup, each login requires an OTP generated by the 2FA app.
