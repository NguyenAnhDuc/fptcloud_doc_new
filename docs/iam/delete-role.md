# Delete a role

You can delete Custom Roles that are no longer needed.

1. From the menu, select **IAM** > **Roles**.
2. Select **Action** at the end of the Custom Role you want to delete, then choose **Delete**.

   [![](/img/migrated/image-1734344452227-353ab722.png)](/img/migrated/image-1734344452227-353ab722.png)

3. Confirm the warning in the popup and click **Delete**.

   [![](/img/migrated/image-1734344485077-982d3c57.png)](/img/migrated/image-1734344485077-982d3c57.png)

:::note
You can only delete a role if it is not assigned to any User Groups.
:::
