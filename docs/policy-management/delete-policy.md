# Delete a policy

To use the delete policy feature, you must log in to the system with Tenant owner permissions.

**Step 1:** In the menu, select **Policy Management**, then select **Actions** > **Delete policy**.

[![file](/img/migrated/image-1719821755207-81a8dba8.png)](/img/migrated/image-1719821755207-81a8dba8.png)

**Step 2:** A warning dialog box appears to confirm the action. Click **Delete policy** to confirm.

[![file](/img/migrated/image-1719821783332-266ee8f8.png)](/img/migrated/image-1719821783332-266ee8f8.png)
