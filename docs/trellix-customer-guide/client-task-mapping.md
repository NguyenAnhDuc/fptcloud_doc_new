# Client Task Mapping

**Step 1** : Select Menu > System Tree > Group > Assigned Client Task
[![file](/img/migrated/image-1765765586369-189ffc4d.png)](/img/migrated/image-1765765586369-189ffc4d.png)
**Step 2** : Users can view scheduled and running tasks on servers where the agent has been successfully installed. [![file](/img/migrated/image-1765765604165-da57bd94.png)](/img/migrated/image-1765765604165-da57bd94.png)
  * Update AMCore Content Package: Update latest signature packages – daily
  * Upgrade Agent – Linux: Check and upgrade to the latest agent version on Linux OS – weekly
  * Upgrade Agent – Windows: Check and upgrade to the latest agent version on Windows OS / Windows Server – weekly
  * Customer – Full Scan: Malware scanning task on servers – weekly
  * Upgrade Endpoint Security – Linux: Upgrade to the latest ENS version on Linux OS – monthly
  * Upgrade Endpoint Security – Windows: Upgrade to the latest ENS version on Windows OS / Windows Server – monthly
  * Linux/Windows Endpoint Security Deploy: Automatically deploy Endpoint Security immediately after agent successfully connects to the server
