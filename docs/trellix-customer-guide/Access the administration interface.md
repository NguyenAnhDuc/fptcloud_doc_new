# Access The Administration Interface

**Step 1** : Log in to the Trellix management console via the domain: <https://am.fptcloud.com:8443> using the username/password provided by FCI.
[![file](/img/migrated/image-1765765382780-8e758eaf.png)](/img/migrated/image-1765765382780-8e758eaf.png)
**Step 2** : After a successful login, the user will access the Trellix management interface.
[![file](/img/migrated/image-1765765391556-388ea73c.png)](/img/migrated/image-1765765391556-388ea73c.png)
  * System Tree: Displays information about servers where the agent has been installed.
    * New Subgroups: Create sub-groups to manage different administration groups

(policies can be created/applied separately for each group/subgroup).
[![file](/img/migrated/image-1765765400751-504e9a39.png)](/img/migrated/image-1765765400751-504e9a39.png)
