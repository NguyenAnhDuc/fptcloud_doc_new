# Using Global Search

1. Log in to [FPT Cloud Portal](https://portal.fptcloud.com).
2. Click the **Search** icon in the top navigation bar.
3. Enter your search keyword (resource name, IP address, etc.).
4. Results are displayed grouped by resource type.
5. Click a result to navigate to the resource details.
