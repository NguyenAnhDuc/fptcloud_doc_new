# Manage Robot Accounts

Robot account là tài khoản dịch vụ dùng để xác thực tự động trong pipeline CI/CD hoặc các hệ thống tích hợp. Thay vì sử dụng tài khoản người dùng, robot account cung cấp thông tin xác thực độc lập với quyền hạn được kiểm soát chặt chẽ.
