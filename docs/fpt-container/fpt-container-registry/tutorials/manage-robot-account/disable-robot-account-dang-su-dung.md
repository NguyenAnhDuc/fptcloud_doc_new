# Disable a robot account

When you need to temporarily suspend access for a robot account without deleting it, you can disable the account. A disabled robot account cannot authenticate until it is re-enabled.

1. In the **Robot Account** list, select the robot account you want to disable, then select **ACTION DISABLE**.

   [![Select ACTION DISABLE](/img/migrated/Picture69-5d9785b0.png)](/img/migrated/Picture69-5d9785b0.png)

2. In the confirmation dialog, enter **DISABLE** to confirm.

   [![Enter DISABLE to confirm](/img/migrated/Picture70-e994d833.png)](/img/migrated/Picture70-e994d833.png)

3. Verify the **Enabled State** after disabling.

   [![Disabled state of robot account](/img/migrated/Picture71-071816c7.png)](/img/migrated/Picture71-071816c7.png)
