# Prerequisites for activating FPT Container Registry

To use FPT Container Registry, you must activate the service on the FPT Portal. Activation requires the appropriate permissions to ensure the security of your organization's resources.

To activate the service, you must be a **Tenant Owner** or a VPC Admin granted the `ContainerRegistry:Active` permission in an IAM role (typically the **SuperAdmin** role). With sufficient permissions, you can activate directly from the FPT Portal interface.
