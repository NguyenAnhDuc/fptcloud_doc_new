# Scale worker nodes

**Step 1:** In the FPT Cloud Portal menu, select **Kubernetes** to open the **Kubernetes Management** page.

[![](/img/migrated/30-c073da78.png)](/img/migrated/30-c073da78.png)

**Step 2:** Select the cluster whose worker node count you want to adjust.

[![](/img/migrated/31-0027f0f8.png)](/img/migrated/31-0027f0f8.png)

**Step 3:** In the **Node Pools** tab, go to **Auto Scale** and click the **Setting** icon.

[![](/img/migrated/32-97ff3240.png)](/img/migrated/32-97ff3240.png)

**Step 4:** In the **Auto Scale Nodes of Cluster** popup, enter the **Min node** and **Max node** values and click **Update**.

[![](/img/migrated/33-89d0a4d3.png)](/img/migrated/33-89d0a4d3.png)

Cluster scaling begins. You cannot interact with the cluster until the process is complete.
