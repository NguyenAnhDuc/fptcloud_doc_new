# Change NFS server configuration

**Step 1:** From the menu, select **Kubernetes**. The **Kubernetes Management** page appears.

[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)

**Step 2:** Select the **Cluster** whose NFS server configuration you want to change.

[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)

**Step 3:** Select the **Advanced** tab. To change the NFS server CPU/RAM flavor, click the Setting icon for the corresponding config.

[![](/img/migrated/Picture42-804cb91c.png)](/img/migrated/Picture42-804cb91c.png)

**Step 4:** Select the desired configuration, then click **Change** to apply.

[![](/img/migrated/Picture43-f4575115.png)](/img/migrated/Picture43-f4575115.png)
