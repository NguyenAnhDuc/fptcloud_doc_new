# Add NFS persistent storage

**Step 1:** From the menu, select **Kubernetes**. The **Kubernetes Management** page appears.

[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)

**Step 2:** Select the **Cluster** you want to add NFS storage to.

[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)

**Step 3:** Select the **Advanced** tab, then in the **NFS Storage** > **Status** section, click the **Setting** icon.

[![](/img/migrated/Picture52-8026ae0a.png)](/img/migrated/Picture52-8026ae0a.png)

**Step 4:** Enter the desired NFS storage capacity (GB), review the information, then click **Enable** to add the NFS storage.

[![](/img/migrated/Picture53-27011f2f.png)](/img/migrated/Picture53-27011f2f.png)
