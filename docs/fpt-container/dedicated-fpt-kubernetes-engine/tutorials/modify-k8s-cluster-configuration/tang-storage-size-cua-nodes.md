# Increase node storage size

**Step 1:** From the menu, select **Kubernetes**. The **Kubernetes Management** page appears.

[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)

**Step 2:** Select the **Cluster** whose node storage you want to expand.

[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)

**Step 3:** Select the **Node Pools** tab. To change the storage size for Master Nodes or Worker Nodes, click the Setting icon for the corresponding config.

[![](/img/migrated/Picture46-45189119.png)](/img/migrated/Picture46-45189119.png)

**Step 4:** Enter the additional capacity you want to add, then click **Extend** to expand the storage.

[![](/img/migrated/Picture47-e9a1ce8b.png)](/img/migrated/Picture47-e9a1ce8b.png)
