# Upgrade Kubernetes version

**Step 1:** From the menu, select **Kubernetes**. The **Kubernetes Management** page appears.
[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)
**Step 2:** Select the **Cluster** you want to upgrade from the list.
[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)
**Step 3:** Select the **Essential Properties** tab, then under **Cluster Information** > **Version**, click the **Setting** icon.
[![](/img/migrated/Picture50-471159a1.png)](/img/migrated/Picture50-471159a1.png)
**Step 4:** Select the version to upgrade to, then click **Upgrade**.
[![](/img/migrated/Picture51-82e15c60.png)](/img/migrated/Picture51-82e15c60.png)

:::note
You can only upgrade to a higher version — downgrading is not supported.
:::
