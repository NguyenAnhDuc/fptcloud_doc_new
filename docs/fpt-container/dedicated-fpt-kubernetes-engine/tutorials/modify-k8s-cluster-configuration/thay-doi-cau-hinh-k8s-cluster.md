# Change K8s cluster configuration

You can make the following configuration changes to an existing Kubernetes cluster in D-FKE:

- Scale worker nodes
- Change node flavor (configuration)
- Change NFS Server flavor (configuration)
- Increase node storage size
- Increase NFS Server storage size
- Upgrade Kubernetes version
- Add NFS Persistent Storage
