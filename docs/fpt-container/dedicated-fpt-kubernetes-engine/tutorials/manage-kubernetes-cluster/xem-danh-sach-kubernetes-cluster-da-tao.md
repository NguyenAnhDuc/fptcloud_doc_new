# List created Kubernetes clusters

The **Kubernetes Management** page lets you view and manage all created Kubernetes clusters.

To open **Kubernetes Management**, select **Kubernetes** from the **FPT Portal** menu. The system displays a list of created clusters with key information including **Id**, **Name**, **Version**, **Nodes**, **Type**, **Status**, and **Created at**.
