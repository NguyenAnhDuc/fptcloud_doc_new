# Change K8s Cluster Configuration

Bạn có thể thực hiện các thay đổi cấu hình sau trên Kubernetes cluster hiện có:

- Thay đổi Cluster Endpoint Access
- Thêm worker group
- Thay đổi Internal Subnet Load Balancer (CIDR)
- Chỉnh sửa label/taint của worker group
- Bật/tắt Node Auto-repair
- Chuyển đổi base worker group
- Nâng cấp phiên bản Kubernetes
- Cấu hình Cluster Autoscale
