# Upgrade Kubernetes version

**Step 1:** From the menu, select **Kubernetes** to open the **Kubernetes Management** page. Select the cluster whose Kubernetes version you want to upgrade.

[![](/img/migrated/Picture41-d322b494.png)](/img/migrated/Picture41-d322b494.png)

**Step 2:** Under **Cluster Information** > **Version**, click the **Setting** icon.

[![](/img/migrated/Picture42-6db86334.png)](/img/migrated/Picture42-6db86334.png)

**Step 3:** Select the version to upgrade to and click **Upgrade**.

[![](/img/migrated/Picture7-3-d45ba609.png)](/img/migrated/Picture7-3-d45ba609.png)

:::note
You can only upgrade to a higher version — downgrading is not supported. To avoid issues, upgrade versions sequentially in order.
:::
