# Update a Load Balancer

In the **Action** menu of the Load Balancer you want to update, select **Update Load Balancer**.

Update any of the following fields:

- **Name**: Enter a new name for the Load Balancer.
- **Description**: Enter a new description.
- **Size**: Select a new Load Balancer package.
- **Public IP**: Assign or change the public IP for internet access.
- **Subnet**: Select the subnet for the Load Balancer. If no specific subnet is required, the system selects any available subnet in the VPC.

Click **Update Load Balancer** to apply the changes, or **Cancel** to discard.
