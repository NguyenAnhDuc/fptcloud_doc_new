# Delete SSL Certificate

To delete an **SSL Certificate**, follow the steps below.

1. On the **SSL Certificate** screen, select the trash icon next to the certificate you want to delete.
2. In the confirmation dialog, select **Delete SSL Certificate** to confirm, or **Cancel** to abort.
