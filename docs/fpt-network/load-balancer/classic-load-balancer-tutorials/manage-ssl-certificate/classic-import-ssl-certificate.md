# Import/create an SSL certificate (Classic)

**Step 1:** Go to the **Manage SSL Certificate for Load balancer** page and click **Import SSL Certificate**.

[![](/img/migrated/image-1712807252615-cef0c042.png)](/img/migrated/image-1712807252615-cef0c042.png)

**Step 2:** Enter the SSL certificate details in the **Import SSL Certificate** popup and click **Save**.

[![](/img/migrated/image-1712808061542-b08fa903.png)](/img/migrated/image-1712808061542-b08fa903.png)
