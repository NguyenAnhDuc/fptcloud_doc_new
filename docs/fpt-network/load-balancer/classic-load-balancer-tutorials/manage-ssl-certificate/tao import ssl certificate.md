# Import SSL Certificate

1. Go to the **Manage SSL Certificate for Load Balancer** page and select **Import SSL Certificate**.

   [![Import SSL Certificate page](/img/migrated/image-1712807252615-cef0c042.png)](/img/migrated/image-1712807252615-cef0c042.png)

2. Enter the SSL certificate details in the **Import SSL Certificate** dialog and select **Save**.

   [![Import SSL Certificate dialog](/img/migrated/image-1712808061542-b08fa903.png)](/img/migrated/image-1712808061542-b08fa903.png)
