# Add Virtual Machines to Load Balancer

If you want to scale a Load Balancer, you can add virtual machines to existing Load Balancers. First create a new virtual machine and install the necessary services in Instance Management, then add the virtual machine to the Load Balancer as follows:

**Step 1:** Go to the **Load Balancer details** page by selecting the Load Balancer name in the **Load Balancer Management** menu.

[![file](/img/migrated/image-1712807219267-78e51205.png)](/img/migrated/image-1712807219267-78e51205.png)

**Step 2:** In the **Instances** tab, the list of virtual machines already added to the Load Balancer is shown. Click **Select instances**.

[![file](/img/migrated/image-1712807226342-138669ae.png)](/img/migrated/image-1712807226342-138669ae.png)

**Step 3:** In the **Change instances** popup, select the virtual machines to add in the **Instances** field.

[![file](/img/migrated/image-1712807234311-3e65aba3.png)](/img/migrated/image-1712807234311-3e65aba3.png)

**Step 4:** Click **Save**.

[![file](/img/migrated/image-1712807240125-47822fc9.png)](/img/migrated/image-1712807240125-47822fc9.png)
