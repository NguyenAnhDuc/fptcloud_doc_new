# View VPN details

1. Click on the VPN name on the VPN management page.

[![VPN list](/img/migrated/Xem-VPN-1-1024x262-72bdfe2b.png)](/img/migrated/Xem-VPN-1-1024x262-72bdfe2b.png)

2. The VPN details appear in the **Status** tab, including:
   - Connection information
   - Error information
   - Traffic
   - Connection instructions and VPN test

To sync at the current time, click the **Refresh** button. By default, the system updates periodically.

[![VPN details](/img/migrated/Xem-chi-tie%CC%82%CC%81t-VPN-1-1024x960-7f67cb07.png)](/img/migrated/Xem-chi-tie%CC%82%CC%81t-VPN-1-1024x960-7f67cb07.png)

3. View connection configuration information in the **Connection Info** tab.

Click **Download Config** to save the configuration to a `.txt` file.

[![VPN connection info](/img/migrated/Xem-chi-tie%CC%82%CC%81t-connection-VPN--0bfbb1a4.png)](/img/migrated/Xem-chi-tie%CC%82%CC%81t-connection-VPN--0bfbb1a4.png)
