# Monitor service through the Monitoring feature

1. On the VPN detail screen, select the **Monitoring** tab from the horizontal menu.

[![Monitoring tab](/img/migrated/image-1761328807530-9f13e19d.png)](/img/migrated/image-1761328807530-9f13e19d.png)

2. Monitor Phase 2 of a specific VPN connection by using the **Filter** option under the **Connection** section.

[![Connection filter](/img/migrated/image-1761328747379-fc1e43b3.png)](/img/migrated/image-1761328747379-fc1e43b3.png)

You can view the Phase 2 status of the service, which includes the following information:

[![Phase 2 status](/img/migrated/image-1761328863701-9de4186d.png)](/img/migrated/image-1761328863701-9de4186d.png)

**VPN connection details:**

| **Field name** | **Description** |
| --- | --- |
| Phase 2 status of connection | Displays the status of the IPSec tunnel (Phase 2). |

You can filter the values for **Local network**, **Remote network**, and **Status** by clicking the filter icon.

| **Field name** | **Description** |
| --- | --- |
| Local Network | Local network subnet range. |
| Remote Network | Remote network subnet range. |
| Status | Current status of the tunnel. |
