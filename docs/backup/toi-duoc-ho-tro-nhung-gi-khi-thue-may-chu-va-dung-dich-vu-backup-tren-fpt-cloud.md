# Tôi is hỗ trợ những gì when thuê server and dùng service Backup trên FPT Cloud?

Ngay when you liên hệ with **FPT Cloud** and đồng ý sử dụng service, you will is tạo 1 account to quản trị tài nguyên trên trang **FPT Portal** and is cung cấp the guide sử dụng.
Trong quá trình khởi tạo tài nguyên and sử dụng service, if có bất cứ vấn đề gì need to hỗ trợ. Bạn chỉ need to liên hệ with bộ phận kỹ thuật, the kỹ thuật viên will hỗ trợ you 24/7.
