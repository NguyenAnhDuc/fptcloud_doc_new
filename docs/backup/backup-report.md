# Backup Report

Chức năng report for phép you thống kê hiện trạng sử dụng backup theo fromng VM trên tất cả the VPC. Các information can lấy includes Backup size, Job Name, VM Name, VPC, Private IP, OS, Backup type, Last success at, Created at.
Để Export Report, you thao tác as follows:
**Step 1:** Ở menu chọn **Backup & Recovery** > **Backup & Recovery Management**. Mở tab **Report**.
**Step 2:** Select **Get Report** > Chọn VPC and Backup type need to lấy > **Show results.**
Bạn can Export Excel to tải về report ở định dạng excel.
