# View lịch sử Backup & Recovery

Hiện tại **FPT Portal** is for phép user xem lại toàn bộ lịch sử thao tác đối with service **Backup & Recovery** như tạo/xóa **Backup Job** , restore/clone Restore Point.
Để xem lại lịch sử **Backup & Recovery**, you hãy thao tác as follows:
Ở menu chọn **Backup & Recovery** > **Backup & Recovery Management**. Mở tab **History**.
**Một số Type phổ biến:**  
| **Type**  | **Mô tả**  |  
| --- | --- |  
| **Create backup schedule**  | Create mới Backup Job  |  
| **Delete backup schedule**  | Delete Backup Job  |  
| **Disable backup schedule**  | Disable Backup Job  |  
| **Enable backup schedule**  | Enable Backup Job  |  
| **Full backup**  | Thực hiện Start Full Backup thủ công  |  
| **Restore**  | Khôi phục lại VM from Restore Point  |  
| **Clone**  | Clone VM mới from Restore Point  |
