##  Introduction 
  * Theo mặc định, máy tính (or virtual machine) will cặp địa chỉ MAC and IP liên kết with nhau nhằm chống giả mạo thông tin. Điều this có nghĩa là with traffic to 1 địa chỉ IP, traffic will is gửi to máy có địa chỉ MAC corresponding. 
  * Tính năng Allow IP address pair for phép thêm cặp địa chỉ IP (or CIDR) and địa chỉ MAC ando 1 interface (card NIC) of instance. Từ that người quản trị can tận dụng feature for nhiều mục đích, đặc biệt là cấu hình HA (High Availability)
