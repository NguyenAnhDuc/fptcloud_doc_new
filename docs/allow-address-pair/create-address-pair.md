# Create Address Pair

Một Network interface card can cấu hình nhiều allow address pair and ngược lại, a address pair can cấu hình for nhiều Network interface card of the Instance khác nhau
**Step 1** : Ở Màn hình quản lý Allow address pair. Select **Create allow address pair**
[![file](/img/migrated/image-1708395038189-f6a186ff.png)](/img/migrated/image-1708395038189-f6a186ff.png)
**Step 2** : Configure allow IP address pair
  * Chọn Instance 
  * Chọn card network to cài đặt Allow address pair
  * Enter IP (or CIDR) to allow

[![file](/img/migrated/image-1708395102833-872f3b79.png)](/img/migrated/image-1708395102833-872f3b79.png)
**Step 3** : Configure allow MAC address pair (optional). Nếu không chọn, mặc định hệ thống will lấy địa chỉ MAC of chỉ Network interface card is is chọn
  * Bấm ando Advanced Setting
  * Enter địa chỉ MAC

[![file](/img/migrated/image-1708395142834-551ea26d.png)](/img/migrated/image-1708395142834-551ea26d.png)
**Step 4** : Bấm Create allow address pair
[![file](/img/migrated/image-1708395173721-87a3a5e8.png)](/img/migrated/image-1708395173721-87a3a5e8.png)
**Step 5** : Check thông tin. Information is cập nhật ở màn hình danh sách Allow address pair and thông tin in Network interface of Instance
[![file](/img/migrated/image-1708395209748-2de1f3be.png)](/img/migrated/image-1708395209748-2de1f3be.png)
[![file](/img/migrated/image-1708395219352-74b14a4a.png)](/img/migrated/image-1708395219352-74b14a4a.png)
