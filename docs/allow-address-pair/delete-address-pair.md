# Delete an address pair

## Method 1: Delete from the Allow address pair management screen

1. Click the delete icon next to the address pair you want to remove.

   [![Delete icon on allow address pair list](/img/migrated/image-1708395674995-65847886.png)](/img/migrated/image-1708395674995-65847886.png)

2. Confirm the deletion.

   [![Confirm deletion dialog](/img/migrated/image-1708395828130-9894ec41.png)](/img/migrated/image-1708395828130-9894ec41.png)

## Method 2: Delete from Instance Management

1. Go to **Network Interface** in Instance Management.

   [![Network Interface screen in Instance Management](/img/migrated/image-1708395867460-1d20cf7a.png)](/img/migrated/image-1708395867460-1d20cf7a.png)

2. Click the X icon next to the address pair.

   [![X icon to remove address pair](/img/migrated/image-1708395893416-0efccabb.png)](/img/migrated/image-1708395893416-0efccabb.png)

3. Confirm the deletion.

   [![Confirm deletion dialog](/img/migrated/image-1708395920272-52eddcc4.png)](/img/migrated/image-1708395920272-52eddcc4.png)
