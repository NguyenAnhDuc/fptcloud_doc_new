# Initial setup

If this is your first time using FPT Cloud, complete the following prerequisites before getting started.

**Create an FPT Cloud account and sign in to FPT Portal**

To start using FPT Cloud services, sign in to the FPT Portal. Go to [https://console.fptcloud.com](https://console.fptcloud.com/) and sign in with your FPT Cloud account credentials. Select the correct Tenant, Region, and VPC for the service you want to use.

To register a new account, select **Sign Up** and fill in the required information as prompted. The support team will contact you to verify your details and activate your account.

:::note
If you encounter any errors during the process, contact our Support team immediately for assistance.
:::
