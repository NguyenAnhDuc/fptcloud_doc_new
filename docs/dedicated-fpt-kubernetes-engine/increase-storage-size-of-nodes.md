# Increase nodes storage size

**Step 1:** In the FPT Cloud Portal menu, select **Kubernetes** to open the **Kubernetes Management** page.

[![](/img/migrated/42-b4f01515.png)](/img/migrated/42-b4f01515.png)

**Step 2:** Select the cluster whose node storage you want to expand.

[![](/img/migrated/43-2cd1e63c.png)](/img/migrated/43-2cd1e63c.png)

**Step 3:** In the **Node Pools** tab, click the **Setting** icon next to **Master Nodes** or **Worker Nodes** to expand storage.

[![](/img/migrated/44-b085f7ac.png)](/img/migrated/44-b085f7ac.png)

**Step 4:** Enter the amount of storage capacity to add and click **Extend Disk**.

[![](/img/migrated/45-16968783.png)](/img/migrated/45-16968783.png)
