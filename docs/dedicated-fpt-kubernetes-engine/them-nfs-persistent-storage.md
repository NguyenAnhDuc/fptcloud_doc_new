# NFS Persistent Storage の追加

**手順 1**: メニューで **Kubernetes** を選択します。**Kubernetes Management** ページが表示されます。

[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)

**手順 2**: 一覧からノード構成を変更したい **Cluster** を選択します。

[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)

**手順 3**: **Advanced** タブを選択し、**NFS Storage** > **Status** セクションで **Setting** アイコンを選択します。

[![](/img/migrated/Picture52-8026ae0a.png)](/img/migrated/Picture52-8026ae0a.png)

**手順 4**: 希望する NFS Storage 容量（GB）を入力して情報を確認し、**Enable** を選択して NFS Storage を追加します。

[![](/img/migrated/Picture53-27011f2f.png)](/img/migrated/Picture53-27011f2f.png)
