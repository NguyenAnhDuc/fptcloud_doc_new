# Backup and restore cluster

To back up a Kubernetes cluster, perform a backup of the master nodes using the **Backup & Recovery** service.

**Step 1:** In the FPT Cloud Portal menu, select **Backup & Recovery** to open the **Backup & Recovery** page.

[![](/img/migrated/63-4d12c51d.png)](/img/migrated/63-4d12c51d.png)

**Step 2:** Click **Create Backup Job**.

[![](/img/migrated/64-72ed6cb2.png)](/img/migrated/64-72ed6cb2.png)

**Step 3:** On the **Backup Job** screen, select all Master Node VMs for the cluster to be backed up. Configure **Retention** and **Schedule** as needed, then click **Create**.

[![](/img/migrated/65-43750a84.png)](/img/migrated/65-43750a84.png)
