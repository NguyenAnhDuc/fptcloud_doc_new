# Increase NFS server storage size

**Step 1** : In the FPT Cloud Portal menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page.
[![](/img/migrated/46-efd3104e.png)](/img/migrated/46-efd3104e.png)
**Step 2** : Choose a Cluster that you want to adjust node’s storage from the list.
[![](/img/migrated/47-2a2dba14.png)](/img/migrated/47-2a2dba14.png)
**Step 3** : In the **Advanced** tab, you can expand the Storage Size of **NFS Server** by clicking the **Setting** icon
[![](/img/migrated/48-58f0a64c.png)](/img/migrated/48-58f0a64c.png)
**Step 4** : Enter the amount of storage capacity to add > Click **Extend Disk** to make changes
[![](/img/migrated/49-42d731db.png)](/img/migrated/49-42d731db.png)
