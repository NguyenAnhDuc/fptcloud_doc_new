# Increase NFS server storage size

**Step 1:** In the FPT Cloud Portal menu, select **Kubernetes** to open the **Kubernetes Management** page.

[![](/img/migrated/46-efd3104e.png)](/img/migrated/46-efd3104e.png)

**Step 2:** Select the cluster whose NFS server storage you want to expand.

[![](/img/migrated/47-2a2dba14.png)](/img/migrated/47-2a2dba14.png)

**Step 3:** In the **Advanced** tab, click the **Setting** icon next to **NFS Server** to expand storage.

[![](/img/migrated/48-58f0a64c.png)](/img/migrated/48-58f0a64c.png)

**Step 4:** Enter the amount of storage capacity to add and click **Extend Disk**.

[![](/img/migrated/49-42d731db.png)](/img/migrated/49-42d731db.png)
