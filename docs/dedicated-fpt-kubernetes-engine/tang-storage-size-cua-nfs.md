# Increase NFS server storage size

**Step 1:** From the menu, select **Kubernetes**. The **Kubernetes Management** page appears.

[![](/img/migrated/Picture30-c657a04c.png)](/img/migrated/Picture30-c657a04c.png)

**Step 2:** Select the **Cluster** whose NFS server storage you want to expand.

[![](/img/migrated/Picture31-f3d03b46.png)](/img/migrated/Picture31-f3d03b46.png)

**Step 3:** Select the **Advanced** tab. To change the NFS server storage size, click the Setting icon for the corresponding config.

[![](/img/migrated/Picture48-973e4b67.png)](/img/migrated/Picture48-973e4b67.png)

**Step 4:** Enter the additional capacity you want to add, then click **Extend** to expand the storage.

[![](/img/migrated/Picture49-e1270d21.png)](/img/migrated/Picture49-e1270d21.png)
