# Change NFS server configuration

**Step 1:** In the FPT Cloud Portal menu, select **Kubernetes** to open the **Kubernetes Management** page.

[![](/img/migrated/38-b0b893d9.png)](/img/migrated/38-b0b893d9.png)

**Step 2:** Select the cluster whose NFS server configuration you want to change.

[![](/img/migrated/39-7ccddcd2.png)](/img/migrated/39-7ccddcd2.png)

**Step 3:** In the **Advanced** tab, click the **Setting** icon next to **NFS Server** to change its flavor (CPU, RAM).

[![](/img/migrated/40-06af27b8.png)](/img/migrated/40-06af27b8.png)

**Step 4:** Choose the desired configuration and click **Change**.

[![](/img/migrated/41-a03a247c.png)](/img/migrated/41-a03a247c.png)
