# Delete a Kubernetes cluster

**Step 1:** In the FPT Cloud Portal menu, select **Kubernetes** to open the **Kubernetes Management** page.

[![](/img/migrated/22-4c7e7f97.png)](/img/migrated/22-4c7e7f97.png)

**Step 2:** Click **Action** for the cluster you want to delete, then click **Delete**.

[![](/img/migrated/23-4396a514.png)](/img/migrated/23-4396a514.png)

**Step 3:** Type **Delete** in the confirmation popup and click **Confirm**.

[![](/img/migrated/24-e402ec48.png)](/img/migrated/24-e402ec48.png)
