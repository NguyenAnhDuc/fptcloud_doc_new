# Add NFS storage

**Step 1:** In the FPT Cloud Portal menu, select **Kubernetes** to open the **Kubernetes Management** page.

[![](/img/migrated/54-847e7eb1.png)](/img/migrated/54-847e7eb1.png)

**Step 2:** Select the cluster to which you want to add NFS storage.

[![](/img/migrated/55-f0c4c08a.png)](/img/migrated/55-f0c4c08a.png)

**Step 3:** In the **Advanced** tab, go to **NFS Storage** > **Status** and click the **Setting** icon.

[![](/img/migrated/56-405c70bf.png)](/img/migrated/56-405c70bf.png)

**Step 4:** Enter the desired NFS storage capacity (GB) and click **Enable**.

[![](/img/migrated/57-1f708a39.png)](/img/migrated/57-1f708a39.png)
