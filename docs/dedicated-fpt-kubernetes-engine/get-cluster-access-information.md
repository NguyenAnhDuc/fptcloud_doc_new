# Get cluster access information

**Step 1** : In the menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page. 
[![](/img/migrated/18-37f7fe3d.png)](/img/migrated/18-37f7fe3d.png)
**Step 2** : Choose the Cluster for which you want to obtain access information.
[![](/img/migrated/19-583eb55b.png)](/img/migrated/19-583eb55b.png)
**Step 3** : In **Essential Properties** > **Cluster Information** > **Configuration** tab, you can download the _.kubeconfig_ file to log in to the Cluster using kubectl or other tools.
[![](/img/migrated/20-ce28c757.png)](/img/migrated/20-ce28c757.png)
**Step 4** : In the **API** section, FPT Portal will display the information of the API URL. Click the **copy** icon to copy.
[![](/img/migrated/21-65accbad.png)](/img/migrated/21-65accbad.png)
FKE utilizes native Kubernetes, allowing users to interact with the cluster using tools like kubectl and dashboard, similar to a regular Kubernetes Cluster.
