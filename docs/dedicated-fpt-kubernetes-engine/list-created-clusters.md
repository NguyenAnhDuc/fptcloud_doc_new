# List created clusters

You can view and manage the list of created Kubernetes clusters on the **Kubernetes Management** page.

In the FPT Portal menu, select **Kubernetes**. The system displays the list of created clusters with the following information: **Id**, **Name**, **Version**, **Nodes**, **Type**, **Status**, and **Created at**.
