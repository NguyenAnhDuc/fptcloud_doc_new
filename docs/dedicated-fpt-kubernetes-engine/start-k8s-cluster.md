# Start a Kubernetes cluster

**Step 1:** In the FPT Cloud Portal menu, select **Kubernetes** to open the **Kubernetes Management** page.

[![](/img/migrated/27-1c29e9f6.png)](/img/migrated/27-1c29e9f6.png)

**Step 2:** Click **Action** for the cluster you want to start, then click **Start**.

[![](/img/migrated/28-f09a3ace.png)](/img/migrated/28-f09a3ace.png)

**Step 3:** Type **Management** in the confirmation popup and click **Confirm**.

[![](/img/migrated/29-0c373207.png)](/img/migrated/29-0c373207.png)
