# Analytics

_User Type: API Provider_
### 1. Analytics
Providers can access the analytics menu to view statistics regarding API usage.
Currently, the Analytics section displays 2 charts:
  * Chart displaying API call counts per consumer.
  * Chart displaying API call counts per API.

### 2. Logging
API Providers can access the logging section to view information regarding API calls made by each consumer, including the API called from which IP address, where the API was invoked, along with details such as time, method, status, base path, etc.
**Step 1** : In the **Application** menu, select **Analytics** > **Logging**.
**Step 2** : Choose a specific consumer to check their log. The system will display logs containing information about the consumer's API calls.
