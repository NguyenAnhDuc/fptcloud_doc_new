# コストエクスプローラーを確認できない

Billingモジュール全般、特にコストエクスプローラーを確認できない原因は3つあります：

**1. アカウントがTenant Ownerロールを持っているが、請求データが空の場合。**

FCI Sale Supportに連絡し、サービス価格の設定とポータルへの表示を依頼してください。

**2. アカウントがTenant Ownerロールを持っていない場合。**

組織のTenant Ownerに[Billing権限設定](../billing/index.md)のガイドに従ってIAM権限の付与を依頼してください。

**3. アカウントに十分なBilling閲覧権限があるが、データが表示されない場合。**

現在選択しているProjectまたはRegionを確認してください。デフォルトProjectと請求対象のリソースRegionを選択してみてください。問題が解決しない場合は、FCIサポートに連絡してください。
