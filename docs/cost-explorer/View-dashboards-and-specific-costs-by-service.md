# View Dashboards And Specific Costs By Service

**Step 1** : In the tab bar menu, select **Billing** > select **Cost Explorer**. The system displays the cost of all services used by ORG in the last 6 months.
[![file](/img/migrated/image-1717143299538-701b2da1.png)](/img/migrated/image-1717143299538-701b2da1.png)
**Step 2** : Drag the scroll bar or scroll the mouse to zoom in/zoom out to one point on the chart to see the data at each time.
[![file](/img/migrated/image-1717143315747-9bf09e34.png)](/img/migrated/image-1717143315747-9bf09e34.png)
