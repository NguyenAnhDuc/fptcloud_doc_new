# Product information

## How to use an AI model and API integration?

On this tab, you can view input parameters, and sample input/output for each AI model and API integration documentation.

[![Product information tab showing input parameters and sample code](/img/migrated/Thiet-ke-chua-co-ten-2-e1754983791648-e6aef34e.png)](/img/migrated/Thiet-ke-chua-co-ten-2-e1754983791648-e6aef34e.png)

## How to see detailed information of a model?

Visit the product detail screen from the home page or via the product URL. On this page, you can find detailed information about the AI model, try the model, and view sample code for the inference API.

[![Model detail page with benchmark results and inference sample code](/img/migrated/detail-8eb65be6.png)](/img/migrated/detail-8eb65be6.png)

## How to know the model's benchmarks?

On this tab, you can view the model's benchmark results for comparison with other models.

[![Benchmark comparison tab showing model performance metrics](/img/migrated/detail-8eb65be6.png)](/img/migrated/detail-8eb65be6.png)

## How to understand and check rate limits?

This page explains the API rate limits, detailing the maximum allowed tokens and requests per minute for each model size to ensure fair and stable system performance.
