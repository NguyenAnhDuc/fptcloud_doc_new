# Xoa Router

Để xóa **Router** , user thực hiện the bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Step 2:** Tại list **Event Gateway** > chọn **Event Gateway** > chọn **tab Security**
**Bước** 3: Tại list **Router** > chọn **Edit** ở Group chứa router need to xóa > chọn icon Delete tại Router need to xóa 
[![Hình ảnh](/img/migrated/image27-bc4dff9c.png)](/img/migrated/image27-bc4dff9c.png)
[![Hình ảnh](/img/migrated/image28-fb9a35ef.png)](/img/migrated/image28-fb9a35ef.png)
**Step 3** : Sau when hoàn tất việc xóa, nhấn **Save** to lưu lại the thay đổi.
