# View Thong tin Event Gateway

Để xem information **Event Gateway** , user thực hiện the bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Step 2:** Tại list **Event Gateway** > chọn **Event Gateway** need to xem information 
  * Tab **Essential Information**
Hiển thị information cơ bản of Event Gateway: **Basic information, Launch Configuration,** **Kafka cluster, Endpoint**
[![Hình ảnh](/img/migrated/image10-60575f21.png)](/img/migrated/image10-60575f21.png)
  * Tab **Security**
Hiển thị information list Security of **Event Gateway**
[![Hình ảnh](/img/migrated/image18-37e84895.png)](/img/migrated/image18-37e84895.png)
  * Tab **Router group**
Hiển thị information list **Router group** of **Event Gateway**
[![Hình ảnh](/img/migrated/image19-cfa604c3.png)](/img/migrated/image19-cfa604c3.png)
  * Tab **Router**
Hiển thị information list the **Router** theo group of **Event Gateway**
