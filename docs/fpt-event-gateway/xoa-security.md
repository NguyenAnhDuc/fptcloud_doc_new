# Xoa Security

Để xóa Security, user thực hiện the bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Step 2:** Tại danh sách **Event Gateway** > chọn **Event Gateway** > chọn **tab Security**
**Bước** 3: Tại danh sách **Security** > chọn **Security** cần xóa > chọn **Action** > chọn **Delete**
[![Hình ảnh](/img/migrated/image1f-77b61a24.png)](/img/migrated/image1f-77b61a24.png)
**Step 3** : Hiển thị hộp thoại **Delete Application** > chọn **Delete** to xóa hoàn thành việc xóa **Security**
[![Hình ảnh](/img/migrated/image20-e743639c.png)](/img/migrated/image20-e743639c.png)
