# View Thong tin Router Group

Để xem information Router group, user thực hiện the bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Step 2:** Tại list **Event Gateway** > chọn **Event Gateway** > chọn **tab Security**
**Bước** 3: Tại list **Router group** > chọn **Router group** need to xem information details
