# View Thong tin Router Group

Để xem thông tin Router group, user thực hiện the bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Step 2:** Tại danh sách **Event Gateway** > chọn **Event Gateway** > chọn **tab Security**
**Bước** 3: Tại danh sách **Router group** > chọn **Router group** cần xem thông tin chi tiết
