# Xoa Router Group

Để xóa **Router group** , user thực hiện the bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**
**Step 2:** Tại danh sách **Event Gateway** > chọn **Event Gateway** > chọn **tab Router group**
**Bước** 3: Tại danh sách **Router group** > chọn **group** cần xóa > chọn **Action** > chọn **Delete**
[![Hình ảnh](/img/migrated/image25-3bec02b1.png)](/img/migrated/image25-3bec02b1.png)
**Step 3** : Hiển thị hộp thoại **Delete Application** > chọn **Delete** to xóa hoàn thành việc xóa **Security**
[![Hình ảnh](/img/migrated/image26-fb5457d4.png)](/img/migrated/image26-fb5457d4.png)
