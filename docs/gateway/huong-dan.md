# Huong Dan

Step 1: Đăng nhập console.fptcloud.com
Step 2: Sau khi đăng nhập thành công, chọn VPC cần xem dữ liệu Gateway
Step 3: Ở sidebar, chọn và mở rộng menu Network → Chọn Menu Gateway
Note: Tính năng này chỉ dành riêng cho một số Tenant có cấu hình đặc biêt, vui lòng liên hệ để được hỗ trợ.
