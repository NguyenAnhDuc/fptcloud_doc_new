# Huong Dan

Step 1: Log in console.fptcloud.com
Step 2: Sau when đăng nhập successfully, chọn VPC cần xem dữ liệu Gateway
Step 3: Ở sidebar, chọn and mở rộng menu Network → Chọn Menu Gateway
Note: Tính năng this chỉ dành riêng for a số Tenant có cấu hình đặc biêt, vui lòng liên hệ to is hỗ trợ.
