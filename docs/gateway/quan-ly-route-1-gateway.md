# Managing Gateway Routes

## Viewing the Route List

[![Route list screen](/img/migrated/image-1754290897920-2416f962.png)](/img/migrated/image-1754290897920-2416f962.png)

The information displayed for each Route includes:

- **Route name**
- **CIDR Destination**
- **IP next hop**
- **Status**
- **Created at**

## Creating a Route

Click the **Create route** button to create a new Route.

[![Create Route form](/img/migrated/image-1754290924141-cfae2a04.png)](/img/migrated/image-1754290924141-cfae2a04.png)

:::note
**Basic** type Gateways do not support Route creation. You can only create Routes when the Gateway is of type **Custom**.
:::

## Editing a Route

In the **Actions** column, click the **Edit** button to edit the Route information.

:::note
**Basic** type Gateways do not support Route editing. You can only edit Routes when the Gateway is of type **Custom**.
:::

## Deleting a Route

In the **Actions** column, click the **Delete** button to delete the Route.

:::note
**Basic** type Gateways do not support Route deletion. You can only delete Routes when the Gateway is of type **Custom**.
:::
