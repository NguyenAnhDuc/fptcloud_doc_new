# 1. Manage danh sách Route

## **1. Manage danh sách Route**
[![file](/img/migrated/image-1754290897920-2416f962.png)](/img/migrated/image-1754290897920-2416f962.png)
Các thông tin of 1 Route includes:
  * Route name
  * CIDR Destination
  * IP next hop
  * Status
  * Created at 

## **2 Create Route**
Người dùng can bấm nút **Create route** to tạo mới 1 Route
[![file](/img/migrated/image-1754290924141-cfae2a04.png)](/img/migrated/image-1754290924141-cfae2a04.png)
  * Với gateway loại Basic will không is tạo Route, chỉ is tạo Route when Gateway loại Custom

## **3 Chỉnh sửa Route**
Ở cột Actions, user can bấm nút Edit to chỉnh sửa thông tin of Route
  * Với gateway loại Basic will không is chỉnh sửa Route, chỉ is chỉnh sửa Route when Gateway loại Custom

## **4 Xoá Route**
Ở cột Actions, user can bấm nút Delete to chỉnh sửa thông tin of Route
  * Với gateway loại Basic will không is xoá Route, chỉ is xoá Route when Gateway loại Custom
