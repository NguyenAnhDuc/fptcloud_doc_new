# Tính năng chính

Cloud Advisor là service of FPT Cloud cung cấp the khuyến nghị tối ưu hóa for hạ tầng cloud of you, includes the lĩnh vực:

- **Chi phí (Cost)**: Phân tích and đề xuất theh giảm cost sử dụng cloud
- **Hiệu suất (Performance)**: Tối ưu hóa hiệu năng of the tài nguyên
- **Bảo mật (Security)**: Đánh giá and cải thiện tình trạng security
- **Độ tin cậy (Reliability)**: Đảm bảo tính sẵn sàng and khả năng phục hồi of system

## Tính năng chính

- Phân tích tự động the tài nguyên cloud is sử dụng
- Đưa ra khuyến nghị cụ thể to tối ưu hóa
- Dashboard trực quan to theo dõi status
- Báo cáo details theo fromng danh mục

## Bắt đầu sử dụng

Truy cập Cloud Advisor from [FPT Cloud Console](https://console.fptcloud.com) to bắt đầu nhận the khuyến nghị tối ưu hóa for hạ tầng of you.
