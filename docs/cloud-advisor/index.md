# Cloud Advisor

Cloud Advisor is an FPT Cloud service that provides optimization recommendations for your cloud infrastructure across the following areas:

- **Cost**: Analyze and suggest ways to reduce cloud spending
- **Performance**: Optimize resource performance
- **Security**: Assess and improve your security posture
- **Reliability**: Ensure availability and recoverability of your systems

## Key features

- Automatic analysis of cloud resources in use
- Specific recommendations for optimization
- Visual dashboard for status monitoring
- Detailed reports by category

## Getting started

Access Cloud Advisor from the [FPT Cloud Console](https://console.fptcloud.com) to start receiving optimization recommendations for your infrastructure.
