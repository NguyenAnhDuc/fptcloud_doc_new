# Introduction

FPT Cloud Advisor là công cụ giúp khách hàng:

- Đánh giá mức độ an toàn, hiệu năng and tối ưu cost of tài nguyên cloud
- Phát hiện rủi ro cấu hình sai
- Đưa ra khuyến nghị (recommendations) to cải thiện hệ thống
