# Zalo ticket support

This guide explains how to create FPT Cloud support tickets via the Zalo OA channel or directly on the support portal: [https://support.fptcloud.com](https://support.fptcloud.com)

## Create a support ticket via Zalo OA

Follow the official FPT Cloud account (OA) on Zalo by using the Zalo app to scan the QR code, then select **Follow**.

[![Zalo OA QR code](/img/migrated/image-1702289152401-f56c10ef.png)](/img/migrated/image-1702289152401-f56c10ef.png)

Screen displayed after following successfully:

## Submit a support request from FPT Cloud Portal

**Step 1:** Access the FPT Cloud support page.

**Option 1:** Go directly to [https://support.fptcloud.com](https://support.fptcloud.com/). Log in with your registered FPT Cloud account.

[![Login page](/img/migrated/image-1702289190502-0108bf6c.png)](/img/migrated/image-1702289190502-0108bf6c.png)

Select **Submit a ticket** to create a support request.

[![Submit a ticket](/img/migrated/image-1702289199279-8b54294a.png)](/img/migrated/image-1702289199279-8b54294a.png)

**Option 2:** Via FPT Cloud Zalo OA — type `/support` in the official support group chat or chat directly with the FPT Cloud Official Account. Zalo will send you a link to the support page.

[![Zalo support](/img/migrated/image-1702289383339-7cfae5fb.png)](/img/migrated/image-1702289383339-7cfae5fb.png) [![Support link](/img/migrated/image-1702289397554-222cc3fe.png)](/img/migrated/image-1702289397554-222cc3fe.png)

After clicking the link, log in to create a ticket.

[![Support portal](/img/migrated/image-1702289421857-a00d06a2.png)](/img/migrated/image-1702289421857-a00d06a2.png)

**Step 2:** Fill in the required information.

- **Subject:** Title of the support request.
- **Organization name:** Your tenant name on FPT Cloud.
- **I need support for:** Select the service you need help with.
- **Priority level:**
  - **Urgent:** System interrupted, inaccessible, or lost internet connectivity.
  - **High:** System slow, intermittent, or significantly impacting service.
  - **Medium:** Low impact — system can still operate normally.
  - **Low:** Service inquiries or basic support requests.
- **Description:** Describe the issue in detail.

[![Ticket form](/img/migrated/image-1702289433410-a3de407d.png)](/img/migrated/image-1702289433410-a3de407d.png)

After filling in the form, click **Submit**. You will receive a confirmation email.

[![Confirmation](/img/migrated/image-1702289454316-840e1c55.png)](/img/migrated/image-1702289454316-840e1c55.png)

**Step 3:** Enable automatic ticket status notifications via Zalo.

In the confirmation email, click the link at the bottom. Scan the QR code with the Zalo app, then select **Allow**.

[![Zalo notification setup](/img/migrated/image-1702289472711-917181f5.png)](/img/migrated/image-1702289472711-917181f5.png)

You will receive automatic ticket status updates via Zalo chat.

[![Zalo notification](/img/migrated/image-1702289485901-918b287b.png)](/img/migrated/image-1702289485901-918b287b.png)

## Check ticket status

- The system sends automatic status updates to your Zalo if you completed the notification setup.
- You can also check ticket status directly on the [FPT Cloud Support portal](https://support.fptcloud.com).
