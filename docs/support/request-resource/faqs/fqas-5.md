# Why can't I click Submit to send the ticket?

Please check the following information:

  * The Title and Request type fields must be filled in completely.
  * You must select at least 1 service and must select the Quantity you wish to request for that service.
  * The Quantity for each service must be a whole number with a value greater than or equal to 1.
  * A valid Start date must be a date after the current date when the request ticket is created.
  * A valid End date must be a date after the Start date (if provided) or after the current date the ticket is created (if no Start date is selected).
