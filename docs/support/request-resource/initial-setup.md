# Initial Setup

If this is your first time using FPT Cloud Server, please check and complete the following steps first:

  * Contact the FPT Cloud Sales/Support team to create and activate at least one Commercial contract.
  * Be provided with an account to access the FPT Portal with the role of Tenant Owner or VPC Owner.

After completing the above steps, you are ready to create a resource request on the FPT Cloud Server system.
