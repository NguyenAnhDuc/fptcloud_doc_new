# Create a Resource Request

**Step 1**: In the tab bar, select the **Tenant** for which you need to provision resources. In the menu bar, select **Request Resource**, then select **Create request**.

[![file](/img/migrated/image-1703744342441-a0f93b7b.png)](/img/migrated/image-1703744342441-a0f93b7b.png)

**Step 2**: Fill in the information in the resource request form.

  * Required fields include: Title, Request type, Service, Quantity.
  * The remaining fields are optional.

[![file](/img/migrated/image-1703744369903-d102105e.png)](/img/migrated/image-1703744369903-d102105e.png)
[![file](/img/migrated/image-1703744387522-1dd6ab81.png)](/img/migrated/image-1703744387522-1dd6ab81.png)

**Step 3**: Select **Submit request** to send the request to FCI. The system will send a confirmation email acknowledging receipt of the request to the creator, and simultaneously notify the FCI Sales/Support team to process the request.

[![file](/img/migrated/image-1703744428466-add11d9d.png)](/img/migrated/image-1703744428466-add11d9d.png)
