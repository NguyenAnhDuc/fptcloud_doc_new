# Delete Topics

In case you want to delete a topic that is no longer in use, you can follow these instructions:
**Step 1** : From the menu, navigate to **Application** > **Topics**.
**Step 2** : Choose a topic to delete > Click **Remove**.
[![](/img/migrated/Pic1_Remove-a0d961bc.png)](/img/migrated/Pic1_Remove-a0d961bc.png)
**Step 3** : Click **Yes** to confirm deletion.
[![](/img/migrated/Pic2_remove-354c9e9d.png)](/img/migrated/Pic2_remove-354c9e9d.png)
