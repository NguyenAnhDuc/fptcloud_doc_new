# Delete Router

To delete a **Router**, follow the steps below:

**Step 1:** In the menu bar, select **Integration** > **Event Gateway**.

**Step 2:** In the **Event Gateway** list, select an **Event Gateway** > select the **Security** tab.

**Step 3:** In the **Router** list, click **Edit** on the Group containing the Router to delete > click the Delete icon on the Router to remove.

[![Router list in edit mode with delete icon visible](/img/migrated/image27-bc4dff9c.png)](/img/migrated/image27-bc4dff9c.png)

[![Router selected for deletion](/img/migrated/image28-fb9a35ef.png)](/img/migrated/image28-fb9a35ef.png)

**Step 4:** After completing the deletion, click **Save** to save the changes.

[![Save confirmation after router deletion](/img/migrated/image29-579cd85c.png)](/img/migrated/image29-579cd85c.png)
