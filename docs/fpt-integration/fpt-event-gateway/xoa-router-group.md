# Delete Router Group

To delete a **Router Group**, follow the steps below:

**Step 1:** In the menu bar, select **Integration** > **Event Gateway**.

**Step 2:** In the **Event Gateway** list, select an **Event Gateway** > select the **Router group** tab.

**Step 3:** In the **Router group** list, select the group to delete > click **Action** > select **Delete**.

[![Router Group list with Action menu open](/img/migrated/image25-3bec02b1.png)](/img/migrated/image25-3bec02b1.png)

**Step 4:** The **Delete Application** dialog appears > click **Delete** to complete the deletion of the **Router Group**.

[![Delete Router Group confirmation dialog](/img/migrated/image26-fb5457d4.png)](/img/migrated/image26-fb5457d4.png)
