# Delete Security

To delete a **Security** configuration, follow the steps below:

**Step 1:** In the menu bar, select **Integration** > **Event Gateway**.

**Step 2:** In the **Event Gateway** list, select an **Event Gateway** > select the **Security** tab.

**Step 3:** In the **Security** list, select the **Security** entry to delete > click **Action** > select **Delete**.

[![Security list with Action menu open](/img/migrated/image1f-77b61a24.png)](/img/migrated/image1f-77b61a24.png)

**Step 4:** The **Delete Application** dialog appears > click **Delete** to complete the deletion of the **Security** entry.

[![Delete Security confirmation dialog](/img/migrated/image20-e743639c.png)](/img/migrated/image20-e743639c.png)
