# FPT Event Gateway

**Event Gateway** is an intermediary component that receives, processes, and routes events in real time between systems. It acts as a **centralized event data gateway**, enabling flexible connectivity between event sources and processing systems such as Kafka, webhooks, or analytics pipelines.

With support for **event filtering, transformation, and routing**, Event Gateway is well-suited for event-driven architectures and is commonly used in **real-time analytics**, **monitoring**, **IoT**, and distributed systems that require reliable, controlled event data processing.
