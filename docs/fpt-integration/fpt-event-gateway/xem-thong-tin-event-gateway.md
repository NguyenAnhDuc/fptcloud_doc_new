# View Event Gateway Information

To view **Event Gateway** information, follow the steps below:

**Step 1:** In the menu bar, select **Integration** > **Event Gateway**.

**Step 2:** In the **Event Gateway** list, select the **Event Gateway** whose information you want to view.

- **Essential Information** tab

  Displays basic information about the Event Gateway: **Basic information**, **Launch Configuration**, **Kafka cluster**, **Endpoint**.

[![Essential Information tab showing Event Gateway details](/img/migrated/image10-60575f21.png)](/img/migrated/image10-60575f21.png)

- **Security** tab

  Displays the list of Security configurations for the **Event Gateway**.

[![Security tab showing JWT auth list](/img/migrated/image18-37e84895.png)](/img/migrated/image18-37e84895.png)

- **Router group** tab

  Displays the list of **Router groups** for the **Event Gateway**.

[![Router group tab showing router group list](/img/migrated/image19-cfa604c3.png)](/img/migrated/image19-cfa604c3.png)

- **Router** tab

  Displays the list of **Routers** grouped by Router group for the **Event Gateway**.

[![Router tab showing routers organized by group](/img/migrated/image1a-f6ba0423.png)](/img/migrated/image1a-f6ba0423.png)
