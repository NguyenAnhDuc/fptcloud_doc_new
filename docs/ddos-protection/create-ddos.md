# Create DDoS Protection

[![DDoS Protection not yet activated](/img/migrated/corero-not-active-1024x462-fa6af4dd.png)](/img/migrated/corero-not-active-1024x462-fa6af4dd.png)

:::note
If you are unable to activate DDoS Protection yourself on the FPT Cloud Portal, contact support:
- Hotline: **1900 638 399**
- Email: **support@fptcloud.com**
:::

After activation, the following features become available.

[![DDoS Protection activated](/img/migrated/corero-active-1024x462-24b1a42c.png)](/img/migrated/corero-active-1024x462-24b1a42c.png)

The system lets you manage the list of protected IPs across all IPs in a Tenant.

[![Protected IP list](/img/migrated/Create-ip-1-1024x416-b730e124.png)](/img/migrated/Create-ip-1-1024x416-b730e124.png)

## Add an IP to protect

Click **Create IP Protection** and fill in the following fields:

- **Traffic Protection**: The protection traffic package for the IP.
- **VPC**: Select a VPC to filter the corresponding Floating IP list.
- **Floating IP**: Select the Floating IP to protect.

:::tip
You can select a VPC first to filter the Floating IP list, or select a Floating IP first and the system will automatically fill in the corresponding VPC.
:::
