# Manage App Catalogs

App Catalogs allow you to manage app catalogs by repository. FPT Cloud supports deploying app catalogs from Bitnami, Grafana, and FPT Cloud-managed chart repositories. You can search for a chart and deploy it to a cluster that has been added to the app catalog.
