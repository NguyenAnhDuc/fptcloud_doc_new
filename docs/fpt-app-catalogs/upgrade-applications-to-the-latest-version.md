# Upgrade Applications To The Latest Version

The system allows upgrading the application version to the latest version available from the repository. However, users need to be mindful of potential incompatibility when performing the upgrade.
To do so, follow these steps
**Step 1** : From the application detail page, click **Upgrade to v** (the button only appears when the current application version are not the latest version).
[![](/img/migrated/Picture33-1-a6c06913.png)](/img/migrated/Picture33-1-a6c06913.png)
**Step 2** : Adjust **User Define Values** if desired.
[![](/img/migrated/Picture34-1-e7b42635.png)](/img/migrated/Picture34-1-e7b42635.png)
**Step 3** : Click **Confirm** to perform the chart upgrade
_**Note** : Chart upgrades may pass or fail depend on the cluster that the chart has been installed to. Depending on the chart version, supported K8s cluster versions might differ._
