# Proxy IP

Chức năng thực hiện thiết lập địa chỉ proxy IP if có sử dụng nhằm thiết lập the service of ip private in cloud [![file](/img/migrated/image-1714112480774-859e0576.png)](/img/migrated/image-1714112480774-859e0576.png)
  * Các chức năng can sử dụng includes thêm, sửa, xoá Proxy IP

  1. Add IP Proxy
     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn ando biểu tượng + to thêm Proxy IP
     * Enter ando the giá trị cần thiết to thiết lập IP như hình and chọn Apply [![file](/img/migrated/image-1714112494432-ca5d7a26.png)](/img/migrated/image-1714112494432-ca5d7a26.png)
     * Select **Save** to lưu thay đổi [![file](/img/migrated/image-1714112501548-e3275d5b.png)](/img/migrated/image-1714112501548-e3275d5b.png)
  2. Edit IP Proxy
     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn ando Modify
     * Thực hiện sửa đổi the thông tin cần thiết: [![file](/img/migrated/image-1714112513079-760a45f0.png)](/img/migrated/image-1714112513079-760a45f0.png)
     * Nhấn chọn Apply and chọn Save to lưu thiết lập [![file](/img/migrated/image-1714115110737-1154d081.png)](/img/migrated/image-1714115110737-1154d081.png)
  3. Xoá IP Proxy
     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn ando ip cần xoá [![file](/img/migrated/image-1714115123188-4dfc9591.png)](/img/migrated/image-1714115123188-4dfc9591.png)
     * Chọn ando biểu tượng thùng rác to thực hiện xoá ip has been chọn and chọn **Save** nhằm thực hiện lưu lại thiết lập. [![file](/img/migrated/image-1714115133281-726201bd.png)](/img/migrated/image-1714115133281-726201bd.png)
