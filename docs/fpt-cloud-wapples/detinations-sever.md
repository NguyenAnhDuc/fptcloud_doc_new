# Destination Server

Chức năng thực hiện setup web service from the ip host nằm in cloud [![file](/img/migrated/image-1714121626519-4364137e.png)](/img/migrated/image-1714121626519-4364137e.png)
  * Thêm/sửa/xóa Destination server [![file](/img/migrated/image-1714719775318-5890dbc9.png)](/img/migrated/image-1714719775318-5890dbc9.png)
  * Mode: Proxy
  * Proxy IP: IP Proxy has been is setup
  * Proxy Port: port proxy is chuyển đổi to sử dụng from port gốc
  * Web Server IP (Domain): ip or domain service of webserver
  * Web Server Port: port service of webserver
  * Health Check: kiểm tra status of webserver

[![file](/img/migrated/image-1714719898104-ec1e303d.png)](/img/migrated/image-1714719898104-ec1e303d.png)
  * Gateway: IP Gateway is setup
  * VLAN ID: Hiển thị VLAN ID is nhập. Trống if không dùng SSL
  * SSL Use: Hiển thị X if không dùng, O if sử dụng
  * SSL Termination: Hiển thị X if không dùng, O if dùng
  * Certificate Expiration: Hiển thị hạn of chứng chỉ is chọn if sử dụng SSL
