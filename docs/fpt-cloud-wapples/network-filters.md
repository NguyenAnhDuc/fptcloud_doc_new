# Sử dụng to cấu hình network (tương tự security group)

Sử dụng to cấu hình network (tương tự security group)
  * Các trường displayed
  * Filter No.: Hiển thị số hiệu of policy
  * Source IP: ip truy cập
  * Source port: port truy cập
  * Source netmask: subnet is truy cập
  * Destination IP: ip đích service is truy cập to
  * Destination Port: port đích service
  * Destination Netmask: subnet đích is truy cập 
  * Protocol: Giao thức truy cập
  * Connection: Hành động thực hiện when phát hiện kết nối(Allow, Deny, Detection)
