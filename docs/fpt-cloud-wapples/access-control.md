# Access Control

TÍnh năng kiểm soát truy cập nhằm phòng chống các hành vi không hợp lệ như spamming, clicking
  * Add Access Control [![file](/img/migrated/image-1714116678455-4855ca28.png)](/img/migrated/image-1714116678455-4855ca28.png)
  * Thiết lập tên 
  * Thiết lập source ip
  * Thiết lập destination ip
  * Thiết lập các thông tin destination (Host: domain, Port, Method: Get Post..., URL: subdomain) [![file](/img/migrated/image-1714116684905-93057588.png)](/img/migrated/image-1714116684905-93057588.png)
  * Thiết lập thời gian ngăn truy cập từ lúc có hành vi, số hành vi tối đa để thực hiện block, quãng thời gian cho số hành vi tối đa được phép
  * Tách các filter được chọn trong [Source IP/Destination IP/URL/Host/Method] bằng dấu ‘,’.
