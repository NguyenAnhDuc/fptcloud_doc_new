# View Backup & Recovery History

**FPT Portal** allows users to view the complete operation history for the **Backup & Recovery** service, such as creating/deleting **Backup Jobs**, restoring/cloning Restore Points.

To view the **Backup & Recovery** history, follow these steps:

In the menu, select **Backup & Recovery** > **Backup & Recovery Management**. Open the **History** tab.

**Common operation types:**

| **Type** | **Description** |
| --- | --- |
| **Create backup schedule** | Create a new Backup Job |
| **Delete backup schedule** | Delete a Backup Job |
| **Disable backup schedule** | Disable a Backup Job |
| **Enable backup schedule** | Enable a Backup Job |
| **Full backup** | Manually start a Full Backup |
| **Restore** | Restore a VM from a Restore Point |
| **Clone** | Clone a new VM from a Restore Point |
