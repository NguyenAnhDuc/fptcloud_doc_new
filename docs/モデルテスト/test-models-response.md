# Test models response

Select a session to view the model's response. The system opens the test playground where you can interact with the model.

[![file](/img/migrated/image-1746694482798.30.25-2511f2f2.png)](/img/migrated/image-1746694482798.30.25-2511f2f2.png)

### **1. System message**

Instructions that set the behavior and tone of the AI model. You can guide how the model responds throughout the conversation.

The system message can be changed based on your preference. Default: _"You are a helpful assistant"_

**Example:** _You are a cloud computing expert supporting users with technical issues._

### **2. User message**

The input or question the user provides to the AI. This is the main prompt the model responds to in the conversation.

**Example:** _What are the advantages of using a private cloud?_

Uploading images in **.jpeg** and **.jpg** format is supported for testing with VLM (Vision-Language Models). List of models that support image testing:

### **3. Settings**

- **Temperature:** The level of creativity permitted in answers, typically ranging from **0 to 2**.

Default: **1**

- **Low temperature (close to 0):**
  - The model generates more **predictable and deterministic** responses.
  - Prioritizes high-probability words and tokens, producing more **focused and precise** output.
- **High temperature (above 1):**
  - The model generates more **creative**, diverse, and **unexpected** responses.
  - Samples from a broader range of words, so output is **more varied** but may be less accurate.

- **Advanced settings:**
  - Add stop sequences: Stop sequences let you specify one or more sequences that will stop the model when detected, allowing you to control the **length and content** of the generated text.
  - Output length: Controls the length of text generated by the model, setting the maximum number of tokens (words or subwords) the model can generate in response to a prompt. Default: **8192**
  - Top-P: Used in generative models to manage the randomness and diversity of generated text, functioning as an alternative to temperature when sampling from the model's output distribution. Default: **0.95**

Example output model settings:

- Add stop sequence: third-party
- Output length: 1000
- Top-P: 0.95
