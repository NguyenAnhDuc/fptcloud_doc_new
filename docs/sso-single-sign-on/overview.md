# Overview

## 1. Overview

SSO (Single Sign-On) is an authentication mechanism that lets users securely access multiple applications and services using a single ID. With SSO, you can access many applications at once without logging in each time. SSO is built on the concept of federated identity, which allows identity information to be shared across trusted but independent systems.

## 2. Benefits

Some benefits of Single Sign-On:

- Fewer usernames and passwords to remember
- Fewer times you need to enter login credentials
- Reduced risk of user credential exposure
