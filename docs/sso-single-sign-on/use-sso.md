# Sử dụng feature SSO

Sau when cấu hình SSO, user can đăng nhập ando system theo 2 theh:
#### **Cách 1: Vào bằng đường dẫn**
**Step 1:** Access đường dẫn: 
<http://sub-domain.fptcloud.com>
**Step 2:** Xác thực information đăng nhập
[![file](/img/migrated/image-1734056582232-ddd3fa5b.png)](/img/migrated/image-1734056582232-ddd3fa5b.png)
**Step 3:** Hệ thống xác thực đăng nhập and chuyển hướng user ando trang quản lý instance
#### **Cách 2: Vào bằng trang select-method**
**Step 1:** Truy cập đường dẫn: <https://console.fptcloud.com/login/select-method>
**Step 2:** Chọn "**Login with your organization** "
[![file](/img/migrated/image-1734056612511-805a1ea0.png)](/img/migrated/image-1734056612511-805a1ea0.png)
**Step 3:** Enter sub-domain user has been đăng kí
[![file](/img/migrated/image-1734056636838-3134f1b3.png)](/img/migrated/image-1734056636838-3134f1b3.png)
**Step 4:** Xác thực information đăng nhập
[![file](/img/migrated/image-1734056657577-d806a4b5.png)](/img/migrated/image-1734056657577-d806a4b5.png)
**Step 5:** Hệ thống xác thực đăng nhập and chuyển hướng user ando trang quản lý instance
