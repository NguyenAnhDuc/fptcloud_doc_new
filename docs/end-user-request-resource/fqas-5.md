# Vì sao tôi cannot chọn submit to gửi ticket?

Bạn hãy kiểm tra the information sau: 
  * Các trường information Title, Request type must is điền đầy đủ. 
  * Bạn need to must chọn tối thiểu 1 service and must chọn Quantity (số lượng) mà you muốn yêu cầu for service that.
  * Quantity of mỗi service must là số nguyên and giá trị lớn hơn or tối thiểu bằng 1.
  * Start date hợp lệ need to must là ngày sau ngày hiện tại tạo ticket yêu cầu.
  * End date hợp lệ need to must là ngày sau ngày Start Date (if có) or sau ngày hiện tại tạo ticket (if không có ngày Start Date nào is chọn).
