# Fqas 5

Bạn hãy kiểm tra the thông tin sau: 
  * Các trường thông tin Title, Request type phải is điền đầy đủ. 
  * Bạn must chọn tối thiểu 1 service and phải chọn Quantity (số lượng) mà you muốn yêu cầu for service that.
  * Quantity of mỗi service phải là số nguyên and giá trị lớn hơn or tối thiểu bằng 1.
  * Start date hợp lệ must là ngày sau ngày hiện tại tạo ticket yêu cầu.
  * End date hợp lệ must là ngày sau ngày Start Date (if có) or sau ngày hiện tại tạo ticket (if không có ngày Start Date nào is chọn).
