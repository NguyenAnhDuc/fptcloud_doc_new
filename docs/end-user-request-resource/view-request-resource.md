# View the yêu cầu has been tạo

Với the account is cấp permission Tenant Owner, user can xem is tất cả the yêu cầu do chính họ tạo, or the yêu cầu do VPC Owner khác tạo. 
Với the account is cấp permission VPC Owner, user chỉ can xem is yêu cầu do chính họ tạo ra mà không xem is the yêu cầu of VPC Owner or Tenant Owner khác tạo.
**Step 1** : Ở menu chọn **Request Resource** , màn hình displayed danh sách the ticket yêu cầu tài nguyên.
[![file](/img/migrated/image-1703744448858-b0177c6f.png)](/img/migrated/image-1703744448858-b0177c6f.png)
**Step 2** : Chọn 1 ticket bằng theh bấm chọn 1 ID request. 
[![file](/img/migrated/image-1703744468083-2c0e0014.png)](/img/migrated/image-1703744468083-2c0e0014.png)
**Step 3** : Màn hình displayed thông tin chi tiết of ticket yêu cầu. 
[![file](/img/migrated/image-1703744483432-698ab803.png)](/img/migrated/image-1703744483432-698ab803.png)
