# Fqas 3

Bạn can chọn VPC mới (New) or 1 VPC bất kỳ mà you có permission quản lý. Với account permission VPC Owner, you chỉ can chọn VPC New or VPC you is quản lý mà không is chọn the VPC khác không thuộc permission quản lý.
