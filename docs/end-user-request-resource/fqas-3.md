# Fqas 3

Bạn có thể chọn VPC mới (New) hoặc 1 VPC bất kỳ mà bạn có quyền quản lý. Với tài khoản quyền VPC Owner, bạn chỉ có thể chọn VPC New hoặc VPC bạn đang quản lý mà không được chọn các VPC khác không thuộc quyền quản lý.
