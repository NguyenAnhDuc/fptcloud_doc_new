# Create a yêu cầu cung cấp tài nguyên

**Step 1** : Ở tab bar chọn **Tenant** cần cung cấp tài nguyên, ở thanh menu bar chọn **Request Resource** , chọn **Create reques** t.
[![file](/img/migrated/image-1703744342441-a0f93b7b.png)](/img/migrated/image-1703744342441-a0f93b7b.png)
**Step 2** : Fill in the thông tin ando form yêu cầu cung cấp tài nguyên
  * Các trường thông tin bắt buộc includes: Title, Request type, Service, Quantity
  * Các trường thông tin còn lại không bắt buộc.

[![file](/img/migrated/image-1703744369903-d102105e.png)](/img/migrated/image-1703744369903-d102105e.png)
[![file](/img/migrated/image-1703744387522-1dd6ab81.png)](/img/migrated/image-1703744387522-1dd6ab81.png)
**Step 3** : Chọn **Submit request** to gửi yêu cầu tới FCI. System will gửi email xác nhận has been tiếp nhận yêu cầu tới người tạo and đồng thời notification tới đội ngũ Sale/Support of FCI to hỗ trợ thực hiện yêu cầu. 
[![file](/img/migrated/image-1703744428466-add11d9d.png)](/img/migrated/image-1703744428466-add11d9d.png)
