# Fqas 4

Nếu you không có permission VPC Owner or Tenant Owner thì you cannot tạo yêu cầu cung cấp tài nguyên. Hãy liên hệ with đội ngũ hỗ trợ of FCI to is hỗ trợ cấp permission.
