# Initial setup

Nếu đây là lần đầu tiên you sử dụng FPT Cloud Server, trước tiên hãy kiểm tra and hoàn thành the công việc sau:
  * Liên hệ with đội ngũ Sale/Support of FPT Cloud to create and chạy tối thiểu 01 hợp đồng Commercial.
  * Được cung cấp account truy cập ando FPT Portal with vai trò là Owner of Tenant or Owner of VPC

Sau when thực hiện the thao tác trên, you has been sẵn sàng to create 1 yêu cầu cung cấp tài nguyên trên system FPT Cloud Server.
