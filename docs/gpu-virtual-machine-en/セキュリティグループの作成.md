# セキュリティグループの作成

**ステップ1：** GPU VMを作成したら、**[Network] > [Security groups]**に移動します
[![Alt text](/img/migrated/14-1-281806c6.png)](/img/migrated/14-1-281806c6.png)
**ステップ2：「Create security group」** ボタンをクリックして、新しい**Security group** を作成します。
**ステップ3：「Create security group」** 画面で、次のフィールドに入力します。
  * **名前：** セキュリティグループの名前をセットします(デフォルトではランダムな名前が生成されます)。
  * **適用インスタンス：** 新規作成した**GPU VM** を選択して、セキュリティグループに関連付けます。
  * **タグを追加：** オプション、リソースをより適切に整理できます。
  * **セキュリティルールの設定：**
  * 次の内容で**インバウンドルール** を作成します:

```
Copy* **タイプ：**SSH

* **ポート範囲:**22

* **ソース：**すべてIPv4
```

[![Alt text](/img/migrated/15-1-4fa4db41.png)](/img/migrated/15-1-4fa4db41.png)
**ステップ4:** 詳細を入力したら、 **「Create security group」** をクリックして確認します。新しく作成されたSecurity groupがリストに表示されます。
**注:Security group** を削除するには、まずすべてのルールを削除し、関連付けられているインスタンスをデタッチする必要があります。
