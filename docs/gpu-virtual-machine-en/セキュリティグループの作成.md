# Create a Security Group

**Step 1:** After creating your GPU VM, go to **Network** > **Security Groups**.

[![Alt text](/img/migrated/14-1-281806c6.png)](/img/migrated/14-1-281806c6.png)

**Step 2:** Click **Create Security Group** to create a new Security Group.

**Step 3:** In the **Create Security Group** screen, fill in the following fields.

  * **Name:** Set a name for the Security Group (a random name is generated by default).
  * **Applied instance:** Select the newly created **GPU VM** to associate with the Security Group.
  * **Add tag:** Optional — helps organize resources more effectively.
  * **Configure security rules:** Create an **inbound rule** with the following settings:

```
Type: SSH
Port range: 22
Source: All IPv4
```

[![Alt text](/img/migrated/15-1-4fa4db41.png)](/img/migrated/15-1-4fa4db41.png)

**Step 4:** After entering the details, click **Create Security Group** to confirm. The newly created Security Group appears in the list.

> 📝 **Note:** To delete a Security Group, you must first remove all rules and detach all associated instances.
