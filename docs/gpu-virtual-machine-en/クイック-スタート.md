# Quick Start

GPU Virtual Machine is a high-performance cloud server combining cloud flexibility with NVIDIA H100 and H200. Follow these steps to deploy your GPU VM.

**Log in to the Console.**
  1. Go to <https://console.fptcloud.com> and sign in.
  2. Navigate to the Instance screen (default screen) on the console page.
  3. Follow the initial setup guide [here](/docs/en/gpu-virtual-machine-en/)/.

**Create a Subnet.**
A Subnet is required to create and deploy a GPU VM.
  1. Click **Network** and select **Subnets** in the side menu.
  2. See the detailed guide [here](/docs/en/gpu-virtual-machine-en/)/.

**Create GPU VMs.**
  1. Click **Compute Engine** and select **Instance Management** in the side menu.
  2. Select **Create Instance** in the Instance Management screen.
  3. Configure the virtual machine deployment.
  4. **Floating IP and Security Group** can be configured after VM creation, not during this step.
  5. See the detailed guide [here](/docs/en/gpu-virtual-machine-en/)/.

**Allocate an IP address (Floating IP).**
  1. Click **Network** and select **Floating IPs** in the side menu.
  2. Select **Allocate IP address** in the Floating IPs screen.
  3. Assign the IP address to your newly created VM.
  4. See the detailed guide [here](/docs/en/gpu-virtual-machine-en/)/.

**Create a Security Group.**
  1. Click **Network** and select **Security Groups** in the side menu.
  2. Select **Create Security Group** in the Security Groups screen.
  3. Create inbound rules for the newly created VM.
  4. See the detailed guide [here](/docs/en/gpu-virtual-machine-en/)/.

**Access the server.**
  1. After the GPU VM is created, connect to the server via SSH using your SSH key.
  2. You can also access it by clicking **"Open at Console"** on the server detail page and logging in with a password.
  3. See the detailed guide [here](/docs/en/gpu-virtual-machine-en/)/.
