# FAQ

### 1. What is a GPU Virtual Machine?

A GPU Virtual Machine (GPU VM) is an independent system created from a GPU cluster of hardware resources, equipped with a dedicated GPU, CPU, memory, network interface, and storage. A wide range of pre-configured virtual machines designed to match workload requirements are available, offering flexible options from 1 to 8 GPUs per VM.

### 2. Can I resize a GPU instance — CPU, RAM, or disk?

GPU VM offers pre-configured flavors for GPU, CPU, RAM, and disk, and these cannot be customized to order. Select the appropriate configuration or contact **FPT Cloud Support**.

### 3. How do I assign a public IP to a GPU instance?

A **Floating IP** can only be assigned after the GPU instance has been successfully created. Note that you must also configure the IP port (port forwarding) to complete the IP configuration. See the detailed guide for more information.

### 4. Which features are not supported for GPU Virtual Machines?

The following features are not supported for GPU Virtual Machines with NVIDIA Hopper (H100, H200):

  * Instance resizing: adding or removing GPUs, resizing CPU, RAM, or disk
  * Snapshots
  * Creating a template from a GPU instance
  * Only backup instances using Veeam are currently supported.

### 5. What storage options are available for GPU Virtual Machines?

  * Currently, only NVMe physical disks are supported for GPU instances.
  * GPU Virtual Machines with Block Storage are coming soon.
  * File Storage (high-performance tier) service is supported.

### 6. In which regions are GPU Virtual Machines available?

GPU Virtual Machines with NVIDIA Hopper (H100, H200) are available in Hanoi 2, Vietnam and Tokyo, Japan.

### 7. How am I billed?

  * **Reserved:** Resources are limited based on demand at a fixed price, paid upfront (partially or in full) or post-paid. Billing periods are monthly (3–9 months) or yearly (1–5 years).
  * **Pay-As-You-Go (PAYG):** Start with unlimited resources and pay later. Charges are typically incremented daily or hourly.

### 8. Does FPT charge for a stopped GPU Virtual Machine server?

Yes. An instance in the "stopped" state still reserves the server for your use, so charges continue until you release the server. If you no longer want to be charged for the server, delete the instance from the FPT Cloud Portal. You are also billed for any other services or resources attached to the GPU instance.
