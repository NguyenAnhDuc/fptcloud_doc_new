# Fpt_Images

**FPT Image** is a custom image built by **FPT** that you can use to **get started quickly with any of the GPU VMs available**.  
This image comes with several components needed for **AI workloads** and selects **Ubuntu** as the **Operating System (OS).**
The versions of the installed dependencies* are **optimized for compatibility** and might not be the latest versions available.   
| **Operating System**  | **Version**  |  
| --- | --- |  
| Ubuntu  | 22.04  |  
| Ubuntu  | 24.04  |  
[![Alt text](https://fptcloud.com/wp-content/uploads/2025/03/OS-Images-1-1024x494.png)](https://fptcloud.com/wp-content/uploads/2025/03/OS-Images-1-1024x494.png)
* The dependency is excluded from OS.   
| **Dependencies**  | **Version**  |  
| --- | --- |  
| **NVIDIA Driver**  | 560.35.03  |  
| **NVIDIA CUDA Toolkit**  | 12.6.85  |  
| **NVIDIA Fabric Manager (NVSwitch Driver)**  | 560.35  |  
| **NVIDIA Datacenter GPU Manager**  | 3.3.9  |
