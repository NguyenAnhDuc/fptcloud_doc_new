# Create a Subnet

To create a GPU Virtual Machine (instance), you need an active Subnet within your VPC. The system assigns a private IP to the new virtual machine from this Subnet.

Steps to create a new Subnet:

**Step 1:** In the side menu, go to **Network** > **Subnets** and select **Create Subnet**.

[![Alt text](/img/migrated/1-057589e4.png)](/img/migrated/1-057589e4.png)

**Step 2:** Enter the required information.

  * **Name:** Enter a recognizable name for the Subnet.
  * **Type:** Two types are currently supported.
    * **Routed:** The Subnet is routed to the internet via a NAT gateway.
    * **Isolated:** The Subnet has no internet routing.
  * **Network address (CIDR):** Enter a valid CIDR.
  * **Gateway IP:** Enter a valid gateway address.
  * **Static IP pool (optional):** Specify the IP range to use. If left blank, the system allocates the entire CIDR range.
  * **Primary DNS:** The DNS server required for domain name resolution.
  * **Secondary DNS (optional):** A backup DNS server in case the primary DNS server fails.
  * **Add tag (optional):** Select tags to assign to the Subnet.

[![Alt text](/img/migrated/2-b8e89ee4.png)](/img/migrated/2-b8e89ee4.png)

**Step 3:** Click **Create Subnet** to create the new Subnet. The system processes the request and displays the result.

On success, the newly created Subnet appears in the Subnet table.
