# View Vpn 

**Step 1** : Click on the VPN name on the VPN management page. 
[![Xem VPN 1](/img/migrated/Xem-VPN-1-1024x262-72bdfe2b.png)](/img/migrated/Xem-VPN-1-1024x262-72bdfe2b.png)
**Step 2** : The VPN's information will appear in the Status Tab, including: 
  * Connection information. 
  * Error information. 
  * Traffic. 
  * Connection instructions, VPN test. 

To sync at the current time, please click the Refresh button. By default, the system will update periodically. 
[![Xem chi tiết VPN 1](/img/migrated/Xem-chi-tie%CC%82%CC%81t-VPN-1-1024x960-7f67cb07.png)](/img/migrated/Xem-chi-tie%CC%82%CC%81t-VPN-1-1024x960-7f67cb07.png)
**Step 3:** Connection configuration information at the **Connection Info Tab**
Click **Download Config** if you need to save information about the *.txt file
