# Delete VPN connection

1. Select **Delete** in the Action menu of the VPN connection to remove.

[![Delete VPN connection](/img/migrated/image-1744270418841-8bbb82f1.png)](/img/migrated/image-1744270418841-8bbb82f1.png)

2. In the dialog box, type `delete` and click **Delete VPN connection** to confirm, or click **Cancel** to discard.

[![Confirm deletion](/img/migrated/image-1744181528169-3495abb5.png)](/img/migrated/image-1744181528169-3495abb5.png)
