# Delete Vpn Connection

To delete a VPN Connection, select **Delete** in the Action menu of the VPN you want to remove. [![file](/img/migrated/image-1744270418841-8bbb82f1.png)](/img/migrated/image-1744270418841-8bbb82f1.png)
A dialog box will appear — type **delete** and click **Delete VPN connection** to confirm, or click **Cancel** to discard. [![file](/img/migrated/image-1744181528169-3495abb5.png)](/img/migrated/image-1744181528169-3495abb5.png)
