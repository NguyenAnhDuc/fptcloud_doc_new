# Enable Disable Vpn Connection

**1. Disable VPN Connection**
To disable a VPN connection, select **Disable VPN connection** from the Action menu. [![file](/img/migrated/image-1744269955705-6bd15bad.png)](/img/migrated/image-1744269955705-6bd15bad.png) A dialog box will appear — click **Disable VPN connection** to proceed, or **Cancel** to abort. [![file](/img/migrated/image-1744185325093-84bb6317.png)](/img/migrated/image-1744185325093-84bb6317.png)
> **Warning: Disable a VPN connection may take some time. Please wait until the process is completed before performing further actions.**
Check the VPN connection status once it has been successfully disabled: [![file](/img/migrated/image-1744269968890-4a79ced3.png)](/img/migrated/image-1744269968890-4a79ced3.png)
**2. Enable VPN Connection**
To enable a VPN connection, select **Enable VPN connection** from the Action menu.
[![file](/img/migrated/image-1744269980695-4e2933aa.png)](/img/migrated/image-1744269980695-4e2933aa.png)
A dialog box will appear — click **Enable VPN connection** to proceed, or **Cancel** to abort. [![file](/img/migrated/image-1744185386777-28068112.png)](/img/migrated/image-1744185386777-28068112.png)
> **Warning: Enable a VPN connection may take some time. Please wait until the process is completed before performing further actions.**
Check the VPN connection status once it has been successfully enabled: [![file](/img/migrated/image-1744269992510-e4a6ef08.png)](/img/migrated/image-1744269992510-e4a6ef08.png)
