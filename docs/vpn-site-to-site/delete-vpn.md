# Delete Vpn

Remove the active VPN. 
**Step 1** : Select **Delete**. 
[![Config delete](/img/migrated/Config-delete-1024x354-9bb9ed67.png)](/img/migrated/Config-delete-1024x354-9bb9ed67.png)
**Step 2** : Confirm Delete. 
Record VPN will switch to Deleting and be deleted. 
[![Config delete confirm](/img/migrated/Config-delete-confirm-1024x445-4c33a072.png)](/img/migrated/Config-delete-confirm-1024x445-4c33a072.png)
