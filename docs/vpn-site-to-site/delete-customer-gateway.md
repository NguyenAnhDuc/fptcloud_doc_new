# Delete Customer Gateway

> **Note: A Customer Gateway can only be deleted when no VPN Connection is assigned to it (Usage VPN Connection = none).**
On the Customer Gateway list screen, click **Delete** in the Action column of the Customer Gateway you want to remove. [![file](/img/migrated/image-1744270299322-d556bbcc.png)](/img/migrated/image-1744270299322-d556bbcc.png) A dialog box will appear — type **delete** and click **Delete Customer Gateway** to confirm, or click **Cancel** to discard. [![file](/img/migrated/image-1744183626743-165bf74f.png)](/img/migrated/image-1744183626743-165bf74f.png)
