# Delete Customer Gateway

:::note
A Customer Gateway can only be deleted when no VPN connection is assigned to it (Usage VPN Connection = none).
:::

1. On the **Customer Gateway** list screen, click **Delete** in the Action column of the gateway to remove.

[![Delete Customer Gateway](/img/migrated/image-1744270299322-d556bbcc.png)](/img/migrated/image-1744270299322-d556bbcc.png)

2. In the dialog box, type `delete` and click **Delete Customer Gateway** to confirm, or click **Cancel** to discard.

[![Confirm deletion](/img/migrated/image-1744183626743-165bf74f.png)](/img/migrated/image-1744183626743-165bf74f.png)
