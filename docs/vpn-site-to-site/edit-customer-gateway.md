# Edit Customer Gateway

:::note
A Customer Gateway can only be edited when it is not assigned to any VPN connection.
:::

1. On the **Customer Gateway** list screen, click **Edit Customer Gateway** in the Action column of the gateway to modify.

[![Edit Customer Gateway](/img/migrated/image-1744269804714-c8f82034.png)](/img/migrated/image-1744269804714-c8f82034.png)

2. Edit the following fields:

[![Customer Gateway form](/img/migrated/image-1744269819671-b090793a.png)](/img/migrated/image-1744269819671-b090793a.png)

| **Field** | **Description** |
| --- | --- |
| **Customer gateway name** | Name of the Customer Gateway. |
| **Description** | Description of the Customer Gateway. |
| **Remote IP public** | Public IP address of the on-premises VPN device. |
| **Remote network** | Subnet of the on-premises network. |
| **Healthcheck IP address** | IP address used to monitor the VPN connection health between the two sites. |

3. Click **Update Customer Gateway** to save changes, or **Cancel** to discard.

[![Update result](/img/migrated/image-1744269832390-ef24471f.png)](/img/migrated/image-1744269832390-ef24471f.png)
