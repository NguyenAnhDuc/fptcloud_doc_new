# Delete VPN Site-to-Site

:::danger
Deleting a VPN Site-to-Site connection is irreversible. The connection will be permanently terminated.
:::

1. Select the VPN connection, then click **Actions** → **Delete**.

   ![Delete VPN menu](/img/migrated/image-1742916525951-13239d02.png)

2. Confirm the deletion. The VPN status changes to **Deleting** and the connection is removed from the list.

   ![VPN deletion confirmation dialog](/img/migrated/image-1742916584099-b13362ab.png)
