# Initial setup

If this is your first time using FPT Cloud, complete the following steps before getting started.

## Create an FPT Cloud account and log in to FPT Portal

To start using FPT Cloud services, you need to log in to FPT Portal.

1. Go to [https://console.fptcloud.com](https://console.fptcloud.com/) and log in with your FPT Cloud account credentials.
2. Select the correct Tenant, Region, and VPC for the service you want to use.

If you do not have an account yet, click **Sign Up** and fill in the required information. The support team will contact you to verify your details and create your account.

:::note
If you encounter any errors during the process, contact our Support team immediately for assistance.
:::
