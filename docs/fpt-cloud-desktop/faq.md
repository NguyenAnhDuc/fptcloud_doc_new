# Faq

#### **Q: FCD là gì?**
A: FCD là viết tắt of FPT Cloud Desktop, service ảo hóa máy tính trên Cloud do FPT Smart Cloud phát triển.
#### **Q: Thiết bị kết nối phù hợp to sử dụng FCD?**
A: Thiết bị kết nối to máy tính ảo (Hay còn is gọi là Local Device) như:
  * Máy tính bàn (PC), Máy tính xách tay (Laptop) or Thin Client, MiniPC
  * Điện thoại thông minh (Smartphone) to xác thực hai (02) lớp

Hai loại thiết bị trên có kết nối Internet/ network LAN
#### **Q: Có thể upgrade cấu hình máy tính ảo FCD is không?**
A: Có thể, tuỳ theo chính sách of fromng công ty. Configure can upgrade includes: Thông số Core CPU, RAM (Memory) and Dung lượng Ổ đĩa (Disk)
#### **Q: Có thể cài đặt the ứng dụng mới ando FCD is không?**
A: Nếu you is Manage in Tổ chức cấp permission Quản trị viên of Máy tính ảo (Administrator), you can thực hiện việc cài đặt the ứng dụng ngoài danh sách mặc định when is cấp phát. Nếu không permission Quản trị viên of Máy tính ảo, you cần liên hệ for IT Admin to is hỗ trợ
#### **Q: Tệp dữ liệu of tôi ở đâu? Chúng is lưu trữ như thế nào and in bao lâu?**
A: Dữ liệu of you is lưu trữ security tại hệ thống of FPT Cloud Desktop. Bạn can truy cập the tệp this trên thư mục Desktop and Documents in hệ điều hành you has been chọn. Tài khoản user will bị vô hiệu hóa and bị xóa sau theo quy định of Tổ chức when có yêu cầu phê duyệt Thu hồi virtual machine has been cấp phát for you.
#### **Q: Một user can đăng nhập from nhiều thiết bị/địa điểm cùng a lúc không?**
A: Một user can đăng nhập from nhiều thiết bị, nhưng chỉ a thiết bị can truy cập virtual machine ando a thời điểm.
