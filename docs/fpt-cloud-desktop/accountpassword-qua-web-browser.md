# Truy cập qua Web browser

Dành for user muốn truy cập virtual machine nhanh, không need to cài đặt FCDClient.
**1.Truy cập ando Homepage service with URL phù hợp**
Các định dạng URL hợp lệ:
  * URL riêng of doanh nghiệp/tổ chức dùng for FCD (quản trị viên customer cung cấp for user)
  * URL has been chứa authentication code hợp lệ (định dạng code.domain). Example: pil783454741.pilotfcd.online
  * URL mặc định of service

**Information URL this do quản trị viên customer cung cấp**
Truy cập đường link service bằng trình duyệt web, chọn **Access through FPT Cloud Desktop Client**
[![file](/img/migrated/image-1767859575680-f5ed3897.png)](/img/migrated/image-1767859575680-f5ed3897.png)
**2.Log in ando Authenticator (Server) phù hợp**
Nếu user truy cập **truy cập bằng URL has been chứa authentication code hợp lệ:** (ví dụ URL chứa code hợp lệ: pil783454741.pilotfcd.online)
  * Log in bằng account/password corresponding:

[![file](/img/migrated/image-1767956660879-1796a9b2.png)](/img/migrated/image-1767956660879-1796a9b2.png)
=> Log in Authenticator (Server) successfully
* * *
Nếu user truy cập from URL mặc định of service:
  * Enter information Authentication Code (information do quản trị viên customer quản lý) (Ví dụ Authentication Code hợp lệ: pil783454741)

[![file](/img/migrated/image-1767863050735-fee7cee6.png)](/img/migrated/image-1767863050735-fee7cee6.png)
  * Enter information account corresponding theo Server muốn đăng nhập

[![file](/img/migrated/image-1767955827414-9e7265a7.png)](/img/migrated/image-1767955827414-9e7265a7.png)
**Note:** Nếu có displayed mã QR Xác thực 2 lớp: Tải and cài đặt **ứng dụng xác thực Microsoft Authenticator** trên Điện thoại tại Apple Store or CH Play/ Google Play
[![file](/img/migrated/image-1767955899835-0408f00d.png)](/img/migrated/image-1767955899835-0408f00d.png)
Mở ứng dụng Authenticator and thực hiện quét (scan) QR Code
[![file](/img/migrated/image-1767955938951-3aac0b75.png)](/img/migrated/image-1767955938951-3aac0b75.png)
  * Ứng dụng thực hiện đồng bộ and displayed OTP to thực hiện Log in ando FCD. 
  * Enter mã OTP and chọn Submit => Log in Authenticator (Server) successfully

Tại màn hình list the virtual machine, chọn truy cập ando virtual machine mong muốn
[![file](/img/migrated/image-1767860779789-05426af5.png)](/img/migrated/image-1767860779789-05426af5.png)
Enter information đăng nhập ando virtual machine if system yêu cầu => Truy cập virtual machine successfully.)
