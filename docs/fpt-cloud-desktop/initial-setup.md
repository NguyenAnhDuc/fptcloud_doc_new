# Prerequisites

Before accessing FPT Cloud Desktop, ensure the following conditions are met:

## SSO login (Azure AD)

- A PC or laptop that can access the FPT Cloud Desktop link via Internet or LAN
- Full virtual desktop access credentials provided by your administrator, including the FCD URL, account details, and corresponding Authenticator app

## Account/Password login

- A PC or laptop that can access the FPT Cloud Desktop link via Internet or LAN
- Full virtual desktop access credentials provided by your administrator, including the FCD URL and account details (FCD FCI URL: `fci.fcd.fptcloud.com`)
