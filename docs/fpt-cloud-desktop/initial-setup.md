# Điều kiện yêu cầu

Trước when truy cập, user need to đảm bảo:
**Log in with account SSO (Azure):**
  * Sử dụng PC, laptop can truy cập link FPT Cloud Desktop thông qua Internet/ network LAN
  * Được cung cấp đầy đủ information truy cập virtual machine includes URL FCD, account and Authenticator corresponding

**Log in with Account/ Password:**
  * Sử dụng PC, laptop can truy cập link FPT Cloud Desktop thông qua Internet/ network LAN
  * Được cung cấp đầy đủ information truy cập virtual machine includes URL FCD and account (URL FCD FCI: fci.fcd.fptcloud.com)
