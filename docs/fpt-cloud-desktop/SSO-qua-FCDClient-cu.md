# Truy cập qua FCDClient cũ (luồng cũ)

Dành for user is sử dụng FCDClient cũ has been cài đặt trước that
**Note:**
  * Việc truy cập bằng FCDClient cũ trên PC, laptop will bị dừng hỗ trợ **sau ngày 31/03/2026.** Truy cập bằng the thiết bị khác vẫn will is hỗ trợ for to when for notification mới. Người dùng chủ động cài đặt FCDClient mới to tránh bị gián đoạn công việc
**1.Truy cập ando Homepage service with URL phù hợp**
Các định dạng URL hợp lệ:
  * URL riêng of doanh nghiệp/tổ chức dùng for FCD (quản trị viên customer cung cấp for user)
  * URL has been chứa authentication code hợp lệ (định dạng code.domain). Example: pil783454741.pilotfcd.online
  * URL mặc định of service

**Information URL this do quản trị viên customer cung cấp**
Truy cập đường link service bằng trình duyệt web, chọn **Access through FPT Cloud Desktop Client**
[![file](/img/migrated/image-1767859575680-f5ed3897.png)](/img/migrated/image-1767859575680-f5ed3897.png)
**2.Log in ando Authenticator (Server) phù hợp**
Nếu user **truy cập bằng URL has been chứa authentication code hợp lệ** (ví dụ URL chứa code hợp lệ: pil783454741.pilotfcd.online)
  * Chỉ need to đăng nhập bằng account SSO corresponding (ví dụ Log in bằng account Microsoft), nhập OTP corresponding theo SSO => Log in Authenticator (Server) successfully [![file](/img/migrated/image-1767953411678-b06915c9.png)](/img/migrated/image-1767953411678-b06915c9.png)

* * *
Nếu user tải xuống Client from URL mặc định of service:
  * Enter information Authentication Code (information do quản trị viên customer quản lý) (Ví dụ Authentication Code hợp lệ: pil783454741)

[![file](/img/migrated/image-1767863050735-fee7cee6.png)](/img/migrated/image-1767863050735-fee7cee6.png)
  * Log in bằng account SSO corresponding (ví dụ Log in bằng account Microsoft), nhập OTP corresponding theo SSO => Log in Authenticator (Server) successfully [![file](/img/migrated/image-1767953411678-b06915c9.png)](/img/migrated/image-1767953411678-b06915c9.png)

**3.Truy cập ando virtual machine** Tại màn hình list the virtual machine, chọn truy cập ando virtual machine mong muốn. **Note:** tại bước this user chọn option truy cập qua FCDClient. 
  * Nếu user is cài đặt FCDClient cũ: System will mở FCDClient cũ to truy cập virtual machine Người dùng nhập information đăng nhập ando virtual machine if system yêu cầu => Truy cập virtual machine successfully
[![file](/img/migrated/image-1767954262885-f3b50a93.png)](/img/migrated/image-1767954262885-f3b50a93.png)
  * Enter information account corresponding theo Server muốn đăng nhập

**Note:** Nếu có displayed mã QR Xác thực 2 lớp: Tải and cài đặt **ứng dụng xác thực Microsoft Authenticator** trên Điện thoại tại Apple Store or CH Play/ Google Play
[![file](/img/migrated/image-1767860622852-f00b8ad2.png)](/img/migrated/image-1767860622852-f00b8ad2.png)
Mở ứng dụng Authenticator and thực hiện quét (scan) QR Code
[![file](/img/migrated/image-1767860691470-919c5b65.png)](/img/migrated/image-1767860691470-919c5b65.png)
  * Ứng dụng thực hiện đồng bộ and displayed OTP to thực hiện Log in ando FCD. 
  * Enter mã OTP and select **Submit** => Log in Authenticator (Server) successfully

**3.Truy cập ando virtual machine**
Tại màn hình list the virtual machine, chọn truy cập ando virtual machine mong muốn. 
**Note:** tại bước this user chọn option truy cập qua FCDClient. 
  * Nếu user is cài đặt FCDClient cũ: System will mở FCDClient cũ to truy cập virtual machine Người dùng nhập information đăng nhập ando virtual machine if system yêu cầu => Truy cập virtual machine successfully
[![file](/img/migrated/image-1767865288885-77987978.png)](/img/migrated/image-1767865288885-77987978.png)
