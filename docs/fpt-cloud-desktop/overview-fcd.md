# Detailed overview

## Advantages over traditional PCs

- **Data security:** Encrypted storage, centralized management, and access control
- **Continuous security patching:** Centralized and consistent updates that save time and resources
- **Data integrity during incidents:** Automatic backups combined with disaster recovery solutions
- **Fast and flexible resource scaling:** Provision thousands of virtual desktops in just a few hours

## Supported connection devices

- Desktop, PC, Laptop
- Mobile (Smartphone) or Tablet
- ThinClient or FPT MiniPC

## Access methods

- **Native FCDClient (recommended):** Access your virtual desktop directly through the application for the smoothest and most stable experience.
- **Web browser:** Quick access through a browser with no installation required.

## Terms and definitions

| Term (English) | Term (Vietnamese) | Abbreviation | Definition |
| --- | --- | --- | --- |
| **Virtual Desktop** | Máy ảo / Máy tính ảo | VD | A virtual machine that provides software features and performance similar to a physical computer. Virtual desktops share FPT Cloud infrastructure. Each user can be allocated multiple virtual desktops simultaneously with different configurations and operating systems. |
| **Desktop Pool** | Nhóm máy ảo | Pool | A group of virtual desktops sharing the same Cloud processing and storage infrastructure. Typically used to group virtual desktops by department, function, usage schedule, etc. |
| **Operating System** | Hệ điều hành | OS | Examples: Microsoft Windows, macOS |
| **End-User / User** | Người dùng cuối | N/A | The end user of an FCD virtual desktop. Users can: log in to the FCD User Portal, access their allocated virtual desktop, and use it like a regular physical computer. |
| **Supported Device** | Thiết bị hỗ trợ | N/A | A device that can access and use the allocated virtual desktop |
| **Local Device** | Thiết bị kết nối | N/A | The device used to connect to the virtual desktop |
| **Local Area Network** | Mạng nội bộ | LAN | A network system used to connect computers within an organization |
| **Authenticator App** | Phần mềm xác thực | N/A | An app that generates a 6-digit random verification code used for account login, valid for 30 seconds |
| **Active Directory** | Quản lý thư mục và xác thực | AD | A centralized user management system |
