# Introduction chi tiết

## Lợi thế so with PC Truyền thống
  * Bảo mật dữ liệu: Save trữ mã hóa, quản lý tập trung and giới hạn truy cập
  * Liên tục cập nhật vá lỗi security: Thực hiện tập trung, nhất quán; Tiết kiệm nguồn lực and thời gian
  * Đảm bảo toàn vẹn dữ liệu when có sự cố: Sao lưu tự động, kết hợp giải pháp restore dữ liệu sau thảm họa
  * Nhanh chóng and Linh hoạt mở rộng/ thu hồi tài nguyên: Cấp phát hàng nghìn máy tính ảo chỉ in a andi giờ

## Concept
**Loại thiết bị kết nối hỗ trợ**
  * Desktop/ PC/ Laptop 
  * Mobile (Smartphone/ Điện thoại thông minh) or Tablet (Máy tính bảng)
  * ThinClient or FPT MiniPC

**Hình thức truy cập**
  * Native FCDClient (Khuyên dùng): Cho phép truy cập virtual machine trực tiếp qua ứng dụng, mang lại trải nghiệm mượt mà and ổn định nhất.
  * Web Browser: Phương thức truy cập nhanh gọn qua trình duyệt, không cần cài đặt.

## Terms & definitions  
| Terms (English)  | Terms (Vietnamese)  | Abbreviations  | Definitions  |  
| --- | --- | --- | --- |  
| **Virtual Desktop**  | Máy ảo / Máy tính ảo  | VD  | - Máy ảo cung cấp feature phần mềm and hiệu năng tương tự như máy tính vật lý thông thường  
- Các virtual machine sử dụng chung hạ tầng FPT Cloud  
- Mỗi user can is cấp phát/sử dụng nhiều virtual machine đồng thời with the thông số cấu hình and hệ điều hành khác nhau  |  
| **Desktop Pool**  | Nhóm virtual machine  | Pool  | - Một nhóm gồm nhiều virtual machine có chung hạ tầng xử lý and lưu trữ Cloud  
- Thông thường dùng to gom nhóm virtual machine theo Bộ phận, Chức năng, Khung giờ sử dụng, v.v.  |  
| **Operating System**  | Hệ điều hành  | OS  | Example: Microsoft Windows, macOS  |  
| **End-User / User**  | Người dùng cuối  | N/A  | Người dùng cuối sử dụng virtual machine FCD  
Người dùng can:  
- Log in ando FCD User Portal  
- Truy cập ando virtual machine is cấp phát  
- Sử dụng virtual machine như máy tính vật lý thông thường  |  
| **Supported Device**  | Thiết bị hỗ trợ  | N/A  | Thiết bị can sử dụng and truy cập ando virtual machine is cấp phát  |  
| **Local Device**  | Thiết bị kết nối  | N/A  | Thiết bị is sử dụng to thực hiện kết nối ando virtual machine  |  
| **Local Area Network**  | Mạng nội bộ  | LAN  | System network dùng to kết nối the máy tính in a phạm vi of tổ chức  |  
| **Authenticator App**  | Phần mềm xác thực  | N/A  | Ứng dụng tạo mã xác thực gồm 6 chữ số ngẫu nhiên dùng to đăng nhập account, có thời hạn sử dụng in 30 giây  |  
| **Active Directory**  | Manage thư mục and xác thực  | AD  | System quản trị tập trung user  |
