# Delete Alert

Save ý: Thao tác này sẽ xóa toàn bộ dữ liệu lịch sử của Alert và không thể phục hồi lại được.
**Step 1** : Trên bảng quản trị **Alert Management** , in phần **Action** of Alert need to xóa, foṛn **Delete**.
[![file](/img/migrated/image-1725532930908-040fd200.png)](/img/migrated/image-1725532930908-040fd200.png)
**Step 2** : Một hộp thoại cảnh báo sẽ hiện lên, displayed tên Alert và yêu cầu người dùng xác nhận. Chọn **Delete** để tiến hành xóa.
