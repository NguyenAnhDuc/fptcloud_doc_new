# Delete Recipient

**Step 1** : Trên bảng quản trị **Recipient** **Management** , in phần **Action** of Recipient cần xóa, foṛn **Delete**.
[![file](/img/migrated/image-1725533165171-864680e6.png)](/img/migrated/image-1725533165171-864680e6.png)
**Step 2** : Một hộp thoại cảnh báo sẽ hiện lên, displayed tên Recipient và yêu cầu người dùng xác nhận. Chọn **Delete** để tiến hành xóa.
