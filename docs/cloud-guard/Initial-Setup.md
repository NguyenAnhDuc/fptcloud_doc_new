# Initial setup

If this is your first time using **FPT Cloud Guard – Alert**, complete the following steps before creating your first alert.

## Create an FPT Cloud account and log in to FPT Portal

1. To register for an FPT Cloud account, visit [https://fptcloud.com/](https://fptcloud.com/).
2. Select **Sign Up** and enter the required information following the on-screen instructions. The support team will contact you to confirm your account details.
3. To log in to FPT Portal, go to [https://console.fptcloud.com/](https://console.fptcloud.com/).
4. Log in with your assigned credentials, then select the correct Tenant, Region, and VPC containing the resources you want to monitor.
5. You are now ready to create your first **Alert**.

:::tip
If you are unsure about any of the above details, or if the system returns an error after 3 attempts, contact our Support team immediately for assistance.
:::
