# Edit an alert

**Step 1:** In the menu, go to **Cloud Guard** and select the **Resource alert** tab.

[![file](/img/migrated/image-1725532789529-6dfc0d32.png)](/img/migrated/image-1725532789529-6dfc0d32.png)

**Step 2:** Select the alert to edit. The Update screen appears.

[![file](/img/migrated/image-1725532811064-3d095f59.png)](/img/migrated/image-1725532811064-3d095f59.png)

**Step 3:** Enter the new values in the relevant fields and click **Update**.

:::note
FPT Cloud Guard only supports editing certain fields of an existing alert.
:::
