# Initialize a Activity Alert mới

**Step 1** : Ở menu chọn **Security** > **Cloud Guard** , mở tab **Activity** **Alerts** and foṛn **Create Alert**. [![file](/img/migrated/image-1725532325024-78ca8f1c.png)](/img/migrated/image-1725532325024-78ca8f1c.png) Một hộp thoại mới will hiện lên and hướng dẫn user thực hiện the bước tiếp theo.
**Step 2** : Configure alert theo nhu cầu bằng theh nhập thông tin ando trường dữ liệu corresponding:  
|   |   |  
| --- | --- |  
| **Field**  | **Description**  |  
| **Name**  | Enter tên of Alert  |  
| **Resource Type**  | Select **Loại tài nguyên** will alert  |  
| **Action**  | Chọn the loại tác động mà hệ thống bắn alert  |  
**Save ý** : Mỗi loại tài nguyên will có những action khác nhau for phép đặt alert. Khi đặt alert Activity log will mặc định apply for toàn bộ VPC | | **Severity** | Chọn mức độ nghiêm trọng of alert, có the mức: Critical, High, Medium, Low | | **Notify to** | Chọn người nhận alert |
[![file](/img/migrated/image-1725532405898-58b580bc.png)](/img/migrated/image-1725532405898-58b580bc.png)
Step 3: Chọn Create to thực hiện khởi tạo Alert. Hệ thống sẽ thông báo xác nhận, kiểm tra tài nguyên and tiến hành quá trình khởi tạo dựa ando the thông tin has been chọn.
Sau when khởi tạo successfully, người dùng can thấy Alert mới tạo and kiểm tra thông tin trên bảng quản trị.
