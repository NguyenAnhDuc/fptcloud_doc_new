# Create Veeam Recovery Media

**Step 1:** Khởi động ứng dụng Create Recovery Media. [![file](/img/migrated/image-1695292809658-75c1dd96.png)](/img/migrated/image-1695292809658-75c1dd96.png)
**Step 2:** Cửa sổ hiện ra, tùy chọn the kết nối network, driver mà you muốn tích hợp ando file Veeam Recovery Media sau that bấm Next to tiếp tục. [![file](/img/migrated/image-1695292817914-6d9e55d8.png)](/img/migrated/image-1695292817914-6d9e55d8.png)
**Step 3:** Đặt tên and chọn nơi lưu trữ file ISO. Sau that bấm Next to tiếp tục.
  * Save ý không should lưu file Veeam Recovery Media tại server need to backup, to tránh trường hợp when server bị xóa or cannot khởi động you cannot lấy file this to sử dụng. [![file](/img/migrated/image-1695292826165-585e6c60.png)](/img/migrated/image-1695292826165-585e6c60.png) **Step 4:** Check lại information sau that bấm Create. [![file](/img/migrated/image-1695292833515-81e26e7e.png)](/img/migrated/image-1695292833515-81e26e7e.png) **Step 5:** Sau when hoàn thành việc khởi tạo, bấm Finish to kết thúc.
