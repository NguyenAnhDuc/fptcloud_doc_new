# Restore Window vm From Cloud Repository

**Step 1:** Attach File ISO Veeam recovery media sau that khởi đội VM from File ISO (lưu ý VM cần có cùng số lượng disk and dung lượng disk). Interface console after VM boot successfully.
Chọn Bare Metal Recovery
**Step 2:** Chọn Network storage sau that chọn Next.
Chọn Veeam Cloud Connect repository
**Step 3:** Enter URL of BaaS: baas-sgn-gw.fptcloud.com đối with site Hồ Chí Minh and baas-han-gw.fptcloud.com đối with site Hà Nội
**Step 4:** Enter Username/password of you. Sau that ấn Enter to tiếp tục.
**Step 5:** Lựa chọn VM cần restore and restore point mong muốn.
**Step 6:** Chọn Manual restore to Map the disk corresponding.
Restore the disk corresponding.
**Step 7:** Check lại thông tin and chọn Restore to tiến hành restore lại VM of you
