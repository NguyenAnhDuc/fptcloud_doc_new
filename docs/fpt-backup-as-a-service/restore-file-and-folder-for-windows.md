# Restore File and Folder for Windows

**Step 1:** Khởi động Veeam Agent For Windows [![file](/img/migrated/image-1695292398449-9e91c3cb.png)](/img/migrated/image-1695292398449-9e91c3cb.png) Note: Trường hợp thời điểm bạn muốn restore cũ hơn bạn phải sử dụn tính năng File Level Restore tham khảo bước 5
**Step 2:** Click vào backup job chứa file mà bạn muốn restore sau đó chọ Restore Files [![file](/img/migrated/image-1695292405251-6f9dec6f.png)](/img/migrated/image-1695292405251-6f9dec6f.png) **Step 3:** Cửa sổ hiện ra, chọn file hoặc folder bạn muốn khôi phục. Chọn Restore Overwrite để khôi phục file đè lên file hiện tại. Hoặc chọn Keep để giữ file hiện tại, khi đó file restore sẽ được đổi tên. [![file](/img/migrated/image-1695292409737-ad7d5f87.png)](/img/migrated/image-1695292409737-ad7d5f87.png) **Step 4:** Bạn cũng có thể chọn Copy To để khôi phục ra một nơi khác. Chọn nơi lưu trũ file, folder sau đó bấn OK. [![file](/img/migrated/image-1695292415057-bac4c7be.png)](/img/migrated/image-1695292415057-bac4c7be.png) **Step 5:** Khởi động công cụ File Level Restore có sẵn trong bộ Veeam Agent For Windows. [![file](/img/migrated/image-1695292421939-897334d8.png)](/img/migrated/image-1695292421939-897334d8.png) **Step 6:** Trong phần Restore Point. Chọn thời điểm bạn muốn khôi phục sau đó bấm Next. [![file](/img/migrated/image-1695292427514-2d131ec1.png)](/img/migrated/image-1695292427514-2d131ec1.png) **Step 7:** Chọn Open để tiếp tục. Sau khi cửa sổ restore hiện lên, quay lại bước 3 để tiếp tục. [![file](/img/migrated/image-1695292434984-0161da63.png)](/img/migrated/image-1695292434984-0161da63.png)
