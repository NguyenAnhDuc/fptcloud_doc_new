# Restore File and Folder for Windows

**Step 1:** Khởi động Veeam Agent For Windows [![file](/img/migrated/image-1695292398449-9e91c3cb.png)](/img/migrated/image-1695292398449-9e91c3cb.png) Note: Trường hợp thời điểm you muốn restore cũ hơn you must sử dụn feature File Level Restore tham khảo bước 5
**Step 2:** Click ando backup job chứa file mà you muốn restore sau that chọ Restore Files [![file](/img/migrated/image-1695292405251-6f9dec6f.png)](/img/migrated/image-1695292405251-6f9dec6f.png) **Step 3:** Cửa sổ hiện ra, chọn file or folder you muốn restore. Chọn Restore Overwrite to restore file đè lên file hiện tại. Hoặc chọn Keep to giữ file hiện tại, when that file restore will is đổi tên. [![file](/img/migrated/image-1695292409737-ad7d5f87.png)](/img/migrated/image-1695292409737-ad7d5f87.png) **Step 4:** Bạn cũng can chọn Copy To to restore ra a nơi khác. Chọn nơi lưu trũ file, folder sau that bấn OK. [![file](/img/migrated/image-1695292415057-bac4c7be.png)](/img/migrated/image-1695292415057-bac4c7be.png) **Step 5:** Khởi động công cụ File Level Restore có sẵn in bộ Veeam Agent For Windows. [![file](/img/migrated/image-1695292421939-897334d8.png)](/img/migrated/image-1695292421939-897334d8.png) **Step 6:** Trong phần Restore Point. Chọn thời điểm you muốn restore sau that bấm Next. [![file](/img/migrated/image-1695292427514-2d131ec1.png)](/img/migrated/image-1695292427514-2d131ec1.png) **Step 7:** Chọn Open to tiếp tục. Sau when cửa sổ restore hiện lên, quay lại bước 3 to tiếp tục. [![file](/img/migrated/image-1695292434984-0161da63.png)](/img/migrated/image-1695292434984-0161da63.png)
