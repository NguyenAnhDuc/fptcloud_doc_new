# Install Veeam Agent for Windows

**Step 1** : Trên server cần backup, login vào trang portal với account của bạn: Backup as a Service : <https://baas.fptcloud.com> [![file](/img/migrated/image-1695292106378-41580e19.png)](/img/migrated/image-1695292106378-41580e19.png) Đây là giao diện quản trị của bạn sau khi login thành công: [![file](/img/migrated/image-1695292115345-71164f38.png)](/img/migrated/image-1695292115345-71164f38.png) **Step 2** : Vào mục **Managed Computers** chọn **Download Agent** sau đó chọn **Window**. [![file](/img/migrated/image-1695292124121-91d52bda.png)](/img/migrated/image-1695292124121-91d52bda.png) Chỉnh thời gian Token hết hạn theo nhu cầu. Mặc định là 12 Tháng. [![file](/img/migrated/image-1695292132688-e41cc91e.png)](/img/migrated/image-1695292132688-e41cc91e.png) **Step 3:** Sau khi download thành công ta tiến hành cài đặt Management Agent trên server cần backup: [![file](/img/migrated/image-1695292142746-c8534ab5.png)](/img/migrated/image-1695292142746-c8534ab5.png) Chọn Next để tiếp tục: [![file](/img/migrated/image-1695292150003-ee0a607d.png)](/img/migrated/image-1695292150003-ee0a607d.png) Chấp nhận điều khoản sau đó bấm 
Cloud Gateway: baas-sgn-gw.fptcloud.com hoặc baas-han-gw.fptcloud.com Port: 6180 [![file](/img/migrated/image-1695292180601-c17bd7f4.png)](/img/migrated/image-1695292180601-c17bd7f4.png)
**Step 5:** Check VM xuất hiện trên portal và tiến trình cài đặt Veeam Agent thành công. [![file](/img/migrated/image-1695292191409-db0a966c.png)](/img/migrated/image-1695292191409-db0a966c.png) **Step 6:** Check Job backup xuất hiện trên portal và Veeam Agent For Windows đã được cài đặt trên Server.
