# Restore Linux vm From Cloud Repository

**Step 1:** Để restore 1 VM Linux you cần download file ISO Veeam recovery media from trang chủ of VEEAM. Download file ISO phù hợp with hệ điều hành of VM cần restore. [![file](/img/migrated/image-1695294618113-7017717b.png)](/img/migrated/image-1695294618113-7017717b.png)
**Step 2:** Khởi động lại VM and boot from ISO Veeam recovery media. Đồng ý with điều khoản sau that tiếp tục. [![file](/img/migrated/image-1695294644895-2a9750be.png)](/img/migrated/image-1695294644895-2a9750be.png) **Step 3:** Configure lại network if cần Trường hợp VM khởi tạo to restore add network có DHCP thì không cần Configure network and can bỏ qua bước 3 this Còn lại must click ando “Configure network” to đặt IP static for VM restore. Note: IP of VM this cần is mở kết nối ra internet or to baas-sgn-gw.fptcloud.com (site HCM) and baas-han-gw.fptcloud.com port 6180 to thực hiện restore [![file](/img/migrated/image-1695294671316-44391ccf.png)](/img/migrated/image-1695294671316-44391ccf.png) Thực hiện chọn edit a connection to đặt IP manual for VM restore: [![file](/img/migrated/image-1695294681820-3a5125db.png)](/img/migrated/image-1695294681820-3a5125db.png) Chọn Edit to set IP for VM [![file](/img/migrated/image-1695294692535-3e2748f0.png)](/img/migrated/image-1695294692535-3e2748f0.png) Chọn IPv4 Configuration sau that chọn mode Manual -> Show to đặt IP [![file](/img/migrated/image-1695294702919-9bc47e74.png)](/img/migrated/image-1695294702919-9bc47e74.png) Sau when đặt xong IP chọn OK and nhấn ESC to back lại màn hình chính to tiến hành restore [![file](/img/migrated/image-1695294720487-cfae6dc7.png)](/img/migrated/image-1695294720487-cfae6dc7.png) **Step 4:** Chọn Add Cloud provider [![file](/img/migrated/image-1695294739863-99ac30bf.png)](/img/migrated/image-1695294739863-99ac30bf.png) **Step 5:** Enter URL gateway of BaaS: baas-sgn-gw.fptcloud.com (site HCM) or baas-han-gw.fptcloud.com (site Hà Nội) [![file](/img/migrated/image-1695294750751-e258de28.png)](/img/migrated/image-1695294750751-e258de28.png) **Step 6:** Enter Username/password of you. Sau that ấn Enter to tiếp tục. [![file](/img/migrated/image-1695294761178-6db27dc0.png)](/img/migrated/image-1695294761178-6db27dc0.png) **Step 7:** Chọn tên backup job and restore poin mong muốn. [![file](/img/migrated/image-1695294772745-817c2f15.png)](/img/migrated/image-1695294772745-817c2f15.png) **Step 8:** Map the disk corresponding. [![file](/img/migrated/image-1695294786115-e28a2a8c.png)](/img/migrated/image-1695294786115-e28a2a8c.png) Restore the disk corresponding. [![file](/img/migrated/image-1695294797487-07dc8c7b.png)](/img/migrated/image-1695294797487-07dc8c7b.png) [![file](/img/migrated/image-1695294806424-3d68dddb.png)](/img/migrated/image-1695294806424-3d68dddb.png) [![file](/img/migrated/image-1695294815857-68eb070f.png)](/img/migrated/image-1695294815857-68eb070f.png) **Step 9:** Ấn S to Start restore [![file](/img/migrated/image-1695294824016-efac8f85.png)](/img/migrated/image-1695294824016-efac8f85.png)
