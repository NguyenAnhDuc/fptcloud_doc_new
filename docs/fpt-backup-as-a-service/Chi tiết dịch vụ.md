# Chi Tiết Dịch Vụ

## Client Backup Server
FPT Cloud cung cấp server backup is cấu hình and triển khai trên hệ thống on-premise is sử dụng the hệ thống ảo hóa như VMware or Hyper V. Máy chủ backup is quản lý and điều hành from xa bởi quản trị viên of FPT Cloud  
FPT Cloud cung cấp as follows:
  * Monitor server - tính khả dụng, hiệu suất and status.
  * Quản trị thiết bị from xa
  * Bảo trì hệ thống, hệ điều hành and công cụ backup

## BaaS Portal
FPT Cloud cung cấp permission truy cập ando cổng web địch vụ BaaS. Portal will cung cấp những tiện ích sau:
  * Monitor and xem the server, hệ thống has been is bảo vê
  * Monitor tiến trình, lich sử and status job backup
  * View chi tiết hệ thống server như kích thước backup, lịch trình, the điểm restore 
  * Chủ động restore dữ liệu 
  * Thông báo tiến trình job backup, tài nguyên status kết nối thông qua email 

## Definitions
  * **Agent:** phần mềm in hệ thống thực hiện the thao tác bảo vệ dữ liệu of the thiết bị vật lý and ảo hóa
  * **VBR server:** Giao tiếp with tất cả the server in hệ thống and điều phối the hoạt động như bảo vệ, restore, the hoạt động quản trị, quản lý job backup, quản lý sự kiện tiến trình.
  * **Data Compression:** Giả pháp tiết kiểm dữ liệu làm giảm dung lượng lưu trữ
  * **Data Encryption:** Giải pháp security dữ liệu
  * **Data Replication:** Sao chép dữ liệu làm bản sao of the server in hệ thống người dùn. Các bản sao this can is truy cập ngay lập tức to restore nhanh, is sử dụng tạo nhiều điểm restore mà không ảnh hưởng to hiệu suất of server.
  * **Disaster Recovery:** Việc lập kế hoạch or triển khai chiến lược to ứng phó with the lỗi như mất toàn bộ cơ sở hạ tầng or do lỗi nghiêm trọng.
  * **Backup copy:** Là a version of cũng a tệp backup and is sao chép ando kho lưu trữ to lưu trữ lâu dài. Các khó lưu trữ can is đặt in hệ thống nguồn or can is triển khai bên ngoài hệ thống. Tệp backup có cũng định dạng with tệp backup chính, vì vậy can restore dữ liệu cần thiết trực tiếp in trường hợp xảy ra sự cố.
  * **Retention Period:** Khoảng thời gian a bộ dữ liệu backup cụ thể không vượt quá tiêu chí lưu giữ of nó. Khi khoảng thời gian lưu giữ has been hết and quá trình dữ liệu bị lỗi thời, dữ liệu this will không khả dụng to restore.
  * **Retention Poins:** Số lượng job successfully is lưu trữ liên tiếp
  * **RPO:** Restore point objective. Điểm restore is xác định bởi lịch trình backup
  * **RTO:** Restore time objective: thời gian restore is xác định bởi phương pháp and lượng dữ liệu is restore
