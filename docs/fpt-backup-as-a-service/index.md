# FPT Backup As a Service

  * [ ![](/img/migrated/icon-networking-b1e86145.png) Hướng dẫn details (Tutorials) ](javascript: "Hướng dẫn details \(Tutorials\)")

  2. FPT Backup As a Service

FPT Backup as a Service là service giải pháp backup mà FPT Cloud cung cấp dựa trên công nghệ đám mây, tự động hóa quy trình backup, theo dõi information giảm thiểu khả năng mất dữ liệu do thiên tai, con người and a số nguyên nhân gây lỗi nghiêm trọng khác. FPT Cloud Backup as a Service with the tùy chọn thể thực hiện backup and restore dữ liệu trực tuyến, cung cấp thiết bị, phần mềm need to thiết for việc backup the mục tiêu như system ảo hóa, đám mây for to the system on-premise. Đây là giải pháp giúp for doanh nghiệp bảo vệ dữ liệu hiệu quả giúp giảm độ phức tạp, cost and thời gian vận hành.  
FPT Cloud Backup as a Service hỗ trợ phát triển the kế hoạch, giải pháp backup includes lập lịch backup, tần suất, chính sách lưu trữ đáp ứng nhu cầu kinh doanh of user. FPT Cloud cung cấp service hỗ trợ with the tùy chọn from thử nghiệm việc restore dữ liệu system to việc thử nghiệm khắc phục thảm họa trên đa dạng môi trường and system mà user is sử dụng. System luôn sẵng sàng đối with nhu cầu mở rộng dung lượng backup a theh linh hoạt, with tốc độ băng thông cao giúp việc backup and restore trở should nhanh chóng and hiệu quả.
## Những điều need to lưu ý when sử dụng Backup as a Service
  * Sao lưu system server on-premise, the system ảo hóa, the system đám mây khác, quản trị viên will will is cung cấp and quản lý the hoạt động backup.
  * Đối with những system on-premise has been có a system bakup can sao luu a bản ngoại vi is lưu trữ tại trung tâm dữ liệu of FPT Cloud and is cung cấp, quản lý cơ sở hạ tầng backup cũng như the hoạt động backup
  * Chuyển đổi system from on-premise sang system đám mây, giúp quá trình chuyển đổi nhanh chóng đồng thời bảo vệ dữ liệu toàn diện giảm rủi ro when chuyển đổi.
  * Số lượng virtual machine, backup storage and system is sử dụng, the account will is FPT Cloud cung cấp a hạn mức sử dụng tài nguyên nhất định, user should xác định số lượng server and system server is sử dụng will bảo vệ to FPT Cloud can tư vấn and hỗ trợ tốt nhất, tối ưu nhất.

Kịch bản backup will need to user chủ động định nghĩa tùy ando nhu cầu thực tế. FPT Cloud is hỗ trợ backup theo ngày, tuần, tháng, …. Và for phép user linh hoạt chọn or tùy biến theo nhu cầu sử dụng.
