# FPT Backup as a Service

FPT Backup as a Service is a cloud-based backup solution that automates backup workflows, monitors backup health, and minimizes data loss from hardware failure, human error, and other critical incidents. It supports online backup and restore across virtualized systems, on-premise infrastructure, and cloud environments, helping businesses protect data while reducing operational complexity and cost.

FPT Cloud BaaS supports flexible backup schedules (daily, weekly, monthly) and can be customized to meet your organization's specific recovery requirements.

## Things to know before using

- Backup applies to on-premise servers, virtualized systems, and other cloud environments. FPT Cloud provides and manages the backup infrastructure and operations.
- If you already have an existing backup system, a secondary offsite copy can be stored at FPT Cloud data centers.
- Migrations from on-premise to cloud are supported with full data protection during the transition.
- Each account has a resource quota. Identify how many servers and systems you need to protect so FPT Cloud can advise on the best plan.

Backup scenarios must be defined by you based on actual requirements. FPT Cloud currently supports daily, weekly, and monthly schedules with flexible customization options.
