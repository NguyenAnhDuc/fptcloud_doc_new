# Delete a webhook secret

:::danger
This action cannot be undone.
:::

1. Click **Action** > **Delete** on the webhook secret to delete.

[![](/img/migrated/Screenshot_1-185c8b21.png)](/img/migrated/Screenshot_1-185c8b21.png)

2. Enter `delete` and click **Confirm** to finish.

[![](/img/migrated/Picture73-ffc47e85.png)](/img/migrated/Picture73-ffc47e85.png)
