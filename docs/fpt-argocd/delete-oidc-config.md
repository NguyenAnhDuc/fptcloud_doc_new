# Delete OIDC configuration

:::danger
Deleting the OIDC configuration will disable SSO login for all users who authenticate via OIDC.
:::

FPT Cloud supports deleting the OIDC configuration via the Portal interface.

Go to **Security & Access** > **OIDC** > **Delete** to remove the OIDC configuration.
