# Create a new account on the Portal

1. On the **Security & Access** > **Local Account** screen, click **Add Account**.

[![](/img/migrated/Picture81-3e8fefb9.png)](/img/migrated/Picture81-3e8fefb9.png)

2. Enter the new account information:
   - **Username** (required): the login name — must be unique in the system
   - **Password** (required): the account password
   - **Capability**: select at least one of `apiKey` or `login`
   - **Status**: select `enable` or `disable`

[![](/img/migrated/Picture82-56c3dcc0.png)](/img/migrated/Picture82-56c3dcc0.png)

3. Click **Add** to create the account.

[![](/img/migrated/Picture83-c343b464.png)](/img/migrated/Picture83-c343b464.png)

4. Verify the account has been created in ArgoCD.

[![](/img/migrated/Picture84-3375cc1d.png)](/img/migrated/Picture84-3375cc1d.png)
