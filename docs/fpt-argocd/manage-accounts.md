# Manage accounts on ArgoCD

In addition to the admin account, FPT Cloud supports creating and managing user accounts in the system. After creation, accounts have `readonly` permission by default — they can view all ArgoCD information after logging in.

To grant additional permissions to an account, configure RBAC policy. See the **Configure RBAC policy** section.
