# Upgrading Argocd Version

FPT Cloud supports users to upgrade the **ArgoCD** version to a newer version amongst the FPT Cloud-supported versions. Currently, some supported versions including v2.8.1, v2.8.2, and v2.8.3.
**Note** : Upgrading may cause service downtime for customers' ArgoCD instance. Therefore, customers should carefully consider upgrading or not and the time to upgrade the ArgoCD cluster.
**Step 1** : Navigate to **Essential Properties** > **Instance information**
[![](/img/migrated/Picture79-fddde53e.png)](/img/migrated/Picture79-fddde53e.png)
**Step 2** : Select the **Version** that you want to upgrade to 
[![](/img/migrated/Picture80-1a5785f9.png)](/img/migrated/Picture80-1a5785f9.png)
**Step 3** : Click **Update** to upgrade the instance's version
