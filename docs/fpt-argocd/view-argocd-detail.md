# View ArgoCD instance details and log in

## View instance details

1. Click the **Instance Name** to view the instance details.

[![](/img/migrated/Picture58-fd48905e.png)](/img/migrated/Picture58-fd48905e.png)

The screen displays the details needed to access the ArgoCD cluster, including URL, username, and password.

[![](/img/migrated/Picture59-1438cb52.png)](/img/migrated/Picture59-1438cb52.png)

## Log in to the ArgoCD cluster

2. On the **Essential Properties** screen, copy the URL and access the application:
   - **URL**: from the **Essential Properties** screen
   - **Username**: from the **Essential Properties** screen
   - **Password**: from the **Essential Properties** screen

[![](/img/migrated/Picture60-3afa2ba7.png)](/img/migrated/Picture60-3afa2ba7.png)

The interface after successful login:
