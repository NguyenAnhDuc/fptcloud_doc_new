# Configure Notification Channel

FPT Cloudでは、以下の通知チャンネルの追加・編集が可能です。

## A. Telegram

### Telegram configの追加

1. **Notifications** > **Telegram** 画面を開きます。

[![](/img/migrated/Picture38-3-ecf99d40.png)](/img/migrated/Picture38-3-ecf99d40.png)

2. TelegramへのTokenを設定します。

[![](/img/migrated/Picture39-3-5750e082.png)](/img/migrated/Picture39-3-5750e082.png)

3. 設定後の結果：

[![](/img/migrated/Picture40-1-d2f2c786.png)](/img/migrated/Picture40-1-d2f2c786.png)

### Telegram configの編集

1. **Notifications** > **Telegram** 画面を開きます。

[![](/img/migrated/Picture41-1-7e331436.png)](/img/migrated/Picture41-1-7e331436.png)

2. Telegram通知用のTokenを再設定します。

[![](/img/migrated/Picture42-1-ffe0b22f.png)](/img/migrated/Picture42-1-ffe0b22f.png)

3. 設定後の結果：

[![](/img/migrated/Picture43-1-2262a51b.png)](/img/migrated/Picture43-1-2262a51b.png)

1. **Notifications** > **Telegram** 画面を開きます。
