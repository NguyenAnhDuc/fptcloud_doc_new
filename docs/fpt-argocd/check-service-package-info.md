# Check service package information

FPT Cloud provides ArgoCD service packages under **Quota Detail**.

1. On the FPT Portal menu, select **Argo CD** > **View Details** to see the available FPT ArgoCD service packages.

[![](/img/migrated/Picture2-6-053e4ca3.png)](/img/migrated/Picture2-6-053e4ca3.png)

2. The Portal displays the available service packages.

[![](/img/migrated/Picture3-7-3e2895dc.png)](/img/migrated/Picture3-7-3e2895dc.png)

By default, you can activate the service with the **Basic** package. To upgrade to a higher package, contact Sales.
