# Enable/disable anonymous user

FPT Cloud supports enabling or disabling anonymous user access to ArgoCD via the Portal interface.

When enabled, unauthenticated users can access the ArgoCD interface with read-only permissions.

Go to **Essential Properties** to enable or disable the anonymous user setting.
