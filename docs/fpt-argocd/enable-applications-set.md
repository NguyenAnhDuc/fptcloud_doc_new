# Enable Applications Set

FPT ArgoCD supports enabling the **ApplicationSet** feature, which allows managing multiple ArgoCD applications using a single ApplicationSet resource.

Go to **Configuration** > **Essential Properties** to enable the ApplicationSet feature for your ArgoCD instance.
