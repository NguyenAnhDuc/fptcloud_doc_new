# Manage anonymous user access

FPT Cloud supports managing anonymous user access for ArgoCD via the Portal interface.

When anonymous access is enabled, unauthenticated users can view ArgoCD resources with read-only permissions.

Go to **Essential Properties** to enable or disable anonymous user access for your ArgoCD instance.
