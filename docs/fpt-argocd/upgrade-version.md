# Upgrade ArgoCD version

FPT Cloud supports upgrading ArgoCD to a newer version from the list of supported versions. The Portal currently supports versions: v2.8.1, v2.8.2, v2.8.3.

:::warning
Upgrading may cause downtime for the ArgoCD service in use. Plan the upgrade at an appropriate time before proceeding.
:::

1. Go to **Essential Properties** > **Instance Information**.

[![](/img/migrated/Picture79-fddde53e.png)](/img/migrated/Picture79-fddde53e.png)

2. Select the version to upgrade to.

[![](/img/migrated/Picture80-1a5785f9.png)](/img/migrated/Picture80-1a5785f9.png)

3. Click **Update** to perform the upgrade.
