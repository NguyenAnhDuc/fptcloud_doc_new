# Add a new webhook secret

1. On the **Configuration** > **Webhook** screen, add a new webhook for GitHub, GitLab, Bitbucket, Bitbucket Server, or Gogs.

Example with GitLab:

[![](/img/migrated/Picture68-acd9722c.png)](/img/migrated/Picture68-acd9722c.png)

2. Enter the information:
   - **Secret** (required): the secret to configure the webhook for GitLab/GitHub/Bitbucket, etc.

[![](/img/migrated/Picture69-0fd8a491.png)](/img/migrated/Picture69-0fd8a491.png)

3. Click **Create** to finish.

[![](/img/migrated/Picture70-3559a527.png)](/img/migrated/Picture70-3559a527.png)
