# Edit Capabilities

FPT Cloud allows users to configure account's capabilities:
  * **login** : Allow users to login via ArgoCD
  * **apiKey** : Allow users to create authentication tokens to access via API. This allows easy integration with automated processes such as CI/CD pipelines or other automated processes that needs to interact with ArgoCD APIs

**Step 1** : From **Security & Access** > **Local Account** > **Edit account**
[![](/img/migrated/Picture85-06a2fa3e.png)](/img/migrated/Picture85-06a2fa3e.png)
**Step 2** : Make additional selections or remove existing selections
_Note: It is required to select at least one capabilities (apiKey/login) for the user_
[![](/img/migrated/Picture86-56c75d94.png)](/img/migrated/Picture86-56c75d94.png)
**Step 3** : Click Edit to complete editing
