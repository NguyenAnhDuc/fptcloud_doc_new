# Change Password

The system allows users to change the password for the created account:
_Note: Users can only create a new password when they forget their password. They cannot copy the password from the portal._
**Step 1** : From **Security & Access** > **Local Account** > **Change Password**
[![](/img/migrated/Picture88-2d97201c.png)](/img/migrated/Picture88-2d97201c.png)
**Step 2** : Enter a new password for the account:
[![](/img/migrated/Picture89-37eec235.png)](/img/migrated/Picture89-37eec235.png)
**Step 3** : Click **Change Password** to submit the new password for the account.
