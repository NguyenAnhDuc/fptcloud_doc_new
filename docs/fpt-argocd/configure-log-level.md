# Configure log level

This feature allows you to configure the log level in the Image Updater controller. The default log level is `debug`.

1. On the **Image Updater** > **Log Level** screen, configure the log level.

[![](/img/migrated/Picture6-7-4537dcac.png)](/img/migrated/Picture6-7-4537dcac.png)

2. Select the mode to update.
