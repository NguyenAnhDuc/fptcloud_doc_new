# Delete OIDC config

:::danger
After deletion, the OIDC Scope reverts to a disabled state. ArgoCD will only support login via username/password if the admin account is enabled or a local account exists.
:::

1. On the **Security & Access** > **OIDC Scope** screen, click **Delete** on the OIDC Scope you want to remove.

[![](/img/migrated/Picture30-3-7c37461f.png)](/img/migrated/Picture30-3-7c37461f.png)

2. Type `delete` to confirm the action.

[![](/img/migrated/Picture31-3-c8f7c979.png)](/img/migrated/Picture31-3-c8f7c979.png)
