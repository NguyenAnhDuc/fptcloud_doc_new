# Upgrade service package

FPT Cloud supports upgrading the service package from a lower to a higher tier.

1. Go to **Essential Properties** > **Entry Information**.

[![](/img/migrated/Picture45-2-03e170d8.png)](/img/migrated/Picture45-2-03e170d8.png)

2. View the current resource package information.

[![](/img/migrated/Picture46-2-adb76700.png)](/img/migrated/Picture46-2-adb76700.png)

[![](/img/migrated/Picture47-2-c08b76d2.png)](/img/migrated/Picture47-2-c08b76d2.png)

3. Upgrade the current service package.

[![](/img/migrated/Picture48-2-60f6beb1.png)](/img/migrated/Picture48-2-60f6beb1.png)

Select a package from the list of eligible upgrades:

[![](/img/migrated/Picture49-2-9769829e.png)](/img/migrated/Picture49-2-9769829e.png)

[![](/img/migrated/Picture50-2-06d64d49.png)](/img/migrated/Picture50-2-06d64d49.png)

4. The result after upgrading:

[![](/img/migrated/Picture51-2-e926ba73.png)](/img/migrated/Picture51-2-e926ba73.png)
