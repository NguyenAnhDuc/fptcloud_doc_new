# Change account password

:::note
You can only change the password when you have forgotten it. The current password cannot be copied from the Portal.
:::

1. On the **Security & Access** > **Local Account** > **Change Password** screen.

[![](/img/migrated/Picture88-2d97201c.png)](/img/migrated/Picture88-2d97201c.png)

2. Enter the new password.

[![](/img/migrated/Picture89-37eec235.png)](/img/migrated/Picture89-37eec235.png)

3. Click **Change Password** to finish.
