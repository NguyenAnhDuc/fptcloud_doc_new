# Manage Group

Autoscale Group includes a tập hợp the Instance Nodes đồng nhất về thông số kỹ thuật with chung a mục đích, mỗi Instance Node corresponding with a Virtual Instance. Autoscale Group cung cấp cơ chế quản lý and tự động mở rộng (tăng/giảm số lượng Instance Node in group) dựa trên Autoscale Profile has been tạo trước that.
