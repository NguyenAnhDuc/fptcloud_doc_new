# Managed GPU Cluster (Kubernetes)

**FPT Managed GPU Cluster** is built on the open-source K8s platform and automates the deployment, replication, and management of containerized applications. FPT Managed GPU Cluster fully integrates container orchestration, storage, networking, security, and PaaS components, providing you with an optimal environment to develop and deploy applications in the cloud.

**FPT Managed GPU Cluster** delivers the Managed GPU Cluster service model for FKE. With MANAGED GPU CLUSTER, FPT Cloud manages all control plane components while you deploy and manage your worker nodes. MANAGED GPU CLUSTER helps you focus on application deployment without spending resources on K8s cluster management.

**FPT Managed GPU Cluster** is a service model based on Kubernetes — an open-source platform that supports automating the deployment, replication, and management of containerized applications. In addition to fully integrating container orchestration, storage, networking, security, and PaaS components, FPT Managed GPU Cluster also provides GPU resources to support complex compute operations.

**Things to note before using Managed GPU Cluster:**
- **Managed GPU Cluster location:** The geographic location (region) can affect access speed to the servers you are using. To optimize speed, select the region closest to the objects generating traffic.
- **Number of nodes and configuration of each node:** Your FPT Cloud account has certain limits on RAM, CPU, storage, IP, and other resources. You should therefore determine the resources you need and the maximum limits so FPT Cloud can support you optimally.
