# Delete a Managed GPU Cluster

For Managed GPU Clusters that are no longer needed, you can delete them following these steps.

**Step 1:** In the menu, select **[AI Infrastructure] > [Managed GPU Cluster]** to display the **[GPU Cluster Management]** page. Select the cluster you want to access.

[![](/img/migrated/14-b4dc395d.png)](/img/migrated/14-b4dc395d.png)

**Step 2:** In the Action column of the cluster you want to delete, select **[Delete]**.

[![](/img/migrated/15-05e09c5c.png)](/img/migrated/15-05e09c5c.png)

**Step 3:** Review the warning in the popup and click **[Delete]** to confirm.

[![](/img/migrated/16-52efdb8d.png)](/img/migrated/16-52efdb8d.png)
