# Delete report

Để xóa report, user thực hiện as follows:
**Step 1** : Click **Home > Apps > Reporting**
**Step 2** : Tại màn danh sách report, click icon **Trash**
[![](/img/migrated/Picture15-c1a6f8b0.png)](/img/migrated/Picture15-c1a6f8b0.png)
**Step 3** : Xuất hiện hộp thoại xác nhận, chọn **Delete** to xóa report, chọn **Cancel** to hủy bỏ thao tác 
[![](/img/migrated/Picture16-19c4a8c7.png)](/img/migrated/Picture16-19c4a8c7.png)
