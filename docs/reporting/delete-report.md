# Delete a report

1. Go to **Home > Apps > Reporting**.
2. In the report list, click the **Trash** icon for the report you want to delete.

   [![Report list with trash icon](/img/migrated/Picture15-c1a6f8b0.png)](/img/migrated/Picture15-c1a6f8b0.png)

3. A confirmation dialog appears. Click **Delete** to confirm, or **Cancel** to abort.

   [![Delete confirmation dialog](/img/migrated/Picture16-19c4a8c7.png)](/img/migrated/Picture16-19c4a8c7.png)
