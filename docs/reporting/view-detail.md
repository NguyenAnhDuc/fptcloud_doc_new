# View chi tiết report

**Step 1** : Click **Home > Apps > Reporting**
[![Alt text](/img/migrated/Picture11-f73c4b93.png)](/img/migrated/Picture11-f73c4b93.png)
List report has been tạo is displayed dưới dạng bảng, includes the thông tin: 
  * **Name** : Tên report has been tạo 
  * **Report status** : Có 2 status of report là **Active** and **Draft**
  * **Dashboard name** : Hiển thị tên dashboard has been chọn 
  * **Recipient** : Hiển thị email of người nhận 
  * **Created date** : Hiển thị thời điểm tạo or chỉnh sửa report lần cuối cùng tính to hiện tại 
  * **Last sent** : Hiển thị ngày gửi email report cuối cùng for to hiện tại 
  * **Schedule** : Hiển thị lịch gửi email report 
  * **Schedule status** : Hiển thị status of việc gửi report theo lịch trình. Bao gồm: **Active** or **Inactive**. Đối with Report ở status **Draft** => Không displayed Schedule status. 

**Step 2** : Click chi tiết ando tên report to xem chi tiết. 
[![Alt text](/img/migrated/Picture12-94a2f87d.png)](/img/migrated/Picture12-94a2f87d.png)
Information report is displayed, includes: 
  * **Tên report**
  * **Status report** : Hiển thị status of report. 
    * **Active** : Trạng thái when user tạo report successfully 
    * **Draft** : Trạng thái when user chọn **Save as Draft**
  * **Initialization time** : Hiển thị thời gian tạo or chỉnh sửa report lần cuối cùng tính to hiện tại report. 
  * **Status Schedule** : Hiển thị status of lịch trình 
    * **Scheduled** : Trạng thái when has been đặt lịch gửi report
    * **Expired** : Trạng thái when report has been quá hạn gửi 
    * **Draft** : Trạng thái when report is ở Draft 
    * **Paused** : Trạng thái when schedule vẫn is in lịch trình, nhưng user tắt status (Tạm dừng việc gửi report)
