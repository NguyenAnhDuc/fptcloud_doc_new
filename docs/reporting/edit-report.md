# Edit a report

1. Go to **Home > Apps > Reporting**.
2. Click a report name to open its details.
3. On the **Detail report** screen, click the **Pencil** icon in the section you want to edit — **Dashboard**, **Schedule**, or **Format report**.

   The system navigates to that section so you can update the relevant information.
