# Edit report

**Step 1** : Click **Home > Apps > Reporting**
**Step 2** : Click ando tên report to xem chi tiết. 
**Step 3** : Hiển thị màn **Detail report** , to chỉnh sửa report, user click ando icon **Pencil** in fromng phần Dashboard, Schedule, Format report.
System will nhảy to fromng phần corresponding to user chỉnh sửa thông tin mong muốn.
