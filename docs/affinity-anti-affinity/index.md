##  Introduction 
Affinity and anti-affinity là hai khái niệm quan trọng liên quan to việc đặt the virtual machine trên the server. Để thực hiện điều this, you can tạo the nhóm server (instance groups) with chính sách affinity or anti-affinity. Khi you khởi chạy a virtual machine, you can truyền information về nhóm server như a gợi ý to thực hiện việc đặt virtual machine theo chính sách mong muốn.
