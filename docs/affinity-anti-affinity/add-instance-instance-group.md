# Add Instance to Instance Group

When creating a new virtual machine, you can assign it to an Instance Group to apply an Affinity or Anti-Affinity policy.

:::note
Each Instance Group holds up to 10 Instances.
:::

1. On the **Create Instance** screen, in the **Instance Group** section, select the Instance Group name.

   ![Select Instance Group when creating Instance](/img/migrated/image-1705025405311-fa794fe3.png)

2. Click **Create Instance**.

The Instance Group information appears on the **Instance Detail** page.

![Instance assigned to Instance Group](/img/migrated/image-1705025498532-8edf7a7b.png)
