# Model Details

View the details of a model. A model includes the following components:

- **Model title**: Name of the model.
- **Description**: Description of the model.
- **Latest version**: The most recently created version at the time of viewing.
- **Created date**: Date the model was created.
- **Modified**: Date the model information was last updated.
- **Model visibility**: Whether the model is Private, Public, or Public metadata.
- **Tags**: Tags attached to the model.
- **Model card**: Overview information about the model.
[![](/img/migrated/Picture5-2bcf13c8.png)](/img/migrated/Picture5-2bcf13c8.png)

- **Version**:
  - **Version**: Version code.
  - **Created date**: Date the version was created.
  - **Size**: Storage size.
  - **Type**: Whether the version is Private or Public.
[![](/img/migrated/Picture6-db90f2cf.png)](/img/migrated/Picture6-db90f2cf.png)

- **File Browser**: Displays uploaded files for each version.
  - **File**: File name.
  - **Modified date**: Date last modified.
  - **Size**: File size.
[![](/img/migrated/Picture7-4a2eec64.png)](/img/migrated/Picture7-4a2eec64.png)

- **Share**:
  - **Email**: Email of the user the model is shared with.
  - **Name**: User's name.
  - **Role**: Permission level of the shared user.
  - **Date invited**: Date the user was added to the model.
  - **Date joined**: Date the user first viewed the model details.
  - **Action**: Actions to Update or Delete a shared user.
[![](/img/migrated/Picture8-34374d24.png)](/img/migrated/Picture8-34374d24.png)
