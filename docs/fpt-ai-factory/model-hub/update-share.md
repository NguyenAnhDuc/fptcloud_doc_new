# Update Share

Update the permissions of a user who has been shared access to a model.

## Steps

1. Click on a model to view its details, then select the **Share** tab.

2. Click the 3-dot icon in the **Action** column, then select **Update**.
[![](/img/migrated/Picture33-2ec2c209.png)](/img/migrated/Picture33-2ec2c209.png)

3. The update form appears. You can edit the following field:
   - **Role**: Modify the user's permission level.
