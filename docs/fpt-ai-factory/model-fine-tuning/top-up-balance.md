# Top Up Balance

* 下のメニューで、「課金」をクリックして課金情報と設定にアクセスします。 

[![file](/img/migrated/image-1742892495834-e94b7ee9.png)](/img/migrated/image-1742892495834-e94b7ee9.png)
  * モデルの微調整サービスの料金が確認できます。 [![file](/img/migrated/image-1742892509465-81e04abd.png)](/img/migrated/image-1742892509465-81e04abd.png)
  * **1,500,000 VND** や**2,500,000 VND** などの推奨額をチャージするか、希望の金額を入力するかを選択できます。最低取引額は**26,000 VND** です。できたら、「購入」をクリックします。 
  * **VNPay** 支払いにリダイレクトされ、選択した金額で取引完了することができます。支払い方法は**VNPayQR** または国際支払いカードなど 

[![file](/img/migrated/image-1742892528929-dc30d234.png)](/img/migrated/image-1742892528929-dc30d234.png)
  * チャージ後、クレジット履歴で、チャージ取引履歴を確認します。 [![file](/img/migrated/image-1742892550518-3a6ee216.png)](/img/migrated/image-1742892550518-3a6ee216.png)
