# Quick Start

## Quick Start

Quick start guide for **Model Testing – Interactive Sessions**:

1. [Sign up for an account](/docs/fpt-ai-factory/model-testing-interactive-sessions/sign-up-for-an-account) at <https://ai.fptcloud.com/> or <https://ai.fptcloud.jp/>.
2. Follow the [Step By Step](/docs/fpt-ai-factory/model-testing-interactive-sessions/step-by-step) guide to create and manage sessions.
3. Use the [Playground](/docs/fpt-ai-factory/model-testing-interactive-sessions/playground) to interact with and evaluate the model.
4. View [Completion History](/docs/fpt-ai-factory/model-testing-interactive-sessions/completion-history) to track completed interactions.
