# Overview

**Model Testing – Interactive Sessions** is a service within FPT AI Factory Portal that allows you to test and evaluate fine-tuned AI models in a direct, interactive chat environment.

This service helps you:

* Simulate real user interactions with the model.
* Evaluate response quality, tone, and contextual relevance.
* Detect edge cases before deployment.
* Integrate the model into applications via API endpoint.

To get started, see [Quick Start](/docs/fpt-ai-factory/model-testing-interactive-sessions/quick-start) or follow the [Step By Step](/docs/fpt-ai-factory/model-testing-interactive-sessions/step-by-step) guide.
