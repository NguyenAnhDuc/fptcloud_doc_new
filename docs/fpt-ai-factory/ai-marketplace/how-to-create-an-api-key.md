# Create an API key

To create and manage API keys, go to **My Account → My API Keys** in the AI Marketplace dashboard.

:::tip
For a full step-by-step walkthrough including creating, editing, and revoking keys, see the [My account](/docs/fpt-ai-factory/ai-marketplace/my-account) guide.
:::
