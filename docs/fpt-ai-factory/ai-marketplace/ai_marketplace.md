# AI Marketplace

AI Marketplace is FPT Cloud's Model-as-a-Service platform, giving you instant access to pre-trained AI models via API — no infrastructure setup required.

## What you can do

- **Run inference** — call LLM, VLM, TTS, STT, and embedding models via a standard REST API.
- **Fine-tune models** — customize foundation models for your use case using LoRA.
- **Deploy custom models** — serve your fine-tuned LoRA adapters as dedicated API endpoints.
- **Test in Playground** — experiment with any model interactively before writing code.
