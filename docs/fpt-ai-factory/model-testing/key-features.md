# Key features

- **Prompt testing:** Submit prompts and observe model completions.
- **System/user message customization:** Define roles and control the conversation context.
- **Parameter tuning:** Adjust settings such as temperature, top P, stop sequences, and output length.
- **Image input (VLM):** Upload .jpg or .jpeg files for Vision-Language Models.
- **Completion history:** View, compare, and manage past results.
- **Session management:** Easily start, continue, or delete interactive sessions.
