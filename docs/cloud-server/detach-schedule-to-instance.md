# Detach an Instance from an Instance Schedule

After detaching, the instance retains its current state until you perform manual operations.

## Detach an Instance

1. In **Instance Schedule Management**, click **Action** → **Apply instances**.
2. Click the **x** next to the instance name to detach it (or click **x** at the end of the listbox to detach all).
3. Click **Apply instances**.
