# Gateway

A Gateway manages network connectivity between a VPC and the internet or external networks.

:::note
This feature is only available to certain Tenants with special configurations. Contact the support team to have it activated.
:::

## Accessing Gateway

1. Log in to [console.fptcloud.com](https://console.fptcloud.com).
2. Select the VPC whose Gateway data you want to view.
3. In the sidebar, expand the **Network** menu and select **Gateway**.
