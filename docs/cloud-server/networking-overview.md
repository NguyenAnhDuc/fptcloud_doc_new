# Networking Details

The FCI network architecture is designed to meet diverse requirements — from traditional networking systems to VPC or the specialized No-gateway model.

## No-gateway Model

You can set up your own system and manage it through a server acting as a router (for example, a Check-point firewall). IPs are assigned directly to the server and traffic is routed through the firewall.
