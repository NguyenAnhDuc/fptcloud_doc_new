# Edit a Storage Disk

You can edit the name, storage type, or increase the size of a Storage Disk when you need to change its storage configuration.

:::note
You can only edit a Storage Disk that is not currently attached to any Instance.
:::

## Edit a Storage Disk

1. In the Storage Disk list, click **Actions** → **Edit**.

   [![Actions menu with Edit option](/img/migrated/image-1712215559562-97286a15.png)](/img/migrated/image-1712215559562-97286a15.png)

2. Update the details:
   - **Name**: Storage Disk name.
   - **Storage Policy**: Storage type.
   - **Size**: Storage size.

   Click **Edit storage disk** to save.

   [![Edit Storage Disk dialog](/img/migrated/image-1712215566139-704f934e.png)](/img/migrated/image-1712215566139-704f934e.png)

:::warning
The new size must be larger than the current size — you cannot reduce a Storage Disk's capacity.
:::
