# Create a Token

Creating a Token lets you authenticate when using Terraform or the API to manage resources on FPT Cloud.

## Create a Token

1. On the **Token** page, click **Create token**.

   [![Token page with Create token button](/img/migrated/image-1731024975251-9874a41d.png)](/img/migrated/image-1731024975251-9874a41d.png)

2. Enter the details:
   - **Token name**: Maximum 50 characters (letters, numbers, `-`, `_`, `.`, and spaces).
   - **Expiry time**: Select from the list.

   [![Create token dialog](/img/migrated/image-1731024985103-48bcc2ab.png)](/img/migrated/image-1731024985103-48bcc2ab.png)

3. Click **Create**.

4. The system displays your Token. Copy and store it securely.

   [![Token created successfully](/img/migrated/image-1731024996987-252d326d.png)](/img/migrated/image-1731024996987-252d326d.png)

:::danger
The Token is displayed only once. After clicking **Done**, you cannot view it again. FPT Cloud Portal does not store your Token.
:::
