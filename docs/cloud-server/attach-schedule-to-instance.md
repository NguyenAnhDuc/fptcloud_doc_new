# Attach an Instance to an Instance Schedule

## Attach an Instance

1. In **Instance Schedule Management**, click **Action** → **Apply instances**.

   [![Action menu](/img/migrated/image-1695609073911-03ac0ac2.png)](/img/migrated/image-1695609073911-03ac0ac2.png)

2. Select the instances to attach, then click **Apply instances**.

   [![Select instances](/img/migrated/image-1695609084366-c567a73d.png)](/img/migrated/image-1695609084366-c567a73d.png)
