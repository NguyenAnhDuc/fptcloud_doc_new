# HPC Portal Quick Start

HPC Portal lets you manage physical GPU Servers for RTX 3090 and RTX 4090 lines. Follow the basic steps below to get started.

## Prerequisites

- ✅ You have an HPC Portal account (contact your administrator to be granted access).
- ✅ GPU resources have been provisioned for your account.

## Getting Started

1. Go to [HPC Portal](https://hpc.fptcloud.com/) and log in.
2. On the **Hosts** screen, select an image and click **Start** to boot the machine.
3. After the machine starts, download the **RDP** file to connect via Remote Desktop.
