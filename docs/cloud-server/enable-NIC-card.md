# Enable Network Interface Card

You can re-enable a previously disabled NIC card to restore network connectivity through that interface.

## Enable a NIC card

1. Navigate to the **Instance Management** page and select your Instance.
2. Go to the **NIC** tab.
3. Locate the disabled NIC card, then select **Actions** → **Enable**.
4. Confirm the action in the dialog box.

The NIC card status changes to **Enabled** and network connectivity is restored.
