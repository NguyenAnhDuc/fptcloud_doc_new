# Using Services with PAYG

When your prepaid resources are exhausted, you can continue creating resources using PAYG (Pay As You Go) billing.

## Services That Support PAYG

1. **Instances**: Create, clone, resize, create snapshot/template.
2. **Storage**: Create, resize, create snapshot from storage.
3. **IP**: Allocate IP.
4. **Load Balancer v1** (General only).
5. **Auto Scaling**.

## How to Use

When creating a new resource, if the quota is exhausted, the system notifies you of which resource types have run out and automatically switches to PAYG billing.
