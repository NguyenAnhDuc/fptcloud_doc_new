# Change Machine Configuration

When your requirements change and the current server configuration is either too high or too low, you can use the Change Package feature to adjust it without affecting your machine's setup.

:::warning
You must shut down the machine before using the Change Package feature.
:::

## Change Configuration

1. Select the image you want to modify, then click **Change package**.
2. Select the new configuration package that suits your needs and confirm.

:::note
After changing the package, verify the GPU card on the machine and update to the latest NVIDIA Studio Driver version to ensure stable operation.
:::
