# View Instance Snapshot Schedule Details

1. In the **Instance snapshot schedule** tab, click the schedule name.

   [![Select schedule](/img/migrated/image-1766044475260-90c9037f.png)](/img/migrated/image-1766044475260-90c9037f.png)

2. View the information: Name, description, start/end date, frequency, and instance list.

   [![Schedule details](/img/migrated/image-1766044510198-7f67e63b.png)](/img/migrated/image-1766044510198-7f67e63b.png)
