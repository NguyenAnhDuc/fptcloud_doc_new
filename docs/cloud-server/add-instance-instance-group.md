Attach a Virtual Machine to an Instance Group


_This feature applies to users using General and Specific service types._

When a user launches a virtual machine, they can pass instance group information as a hint to place the virtual machine according to the desired policy.

To do this, follow these simple steps:

**Note: Each instance group can be attached to a maximum of 10 instances.**

**Step 1**: In the menu, select **Instance Management** > **Create instance**. In the **Instance Group** section, select the **Name** of the instance group where you want to place the virtual machine.

![file](images/add-instance-instance-group/img-001.png)

**Step 2**: Click **Create instance**. The system will initialize and notify you of the result.

If successful, the instance group information will be displayed on the Instance Detail page.

![file](images/add-instance-instance-group/img-002.png)
