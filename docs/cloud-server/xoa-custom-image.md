# Delete a Custom Image

When a Custom Image is no longer needed, you can delete it from the system.

:::danger
A deleted Custom Image cannot be recovered. Make sure no Instances are currently using this image before deleting it.
:::

## Delete a Custom Image

1. In **Custom Image**, click **Actions** → **Delete**.

   [![Actions menu with Delete option](/img/migrated/image-1734596886940-61bbf341.png)](/img/migrated/image-1734596886940-61bbf341.png)

2. Confirm in the popup, then click **Delete custom image**.

   [![Delete Custom Image confirmation dialog](/img/migrated/image-1734596914709-3da0ab77.png)](/img/migrated/image-1734596914709-3da0ab77.png)
