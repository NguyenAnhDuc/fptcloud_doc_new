# Revoke a Token

When a Token is no longer needed or you suspect it has been compromised, revoke it immediately to maintain security.

## Revoke a Token

1. On the **Token** page, click **Actions** → **Revoke token**.

   [![Actions menu with Revoke token option](/img/migrated/image-1731025026905-158de23e.png)](/img/migrated/image-1731025026905-158de23e.png)

2. Confirm the revocation in the dialog.

   [![Revoke token confirmation dialog](/img/migrated/image-1731025048482-957c5ad2.png)](/img/migrated/image-1731025048482-957c5ad2.png)

3. The system revokes the Token automatically.

   [![Token revoked](/img/migrated/image-1731025071939-e64b80bc.png)](/img/migrated/image-1731025071939-e64b80bc.png)
