# Frequently Asked Questions about Tokens

## Why can't some Tokens be revoked?

Some older Tokens created before a FPT Cloud Portal upgrade do not support revocation. The system displays a tooltip: "Tokens created earlier cannot be revoked".

## When does a Token change to "Expired" status?

When a Token passes its expiry time (calculated from the time it was created), the system automatically changes its status to **Expired**. You will no longer be able to authenticate using that Token.
