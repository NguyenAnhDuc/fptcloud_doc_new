# Subnet Management

A Subnet is a sub-network within a VPC — equivalent to a LAN in a physical system. You can attach one or more Subnets to an instance depending on your requirements.

## Creating a New Subnet

1. Go to **Networking** → **Subnets** and click **Create subnet**.

   [![Subnets screen](/img/migrated/image-1716262599480-f51ee2ed.png)](/img/migrated/image-1716262599480-f51ee2ed.png)

2. Enter the required information: Name, Type (Routed/Isolated), CIDR, Gateway IP, DNS.

3. Click **Create subnet**.
