# Delete an Instance Schedule

:::danger
Deleting a schedule cannot be undone. A schedule that is currently running cannot be deleted.
:::

## Delete a Schedule

1. In **Instance Schedule Management**, click **Action** → **Delete**.

   [![Action Delete menu](/img/migrated/image-1695609049584-8754accf.png)](/img/migrated/image-1695609049584-8754accf.png)

2. Type **delete**, then click **Delete schedule**.

   [![Confirm delete schedule](/img/migrated/image-1695609061103-74aa2b28.png)](/img/migrated/image-1695609061103-74aa2b28.png)

Attached instances are released and retain their current state.
