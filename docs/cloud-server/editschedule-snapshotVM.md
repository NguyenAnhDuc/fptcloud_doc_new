# Edit an Instance Snapshot Schedule

1. In the **Instance snapshot schedule** tab, click **Action** → **Edit**.

   [![Action Edit menu](/img/migrated/image-1766044561523-30e56e38.png)](/img/migrated/image-1766044561523-30e56e38.png)

2. Update the information as needed.

   [![Edit schedule](/img/migrated/image-1766044589323-730f2f46.png)](/img/migrated/image-1766044589323-730f2f46.png)

3. Click **Update schedule**.
