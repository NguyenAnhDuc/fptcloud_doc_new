# Delete a template

When you no longer need a template, you can delete it to free up resources.

:::danger
Deleted templates cannot be recovered.
:::

## Delete a template

1. On the **Custom Images** page, select **Actions** → **Delete custom image** on the template you want to delete.

   [![Actions menu with Delete custom image option](/img/migrated/image-1719539882859-4f7b095b.png)](/img/migrated/image-1719539882859-4f7b095b.png)

2. Confirm the deletion in the dialog box.
