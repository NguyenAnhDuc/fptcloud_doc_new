# Delete a virtual machine

When you no longer need a VM, you can delete it from the system to avoid incurring additional costs.

:::danger
Deleting a VM permanently removes all data and cannot be recovered.
:::

## Delete a VM

1. In **Instance Management**, select **Actions** → **Delete**.

   [![Actions menu with Delete option](/img/migrated/image-1712200768564-6389b6eb.png)](/img/migrated/image-1712200768564-6389b6eb.png)

   [![Confirm Delete selection](/img/migrated/image-1712200780901-998413fe.png)](/img/migrated/image-1712200780901-998413fe.png)

2. A warning dialog displays the VM name. Type **delete** to confirm.

   [![Delete confirmation dialog](/img/migrated/image-1712200787729-9546cde9.png)](/img/migrated/image-1712200787729-9546cde9.png)

   [![Type delete to confirm](/img/migrated/image-1712200798790-c16164fa.png)](/img/migrated/image-1712200798790-c16164fa.png)

3. The system proceeds to delete and reclaim the VM resources.

   [![VM being deleted](/img/migrated/image-1712200803567-91c8fdd5.png)](/img/migrated/image-1712200803567-91c8fdd5.png)
