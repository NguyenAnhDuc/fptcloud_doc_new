# Edit a Storage Disk Snapshot Schedule

1. In the **Storage snapshot schedule** tab, click **Action** → **Edit**.

   [![Action Edit menu](/img/migrated/image-1766045109643-13e019dd.png)](/img/migrated/image-1766045109643-13e019dd.png)

2. Update the information as needed.

   [![Edit schedule](/img/migrated/image-1766045137152-94c1a305.png)](/img/migrated/image-1766045137152-94c1a305.png)

3. Click **Update schedule**.
