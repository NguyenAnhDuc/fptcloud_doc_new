# Priority Rule Management

:::note
This feature is only available to certain Tenants with special configurations. Contact the support team to have it activated.
:::

Priority Rules let you set the execution order for firewall rules — determining which rule is evaluated first when allowing or denying traffic.

## Benefits

- Arrange rules by importance.
- Ensure specific rules are processed before general ones.

## Configuring Priority

1. In the Security Group, select the rule you want to change.
2. Set the **Priority** value (lower number = higher priority).
3. Click **Save**.
