# Managing GPU with Portal Console

FPT GPU Server is a virtual machine service deployed on FPT Cloud with dedicated GPU cards — delivering superior computing performance for AI, Machine Learning, graphics rendering, and image processing.

The service combines the flexibility of FPT Cloud Server with the powerful performance of GPU card lines such as NVIDIA Tesla A10, A30, A100, RTX 3090, and RTX 4090.

:::tip
If you need to use GPU services or encounter technical issues, contact the customer support team for assistance.
:::
