# Delete a virtual machine

When you no longer need a virtual machine, you can delete it from the system to avoid incurring additional costs.

:::danger
Deleting a virtual machine permanently removes all data and cannot be undone. Make sure the virtual machine is in **Unlock Deletion** state before proceeding.
:::

## Delete a virtual machine

1. In the **Instance Management** dashboard, select **Actions** → **Delete**.

   [![Actions menu with Delete option](/img/migrated/image-1712200768564-6389b6eb.png)](/img/migrated/image-1712200768564-6389b6eb.png)

   [![Confirm Delete selection](/img/migrated/image-1712200780901-998413fe.png)](/img/migrated/image-1712200780901-998413fe.png)

2. A confirmation dialog displays the virtual machine name. Type **delete** to confirm.

   [![Delete virtual machine confirmation dialog](/img/migrated/image-1712200787729-9546cde9.png)](/img/migrated/image-1712200787729-9546cde9.png)

   [![Type delete to confirm](/img/migrated/image-1712200798790-c16164fa.png)](/img/migrated/image-1712200798790-c16164fa.png)

3. The system deletes the virtual machine and reclaims its resources.

   [![Virtual machine being deleted](/img/migrated/image-1712200803567-91c8fdd5.png)](/img/migrated/image-1712200803567-91c8fdd5.png)
