# Delete a Storage Disk

When a Storage Disk is no longer needed, you can delete it to avoid incurring additional costs.

:::danger
You can only delete a Storage Disk that is not currently attached to any Instance. Deleted data cannot be recovered.
:::

## Delete a Storage Disk

1. In the Storage Disk list, click **Actions** → **Delete**.

   [![Actions menu with Delete option](/img/migrated/image-1712215573891-02608fa9.png)](/img/migrated/image-1712215573891-02608fa9.png)

2. Review the details in the popup, then click **Delete storage disk**.

   [![Delete Storage Disk confirmation dialog](/img/migrated/image-1712215582910-1868700d.png)](/img/migrated/image-1712215582910-1868700d.png)
