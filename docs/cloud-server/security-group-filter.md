# Filter Security Group

The Filter feature lets you quickly search and narrow down the Security Group list using multiple criteria: name, tag, IP, status, firewall type, and creation date.

## Filter Criteria

| Field | Description |
|-------|-------------|
| **Name** | Enter a full or partial name. For example, enter `sec` to find "security-group-9609". |
| **Tag** | Filter by attached tag. |
| **IP** | Filter by IP address. |
| **Status** | Filter by status. |
| **Type** | Security Group or Distributed Firewall. |
| **Created at** | Filter by creation date. |
