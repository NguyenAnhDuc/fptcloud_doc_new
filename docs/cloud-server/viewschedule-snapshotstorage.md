# View Storage Disk Snapshot Schedule Details

1. In the **Storage snapshot schedule** tab, click the schedule name.

   [![Select schedule](/img/migrated/image-1766045049451-2b6c1676.png)](/img/migrated/image-1766045049451-2b6c1676.png)

2. View the information: Name, description, start/end date, frequency, and storage disk list.

   [![Schedule details](/img/migrated/image-1766045074035-865020b6.png)](/img/migrated/image-1766045074035-865020b6.png)
