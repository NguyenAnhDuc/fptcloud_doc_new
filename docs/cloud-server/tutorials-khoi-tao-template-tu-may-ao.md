# Create a template from a virtual machine

Creating a template from a pre-configured virtual machine lets you deploy multiple virtual machines with the same environment quickly.

## Create a template

1. In **Instance Management**, select **Actions** → **Create Template**.

   [![Actions menu with Create Template option](/img/migrated/image-1715855344608-be2905c7.png)](/img/migrated/image-1715855344608-be2905c7.png)

2. Fill in the template details, then click **Create template**.

3. The system redirects you to the **Custom Images** page to monitor the progress.

   [![Template creation progress](/img/migrated/image-1715855896164-3bb4e6c2.png)](/img/migrated/image-1715855896164-3bb4e6c2.png)

   [![Template created successfully](/img/migrated/image-1715855902172-92a593fa.png)](/img/migrated/image-1715855902172-92a593fa.png)
