# Cost Explorer

Cost Explorer helps you analyze and track FPT Cloud service costs across multiple dimensions — by service, resource type, VPC, or tag. This is especially useful when you need to control cloud spending and optimize your budget.

:::note
Costs shown in Cost Explorer are customized based on the filters you set in Customize Filter and may differ from the actual billed amount in the end-of-month invoice.
:::
