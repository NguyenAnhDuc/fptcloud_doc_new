# Why do I have multiple reconciliation entries in the same billing period?

If the Project/VPC resources within a Tenant are split into multiple billing groups for separate payment processing, the system displays one reconciliation entry per billing group.

:::tip
Contact FCI Sale Support or the FCI team for assistance if you need more information about billing group configuration.
:::
