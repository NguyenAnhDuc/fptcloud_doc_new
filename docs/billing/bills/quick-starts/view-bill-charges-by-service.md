# View charges by service

The **Charges by service** tab lets you view the billed quantity, pricing model, unit price, and total cost for each service. This data corresponds to the **Invoice** sheet in the downloaded reconciliation file.

[![Charges by service screen](/img/migrated/image-1744088527526-341fc3c4.png)](/img/migrated/image-1744088527526-341fc3c4.png)

## Filter data

You can filter Charges by service data using the following criteria:

- **Invoice group**
- **Payment type**
- **Service category**
- **Service type**
- **Service name**

[![Filter panel for Charges by service](/img/migrated/image-1744088537330-29a3fbfc.png)](/img/migrated/image-1744088537330-29a3fbfc.png)

Filter results show only services that satisfy all selected conditions (AND logic). The total amount is the sum of all services matching the filter.

[![Filtered Charges by service results](/img/migrated/image-1744088542322-9ec1fe0b.png)](/img/migrated/image-1744088542322-9ec1fe0b.png)
