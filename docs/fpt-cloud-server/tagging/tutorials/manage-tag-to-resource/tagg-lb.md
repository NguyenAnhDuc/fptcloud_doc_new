# Manage tags for Load Balancer

Assigning tags to Load Balancers helps you categorize load balancers by project or environment — making network infrastructure management easier.

## Assign tags when creating a Classic Load Balancer

1. Select **Classic load balancer** from the menu, then click **Create load balancer**.

   [![Classic load balancer screen with Create load balancer button](/img/migrated/image-1721034946476-8389286e.png)](/img/migrated/image-1721034946476-8389286e.png)

2. In the **Create load balancer** dialog, select tags in the **Add tag (optional)** field.

   [![Create load balancer dialog with Add tag field](/img/migrated/image-1721034960349-bc880785.png)](/img/migrated/image-1721034960349-bc880785.png)

## Manage tags for an existing Load Balancer

1. On the **Classic Load Balancer** screen, select a load balancer, then click **Actions** → **Manage tags**.

   [![Actions menu with Manage tags option for Load Balancer](/img/migrated/image-1721034970802-6fc88cac.png)](/img/migrated/image-1721034970802-6fc88cac.png)

2. A dialog appears. Select tags to assign to the Load Balancer.

   [![Manage tags dialog for Load Balancer](/img/migrated/image-1721034981643-db1aeca2.png)](/img/migrated/image-1721034981643-db1aeca2.png)

3. To remove a tag, click the **x** next to the tag name. Click the **x** at the end of the list box to remove all tags at once. Click **Save** to confirm.

   [![Manage tags dialog with remove option](/img/migrated/image-1721034991057-8426f650.png)](/img/migrated/image-1721034991057-8426f650.png)

:::warning
When a Load Balancer is deleted, the system automatically removes all associated tags and updates the tag list on the Tagging screen.
:::
