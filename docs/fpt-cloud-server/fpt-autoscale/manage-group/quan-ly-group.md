# Manage Group

An Autoscale Group is a collection of Instance Nodes that are uniform in technical specifications and share a common purpose, where each Instance Node corresponds to one Virtual Instance. An Autoscale Group provides mechanisms for managing and automatically scaling (increasing or decreasing the number of Instance Nodes) based on a previously created Autoscale Profile.
