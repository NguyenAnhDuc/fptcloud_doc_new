# What is FPT Cloud Desktop?

FPT Cloud Desktop (FCD) is a flexible, secure virtual desktop service built on the FPT Cloud platform, developed in-house by FPT Smart Cloud. You can access your virtual desktop from a wide range of devices and operating systems, from anywhere with an Internet connection.

## Advantages over traditional PCs

- **Data security:** Encrypted storage, centralized management, and access control
- **Continuous security patching:** Centralized and consistent updates that save time and resources
- **Data integrity during incidents:** Automatic backups combined with disaster recovery solutions
- **Fast and flexible resource scaling:** Provision thousands of virtual desktops in just a few hours

## Supported connection devices

- Desktop, PC, Laptop
- Mobile (Smartphone) or Tablet
- ThinClient or FPT MiniPC

## Access methods

- **Native FCDClient (recommended):** Access your virtual desktop directly through the application for the smoothest and most stable experience.
- **Web browser:** Quick access through a browser with no installation required.

## Terms & definitions

| Terms (English) | Terms (Vietnamese) | Abbreviation | Definition |
|---|---|---|---|
| **Virtual Desktop** | Máy ảo / Máy tính ảo | VD | A virtual machine that provides software features and performance comparable to a standard physical computer. Virtual machines share the FPT Cloud infrastructure. Each user can be provisioned with and use multiple virtual machines simultaneously, each with different configurations and operating systems. |
| **Desktop Pool** | Nhóm máy ảo | Pool | A group of multiple virtual machines sharing common Cloud processing and storage infrastructure. Typically used to organize virtual machines by department, function, usage schedule, etc. |
| **Operating System** | Hệ điều hành | OS | Examples: Microsoft Windows, macOS |
| **End-User / User** | Người dùng cuối | N/A | An end user of FCD virtual machines. Can log in to the FCD User Portal, access provisioned virtual machines, and use them like standard physical computers. |
| **Supported Device** | Thiết bị hỗ trợ | N/A | A device that can access and use a provisioned virtual machine. |
| **Local Device** | Thiết bị kết nối | N/A | The device used to establish a connection to a virtual machine. |
| **Local Area Network** | Mạng nội bộ | LAN | A network system used to connect computers within an organization's premises. |
| **Authenticator App** | Phần mềm xác thực | N/A | An application that generates a 6-digit random verification code used for account login, valid for 30 seconds. |
| **Active Directory** | Quản lý thư mục và xác thực | AD | A centralized user management system. |
