# Register an Account

Follow the steps below to register an FPT ID account on FPT Cloud Portal.

1. Go to [console.fptcloud.com](https://console.fptcloud.com) and register an FPT ID account.

[![FPT ID account registration page](/img/migrated/image-1694073641641-fb287cef.png)](/img/migrated/image-1694073641641-fb287cef.png)

2. Confirm your registration by checking the inbox of the email address you used to register and following the instructions in the confirmation email.
