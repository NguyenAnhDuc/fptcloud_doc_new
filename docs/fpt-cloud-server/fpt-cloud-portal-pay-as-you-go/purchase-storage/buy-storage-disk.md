# Purchase a Storage Disk

After topping up your account, you can purchase a storage disk.

:::note
- You must have at least 50,000 VND plus the cost of one hour of storage usage to create a disk.
- The system will immediately deduct one hour of storage usage cost upon creation.
:::

1. Go to the **Compute Engine – Storage Disks** page.

[![Storage Disks page](/img/migrated/image-1694073928674-97d41009.png)](/img/migrated/image-1694073928674-97d41009.png)

2. Click **New Storage**.

[![Create new storage](/img/migrated/image-1694073937844-c9e0ea1f.png)](/img/migrated/image-1694073937844-c9e0ea1f.png)

3. Select a **Storage policy** and the required capacity. The system estimates the hourly usage cost for your reference.

4. Click **Create storage disk**.
   - If the information is valid, the system creates the storage and redirects you to the Storage Disks page.
   - If you have reached your resource limit, the system displays an error message.

[![Storage disk creation result](/img/migrated/image-1694073957418-7a7b3dac.png)](/img/migrated/image-1694073957418-7a7b3dac.png)

[![Cost deducted after creation](/img/migrated/image-1694073966294-6d3c6da1.png)](/img/migrated/image-1694073966294-6d3c6da1.png)
