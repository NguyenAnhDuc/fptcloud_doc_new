# Change Virtual Machine Configuration

You can change the configuration of a virtual machine that is currently running.

:::note
The system will immediately deduct one hour of usage cost based on the new configuration when the change is applied.
:::

1. Go to the **Instance Management** page.

2. On the virtual machine you want to change, click **Action** and select **Resize VM**.

[![Select Resize VM](/img/migrated/image-1694073866010-73330cc8.png)](/img/migrated/image-1694073866010-73330cc8.png)

3. Select the new configuration.

[![Select new configuration](/img/migrated/image-1694073874856-91ac98f3.png)](/img/migrated/image-1694073874856-91ac98f3.png)

4. Click **Resize instance**.

[![Confirm configuration change](/img/migrated/image-1694073882943-1c379eb9.png)](/img/migrated/image-1694073882943-1c379eb9.png)
