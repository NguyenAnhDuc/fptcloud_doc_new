# Purchase a Virtual Machine

On the Instance page, you can create virtual machines and pay under the PAYG model.

:::note
You must have at least 50,000 VND plus the cost of one hour of service to create a virtual machine. The system will immediately deduct one hour of usage cost upon creation.
:::

1. Go to the **Instance Management** page.

[![Instance Management page](/img/migrated/image-1694073823080-04d5dd8a.png)](/img/migrated/image-1694073823080-04d5dd8a.png)

2. Click **New Instance**.

3. Select a virtual machine configuration:
   - Choose a configuration package; the system displays the configuration details and corresponding price.
   - The system estimates the monthly usage cost for your reference.

[![Select virtual machine configuration](/img/migrated/image-1694073838317-c01bfab4.png)](/img/migrated/image-1694073838317-c01bfab4.png)

4. Click **Create instance**.
   - If the information is valid, the system creates the virtual machine and redirects you to the Instance Management page.
   - If you have reached your resource limit, the system displays an error message and prevents further purchases.

[![Virtual machine creation result](/img/migrated/image-1694073846242-2bb9076d.png)](/img/migrated/image-1694073846242-2bb9076d.png)

[![Cost deducted after creation](/img/migrated/image-1694073853907-420d469b.png)](/img/migrated/image-1694073853907-420d469b.png)
