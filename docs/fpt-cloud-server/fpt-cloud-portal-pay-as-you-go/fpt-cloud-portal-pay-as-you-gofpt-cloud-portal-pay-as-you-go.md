# FPT Cloud Portal – Pay As You Go

On FPT Cloud Portal, you can register an account and prepay to use services under the Pay As You Go (PAYG) model — you only pay for what you use. Services are billed down to the hour, giving you flexibility and cost efficiency.

Steps to get started:

1. Register an account
2. Log in to the system
3. Select PAYG and verify your phone number
4. Create an Organization
5. Explore the Portal home page
6. Top up via gift code
7. Top up via VNPay
8. Purchase a virtual machine
9. Change virtual machine configuration
10. Purchase a Public IP
11. Purchase a storage disk
12. Change storage disk capacity
13. View billing by monthly reconciliation
14. View performed transactions
