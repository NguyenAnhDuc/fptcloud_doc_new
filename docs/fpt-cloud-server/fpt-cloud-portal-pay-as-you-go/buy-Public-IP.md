# Purchase a Public IP

After topping up your account, you can purchase additional Public IPs.

:::note
- You must have at least 50,000 VND plus the cost of one hour of IP usage to allocate an IP.
- The system will immediately deduct one hour of IP usage cost upon allocation.
:::

1. Go to the **Networking – Floating IPs** page.

[![Floating IPs page](/img/migrated/image-1694073889613-41f69bf5.png)](/img/migrated/image-1694073889613-41f69bf5.png)

2. Click **Allocate IP address**.

3. Select **Allocate new IP from pool**:
   - Choose the number of IPs to allocate.
   - The system estimates the hourly IP usage cost for your reference.

[![IP allocation configuration](/img/migrated/image-1694073898945-f373c2b3.png)](/img/migrated/image-1694073898945-f373c2b3.png)

4. Click **Allocate floating IP**. If the information is valid, the system allocates the IP to your account.

[![IP allocation result](/img/migrated/image-1694073904922-0dd35f26.png)](/img/migrated/image-1694073904922-0dd35f26.png)

[![Cost deducted after allocation](/img/migrated/image-1694073916507-22fc26cb.png)](/img/migrated/image-1694073916507-22fc26cb.png)
