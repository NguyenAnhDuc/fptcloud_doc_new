# Initial setup

If this is your first time using the service, check and complete the following steps.

**Create an FPT Cloud account and log in to the FPT Portal**

To start using FPT Cloud services, you need to log in to the FPT Portal. Visit [https://console.fptcloud.com](https://console.fptcloud.com/) and sign in with your FPT Cloud account credentials. Select the correct Tenant, Region, and VPC for the service you want to use.

If you do not have an FPT Cloud account, visit the homepage at [https://fptcloud.com](https://fptcloud.com/). Select the **Sign Up** function and enter the required information as instructed. The support team will contact you shortly to confirm your details and create your account.

If you encounter any errors during the process, contact our Support team immediately for assistance.
