# Enable a policy

To use the Enable feature, you must log in to the system with Tenant owner permissions.

**Step 1:** In the menu, select **Policy Management**. Select a policy with **Inactive** status, then select **Actions** > **Enable policy**.

[![file](/img/migrated/image-1719821838733-6dc53530.png)](/img/migrated/image-1719821838733-6dc53530.png)

**Step 2:** A dialog box appears. Click **Enable policy** to confirm the action. The system will enable the policy and update its status on the **Policy Management** screen.

[![file](/img/migrated/image-1719821851124-6c464cf2.png)](/img/migrated/image-1719821851124-6c464cf2.png)

**Step 3:** The system enables the policy and updates its status from **Inactive** to **Active** on the **Policy Management** screen.

[![file](/img/migrated/image-1719821863943-2b379a0c.png)](/img/migrated/image-1719821863943-2b379a0c.png)
