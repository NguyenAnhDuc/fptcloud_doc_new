# Lock and unlock VM deletion

The deletion lock feature protects important virtual machines from accidental deletion — reducing the risk of data loss caused by operator error.

## Lock VM deletion (Lock Deletion)

1. In **Instance Management**, select the virtual machine, then click **Actions** → **Lock Deletion**.

   ![Actions menu with Lock Deletion option](/img/migrated/image-1712738228214-76b8b81a.png)

2. A confirmation dialog displays the virtual machine name. Click **Lock Instance Deletion** to confirm.

   ![Lock Instance Deletion confirmation dialog](/img/migrated/image-1712738219490-01d185fc.png)

The virtual machine cannot be deleted until it is unlocked.

## Unlock VM deletion (Unlock Deletion)

1. In **Instance Management**, select the virtual machine, then click **Actions** → **Unlock Deletion**.

   ![Actions menu with Unlock Deletion option](/img/migrated/image-1712738202636-17abdc91.png)

2. A confirmation dialog displays the virtual machine name. Click **Unlock Instance Deletion** to confirm.

   ![Unlock Instance Deletion confirmation dialog](/img/migrated/image-1712738209302-86a7aea2.png)

The virtual machine is now unlocked and can be deleted normally.
