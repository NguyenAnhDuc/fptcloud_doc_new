# Initial setup

If this is your first time using FPT Cloud, complete the following steps before getting started.

## Create an account and sign in to FPT Cloud

To use FPT Cloud services, sign in to the FPT Portal:

1. Go to [https://console.fptcloud.com](https://console.fptcloud.com/) and sign in with your FPT Cloud credentials.
2. Select the correct **Tenant**, **Region**, and **VPC** for your needs.

If you do not have an account yet, click **Sign Up** and follow the on-screen instructions. The support team will contact you to confirm your details and create your account.

:::note
If you encounter any errors during the process, contact our support team immediately for assistance.
:::
