# Initial Setup

If this is your first time using the service, complete the following steps.

## Prerequisites

- An FPT Cloud account.
- Logged in to FPT Cloud Portal.

## Log in to FPT Cloud Portal

1. Go to [https://console.fptcloud.com](https://console.fptcloud.com/).
2. Log in with your FPT Cloud credentials.
3. Select the correct **Tenant**, **Region**, and **VPC** for your service.

:::tip
If you do not have an FPT Cloud account, visit [https://fptcloud.com](https://fptcloud.com/) and select **Sign Up** to register. The support team will contact you to confirm your information.
:::
