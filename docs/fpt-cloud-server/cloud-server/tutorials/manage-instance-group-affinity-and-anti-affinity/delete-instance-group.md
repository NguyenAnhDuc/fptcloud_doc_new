# Delete an Instance Group

:::danger
This action cannot be undone. Deleting an Instance Group will also detach it from all assigned Instances.
:::

1. On the **Instance Group Management** table, click **Delete** in the **Action** column.

   ![Delete button on Instance Group Management table](/img/migrated/image-1705025148984-2af9c584.png)

2. A confirmation dialog appears. Type `delete` and click **Delete Instance Group**.

   ![Instance Group deletion confirmation dialog](/img/migrated/image-1705025187846-59abb769.png)

The Instance Group is permanently deleted. All previously assigned Instances are released.
