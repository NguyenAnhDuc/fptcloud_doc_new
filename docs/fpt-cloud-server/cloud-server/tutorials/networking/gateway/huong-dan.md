# Accessing Gateway

1. Log in to [console.fptcloud.com](https://console.fptcloud.com).
2. After logging in successfully, select the VPC whose Gateway data you want to view.
3. In the sidebar, select and expand the **Network** menu, then choose **Gateway**.

:::note
This feature is only available to Tenants with a special configuration. Please contact the support team to have it enabled.
:::
