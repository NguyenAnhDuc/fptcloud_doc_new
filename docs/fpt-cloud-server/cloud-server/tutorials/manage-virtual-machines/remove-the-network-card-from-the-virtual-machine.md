# Remove a network card (NIC) from the virtual machine

When you no longer need a connection to a Subnet, you can remove the network card from the virtual machine to simplify your network configuration.

## Remove a network card

1. Open the **Network Interface** tab of the virtual machine.

   [![Network Interface tab showing the list of network cards](/img/migrated/image-1744793241168-180450d8.png)](/img/migrated/image-1744793241168-180450d8.png)

2. In the **Network** table, select **Actions** → **Delete** for the network card you want to remove.

   [![Actions menu with Delete option for the network card](/img/migrated/image-1744793619214-47123e3d.png)](/img/migrated/image-1744793619214-47123e3d.png)

3. A confirmation dialog appears. Click **Delete** to confirm removal of the network card.

   [![Confirm network card removal dialog](/img/migrated/image-1712723201176-b2e4e0e8.png)](/img/migrated/image-1712723201176-b2e4e0e8.png)
