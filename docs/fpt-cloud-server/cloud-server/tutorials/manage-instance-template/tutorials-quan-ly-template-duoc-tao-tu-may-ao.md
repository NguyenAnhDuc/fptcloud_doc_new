# Manage templates on the Custom Images page

The **Custom Images** page displays the list of templates created from virtual machines under the **General Image** tab.

## Template information

- **Name**: The template name.
- **Description**: The template description.
- **Status**: Current state — Pending / Success / Failed.

[![Template list on Custom Images](/img/migrated/image-1715855590500-9c6d17bc.png)](/img/migrated/image-1715855590500-9c6d17bc.png)

- **Progress**: A progress bar shows the initialization progress.

[![Template initialization progress](/img/migrated/image-1715855691493-7d373533.png)](/img/migrated/image-1715855691493-7d373533.png)
