# Create an Instance from a template

Creating a virtual machine from a template lets you deploy quickly with a pre-configured setup.

## Create an Instance

1. On the **Custom Images** page, select **Actions** → **Create Instance** on the desired template.

   [![Actions menu with Create Instance option](/img/migrated/image-1715856847338-4cbbe896.png)](/img/migrated/image-1715856847338-4cbbe896.png)

:::note
Only templates with a **Success** status can be used to create an Instance.
:::

2. Fill in the Instance details.

   [![Create Instance dialog from template](/img/migrated/image-1744797552995-eb6de7b6.png)](/img/migrated/image-1744797552995-eb6de7b6.png)

:::warning
The Instance storage capacity must be greater than the image size.
:::

3. Click **Create Instance** and monitor the progress on the **Instances** page.
