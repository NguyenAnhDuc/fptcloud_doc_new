# Select a Project When Creating a VPC

When creating a new VPC, you can assign it to an existing Project. This ensures all resources within the VPC are associated with the correct project for cost tracking and access control.

:::tip
Assigning a VPC to a Project at creation time is the recommended approach — it avoids having to reassign resources later.
:::
