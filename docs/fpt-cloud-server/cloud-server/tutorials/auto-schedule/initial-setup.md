# Initial setup

If this is your first time using FPT Cloud, complete the following steps before getting started.

**Create an FPT Cloud account and sign in to FPT Portal**

1. Go to [https://console.fptcloud.com](https://console.fptcloud.com/) and sign in with your FPT Cloud credentials.
2. Select the correct Tenant, Region, and VPC for the service you want to use.
3. If you do not have an account yet, click **Sign Up** and fill in the required information. The support team will contact you to verify the details and create your account.

:::note
If you encounter any error during the process, contact our Support team immediately for assistance.
:::
