# Schedule Shutdown

Schedule Shutdown lets you set a future automatic shutdown time for your machine — useful when you are busy or do not have time to monitor the machine after your work is done.

## Set a Shutdown Schedule

1. Select the image → **Schedule**.
2. In the dialog, choose your desired time and click **Save settings**.

   [![Schedule shutdown settings dialog](/img/migrated/schedule-2-c46f16f3.jpg)](/img/migrated/schedule-2-c46f16f3.jpg)

After configuring, you can view the remaining time before the automatic shutdown on the **Hosts** list interface.
