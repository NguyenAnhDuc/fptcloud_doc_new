# Set Level 2 Password

The Level 2 password is a password you set after logging in to HPC Portal — used to log in to the machine when connecting via Remote Desktop or SSH.

## Set Up Level 2 Password

1. Click the user icon in the header, then click your username.
2. On the settings page, select the **Level 2 password** tab.
3. Enter the new password, then click **Submit**.
