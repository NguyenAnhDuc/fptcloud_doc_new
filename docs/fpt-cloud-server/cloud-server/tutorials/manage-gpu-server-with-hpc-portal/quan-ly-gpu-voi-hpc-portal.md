# Managing GPU Server with HPC Portal

[HPC Portal](https://hpc.fptcloud.com/) is the management interface for physical GPU Servers with RTX 3090 and RTX 4090 lines on FPT Cloud — providing a specialized management interface for high-performance GPU workloads.

:::note
To log in and create machines on HPC Portal, contact your administrator to grant permissions and provision resources for your account.
:::
