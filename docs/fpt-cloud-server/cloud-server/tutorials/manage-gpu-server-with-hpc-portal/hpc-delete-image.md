# Delete Image

When you no longer need an image, you can delete it from the machine list on HPC Portal.

:::danger
Deleting an image is irreversible. Make sure you have backed up all important data before proceeding.
:::

## Delete an Image

1. Select the image you want to delete, then click **Action** → **Remove**.
2. In the confirmation dialog, click **Remove** to complete the deletion.
