# Clone Image

Clone Image lets you duplicate existing images along with all storage devices — without needing to set up the environment from scratch.

:::warning
Make sure you have completed all environment setup and shut down the machine before cloning.
:::

## Clone an Image

1. Select the image and click **Clone**.
2. Enter the number of machines to create, then click **Clone**.
