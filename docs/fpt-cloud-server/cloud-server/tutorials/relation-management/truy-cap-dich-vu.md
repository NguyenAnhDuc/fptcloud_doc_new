# Accessing the Service

The steps are as follows:

**Step 1**: Log in to **FPT Cloud Portal** at **<https://console.fptcloud.com>**

**Step 2**: In the **FPT Cloud Console Portal** menu, click **Monitoring**. The list of workspaces is displayed as shown below.

[![Alt text](/img/migrated/Picture1-931a5e8c.png)](/img/migrated/Picture1-931a5e8c.png)

**Step 3**: Click the workspace name to view its details. The workspace detail screen will be displayed.

**Step 4**: Click the "**Configuration**" tab.

[![Alt text](/img/migrated/Picture2-d44be0fa.png)](/img/migrated/Picture2-d44be0fa.png)

Here you can see the **Dashboard URL**. Click that link to go to the dashboard page.

The dashboard page is displayed as follows:

[![Alt text](/img/migrated/Picture5-d0c28feb.png)](/img/migrated/Picture5-d0c28feb.png)

**Step 5**: In the menu, click **App > Fmon - Relation**. You can now use our service.
