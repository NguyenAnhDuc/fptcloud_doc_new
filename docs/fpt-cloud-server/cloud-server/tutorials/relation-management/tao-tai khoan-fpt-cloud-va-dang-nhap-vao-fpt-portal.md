# Create an FPT Cloud Account and Log in to FPT Portal

To start using the **Relation management** service, you need to log in to **FPT Portal**.

Visit **<https://console.fptcloud.com>** and log in with the FPT Cloud account credentials provided to you. Select the correct **Tenant, Region, and VPC** for the service you want to use. If you do not yet have an FPT Cloud account, visit the homepage at **[https://fptcloud.com](https://fptcloud.com/)**.

Then select the **Sign Up** function and enter the required information as guided by the system. Our support team will contact you shortly to confirm your information and create your account.

If you encounter any errors during the process, please contact our Support team immediately for assistance.
