# Disable a NIC card

1. Click **Disable Card** on the NIC you want to disable.

[![Disable Card button](/img/migrated/image-1691381843537-90d61d72.png)](/img/migrated/image-1691381843537-90d61d72.png)

2. Confirm the disable action.

[![Confirm Disable Card](/img/migrated/image-1691381854527-265d8dd4.png)](/img/migrated/image-1691381854527-265d8dd4.png)

3. Wait for the system to process the request.

[![System processing](/img/migrated/image-1691381866783-e7158e52.png)](/img/migrated/image-1691381866783-e7158e52.png)

4. The card moves to **Inactive** status. The Instance can no longer connect to the network through this card.

[![NIC card in Inactive status](/img/migrated/image-1691381878032-5f233510.png)](/img/migrated/image-1691381878032-5f233510.png)
