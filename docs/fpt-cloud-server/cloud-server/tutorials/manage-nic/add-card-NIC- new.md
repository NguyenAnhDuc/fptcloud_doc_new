# Add a new NIC card

FPT Cloud supports adding new NIC cards easily and flexibly to meet your needs.

1. Click **Add NIC**.

[![Add NIC button](/img/migrated/image-1691382699972-7f2687d8.png)](/img/migrated/image-1691382699972-7f2687d8.png)

2. Select the **Subnet** you need.

[![Select Subnet](/img/migrated/image-1691382717830-0934edbe.png)](/img/migrated/image-1691382717830-0934edbe.png)

[![Subnet confirmation](/img/migrated/image-1691382726787-0afe397a.png)](/img/migrated/image-1691382726787-0afe397a.png)

[![Subnet selected](/img/migrated/image-1691382743112-1538d5f6.png)](/img/migrated/image-1691382743112-1538d5f6.png)

3. Wait for the system to process. The system will randomly assign an IP address from the selected Subnet.

[![System processing](/img/migrated/image-1691382798352-22ba262c.png)](/img/migrated/image-1691382798352-22ba262c.png)

4. The card moves to **Active** status. The Instance can connect to the network through this card.

[![NIC card in Active status](/img/migrated/image-1691382818519-8982a88f.png)](/img/migrated/image-1691382818519-8982a88f.png)
