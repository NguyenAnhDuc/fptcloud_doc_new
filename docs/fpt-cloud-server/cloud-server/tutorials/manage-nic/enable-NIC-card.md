# Enable a NIC card

For a NIC card in **Inactive** status, you can enable it to restore network connectivity.

1. Click **Enable Card** on the NIC you want to enable.

[![Enable Card button](/img/migrated/image-1691382154697-155c022c.png)](/img/migrated/image-1691382154697-155c022c.png)

2. Confirm the enable action.

[![Confirm Enable Card](/img/migrated/image-1691382171803-02c83019.png)](/img/migrated/image-1691382171803-02c83019.png)

3. Wait for the system to process the request.

[![System processing](/img/migrated/image-1691382523333-cd5f8412.png)](/img/migrated/image-1691382523333-cd5f8412.png)

4. The card moves to **Active** status. The Instance can connect to the network through this card.

[![NIC card in Active status](/img/migrated/image-1691382532014-ecb97957.png)](/img/migrated/image-1691382532014-ecb97957.png)
