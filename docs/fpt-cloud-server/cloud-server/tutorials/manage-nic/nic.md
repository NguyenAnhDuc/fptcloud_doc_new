# Network Interface Card (NIC)

## Overview

A Network Interface Card (NIC) on FPT Cloud is a key component for connecting Instances to network resources. NICs provide high-performance, reliable connectivity and enable flexible network environments for cloud-based applications and services.

## Before configuring a NIC

- This is an advanced feature intended for users with knowledge of network infrastructure management.
- Determine the network connection for the NIC to ensure the Instance can communicate with and access network resources effectively.
- Configure the IP address, subnet mask, gateway, and DNS for the NIC.
- After configuration, run tests to confirm the NIC works correctly and does not cause issues in the Instance or network.
