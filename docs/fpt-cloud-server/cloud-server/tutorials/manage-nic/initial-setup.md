# Initial setup

Complete the following steps before using the NIC (Network Interface Card) feature on FPT Cloud.

## Create an FPT Cloud account and log in

1. Log in at [https://console.fptcloud.com](https://console.fptcloud.com) with your FPT Cloud credentials.
2. Select the correct **Tenant**, **Region**, and **VPC** for your service.

If you do not have an account, visit [https://fptcloud.com/](https://fptcloud.com/), click **Sign Up**, and follow the on-screen instructions. The support team will contact you to confirm your account.

:::tip
If you encounter any issues, contact the support team for assistance.
:::
