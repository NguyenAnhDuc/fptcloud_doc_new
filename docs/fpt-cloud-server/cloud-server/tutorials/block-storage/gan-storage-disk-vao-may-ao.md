# Attach a Storage Disk to an Instance

After creating a Storage Disk, you can attach it to an Instance to start using it — similar to connecting an external hard drive to a physical machine.

## Attach a Storage Disk

1. Select the Storage Disk you want to attach, then click **Actions** → **Attach**.

   [![Actions menu with Attach option](/img/migrated/image-1712215517268-10c0d6df.png)](/img/migrated/image-1712215517268-10c0d6df.png)

2. Select the target Instance in the popup, then click **Attach**.

   [![Instance selection dialog for Attach](/img/migrated/image-1712215524338-f47bb248.png)](/img/migrated/image-1712215524338-f47bb248.png)

:::note
- Each Storage Disk can only be attached to one Instance at a time.
- For Windows Instances, additional disk configuration steps are required after attaching.
:::
