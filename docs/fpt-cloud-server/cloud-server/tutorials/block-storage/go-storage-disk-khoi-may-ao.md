# Detach a Storage Disk from an Instance

When a Storage Disk is no longer needed on an Instance, you can detach it. Your data remains intact and the disk can be attached to a different Instance later.

## Detach a Storage Disk

1. In the Storage Disk list, click **Actions** → **Detach**.

   [![Actions menu with Detach option](/img/migrated/image-1712215548327-dc507d37.png)](/img/migrated/image-1712215548327-dc507d37.png)

2. Review the details in the popup, then click **Detach**.

   [![Detach confirmation dialog](/img/migrated/image-1712215553461-87cc9cfe.png)](/img/migrated/image-1712215553461-87cc9cfe.png)
