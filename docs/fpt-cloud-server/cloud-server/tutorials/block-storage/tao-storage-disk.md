# Create a Storage Disk

Creating a new Storage Disk lets you expand the storage capacity for an Instance — useful when you need more space for data or applications.

## Create a Storage Disk

1. Select **Compute Engine** → **Storage Disk**, then click **Create Storage**.

   [![Storage Disk screen with Create Storage button](/img/migrated/image-1712215492491-8d7d7c0c.png)](/img/migrated/image-1712215492491-8d7d7c0c.png)

2. Enter the details:
   - **Name**: Name of the Storage Disk.
   - **Storage Policy**: Type of storage.
   - **Size**: Storage size.
   - **Applied instance** *(optional)*: The Instance to attach to (you can attach it later).

   [![Create Storage Disk dialog](/img/migrated/image-1712215502018-5f358995.png)](/img/migrated/image-1712215502018-5f358995.png)

3. Click **Create storage disk**.

The new Storage Disk appears on the **Storage Disk Management** page.

[![Storage Disk after creation](/img/migrated/image-1712215510866-1cd01f25.png)](/img/migrated/image-1712215510866-1cd01f25.png)
