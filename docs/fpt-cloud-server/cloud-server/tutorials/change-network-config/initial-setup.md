# Initial setup

If this is your first time, check and complete the following steps before proceeding.

## Create an FPT Cloud account and sign in to FPT Portal

1. Go to [https://console.fptcloud.com](https://console.fptcloud.com/) and sign in with your FPT Cloud credentials.

2. Select the correct **Tenant**, **Region**, and **VPC** for the service you want to use.

If you do not have an FPT Cloud account, visit [https://fptcloud.com/](https://fptcloud.com/), click **Sign Up**, and follow the on-screen instructions. The support team will contact you to confirm your details and create your account.

:::note
If you encounter any errors during the process, contact the Support team for assistance.
:::
