# FPT Cloud Monitoring

FPT Cloud Monitoring (FMON) is a comprehensive observability platform that lets you monitor metrics, logs, and traces for infrastructure, applications, and public endpoints on FPT Cloud. It is built on Grafana and provides dashboards, alerting, and data exploration in a single workspace.
