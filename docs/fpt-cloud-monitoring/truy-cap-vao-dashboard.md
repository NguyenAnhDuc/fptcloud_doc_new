# 7.1. Truy cập vào dashboard

**Step 1** : Đăng nhập vào FPT portal cloud **<https://console.fptcloud.com>**
**Step 2** : Trên menu của FPT Cloud Portal, click **Monitoring**. 
**Step 3** : Click chi tiết vào một Workspace, sau đó di chuyển tới **Configuration Data** tab.
**Step 4** : Click vào dashboard url, dashboard sẽ tự động login bằng Single Sign On (SSO)
