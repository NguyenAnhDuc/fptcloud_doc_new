# 7.1. Truy cập ando dashboard

**Step 1** : Log in ando FPT portal cloud **<https://console.fptcloud.com>**
**Step 2** : Trên menu of FPT Cloud Portal, click **Monitoring**. 
**Step 3** : Click chi tiết ando a Workspace, sau that di chuyển tới **Configuration Data** tab.
**Step 4** : Click ando dashboard url, dashboard will tự động login bằng Single Sign On (SSO)
