# 7.2.1. Cài đặt dashboard template

Giúp you triển khai nhanh chóng thông qua the mẫu dashboard mà FPT Cloud Monitoring cung cấp, sẵn sàng ngay to sử dụng.
**7.2.1. Cài đặt dashboard template**
**Step 1** : Log in ando dashboard (xem mục 7.1. Truy cập ando dashboard)
**Step 2** : Trên menu trái, click **Apps** > **FMON Template**
[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)
**Step 3** : Click **Dashboard Template**
**Step 4** : Chọn a dashboard, click **Integrate**
[![Alt text](/img/migrated/Picture14-2-d9a00088.png)](/img/migrated/Picture14-2-d9a00088.png)
**7.2.2. Remove bỏ Dashboard Template**
**Step 1** : Log in ando dashboard (xem mục 7.1. Truy cập ando dashboard)
**Step 2** : Trên menu trái, click **Apps** > **FMON Template**
[![Alt text](/img/migrated/Picture13-2-4dc26f2f.png)](/img/migrated/Picture13-2-4dc26f2f.png)
**Step 3** : Click **Dashboard Template**
**Step 4** : Chọn a dashboard has been cài đặt, click **Remove**
[![Alt text](/img/migrated/Picture15-2-7940295e.png)](/img/migrated/Picture15-2-7940295e.png)
