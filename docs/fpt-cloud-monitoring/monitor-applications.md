# Monitor Applications

**Step 1** : Log in ando FPT portal cloud **<https://console.fptcloud.com>**
**Step 2** : Trên menu of FPT Cloud Portal, click **Monitoring**. 
**Step 3** : Click details ando a Workspace, sau that di chuyển tới **Add Data** tab.
**Step 4** : Trong **Applications** , displayed list the ngôn ngữ lập trình is sẵn sàng hỗ trợ, click ando a ngôn ngữ lập trình and làm theo guide sau that.
