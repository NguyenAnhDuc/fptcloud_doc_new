# Initial setup

## Log in to FPT Portal

To start using **FPT Cloud Monitoring**, log in to FPT Portal.

1. Go to **[https://console.fptcloud.com](https://console.fptcloud.com)** and log in with your FPT Cloud username and password.
2. Select the correct Tenant, Region, and VPC for the service.

## Create an FPT Cloud account

If you do not have an FPT Cloud account yet:

1. Visit **[https://fptcloud.com](https://fptcloud.com/)**.
2. Select **Sign Up** and enter the required information following the on-screen instructions.
3. The support team will contact you to confirm your account details.

:::tip
If you encounter any errors during the process, contact our Support team immediately for assistance.
:::
