# Create Vault Instance

**Step 1:** In the FPT Portal menu, select **Key Vault** > Click **Create**
[![](/img/migrated/k1-ff2abcff.png)](/img/migrated/k1-ff2abcff.png)
**Step 2** : In the **Create Key Vault** popup, enter the necessary information:
[![](/img/migrated/k2-1e9263de.png)](/img/migrated/k2-1e9263de.png)
**Step 3** : Click **Create** to proceed. The system will check the resource requirement, send a confirmation message and proceed with the process of creating a new **Vault Instance** based on the information entered.
List of Key Vault after success creation:
[![](/img/migrated/k3-612b2fb5.png)](/img/migrated/k3-612b2fb5.png)
