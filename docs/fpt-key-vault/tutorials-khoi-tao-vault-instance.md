# Create a Vault Instance

A Vault Instance is the basic unit of FPT Key Vault — it stores all your Secret Engines, Secrets, and security configurations. Create a Vault Instance before performing any Secret management operations.

1. In the menu, select **Security** > **Key Vault** > **Create**.

   [![Key Vault list screen with Create button](/img/migrated/Screenshot-2024-08-20-103813-1-bebc2790.png)](/img/migrated/Screenshot-2024-08-20-103813-1-bebc2790.png)

2. On the **Create Key Vault** screen, enter the required information.

   [![Form for entering Key Vault initialization details](/img/migrated/Userguide-FPT-KeyVault-2022-2-125824c1.png)](/img/migrated/Userguide-FPT-KeyVault-2022-2-125824c1.png)

3. Click **Create** to finish. The system checks resources, sends a confirmation message, and creates a new **Vault Instance** based on the information entered.

   [![Key Vault list after successful creation](/img/migrated/Userguide-FPT-KeyVault-2022-3-1024x290-02d8d8d5.png)](/img/migrated/Userguide-FPT-KeyVault-2022-3-1024x290-02d8d8d5.png)
