# View And Update Secret Information

**Step 1:** Select the Secret Engine to view **Secrets** on the Secret Engine Management screen:
[![](/img/migrated/v1-88a45c58.png)](/img/migrated/v1-88a45c58.png)
**Step 2** : Select the **Secret** that you want to view or update information
[![](/img/migrated/v2-90d1e587.png)](/img/migrated/v2-90d1e587.png)
**Step 3** : In the **Edit Screen** popup, user can view information or update those information.
Click **Edit** to save changes
[![](/img/migrated/v4-534b24ad.png)](/img/migrated/v4-534b24ad.png)
