# Thay đổi dung lượng disk rời

* B1. Vào trang Compute Engine - Storage Disks
  * B2. Trên mỗi ổ đĩa, bấm vào action và foṛn Edit [![file](/img/migrated/image-1694073979781-7ec29458.png)](/img/migrated/image-1694073979781-7ec29458.png)
  * B3. Chọn policy và dung lượng cần thay đổi 
    * Save ý: Ổ đĩa không gắn vào máy ảo không thể edit policy
    * Hệ thống estimate số tiền sử dụng storage in 1 giờ for customer. [![file](/img/migrated/image-1694073989051-4fa42062.png)](/img/migrated/image-1694073989051-4fa42062.png)
  * B4. Bấm Edit storage disk 
    * Nếu the information hợp lệ thì system thay đổi storage and đi đến trang Storage disk
    * Nếu user has been sử dụng hết giới hạn tài nguyên, system will báo lỗi and không for phép thay đổi. [![file](/img/migrated/image-1694073998971-887af5ca.png)](/img/migrated/image-1694073998971-887af5ca.png)

Save ý: 
  * Khách hàng cần có ít nhất 50.000đ + tiền sử dụng storage in 1h để thay đổi 
  * Hệ thống sẽ trừ tiền chi phí sử dụng storage in 1h ngay when thay đổi.
