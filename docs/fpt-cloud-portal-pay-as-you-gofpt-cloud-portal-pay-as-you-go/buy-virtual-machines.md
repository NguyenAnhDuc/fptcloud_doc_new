# Mua virtual machine

Trên trang Instance, người dùng có thể tạo máy ảo và trả tiền theo PAYG
Người dùng phải có ít nhất̉ 50000 VND + số tiền của một giờ sử dụng dịch vụ to khởi tạo service.
  * B1. Vào trang Instance Management

[![file](/img/migrated/image-1694073823080-04d5dd8a.png)](/img/migrated/image-1694073823080-04d5dd8a.png)
  * B2. Chọn New Instance
  * B3. Chọn cấu hình máy, in that: 
    * Chọn gói cấu hình, hệ thống displayed thông tin cấu hình and giá.
    * System estimate số tiền sử dụng in 1 tháng for khách hàng.

[![file](/img/migrated/image-1694073838317-c01bfab4.png)](/img/migrated/image-1694073838317-c01bfab4.png)
  * B4. Bấm create instance 
    * Nếu the thông tin hợp lệ thì hệ thống tạo máy and đi to trang Instance management.
    * Nếu user has been sử dụng hết giới hạn tài nguyên, hệ thống will báo lỗi and không for phép mua service. [![file](/img/migrated/image-1694073846242-2bb9076d.png)](/img/migrated/image-1694073846242-2bb9076d.png)

Save ý: Hệ thống sẽ trừ tiền chi phí sử dụng máy ảo in 1h ngay when khởi tạo máy. [![file](/img/migrated/image-1694073853907-420d469b.png)](/img/migrated/image-1694073853907-420d469b.png)
