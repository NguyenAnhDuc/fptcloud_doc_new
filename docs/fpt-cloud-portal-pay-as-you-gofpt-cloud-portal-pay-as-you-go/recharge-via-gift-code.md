# Nạp tiền thông qua gift code.

Khách hàng can nạp tiền thông qua gift code nhận is from FPT Smart Cloud. 
  * B1. Vào trang Billing, chọn menu Topup. [![file](/img/migrated/image-1694073773226-c126e916.png)](/img/migrated/image-1694073773226-c126e916.png)
  * B2. Enter mã gift code. [![file](/img/migrated/image-1694073782136-bded0c89.png)](/img/migrated/image-1694073782136-bded0c89.png)
  * B3. Bấm Redeem gift code 
    * Nếu gift code đúng and is in thời gian có hiệu lực, system will cộng tiền ando account.
