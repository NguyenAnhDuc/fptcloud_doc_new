# Top Up via Gift Code

You can top up your account balance using a gift code received from FPT Smart Cloud.

1. Go to the **Billing** page and select the **Topup** menu.

[![Billing – Topup page](/img/migrated/image-1694073773226-c126e916.png)](/img/migrated/image-1694073773226-c126e916.png)

2. Enter the gift code in the input field.

[![Enter gift code](/img/migrated/image-1694073782136-bded0c89.png)](/img/migrated/image-1694073782136-bded0c89.png)

3. Click **Redeem gift code**.

:::note
If the gift code is valid and within its validity period, the system will automatically credit the amount to your account.
:::
