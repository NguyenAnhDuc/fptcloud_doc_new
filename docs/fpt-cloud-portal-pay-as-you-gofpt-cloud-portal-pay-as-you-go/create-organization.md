# Create an Organization

After verifying your phone number via OTP, the system redirects you to the Organization creation page.

1. Fill in your organization or individual details:
   - **Organization name**: the name of your organization or customer name.
   - **Organization code**: the display name of your organization in the system.
   - **Organization address**: the address of your organization (optional).

[![Fill in Organization details](/img/migrated/image-1694073747297-30c6d924.png)](/img/migrated/image-1694073747297-30c6d924.png)

2. Click the create Organization button. The system will provision default resources for you to start using the service. Resource provisioning takes approximately 1–3 minutes.

[![System initializing resources](/img/migrated/image-1694073758532-40c000e0.png)](/img/migrated/image-1694073758532-40c000e0.png)
