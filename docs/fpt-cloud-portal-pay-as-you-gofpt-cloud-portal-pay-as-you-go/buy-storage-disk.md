# Mua storage disk (disk rời)

Khách hàng has been nạp tiền trên hệ thống can mua disk rời.
  * B1. Vào trang Compute Engine - Storage Disks [![file](/img/migrated/image-1694073928674-97d41009.png)](/img/migrated/image-1694073928674-97d41009.png)
  * B2. Chọn New Storage [![file](/img/migrated/image-1694073937844-c9e0ea1f.png)](/img/migrated/image-1694073937844-c9e0ea1f.png)
  * B3. Chọn Storage policy and size 
    * System estimate số tiền sử dụng storage in 1 giờ for khách hàng.
  * B4. Bấm Create storage disk
    * Nếu the thông tin hợp lệ thì hệ thống tạo storage and đi to trang Storage disk
    * Nếu user has been sử dụng hết giới hạn tài nguyên, hệ thống will báo lỗi [![file](/img/migrated/image-1694073957418-7a7b3dac.png)](/img/migrated/image-1694073957418-7a7b3dac.png)
Save ý: 
  * Khách hàng cần có ít nhất 50.000đ + tiền sử dụng storage in 1h để khởi tạo
  * Hệ thống sẽ trừ tiền chi phí sử dụng storage in 1h ngay when khởi tạo. [![file](/img/migrated/image-1694073966294-6d3c6da1.png)](/img/migrated/image-1694073966294-6d3c6da1.png)
