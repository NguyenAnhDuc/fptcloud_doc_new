# Delete a Budget Alert

:::danger
Deleting a Budget Alert cannot be undone. After deletion, the system stops calculating costs and sending alerts for that Budget Alert.
:::

1. Select **Billing** → **Budget Alert**. The system displays the list of existing Budget Alerts. Select **Actions** on the Budget Alert you want to delete.

   ![Select Actions on Budget Alert](/img/migrated/image-1722238402560-f7ffe076.png)

2. Select **Delete**.

   ![Delete button](/img/migrated/image-1722238623649-f0020ae9.png)

3. A confirmation dialog appears. Click **Delete**.

   ![Budget Alert deletion confirmation dialog](/img/migrated/image-1722396453644-8e9f9bd6.png)

The Budget Alert is removed from the list.
