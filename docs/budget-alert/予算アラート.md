# 予算アラート

予算アラートはプロアクティブな予算アラートシステムの構築を支援します。リアルタイムでコストを監視し、設定したしきい値に予算が達した際に通知を受け取ることができます。日次、月次、四半期、年次でクラウドコストを管理する必要がある場合に特に有用です。

## 予算を作成し、アラートを設定する

- 日次／月次／四半期／年次の期間で、実績使用量または期末までの予測予算に基づいて予算を設定できます。
- アラート対象のリソースグループを選択・管理できます。
- 目的に応じてアラートしきい値と通知方法（システム通知またはメール）を設定できます。

## 予算履歴とアラートの確認

- 実績コスト、実績コストと予算の差異、期末までの予測コストを追跡できます。
- 設定したアラートしきい値に基づくアラート送信履歴を確認できます。
