# View trace data

**Step 1:** On the menu, select **Explore**.

[![](/img/migrated/Screenshot_22-f2b7b75d.png)](/img/migrated/Screenshot_22-f2b7b75d.png)

**Step 2:** Select the datasource: **tempo**.

The system opens the **TraceQL** tab where you can enter your query.

[![](/img/migrated/Screenshot_29-a8f6ab7d.png)](/img/migrated/Screenshot_29-a8f6ab7d.png)

**Step 3:** Click **Run query** to display results.

[![](/img/migrated/Screenshot_30-e54fb90f.png)](/img/migrated/Screenshot_30-e54fb90f.png)

For detailed guidance, refer to the official Grafana documentation [here](https://grafana.com/docs/tempo/latest).
