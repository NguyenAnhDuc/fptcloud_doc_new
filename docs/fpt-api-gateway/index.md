# Fpt Api Gateway

  2. FPT API Gateway

**API Gateway** là thành phần trung gian giúp quản lý, kiểm soát and định tuyến the yêu cầu (requests) giữa client and the service backend. Nó hoạt động như a cổng truy cập tập trung for toàn bộ API, đảm bảo việc giao tiếp an toàn, thống nhất and dễ dàng mở rộng. 
**API Gateway** hỗ trợ the feature như xác thực, phân permission, giới hạn tần suất (rate limiting), cân bằng tải, monitoring and quản lý vòng đời API. Với **APISIX Dashboard** , user can cấu hình trực quan the route, service, upstream, plugin and chứng chỉ TLS, giúp đơn giản hóa việc quản trị and vận hành. 
API Gateway phù hợp with the kiến trúc microservices and system phân tán, thường is sử dụng to tăng cường security, tính linh hoạt and khả năng quan sát in việc quản lý API.
