# Initial setup

To start using **FPT API Gateway**, you need to log in to the FPT Portal.

Go to [https://console.fptcloud.com](https://console.fptcloud.com) and sign in with your FPT Cloud account credentials. Select the correct Tenant, Region, and VPC for your service.

If you do not have an FPT Cloud account, visit [https://fptcloud.com](https://fptcloud.com), click **Sign Up**, and follow the on-screen instructions. The support team will contact you to verify your information and create your account.

:::note
If you encounter any errors during the process, contact our Support team immediately for assistance.
:::
