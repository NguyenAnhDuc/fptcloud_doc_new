# Xoa api Gateway

Để xóa API Gateway, user thực hiện the bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **API Gateway**
**Step 2:** Tại list **API Gateway** > chọn **API Gateway** need to xóa > nhấn **Action** > select **Delete**
[![Hỉnh ảnh](/img/migrated/z7045391397635_d1a00a9228efc6171ae123d65-6c413f1c.jpg)](/img/migrated/z7045391397635_d1a00a9228efc6171ae123d65-6c413f1c.jpg)
**Step 3** : Hiển thị hộp thoại **Delete Application** > nhập **delete** > nhấn **confirm** to xóa hoàn thành việc xóa **API Gateway**
