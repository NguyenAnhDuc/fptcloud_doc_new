# Xem Thong tin api Gateway

Để xem thông tin **API Gateway** , người dùng thực hiện các bước sau: 
**Step 1:** Tại thanh menu chọn **Integration** > chọn **API Gateway**
**Step 2:** Tại danh sách **API Gateway** > chọn **API Gateway** cần xem thông tin 
  * Tab **Overview**

Hiển thị thông tin cơ bản của API Gateway: **Basic information** , **Launch Configuration** , **Etcd** , **LB size** , **Endpoint**
[![Hỉnh ảnh](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)
  * Tab **Autoscaling**

Hiển thị thông tin Autoscaling của **API Gateway**
[![Hỉnh ảnh](/img/migrated/SCR-20250924-nuvs-077cada2.png)](/img/migrated/SCR-20250924-nuvs-077cada2.png)
  * Tab **Dashboard**

Hiển thị thông tin dashboard của API Gateway 
Nếu khi tạo API Gateway tùy chọn **Dashboard = Disable** , thì ở màn hình **Details** sẽ **không hiển thị tab Dashboard**
