# View API Gateway details

To view **API Gateway** details, follow these steps:

**Step 1:** In the menu bar, select **Integration** > **API Gateway**.

**Step 2:** In the **API Gateway** list, select the API Gateway you want to inspect.

- **Overview** tab: Displays basic information — Basic information, Launch Configuration, Etcd, LB size, and Endpoint.

[![Image](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)](/img/migrated/z7045370518330_c37c9eb76ab638d6806679fdd-1f519ff1.jpg)

- **Autoscaling** tab: Displays autoscaling configuration for the **API Gateway**.

[![Image](/img/migrated/SCR-20250924-nuvs-077cada2.png)](/img/migrated/SCR-20250924-nuvs-077cada2.png)

- **Dashboard** tab: Displays Dashboard information for the API Gateway. If you selected **Dashboard = Disable** during creation, this tab will not appear.

[![Image](/img/migrated/z7045379122842_6adcfbe63dafe93587e68b587-a1a084d9.jpg)](/img/migrated/z7045379122842_6adcfbe63dafe93587e68b587-a1a084d9.jpg)
