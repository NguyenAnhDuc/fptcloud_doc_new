# Manage issue status in the Issue detail screen

**Step 1:** In the Issue list for an asset, click **Details** on any issue to open the detail view.

[![anhfsec](/img/migrated/i6-e0741810.png)](/img/migrated/i6-e0741810.png)

**Step 2:** On the Issue detail screen, you can view the current status of the issue from the most recent scan. To change the status, click **Note status**.

[![anhfsec](/img/migrated/i7-c61cdbb2.png)](/img/migrated/i7-c61cdbb2.png)

**Step 3:** To track the issue's status change history, view the **Activity** section.

[![anhfsec](/img/migrated/i8-8cf185fa.png)](/img/migrated/i8-8cf185fa.png)
