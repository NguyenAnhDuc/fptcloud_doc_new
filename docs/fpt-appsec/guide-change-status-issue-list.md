# Change issue status in the Issue list

**Step 1.** In the asset list, select any asset and open the **Issue** tab. The issue status is displayed on each issue card.

[![anhfsec](/img/migrated/i1-c8b75478.png)](/img/migrated/i1-c8b75478.png)

You can filter issues by status: Open, Ignored, and the ignore reasons.

[![anhfsec](/img/migrated/i2-c41aa480.png)](/img/migrated/i2-c41aa480.png)

**Step 2.** To change an issue's status, click **Note status**.

[![anhfsec](/img/migrated/i3-21de070c.png)](/img/migrated/i3-21de070c.png)

**Step 3.** Select the target status (currently supports two statuses: Open and Ignore).

For the Ignored → Open flow, you may optionally add a comment, then click **Save**.

[![anhfsec](/img/migrated/i4-454465e4.png)](/img/migrated/i4-454465e4.png)

For the Open → Ignored flow, you must select an ignore reason and may optionally add a comment, then click **Save**.

[![anhfsec](/img/migrated/i5-1284ea67.png)](/img/migrated/i5-1284ea67.png)

> **Note:** The issue status change takes effect at the next scan.
