# Hướng dẫn xem danh sách & chi tiết schedule

**Step 1:** Tại thanh menu bên trái, chọn Security Platform → DevSec → Schedule. Màn hình **Schedule Management** displayed danh sách the Schedule is sắp xếp mặc định theo ngày tạo from mới nhất to cũ nhất: 
[![Alt text](/img/migrated/1-1-461969c3.png)](/img/migrated/1-1-461969c3.png)
**Step 2** : Quan sát the thông tin displayed:   
| Trường  | Mô tả  |  
| --- | --- |  
| Schedule ID  | Mã định danh tự tăng do hệ thống sinh.  |  
| Schedule Name  | Tên of Schedule. Click to xem chi tiết.  |  
| Frequency  | Tần suất chạy (On-demand / Weekly / Monthly).  |  
| Interval  | Thời điểm chạy job, định dạng “Ngày + Giờ”.  |  
| Last Launch  | Hiển thị thời gian and status chạy gần nhất (Running / Success Finished / Failed Error / None).  |  
| Next Run  | Thời gian chạy tiếp theo or to trống.  |  
| Created By  | Người tạo Schedule.  |  
| Action  | Edit – chỉnh sửa. Delete – xóa  |  
**Step 3:** View chi tiết schedule 
Nhấn ando tên schedule bất kỳ to chuyển to màn hình Schedule detail:
