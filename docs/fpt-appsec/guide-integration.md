# Integration management

Only **Org Admin** or **Team Manager** has permission to edit.

**Step 1:** Go to the **DevSec** tab in the **Security Platform** menu, then click **Integration** to view the integration list. Select your Org or Team:

[![anhfsec](/img/migrated/in1-1-bbf498e7.png)](/img/migrated/in1-1-bbf498e7.png)

> **Note:** Integration types are disabled and cannot be selected until you choose an org/team.

**Step 2:** Select the Source Control or Container Registry type to integrate:

[![anhfsec](/img/migrated/in2-6701890e.png)](/img/migrated/in2-6701890e.png)
