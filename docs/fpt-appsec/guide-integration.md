# Integration Management - Manage tích hợp

Chỉ **Org Admin** or **Team Manager** có permission chỉnh sửa
**Step 1** : Đến tab DevSec in menu Security Platform, chọn **Integration** to xem is danh sách **integration**. Tiến hành chọn Org or Team of you:
[![anhfsec](/img/migrated/in1-1-bbf498e7.png)](/img/migrated/in1-1-bbf498e7.png)
_Note: Các loại tích hợp will bị disabled and you cannot chọn tích hợp when chưa chọn org/team_
**Step 2** : Chọn loại Source Control or Container Registry cần tích hợp:
[![anhfsec](/img/migrated/in2-6701890e.png)](/img/migrated/in2-6701890e.png)
