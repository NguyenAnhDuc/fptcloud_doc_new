# Edit a schedule

**Option 1:** In the Schedule Management screen, click the **⋮** icon on a schedule, then select **Edit**.

[![Alt text](/img/migrated/Picture19-c62f7e3c.png)](/img/migrated/Picture19-c62f7e3c.png)

The system navigates to the edit screen where you can modify the schedule as needed.

[![Alt text](/img/migrated/Picture20-11454da7.png)](/img/migrated/Picture20-11454da7.png)

**Option 2:** In the **Schedule Detail** screen, click the **Edit** button in the top right.

[![Alt text](/img/migrated/Picture21-18abe5d7.png)](/img/migrated/Picture21-18abe5d7.png)
