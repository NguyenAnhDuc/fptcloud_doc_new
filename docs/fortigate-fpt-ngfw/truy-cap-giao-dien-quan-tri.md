# Access the admin interface

1. Log in to the Fortigate NGFW management interface using the configured IP address, username, and password.

   [![Fortigate login screen](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-1-1024-b1cda52c.png)](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-1-1024-b1cda52c.png)

2. After logging in successfully, the Dashboard displays system parameters including: **Serial Number**, **Licenses**, **CPU**, **Memory**, and more.

   [![Fortigate dashboard after login](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-2-1024-898484e4.png)](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-2-1024-898484e4.png)
