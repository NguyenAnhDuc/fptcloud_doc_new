# Configure Static Routes

**Step 1:** **Network** > **Static Routes**. Ở phần cấu hình ban đầu trên CLI has been tạo sẵn static route for WAN port có gateway 10.0.0.1.  
[![Userguide Fortigate FPT NGFW 2022 5](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-5-1024-f0a1691a.png)](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-5-1024-f0a1691a.png)  
**Step 2:** Người dùng can tạo static route trên giao diện web. **Create New** > Chọn Interface cần Route > chọn **Gateway** or **Destination** (Destination can to default if không có destination cụ thể).
