# Faq

## Fortigate là gì?
FortiGate hay Firewall Fortinet là the thiết bị security đa lớp thế hệ tiếp theo dưới dạng tường lửa of Fortinet. Fortigate is tích hợp the feature hàng đầu to cung cấp a giải pháp toàn diện in việc security thông tin cũng như là bảo vệ hệ thống network nội bộ tại the doanh nghiệp hay tổ chức with mọi quy mô.  
Firewall Fortinet mang to for user khả năng security with tốc độ cao, ngăn ngừa virus and chặn đứng the mối đe doạ to trung tâm dữ liệu and hệ thống network of you. Fortigate cũng is hỗ trợ feature năng lọc web, liên tục kiểm tra the gói tin is trao đổi giữa the thiết bị in hệ thống network, hỗ trợ khả năng displayed giúp người quản trị dễ dàng monitoring, phát hiện and giải quyết kịp thời những xâm nhập trái phép.
## Vì sao should sử dụng FortiGate?
Hãy cùng điểm qua những feature vượt trội of sản phẩm Fortigate so with the thiết bị tường lửa thông thường, to from that tìm is câu trả lời for câu hỏi “Vì sao should sử dụng Fortigate?”.
  * **Khả năng displayed and bảo vệ toàn diện:** FortiGate is tích hợp the feature như ngăn chặn Ransomware, Command & Control, công nghệ SSL (includes TLS1.3) for phép thiết lập mã hoá an toàn, cùng khả năng tự động bảo vệ trước the mối đe dọa
  * **Hỗ trợ FortiGuard:** Fortigate is hỗ trợ FortiGuard with khả năng hợp nhất and chạy đồng thời the service IPS, lọc web & video, security DNS to giảm cost and quản lý mọi rủi ro.
  * **Tích hợp Proxy:** FortiGate cung cấp trải nghiệm user liền mạch and security an toàn for thiết bị of nhân viên with feature Zero Trust Network Acess (ZTNA).
  * **Bảo mật cao cấp:** Các giải pháp security nâng cao giúp ngăn chặn truy cập trái phép, phân vùng truy cập and bộ lọc gói tin a theh rõ ràng.
  * **Manage network theo hướng tự động hoá:** Giữ permission kiểm soát with khả năng displayed qua lưu lượng truy cập network, kiểm soát chính sách a theh chi tiết and trực quan, giúp mở rộng khả năng security and quản lý network.

## Sao lưu and phục hồi cấu hình Firewall như thế nào?
**FortiGate** hỗ trợ việc backup and phục hồi cấu hình 1 theh đơn giản and thuận tiện.  
Login ando hệ thống, chọn **System configuration**.
[![Userguide Fortigate FPT NGFW 2022 37](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-37-34b5a5bf.png)](/img/migrated/Userguide-Fortigate-FPT-NGFW-2022-37-34b5a5bf.png)
Muốn lưu cấu hình hiện tại of hệ thống ta chọn **Backup**.
Muốn phục hồi 1 cấu hình has been lưu, ta chọn ando nút **Restore**. Chọn Browse to nơi lưu file cấu hình, rồi nhấn **OK**. Sau when phục hồi xong should khởi động lại hệ thống.
