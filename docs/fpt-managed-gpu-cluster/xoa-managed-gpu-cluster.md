# Delete a Managed GPU Cluster

For Managed GPU Clusters that are no longer needed, customers can delete them by following these steps:

**Step 1:** In the menu, select **AI Infrastructure** > **Managed GPU Cluster**. The system will display the **GPU Cluster Management** page. Select the cluster you want to delete.

[![](/img/migrated/14-b4dc395d.png)](/img/migrated/14-b4dc395d.png)

**Step 2:** Select **Action** at the end of the row for the cluster you want to delete. Select **Delete**.

[![](/img/migrated/15-05e09c5c.png)](/img/migrated/15-05e09c5c.png)

**Step 3:** Confirm the warning information in the popup and click **Delete**.

[![](/img/migrated/16-52efdb8d.png)](/img/migrated/16-52efdb8d.png)
