# Xoa Managed gpu Cluster

Với the cụm Managed GPU Cluster không còn nhu cầu sử dụng, customer can xóa theo guide sau:
**Step 1** : Ở menu chọn **AI Infrastructure** > **Managed GPU Cluster** , system will displayed trang **GPU Cluster Management**. Chọn Cluster muốn lấy information truy cập Cluster.
[![](/img/migrated/14-b4dc395d.png)](/img/migrated/14-b4dc395d.png)
**Step 2** : Select **Action** ở cuối Cluster need to xóa trên list. Chọn Delete.
[![](/img/migrated/15-05e09c5c.png)](/img/migrated/15-05e09c5c.png)
**Step 3** : Confirm information alert in popup and select **Delete**.
[![](/img/migrated/16-52efdb8d.png)](/img/migrated/16-52efdb8d.png)
