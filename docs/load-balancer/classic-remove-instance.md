# Remove a virtual machine from a Classic Load Balancer

**Step 1:** Go to the Load Balancer details page by clicking the Load Balancer name in the **Load Balancer Management** dashboard.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-4-1024x-511c9674.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-4-1024x-511c9674.png)

**Step 2:** On the **Instances** tab, the current list of attached virtual machines is displayed. Click **Select instances**.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-5-1024x-96497cb3.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-5-1024x-96497cb3.png)

**Step 3:** In the **Change instances** popup, click the [![](/img/migrated/Userguide-FPT-Load-Balancer-2022-8-da4ac7b6.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-8-da4ac7b6.png) icon at the end of the virtual machine name to remove it.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-9-d29dbacb.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-9-d29dbacb.png)

**Step 4:** Click **Save**.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-7-300x1-828020a4.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-7-300x1-828020a4.png)
