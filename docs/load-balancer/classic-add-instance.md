# Add a virtual machine to a Classic Load Balancer

To scale a Load Balancer, you can add more virtual machines to an existing Load Balancer. First create the new virtual machine and install the required services in Instance Management, then add it to the Load Balancer.

**Step 1:** Go to the Load Balancer details page by clicking the Load Balancer name in the **Load Balancer Management** dashboard.

[![](/img/migrated/image-1712807219267-78e51205.png)](/img/migrated/image-1712807219267-78e51205.png)

**Step 2:** On the **Instances** tab, the current list of attached virtual machines is displayed. Click **Select instances**.

[![](/img/migrated/image-1712807226342-138669ae.png)](/img/migrated/image-1712807226342-138669ae.png)

**Step 3:** In the **Change instances** popup, select the virtual machines to add in the **Instances** field.

[![](/img/migrated/image-1712807234311-3e65aba3.png)](/img/migrated/image-1712807234311-3e65aba3.png)

**Step 4:** Click **Save**.

[![](/img/migrated/image-1712807240125-47822fc9.png)](/img/migrated/image-1712807240125-47822fc9.png)
