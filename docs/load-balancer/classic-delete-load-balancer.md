# Delete a Classic Load Balancer

When a Load Balancer is no longer needed, delete it to free up resources and avoid unnecessary costs.

**Step 1:** In the **Load Balancer Management** dashboard, click **Delete** in the **Action** menu for the Load Balancer you want to delete.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-15-1024-225988fc.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-15-1024-225988fc.png)

**Step 2:** A confirmation popup appears. Click **Delete** to confirm.

[![](/img/migrated/Userguide-FPT-Load-Balancer-2022-16-6b6db13c.png)](/img/migrated/Userguide-FPT-Load-Balancer-2022-16-6b6db13c.png)
