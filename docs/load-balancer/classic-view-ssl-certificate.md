# View SSL certificate details (Classic)

You can view all SSL certificates created within the VPC in the **Manage SSL Certificate for Load balancer** section.

[![](/img/migrated/image-1712808075335-de5bd0cf.png)](/img/migrated/image-1712808075335-de5bd0cf.png)

When you select a specific SSL certificate, the system displays the certificate's **Name** and **Public Key**.

[![](/img/migrated/image-1712808088237-03369751.png)](/img/migrated/image-1712808088237-03369751.png)
