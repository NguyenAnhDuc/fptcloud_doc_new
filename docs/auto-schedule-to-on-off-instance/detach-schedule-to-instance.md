# Remove a or nhiều virtual machine ando instance schedule

Bạn can gỡ the virtual machine ra khỏi schedule when không còn nhu cầu sử dụng. Sau when gỡ, virtual machine will đứng yên ở status lúc gỡ to when nào you thao tác tay lại mới thay đổi. 
**Step 1** : Trên bảng quản trị **Instance Schedule Management,** in phần **Action** , chọn **Apply instances**
**Step 2** : Chọn dấu “x” bên cạnh tên the instances muốn gỡ to thực hiện thao tác gỡ virtual machine khỏi schedule (you cũng can click ando dấu “x” ở cuối listbox to gỡ cùng lúc toàn bộ virtual machine khỏi schedule), sau that chọn **Apply instances**.
