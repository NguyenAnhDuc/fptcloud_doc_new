# Gắn a or nhiều virtual machine ando instance schedule

Sau when tạo ra instance schedule, you need to gắn the virtual machine ando in schedule that to sử dụng feature tự động tắt/bật. 
**Step 1:** Trên bảng quản trị **Instance Schedule Management,** in phần **Action** , chọn **Apply instances**
[![file](/img/migrated/image-1695609073911-03ac0ac2.png)](/img/migrated/image-1695609073911-03ac0ac2.png)
**Step 2:** Chọn tên the instances muốn gắn ando schedule & chọn **Apply instances**
[![file](/img/migrated/image-1695609084366-c567a73d.png)](/img/migrated/image-1695609084366-c567a73d.png)
