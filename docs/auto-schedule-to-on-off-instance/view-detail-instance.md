# View instance schedule details

**Step 1:** On the **Instance Schedule Management** page, click the name of the instance schedule you want to view.

[![Instance schedule list](/img/migrated/image-1695609011987-33ef2b1d.png)](/img/migrated/image-1695609011987-33ef2b1d.png)

**Step 2:** In the **Overview** section, review the schedule details: Name, Description, Start time, Stop time, Start Date, End Date, Frequency, and the list of attached virtual machines.

[![Instance schedule overview](/img/migrated/image-1695609025652-03872f17.png)](/img/migrated/image-1695609025652-03872f17.png)

**Step 3:** In the **Attached instances** section, review the list of attached virtual machines and their details: instance name, instance status, latest start time and status, and latest stop time and status.

[![Attached instances list](/img/migrated/image-1695609035976-7b5310a1.png)](/img/migrated/image-1695609035976-7b5310a1.png)
