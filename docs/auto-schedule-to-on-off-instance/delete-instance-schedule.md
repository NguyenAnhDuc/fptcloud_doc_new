# Delete a instance schedule

**Save ý** : Thao tác this will không thực hiện is if lịch has been and is chạy dở chưa xong. Việc xóa instance schedule this will cannot restore lại is. 
**Step 1** : Trên bảng quản trị **Instance Schedule Management,** in phần **Action** , chọn **Delete**
[![file](/img/migrated/image-1695609049584-8754accf.png)](/img/migrated/image-1695609049584-8754accf.png)
**Step 2** : Một hộp thoại cảnh báo sẽ hiện lên, displayed tên schedule và yêu cầu người dùng xác nhận. Gõ chữ **delete** and chọn **Delete schedule** để tiến hành xóa, sau đó hệ thống will xóa hoàn toàn schedule and the virtual machine is attach (if có) thì will is giải phóng and giữ nguyên status nó is đứng.
