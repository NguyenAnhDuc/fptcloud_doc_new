# Manage instance schedules

The **Instance Schedule Management** page lists all instance schedules you have created, including their name, time, frequency, and attached virtual machines.

From this page you can:

- Create a new instance schedule.
- View the details of an instance schedule.
- Attach or detach virtual machines from a schedule.
- Delete an instance schedule.
