# Access the scan results dashboard

1. From the menu, select **Security > Cloud VA**. Click a target to open its statistics dashboard showing results across all scans.

   For targets that have been scanned successfully at least once, the dashboard displays results across the following main tabs:

   | Tab | Description |
   |---|---|
   | **Security status** | Overall scan results summary. |
   | **Server & URL** | List of paths scanned and detailed vulnerabilities per path. |
   | **Risk** | Full list of vulnerabilities found, sorted by severity in descending order. |
   | **Task** | Scan task history. |
   | **Report** | Create and view reports based on the most recent scan results. |

2. Select the **Security Status** tab to view the overall scan results summary.

   For web application targets, the scan results include an overview of detected vulnerabilities, categorized by severity level.
