# Check target scan status

When a target has been successfully scheduled for scanning, the system continuously synchronizes information to FPT Portal. You can check the scan status of a target as follows:

On the **Vulnerability Assessment Management** dashboard, find the target you want to monitor. The most recent scan status is shown in the **Scanning status** and **Last scanned** fields.

| Status | Description |
|---|---|
| **Scheduled** | Scan has been scheduled but has not yet started. |
| **Success** | Scan completed successfully. |
| **Pending** | Reading input and preparing to scan. |
| **Processing** | Scanning the target for vulnerabilities. |
| **Stopped** | Scan was stopped by the user. |
| **Error** | An issue occurred during scanning; the scan was stopped by the system. |
