# Manage Security Rule Set

In addition to the default Rule Set provided by FPT Cloud WAF, users can create a custom Rule Set with custom Rules to suit the specific needs of the protected endpoint.
To add a new custom Rule Set, administrators can follow these steps:
**Step 1** : Select **Security Rule Sets** in the menu to open the **Security Rule Sets** > **Create Security Rule Set.**
[![Userguide FPT WAF 2022 42](/img/migrated/Userguide-FPT-WAF-2022-42-1024x578-3ae15c86.png)](/img/migrated/Userguide-FPT-WAF-2022-42-1024x578-3ae15c86.png)
**Step 2:** Choose **Create Security Rule Set** to open the new Rule Set setup screen and fill in the required information:
  * **Name:** Name of the new Rule Set.
  * **Description** (optional): Description
  * **Replicated From:** Choose the Rule Set you want to clone and modify.

[![Userguide FPT WAF 2022 43](/img/migrated/Userguide-FPT-WAF-2022-43-1024x578-ee9a0838.png)](/img/migrated/Userguide-FPT-WAF-2022-43-1024x578-ee9a0838.png)
**Step 3:** Select **Create Security Rule Set** to save and complete the creation of the Rule Set.
[![Userguide FPT WAF 2022 44](/img/migrated/Userguide-FPT-WAF-2022-44-1024x578-9ec97a49.png)](/img/migrated/Userguide-FPT-WAF-2022-44-1024x578-9ec97a49.png)
##### 
Create Custom Rules in Rule Sets
After creating a Rule Set, you can add custom Rules as needed.
To add a new custom Rule, follow these steps:
**Step 1:** In the **Rule Sets Detail** where you want to add a Rule, choose Create Rule.
[![Userguide FPT WAF 2022 45](/img/migrated/Userguide-FPT-WAF-2022-45-1024x578-d2293080.png)](/img/migrated/Userguide-FPT-WAF-2022-45-1024x578-d2293080.png)
**Step 2:** Enter the necessary information for the Rule:
  * **File:** Name the rule.
  * **Rule ID:** Rule ID (ranging from 5,000,000 to 5,999,999).
  * **Status:** Operational status (Enabled/Disabled).
  * **Content:** Rule content.
  * **Description:** Description for the rule.

[![Userguide FPT WAF 2022 46](/img/migrated/Userguide-FPT-WAF-2022-46-1024x578-74173036.png)](/img/migrated/Userguide-FPT-WAF-2022-46-1024x578-74173036.png)
**Step 3:** Select **Creat Rule** to create the new Rule.
[![Userguide FPT WAF 2022 47](/img/migrated/Userguide-FPT-WAF-2022-47-1024x578-af74e7d2.png)](/img/migrated/Userguide-FPT-WAF-2022-47-1024x578-af74e7d2.png)
