# Manage stream host

A **stream host** is an Nginx feature that forwards TCP/UDP traffic directly to another machine in the network.

1. Select **Hosts** > **Streams** from the menu to open the stream host management screen.

   [![Stream host management screen](/img/migrated/Userguide-FPT-WAF-2022-29-1024x503-9a982ba9.png)](/img/migrated/Userguide-FPT-WAF-2022-29-1024x503-9a982ba9.png)

2. Click **Create Stream** to open the creation screen and enter the details:

   - **Incoming Port**
   - **Forward Host**
   - **Forward Port**

   Optional features:
   - **TCP Forwarding**
   - **UDP Forwarding**

   [![Enter stream host details](/img/migrated/Userguide-FPT-WAF-2022-30-1024x502-e7c3ef10.png)](/img/migrated/Userguide-FPT-WAF-2022-30-1024x502-e7c3ef10.png)

3. Click **Create Stream** to create, or **Create & Add Another** to create and add another stream host.

   [![Stream host detail after creation](/img/migrated/Userguide-FPT-WAF-2022-31-1024x504-c73172be.png)](/img/migrated/Userguide-FPT-WAF-2022-31-1024x504-c73172be.png)
