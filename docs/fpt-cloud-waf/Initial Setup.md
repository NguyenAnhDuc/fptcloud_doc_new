# Initial Setup

If this is your first time using **FPT Cloud WAF** , please follow these steps:
### Create an FPT Cloud account and log in to FPT Cloud Portal
To get started with the **FPT Cloud WAF** , service, you need to log in to the **FPT Portal**.
Visit FPT Portal and log in with your FPT Cloud username and password. Select the correct Tenant, Region, and VPC
If you don't have an FPT Cloud account, visit the homepage at FPT Cloud.
Then, choose the Sign Up function and enter the required information as guided by the system. Our support team will contact you shortly to confirm the details and create your account.
If you encounter any errors during the process, please contact our support team for assistance.
After completing these steps, you are ready to start using WAF on the FPT Cloud system.
