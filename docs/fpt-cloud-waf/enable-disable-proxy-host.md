# Enable or disable a proxy host

1. In the **WAF Dashboard**, select **Hosts** > **Proxy**.

   [![Proxy host list](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. On the proxy host you want to toggle, click the **More** icon, then select **Toggle**.

   [![Select Toggle on proxy host](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)](/img/migrated/Userguide-FPT-WAF-2022-22-1024x578-a1462ad1.png)

3. In the confirmation dialog, click **Run Action** to confirm.

   [![Confirm enable/disable proxy host](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)](/img/migrated/Userguide-FPT-WAF-2022-23-48fa2290.png)
