# Update a proxy host

1. In the **WAF Dashboard**, select **Hosts** > **Proxy**.

   [![Proxy host list](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. On the proxy host you want to update, click the **Edit** icon.

   [![Select Edit on proxy host](/img/migrated/Userguide-FPT-WAF-2022-19-1024x578-234f8c82.png)](/img/migrated/Userguide-FPT-WAF-2022-19-1024x578-234f8c82.png)

3. Enter the updated information.

   [![Edit proxy host details](/img/migrated/Userguide-FPT-WAF-2022-20-1024x578-e34b9e78.png)](/img/migrated/Userguide-FPT-WAF-2022-20-1024x578-e34b9e78.png)

4. Click **Update Proxy** to save and return to the **Proxy Management** page.

   [![Save proxy host](/img/migrated/Userguide-FPT-WAF-2022-21-1024x578-9594ad2a.png)](/img/migrated/Userguide-FPT-WAF-2022-21-1024x578-9594ad2a.png)
