# FPT Cloud WAF

FPT Cloud Web Application Firewall (FPT Cloud WAF) is a firewall service that protects web applications from common attacks including DOS, Local and Remote File Inclusion, SQL Injection, Cross Site Scripting, and attacks exploiting vulnerabilities in NodeJS, Java, and PHP.

## Key features

- Rapid setup with three configuration sizes: Small, Medium, and Large.
- Protects web applications against the OWASP Top 10 threats.
- Automatically monitors, analyzes, and blocks botnet traffic.
- Supports SSL certificate registration and automatic renewal.
- Continuously updates virtual security patches to prevent zero-day attacks.

- **Initial setup** — Create an account and log in to FPT Portal for the first time.
- **Quick start** — Add a WAF server, log in to the dashboard, and configure connections.
