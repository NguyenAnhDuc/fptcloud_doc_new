# View proxy host details

1. In the **WAF Dashboard**, select **Hosts** > **Proxy**.

   [![Proxy host list](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)](/img/migrated/Userguide-FPT-WAF-2022-17-1024x578-72760d67.png)

2. Select the proxy host you want to view.

   [![Proxy host details](/img/migrated/Userguide-FPT-WAF-2022-18-1024x578-0cd1027d.png)](/img/migrated/Userguide-FPT-WAF-2022-18-1024x578-0cd1027d.png)
