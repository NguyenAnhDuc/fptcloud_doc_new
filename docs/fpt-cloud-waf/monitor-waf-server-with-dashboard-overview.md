# Monitor Waf Server With Dashboard Overview

**Home dashboard** is an overview page that allows administrators to have a comprehensive view of the system by monitoring various parameters and statistics of the WAF server.
In the **WAF Dashboard** , select **Overview**. The screen will display continuously updated statistics.  
| **Metrics**  | **Description**  |  
| --- | --- |  
| **Total Hosts**  | The number of Proxy Hosts created under the WAF server.  |  
| **Total Redirection**  | The number of Redirection Hosts created under the WAF server.  |  
| **Total Stream**  | The number of Stream Hosts created under the WAF server.  |  
| **Total Not Found Hosts**  | The number of 404 Not Found Hosts created under the WAF server.  |  
| **Active Connections**  | The number of connections currently connected to the proxy hosts.  |  
| **Security Events**  | The total number of security events that the WAF server has blocked and recorded for the proxy hosts.  |  
| **Top By Attack Type**  | The most frequent types of attacks that the WAF server has blocked and recorded.  |  
| **Top By RuleID**  | The most violated rule IDs by the WAF server.  |  
| **Top By Host**  | The proxy hosts with the highest number of blocked and recorded security events.  |  
| **Top By Source Attack**  | The IP addresses that have been the source of the most frequent attacks recorded by the WAF server.  |
