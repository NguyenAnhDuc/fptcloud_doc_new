# Delete Waf Server

**Note:** This operation will permanently delete all data associated with the WAF server, and recovery will not be possible. Please double-check before proceeding with the deletion.
**Step 1** : To delete a created WAF server, select **Delete WAF** in the **Cloud Web Application Firewall Management.**
**[![Userguide FPT WAF 2022 13](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)](/img/migrated/Userguide-FPT-WAF-2022-13-1024x538-02cafd36.png)**
**Step 2** : Confirm the deletion by selecting **Delete** in the confirmation popup.
**[![Userguide FPT WAF 2022 14](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)](/img/migrated/Userguide-FPT-WAF-2022-14-73224f68.png)**
