# Log In To Waf Dashboard

**Step 1:** After successfully initializing the WAF server, the user needs to access the inbox of the email address used to create the WAF. Retrieve the login information for the WAF Dashboard, including the **Dashboard link** , **Username** , **Password**. 
[![Userguide FPT WAF 2022 3](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)
**Step 2:** Access the **Dashboard link** and log in.
[![Userguide FPT WAF 2022 4](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)
**Note** : User should change the WAF Dashboard password to ensure security during usage.
