# Log in to WAF Dashboard

1. After the WAF server is created successfully, access the inbox of the email used to create the WAF. Retrieve the login credentials: **Dashboard link**, **Username**, **Password**.

   [![Email with WAF Dashboard login credentials](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)](/img/migrated/Userguide-FPT-WAF-2022-3-1024x713-e3f45723.png)

2. Access the **Dashboard link** and log in.

   [![WAF Dashboard login screen](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)](/img/migrated/Userguide-FPT-WAF-2022-4-1024x517-0f1a7786.png)

:::warning
Change your WAF Dashboard password immediately after the first login to ensure security.
:::
